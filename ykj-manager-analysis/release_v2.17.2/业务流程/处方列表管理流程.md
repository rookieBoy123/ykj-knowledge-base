# 处方列表管理流程

> 文档版本：v1.0 | 面向读者：运营管理员、药房管理员

## 概述

处方列表管理用于查看和管理平台上所有医生开具的处方记录。管理员可以按多维度筛选处方，查看处方的电子版文件（PDF或图片格式），患者姓名默认脱敏保护隐私。

## 触发条件

- **前置条件**：已登录管理后台，具备诊疗管理权限
- **入口路径**：左侧菜单 → 诊疗管理 → 处方管理 → 处方列表

## 步骤详解

### 第一步：筛选查询处方

支持以下筛选维度：

| 筛选项 | 说明 |
|--------|------|
| 医生姓名 | 最多输入5个字 |
| 科室 | 树形科室选择 |
| 处方类型 | 中药处方 / 成药处方 |
| 处方编号 | 精确搜索 |
| 开具时间 | 日期范围 |
| 状态 | 开方中 / 待审核 / 审核中 / 审核通过 / 审核不通过 / 已失效 |
| 所属团队 | 仅专家团队功能开启时显示，下拉选择 |

### 第二步：查看处方列表

| 列名 | 说明 |
|------|------|
| 处方编号 | 门诊处方号 |
| 处方类型 | 中药处方（类型=2）/ 成药处方（类型=1） |
| 处方医生 | 开具处方的医生姓名 |
| 所属团队 | 仅专家团队功能开启时显示 |
| 科室 | - |
| 开具时间 | - |
| 诊断 | 临床诊断 |
| 患者姓名 | 脱敏显示（如"张**"），点击可查看全名 |
| 状态 | 开方中(10) / 待审核(20) / 审核中(25) / 审核通过(30) / 审核不通过(40) / 已失效(50) |
| 操作 | 查看电子处方 |

### 第三步：查看电子处方

点击"查看电子处方"，根据处方文件类型自动选择打开方式：
- **PDF文件**：在新窗口中通过PDF查看器打开（`/static/pdf/web/viewer.html`）
- **图片文件**：使用图片预览组件全屏查看

注意：操作列只在处方有电子处方文件（treatmentOrderUrl不为空）时才显示"查看电子处方"按钮。

### 第四步：患者隐私保护

患者姓名采用脱敏显示策略——姓的第一个字保留，其余字符用星号（*）替代。点击脱敏区可以切换显示完整姓名。每次切换只影响当前行，不影响其他行的脱敏状态。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 没有电子处方文件 | 操作列不显示"查看电子处方"按钮 |
| 处方状态变更 | 处方状态是后端决定的，前端只做筛选展示 |
| 专家团队功能未开启 | "所属团队"筛选和表格列均隐藏 |

## 关联功能

- **处方审核**：审核通过/不通过的状态与药师审核流程联动
- **药房发药**：审核通过的处方进入药品调配和发放流程
- **专家团队**：处方的所属团队与专家团队管理模块关联
