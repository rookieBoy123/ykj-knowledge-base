# 会诊受邀机构管理流程

> 文档版本 2026-07-04 | 面向远程医疗管理员

## 概述
远程医疗管理员通过此功能管理可参与远程会诊的外部受邀机构。为每个机构创建账号、录入联系信息，并从机构列表快捷跳转至该机构的医生管理和服务管理页面。

> ⚠️ **权限限制**：此页面仅对角色为"远程会诊管理员"（remoteMedicalManager）的用户开放。其他角色的用户访问时，页面只显示提示文字"当前页面仅供远程医疗管理员使用！"，无法进行任何操作。

## 触发条件
- 入口：诊疗管理 → 远程医疗中心 → 受邀机构管理
- 前置条件：当前登录用户角色必须是"远程会诊管理员"
- 使用场景：新增合作医院作为会诊受邀方、管理受邀机构的联系信息、为受邀机构配置可参与会诊的医生和服务

## 步骤详解

### 第一步：查看受邀机构列表
页面展示所有受邀机构，每条记录包含：

| 信息 | 说明 |
|------|------|
| 机构编号 | 系统自动生成的唯一ID |
| 机构名称 | 受邀机构的名称 |
| 机构账号 | 机构的登录账号 |
| 联系人 | 机构的联系人姓名 |
| 联系电话 | 机构的联系电话 |
| 联系地址 | 机构的联系地址 |
| 状态 | 启用/禁用 |
| 操作 | 编辑、删除、管理服务、管理医生 |

可通过机构名称进行模糊搜索。

### 第二步：新增受邀机构
点击"新增"按钮，弹出填写窗口，包含以下字段：

**基本信息**（新增和编辑均需填写）：
- **机构名称**：必填，最多200个字符
- **联系人**：必填，最多30个字符
- **联系电话**：必填，必须是11位中国大陆手机号（以1开头），不符合格式时提示"请输入正确的手机号"
- **联系地址**：必填，最多500个字符
- **状态**：选择"启用"或"禁用"

**账号信息**（仅新增时显示，编辑时不显示）：
- **机构账号**：必填，最多15个字符
- **密码**：必填，需符合系统密码强度要求
- **确认密码**：必填，必须与密码一致，不一致时提示"两次密码不一致，请重新填写"

### 第三步：编辑机构信息
点击操作列"编辑"，弹出窗口加载当前机构信息，可修改机构名称、联系人、联系电话、联系地址和状态。编辑模式不显示账号和密码字段。

### 第四步：删除机构
点击操作列"删除"，弹出确认框（"确认删除受邀机构【机构名称】吗？"），确认后删除。

### 第五步：管理医生的跳转
点击操作列"管理医生"，跳转到会诊医生管理页面，并携带当前机构的ID（`instId`）作为查询参数，进入该机构的医生列表。

### 第六步：管理服务的跳转
点击操作列"管理服务"，跳转到会诊服务管理页面，同样携带机构ID，进入该机构的服务管理。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 非远程医疗管理员访问 | 页面显示提示文字，隐藏所有操作功能 |
| 手机号格式不正确 | 提示"请输入正确的手机号" |
| 两次密码不一致 | 提示"两次密码不一致，请重新填写" |
| 未填写必填字段 | 对应字段下方出现红色提示文字 |
| 最后一页数据被全部删除 | 自动回退到上一页 |

## 关联功能
- **会诊医生管理**：从此页面跳转，管理该受邀机构下可参与会诊的医生
- **会诊服务管理**：从此页面跳转，管理该受邀机构提供的会诊服务
- **远程会诊列表**：受邀机构会出现在会诊申请的"受邀方"选择中
