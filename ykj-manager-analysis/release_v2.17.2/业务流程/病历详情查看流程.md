# 病历详情查看流程

> 文档版本 v1.0 | 面向读者：运营管理员、客服人员

## 概述

病历详情页面集中展示某次线上问诊的完整医疗记录，包括患者基本信息、病史、就诊信息、会诊记录和处方建议。管理员可以在此页面全面了解一次问诊的来龙去脉，支持查看问诊对话记录和会诊详情。

## 触发条件

- **前置条件**：管理员已登录管理后台，拥有患者病历查看权限
- **入口路径**：患者管理 → 患者详情 → 病历记录列表 → 点击某条记录进入详情

## 步骤详解

### 第一步：进入页面——查看患者信息

页面左侧显示患者基本信息卡片，包含：
- **就诊卡号**：患者在医院HIS系统中的唯一标识
- **姓名、性别、出生日期、年龄**：年龄以"岁"或"月"显示（不满1岁按月龄显示）
- **证件号码**：患者的身份证号
- **监护人信息**：仅当患者有监护人时显示（`hadGuardian === 1`），包含监护人姓名、关系（父亲/母亲/其他）、身份证号。监护人身份证号旁会显示"已校验"标记
- **门慢医保复诊**：如患者登记了慢性病种，显示对应的病种名称标签

右侧为锚点导航菜单，包含：患者信息、病史信息、就诊信息、会诊记录、处方建议。点击任一项可平滑滚动到对应区域。当前所在区域在导航中以蓝色高亮标记。

### 第二步：查看病史信息

显示本次问诊填写的病史资料，内容根据问诊类型（中医/西医）有所不同：

**通用病史字段**：
- 主诉：患者对自身病情的描述
- 现病史
- 既往史、家族史、个人史、过敏史、月经史、婚育史
- 常见体征
- 辅助检查

**中医特有字段**（当病历类型为中医时）：
- 脉象
- 舌象
- 治法

**西医特有字段**（当病历类型为西医时）：
- 治疗意见

所有未填写的字段统一显示"未填写"。

### 第三步：查看就诊信息

显示本次问诊的就诊相关信息：
- **就诊方式**：固定显示"线上就诊"
- **首诊机构、首诊时间**
- **问诊时间**：问诊实际开始的时间
- **服务类型**：图文问诊、视频问诊、咨询服务等
- **科室名称**
- **病情描述**：患者的病情自述
- **病历资料**：患者上传的图片资料，以缩略图网格展示，点击可放大预览。如未上传则显示"记录遗失/不在身边"
- **主治医生**：医生姓名 | 科室 | 职称 | 医院名称（三甲医院标注"三甲"）

底部有"问诊对话记录"按钮，点击弹出对话窗口。

### 第四步：查看问诊对话记录

点击"问诊对话记录"按钮打开弹窗，展示医生和患者之间的完整对话：
- 默认加载最近的消息记录，向上滚动自动加载更早的消息
- 支持文字、图片、语音消息的展示
- 语音消息点击可播放，播放其他语音时自动暂停前一条
- 如果是旧版问诊记录，显示搜索框，可搜索对话内容

### 第五步：查看会诊记录

仅当系统开启了"远程会诊"功能开关（`consultationEnabled`）时显示此区域。

列出与该问诊关联的所有会诊记录，每项显示：
- **会诊时间**
- **会诊状态**：会诊申请(1) → 会诊待备案(2) → 会诊待患者确认(3) → 会诊待开始(4) → 会诊进行中(5) → 待会诊报告(6) → 已完成(7)

点击某条会诊记录可弹出会诊详情弹窗，查看会诊的完整信息。

如果没有关联会诊，显示"暂无数据"。

### 第六步：查看处方建议

显示本次问诊开具的处方列表，每张处方包含：
- **处方类型**：成药处方（type=1）或中药处方（type≠1）
- **处方编号**（门诊号）
- **查看对应订单**按钮：如果处方已生成订单，可跳转到订单详情
- **查看原始处方**按钮：如果有处方PDF文件，在新窗口打开PDF预览；如果是图片则弹窗预览

**成药处方详情**：
- 诊断、辨病、辨证
- 药品列表：药品名称、规格、数量、用法用量（用法、每日次数、每次用量）
- 处方医生

**中药处方详情**：
- 诊断、辨病、辨证
- 药品列表：药品名称及数量（以销售单位显示）
- 处方医生
- 医嘱禁忌：用药说明（煎煮方法）、用药时间（从字典转换显示）、用药禁忌（从字典转换显示）

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 病史字段未填写 | 显示"未填写" |
| 患者未上传病历资料 | 显示"记录遗失/不在身边" |
| 无关联会诊记录 | 显示"暂无数据" |
| 处方无订单号 | 不显示"查看对应订单"按钮 |
| 处方无原始处方文件 | 不显示"查看原始处方"按钮 |
| 功能开关关闭会诊 | 不显示"会诊记录"区域和导航项 |
| 处方列表为空 | 显示空状态占位图 |

## 关联功能

- 跳转到订单详情页（点击处方中的"查看对应订单"）
- 跳转到患者详情
- 查看会诊详情弹窗
- 问诊对话记录弹窗
