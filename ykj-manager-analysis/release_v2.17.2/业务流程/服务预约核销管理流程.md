# 服务预约核销管理流程

> 文档版本 2026-07-06 | 面向管理员/运营人员

## 概述

管理医院线下服务（如体检、健康证、核酸检测等）的预约订单。运营人员可以查询预约信息、核销预约订单（输入核销码确认到店消费），并查看预约详情。预约手机号支持脱敏保护，需单独点击查看真实号码。

## 触发条件

- **前置条件**：拥有运营管理权限的账号登录管理后台
- **入口路径**：管理后台 → 诊疗管理 → 服务项目管理 → 转发人管理

## 步骤详解

### 第一步：查询预约订单

页面提供六种筛选条件：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 预约编号 | 文本输入 | 模糊搜索，最多30字 |
| 预约手机号 | 文本输入 | 精确搜索，最多11位 |
| 预约时间 | 日期范围 | 选择起止日期 |
| 服务名称 | 文本输入 | 预约服务名称，最多20字 |
| 服务方式 | 下拉选择 | 从字典获取选项列表 |
| 服务类型 | 下拉选择 | 区分体检/健康证/核酸检测/检查等 |
| 人员类型 | 下拉选择 | 个人/团体企业 |
| 预约单状态 | 下拉选择 | 按状态筛选 |

点击查询后，列表展示各预约订单的核心信息。

### 第二步：列表查看

列表展示字段：

- 预约编号、预约服务名称、服务类型、人员类型
- 服务方式、价格（元）、联系人
- 联系电话（默认脱敏显示，点击可查看完整号码）
- 预约时间（含时间段范围）
- 状态、服务核销时间
- 操作：查看详情

**联系电话脱敏机制**：列表中手机号默认加密显示。点击某行的联系电话，系统调用后端接口获取完整号码，同一时间只展示一条记录的完整号码（点击新记录时之前展开的号码自动隐藏）。

### 第三步：核销订单

1. 在列表中选中要核销的预约订单（支持勾选）
2. 点击列表上方"核销订单"按钮
3. 弹出核销码输入对话框，输入16位核销码
4. 点击"确定"完成核销
5. 核销后订单状态更新，记录核销时间

### 第四步：查看详情

点击某条记录的"查看详情"，跳转到该预约的完整信息页面（包含预约服务的详细信息、支付情况、核销历史等）。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未输入核销码 | 弹窗有字数限制提示，最多16位 |
| 核销码错误 | 后端返回错误提示 |
| 未选中订单点核销 | 按钮不可用或提示先选择订单 |

## 关联功能

- **服务项目管理**：健康服务包的创建和配置
- **订单管理**：门诊服务订单的查询管理
