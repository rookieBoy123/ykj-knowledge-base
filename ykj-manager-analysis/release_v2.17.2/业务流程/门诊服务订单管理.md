# 门诊服务订单管理

> 文档版本：v1.0 | 最后更新：2026-07-02 | 面向读者：运营人员

## 概述

管理人员可以通过该页面查看和筛选所有患者的问诊订单，支持多种维度筛选、查看问诊对话记录、视频回放、知情同意书确认信息，以及手动结束问诊。

## 触发条件

- **前置条件**：拥有管理后台登录权限
- **入口路径**：管理后台 → 诊疗管理 → 门诊服务订单 → 订单列表

## 步骤详解

### 第一步：筛选订单

进入页面后，可以通过以下条件筛选订单：

| 筛选条件 | 说明 |
|----------|------|
| 问诊编号 | 输入问诊编号精确查找 |
| 服务类型 | 多选：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊、AI图文问诊 |
| 问诊流程 | 多选：在线问诊、线上复诊、慢病医保、预约挂号、快速开方、AI问诊 |
| 问诊状态 | 单选：已预约、问诊中、问诊结束、取消问诊 |
| 支付时间 | 时间范围筛选 |
| 问诊时间 | 时间范围筛选 |
| 就诊时间 | 时间范围筛选 |
| 有无病历 | 全部/有/无 |
| 医生姓名 | 输入医生姓名搜索 |
| 患者姓名 | 输入患者姓名搜索 |
| 所属机构 | 输入机构名称搜索（如开启机构功能） |
| 所属团队 | 下拉选择专家团队（如开启团队功能） |

点击搜索按钮后，列表按分页显示结果，每页默认10条。

### 第二步：查看订单列表

列表展示以下信息：

| 列名 | 说明 |
|------|------|
| 问诊编号 | 唯一标识 |
| 服务类型 | 图文问诊/视频问诊等 |
| 问诊流程 | 在线问诊/线上复诊等 |
| 医生姓名 | 接诊医生 |
| 支付时间 | 患者付款时间 |
| 诊费 | 问诊费用金额 |
| 患者姓名 | 脱敏显示（姓+*号），需点击查看完整姓名 |
| 患者联系方式 | 脱敏显示，需点击查看完整号码 |
| 所属机构 | 患者所属机构（如开启） |
| 问诊时间 | 问诊创建时间 |
| 就诊时间 | 实际就诊开始时间 |
| 状态 | 问诊当前状态 |
| 有无病历 | 有/无 |
| 操作 | 可执行的操作按钮 |

**注意**：患者姓名和联系方式默认脱敏显示。点击脱敏字段可申请查看完整信息，查看后仅当前行显示明文，其他行保持脱敏。切换行查看时，之前行的明文会重新隐藏。

### 第三步：执行操作

每条订单记录可执行以下操作（根据状态和配置动态显示）：

| 操作 | 适用条件 | 说明 |
|------|----------|------|
| 查看详情 | 所有订单 | 查看问诊对话记录（IM聊天记录含文字/图片/语音/视频消息） |
| 结束问诊 | 问诊中状态（flowStatus=20） | 管理员强制结束当前问诊，确认后立即生效 |
| 视频记录 | 视频问诊类型（serviceType=2） | 查看问诊过程中的视频回放列表，点击可播放 |
| 知情同意书 | 已签署知情同意书 | 查看患者签署的知情同意书信息，包括确认人、IP地址、设备信息、微信OpenId、签署时间 |

### 第四步：查看问诊详情

点击"查看详情"按钮后，弹出对话记录窗口：

- **普通问诊**：显示IM聊天记录（文字消息、图片、语音、视频等），支持向下滚动加载更早的消息
- **AI问诊**：显示AI助手与患者的对话记录，分为"问诊对话记录"和"AI问诊报告"两种视图：
  - 对话记录：展示AI和用户的完整对话，AI回复可能包含卡片形式的结构化信息
  - AI问诊报告：包含主诉、病史信息（现病史/既往史/过敏史/家族史等）、体检（辅助检查）、AI诊断建议

所有对话记录窗口均显示水印（当前用户名@系统名称），防止截图泄露。

### 第五步：查看视频记录

对于视频问诊订单，点击"视频记录"按钮后：
- 显示该问诊的所有视频录制列表，每条记录显示录制时间
- 点击"播放"按钮在新窗口中播放视频

### 第六步：查看知情同意书

点击"知情同意书确认信息"按钮后，弹出窗口显示：
- 知情同意书文本内容
- 确认人姓名
- 签署时网络IP地址
- 设备信息
- 微信用户OpenId
- 签署时间

### 第七步：结束问诊

点击"结束问诊"按钮后：
1. 弹出确认提醒："确认是否结束所选问诊?"
2. 点击确定后调用后台接口
3. 成功后提示"成功结束问诊"，自动刷新列表

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 问诊编号为空 | 结束问诊时提示"问诊编号为空！"无法继续 |
| 筛选条件中"问诊流程"选了"全部" | 自动清空该条件（等同于不限制） |
| 问诊状态复合值（如"已预约"对应20,0） | 后台自动解析为flowStatus=20且isCheckIn=0 |
| 切换患者联系方式查看 | 之前显示明文的行自动恢复脱敏，同时只能查看一行完整号码 |
| 问诊对话记录为空 | 提示"没有找到相关问诊对话记录!" |
| 分页数据异常（当前页无数据但非第1页） | 自动回退到上一页重新加载 |

## 关联功能

- **处方列表**：从订单可关联查看该问诊开具的处方
- **患者病历详情**：从患者信息可跳转查看完整病历
- **医生列表**：从医生姓名可跳转查看医生详情
- **问诊列表查询**：另一个问诊查询入口（更细化的查询页面）
