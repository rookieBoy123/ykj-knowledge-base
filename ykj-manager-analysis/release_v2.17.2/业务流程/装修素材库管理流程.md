# 装修素材库管理流程

> 文档版本：2026-07-16 | 面向读者：运营、管理员

## 概述

装修素材库是管理后台装修设计模块的图片和视频资产管理功能。管理员可以在素材库中建立分类目录树，上传图片和视频素材，用于自定义页面的搭建。素材按分类组织，支持上传、删除、移动、预览等操作。

## 触发条件

- 需要为装修页面准备图片/视频素材
- 需要整理和归类已有的设计素材
- 需要在装修设计中选择素材插入页面

## 页面入口

管理后台 → 运营管理 → 装修设计 → 素材管理

## 步骤详解

### 第一步：进入素材库

页面分为左右两栏布局：
- **左侧**：分类目录树，展示素材的分类层级结构，默认选中「全部」
- **右侧**：素材展示区，显示当前选中分类下的所有素材文件

顶部有两个 Tab 切换：「图片」和「视频」，分别管理不同类型的素材。

### 第二步：管理分类目录

分类树支持三级层级结构：

**添加分类**：点击「全部」旁的加号按钮，或在已有分类上点击「更多 → 添加分类」。弹出表单需要选择「上级分类」和填写「分类名称」。系统校验：
- 分类名称不能为空（提示「请先填写分类名称」）
- 上级分类不能为空（提示「请选择上级分类」）
- 同一父级下分类名称不能重复（服务端提示「当前目录下已经有相同名称的分类了，请重新输入」）

**编辑分类**：在已有分类上点击「更多 → 编辑分类」，修改上级分类或分类名称。注意：编辑时不能将上级分类选为当前分类本身（前端提示「上级分类不能选当前分类」）。

**删除分类**：在已有分类上点击「更多 → 删除分类」，弹出二次确认弹窗后执行删除。服务端会**级联删除**该分类下的所有子分类，但不会自动删除分类下的素材文件。

### 第三步：上传素材

选中目标分类后，点击「上传图片」或「上传视频」按钮。

- **图片格式**：支持 `.jpg`、`.jpeg`、`.png`、`.gif`
- **视频格式**：支持 `.mp4`
- 素材会上传到 OSS 对象存储，服务端自动提取文件名和后缀
- 如果素材列表为空，前端提示「要保存的素材不能为空」

### 第四步：浏览和选择素材

素材支持两种展示模式：
- **大图模式**（默认）：以网格形式展示素材缩略图
- **列表模式**：以列表形式展示，可显示更多信息

在装修设计器中调用素材库时，会显示「使用选中图片/视频」按钮，确认后将选中素材插入到装修页面中。非设计器场景（素材管理页面）不显示此按钮。

### 第五步：移动素材

选中需要移动的素材后，通过「图片/视频移动至」下拉框选择目标分类。下拉框内嵌分类树，点击目标分类即可完成移动。

如果素材已被并发删除，服务端返回空列表时不执行任何操作。如果更新失败，提示「移动素材失败」。

### 第六步：删除素材

选中素材后点击「删除图片/视频」按钮。支持**批量删除**，同时选中多个素材一次性删除。删除失败时提示「删除素材失败」。素材从 OSS 上的原始文件不会被自动删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 批量保存时素材列表为空 | 服务端提示「要保存的素材不能为空」 |
| 保存素材失败（数据库写入异常） | 服务端提示「保存素材失败」 |
| 分类下无素材 | 显示空状态页「暂无数据」 |
| 批量删除失败 | 服务端提示「删除素材失败」 |
| 移动素材失败 | 服务端提示「移动素材失败」 |
| 分类名称重复 | 服务端提示「当前目录下已经有相同名称的分类了」 |

## 关联功能

- **自定义页面设计**：设计器中插入图片时调起素材库选择器
- **底部菜单配置**：设置图标时调起素材库选择器
- **系统分类管理**：素材分类数据通过 `SysCategoryServiceImpl`（151 行）管理，类型固定为 `MATERIAL`
- **OSS 对象存储**：素材文件存储在 OSS 中，前端展示时拼接 OSS 域名为完整 URL
