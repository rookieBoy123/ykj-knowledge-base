# 药师列表管理流程

> 文档版本：v1.0 | 面向读者：运营管理员、药房管理员

## 概述

药师列表管理用于管理平台所有药师的账号信息。管理员可以新增、编辑、删除药师，管理药师的启用/禁用状态，进行CA认证同步和手机号变更等操作。

## 触发条件

- **前置条件**：已登录管理后台，具备医院管理权限
- **入口路径**：左侧菜单 → 医院管理 → 药师列表

## 步骤详解

### 第一步：查看药师列表

进入页面后看到药师列表表格，展示信息：

| 列名 | 说明 |
|------|------|
| 姓名 | 药师姓名 |
| 性别 | - |
| 联系电话 | 默认脱敏显示，点击可查看完整号码 |
| 邮箱地址 | - |
| CA认证状态 | 仅CA功能开启时显示（未认证/认证中/已认证/认证失败） |
| CA用户同步状态 | 详细同步步骤状态 |
| 入驻时间 | - |
| 状态 | 启用/禁用开关 |
| 操作 | 编辑、删除、同步资料、去认证、变更手机号 |

### 第二步：搜索筛选药师

支持按药师姓名搜索（最多5个字）。

### 第三步：新增药师

点击"新增"按钮，打开药师信息录入表单。需要填写的信息包括：
- **基本资料**：姓名、性别、手机号、邮箱
- **职业信息**：所属医院、所在科室、职称、职务、药师专业
- **学历信息**：学历、民族
- **证件信息**：身份证号、身份证正反面照片、职称证书照片
- **头像**：个人头像上传

新增药师时系统需要具备 `hosp:pharmacists:write` 权限。

### 第四步：编辑药师

点击"编辑"按钮修改该药师的各项信息，操作流程与新增类似。

### 第五步：启用/禁用药师

在列表的状态列直接切换开关：
- 启用：药师可以正常登录和使用系统
- 禁用：药师账号被停用

### 第六步：药师认证相关操作

与医生类似，药师也支持CA（电子签章）认证：
- **同步资料**：将药师信息同步到CA系统（仅CA功能开启时显示，且药师CA状态为"未同步"或"认证失败"时可用）
- **去认证**：引导药师进行CA实名认证（仅在CA状态为"认证中"时显示）

### 第七步：变更手机号

点击"变更手机号"，输入新手机号。系统实时校验：手机号不能为空、格式要正确、且不能与已有医生或药师的手机号重复。修改成功后弹窗确认"手机号码修改成功！"。

### 第八步：删除药师

点击"删除"，弹出确认弹窗"此操作将永久删除该药师, 是否继续?"，确认后不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 变更手机号为空 | 提示"手机号不能为空" |
| 手机号格式不对 | 提示"手机号格式错误" |
| 手机号与已有医生/药师重复 | 提示"手机号码已存在，请重新输入" |
| 删除操作 | 弹出二次确认弹窗 |
| CA同步资料按钮不可用 | 仅当CA功能开启 且 状态为"未同步(1)"或"认证失败(4)"时才可用 |
| CA去认证按钮不可用 | 仅当CA功能开启 且 状态为"认证中(2)"时才显示 |

## 关联功能

- **CA认证系统**：药师也需要电子签章认证
- **字典管理**：职称、职务、学历、民族等下拉选项来自系统字典
- **科室管理**：药师所在科室从科室树加载
- **药房管理**：药房审核处方时需要指派药师
