# 商务邀请医生管理流程

> 文档版本 2026-07-04 | 面向运营管理员

## 概述

管理员查看各商务人员邀请的医生列表，支持批量变更邀请商务、导出数据。用于管理商务与医生的绑定关系。

## 触发条件

- **前置条件**：已有商务人员通过审核，且商务通过分享二维码邀请了医生注册
- **入口路径**：运营管理 → 商务管理 → 商务邀请医生列表

## 步骤详解

### 第一步：查看邀请医生列表

进入页面后，系统加载所有被商务邀请的医生记录。列表展示以下信息：
- 编号（医生ID）
- 医生姓名、职称、所属医院、科室
- 邀请商务（该医生由哪位商务邀请）
- 注册时间
- 状态（填写资料中 / 审核中 / 启用 / 驳回）
- 成功入驻时间

支持按以下条件筛选：
- **注册时间**：日期范围筛选
- **商务姓名**：按邀请的商务姓名搜索
- **状态**：下拉选择（全部 / 填写资料中 / 审核中 / 启用 / 驳回）

### 第二步：批量修改邀请商务

勾选多条医生记录后，点击"批量修改邀请商务"按钮。弹出窗口后：
1. 确认已选中的医生列表
2. 从下拉框中选择新的商务人员
3. 点击"确定"提交

系统会批量将这些医生的邀请商务变更为新选择的商务。

> ⚠️ 如果没有勾选任何医生就点击该按钮，系统会提示"请选择要修改的记录！"

### 第三步：导出数据

点击"导出"按钮，系统将当前筛选条件下的所有数据导出为 Excel 文件（文件名：`商务邀请医生.xls`）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未勾选医生就点击"批量修改" | 提示"请选择要修改的记录！" |
| 状态为"填写资料中"的医生 | 医生尚未完成资料填写，尚未进入审核流程 |
| 状态为"驳回"的医生 | 可重新提交资料，或由其他商务重新邀请 |
| 导出时数据量大 | 显示加载中提示，等待导出完成 |

## 关联功能

- **商务审核管理**：商务提交资料后需先通过审核，才能邀请医生
- **医生管理**：邀请成功后医生出现在医生管理列表中，可进行后续管理
