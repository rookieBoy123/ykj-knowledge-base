# 收费项目管理流程

> 文档版本 2026-07-03 | 面向运营/财务人员

## 概述

管理医院的所有收费项目，包括费用编码、医保编码、费用名称、类别、单价等。收费项目是医生开具处方和计算费用的基础数据，支持启用/禁用控制。

## 触发条件

- **前置条件**：已在数据字典中配置费用类别（分类编码"fylb"）
- **入口路径**：运营管理 → 收费项目管理

## 步骤详解

### 第一步：筛选查询

支持三个筛选维度：

1. **费用编码**：按费用编码搜索（最多20位）
2. **费用名称**：按费用名称搜索（最多100字）
3. **费用类别**：从数据字典（分类编码"fylb"）动态加载，如药品费、检查费、治疗费等
4. **状态**：全部 / 启用 / 禁用

### 第二步：查看收费项目列表

| 列 | 说明 |
|:---|:---|
| 费用编码 | 费用的唯一编码标识 |
| 医保编码 | 对应医保结算系统中的编码（如有） |
| 费用名称 | 费用项目的名称 |
| 费用类别 | 所属的费用分类 |
| 单价(元) | 该项费用的单价 |
| 单位 | 计费单位（如次、盒、支、片等） |
| 状态 | 启用/禁用，通过行内开关切换 |

### 第三步：新增收费项目

点击"新增"按钮（需要"hosp:charge-items:write"权限），弹出编辑弹窗：
1. 填写费用编码（必填，唯一标识）
2. 填写医保编码（选填，医保结算时使用）
3. 填写费用名称
4. 选择费用类别
5. 设置单价和单位
6. 保存后默认为"启用"状态

### 第四步：编辑收费项目

点击操作列的"编辑"按钮，弹出编辑弹窗修改费用信息。费用编码建议创建后不要轻易修改（涉及到已有处方的费用关联）。

### 第五步：启用/禁用收费项目

在列表行内点击"状态"列的开关：
- **启用**：医生开方时可以选择该收费项目
- **禁用**：医生开方时该收费项目不再显示，但不影响已有处方

切换即时生效。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 费用类别选项为空 | 数据字典"fylb"未配置，需先在字典管理中维护费用类别 |
| 禁用收费项目后已有处方 | 不影响已有处方中的费用数据，仅新处方不可选择 |
| 新增时无写权限 | "新增"按钮不显示（权限控制：hosp:charge-items:write） |
| 费用编码重复 | 服务端会校验唯一性并提示错误 |

## 关联功能

- **数据字典**：费用类别通过字典"费用类别（fylb）"管理
- **处方开具**：医生在开具处方时从收费项目中选择费用项
- **医保结算**：医保编码字段用于与医保系统对接
- **费用统计**：财务统计时按收费项目汇总

## 业务说明

收费项目是医院收费体系的基础数据。每个收费项目都有：
- **费用编码**：内部唯一标识，一般与医院HIS系统编码保持一致
- **医保编码**：医保目录中的对应编码，用于医保结算报销
- **费用类别**：从字典"fylb"中选取，如挂号费、诊疗费、药品费、检查费、治疗费等
- **状态控制**：禁用后不影响历史数据，但会从医生开方界面中移除
