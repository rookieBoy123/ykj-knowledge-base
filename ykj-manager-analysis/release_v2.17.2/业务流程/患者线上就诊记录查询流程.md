# 患者线上就诊记录查询流程

> 面向管理员 | 管理后台功能

## 概述

管理员通过输入患者身份证号，查看该患者在本平台的所有线上就诊记录和病历详情，包括处方信息和可预览的电子病历/电子处方文件。

## 触发条件

- **前置条件**：拥有管理后台访问权限，需要有患者身份证号和访问令牌（通过患者列表或其他入口传入）
- **入口路径**：管理后台 → 通过患者列表点击进入，或通过携带 `idCard` 和 `token` 参数的 URL 直接访问

## 步骤详解

### 第一步：页面加载与患者识别

页面通过 URL 参数获取患者身份证号和访问令牌。如果缺少任一参数，页面不进行任何查询。

系统根据身份证号查询患者信息：
- 找到患者 → 继续加载就诊记录列表
- 未找到患者 → 页面提示"没有该患者相关信息"

### 第二步：查看就诊记录时间线

页面左侧以时间线形式展示该患者的所有线上就诊记录，每条记录显示：
- **门诊号**：并标注"复诊"标签
- **接诊医生**：显示医生姓名、所属科室、职称
- **主诉**：患者自述的病情描述
- **诊断信息**：分别显示西医诊断、中医辨病、中医辨证（如该次问诊有相应诊断）
- **电子病历按钮**：如该次就诊有对应的电子病历文件，显示"电子病历"按钮

点击任意一条时间线卡片，右侧展示该次就诊的详细病历。

空状态：如果该患者没有线上就诊记录，页面中央显示"暂无线上就诊记录!"。

### 第三步：查看病历详情

右侧区域展示选中就诊记录的完整病历信息：

**基本信息区**：
- 就诊人姓名、性别、年龄（小于1岁显示"X月"，大于1岁显示"X岁"）
- 门诊号、初诊标志（固定显示"复诊"）
- 监护人信息（如有监护人）：显示监护人姓名和身份证号
- 就诊类型（图文问诊/视频问诊等）
- 就诊机构名称、就诊时间、就诊科室、接诊医生

**病史信息区**（未填写的字段统一显示"未填写"）：
- 主诉、现病史、既往史、过敏史、个人史、家族史
- 月经史、婚育史（如适用）

**中医特有信息**（仅当病历类型为中医时显示）：
- 中医四诊：脉象和舌象
- 治法

**西医特有信息**（仅当病历类型为西医时显示）：
- 治疗意见

**诊断信息**：
- 西医诊断
- 中医辨病、中医辨证（仅中医病历显示）

**病历资料**：患者上传的病历图片，点击可放大预览。如无图片则显示"记录遗失/不在身边"。

### 第四步：查看处方信息

当本次就诊有处方时，下方显示"处方信息"区域。每张处方独立展示：

**西药处方**（type=1）：
- 表格形式展示药品清单：药品名称、规格、剂型、每次用量、用药途径、用药频次、数量

**中药处方**（type=2）：
- 药品清单逐行展示：药品名称+数量+单位
- 剂数、用药方法（内服/外用）
- 服用说明：X日X剂，一剂分X次服用
- 用药说明（煎煮方法等）

每张处方底部显示：
- 处方医生姓名
- 审核药师姓名（未审核则显示"-"）
- "电子处方"按钮（如有电子处方文件）

### 第五步：预览电子病历/电子处方

点击"电子病历"或"电子处方"按钮，在新窗口中通过内置 PDF 阅读器预览对应的文件。

---

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| URL缺少 idCard 或 token 参数 | 页面不发起任何查询，空白等待 |
| 根据身份证号未找到患者 | 提示"没有该患者相关信息" |
| 患者无线上就诊记录 | 显示"暂无线上就诊记录!" |
| 病历某字段为空 | 统一显示"未填写" |
| 无病历资料图片 | 显示"记录遗失/不在身边" |
| 就诊记录列表为空 | 第一页就返回空→显示空状态 |
| 某次就诊无电子病历文件 | 不显示"电子病历"按钮 |
| 某处方无审核药师 | 审核药师栏显示"-" |

## 关联功能

- **患者列表管理**：通常从患者列表页携带身份证号跳转至此
- **电子病历管理**：与此页面共享病历数据源
- **处方管理**：处方数据与处方列表管理共享
