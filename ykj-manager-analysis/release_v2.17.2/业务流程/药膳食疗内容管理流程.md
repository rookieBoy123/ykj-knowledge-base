# 药膳食疗内容管理流程

> 文档版本 2026-07-04 | 面向运营/内容编辑

## 概述
药膳食疗管理用于维护小程序"中医百科"模块中的药膳食疗内容。运营人员在这里创建、编辑和删除药膳条目，包括药膳名称、主辅料、功效、适应症、功能作用和做法等。这些内容最终展示在小程序用户端的药膳食疗浏览页面中。

## 触发条件
- 菜单路径：运营管理 → 医学百科 → 药膳食疗
- 使用场景：新增药膳内容、修改已有药膳信息、下架不再推荐的药膳

## 页面布局
页面为标准的列表+新增/编辑弹窗模式。顶部有四个搜索筛选条件，列表展示药膳的主要信息。

## 步骤详解

### 第一步：搜索筛选
支持四个筛选条件（均从数据字典加载选项）：
- **药膳名称**：手动输入，模糊搜索
- **食材**：下拉选择，选项来自字典 `yssl-sc`
- **功效**：下拉选择，选项来自字典 `yssl-gx`
- **适应症**：下拉选择，选项来自字典 `yssl-syz`

注：食材筛选的实现比较特殊——用户选择的是字典编码（如"shanyao"），但查询时会自动将编码转换为中文名称（如"山药"）再传给后端。

### 第二步：浏览列表
列表字段包括：
- **药膳名称**：药膳的显示名称
- **功效**：多个功效用顿号（、）连接显示，无数据时显示"-"
- **适应症**：多个适应症用顿号连接显示，无数据时显示"-"
- **功能作用**：对功效的详细描述
- **主料**：主要食材
- **辅料**：辅助食材
- **做法**：制作步骤描述
- **最后修改时间**：最近一次更新的时间
- **操作**：编辑、删除按钮

### 第三步：新增药膳
1. 点击"新增"按钮，弹出编辑表单
2. 填写各项内容（具体字段由 AddOrEdit 子组件定义）
3. 保存后弹窗关闭，列表跳转到第1页并刷新

### 第四步：编辑药膳
1. 点击某条药膳的"编辑"按钮
2. 弹出编辑表单，自动填充当前数据
3. 修改后保存，列表保持在当前页码刷新

### 第五步：删除药膳
**单个删除**：
1. 点击某行的"删除"按钮
2. 弹出确认框："数据删除后不可恢复，您确认要删除吗？"
3. 确认后删除，页码重置为第1页，列表刷新

**批量删除**：
1. 勾选多条药膳（通过列表左侧复选框）
2. 点击"批量删除"按钮
3. 弹出确认框，确认后批量删除勾选的所有条目
4. 如未勾选任何条目，不执行操作

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未勾选条目就点击批量删除 | 无响应（selectedSelections 检查为空时直接 return） |
| 删除时取消操作 | 不执行删除，列表不变 |
| 食材下拉选项为空 | 查询时食材条件不生效 |
| 药膳名称超过20个字符 | 输入框自动限制，最大20个字符 |

## 关联功能
- **数据字典**：食材、功效、适应症的下拉选项来自字典系统（`yssl-sc`、`yssl-gx`、`yssl-syz`）
- **小程序药膳食疗浏览**：这里维护的内容对应用户在小程序中看到的药膳食疗推荐
