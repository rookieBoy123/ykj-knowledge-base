# 经典名方管理流程

> 文档版本 2026-07-04 | 面向读者：内容运营人员

## 概述

管理员在后台维护中医经典名方百科内容，包括处方的基本信息、适应症、功效作用及四个维度的详细说明（基本用法、药物作用、用药方法、注意事项）。内容通过富文本编辑器编写，支持插入图片，最终展示在患者端"中医百科-经典名方"模块。

## 触发条件

- 入口路径：管理后台 → 运营管理 → 医学百科 → 经典名方管理
- 适用角色：拥有医学百科管理权限的管理员

## 步骤详解

### 第一步：查看经典名方列表

进入页面后，系统展示经典名方列表，每条记录包含：
- **处方名称**：如"六味地黄丸""银翘散"等
- **处方类型**：由字典"jtmf-lx"定义（如汤剂、散剂、丸剂等）
- **适应症**：适用的疾病症状（可多选）
- **书籍出处**：如《伤寒论》《金匮要略》等
- **功效作用**：简要的治疗功效描述
- **最后修改时间**：最近编辑时间

列表支持按处方名称（关键词）、处方类型、适应症、书籍出处四个维度筛选。筛选选项全部来自数据字典。

### 第二步：新增经典名方

点击「新增」按钮，跳转到编辑页面，需要填写：

**基本信息区：**
- **处方名称**：必填，最多20字
- **处方类型**：必填，从下拉选择（如汤剂、丸剂、散剂等）
- **适应症**：可选多个，如"感冒""咳嗽""头痛"等
- **书籍出处**：必填，如《伤寒论》《金匮要略》等
- **功效作用**：必填，最多100字，简要说明治疗功效
- **展示封面图**：可选，建议上传 200×200px 的 1:1 比例图片

**中药详情区（四个标签页，均使用富文本编辑器）：**
- **基本用法**：处方的基本使用方法（必填），可插入图片、设置文字格式
- **药物作用**：药物的治疗机理（必填）
- **用药方法**：具体的用药指导（必填）
- **注意事项**：使用禁忌和提醒（必填）

四个标签页均不能为空。如果未填写某个标签页就提交，系统会自动切换到对应的标签页并提示。

### 第三步：编辑经典名方

点击列表中的「编辑」按钮，跳转到编辑页面，系统自动回填已有的所有信息。修改后点击「确定」保存。

### 第四步：删除经典名方

- **单个删除**：点击「删除」按钮，二次确认后删除
- **批量删除**：勾选多条后点击「批量删除」，二次确认后删除

删除后数据不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 处方名称为空 | 前端提示"请填写处方名称" |
| 处方类型未选择 | 前端提示"请选择处方类型" |
| 书籍出处未选择 | 前端提示"请选择书籍出处" |
| 功效作用未填写 | 前端提示"请填写功效作用" |
| 基本用法未填写 | 提交时自动切换到"基本用法"标签页，提示"请填写基本用法" |
| 药物作用未填写 | 提交时自动切换到"药物作用"标签页，提示"请填写药物作用" |
| 用药方法未填写 | 提交时自动切换到"用药方法"标签页，提示"请填写用药方法" |
| 注意事项未填写 | 提交时自动切换到"注意事项"标签页，提示"请填写注意事项" |
| 批量删除未勾选 | 静默无提示 |

## 关联功能

- **患者端经典名方浏览**：小程序「中医百科 → 经典名方」展示此处维护的内容
- **数据字典**：处方类型、适应症、书籍出处的下拉选项全部来自字典（类别编码：jtmf-lx、jtmf-syz、jtmf-sj）
