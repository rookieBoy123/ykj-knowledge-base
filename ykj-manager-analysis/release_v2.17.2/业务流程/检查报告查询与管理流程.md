# 检查报告查询与管理流程

> 文档版本 2026-07-04 | 面向读者：运营人员、产品经理

## 概述

管理后台提供检查报告的查询和报告填写功能。运营人员可以按检查日期、患者姓名、报告状态筛选查看患者的各项检查报告，并可以为尚未出报告的检查填写报告内容，或查看已完成的检查报告详情。

## 触发条件

- 运营人员需要查看某位患者的检查报告时
- 医生需要回填检查报告结果时
- 需要按时间范围统计已出/未出报告时

## 步骤详解

### 第一步：进入检查报告列表

进入「诊疗管理 → 检查报告」页面，系统默认加载全部检查报告记录，每页显示10条。

列表展示的信息包括：
- **检查号**：每份检查的唯一编号
- **姓名、性别、年龄**：患者基本信息
- **检查项目**：该项检查的具体类型（如血常规、CT等，由数据字典维护）
- **科室**：开单科室
- **检查时间**：进行检查的日期
- **报告医生**：填写报告的医生姓名
- **报告时间**：报告完成的日期
- **状态**：未出报告 / 已出报告

### 第二步：筛选查找

页面顶部提供三个筛选条件：

1. **检查日期**：选择日期范围，按检查时间筛选
2. **姓名**：按患者姓名搜索（最多输入5个字）
3. **状态**：下拉选择「全部」「未出报告」「已出报告」

点击搜索后自动刷新列表，回到第一页。

### 第三步：查看或填写报告

每条记录的操作列会根据报告状态显示不同按钮：

- **已出报告**（状态=1）：显示「查看报告」按钮，点击后可查看报告详情
- **未出报告**（状态=0）：显示「填写报告」按钮，医生点击后可进入报告填写界面，提交后状态变为"已出报告"

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 搜索姓名超过5个字 | 输入框限制最大5个字符，无法输入更多 |
| 报告尚未生成 | 状态显示"未出报告"，只有「填写报告」按钮可点击 |
| 查询结果为空 | 列表区域显示空数据提示 |
| 接口返回错误 | 表格不刷新数据，保持上次查询结果 |

## 关联功能

- **患者病历详情**：从患者详情页可查看该患者的所有检查报告
- **患者列表**：可从患者维度查询关联的检查记录
