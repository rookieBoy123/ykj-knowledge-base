# 中药特殊煎煮方法管理流程

> 文档版本 1.0 | 面向读者：运营管理员、药房管理员

## 概述
管理人员可以为特定的中药设置"特殊煎煮方法"。例如某些药材需要"先煎"、"后下"、"包煎"等特殊处理方式。医生在开具包含该药材的处方时，系统会自动提示对应的特殊煎煮方法。

## 触发条件
- 药房需要维护中药的特殊煎煮要求
- 入口：管理后台 → 医院管理 → 药品信息 → 中药特殊煎煮方法

## 步骤详解

### 第一步：查看特殊煎煮方法列表
进入页面后看到列表，显示：
- 序号
- 药品名称（通用名）
- 特殊煎煮方法（如"先煎"、"后下"、"冲服"等）

顶部搜索框可按"药品通用名称"搜索。

### 第二步：新增特殊煎煮方法
点击"新增"按钮，弹出表单填写：
- **药品通用名称**：必填，最多255个字符。新增时可输入，编辑时不可修改
- **特殊煎煮方法**：必填，从下拉列表中选择

特殊煎煮方法下拉选项来自数据字典（分类编码：`zytsjzff`，即"中药特殊煎煮方法"），由系统预设，不可在页面上自行创建。

校验规则：
- 药品名称不能为空或纯空格
- 药品名称不能包含特殊符号（如 @#$% 等）
- 药品名称长度在2-20个字符之间
- 特殊煎煮方法必须选择

### 第三步：编辑特殊煎煮方法配置
点击"编辑"按钮，可以修改已配置的药品的特殊煎煮方法。药品名称不可修改（编辑时输入框置灰），只能更换煎煮方法的下拉选择。

> **注意**：本页面没有删除功能（操作列只有"编辑"按钮）。如需移除某个药品的特殊煎煮配置，需联系开发人员或通过数据库操作。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药品名称为空或纯空格 | 提示"药品名称不能为空且不能为空格" |
| 药品名称包含特殊符号 | 提示"药品名称不能包含特殊符号" |
| 药品名称超过20字符 | 表单最大长度限制为255，但校验规则要求2-20字符 |
| 未选择特殊煎煮方法 | 提示"请选择特殊煎煮方法" |
| 同名药品重复添加 | 服务端返回"该药品已存在"错误 |
| 找不到煎煮方法选项 | 下拉列表为空，需检查数据字典配置 |

## 关联功能
- **处方开具**：医生在开具含该药材的处方时，煎煮方法会自动显示
- **数据字典**：特殊煎煮方法的可选值来自数据字典的"中药特殊煎煮方法"分类
