# 处方药师审核与电子签名管理流程

> 文档版本 v1.0 | 面向读者：运营人员、药房管理员

## 概述

药师在审核处方时，需要上传个人电子签名图片用于处方签章。此页面展示当前登录药师的基本信息和资质证书（身份证、资格证书），并支持上传、裁剪和提交电子签名图片。

## 前置条件

- 当前用户必须是已录入系统的药师身份
- 药师基本信息（姓名、性别、手机号、邮箱、证书照片）已在系统中维护

## 触发入口

管理后台 → 诊疗管理 → 处方管理 → 处方药师审核 → 签名录入

---

## 步骤详解

### 第一步：查看药师信息

进入页面后，系统自动加载当前药师的资料：

| 信息区域 | 展示内容 |
|----------|---------|
| 基本信息 | 药师姓名、性别、手机号码、邮箱地址 |
| 身份证 | 身份证正反面图片，支持点击放大查看 |
| 资格证书 | 药师资格证书图片，支持点击放大查看 |

如果当前用户没有关联药师身份，页面显示空状态提示"暂无数据"。

### 第二步：上传电子签名

签名区域支持以下操作：

1. 点击「点击上传签名」按钮，选择本地图片文件
2. 上传前系统校验：
   - **格式限制**：仅支持 jpg/png 格式
   - **大小限制**：不超过 2MB
   - 不符合条件时弹出错误提示

3. 上传成功后会打开「图片剪裁」弹窗：
   - 拖拽调整裁剪区域（宽高比固定 2:1）
   - 鼠标滚轮可放大缩小图片
   - 点击「确认」完成裁剪
   - 裁剪后的签名图片即时预览在签名区域

### 第三步：提交签名

1. 确认签名图片无误后，点击「提交」按钮
2. 系统校验：
   - 如果未上传签名，提示"签名必须上传!"
   - 如果药师信息未加载成功，无法提交
3. 提交成功后提示"操作成功"

---

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 上传非jpg/png格式图片 | 提示"上传图片只能是jpg/png格式!" |
| 上传超过2MB的图片 | 提示"上传图片大小不能超过 2M!" |
| 未上传签名直接提交 | 提示"签名必须上传!" |
| 当前用户无药师身份 | 页面显示空状态，无法操作 |

## 关联功能

- **处方审核列表**：药师审核处方时使用此签名进行签章
- **药师管理**：药师基本信息在药师列表管理中维护
