# 量表发布与管理流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、医务管理人员

## 概述

量表管理用于创建和发布健康测评量表（如焦虑自评量表、抑郁筛查量表等）。管理员可创建量表、添加题目、设置评分标准，发布后供医生在问诊中下发给患者填写。

## 触发条件

- 需要新增一种健康测评量表
- 需要修改现有量表的题目或评分标准
- 需要禁用不再使用的量表

## 步骤详解

### 第一步：量表列表管理

量表列表页面展示所有已创建的量表，支持以下筛选：
- 量表分类：下拉选择量表分类
- 量表名称：模糊搜索（最多128字）
- 创建时间：时间范围筛选

列表展示信息：量表名称、所属分类、创建时间、启用状态、操作按钮。

### 第二步：新增量表

点击「新增量表」进入量表编辑页面。量表编辑分为三个步骤：

**步骤1 — 基本信息**：
- 填写量表名称
- 选择量表分类
- 填写完成后点击「下一步」进入题目编辑

**步骤2 — 量表题目**：
- 左侧面板显示题目列表（支持新增、搜索题目名称）
- 点击题目可在右侧编辑题目内容、选项和分值
- 支持题目上移/下移调整顺序
- 支持删除题目
- 首次进入时若无题目，显示空状态提示

**步骤3 — 评分标准**：
- 设置量表的评分规则
- 定义不同分数段对应的评价等级
- 如：0-10分→正常、11-20分→轻度、21-30分→中度、31分以上→重度

### 第三步：量表发布

量表在草稿状态下需要发布才能使用。点击「发布」按钮时，系统会自动检查量表内容完整性：

- **有题目但无评分标准**：提示"该量表有内容未填写，请确认填写完整再发布"，可选择「完善量表」跳转到未完成部分继续编辑
- **题目和评分标准均完整**：直接发布成功

发布成功后的量表状态变为"已发布"。

### 第四步：量表状态管理

量表有三种状态：

| 状态 | 含义 | 可执行操作 |
|------|------|-----------|
| 草稿 | 正在编辑中，未发布 | 编辑、发布 |
| 已发布 | 已发布，可在问诊中使用 | 禁用、上移、下移 |
| 禁用 | 已禁用，不再可用 | 启用 |

- **发布**（草稿→已发布）：需通过内容完整性检查
- **禁用**（已发布→禁用）：操作后量表不再在问诊中可选
- **启用**（禁用→已发布）：重新启用量表
- **上移/下移**：调整已发布或已禁用量表在小程序中的显示顺序（支持跨页排序）
- **编辑**：仅草稿状态可编辑，进入编辑页面从上次未完成步骤开始

### 第五步：量表分类管理

量表分类独立管理，在量表列表页通过分类下拉筛选。分类选项在页面加载时动态获取。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 发布时题目列表为空 | 弹窗提示"该量表有内容未填写"，可跳转继续完善 |
| 发布时评分标准为空 | 弹窗提示"该量表有内容未填写"，可跳转继续完善 |
| 量表编辑时未保存就切换页面 | 量表保持草稿状态，数据已自动保存 |
| 禁用的量表试图编辑 | 编辑按钮不可见，只能启用后重新编辑 |
| 搜索题目无结果 | 显示"无查询结果"空状态 |

## 关联功能

- 医生端问诊中下发量表给患者
- 患者端填写量表并提交
- 量表测评结果自动计算和展示
