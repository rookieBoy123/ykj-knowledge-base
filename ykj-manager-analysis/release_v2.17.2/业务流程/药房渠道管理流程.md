# 药房渠道管理流程

> 文档版本：2026-07-06 | 面向读者：运营/管理人员

## 概述

管理人员在后台统一管理平台的药房渠道，支持线上药房和线下药房的创建、编辑、启停、库存管理开关和排序调整。药房的类型（线上/线下）受系统模式（严格模式）和配置开关控制。

## 触发条件

- **前置条件**：管理人员已登录后台，拥有药房管理权限（`med:pharmacies:write`）
- **入口路径**：后台 → 医院管理 → 药房渠道

## 步骤详解

### 第一步：查看药房列表

进入页面自动加载药房列表。系统根据当前模式决定显示内容：

**严格模式（system.mode='strict'）或 未开启线下药房**：
- 隐藏"药房类型"列
- 隐藏"药房类型"搜索条件
- 仅显示和管理线上药房

**普通模式且开启线下药房**：
- 显示完整列表，可按药房类型筛选（全部/线上药房/线下药房）

列表展示：

| 列名 | 说明 |
|------|------|
| ID | 药房编号 |
| 药房类型 | 线上药房 / 线下药房（严格模式下隐藏） |
| 药房名称 | 药房名称 |
| 负责人 | 药房负责人姓名 |
| 状态 | 启用/禁用开关 |
| 库存管理 | 启用库存/禁用库存状态标签 |
| 创建时间 | 药房创建时间 |

### 第二步：搜索与筛选

支持按以下条件筛选：
- **药房名称**：输入药房名称搜索（最多20个字符）
- **状态**：启用 / 禁用
- **药房类型**：全部 / 线上药房 / 线下药房（严格模式下隐藏）

### 第三步：新增药房

点击「新增」按钮跳转到药房编辑页面。在编辑页填写药房信息后保存。

### 第四步：编辑药房

点击某条记录的「编辑」按钮，跳转到编辑页面（携带药房ID参数）。

### 第五步：管理药品

点击「管理药品」按钮，跳转到该药房的药品管理页面，管理该药房下的药品库存和上架状态。

### 第六步：排序调整

支持上下移动和置顶/置底操作：
- **排至最前**：将该药房排序到第一位
- **排至最后**：将该药房排到最后一位
- **上移**：将该药房向前移动一位
- **下移**：将该药房向后移动一位

### 第七步：启用/禁用

点击状态开关可快速启用或禁用某个药房。
- 启用：调用 `pharmaciesEnable` 接口
- 禁用：调用 `pharmaciesDisable` 接口

### 第八步：路由缓存

从药品管理页面返回时，保留之前的搜索条件和分页状态。从其他页面进入时，自动重置搜索条件。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 严格模式或未开启线下药房 | 隐藏药房类型相关列和搜索条件 |
| 操作失败 | 提示"操作失败"并回滚状态 |

## 关联功能

- **药品管理**：点击"管理药品"跳转到药房药品管理页
- **药房编辑**：新增/编辑药房跳转到编辑页面
- **系统配置**：系统模式（严格/普通）和线下药房开关控制药房类型显示
