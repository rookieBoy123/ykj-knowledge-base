# 远程会诊管理

> 文档版本：v1.0 | 最后更新：2026-07-02 | 面向读者：运营人员、远程医疗管理员

## 概述

远程会诊管理是供"远程医疗管理员"角色使用的专属功能，用于管理跨医疗机构的远程会诊申请全流程——从创建会诊申请、等待应答、收费、开始会诊、进行中、结束到归档的完整生命周期。

## 触发条件

- **前置条件**：登录用户的角色为"远程医疗管理员"（remoteMedicalManager），非该角色用户看到提示"当前页面仅供远程医疗管理员使用！"
- **入口路径**：管理后台 → 诊疗管理 → 远程医疗中心 → 远程会诊

## 步骤详解

### 第一步：筛选会诊申请

进入页面后，可通过以下条件筛选：

| 筛选条件 | 说明 |
|----------|------|
| 会诊单号 | 输入会诊单号精确查找 |
| 申请科室 | 输入科室名称搜索 |
| 申请医师 | 输入医生姓名搜索 |
| 患者姓名 | 输入患者姓名搜索 |
| 拟申请会诊机构 | 下拉选择目标会诊机构 |
| 状态 | 下拉选择当前状态 |

### 第二步：查看会诊列表

列表展示以下字段：

| 列名 | 说明 |
|------|------|
| 会诊单号 | 唯一标识 |
| 申请科室 | 发起会诊的科室 |
| 申请医师 | 发起会诊的医生 |
| 患者姓名 | 会诊患者 |
| 患者性别 | 男/女 |
| 患者年龄 | 患者年龄 |
| 拟申请会诊机构 | 目标会诊机构名称 |
| 拟申请会诊服务 | 目标会诊服务名称 |
| 拟申请会诊科室 | 目标科室名称 |
| 拟申请会诊医生 | 目标医生姓名 |
| 拟申请会诊时间 | 计划会诊时间 |
| 状态 | 当前流程状态 |
| 会诊时间 | 实际会诊时间 |
| 操作 | 根据状态动态显示 |

### 第三步：会诊全流程状态管理

远程会诊有10种状态，对应不同的可执行操作：

| 状态码 | 状态名称 | 可执行操作 |
|--------|----------|-----------|
| 1 | 草稿 | 编辑、删除 |
| 2 | 待应答 | 撤回 |
| 9 | 已撤回 | 编辑、删除 |
| 10 | 已拒绝 | 编辑、取消 |
| 3 | 待收费 | 收费、取消 |
| 4 | 待开始 | 取消、开始会诊 |
| 5 | 进行中 | 进入会诊、结束会诊 |
| 6 | 已结束 | 补充资料、归档 |
| 7 | 已归档 | —（终态） |
| 8 | 已取消 | —（终态） |

**状态流转图**：
```
草稿 → 待应答 → 待收费 → 待开始 → 进行中 → 已结束 → 已归档
  ↓       ↓        ↓        ↓                   ↓
 删除   已撤回    取消     取消                取消(8)
          ↓
        (可重新编辑提交)
```

### 第四步：新增会诊申请

点击"新增"按钮弹出申请表单，填写以下信息：
- 患者基本信息
- 申请科室和医生
- 拟申请的会诊机构、服务、科室、医生
- 计划会诊时间
- 可先"保存草稿"或直接"提交申请"

### 第五步：编辑草稿/已撤回/已拒绝的申请

对于状态为"草稿"、"已撤回"、"已拒绝"的记录，可点击"编辑"修改会诊信息后重新提交。

### 第六步：撤回待应答的申请

对于状态为"待应答"的记录，点击"撤回"按钮：
1. 弹出确认提示："确认撤回申请吗？"
2. 确认后申请状态变为"已撤回"

### 第七步：收费确认

对于状态为"待收费"的记录，点击"收费"按钮：
1. 弹出确认提示："确认已收费吗？"
2. 确认后状态流转为"待开始"

### 第八步：取消会诊

对于状态为"已拒绝"、"待收费"、"待开始"的记录，点击"取消"按钮：
1. 弹出取消原因填写窗口
2. 必须填写取消原因（最多500字）
3. 提交后状态变为"已取消"

### 第九步：开始会诊

对于状态为"待开始"的记录，点击"开始会诊"按钮：
1. 调用开始接口
2. 自动跳转到"会诊中心"页面进行实时会诊操作

### 第十步：进入会诊

对于状态为"进行中"的记录，点击"进入会诊"按钮跳转到会诊中心页面。

### 第十一步：结束会诊

对于状态为"进行中"的记录，点击"结束会诊"按钮：
1. 弹出确认提示："确认结束会诊吗？"
2. 确认后状态变为"已结束"

### 第十二步：补充资料

对于状态为"已结束"的记录，点击"补充资料"按钮：
1. 弹出文件上传窗口
2. 支持上传治疗方案及效果资料（图片、PDF、Word、Excel等）
3. 单个文件不超过10MB
4. 可多次上传补充
5. 已上传的文件可查看和管理

### 第十三步：归档

对于状态为"已结束"的记录，点击"归档"按钮：
1. 弹出提示："归档后不再支持补充上传资料，确认相关会诊资料已全部上传吗？"
2. 确认后状态变为"已归档"（终态，不可再操作）

### 第十四步：删除

对于状态为"草稿"或"已撤回"的记录，点击"删除"按钮：
1. 弹出提示："删除后数据不可恢复，确认删除吗？"
2. 确认后记录被永久删除

### 查看详情

任意状态的记录均可点击"查看详情"查看完整信息。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 非远程医疗管理员访问 | 页面显示"当前页面仅供远程医疗管理员使用！" |
| 取消会诊未填写原因 | 提示"请输入取消会诊原因"，阻止提交 |
| 补充资料未选择文件 | 提示"请选择文件后再保存"，阻止提交 |
| 归档操作 | 二次确认后执行，归档后不可补充资料 |
| 删除操作 | 二次确认"删除后数据不可恢复"后才执行 |

## 关联功能

- **会诊中心**：从"进入会诊"或"开始会诊"操作跳转，进行实时会诊
- **受邀机构管理**：管理可参与会诊的医疗机构
- **医生管理（会诊）**：管理可参与会诊的医生
- **服务管理（会诊）**：管理会诊服务类型和配置
- **患者病历详情**：可关联查看患者的完整病历信息
