# 问诊评价管理流程

> 文档版本 2026-07-04 | 面向运营管理员

## 概述

管理员查看、审核、管理患者对问诊服务的评价内容，控制评价在前端是否可见。

## 触发条件

- **前置条件**：患者完成问诊后提交了评价
- **入口路径**：诊疗管理 → 门诊服务订单 → 评价列表

## 步骤详解

### 第一步：查看评价列表

进入页面后，系统加载所有评价记录。列表展示：
- 会员名称
- 问诊类型（图文/视频/快速问诊）
- 接诊医生
- 分数
- 评价内容（长文本支持悬浮查看全文）
- 评价时间
- 显示状态（显示/隐藏的开关）
- 操作按钮

支持筛选：
- **关键字**：按评价内容搜索
- **评价时间**：日期范围
- **状态**：待审核 / 审核通过 / 审核不通过

### 第二步：审核评价

每个待审核的评价（flowStatus=10）旁边有"通过"按钮。点击后该评价状态变为"审核通过"并自动设置为"显示"。

### 第三步：查看评价详情

点击"详情"按钮，弹出评价详情窗口：
- 会员名称、问诊类型、接诊医生
- 评价日期、分数
- **所选标签**：以标签形式展示（如"态度好""专业"等）
- **评价内容**：完整文本
- **是否显示**：开关按钮（1=显示，0=隐藏）

在详情窗口中：
- 如果评价为"待审核"状态，按钮显示为"通过"，点击后同时完成审核和显示设置
- 如果评价已审核，按钮显示为"确定"，仅更新显示状态

### 第四步：控制评价显示

列表中有"显示/隐藏"开关，直接点击即可切换评价在小程序端的可见性。隐藏后患者在小程序中看不到该条评价。

### 第五步：删除评价

点击"删除"按钮，系统弹出二次确认："此操作将永久删除该文件，是否继续？"。确认后评价被删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 评价无标签 | 详情中标签区域为空 |
| 删除操作取消 | 提示"已取消删除" |
| 评价已审核通过 | "通过"按钮隐藏，仅显示"详情"和"删除" |

## 关联功能

- **门诊服务订单**：评价来自问诊订单完成后患者的反馈
- **服务评价审核**：管理后台另一入口可进行评价审核操作
