# 服务项目管理流程（管理后台）

> 文档版本：release/v2.17.2 | 面向读者：运营、管理员

---

## 一、服务项目字段

| 字段 | 说明 | 必填 |
|------|------|------|
| 服务名称 | 显示名称 | 必填 |
| 服务类型 | 从字典加载（体检/健康证/检查/核酸等） | 必填 |
| 服务方式 | 上门(202)/自行到院(206)/两者(208) | 必填 |
| 人员类型 | 个人(1)/团体(2)/两者(3) | 必填 |
| 费用编码 | 内部编码 | 必填 |
| 医保编码 | 医保系统编码 | 选填 |
| 开单科室 | 关联科室 | 选填 |

---

## 二、筛选与列表

支持按服务名称、服务方式、服务类型、人员类型、启用状态筛选。列表直接在线切换启用/禁用状态。

---

## 三、关键规则

1. 医保编码选填（自费项目不填）
2. 禁用后患者端完全不可见，已生成订单不受影响
3. 状态切换即时生效，无需保存
4. 团体/企业类型预约只收定金
