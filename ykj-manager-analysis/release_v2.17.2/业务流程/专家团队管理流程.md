# 专家团队管理流程

> 文档版本 2026-07-03 | 面向运营人员

## 概述

管理在患者端展示的专家团队。每个专家团队包含团队名称、头像、首席专家、团队简介等信息，可以设置展示/隐藏状态，支持排序（置顶/上移/下移/置底），以及管理团队中的医生成员。

## 触发条件

- **前置条件**：已有入驻医生可供添加到团队中
- **入口路径**：运营管理 → 专家团队

## 步骤详解

### 第一步：查看团队列表

进入页面后展示所有专家团队：

| 列 | 说明 |
|:---|:---|
| 编号 | 系统自动分配的编号 |
| 团队头像 | 圆形头像缩略图，未设置时显示默认头像 |
| 团队名称 | 团队名称 |
| 首席专家 | 团队的首席专家姓名 |
| 团队简介 | 团队的简要介绍 |
| 展示状态 | 显示/隐藏，通过行内开关切换 |
| 修改时间 | 最近修改时间 |

可按"团队名称"和"展示状态"筛选。

### 第二步：新增团队

点击"新增"按钮，跳转到专家团队详情编辑页：
1. 上传团队头像
2. 填写团队名称
3. 选择首席专家
4. 编写团队简介
5. 保存后团队出现在列表中

### 第三步：编辑团队

点击"编辑"按钮，跳转到详情编辑页修改团队信息（名称、头像、首席专家、简介等）。

### 第四步：管理团队成员

点击"管理成员"按钮，跳转到成员管理页面：
1. 查看当前团队中的成员列表
2. 添加新成员（从已入驻医生中选择）
3. 移除现有成员
4. 成员可以是团队中的普通专家，不一定是首席专家

### 第五步：设置展示状态

在列表行内点击"展示状态"列的开关：
- **显示**：团队信息在患者端展示
- **隐藏**：团队信息在患者端隐藏
- 切换为"隐藏"时会弹出确认提示："设置展示状态为'隐藏'后患者端将不显示该专家团队信息，您确认要修改吗？"

### 第六步：调整排序

每个团队行操作列提供四个排序按钮：

| 按钮 | 效果 |
|:---|:---|
| 排至最前 | 将该团队移动到列表最顶部（患者端展示优先级最高） |
| 上移 | 将该团队向上移动一位 |
| 下移 | 将该团队向下移动一位 |
| 排至最后 | 将该团队移动到列表最底部（患者端展示优先级最低） |

排序立即生效并刷新列表。

### 第七步：删除团队

点击"删除"按钮，系统弹出确认提示"删除后数据不可恢复，您确认要删除该专家团队吗？"，确认后永久删除。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 未设置团队头像 | 显示系统默认头像图片（圆形占位图） |
| 隐藏团队 | 患者端不可见，但后台仍可查看和编辑 |
| 删除团队 | 团队数据永久删除，但成员医生信息不受影响（医生仍是平台入驻医生） |
| 排序并发操作 | 每次排序操作独立执行，可能需手动刷新 |
| 删除后列表最后一页无数据 | 自动回退到上一页 |

## 关联功能

- **医生管理**：团队成员从已入驻医生中选择
- **患者端首页**：专家团队在患者端首页展示，用户可查看团队详情和成员
- **专家团队详情页**：患者端可查看团队的详细介绍和成员医生列表

## 业务说明

专家团队功能用于在患者端展示医院的优势学科和专家阵容。每个团队由一名"首席专家"领衔，可包含多名成员医生。团队在患者端按排序顺序展示，管理员可以通过置顶操作将重点团队排在前面。
