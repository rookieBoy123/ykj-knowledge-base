# 服务项目添加与编辑流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、系统配置人员

## 概述

服务项目是门诊预约服务的基础配置单元。管理员可新增或编辑服务项目，配置服务名称、类型、价格、服务方式、排班时间等完整信息。配置完成后，服务项目会显示在小程序的预约页面供患者选择。

## 触发条件

- 医院新增了一项门诊服务（如体检、核酸检测）
- 需要调整现有服务的价格或排班
- 需要修改服务须知或包含的项目内容

## 步骤详解

### 第一步：进入新增/编辑页面

从服务项目列表页点击「新增项目」或「编辑」，进入配置表单页面。新增时服务名称和服务类型可以自由填写；编辑时这两个字段锁定不可修改。

### 第二步：填写基本信息

**服务名称**（必填）：
- 字数限制：2-20字
- 不能包含特殊符号（除小括号外的 ~!@#$^&* 等）
- 编辑模式下锁定不可修改

**服务类型**（必填）：
- 从数据字典的「服务类型（fwlx）」中下拉选择
- 编辑模式下锁定不可修改

### 第三步：配置支持人员类型与价格

至少勾选一种人员类型：

| 人员类型 | 价格设置 |
|----------|---------|
| 个人 | 设置个人服务价格（元），支持两位小数 |
| 团体/企业 | 设置定金（元），支持两位小数 |

默认勾选个人；勾选团体/企业需填写定金，勾选后该服务显示在团体/企业预约服务中。

### 第四步：配置项目明细（自动发票模式）

当系统开启自动发票功能时，需要配置以下内容：

**项目明细**（必填）：
- 从收费项目库中选择一个或多个项目
- 每个项目设置数量，系统自动计算金额（单价 × 数量）
- 同一项目不能重复选择
- 支持新增/删除明细行

**开单科室**（必填）：
- 从科室树形结构中选择，支持搜索过滤

### 第五步：配置服务方式

至少勾选一种服务方式：

| 服务方式 | 配置内容 |
|----------|---------|
| 自行到院 | 医院名称（最多256字）、地址（支持地图搜索获取经纬度） |
| 上门服务 | 上门服务费（元），价格统计在总费用中 |

地图选址支持高德地图和腾讯地图两种模式（由系统配置决定）。点击地址输入框的「搜索」按钮打开地图选择位置，自动回填地址和经纬度。

### 第六步：配置服务排班

勾选周一至周日中可预约的日期，勾选表示可预约。排班按每月对应日期显示，默认显示近2个月的排班。

### 第七步：配置开放时间段

支持勾选两种时间段类型：

**工作时间段**（可添加多个）：
- 每个时间段设置起始时间和结束时间（5分钟步长，范围06:00-23:59）
- 支持添加/删除时段
- 后续时间段起始时间不能早于前一个时间段的结束时间

**非工作时间段**：
- 设置一个非工作时段
- 可设置非工作时间段费用（选填，价格自动统计在总费用中）

**重要校验**：
- 工作时间段和非工作时间段不能有重叠
- 勾选了对应类型必须填写对应的时间值

### 第八步：上传图片

上传服务展示图片，建议尺寸 750×750（1:1比例）。图片将显示在小程序预约页面。

### 第九步：设置状态与内容

**状态**：启用/禁用。禁用后不在小程序显示，可再次编辑后启用。

**包含项目**（必填）：通过富文本编辑器填写服务包含的项目内容。

**服务须知**（选填）：通过富文本编辑器填写服务须知说明。

**客服电话**（必填）：支持手机号（11位）或座机号（区号+号码）格式。

### 第十步：提交保存

点击「确认提交」按钮，系统执行完整校验后保存。新增调用新增接口，编辑调用编辑接口。成功提示后返回上一页。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未选择任何支持人员类型 | 提示"请选择支持人员类型" |
| 未选择任何服务方式 | 提示"请选择服务方式" |
| 项目明细重复选择同一项目 | 提示"项目明细选择重复" |
| 项目明细选择不完整（缺项目或数量） | 提示"项目明细选择有误" |
| 工作时间段勾选了但未填写时间 | 提示"没有勾选工作时间段但填写了相关值，请勾选" |
| 非工作时间段勾选了但未填写时间 | 提示"没有勾选非工作时间段但填写了相关值，请勾选" |
| 工作与非工作时间段重叠 | 提示"工作时间段与非工作时间段时间重复,请重新修改" |
| 经纬度为空 | 提示"经纬度不能为空，请点击搜索从地图进行获取" |
| 包含项目为空 | 提示"包含项目不能为空且不能为空格" |
| 服务名称包含特殊符号 | 提示"服务名称不能包含特殊符号（除()）" |
| 客服电话格式不正确 | 提示"请填写正确的联系方式" |

## 关联功能

- 关联收费项目管理（项目明细中的项目来源）
- 关联科室管理（开单科室的选择来源）
- 关联数据字典（服务类型下拉列表）
