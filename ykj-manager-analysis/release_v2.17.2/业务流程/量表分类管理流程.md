# 量表分类管理流程

> 文档版本 2026-07-04（第5轮） | 面向运营/产品人员

## 概述

量表分类用于对系统内的各类心理测评量表进行分组管理，方便用户在小程序端按分类浏览和选择量表。管理员可以创建、编辑、启用/禁用分类，并调整分类的显示顺序。

## 触发条件

- 前置条件：具有量表管理权限的后台用户
- 入口路径：后台 → 诊疗管理 → 量表管理 → 量表分类

## 步骤详解

### 第一步：查看分类列表

进入页面后，系统展示所有量表分类的列表，每列显示：
- **量表分类**：分类的名称
- **状态**：启用或禁用（可在列表中直接切换）
- **创建时间**：分类创建的时间
- **操作**：编辑、上移、下移

列表支持按分类名称和状态（启用/禁用）筛选。

### 第二步：新增分类

点击「新增」按钮，弹出对话框，需要填写：
- **分类名称**：必填，最长128个字符，系统会自动去除首尾空格
- **状态**：默认「启用」，可改为「禁用」

点击保存后，新分类会出现在列表末尾。

### 第三步：编辑分类

点击某条分类的「编辑」按钮，弹出编辑对话框（与新增对话框相同）。可以修改分类名称和状态，保存后生效。

> 注意：目前分类名称变更不校验是否与已有分类重名（代码中有 `validExistsName` 方法占位但未实现校验逻辑），所以管理员需要自行注意避免创建重名分类。

### 第四步：调整分类顺序

每个分类都有「上移」和「下移」按钮：
- **上移**：将该分类与上一个分类交换位置。列表第一条不显示上移按钮。
- **下移**：将该分类与下一个分类交换位置。列表最后一条不显示下移按钮。

顺序调整后，小程序端量表分类的展示顺序会同步更新。

### 第五步：启用/禁用分类

在列表中可以直接通过开关切换分类的启用/禁用状态，无需弹窗确认。禁用后，该分类下的量表在小程序端将不可见。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 分类名称为空或纯空格 | 表单校验拦截，提示"请输入名称" |
| 状态未选择 | 表单校验拦截，提示"请选择" |
| 删除分类 | 代码中有删除逻辑但当前列表页没有暴露删除按钮（无法删除分类） |

## 关联功能

- 量表列表管理：分类被用于筛选和归类量表
- 小程序端量表：用户通过分类浏览量表
- 量表创建编辑：创建量表时需要选择一个分类
