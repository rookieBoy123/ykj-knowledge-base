# 业务开展情况管理流程

> 文档版本 v1.0 | 面向读者：运营管理员

## 概述

管理员按月记录和维护互联网医院的经营指标数据，包括人员配置、财务状况、资产情况等共计15项经营数据，支持新增、编辑和删除操作。

## 触发条件

- 前置条件：拥有 `hosp:institution-businesses:write` 权限的管理员
- 入口路径：医院管理 → 业务开展情况

## 步骤详解

### 第一步：查看经营数据列表

进入页面后，以表格形式展示所有已录入的月度经营数据。表格列包含：

| 列名 | 说明 |
|------|------|
| 月份 | 如 2024-06 |
| 职工总数 | 医院在职员工总数 |
| 日均坐诊医生数 | 每天出诊的医生平均数 |
| 客户服务人员总数 | 客服团队人数 |
| 总收入 / 总支出 | 万元为单位 |
| 负债 / 固定资产 / 流动资产 / 总资产 | 万元为单位 |
| 创建时间 / 修改时间 | 操作记录时间 |

支持按月份筛选数据，页面默认每页展示10条。

### 第二步：新增月度经营数据

点击「新增」按钮，弹出填写表单。表单包含以下字段（**全部必填**）：

**人员配置类**：
- 月份（月份选择器）
- 职工总数
- 日均坐诊医生数
- 客户服务人员总数

**财务指标类**（单位：万元）：
- 无形资产及开办费
- 对外投资
- 净资产
- 负债
- 固定资产
- 流动资产
- 总收入
- 总支出
- 总资产

**其他**：
- 派出（分支）机构数量
- 业务用房面积（m²）

### 第三步：月份唯一性校验

新增时，系统会对所选月份进行后台校验。如果该月份已有数据记录，表单的月份字段会提示「当前月份已存在」，阻止重复提交。每条记录按月份唯一。

编辑时，如果月份没有变更（与原始值相同），则直接通过校验；如果变更了月份，则重新校验新月份是否已被占用。

### 第四步：编辑经营数据

点击列表中某行的「编辑」按钮，弹出表单（与新增相同），数据自动回填。修改完成后点击确定保存。

### 第五步：删除经营数据

点击「删除」按钮，弹出二次确认框：「此操作将永久删除该项, 是否继续?」。确认后数据被永久删除，列表自动刷新。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 新增时月份已存在 | 表单校验提示"当前月份已存在"，阻止提交 |
| 修改月份为已存在的月份 | 同上，校验不通过 |
| 必填字段为空 | 各字段对应提示（如"请输入职工总数"） |
| 取消删除操作 | 提示"已取消删除"，不执行删除 |

## 关联功能

- 数据看板 → 经营数据可能用于看板的统计展示
- 互联网医院信息管理 → 同一模块下的基础信息管理
