# 业务开展情况管理流程

> 文档版本 2026-07-04 | 面向读者：运营管理员、机构管理人员

## 概述

管理员按月填报互联网医院的业务开展情况数据，包括职工总数、日均坐诊医生数、资产状况、收支等核心经营指标，用于监管上报和内部管理。支持月度维度的新增、编辑、删除操作。

## 触发条件

- **前置条件**：具有业务开展情况管理权限
- **入口路径**：管理后台 → 医院管理 → 业务开展情况

## 步骤详解

### 第一步：查看月度数据列表

进入页面后展示已填报的业务开展情况列表，按月份分组，每条记录显示：
- **月份**
- **职工总数**
- **日均坐诊医生数**
- **客户服务人员总数**
- **总收入**（万元）
- **总支出**（万元）
- **负债**（万元）
- **创建时间**
- **修改时间**

列表支持按月份筛选（月份选择器），可快速定位特定月份的数据。

### 第二步：新增月度数据

点击列表顶部「新增」按钮，弹出表单弹窗，需要填写以下指标：

| 字段 | 说明 | 必填 |
|------|------|------|
| 月份 | 选择年月 | 是（且不能重复） |
| 职工总数 | 医院职工总人数 | 是 |
| 客户服务人员总数 | 客户服务岗位人数 | 是 |
| 日均坐诊医生数 | 日均在线坐诊医生数量 | 是 |
| 无形资产及开办费 | 无形资产和开办费用（万元） | 是 |
| 对外投资 | 对外投资总额（万元） | 是 |
| 净资产 | 净资产金额（万元） | 是 |
| 负债 | 负债总额（万元） | 是 |
| 固定资产 | 固定资产金额（万元） | 是 |
| 流动资产 | 流动资产金额（万元） | 是 |
| 总收入 | 总收入金额（万元） | 是 |
| 总支出 | 总支出金额（万元） | 是 |
| 总资产 | 总资产金额（万元） | 是 |
| 派出（分支）机构数量 | 分支机构数量 | 否 |
| 业务用房面积 | 业务用房面积（m²） | 否 |

> 金额类字段单位均为「万元」，业务用房面积单位为「m²」。

### 第三步：月份唯一性校验

保存时系统自动校验月份是否重复：
- 如果该月份已有数据，保存时提示「当前月份已存在」
- 编辑已有记录时不会触发重复校验（编辑的是同一条记录）
- 月份选择器和已存在月份数据的联动校验是实时的

### 第四步：编辑月度数据

点击列表行的「编辑」按钮，弹窗回填当前数据，允许修改所有字段。修改后点击「确定」保存。

### 第五步：删除月度数据

点击列表行的「删除」按钮，弹出确认框「此操作将永久删除该项, 是否继续?」，确认后删除该月份数据。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 月份重复 | 保存时校验同月份是否已存在，存在则提示「当前月份已存在」，不允许保存 |
| 月份未选择 | 提示「请选择月份」 |
| 必填字段为空 | 对应字段标红并提示具体错误信息（如「请输入职工总数」） |
| 新增时月份为空 | 表单校验拦截，提示选择月份 |
| 删除确认取消 | 提示「已取消删除」，不执行删除操作 |

## 关联功能

- **数据看板**：业务开展情况数据可汇总到数据看板展示
- **机构管理**：数据与所属机构关联
- **监管上报**：数据用于向监管部门提交互联网医院运行情况
