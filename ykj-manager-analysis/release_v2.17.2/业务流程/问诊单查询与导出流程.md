# 问诊单查询与导出流程

> 文档版本：2026-07-04 | 面向读者：运营人员、财务人员

## 概述

管理员在此页面查询所有问诊单的订单信息和支付明细，包括订单号、问诊单编号、服务类型、问诊流程、医生姓名、患者姓名、诊费、平台服务费、医生收益、问诊状态和支付状态。支持导出为Excel文件。

## 触发条件

- 前置条件：具有问诊管理权限的管理员账号登录
- 入口路径：「诊疗管理 → 门诊服务订单 → 问诊单查询」

## 筛选条件

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 订单编号 | 文本输入 | 模糊搜索，最长20字符 |
| 创建时间 | 日期范围 | 默认预设最近30天 |
| 服务类型 | 多选下拉 | 图文问诊/视频问诊/咨询服务/快速问诊/快速续方/医生复诊/商城问诊/AI图文问诊 |
| 问诊流程 | 多选下拉 | 在线问诊/线上复诊/慢病医保/预约挂号/快速开方/AI问诊 |
| 问诊状态 | 下拉选择 | 已预约/问诊中/问诊结束/取消问诊 |
| 支付状态 | 下拉选择 | 未支付/已支付 |
| 医生姓名 | 文本输入 | 最多5字符 |
| 患者姓名 | 文本输入 | 最多5字符 |
| 支付时间 | 日期范围 | 订单实际支付时间范围 |
| 所属团队 | 下拉选择 | 仅专家团队功能开启时显示 |
| 问诊单编号 | 文本输入 | 最长30字符 |

> 注：选择"全部"问诊流程（serviceClassifies=0）时，该条件不生效，相当于不过滤。

## 列表字段

| 字段 | 说明 |
|------|------|
| 订单号 | 唯一标识，宽度180px |
| 问诊单编号 | 问诊单唯一标识，宽度180px |
| 服务类型 | 图文问诊/视频问诊等 |
| 问诊流程 | 在线问诊/线上复诊等 |
| 医生姓名 | 接诊医生 |
| 所属团队 | 仅专家团队功能开启时显示 |
| 患者姓名 | 脱敏显示（姓+*号），可点击查看明文 |
| 联系方式 | 脱敏显示，点击后验证身份显示明文 |
| 创建时间 | 问诊单创建时间 |
| 订单支付时间 | 特殊格式化显示 |
| 诊费(元) | 保留两位小数 |

| 医生收益(元) | 医生实际获得的收益 |
| 问诊状态 | 问诊中/问诊结束/取消问诊 |
| 支付状态 | 已支付/未支付 |

## 导出功能

点击"导出"按钮将当前筛选条件下的所有数据导出为Excel文件。

**导出限制**：
- 创建时间范围不能超过3个月（超出提示"时间间隔不能超过三个月"）
- 支付时间范围同样不能超过3个月
- 导出过程中显示"加载中"动画，完成后自动关闭

**状态值复合解析**：
问诊状态 "已预约" 对应 flowStatus=20 且 isCheckIn=0，"问诊中" 对应 flowStatus=20 且 isCheckIn=1。系统在请求时将复合值（如"20,0"）拆分为两个独立参数传递。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 导出时间范围超过3个月 | 前端校验拦截，弹出警告"时间间隔不能超过三个月" |
| 搜索无结果后切换页码 | 自动回退到上一页重新查询 |
| 电话号码脱敏查看 | 需验证操作者身份后返回明文，同一时刻仅显示一条明文 |
| 患者姓名脱敏 | 默认显示为"姓+*"格式，点击后显示完整姓名 |
| 专家团队功能未开启 | "所属团队"筛选和列表列自动隐藏 |
| AI医生功能未开启 | 服务类型不显示"AI图文问诊"，问诊流程不显示"AI问诊" |

## 关联功能

- 与医生信息管理关联（医生姓名可点击跳转）
- 与订单管理模块联动（共用订单号和支付信息）
- 与专家团队管理关联
