# 服务项目配置详情查看流程

> 文档版本 2026-07-04 | 面向读者：运营人员、产品经理

## 概述

服务项目是互联网医院对外提供的可预约医疗服务（如体检、核酸、健康证、检查等）。运营人员可在管理后台创建和编辑服务项目，并在详情页以只读方式查看完整的配置信息。本流程聚焦于详情查看——所有字段均为禁用状态，仅用于审阅和确认配置。

## 触发条件

- 运营人员在服务项目列表页点击某条记录查看详情时
- 需要确认某个服务项目的完整配置信息时

## 步骤详解

### 第一步：进入服务项目详情

从「诊疗管理 → 服务项目管理 → 项目列表」中点击某条服务项目记录，进入详情页面。页面顶部有返回按钮，点击可回到列表。

### 第二步：查看基本配置

页面以表单形式只读展示以下配置：

**服务名称与服务类型**
- 显示项目名称
- 服务类型从字典（fwlx）获取，如体检、核酸、健康证、检查等

**支持人员类型**
- **个人**：是否启用个人预约，显示个人价格（元）
- **团体/企业**：是否启用团体预约，显示定金金额（元）
- 说明文字：默认勾选个人，勾选团体或企业需填写定金；勾选后该服务显示在团体/企业预约服务中

**项目明细**（仅当系统开启自动发票时显示）
- 展示服务包含的项目清单，每项显示：名称、数量、金额
- 开单科室名称

### 第三步：查看服务方式

- **自行到院**：是否启用，显示医院名称和地址
- **上门服务**：是否启用，显示上门服务费用
- 说明：服务方式支持多选，选择单一服务方式则前端默认选中；上门服务费会统计在总费用中

### 第四步：查看服务排班

以复选框组展示周一到周日的排班设置：
- 勾选的日期表示该天可预约
- 默认显示近2个月的排班
- 说明：该排班按照每月对应日期显示

### 第五步：查看开放时间段

分两种类型：

**工作时间段**
- 可设置多组时间段（如 09:00-12:00, 14:00-17:00）
- 时间粒度5分钟，范围06:00-23:59

**非工作时间段**
- 设置非工作时间的起止时间
- 非工作时间段费用（选填）：将统计在最终费用中

### 第六步：查看其他配置

- **服务图片**：显示上传的封面图（建议比例1:1，显示在小程序预约页面）
- **启用状态**：禁用/启用，禁用后不在小程序显示但仍可编辑
- **包含项目**：富文本内容，描述服务包含的项目详情
- **服务须知**：富文本内容，描述服务注意事项和详情说明
- **客服电话**：显示联系电话

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 无ID参数进入 | 页面正常加载但无数据显示 |
| 服务类型过滤 | 字典中value为"-1"的类型被过滤，不在类型选项中显示 |
| 自动发票未开启 | 项目明细和开单科室区域不显示 |
| 工作时间为空 | 工作时间段复选框不勾选，时间输入框为空 |
| 非工作时间为空 | 非工作时间段复选框不勾选 |

## 关联功能

- **服务项目列表管理**：新增/编辑/删除服务项目
- **转诊管理**：将服务项目分配给转诊医生
- **小程序预约**：患者在小程序端查看和预约已启用的服务项目
