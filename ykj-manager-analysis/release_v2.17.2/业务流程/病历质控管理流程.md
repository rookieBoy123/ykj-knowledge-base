# 病历质控管理流程

> 文档版本：v1.0 | 面向读者：医疗质控管理员、运营人员

## 概述

病历质控管理功能用于查看患者在线上的完整就诊病历和处方记录。管理人员通过输入患者的身份证号，可以调出该患者所有线上就诊记录的时间线，逐条查看每次就诊的病历详情和处方信息。支持中西医两种病历类型，以及电子病历/电子处方的 PDF 查看。

## 触发条件

- **入口路径**：管理后台 → 诊疗管理 → 病历质控
- **前置条件**：需要患者的身份证号，且该患者在系统中存在线上就诊记录

## 步骤详解

### 第一步：进入病历质控页面

管理人员通过 URL 参数传入患者的身份证号（`idCard`）和授权凭证（`token`）进入页面。页面顶部会显示当前系统的医院名称。

如果患者没有线上就诊记录，页面会显示「暂无线上就诊记录!」的提示。

### 第二步：查看就诊时间线

页面左侧以时间线形式展示该患者所有的线上就诊记录。每条记录显示：
- **门诊号**：问诊编号，并标注「复诊」标签
- **接诊医生**：医生姓名、科室、职称
- **主诉**：患者本次就诊的主诉内容
- **诊断信息**：西医诊断、中医辨病、中医辨证（根据病历类型显示对应内容）
- **电子病历按钮**：如果有电子病历 PDF，可点击查看

点击任意一条就诊记录，右侧详情区域会切换到对应的病历详情。

### 第三步：查看病历详情

右侧详细展示选中就诊记录的病历信息，包括：

**基本信息区**：
- 就诊人姓名、性别、年龄（岁或月）
- 门诊号、初诊标志（固定显示「复诊」）
- 监护人信息（如有监护人）
- 就诊类型（图文问诊/视频问诊等）
- 就诊机构、就诊时间、就诊科室、接诊医生

**病史信息区**：
- 主诉、现病史、既往史、过敏史
- 个人史、家族史、月经史、婚育史
- 中医四诊（脉象、舌象）：仅中医病历显示
- 常见体征
- 治法（中医）/ 治疗意见（西医）
- 辅助检查

**诊断信息区**：
- 西医诊断
- 中医辨病、中医辨证（仅中医病历显示）

**病历资料区**：
- 患者上传的病历图片，点击可放大预览
- 如无资料则显示「记录遗失/不在身边」

### 第四步：查看处方信息

如果该次就诊有开具处方，病历详情下方会展示处方信息列表。

**西药处方**：以表格形式展示，包含药品名称、规格、剂型、每次用量、用药途径、用药频次、数量。

**中药处方**：展示药品清单、剂数、用药方法（内服/外用）、服用频率、煎煮说明。

每条处方还显示：
- 处方医生姓名
- 审核药师姓名（如有）
- 「电子处方」按钮：可查看 PDF 版处方

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未传入身份证号或 token | 页面不加载任何数据 |
| 身份证号查不到患者信息 | 提示「没有该患者相关信息」 |
| 患者无线上就诊记录 | 显示「暂无线上就诊记录!」 |
| 病历资料图片不存在 | 显示「记录遗失/不在身边」 |
| 病史字段未填写 | 显示「未填写」 |
| 电子病历/处方 PDF 地址为空 | 不显示对应按钮 |

## 关联功能

- 患者详情查看（从患者信息跳转过来）
- 电子病历 PDF 查看器（独立窗口打开）
- 电子处方 PDF 查看器（独立窗口打开）
