# 线下病历详情查看流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、医务管理人员

## 概述

查看从医院HIS系统同步过来的线下就诊病历详情。管理员可以通过此页面查看患者在线下医院就诊的完整记录，包括患者基本信息、病史信息和处方详情。

## 触发条件

- **入口路径**：患者管理 → 患者列表 → 点击某位患者的线下就诊记录 → 病历详情
- **前置条件**：用户具有患者管理权限；患者在HIS系统中有线下就诊记录

## 步骤详解

### 第一步：患者信息区

页面顶部展示患者的基本信息：

- **就诊卡号**：患者在医院的HIS系统编号
- **姓名、性别、出生日期、年龄**：患者基础信息（年龄不足1岁时以"月"为单位显示）
- **身份证号**：如已校验，旁边显示"已校验"标签
- **监护人信息**：仅当患者有监护人时显示，包括监护人姓名、与患者的关系、监护人身份证号

### 第二步：病史信息区

展示本次就诊的完整病史记录：

- **医生、科室、就诊时间**：本次就诊的基本情况
- **主诉**：患者的主要症状描述
- **现病史**：当前疾病的发生发展过程
- **既往史**：过去的疾病史
- **个人史、家族史、婚育史、月经史**：患者个人及家庭健康背景
- **过敏史**：药物或食物过敏信息
- **体格检查**：体温、脉率、呼吸频率、血压、身高、体重、BMI等指标
- **中医四诊**：舌像、脉象、望闻问切结果
- **辅助检查**：化验、影像等检查结果
- **初步诊断**：医生的初步诊断结论
- **病状分类**：疾病的分类标签
- **治疗意见**：医生的治疗建议

**注意**：以上各项如无数据，统一显示为"无"或"-"。

### 第三步：处方信息区

展示本次就诊开具的所有处方，每个处方卡片包含：

**中药处方**（type=4）：
- 处方编号
- 剂数（如"7剂"）
- 用药方法和频率
- 用药说明（如煎煮方法）
- 处方医生和审核药师姓名

**西药处方**（type=5）：
- 每个药品的名称、数量、剂型、规格
- 用法用量（用法、频率、每次用量）

**其他类型处方**（治疗项目/理疗/体检/附加项目/材料）：
- 处方类型标签（如"治疗项目处方（普通）"）
- 处方编号
- 每个项目的名称和数量

处方列表支持分页浏览（每页10/30/50/100条可选）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 患者无线下就诊记录 | 页面数据为空，各项显示"-" |
| 处方列表为空 | 处方信息区不显示任何处方卡片 |
| HIS数据同步延迟 | 线下就诊记录可能不完整，各项显示"-" |
| 患者年龄小于1岁 | 年龄显示为"X月"而非"X岁" |
| 无监护人信息 | 监护人区域完全不显示 |

## 关联功能

- **患者列表管理**：从此页面进入，可返回患者列表
- **HIS数据同步**：线下病历数据来源于HIS系统的数据同步
- **线上就诊记录查询**：查看患者在线上的问诊记录
