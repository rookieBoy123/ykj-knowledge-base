# 轮播图管理流程

> 文档版本：v1.0 | 面向读者：运营管理员

## 概述

轮播图管理用于设置小程序首页的轮播广告图。管理员可以新增、编辑、删除、排序轮播图，控制每张轮播图的显示/隐藏状态，设置点击后的跳转目标。

## 触发条件

- **前置条件**：已登录管理后台，具备运营管理权限
- **入口路径**：左侧菜单 → 运营管理 → 文章管理 → 轮播图管理

## 步骤详解

### 第一步：查看轮播图列表

进入页面后看到轮播图表格，展示以下信息：

| 列名 | 说明 |
|------|------|
| 图片 | 缩略图预览 |
| 轮播标题 | 标题文字 |
| 链接类型 | 内部小程序 / H5 / 外部小程序 |
| 状态 | 显示/隐藏（开关可直接切换） |
| 发布时间 | - |
| 操作 | 编辑、删除、上移、下移 |

### 第二步：新增轮播图

点击"新增"按钮，弹出表单弹窗，需要填写：

1. **轮播标题**（必填）：在轮播图上显示的文字，最多255字
2. **轮播图片**（新增时必填）：上传图片，仅支持jpg/png格式，大小不超过2MB，建议尺寸1078×330px
3. **链接类型**（必填）：选择点击轮播图后的跳转方式
   - **内部小程序**：跳转到本小程序的某个页面
   - **H5**：跳转到一个网页链接
   - **外部小程序**：跳转到其他小程序，需额外填写小程序AppID
4. **跳转路径**：填写跳转的页面路径或URL，最多255字。如果选了"外部小程序"但未填路径，则默认跳转到首页
5. **是否显示**：默认隐藏，可选择显示

### 第三步：编辑轮播图

点击"编辑"进入编辑模式，可修改任意字段。与新增的区别是编辑时图片再次上传是可选的。

### 第四步：显示/隐藏

在列表中直接点击状态列的开关，可即时切换轮播图的显示/隐藏状态。该操作无需弹窗确认，直接调用接口切换。

### 第五步：排序

通过"上移"和"下移"按钮调整轮播图的展示顺序：
- **上移**：将当前轮播图与上一条交换位置，如果已经是第一条则提示"第一条不能上移"
- **下移**：将当前轮播图与下一条交换位置，如果已经是最后一条则提示"最后一条不能下移"

### 第六步：删除轮播图

点击"删除"，弹出确认弹窗"此操作将永久删除该文件, 是否继续?"，确认后删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未填写轮播标题 | 提示"请输入轮播标题" |
| 未选择链接类型 | 提示"请选择类型" |
| 新增时未上传图片 | 提示"请上传轮播图片" |
| 图片格式不是jpg/png | 提示"上传头像图片只能是jpg/png格式!" |
| 图片超过2MB | 提示"上传头像图片大小不能超过2MB!" |
| 选择外部小程序但未填AppID | 提示"请填写Appid" |
| 填写了跳转路径但未选链接类型 | 提示"请选择链接类型" |
| 第一条试图上移 | 提示"第一条不能上移！" |
| 最后一条试图下移 | 提示"最后一条不能下移！" |
| 删除操作 | 弹出二次确认弹窗 |

## 关联功能

- **小程序首页**：轮播图最终显示在小程序的患者端/医生端首页
- **文章管理**：同属运营管理模块下的内容管理功能
