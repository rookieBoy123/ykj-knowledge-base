# 问诊单查询管理流程

> 文档版本 v1.0 | 面向读者：运营管理员、财务人员

## 概述

问诊单查询页面为管理员提供所有问诊订单的统一查询入口，支持按多种条件组合筛选，并可将查询结果导出为Excel文件。该页面适用于运营分析、财务对账、问题排查等场景。

## 触发条件

- **前置条件**：管理员已登录管理后台，拥有问诊查询权限
- **入口路径**：门诊服务 → 问诊订单 → 问诊单查询

## 步骤详解

### 第一步：设置查询条件

页面默认将创建时间范围设置为"最近30天"，管理员可按以下条件组合筛选：

| 筛选条件 | 类型 | 说明 |
|---------|------|------|
| 订单编号 | 文本输入 | 精确搜索，最多20字 |
| 问诊单编号 | 文本输入 | 精确搜索，最多30字 |
| 创建时间 | 日期范围 | 默认最近30天 |
| 服务类型 | 多选下拉 | 图文问诊(1)、视频问诊(2)、咨询服务(3)、快速问诊(4)、快速续方(5)、医生复诊(7)、商城问诊(8)。开启AI医生功能后额外显示AI图文问诊(9) |
| 问诊流程 | 多选下拉 | 在线问诊(1)、线上复诊(2)、慢病医保(3)、预约挂号(4)、快速开方(5)。开启AI医生功能后额外显示AI问诊(6) |
| 问诊状态 | 单选下拉 | 已预约(20+未签到)、问诊中(20+已签到)、问诊结束(30)、取消问诊(40) |
| 支付状态 | 单选下拉 | 未支付(0)、已支付(1) |
| 医生姓名 | 文本输入 | 模糊搜索，最多5字 |
| 患者姓名 | 文本输入 | 模糊搜索，最多5字 |
| 支付时间 | 日期范围 | 可选 |
| 所属团队 | 单选下拉 | 仅当系统开启了专家团队功能时显示 |

**问诊状态说明**：已预约和问诊中的区别通过复合参数实现。选择"已预约"实际查询 `flowStatus=20且isCheckIn=0`；选择"问诊中"查询 `flowStatus=20且isCheckIn=1`；问诊结束为 `flowStatus=30`；取消问诊为 `flowStatus=40`。

### 第二步：查看查询结果

列表展示以下信息：
- **订单号**（180px宽）
- **问诊单编号**（180px宽）
- **服务类型**（显示枚举名称）
- **问诊流程**（显示枚举名称）
- **医生姓名**
- **所属团队**（仅专家团队功能开启时显示）
- **患者姓名**（脱敏显示：保留首字，其余用*替代；点击眼睛图标可查看完整姓名）
- **联系方式**（脱敏显示；点击眼睛图标需经过权限校验后才能查看完整手机号）
- **创建时间**
- **订单支付时间**（未支付时显示"-"）
- **诊费(元)**（保留两位小数）
- **平台服务费**（仅已支付且问诊结束时显示，否则"-"）
- **医生收益(元)**（仅已支付且问诊结束时显示，否则"-"）
- **问诊状态**（草稿/问诊中/问诊结束/取消问诊）
- **支付状态**（已支付/未支付）

### 第三步：导出数据

点击"导出"按钮，将当前查询条件下的结果导出为Excel文件（`问诊单查询.xls`）。

**导出限制**：创建时间范围和支付时间范围均不得超过3个月，否则提示"时间间隔不能超过三个月"。

导出过程中显示加载中提示，完成后自动关闭。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 导出时时间跨度超过3个月 | 提示"时间间隔不能超过三个月"，阻止导出 |
| 患者姓名为空 | 不显示脱敏数据列 |
| 问诊状态为复合值 | 前端自动拆分为flowStatus和isCheckIn两个参数 |
| 服务类型/问诊流程选了"全部" | 移除该筛选条件，查询所有 |
| 未支付 | 订单支付时间、平台服务费、医生收益均显示"-" |
| 问诊未结束 | 平台服务费和医生收益显示"-"（仅问诊结束后才产生实际收益数据） |
| 空列表跳页 | 自动回退到上一页 |
| 专家团队功能关闭 | 隐藏"所属团队"筛选项和列表列 |
| AI医生功能未开启 | 不显示AI相关选项 |

## 关联功能

- 订单详情（点击订单号可跳转，但当前列表未提供跳转链接）
- 问诊对话记录（通过病历详情查看）
- 财务管理（诊费、平台服务费、医生收益的统计数据）
