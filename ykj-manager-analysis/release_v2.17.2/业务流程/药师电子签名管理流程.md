# 药师电子签名管理流程

> 文档版本 2026-07-04 | 面向管理员（药师本人）

## 概述

当前登录的药师在后台管理自己的电子签名。页面展示药师基本信息（姓名、性别、手机号、邮箱）和相关证书图片（身份证、资格证书）。药师可以上传手写签名的照片，经过裁剪后保存为电子签名，用于处方审核时的电子签章。

## 触发条件

- 药师首次登录后台，需要设置电子签名
- 药师需要更换已有的电子签名图片
- 入口：后台 → 处方管理 → 药师签名设置

## 步骤详解

### 第一步：查看药师信息

页面加载时自动获取当前登录药师的基本信息：

- **药师姓名**
- **性别**
- **手机号码**
- **邮箱地址**

如果某项信息为空，显示"暂无"。

下方展示药师的相关证书图片：
- **身份证**：正反面图片
- **资格证书**：药师资格证图片

证书图片以大图展示，可滚动查看。

### 第二步：上传签名图片

点击「上传签名」按钮，选择本地图片文件。上传前系统会校验：

- 文件格式限 jpg/png
- 文件大小不超过 2MB

不满足条件会弹出错误提示。

### 第三步：裁剪签名图片

上传图片后会弹出裁剪对话框，可对图片进行裁剪调整：

- 使用鼠标滚轮可放大缩小图片
- 裁剪框固定比例为 2:1（宽:高）
- 裁剪框大小固定，不可拖动改变尺寸
- 图片可拖动调整裁剪区域

调整完成后点击「确认」，系统将裁剪后的图片上传保存为电子签名。

### 第四步：提交保存

签名上传完成后，点击页面右下角的「提交」按钮。系统将签名图片ID与药师ID绑定保存。

- 如果未上传签名就点击提交，弹出提示："签名必须上传！"
- 提交成功后显示"操作成功"

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未登录药师账号或药师信息未加载 | 页面显示"暂无数据"的空状态 |
| 上传非 jpg/png 格式文件 | 提示"上传图片只能是jpg/png格式!" |
| 上传文件超过 2MB | 提示"上传图片大小不能超过 2M!" |
| 未上传签名就点击提交 | 提示"签名必须上传!"，阻止提交 |

## 关联功能

- **处方药师审核**：审核处方时使用此签名进行电子签章
