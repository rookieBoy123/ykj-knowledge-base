# 快捷回复内容管理流程

> 文档版本 v1.0 | 面向读者：运营管理员

## 概述

管理员配置医生在问诊对话中可使用的快捷回复短语。快捷回复按**分类**组织（如"常用问候"、"用药指导"等），每个分类下可以添加多条快捷回复内容。医生在问诊时可以选择对应的快捷回复一键发送。

## 触发条件

- 前置条件：拥有快捷回复管理权限的管理员账号
- 入口路径：运营管理 → 快捷回复设置

## 步骤详解

### 第一步：选择分类

页面顶部展示已配置的分类标签页（TabBar），点击切换不同分类。首次进入时默认选中第一个分类。如果没有任何分类，标签页为空。

分类数据来自系统预设的公共分类库（isCommon=1），而非管理员自行创建。

### 第二步：查看分类下的快捷回复内容

选中分类后，页面展示该分类下所有快捷回复内容列表，包含：
- 内容（快捷回复的文案）
- 创建时间
- 修改时间
- 操作按钮（编辑、删除）

列表支持关键字搜索过滤（最多255字），支持分页浏览。

### 第三步：新增快捷回复内容

点击"新增"按钮弹出表单，包含两个字段：
- **内容**（必填，最多255字）：快捷回复的文案
- **备注**（选填，最多500字）：对这条回复的说明

保存前系统会校验内容名称在**同一分类下**是否重复，重复则提示"内容名称已存在，请重新输入"。

### 第四步：编辑快捷回复内容

点击某条内容的"编辑"按钮，弹出与新增相同的表单，预填充现有数据。编辑同样会校验重名。

### 第五步：删除快捷回复内容

支持两种删除方式：
- **单条删除**：点击某条内容的"删除"按钮，弹出确认框后删除
- **批量删除**：勾选多条内容后，点击"批量删除"按钮，弹出确认框后一次性删除所有选中内容

若不勾选任何内容直接点批量删除，系统提示"请至少选择一条数据"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 新增时内容名称为空 | 表单校验拦截，提示"请输入内容" |
| 新增/编辑时内容名称与已有内容重复 | 弹出警告"内容名称已存在，请重新输入"，阻止保存 |
| 批量删除时未勾选任何条目 | 弹出提示"请至少选择一条数据" |
| 点击弹窗关闭按钮 | 弹出二级确认"确认关闭？"，防止误操作丢失表单数据 |
| 没有任何分类 | 标签页为空，分类名显示为空 |
| 翻页后当前页无数据 | 自动回退到上一页 |

## 关联功能

- 医生问诊对话 → 医生在对话输入框点击快捷回复按钮，选择预设内容快速发送
- 快捷回复分类设置 → 分类在系统公共分类库中统一管理
