# 线上就诊病历查看流程

> 文档版本：2026-07-06 | 面向读者：运营/管理人员

## 概述

管理人员通过患者身份证号和访问凭证，查看该患者的全部线上就诊病历记录，包括病历详情、处方信息和病历图片资料。页面采用左右双栏布局：左侧为病历时间轴列表，右侧为选中病历的详细信息。

## 触发条件

- **前置条件**：管理人员已登录后台，拥有患者身份证号和访问凭证（token）
- **入口路径**：通过特定链接参数 `?idCard=xxx&token=xxx` 进入

## 步骤详解

### 第一步：身份验证

系统根据URL参数中的身份证号查询患者信息：
- 如果查询到患者，自动加载该患者的全部病历列表
- 如果查询不到，提示"没有该患者相关信息"

同时获取系统配置中的机构标题（用于页面展示）。

### 第二步：浏览病历时间轴

左侧以时间轴形式展示该患者的全部线上就诊记录，每条记录显示：
- **门诊号**（标记为"复诊"）
- **接诊医生**：医生姓名 | 科室 | 职称
- **主诉**：患者填写的主要症状描述
- **诊断信息**：
  - 西医诊断（如有）
  - 中医辨病（如有）
  - 中医辨证（如有）
- **电子病历按钮**：如果存在病历PDF文件，显示「电子病历」按钮可点击查看

点击某条病历记录，右侧自动加载对应的详细病历信息。当前选中的记录卡片高亮显示。

### 第三步：查看病历详情

右侧病历详情区域展示：

**基本信息**：
- 就诊人姓名、性别、年龄（显示"X岁"或"X月"）
- 门诊号、初诊标志（固定显示"复诊"）
- 监护人信息（如有）：监护人姓名、身份证号
- 就诊类型、就诊机构、就诊时间、科室、接诊医生
- 主诉

**病史信息**（未填写则显示"未填写"）：
- 现病史、既往史、过敏史、个人史、家族史
- 月经史（女性患者）、婚育史
- 中医四诊（中医病历类型）：脉象、舌象
- 常见体征、治法（中医）/ 治疗意见（西医）
- 辅助检查

**诊断信息**：
- 西医诊断
- 中医辨病
- 中医辨证

**病历资料**：
- 患者上传的病历图片（支持点击放大预览）
- 如无图片则显示"记录遗失/不在身边"

### 第四步：查看处方信息

如果病历包含处方记录，在病历详情下方展示处方信息：

**西药处方**：表格展示
- 药品名称、规格、剂型
- 每次用量、用药途径、用药频次
- 数量

**中药处方**：
- 处方明细：药品名称和数量
- 剂数、用药方法（内服/外用）、服用说明（几日内几剂，一剂分几次）
- 煎煮方法说明

**处方公共信息**：
- 处方医生、审核药师
- 电子处方按钮（如有PDF文件）

### 第五步：查看电子文档

点击「电子病历」或「电子处方」按钮，在新窗口中打开PDF文件进行查看。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未提供身份证号或token | 页面不加载数据 |
| 根据身份证号查不到患者 | 提示"没有该患者相关信息" |
| 患者无线上就诊记录 | 显示"暂无线上就诊记录!" |
| 病历资料图片缺失 | 显示"记录遗失/不在身边" |
| 病史字段为空 | 显示"未填写" |
| 处方无电子处方PDF | 不显示「电子处方」按钮 |

## 关联功能

- **电子病历PDF**：使用内置PDF查看器打开
- **图片预览**：支持多图预览和切换
