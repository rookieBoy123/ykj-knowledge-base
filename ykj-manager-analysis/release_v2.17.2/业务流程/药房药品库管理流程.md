# 药房药品库管理流程

> 文档版本 2026-07-05 | 面向运营人员

## 概述

药房药品库管理用于维护药房侧的药品信息，区分"成药类型"和"中药类型"两种药品，支持新增、查看详情、删除，以及通过 Excel 批量导入导出药品数据。还可以从标准药品数据库同步更新药品信息。

## 触发条件

- **入口路径**：管理后台 → 医院管理 → 药房管理 → 药品管理
- **使用角色**：拥有药品库管理权限的运营人员
- **前置条件**：系统需配置好药品生产厂家字典

## 步骤详解

### 第一步：切换药品类型

页面顶部有两个标签页切换：**成药类型**和**中药类型**。两种类型展示的列表字段和可执行的操作不同：

| 字段/功能 | 成药类型 | 中药类型 |
|-----------|---------|---------|
| 药品监管ID | 显示（搜索栏和列表列） | 隐藏 |
| 通用名 | 显示 | 显示 |
| 商品名称 | 显示 | 显示 |
| 药品剂型 | 显示 | 显示 |
| 计量计价单位 | 隐藏 | 显示 |
| 药品类型（OTC/RX） | 显示 | 隐藏 |
| 生产厂家 | 显示 | 隐藏 |
| 药品规格 | 显示 | 隐藏 |
| 同步药品数据库 | 显示 | 隐藏 |

搜索条件也随类型切换：成药类型额外显示"药品监管ID"搜索框；中药类型不显示。

### 第二步：搜索和浏览

支持以下搜索条件：
- **商品名称**：模糊搜索
- **通用名称**：模糊搜索
- **药品监管ID**：仅成药类型下显示，模糊搜索

列表字段会根据当前药品类型动态显示不同的列。

### 第三步：新增药品

点击"新增"按钮跳转到药品编辑页面（`editdrug`），可在该页面填写完整的药品信息（包括基本信息、规格信息、用法用量等数十个字段）。新增时通过 URL 参数传递当前药品类型（`drugsType=1` 为成药，`drugsType=2` 为中药）。

从药品编辑页面返回时，列表会自动刷新。

### 第四步：查看详情

点击"查看详情"跳转到药品编辑页面，以只读模式显示该药品的完整信息（URL 参数 `type=detail`）。

### 第五步：批量导入导出

**下载模板**：点击"下载模板"下载对应药品类型的 Excel 导入模板（需要 `med:drug-lib-tmpl:read` 权限）。

**导入**：点击"导入"按钮上传填好的 Excel 文件（成药需要 `med:drug-lib-patent:import` 权限，中药需要 `med:drug-lib-herbal:import` 权限）。导入完成后弹出结果对话框，显示：总导入条数、成功条数、失败条数，以及每条失败记录的行号和具体失败原因。

**导出**：点击"导出"将当前搜索结果导出为 Excel 文件（成药需要 `med:drug-lib-patent:export` 权限，中药需要 `med:drug-lib-herbal:export` 权限）。

### 第六步：同步药品数据库

点击"同步药品数据库"按钮（仅成药类型下显示，需要 `med:drug-lib-patent:sync` 权限），系统从标准药品数据库拉取最新药品信息并更新本地药品库。

### 第七步：删除药品

点击"删除"弹出确认提示"此操作将永久删除该药品，是否继续？"，确认后根据当前药品类型调用对应的删除接口（`chengDrugsDel` 或 `zhongDrugsDel`）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 导入 Excel 部分数据失败 | 弹窗显示失败详情表格（序号、通用名、失败原因），成功的部分正常入库 |
| 分页超出范围（当前页无数据） | 自动回退到上一页重新加载 |
| 删除时取消操作 | 提示"已取消删除" |
| 未配置生产厂家字典 | 厂家下拉选项为空，但不影响其他操作 |
| 从编辑页面返回列表 | 通过路由守卫判断是否来自编辑页，来自编辑页则跳过重置搜索条件（保留用户之前的搜索状态） |

## 关联功能

- **药品编辑**：新增和查看详情均跳转到 `editdrug` 页面
- **药品生产厂家**：厂家下拉选项来自生产厂家管理
- **药品字典（ypdw/mcyl/jldw）**：编辑页面的单位选项依赖多个药品相关字典
- **处方开具**：药品库中的药品供医生在开具处方时选择
