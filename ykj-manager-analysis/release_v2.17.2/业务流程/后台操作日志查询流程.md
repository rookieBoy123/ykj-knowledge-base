# 后台操作日志查询流程

> 文档版本 2026-07-03 | 面向系统管理员/安全审计

## 概述
后台操作日志记录了所有管理员在系统中的增删改查操作，包括谁在什么时间、从哪个IP、用什么浏览器、访问了哪个接口、操作了什么内容、结果成功还是失败、耗时多少毫秒。主要用于安全审计和问题追溯。

## 触发条件
- 菜单路径：系统管理 → 系统管理 → 后台操作日志
- 使用场景：排查某个数据被谁改了、审计管理员操作行为、分析系统异常请求

## 页面功能

### 筛选条件
| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 用户名 | 文本输入 | 精确匹配登录账号，最长30字符 |
| 功能模块 | 文本输入 | 模糊匹配模块名（如"用户管理"），最长100字符 |
| 操作时间 | 日期范围 | 选择起止日期进行过滤 |
| 操作内容 | 文本输入 | 模糊匹配操作描述关键字，最长255字符 |

注：代码中存在被注释掉的"操作类型"下拉筛选（登录/登出/查询/更新/新增/删除/导入/导出），说明此功能曾有计划但当前未启用。

### 列表字段
| 字段 | 说明 |
|------|------|
| 用户名 | 登录账号，如果为空则显示"匿名用户" |
| 功能模块 | 操作所属的功能模块名 |
| 操作内容 | 本次操作的描述 |
| 操作时间 | 操作发生的时间 |
| 操作结果 | 成功（绿）或失败（红） |

### 查看详情
每行提供"查看详情"按钮，点击后弹出详情弹窗，展示更完整的信息：

| 详情字段 | 说明 |
|----------|------|
| 用户ID、用户名、姓名 | 操作者身份 |
| 操作类型 | 如"查询""更新""新增""删除"等 |
| 操作时间、IP、客户端 | 操作环境信息 |
| 操作内容 | 操作的具体描述 |
| 请求URI、请求方法 | 调用的是哪个接口（如GET /api/user/list） |
| 请求参数 | 调用接口时传入的参数（如有） |
| 请求结果 | 成功或失败，有绿色/红色图标区分 |
| 错误信息 | 仅失败时显示，展示具体错误原因 |
| 请求耗时 | 接口响应耗时，单位毫秒 |

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 用户名为空 | 列表显示"匿名用户" |
| 查询无结果 | 列表显示空 |
| 操作失败 | 操作结果显示红色"失败"，详情弹窗中额外显示错误信息字段 |

## 关联功能
- **小程序操作日志**（appLog）：结构相同但固定过滤 `sysClient='app'`，只查看小程序端的操作
- **登录日志**：专门记录管理员登录/登出行为
- **小程序登录日志**：专门记录小程序端用户登录行为
- 服务端：`MgmtSysAuditLogController`
