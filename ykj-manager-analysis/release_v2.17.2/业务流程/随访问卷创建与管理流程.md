# 随访问卷创建与管理流程

> 文档版本 2026-07-04 | 面向读者：运营人员、产品经理

## 概述

随访问卷是医生对患者进行定期随访时使用的调查问卷工具。运营人员可以在管理后台创建、编辑、发布和禁用随访问卷，每个问卷由标题、说明和若干题目组成。发布后的问卷可在随访任务中下发给患者填写。

## 触发条件

- 医生需要创建新的随访调查问卷时
- 需要修改已有问卷内容时
- 需要停用或重新启用某份问卷时

## 步骤详解

### 第一步：进入随访问卷管理

进入「诊疗管理 → 随访管理 → 问卷管理」页面，系统展示所有随访问卷列表，每页10条。

列表信息包括：
- **随访问卷标题**：问卷名称
- **随访问卷状态**：草稿 / 已发布 / 禁用
- **创建时间**：问卷创建日期
- **操作**：查看详情、发布、启用、禁用、编辑

页面顶部支持按「问卷标题」、「状态」（草稿/已发布/禁用）、「创建时间」范围筛选。

### 第二步：新增随访问卷

点击「新增」按钮进入问卷创建页面，分为两个步骤：

**步骤一：填写基本信息**

在左侧「基本信息」标签页填写：
- **问卷标题**：必填，用于标识问卷
- **说明/备注**：选填，对问卷用途的说明

填写完成后点击「保存/下一步」，系统自动跳转到题目编辑界面。

**步骤二：添加题目**

进入题目管理界面后，左侧显示题目列表（初始为空），点击「新增」按钮添加题目。每个题目编辑内容包括：
- 题目名称
- 题目类型（选择/填空等）
- 题目选项（如单选/多选的选项列表）

支持以下操作：
- **新增题目**：添加新题目到问卷中
- **编辑题目**：修改已有题目的内容和选项
- **上移/下移**：调整题目的排列顺序
- **删除题目**：删除某个题目（需确认）

右上角有「预览」按钮，可随时预览问卷在前端的展示效果。

### 第三步：发布问卷

问卷创建完成后，需要发布才能被随访任务引用。在问卷列表页，每条草稿状态的问卷操作列显示「发布」按钮。

点击「发布」时，系统会检查问卷完整性：如果存在未填写完整的内容，弹窗提示「该问卷有内容未填写，请确认填写完整再发布」，并提供「完善问卷」按钮跳转到编辑页；如果问卷完整，直接发布成功，状态变为「已发布」。

### 第四步：启用/禁用问卷

- **已发布状态**的问卷可点击「禁用」按钮关闭，禁用后不再在随访任务中可选
- **禁用状态**的问卷可点击「启用」按钮重新启用，同样需要问卷内容完整检查

启用、发布使用的是同一个变更状态接口，启用本质上也是重新发布。

### 第五步：查看详情

点击「查看详情」按钮，以只读模式查看问卷的完整内容：标题、说明和所有题目及其选项。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 发布时问卷内容不完整 | 弹窗提示"该问卷有内容未填写，请确认填写完整再发布"，可点击"完善问卷"跳转编辑 |
| 已发布的问卷编辑 | 编辑按钮只在草稿状态（status=0）时显示，已发布问卷不能直接编辑 |
| 删除题目 | 弹出确认对话框"是否确认删除？"，确定后删除 |
| 问卷标题超长 | 输入框限制最大30个字符 |
| 题目上移至顶部 | 上移按钮置灰，不可操作 |
| 题目下移至底部 | 下移按钮置灰，不可操作 |
| 删除正在编辑的题目 | 删除当前查看的题目后，自动跳转到新增题目界面 |
| 新增问卷无提交按钮 | 基本信息保存后自动转到题目编辑，无独立"提交"按钮——以发布为最终完成标志 |

## 三种状态说明

| 状态码 | 状态名 | 含义 |
|--------|--------|------|
| 0 | 草稿 | 正在创建/编辑中，仅后台可见 |
| 1 | 已发布 | 已发布可用，随访任务中可引用 |
| 2 | 禁用 | 已停用，不可在随访任务中引用 |

## 关联功能

- **随访任务管理**：任务创建时引用已发布的随访问卷
- **随访提醒管理**：配置随访提醒时可关联问卷
