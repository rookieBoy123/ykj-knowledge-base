# 药品订单销售统计流程

> 文档版本：2026-07-04 | 面向读者：运营人员、财务人员

## 概述

在财务模块中查看所有药品订单的销售统计数据，包括每笔订单的支付金额按微信支付、医保统筹、医保个账、线下支付等渠道的详细分布，以及收入汇总概览。支持按日期、订单类型、医生、团队等条件筛选，并可将数据导出为 Excel。

## 触发条件

- **前置条件**：拥有财务模块访问权限
- **入口路径**：管理后台 → 财务统计 → 药品订单统计

## 页面结构

页面分为两个区域：

### 顶部统计概览
页面顶部固定展示当前筛选条件下的汇总数据，包括：
- 合计收入（元）
- 微信支付金额
- 医保统筹金额（仅在开通医保支付时显示）
- 医保个账金额（仅在开通医保支付时显示）
- 医保支付-微信支付金额（仅在开通医保支付时显示）
- 线下支付金额
- 退款金额

### 药品订单列表
以表格形式展示每笔药品订单的销售明细，列包括：订单编号、订单类型、订单支付时间、医生姓名、所属团队、患者姓名、订单总金额、订单实付金额、微信支付金额、医保统筹金额、医保个账金额、医保支付-微信支付金额、线下支付金额、退款金额。

> **说明**：医保相关的金额列（医保统筹、医保个账、医保支付-微信支付）仅在系统开通医保支付功能时显示；所属团队列仅在系统开通专家团队功能时显示。

## 步骤详解

### 第一步：设置筛选条件

点击查询区域的筛选项，可以按以下条件过滤数据：

| 筛选项 | 类型 | 说明 |
|-------|------|------|
| 订单支付日期 | 日期范围选择器 | 选择起止日期，默认最近30天 |
| 订单编号 | 文本输入框 | 精确搜索，最多30个字符 |
| 订单类型 | 下拉选择 | 从数据字典中动态加载（如药品订单、问诊订单等） |
| 医生 | 远程搜索下拉选择 | 输入医生姓名后系统自动模糊匹配，仅展示已审核通过的医生 |
| 所属团队 | 下拉选择 | 可选择专家团队（仅在专家团队功能启用时显示） |

### 第二步：查看统计数据

设置筛选条件后点击"查询"按钮，系统自动加载两个部分的数据：
1. **顶部统计概览**：根据筛选条件聚合计算各支付渠道的收入合计和退款合计
2. **订单列表**：分页展示每笔订单的详细收款信息

列表支持分页浏览，每页默认显示10条记录。

### 第三步：导出数据

点击表格上方的"导出"按钮，系统会将当前筛选条件下的所有订单数据导出为 Excel 文件（文件名：订单销售统计.xls）。导出过程中页面显示加载提示。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未开通医保支付 | 医保统筹、医保个账等列和统计项自动隐藏 |
| 未开通专家团队 | 所属团队筛选项和列自动隐藏 |
| 医生搜索无匹配结果 | 下拉列表为空 |
| 筛选条件无匹配数据 | 表格显示空列表，统计概览显示数值为0或"-" |
| 导出数据量较大 | 显示加载提示至导出完成 |

## 关联功能

- **交易流水查询**：可跳转查看每笔订单的原始支付流水
- **数据看板**：以图表形式呈现按月份和来源的统计趋势
- **医保结算上传**：医保相关数据的对账和上传管理
