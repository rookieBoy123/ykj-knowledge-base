# 药品订单销售统计流程

> 文档版本 | 面向读者：运营管理员 / 财务人员

## 概述
提供药品订单的销售收入统计功能，按支付渠道（微信支付、医保统筹、医保个账、线下支付、退款）分别展示金额，支持按日期、订单编号、订单类型、医生、所属团队等多维度筛选，并可将数据导出为Excel。

## 触发条件
- **前置条件**：拥有药品销售统计权限的管理员账号
- **入口路径**：管理后台 → 财务管理 → 药品订单销售统计

## 步骤详解

### 第一步：查看统计概览
进入页面后，系统默认展示最近30天的数据。页面顶部显示**合计收入统计条**，包含以下维度：

- **合计收入**：所有订单的总收入金额
- **微信支付**：通过微信支付的金额
- **医保统筹**：医保统筹基金支付金额（仅在开启医保支付时显示）
- **医保个账**：医保个人账户支付金额（仅在开启医保支付时显示）
- **医保支付-微信支付**：医保订单中用户通过微信补付的金额（仅在开启医保支付时显示）
- **线下支付**：线下渠道支付的金额
- **退款**：已退款金额

所有金额均精确显示到两位小数。

### 第二步：筛选查询
可通过以下条件筛选数据：

| 筛选条件 | 说明 |
|---------|------|
| 订单支付日期 | 日期范围选择，默认最近30天 |
| 订单编号 | 精确搜索，最多30个字 |
| 订单类型 | 从数据字典动态加载（如药品订单、问诊订单等） |
| 医生 | 远程搜索，支持按医生姓名模糊搜索，仅显示已入驻医生 |
| 所属团队 | 可选，该筛选条件仅在系统开启「专家团队」功能时显示 |

筛选后，概览统计和列表数据会同步更新。

### 第三步：查看订单明细列表
列表展示每条订单的详细信息：
- 订单编号、订单类型、订单支付时间
- 医生姓名（所属团队名称在开启专家团队功能时显示）
- 患者姓名
- 订单总金额、订单实付金额
- 各支付渠道明细：微信支付、医保统筹、医保个账、医保支付-微信支付、线下支付、退款

### 第四步：导出数据
点击「导出」按钮，导出当前筛选条件下的全部数据为Excel文件（文件名：订单销售统计.xls）。导出时会显示"加载中"提示，完成后自动下载。

> ⚠️ **注意**：导出有时间限制，最多只能导出3个月（92天）内的数据。如果选择的日期范围超过3个月，会提示"只能导出三个月之内的数据"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 导出时间范围超过92天 | 提示"只能导出三个月之内的数据"，导出失败 |
| 导出过程异常 | 关闭加载提示，停止导出 |
| 专家团队功能关闭 | 所属团队筛选条件和列表列自动隐藏 |

## 关联功能
- **订单管理**：订单列表中的每条记录对应一笔实际订单
- **退款审核**：退款金额统计与退款审核模块关联
- **医保开关**：医保统计列是否显示取决于系统配置中的医保支付开关
