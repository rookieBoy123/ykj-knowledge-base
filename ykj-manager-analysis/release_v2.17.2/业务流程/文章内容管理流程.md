# 文章内容管理流程

> 文档版本 2026-07-03 | 面向运营/内容编辑人员

## 概述

管理小程序端展示的科普文章内容。运营人员可以新增、编辑、删除文章，按文章类型和状态筛选，设置文章的显示/隐藏状态，支持批量删除操作。

## 触发条件

- **前置条件**：拥有内容管理权限
- **入口路径**：运营管理 → 文章管理 → 文章列表

## 步骤详解

### 第一步：筛选查询

支持三个筛选维度：

1. **文章类型**：从数据字典（分类编码"ysnrfl"，即"医生内容分类"）动态加载，如健康科普、疾病知识、用药指导等
2. **文章标题**：输入文章标题进行模糊搜索
3. **状态**：显示 / 隐藏

### 第二步：查看文章列表

列表中展示：

| 列 | 说明 |
|:---|:---|
| 勾选框 | 用于批量选择删除 |
| 缩略图 | 文章的封面图片 |
| 标题 | 文章标题，过长时鼠标悬停显示完整内容 |
| 作者 | 文章来源或作者名称 |
| 状态 | 显示/隐藏，通过行内开关直接切换 |
| 发布时间 | 文章的发布日期 |
| 操作 | 编辑、删除 |

### 第三步：新增文章

点击"新增"按钮，系统全屏打开文章编辑器：
1. 选择文章类型（从数据字典的分类中选择）
2. 输入文章标题
3. 填写作者信息
4. 上传缩略图（封面图）
5. 使用富文本编辑器撰写正文内容
6. 设置发布时间
7. 点击保存

编辑器采用全屏设计，提供完整的富文本编辑功能（排版、图片、链接等）。

### 第四步：编辑文章

点击"编辑"按钮，同样全屏打开编辑器，预填当前文章的所有内容。编辑完成后点击"修改"，保存更新。

### 第五步：显示/隐藏文章

在列表行内点击"状态"列的开关：
- **显示**：文章在小程序端正常展示
- **隐藏**：文章在小程序端不显示（不会删除数据，可重新设为显示）

切换即时生效，不需要额外确认。

### 第六步：删除文章

支持两种删除方式：
- **单个删除**：点击操作列的"删除"，确认后删除
- **批量删除**：勾选多篇文章后，点击"批量删除"按钮。如果未勾选任何文章，会提示"请至少选择一条数据"

删除操作有确认弹窗，确认后不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 未勾选文章就点批量删除 | 提示"请至少选择一条数据" |
| 文章类型选项为空 | 数据字典未配置"医生内容分类"，需先在字典管理中维护 |
| 删除后列表翻页异常 | 如果当前页是最后一页且删除后无数据，自动回退到上一页 |
| 确认删除时取消 | 提示"已取消删除" |

## 关联功能

- **数据字典**：文章类型分类通过字典"医生内容分类（ysnrfl）"配置
- **患者端文章展示**：文章在小程序端按分类展示给用户阅读
- **轮播图管理**：首页轮播图可以配置跳转到具体文章

## 业务说明

文章分类编码为"ysnrfl"（医生内容分类），这是小程序端健康科普内容的基础分类体系。运营人员可以在数据字典中自行增删改分类，灵活调整内容组织结构。

与"推荐医生"、"轮播图"等功能不同，文章管理是一个独立的内容生产模块，侧重于给患者提供专业的医疗健康知识，帮助患者了解疾病、用药、养生等信息。
