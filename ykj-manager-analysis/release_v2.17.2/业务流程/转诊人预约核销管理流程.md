# 转诊人预约核销管理流程

> 文档版本 2026-07-04 | 面向运营管理员

## 概述

管理员管理健康服务预约订单的全流程：查询预约单、核销服务、查看详情、退款处理、打印发票。

## 触发条件

- **前置条件**：患者在小程序中下单了健康服务预约（体检、核酸检测等线下服务）
- **入口路径**：诊疗管理 → 服务项目管理 → 转诊人管理

## 步骤详解

### 第一步：查询预约单

进入列表页，支持以下多维度筛选：
- **预约编号**：精确搜索
- **预约手机号**：11位手机号搜索
- **预约时间**：日期范围
- **服务名称**：模糊搜索
- **服务方式**：下拉选择（上门服务/到店服务等，从字典获取）
- **服务类型**：下拉选择（体检/核酸检测/检查等，从字典获取）
- **人员类型**：下拉选择（个人/团体企业，从字典获取）
- **预约单状态**：下拉选择

列表展示：预约编号、服务名称、服务类型、人员类型、服务方式、价格、联系人、联系电话（脱敏）、预约时间、状态、服务核销时间。

### 第二步：核销订单

点击顶部的"核销订单"按钮，弹出窗口输入核销码（最多16位）。输入完8位后自动确认或手动点击确定。

系统根据核销码查找对应预约单，跳转到预约详情页。如果核销码不存在，提示错误。

### 第三步：查看预约详情

详情页展示完整的预约信息：

**进度条**：生成订单 → 预约单状态（待支付/已取消/退款中/已退款/待核销/已核销）→ 服务状态

**预约单信息**：
- 预约单号、联系人、手机号、人员类型
- 预约服务名称、预约时间
- 服务方式、上门地址（如果选择上门服务）
- 团体预约时显示预约人数
- 订单支付时间

**费用明细**：
- 序号、服务名称、单价、合计
- 应付金额、实际支付金额
- 可查看"服务说明"（包含项目和须知）

**底部操作按钮**（根据状态显示）：
- **退款**：状态为"待核销"时显示。点击后二次确认，提示"预付款项将在48小时内原路退回"
- **确认核销**：状态为"待核销"时显示。弹出窗口输入核销码和发票信息
- **打印发票**：已开发票时显示，点击打开发票图片

### 第四步：确认核销（含发票）

点击"确认核销"后弹出窗口：
1. 输入核销码（16位）
2. 如果预约支持发票：选择发票抬头类型（个人/企业），企业需输入企业名称（最多64字）
3. 点击确定提交核销

核销成功后预约单状态变为"已核销"。

### 第五步：退款处理

点击"退款"按钮，系统：
1. 弹出确认提示："是否确认退款，确认后，预付款项将在48小时内原路退回？"
2. 确认后提交退款申请（退款原因自动填写为管理员操作）
3. 退款成功后刷新页面

## 状态流转

```
待支付 → (支付成功) → 待核销 → (核销) → 已核销
待支付 → (取消)    → 已取消
待核销 → (退款申请) → 退款中 → (退款成功) → 已退款
```

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 核销码不存在 | 提示错误，核销失败 |
| 核销码输入框为空 | 提示"请输入核销码" |
| 企业发票未填企业名称 | 提示"请输入企业名称" |
| 未选择发票抬头类型 | 提示"请选择发票抬头" |
| 退款时数据异常 | 提示错误信息 |
| 自动核销（从列表页核销码弹窗进入） | 跳转到详情页后自动弹出核销窗口，输入核销码自动提交 |
| 图片发票未生成 | 点击打印时提示"开票中！" |

## 关联功能

- **发票管理**：核销时可关联发票信息
- **订单管理**：退款关联订单退款流程
- **字典管理**：服务方式、服务类型、人员类型、预约单状态从字典获取
