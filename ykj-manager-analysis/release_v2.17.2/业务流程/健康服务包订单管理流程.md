# 健康服务包订单管理流程

> 文档版本 2026-07-03 | 面向运营人员

## 概述

管理用户购买健康服务包后产生的所有订单。支持按订单状态筛选、查看详情、订单核销、数据导出等操作。健康服务包是一种套餐产品，用户一次性购买后可享受多次服务，订单需要核销后方可使用。

## 触发条件

- **前置条件**：用户在患者端购买了健康服务包
- **入口路径**：运营管理 → 健康服务包 → 服务包订单

## 步骤详解

### 第一步：按状态标签筛选

页面顶部提供状态标签页快速筛选：

| 状态 | 说明 |
|:---|:---|
| 全部 | 查看所有订单 |
| 待支付 | 用户已下单但未付款 |
| 已付款 | 用户已完成支付，等待核销 |
| 已完成 | 订单已核销完成 |
| 已取消 | 订单已取消 |
| 退款中 | 用户申请退款，正在处理中 |
| 退款成功 | 退款已到账 |

### 第二步：精确筛选

可通过以下条件进一步精确查找：
- **订单编号**：输入完整或部分订单号
- **服务包名称**：按服务包名称搜索
- **联系电话**：按下单用户的手机号搜索（支持脱敏查看）
- **创建时间**：按日期范围筛选

### 第三步：查看订单列表

列表展示关键信息：

| 列 | 说明 |
|:---|:---|
| 订单编号 | 唯一订单号 |
| 服务包名称 | 用户购买的服务包名称 |
| 服务项目 | 服务包包含的服务项目列表 |
| 订单金额（元） | 订单支付金额 |
| 联系电话 | 用户手机号，默认脱敏显示，点击可查看完整号码（需服务端校验） |
| 状态 | 当前订单状态（待支付/已付款/已完成/已取消/退款中/退款成功） |
| 创建时间 | 下单时间 |
| 订单支付时间 | 实际付款时间 |

### 第四步：查看订单详情

点击操作列的"查看详情"，跳转到订单详情页面。详情页展示订单的完整信息，包括服务包内容、支付记录、核销记录等。

### 第五步：订单核销

点击列表上方的"订单核销"按钮，弹出核销弹窗：

1. 输入订单编号或扫码获取订单信息
2. 确认订单内容和服务项目
3. 执行核销操作
4. 核销后订单状态从"已付款"变为"已完成"

### 第六步：数据导出

点击"导出"按钮，将当前筛选条件下的所有订单数据导出为Excel文件。导出过程中显示加载提示。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 联系电话脱敏查看 | 默认只显示部分数字，点击后需服务端校验权限，每次只能查看一个用户的完整手机号 |
| 已取消/退款订单是否可以核销 | 只有"已付款"状态的订单可以核销，其他状态无核销入口 |
| 导出数据过多 | 按当前筛选条件导出，无数量限制（与问诊订单的3个月限制不同） |
| 空结果 | 显示空状态，不提示错误 |

## 关联功能

- **健康服务包管理**：管理可供购买的服务包配置
- **服务项目管理**：管理服务包中包含的服务项
- **订单核销**：核销是订单完成的关键步骤
