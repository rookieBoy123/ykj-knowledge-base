# 药师电子签名上传流程

> 文档版本：2026-07-05 | 面向读者：药师、运营管理员

## 概述

药师在审方工作台设置个人电子签名图片，用于电子处方签章。流程包括查看个人基本信息和证书、上传签名图片并进行裁剪。

## 触发条件

- 前置条件：药师已登录并通过认证
- 入口路径：管理后台 → 诊疗管理 → 处方管理 → 药师审方 → 审方录入

## 步骤详解

### 第一步：查看基本信息

进入审方录入页面后，顶部展示药师的基本信息卡片：
- 药师姓名、性别、手机号码、邮箱地址
- 证件照片区：展示身份证正反面图片、资格证书图片（如已上传）

### 第二步：上传签名图片

点击"点击上传签名"按钮，从本地选择图片文件。

**系统校验规则**：
- 仅支持 JPG、PNG 格式（其他格式会提示"上传图片只能是jpg/png格式！"）
- 文件大小不超过 2MB（超过提示"上传图片大小不能超过 2M！"）

### 第三步：裁剪签名图片

上传图片后自动弹出裁剪对话框：
- 裁剪框固定宽高比为 2:1（宽度是高度的两倍）
- 支持鼠标滚轮缩放图片、拖动图片调整位置
- 裁剪框大小固定不可改变，位置不可拖动
- 下方提示："鼠标滚轮可放大缩小图片,可拉动图片"

### 第四步：确认裁剪

点击"确认"按钮完成裁剪，系统自动将裁剪后的图片上传到服务器。成功后：
- 裁剪对话框关闭
- 签名预览区域立即显示新的签名图片

### 第五步：提交签名

点击页面底部的"提交"按钮，将签名图片与当前药师绑定。系统会校验：
- 如果没有上传签名，提示"签名必须上传！"
- 成功提示"操作成功"

### 空状态

如果药师信息未加载（无数据），页面显示空状态图标和"暂无数据"文字，不显示任何表单。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 上传非JPG/PNG文件 | 前端校验拦截，提示"上传图片只能是jpg/png格式！" |
| 文件超过2MB | 前端校验拦截，提示"上传图片大小不能超过 2M！" |
| 未上传签名直接提交 | 弹出警告"签名必须上传！" |
| 药师信息不存在 | 页面显示"暂无数据"空状态 |

## 关联功能

- **处方电子签章流程**：药师签名是电子处方签章的基础
- **药师审核流程**：签名上传完成后才能在审方环节使用
