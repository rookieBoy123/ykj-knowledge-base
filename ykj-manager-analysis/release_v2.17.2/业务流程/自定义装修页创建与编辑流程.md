# 自定义装修页创建与编辑流程

> 文档版本 1.0 | 面向读者：运营/设计人员

## 概述

这是医院小程序的**可视化页面编辑器**，运营人员可以通过拖拽方式自由搭建小程序首页或专题页的布局和内容。不需要写代码，从左侧组件库中选择模块拖到中间的"手机模拟屏"上，再在右侧配置各项参数即可。

## 触发条件

- 在「页面装修列表」中点击"新增页面"，或点击已有页面的"编辑"按钮
- 支持三种操作模式：新增（pageType=add）、编辑（pageType=edit）、复制（pageType=copy）
- 复制模式下页面名称会自动加上"-副本"后缀

## 页面布局

编辑器采用**三栏式布局**：

| 区域 | 位置 | 作用 |
|------|------|------|
| 左侧栏 | 组件库 | 展示所有可用的页面组件，支持展开/收起 |
| 中间栏 | 手机预览区 | 模拟手机屏幕（375px宽），实时预览页面效果 |
| 右侧栏 | 配置面板 | 点击某个组件后，在这里编辑该组件的详细参数 |

## 步骤详解

### 第一步：设置页面基本信息

点击中间预览区顶部的编辑图标，可以：
- **修改页面名称**：不超过15个字，用于后台识别
- **点击"页面设置"按钮**→进入页面级配置：
  - 页面标题（显示在手机顶部导航栏）
  - 背景颜色或背景图片（支持平铺/全屏两种模式）
  - 标题字体颜色和背景色
  - 返回跳转地址

### 第二步：从组件库选择组件

左侧的「基础组件」库提供了各类页面模块，点击组件名称右侧箭头可展开查看：

- **搜索框**：每个页面只能添加一个，自动固定在顶部
- **轮播图（Banner）**：可配置多张图片轮播，支持设置轮播高度（0-200范围内）
- **医生列表**：展示医生信息卡片
- **医生视频列表**：需系统开启doctorVideoEnabled功能才会显示
- **健康服务包**：需系统开启servicePackageEnabled功能才会显示
- **底部导航栏（Footer）**：每个页面只能添加一个，自动固定在底部，禁止拖拽移动

**添加方式**：
- 从左侧**拖拽**组件到中间预览区（拖拽到两个组件之间的空隙）
- 或直接**点击**组件，会自动添加到布局末尾

### 第三步：调整组件顺序

在中间预览区，每个组件右上角有操作按钮：
- **向上/向下箭头**：移动组件位置
- **复制图标**：复制当前组件到下方
- **删除图标（X）**：移除该组件

**限制规则**：
- 搜索框和底部导航栏**不可移动、不可拖拽**
- 如果搜索框上方有组件，会提示"搜索框或分类必须为顶部"

### 第四步：配置组件参数

点击中间预览区的某个组件，右侧面板会显示该组件的配置项：

- 不同组件有不同的配置项（如轮播图可以设置图片链接、跳转地址；医生列表可以设置筛选条件）
- 每个组件的配置项由对应的右侧配置组件动态渲染
- 配置修改在中间预览区实时生效

### 第五步：保存与发布

顶部操作栏提供：
- **保存修改**：保存当前页面配置。新建页面首次保存后自动生成页面ID，下次进入为编辑模式
- **页面重置**：恢复到服务器端当前保存的最新版本（谨慎操作，会丢失未保存的修改）
- **另存模板**：将当前页面另存为一个新的模板（需输入不超过15字的模板名称）
- **手机预览**：生成一个二维码，扫码可在手机上预览实际效果（仅已保存过的页面可用）

**快捷键**：支持 `Ctrl+S` 保存。

## 离开保护

点击返回按钮或切换页面时，系统会弹出确认框："当前为离开页面操作，请检查是否已保存修改，离开后系统不会保存已修改内容。" 防止误操作丢失数据。

## 特殊规则

| 规则 | 说明 |
|------|------|
| 搜索框限制 | 每页只能一个，自动置顶，不可移动/拖拽 |
| 底部导航限制 | 每页只能一个，自动置底，不可移动/拖拽 |
| 医生视频组件 | 仅当系统开启doctorVideoEnabled功能时在组件库中显示 |
| 健康服务包组件 | 仅当系统开启servicePackageEnabled功能时在组件库中显示 |
| 组件排序 | 拖拽后每个组件的时间戳自动更新为新时间×1000+索引，确保顺序稳定 |

## 关联功能

- 页面装修列表：管理所有装修页的上架/下架状态
- 小程序首页：装修页发布后在患者端小程序展示
