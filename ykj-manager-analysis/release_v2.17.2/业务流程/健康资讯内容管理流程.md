# 健康资讯内容管理流程

> 文档版本 2026-07-04 | 面向运营人员/内容编辑

## 概述
运营人员通过此功能管理微信小程序患者端展示的健康资讯文章。支持按文章类型分类管理，可新增、编辑、删除文章，控制文章显示/隐藏状态，以及批量删除。

## 触发条件
- 入口：运营管理 → 内容管理 → 健康资讯
- 使用场景：发布新的健康科普文章、管理文章的显示状态、清理过时内容

## 步骤详解

### 第一步：查询和筛选文章
页面顶部提供三个筛选条件：
- **文章类型**：下拉选择文章分类（数据来源为字典编码 `hznrfl`），如"疾病科普""用药指导""饮食健康"等
- **文章标题**：输入标题关键词模糊搜索，最多255个字符
- **状态**：选择"显示"或"隐藏"进行过滤

### 第二步：查看文章列表
列表展示以下信息：

| 信息 | 说明 |
|------|------|
| 缩略图 | 文章封面图片，缩略形式展示 |
| 标题 | 文章标题，过长时鼠标悬停显示完整内容 |
| 创建人 | 文章的作者/创建者姓名 |
| 状态 | 显示/隐藏开关 |
| 发布时间 | 文章发布的时间 |
| 操作 | 编辑、删除 |

### 第三步：新增文章
点击列表上方"新增"按钮，页面切换为全屏编辑模式，可填写：
- 选择文章类型（从 `hznrfl` 分类中选择）
- 填写文章标题和内容
- 上传缩略图
- 设置发布时间

保存后返回列表，列表自动刷新。

### 第四步：编辑文章
点击操作列的"编辑"按钮，页面切换为全屏编辑模式，加载文章当前数据，修改后保存。

### 第五步：控制文章显示/隐藏
列表中"状态"列为开关控件：
- **显示**：文章在患者端可见
- **隐藏**：文章在患者端不可见

切换开关即时生效，操作成功提示"操作成功"。

### 第六步：删除文章
- **单条删除**：点击操作列"删除"按钮，弹出确认框（"此操作将永久删除该资讯，是否继续？"），确认后删除
- **批量删除**：勾选多条文章的复选框，点击上方"批量删除"按钮，弹出确认框后一次性删除
- 如果未勾选任何文章就点击批量删除，系统提示"请至少选择一条数据"
- 取消删除时提示"已取消删除"

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未选择文章就点击批量删除 | 提示"请至少选择一条数据" |
| 删除操作取消 | 提示"已取消删除" |
| 最后一页数据被全部删除 | 自动回退到上一页 |
| 搜索无结果 | 列表显示空 |

## 关联功能
- **文章类型管理**：文章类型来自数据字典（`hznrfl`），在"字典管理"中维护
- **患者端健康资讯**：显示状态为"显示"的文章在微信小程序患者端展示
- **文章内容管理流程**（另一功能）：管理面向医生的文章内容（使用 `ysnrfl` 分类），与健康资讯是不同的功能模块
