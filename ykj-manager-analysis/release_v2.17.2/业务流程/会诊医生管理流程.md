# 会诊医生管理流程

> 文档版本 2026-07-04 | 面向远程医疗管理员

## 概述

远程医疗管理员在受邀机构下管理参与会诊的医生信息，包括新增、编辑、删除医生，以及启用/禁用医生状态。

## 触发条件

- **前置条件**：当前登录账号具有"远程会诊管理员"角色；已从受邀机构列表页进入
- **入口路径**：诊疗管理 → 远程医疗中心 → 受邀机构管理 → 点击某个机构的"管理医生"

## 步骤详解

### 页面加载

进入页面后，顶部显示当前机构名称（如"机构名称：XX医院"），下方展示该机构下所有会诊医生的列表。列表包含：医生编号、医生姓名、科室、职称、职称等级、联系电话、医生简介、状态（启用/禁用）和操作按钮。

页面必须携带机构 ID 参数（从受邀机构列表传入），缺少时会提示"缺少机构id"并停留在空白页面。

### 新增医生

1. 点击列表上方的「新增」按钮，弹出新增表单窗口
2. 填写以下必填信息：
   - **医生姓名**：最多 30 个字符
   - **科室**：从数据字典中的科室编码树形列表中选择（支持搜索）
   - **职称**：从下拉列表选择（如主任医师、副主任医师等）
   - **职称等级**：从下拉列表选择（如正高、副高、中级等）
   - **联系电话**：必须填写有效的 11 位手机号码
   - **简介**（选填）：最多 300 个字符
   - **状态**：默认"启用"，可切换为"禁用"
3. 点击「确定」提交，新增成功后列表自动刷新

### 编辑医生

1. 在列表中找到目标医生，点击操作列的「编辑」按钮
2. 弹出编辑窗口，表单自动填入该医生的现有信息
3. 修改需要变更的字段后点击「确定」
4. 编辑成功后列表自动刷新

### 删除医生

1. 点击目标医生操作列的「删除」按钮
2. 系统弹出确认提示："删除后数据不可恢复，确认删除医生【姓名】吗？"
3. 确认后执行删除，列表刷新

### 启用/禁用

新增或编辑时可以设置医生状态：
- **启用**：医生可正常参与会诊
- **禁用**：医生暂停参与会诊，但在列表中仍可见

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 页面未携带机构ID | 提示"缺少机构id"，页面空白 |
| 未填写必填字段 | 表单校验提示具体缺失项，阻止提交 |
| 手机号码格式错误 | 提示"请输入正确的手机号码" |
| 医生存在关联会诊记录 | 服务端拦截，提示"医生【姓名】存在被申请或安排会诊记录，不可删除！" |
| 医院被删除后访问 | 可能显示机构名称异常（通过机构详情接口获取） |

## 关联功能

- **受邀机构管理**：入口来源，通过机构 ID 参数跳转进入
- **会诊服务管理**：同级别的机构子功能
- **会诊申请管理**：医生删除的约束来源（已关联会诊的不可删）
