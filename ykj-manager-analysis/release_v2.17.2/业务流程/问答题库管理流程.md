# 问答题库管理流程

> 文档版本：2026-07-05 | 面向读者：运营管理员、问卷设计人员

## 概述

管理可复用的题目库，供问卷创建时引用。支持单选题、多选题、问答题、上传图片四种题型。题目可独立管理（增删改查、启用/禁用），与问卷系统解耦。

## 触发条件

- 前置条件：拥有题库管理权限（`hosp:ql-questions`）
- 入口路径：管理后台 → 诊疗管理 → 问卷管理 → 问答题库

## 步骤详解

### 第一步：搜索与筛选

支持四维搜索条件：
- **题目编码**：按系统自动生成的题目编号搜索（如"TM20260705143022"）
- **题目名称**：模糊搜索题目内容
- **题目类型**：单选（单选/多选/问答/上传图片）
- **题目状态**：启用/禁用

### 第二步：查看题目列表

列表展示以下信息：
- 题目编码（自动生成，格式为"TM"+年月日时分秒毫秒，如TM20260705143022123）
- 题目名称
- 题目类型（单选题/多选题/问答题/上传图片）
- 是否必填（是/否）
- 创建时间
- 题目状态（开关，可一键切换启用/禁用）
- 操作（编辑、删除）

列表支持多选复选框和批量删除操作。

### 第三步：新建题目

点击"新建"按钮弹出600px宽的对话框，Tab页切换四种题型：

**单选题**（默认Tab）：
- 题目名称（必填，最长100字）
- 是否必填（是/否）
- 备注（最长300字）
- 选项列表：至少2个选项（不足2个提交时提示"单选题选项不能低于2个"）
- 可添加/删除选项

**多选题**：与单选题结构相同，但支持多个正确答案

**问答题**：无需选项，只需填写题目名称

**上传图片**：无需选项，需填写题目名称

**校验规则**：
- 题目名称必填（为空提示"题目名称不能为空"）
- 题目名称最长100字
- 题目名称不可重复（提交时提示"题目名称XXX已存在"）
- 单选/多选题的选项至少2个
- 备注最长300字

### 第四步：编辑题目

点击"编辑"按钮，根据题目类型加载对应的编辑组件。编辑时的校验规则与新增相同。题目类型不可更改（编辑时不显示类型切换Tab，直接加载对应类型的编辑表单）。

### 第五步：启用/禁用题目

列表每行的"题目状态"开关可直接操作，即时生效。禁用后该题目在问卷创建时不可选。

### 第六步：删除题目

**单个删除**：点击操作列的"删除"，弹出确认框："删除后数据不可恢复。确认删除该题目？"，确认后删除题目及其所有选项。

**批量删除**：勾选多个题目复选框后，点击"批量删除"，弹出确认框，确认后一次性删除所有选中题目及其选项。未选择任何题目时提示"请至少选择一条数据"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 题目名称为空 | 提交时提示"题目名称不能为空" |
| 题目名称超100字 | 提交时提示"题目名称长度不能大于100字" |
| 题目名称重复 | 提交时提示"题目名称XXX已存在" |
| 单选/多选题选项少于2个 | 提交时提示"单选题选项不能低于2个" |
| 题目ID不存在 | 后端提示"题目不存在" |
| 批量删除未选择数据 | 前端提示"请至少选择一条数据" |
| 翻页到超出数据范围的页码 | 自动回退到上一页 |

## 关联功能

- **问卷创建与管理**：创建问卷时从题库中引用题目
- **问卷题目编辑**：问卷中的题目可以来自题库
- **随访问卷管理**：随访功能也使用题库
