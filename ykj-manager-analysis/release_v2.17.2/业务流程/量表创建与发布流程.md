# 量表创建与发布流程

> 文档版本 2026-07-04（第5轮） | 面向运营/产品人员

## 概述

量表是用于心理健康评估的标准化测评工具。管理员在后台创建量表需要经过三个步骤：填写基本信息、添加题目、设置评分标准。量表有三种状态：草稿、发布、禁用。

## 触发条件

- 前置条件：具有量表管理权限，已创建量表分类
- 入口路径：后台 → 诊疗管理 → 量表管理 → 量表列表 → 新增量表

## 步骤详解

### 第一步：填写基本信息

点击「新增量表」进入编辑页面。左侧显示三个步骤导航：**基本信息 → 量表题目 → 评分标准**。第一步默认展开基本信息区域，需要填写量表的基础属性（如名称、所属分类等）。

> 填写完成后点击下一步，系统保存基本信息并自动跳转到第二步。

### 第二步：添加量表题目

进入「量表题目」区域后，可以在题目列表中管理题目：
- **新增题目**：点击「新增」按钮，清空右侧表单，输入新的题目内容
- **编辑题目**：点击题目列表中的某条题目，右侧加载该题目的详细信息进行编辑
- **删除题目**：点击题目右侧的删除按钮
- **调整顺序**：使用「上移」「下移」按钮排列题目的显示顺序
- **搜索题目**：在顶部搜索框输入题目名称关键词，支持精确搜索

> 注意：新增量表时题目列表为空，必须至少添加一道题目才能进入下一步。如果没有添加题目就切换走，后续发布时会检测到题目为空，无法发布。

### 第三步：设置评分标准

点击「评分标准」进入第三步。为量表设置评分规则（如每个选项的分值、总分范围与结果等级的对应关系等）。

> 如果量表没有评分标准，发布时同样会被拦截。

### 第四步：发布量表

完成以上三步后，返回量表列表页面，该量表的状态显示为「草稿」。点击「发布」按钮，系统会先检查：
1. 量表是否有题目（`questionList` 不为空）
2. 量表是否有评分规则（`ratingList` 不为空）

如果检查不通过，弹出提示"该量表有内容未填写，请确认填写完整再发布"，并引导进入缺失的部分继续完善。

如果检查通过，发布成功，状态变为「发布」。发布后的量表在小程序端对用户可见。

### 第五步：管理已发布的量表

已发布的量表可以进行以下操作：
- **禁用**：点击「禁用」将量表下线，小程序端不再显示，状态变为「禁用」
- **启用**：对已禁用的量表点击「启用」可重新上线
- **上移/下移**：调整量表在同分类下的显示顺序（草稿状态不支持排序）

> 草稿状态的量表只能进行「编辑」和「发布」操作，不支持排序和启用/禁用。

## 继续完善功能

如果发布时被拦截（量表内容不完整），点击「完善量表」按钮会跳转到编辑页面，并**自动定位到未完成的步骤**：
- 没有题目 → 自动展开题目区域，显示空状态提示
- 有题目但没有评分 → 自动跳转到评分标准步骤

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 量表没有任何题目就发布 | 弹窗提示"该量表有内容未填写"，引导完善 |
| 量表有题目但没有评分标准就发布 | 同上 |
| 编辑已发布/禁用的量表 | 草稿状态的量表才可编辑，发布/禁用状态下没有编辑按钮 |
| 试图对草稿状态量表排序 | 上移/下移按钮不显示（`item.status !== 0`） |

## 关联功能

- 量表分类管理：创建量表时需要选择所属分类
- 小程序端量表：发布后的量表在小程序端供用户测评
- 量表测评记录：用户完成测评后的结果记录
