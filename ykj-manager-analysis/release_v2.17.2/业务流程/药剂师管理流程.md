# 药剂师管理流程

> 文档版本：2026-07-06 | 面向读者：运营/管理人员

## 概述

管理人员在后台对平台合作的药剂师进行统一管理，包括新增、编辑、删除、启停、CA认证同步、手机号变更等操作。药剂师信息包括姓名、性别、联系电话、邮箱、CA认证状态、入驻时间等。

## 触发条件

- **前置条件**：管理人员已登录后台，拥有药剂师管理权限（`hosp:pharmacists:write`）
- **入口路径**：后台 → 医院管理 → 药剂师列表

## 步骤详解

### 第一步：查看药剂师列表

进入页面自动加载药剂师列表，支持按姓名搜索。列表展示：

| 列名 | 说明 |
|------|------|
| 姓名 | 药剂师姓名 |
| 性别 | 男/女 |
| 联系电话 | 默认脱敏显示，点击可查看完整号码（需调用解密接口） |
| 邮箱地址 | 药剂师邮箱 |
| CA认证状态 | 未认证/认证中/已认证/认证失败（仅在CA功能开启时显示） |
| CA用户同步状态 | CA平台同步状态详情（仅在CA功能开启时显示） |
| 入驻时间 | 药剂师创建时间 |
| 状态 | 启用/禁用开关 |

### 第二步：新增药剂师

点击「新增」按钮，弹出药剂师信息表单：
- 填写基本信息：姓名、性别、联系电话、邮箱
- 选择所在医院、科室、职称、职务
- 选择民族、学历、药师专业
- 上传头像（限制2MB以内）
- 提交后自动创建药剂师账号

### 第三步：编辑药剂师信息

点击某条记录的「编辑」按钮，弹出已填充信息的编辑表单，修改后保存。

### 第四步：启用/禁用

点击状态开关可快速启用或禁用某个药剂师账号：
- 启用：调用 `pharmacistsEnable` 接口
- 禁用：调用 `pharmacistsDisable` 接口

### 第五步：CA认证管理（CA功能开启时显示）

**同步资料**：适用于"未认证"（authStatus=1）或"认证失败"（authStatus=4）的药剂师，点击后向CA平台同步资料。

**去认证**：适用于"认证中"（authStatus=2）的药剂师，点击后弹出二维码，药剂师扫码完成CA认证。

CA认证状态说明：
- 未认证（1）：资料未同步到CA平台
- 认证中（2）：资料已同步，等待认证
- 已认证（3）：已通过CA认证
- 认证失败（4）：数据不符合要求，显示失败原因

### 第六步：变更手机号

点击「变更手机号」弹出变更对话框：
- 输入新手机号（11位数字）
- 系统实时校验手机号是否已被其他医生或药剂师使用
- 如果手机号已存在，提示"手机号码已存在，请重新输入"
- 验证通过后调用 `editPhone` 接口完成变更
- 变更成功后弹出确认提示

### 第七步：删除药剂师

点击「删除」按钮，弹出确认对话框"此操作将永久删除该药师, 是否继续?"，确认后执行删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号为空 | 提示"手机号不能为空" |
| 手机号格式错误 | 提示"手机号格式错误"（需满足1开头11位数字） |
| 手机号已被使用 | 提示"手机号码已存在，请重新输入" |
| 填写格式不正确 | 提示"请填写正确的手机号码" |
| 取消删除操作 | 提示"已取消删除" |
| 删除操作失败 | 提示"操作失败" |

## 关联功能

- **CA认证**：通过二维码扫码完成药剂师CA身份认证
- **手机号解密**：联系电话默认脱敏，点击后通过解密接口展示完整号码
- **医院/科室字典**：表单中的医院、科室、职称等选项从字典数据加载
