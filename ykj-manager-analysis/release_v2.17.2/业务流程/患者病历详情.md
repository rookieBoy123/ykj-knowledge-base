# 患者病历详情

> 文档版本：v1.0 | 最后更新：2026-07-02 | 面向读者：运营人员、医院管理员

## 概述

患者病历详情页面汇总展示患者在一次就诊中的完整医疗信息，包括患者基本信息、病史资料、就诊信息、会诊记录（如有）和处方建议。页面采用左侧内容区+右侧导航锚点的布局，方便快速定位各信息模块。

## 触发条件

- **前置条件**：拥有管理后台登录权限
- **入口路径**：管理后台 → 医院管理 → 患者管理 → 患者详情 → 病历详情

## 页面结构

右侧固定导航菜单，点击可平滑滚动到对应模块：
1. 患者信息
2. 病史信息
3. 就诊信息
4. 会诊记录（如开启会诊功能）
5. 处方建议

## 步骤详解

### 一、患者信息

展示患者的基本资料：

| 字段 | 说明 |
|------|------|
| 就诊卡号 | 医院信息系统的患者编号，无数据时显示"-" |
| 姓名 | 患者真实姓名 |
| 性别 | 男（1）/ 女（2） |
| 出生日期 | 患者的出生年月日 |
| 年龄 | 优先显示"X岁"，新生儿显示"X月"，无法判断时显示"未知" |
| 证件号码 | 身份证号等信息 |

**监护人信息**（如有）：
- 监护人姓名
- 与患者关系（父亲/母亲/其他）
- 监护人身份证号（显示"已校验"标签）

**慢病医保信息**（如有）：
显示门慢医保复诊标签。

### 二、病史信息

展示患者本次就诊填写的病史资料。根据病历类型分为西医和中医两种模式。

**通用病史字段**：

| 字段 | 说明 |
|------|------|
| 主诉 | 患者主要症状描述，未填写显示"未填写" |
| 现病史 | 当前疾病的发生发展过程 |
| 既往史 | 过去的健康问题和疾病 |
| 家族史 | 家族成员的健康状况 |
| 个人史 | 个人生活习惯和环境 |
| 过敏史 | 药物/食物过敏情况 |
| 月经史 | 女性患者的月经情况 |
| 婚育史 | 婚姻和生育情况 |

**中医特有字段**（病历类型为中医时显示）：
- 脉象：中医脉诊结果
- 舌象：中医舌诊结果
- 治法：中医治疗方法

**西医特有字段**（病历类型为西医时显示）：
- 治疗意见：西医治疗建议

**其他字段**：
- 常见体征
- 辅助检查：辅助检查结果

### 三、就诊信息

展示本次就诊的详细情况：

| 字段 | 说明 |
|------|------|
| 就诊方式 | 固定显示"线上就诊" |
| 首诊机构 | 患者首次就诊的医疗机构 |
| 首诊时间 | 首次就诊时间 |
| 门诊号 | 线下门诊号 |
| 问诊时间 | 本次线上问诊的开始时间 |
| 服务类型 | 图文问诊/视频问诊等 |
| 科室名称 | 就诊科室 |
| 病情描述 | 患者填写的病情描述 |
| 病历资料 | 患者上传的病历图片，无图片显示"记录遗失/不在身边"，点击图片可放大查看 |

**主治医生信息**：显示医生姓名、科室、职务、医院名称，如为三甲医院标注"三甲"。

操作按钮：点击"问诊对话记录"可弹窗查看该问诊的完整对话（含文字、图片、语音消息），支持搜索和滚动加载。

### 四、会诊记录

如开启远程会诊功能，展示与该次就诊关联的会诊记录列表。每条记录显示：
- 会诊时间
- 会诊状态（会诊申请/会诊待备案/会诊待患者确认/会诊待开始/会诊进行中/待会诊报告/已完成）

点击某条记录可查看会诊详情。

如未开启会诊功能或没有会诊记录，该模块不显示。

### 五、处方建议

展示该次就诊医生开具的处方。分为两种类型：

**成药处方（type=1）**：
- 处方编号
- 诊断信息：诊断、辨病、辨证
- 药品清单：每种药品显示名称、规格、数量、用法用量
- 处方医生

**中药处方（type=2）**：
- 处方编号
- 诊断信息：诊断、辨病、辨证
- 药品清单：每种药材显示名称和数量
- 处方医生
- 医嘱禁忌：用药说明、用药时间、用药禁忌

操作按钮：
- "查看对应订单"：跳转到该处方关联的药品订单
- "查看原始处方"：查看处方原始图片

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 患者无病历时各字段为空 | 显示"未填写"或"-" |
| 年龄无法计算 | 显示"未知" |
| 无病历资料图片 | 显示"记录遗失/不在身边" |
| 无处方记录 | 显示空状态图标 |
| 无会诊记录 | 显示"暂无数据" |
| 未开启远程会诊功能 | 右侧导航和内容区都不显示会诊记录模块 |
| 问诊对话记录为空 | 提示"没有找到相关问诊对话记录!" |

## 关联功能

- **患者详情**：查看患者完整信息
- **门诊服务订单**：查看该患者的问诊订单
- **处方列表**：查看所有处方记录
- **会诊管理**：查看和管理会诊
