# 药品黑名单管理流程

> 文档版本 | 面向读者：运营管理员

## 概述
运营管理员可以在系统中维护一个"药品黑名单"——将某些药品通用名加入黑名单后，医生在开处方时将无法使用这些药品。这是一种安全管控机制，用于禁止使用某些有风险或不适合在本平台使用的药品。

## 触发条件
- **前置条件**：拥有「药品黑名单写权限」的管理员账号
- **入口路径**：管理后台 → 医院管理 → 药品信息 → 药品黑名单

## 步骤详解

### 第一步：查看黑名单列表
进入页面后，系统自动加载当前已配置的药品黑名单列表。每条记录显示：
- **通用名**：被限制的药品名称
- **更新时间**：最后一次修改的时间

列表支持按通用名搜索，方便在大量数据中快速定位。支持分页浏览。

### 第二步：新增黑名单药品
点击页面上的「新增」按钮，弹出一个表单窗口：
1. 在"通用名"输入框中填写要限制的药品名称（最多255个字）
2. 点击「确定」保存

系统会自动校验该药品是否已在黑名单中，如果重复添加会提示"药品重复"。

### 第三步：编辑黑名单条目
点击列表中某条记录的「编辑」按钮，可修改该药品的通用名。修改时同样会校验是否与其他条目重复。

### 第四步：删除黑名单条目
- **单个删除**：点击某条记录的「删除」按钮，确认后永久删除
- **批量删除**：勾选多条记录后点击「批量删除」按钮，确认后一次性删除所有选中项

删除前会弹出二次确认提示"此操作将永久删除选中项，是否继续？"，防止误操作。

### 第五步：Excel批量导入
通过「下载模板」按钮下载标准模板（包含"通用名"列），填写后通过「导入」按钮上传：
- 系统会自动去重——如果Excel中的药品已在黑名单中，该条记录会被标记为导入失败
- 导入成功后黑名单立即生效
- 导入失败的数据会在结果中列出，方便排查

### 第六步：下载模板
点击「下载模板」按钮，系统生成一个名为"黑名单导入模板.xlsx"的文件，包含表头"通用名"，管理员按模板填写即可。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 新增时药品名称已在黑名单中 | 弹出提示"药品重复"，不允许重复添加 |
| 编辑时药品名称与其他条目重复 | 弹出提示"药品重复"，保存失败 |
| 批量删除时未选中任何记录 | 提示"请至少选择一条数据" |
| Excel导入时药品已存在 | 已存在的药品被标记为导入失败，其他药品正常导入 |
| 导入Excel为空（无有效数据） | 全部标记为导入失败 |

## 业务影响
黑名单中的药品名称会在处方开具时被系统拦截——当医生在处方中搜索或添加药品时，系统会自动校验，如果药品名称包含黑名单中的关键词，该药品将无法被使用。这是一个实时生效的管控机制。

## 关联功能
- **处方开具**（医生端）：黑名单直接影响医生能否开出该药品
- **药品管理**：黑名单与药品主数据相互独立管理

## 权限要求
- 查看列表：`med:blacklists:read`
- 新增/编辑/删除：`med:blacklists:write`
- 导入Excel：`med:blacklists:import`
