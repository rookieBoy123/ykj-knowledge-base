# 商务列表管理流程

> 文档版本 2026-07-05 | 面向系统管理员、运营人员

## 概述

商务列表管理是运营管理后台中用于维护商务人员信息的模块。商务人员是平台中负责拓展医生、推广药品渠道的角色，系统管理员可以在此查看、新增、编辑、启用/禁用、删除商务信息，并为商务人员生成专属的邀请二维码。

## 触发条件

- **前置条件**：拥有运营管理后台的访问权限
- **入口路径**：运营管理 → 商务管理 → 商务列表

## 步骤详解

### 第一步：进入商务列表

页面加载后展示所有已注册的商务人员列表，表格包含以下信息：

- **商务编号**：系统自动生成的唯一编号（格式：BUSI + 数字）
- **商务姓名**：商务人员的姓名
- **性别**：显示为"男"或"女"
- **联系电话**：默认显示为脱敏状态（中间四位隐藏），点击可查看完整号码（单次有效）
- **审核时间**：最近一次审核通过的时间
- **审核状态**：显示为"待审核""审核通过"或"审核不通过"
- **状态开关**：显示当前账号的启用/禁用状态，可直接切换
- **邀请**：点击"查看"按钮可查看该商务已邀请的医生列表和药品渠道统计

### 第二步：搜索和筛选商务

页面顶部提供搜索条件：

- **商务姓名**：输入商务姓名模糊搜索
- **状态**：按"启用"或"禁用"筛选
- **注册时间**：按日期范围筛选注册时间

点击搜索按钮后，列表重新加载符合条件的商务记录。

### 第三步：新增商务

在商务列表页，点击"新增商务"按钮会弹出新增表单，需填写以下信息：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 商务姓名 | 最多20个字符 | 是 |
| 性别 | 从字典选项中选择"男"或"女" | 否 |
| 手机号码 | 11位手机号，支持数字输入 | 否 |
| 身份证号码 | 最多18位 | 否 |
| 开户行 | 银行名称，最多100字符 | 否 |
| 银行账号 | 最多100字符 | 否 |
| 详细地址 | 最多255字符 | 否 |
| 状态 | "启用"或"禁用" | 是 |

提交后，商务信息进入"待审核"状态，需在审核列表中进行审核。

### 第四步：编辑商务信息

点击某条商务记录的"编辑"按钮，弹出与新增相同的表单，可修改除审核状态外的所有字段。修改后保存即可。

> **注意**：如果商务的审核状态为"审核不通过"，操作按钮会显示为"修改资料"而非"编辑"，点击后提交时会自动将审核状态重置为"待审核"（10），重新进入审核流程。

### 第五步：启停商务账号

列表中的状态开关可直接切换商务账号的启用/禁用状态：

- **启用**：确认后会弹出提示"是否确认启用该商务账号"
- **禁用**：确认后会弹出提示"是否确认禁用该商务账号，禁用后该商务账号不能正常使用"

切换后无需审核，即时生效。

### 第六步：查看邀请情况

点击列表中的"查看"按钮，弹出"邀请医生"窗口，展示：

- 该商务已邀请的医生列表（分页展示，可按医生姓名和时间筛选）
- 药品渠道统计（药品数量汇总）
- 支持刷新和重新查询

### 第七步：邀请医生二维码

点击某条商务记录的"邀请医生码"按钮，弹出一个二维码图片弹窗：

- 医生扫描此二维码后，会被记录为该商务邀请的医生
- 点击"刷新二维码"可重新生成（旧二维码失效）
- 如果获取失败，会提示"无法获取商务二维码，请联系管理员"

### 第八步：变更手机号

点击"变更手机号"按钮，弹出单独的手机号修改窗口：

- 输入新的11位手机号码
- 确认后系统会修改该商务关联的会员手机号
- 修改成功后，会先后弹出两个提示：
  1. "手机号码修改成功！"
  2. "请检查该电话号码是否有关联医生或药师账号"

### 第九步：删除商务

点击"删除"按钮，弹出二次确认："此操作将永久删除该商务, 是否继续?"。确认后商务信息被永久删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号码格式不正确 | 提交时提示"请正确填写电话号码" |
| 商务姓名为空 | 提示"请输入名称" |
| 未选择状态 | 提示"请选择状态" |
| 无法获取二维码 | 提示"无法获取商务二维码，请联系管理员" |
| 手机号变更未填写 | 提示"请填写正确的手机号码" |
| 禁用商务账号 | 弹出二次确认："禁用后该商务账号不能正常使用" |

## 关联功能

- **商务审核列表**：新增的商务需在审核列表中审核通过后方可正常使用
- **邀请医生管理**：可查看商务邀请的医生列表
- **会员管理**：商务人员的手机号关联会员体系
- **二维码刷新**：刷新后旧二维码失效
