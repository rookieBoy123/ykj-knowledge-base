# 商务列表管理流程

> 文档版本 2026-07-04 | 面向读者：运营管理员

## 概述

商务人员是平台拓展医生资源的业务人员。管理员可以在此管理所有商务人员的账号信息，包括新增、编辑、启停、删除，以及邀请医生码查看和手机号变更。

## 触发条件

- **前置条件**：拥有运营管理权限
- **入口路径**：管理后台 → 运营管理 → 商务管理 → 商务列表

## 步骤详解

### 第一步：查询商务列表

进入页面后，系统自动加载商务人员列表。列表展示以下信息：
- **商务编号**：系统自动生成的唯一编号
- **商务姓名**
- **性别**：男/女
- **联系电话**：默认脱敏显示，点击可查看完整号码
- **审核时间**
- **审核状态**：待审核 / 审核通过 / 审核不通过
- **状态开关**：启用/禁用切换

**筛选条件**：支持按"商务姓名""状态（启用/禁用）""注册时间"进行筛选查询。

### 第二步：新增/编辑商务

点击列表上方（通过特定入口）或每行的"编辑"/"修改资料"按钮：

**新增/编辑表单字段**：

| 字段 | 说明 | 校验规则 |
|------|------|----------|
| 商务姓名 | 必填 | 最长 20 字 |
| 性别 | 从字典"性别代码"中拉取选项 | — |
| 手机号码 | 必填 | 11 位手机号格式 |
| 身份证号码 | 选填 | 最长 18 位 |
| 开户行 | 选填 | 最长 100 字 |
| 银行账号 | 选填 | 最长 100 字 |
| 详细地址 | 选填 | 最长 255 字 |
| 状态 | 启用/禁用 | 必选 |

提交后需要二次确认。

### 第三步：启用/禁用商务

列表中有状态开关，管理员可直接切换：
- **启用**：确认"是否确认启用该商务账号"
- **禁用**：确认"是否确认禁用该商务账号，禁用后该商务账号不能正常使用"

**重要**：审核不通过的商务，操作按钮显示为"修改资料"而非"编辑"。修改资料时，会将审核状态重置为"待审核"。

### 第四步：查看邀请医生

点击某商务的"查看"按钮（邀请列），弹出"邀请医生"对话框，展示：
- 该商务邀请的医生列表
- 医生相关的药品统计数量
- 支持搜索和分页浏览

### 第五步：查看/刷新邀请医生码

点击"邀请医生码"按钮，弹出二维码展示窗口：
- 显示该商务的专属邀请医生二维码
- 点击"刷新二维码"可重新生成新的邀请码

### 第六步：变更手机号

点击"变更手机号"按钮，弹出输入框：
- 输入新的 11 位手机号码
- 提交后系统更新手机号，并弹出二次提醒："请检查该电话号码是否有关联医生或药师账号"

### 第七步：删除商务

点击"删除"按钮，弹出确认提示："此操作将永久删除该商务，是否继续？"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号格式不正确 | 表单校验提示"请正确填写电话号码" |
| 禁用商务账号 | 提示"禁用后该商务账号不能正常使用"，需确认 |
| 审核不通过的商务编辑 | 操作按钮显示为"修改资料"，修改后自动重置审核状态为"待审核" |
| 变更手机号后 | 提醒管理员检查是否有关联的医生或药师账号 |
| 无法获取商务二维码 | 提示"无法获取商务二维码，请联系管理员" |
| 手机号脱敏显示 | 需管理员主动点击查看完整号码，每次只能查看一个 |

## 关联功能

- **商务审核**：新增的商务需要到"商务审核列表"页面进行审核
- **邀请医生**：商务通过分享二维码邀请医生入驻平台
- **医生管理**：医生列表中有商务归属关系
- **服务端接口**：对应 `MgmtBusinessController`（ykj-service-server）
