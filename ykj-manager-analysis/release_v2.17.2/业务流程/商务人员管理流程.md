# 商务人员管理流程

> 文档版本 v1.0 | 面向读者：运营管理员

## 概述

商务人员是平台运营中的重要角色，负责邀请和管理医生入驻。此页面提供商务人员的完整管理功能：新增、编辑、启用/禁用、查看邀请的医生、管理邀请二维码、变更手机号等。

## 触发条件

- **前置条件**：管理员已登录管理后台，拥有商务管理权限
- **入口路径**：运营管理 → 商务管理 → 商务列表

## 步骤详解

### 第一步：查看和搜索商务列表

页面展示所有商务人员的分页列表，支持以下搜索条件：
- **商务姓名**：模糊搜索
- **状态**：启用/禁用
- **注册时间**：日期范围筛选

列表展示字段：
- 商务编号（系统自动生成）
- 商务姓名
- 性别（男/女）
- 联系电话（默认脱敏显示，点击眼睛图标可查看完整号码，需经过权限校验）
- 审核时间
- 审核状态：待审核(10)、审核通过(20)、审核不通过(30)
- 状态开关：启用(1)/禁用(0)，切换时有确认提示

### 第二步：新增商务

点击新增按钮打开表单弹窗，需填写以下信息：
- **商务姓名**（必填，最长20字）
- **性别**（从数据字典"xbdm"加载选项）
- **手机号码**（必填，11位手机号格式校验）
- **身份证号码**（最长18位）
- **开户行**（最长100字）
- **银行账号**（最长100字）
- **详细地址**（最长255字）
- **状态**（启用/禁用）

提交后，商务的状态自动设为"待审核"（flowStatus=10），等待管理员审核。

### 第三步：编辑商务

点击"编辑"按钮打开表单弹窗，预填已有信息，修改后保存。

**特殊规则**：
- 审核通过的商务（flowStatus=20）可正常编辑
- 审核不通过的商务（flowStatus=30）按钮文案变为"修改资料"，编辑提交后审核状态自动重置为"待审核"（flowStatus=10），重新进入审核流程

### 第四步：查看邀请的医生

点击"邀请"列的"查看"按钮，弹出"邀请医生"弹窗，展示该商务已邀请的医生列表：
- 医生列表分页展示，支持按医生姓名和时间范围搜索
- 同时展示该商务的药品销量统计数据
- 可在弹窗内对医生列表进行分页翻页

### 第五步：管理邀请二维码

点击"邀请医生码"按钮，弹出二维码弹窗，显示该商务的专属邀请二维码图片。医生扫描此二维码后可注册并关联到该商务名下。

**刷新二维码**：点击弹窗中的"刷新二维码"链接可重新生成二维码。刷新失败时提示"无法获取商务二维码，请联系管理员"。

### 第六步：变更手机号

点击"变更手机号"打开确认弹窗：
- 输入新的手机号码（11位格式校验）
- 确认后调用接口修改
- 修改成功后会先后弹出两个提示："手机号码修改成功！" → "请检查该电话号码是否有关联医生或药师账号"
- 这一连续提示确保管理员意识到手机号变更可能影响关联账号

### 第七步：启用/禁用商务

列表中的状态开关可直接切换：
- 启用→禁用：提示"是否确认禁用该商务账号，禁用后该商务账号不能正常使用"
- 禁用→启用：提示"是否确认启用该商务账号"
- 切换后即时生效

### 第八步：删除商务

点击"删除"按钮：
- 弹出确认提示："此操作将永久删除该商务, 是否继续?"
- 确认后执行删除
- 取消时提示"已取消删除"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号格式错误 | 提示"请正确填写电话号码" |
| 手机号码为空 | 提示"电话不能为空" |
| 保存时姓名为空 | 提示"请输入名称" |
| 状态未选择 | 提示"请选择状态" |
| 关闭编辑弹窗 | 弹出确认提示"确认关闭？" |
| 列表空数据跳页 | 自动回退到上一页 |
| 二维码获取失败 | 提示"无法获取商务二维码，请联系管理员" |
| 审核不通过再次编辑 | 按钮显示"修改资料"，提交后状态重置为待审核 |

## 关联功能

- 商务审核（审核列表页面，仅显示待审核的商务）
- 医生管理（邀请的医生列表）
- 会员管理（变更手机号实际修改会员手机号）
