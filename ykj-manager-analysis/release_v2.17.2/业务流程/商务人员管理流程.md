# 商务人员管理流程

> 文档版本 1.0 | 面向读者：运营/商务管理人员

## 概述

商务人员是平台的市场拓展角色，负责**邀请医生入驻平台**。这里管理所有商务人员的资料、审核状态、启用/禁用，并提供邀请医生码、变更手机号等功能。

## 触发条件

- 进入「运营管理 → 商务列表」页面

## 列表信息

商务列表展示以下字段：

| 列名 | 说明 |
|------|------|
| 商务编号 | 系统自动生成的唯一编号 |
| 商务姓名 | 商务人员姓名 |
| 性别 | 男/女（通过字典"xbdm"转换） |
| 联系电话 | 默认脱敏显示，点击"眼睛"图标查看完整号码（需操作权限） |
| 审核时间 | 审核通过的时间 |
| 审核状态 | 待审核(10) / 审核通过(20) / 审核不通过(30) |
| 状态开关 | 启用/禁用开关 |
| 邀请 | 点击"查看"查看该商务邀请的医生列表 |

## 步骤详解

### 第一步：筛选查询

可通过以下条件筛选：
- **商务姓名**：输入姓名模糊搜索
- **状态**：启用/禁用下拉选择
- **注册时间**：日期范围筛选

### 第二步：新增商务

点击「新增商务」按钮，弹出表单：

| 字段 | 校验规则 |
|------|---------|
| 商务姓名 | 必填，最多20字 |
| 性别 | 下拉选择（男/女） |
| 手机号码 | 必填，11位手机号格式校验 |
| 身份证号码 | 最多18位 |
| 开户行 | 最多100字 |
| 银行账号 | 最多100字 |
| 详细地址 | 最多255字 |
| 状态 | 启用/禁用 |

提交后，新商务的审核状态自动设为"待审核"。

### 第三步：编辑商务

点击操作栏的「编辑」或「修改资料」按钮：
- **审核通过或待审核的商务**：显示"编辑"按钮
- **审核不通过的商务**：显示"修改资料"按钮（修改后审核状态自动变回"待审核"）
- 编辑表单回填所有已有信息

### 第四步：启用/禁用

列表中的状态开关：
- 点击「启用」确认："是否确认启用该商务账号"
- 点击「禁用」确认："是否确认禁用该商务账号，禁用后该商务账号不能正常使用"

### 第五步：邀请医生管理

点击「查看」按钮，弹出"邀请医生"弹窗：
- 显示该商务邀请的所有医生列表
- 同时展示药品统计数据（MedicineCounts）
- 支持医生姓名搜索和日期范围筛选

### 第六步：邀请医生码

点击操作栏的「邀请医生码」按钮：
- 弹窗展示一个二维码图片
- 医生扫码后可关联到该商务名下
- 支持「刷新二维码」（重新生成）

### 第七步：变更手机号

点击「变更手机号」按钮，弹出手机号修改窗口：
- 输入新手机号（11位格式校验）
- 提交成功后依次弹出两次提示：
  - 第一次："手机号码修改成功！"
  - 第二次："请检查该电话号码是否有关联医生或药师账号"

### 第八步：删除商务

点击「删除」按钮，弹出二次确认：
"此操作将永久删除该商务, 是否继续?"

## 特殊规则

| 规则 | 说明 |
|------|------|
| 审核不通过可重新提交 | 修改资料后审核状态从30变回10 |
| 联系电话脱敏 | 默认隐藏中间4位，点击眼睛图标需调用专用接口获取明文 |
| 手机号变更提醒 | 变更后会提醒检查关联的医生/药师账号 |
