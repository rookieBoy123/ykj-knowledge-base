# 商务人员管理流程

> 文档版本 1.0 | 面向读者：运营人员、管理员

## 概述

商务人员（地推人员）管理用于维护平台线下推广人员的基本信息，包括姓名、联系方式、身份信息、银行账户等，同时管理商务与医生的邀请关系。商务提交资料后需经过审核，审核通过方可正常使用。

## 触发条件

- **入口路径**：运营管理 → 商机管理 → 商务列表
- **前置条件**：需拥有商务管理权限

## 步骤详解

### 第一步：查看商务列表

列表页面展示所有商务人员信息，包含以下列：
- **商务编号**：系统自动生成的唯一编号
- **商务姓名**：商务人员的真实姓名
- **性别**：显示为"男"或"女"
- **联系电话**：默认脱敏显示（中间数字隐藏），点击可查看完整号码
- **审核时间**：最近一次审核通过的时间
- **审核状态**：待审核 / 审核通过 / 审核不通过
- **启用状态**：开关控件，可随时启用或禁用
- **操作按钮**：编辑、邀请医生码、变更手机号、删除等

支持按**商务姓名**、**启用/禁用状态**、**注册时间范围**进行筛选。

### 第二步：新增/编辑商务

点击列表中的"编辑"按钮（或新增按钮）打开表单弹窗，需填写以下信息：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 商务姓名 | 真实姓名，最多20字 | 是 |
| 性别 | 从下拉选"男"或"女" | 否 |
| 手机号码 | 11位手机号 | 否 |
| 身份证号码 | 18位身份证号 | 否 |
| 开户行 | 银行卡开户行名称，最多100字 | 否 |
| 银行账号 | 银行卡号，最多100字 | 否 |
| 详细地址 | 联系地址，最多255字 | 否 |
| 状态 | 启用/禁用 | 是 |

### 第三步：审核流程

商务资料提交后进入审核流程，状态流转如下：

```
新增/修改资料 → 待审核(10) → 审核通过(20) → 可正常使用
                            → 审核不通过(30) → 需修改后重新提交
```

- **待审核**：商务刚提交资料或修改资料后的状态
- **审核通过**：审核人员通过后在后台操作，状态变为"审核通过"
- **审核不通过**：资料有问题被驳回，操作按钮变为"修改资料"
  - 点击"修改资料"可重新编辑信息，提交后状态自动回到"待审核(10)"
  - 编辑时保留之前的填写内容方便修改

### 第四步：启用/禁用管理

列表中每行有启用/禁用开关：
- **启用**：商务账号可正常登录和使用
- **禁用**：商务账号不能正常使用
- 切换时弹窗确认，提示禁用后的影响

### 第五步：邀请医生管理

点击"邀请"列下的"查看"按钮，弹出邀请医生详情窗口：

- 上半部分展示该商务邀请的医生统计信息（药品相关数据）
- 下半部分展示邀请的医生列表，支持按医生姓名、注册时间筛选
- 可通过"商务邀请医生列表"页面进行批量管理（见关联功能）

### 第六步：邀请医生码

点击"邀请医生码"按钮，弹出二维码弹窗：
- 显示该商务专属的邀请二维码
- 可将二维码分享给医生，医生扫码即可进入入驻流程
- 支持"刷新二维码"生成新的邀请码

### 第七步：变更手机号

点击"变更手机号"按钮，弹出手机号修改弹窗：
- 输入新的11位手机号码
- 修改成功后，系统会连续弹出两个提示：
  1. "手机号码修改成功！"
  2. "请检查该电话号码是否有关联医生或药师账号"
- 确保管理员意识到手机号变更可能影响关联账户

### 第八步：删除商务

点击"删除"按钮，二次确认后可永久删除该商务信息。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 提交时姓名为空 | 提示"请输入名称" |
| 提交时状态未选 | 提示"请选择状态" |
| 手机号码格式不正确 | 提示"请正确填写电话号码" |
| 手机号码为空 | 提示"电话不能为空" |
| 审核不通过时点编辑 | 按钮文字变为"修改资料"，提交后flowStatus重置为10（待审核） |
| 昵名用户查看电话 | 需要授权查看完整号码，系统脱敏显示中间4位 |
| 不能获取二维码 | 提示"无法获取商务二维码，请联系管理员" |

## 关联功能

- **商务邀请医生列表**（运营管理 → 商机管理 → 邀请医生）：查看所有商务邀请的医生，支持批量修改邀请商务、导出数据
- **医生入驻流程**：医生扫描商务邀请码后进入入驻流程，入驻成功关联到对应商务
- **商机审核列表**（运营管理 → 商机管理 → 商机审核）：审核人员在此处理商务审核
- **会员管理**：变更手机号功能调用会员模块接口
