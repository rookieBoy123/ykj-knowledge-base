# 设施信息管理流程

> 文档版本：v1.0 | 面向读者：医院设备管理员、运营人员

## 概述

设施信息管理功能用于记录和维护医院各类医疗设备的台账信息。管理人员可以对设备进行新增、编辑、删除操作，并通过设备名称、型号、产地、使用情况等条件筛选查询。每台设备记录包含编号、类型、产地、购买日期、生产厂家、价值金额、设计寿命等完整信息。

## 触发条件

- **入口路径**：管理后台 → 医院管理 → 设施信息
- **前置条件**：拥有「hosp:devices:write」权限才能新增设备

## 步骤详解

### 第一步：查看设备列表

进入页面后，系统自动加载设备列表，以表格形式展示。每条设备记录包含：
- **设备编号**：系统自动生成的唯一编号
- **设备名称**
- **设备型号**
- **产地**：从数据字典「设备产地」中取值显示
- **设备类型**：从数据字典「设备类型」中取值显示
- **使用情况**：从数据字典「设备使用情况」中取值显示
- **创建时间**、**修改时间**

表格支持分页浏览，底部显示总记录数。

### 第二步：筛选查询设备

页面顶部提供四个筛选条件：
- **设备名称**：输入框，模糊搜索，最长 20 字
- **设备型号**：输入框，模糊搜索
- **产地**：下拉选择，选项来自「设备产地」数据字典
- **使用情况**：下拉选择，选项来自「设备使用情况」数据字典

设置条件后点击查询，列表按筛选条件刷新。切换页码或页大小时保持当前筛选条件。

### 第三步：新增设备

点击「新增设备」按钮（需要权限），弹出新增表单，包含以下字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 设备名称 | 文本 | 是 | 最长 20 字 |
| 设备类型 | 下拉 | 是 | 来自「设备类型」字典 |
| 产地 | 下拉 | 是 | 来自「设备产地」字典 |
| 使用情况 | 下拉 | 是 | 来自「设备使用情况」字典 |
| 购买日期 | 日期 | 是 | 日期选择器 |
| 生产厂家 | 文本 | 否 | 最长 120 字 |
| 设备型号 | 文本 | 否 | 最长 200 字 |
| 设备价值金额 | 文本 | 否 | 金额输入 |
| 理论设计寿命 | 文本 | 否 | 单位：年 |
| 购进时新旧况 | 下拉 | 否 | 来自「设备新旧情况」字典 |
| 同批购进相同型号设备台数 | 文本 | 否 | 数字输入 |
| 设备参数 | 文本 | 否 | 最长 500 字 |
| 用途 | 文本 | 否 | 最长 500 字 |
| 备注 | 文本 | 否 | 最长 100 字 |

填写完成点击「确定」保存。必填字段未填写时会给出红色提示。

### 第四步：编辑设备

在列表操作栏点击「编辑」，弹出编辑表单，表单中预填当前设备的所有信息。修改后点击「确定」保存，列表自动刷新。

### 第五步：删除设备

在列表操作栏点击「删除」，弹出确认对话框：「此操作将永久删除该项, 是否继续?」
- 点击「确定」：执行删除，成功后提示「操作成功」并刷新列表
- 点击「取消」：关闭对话框，提示「已取消删除」

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 设备名称未填写 | 提示「请输入设备名称」 |
| 设备类型未选择 | 提示「请选择设备类型」 |
| 产地未选择 | 提示「请选择产地」 |
| 使用情况未选择 | 提示「请选择使用情况」 |
| 购买日期未选择 | 提示「请选择购买日期」 |
| 无「hosp:devices:write」权限 | 新增按钮不显示 |
| 删除操作取消 | 提示「已取消删除」 |

## 关联功能
