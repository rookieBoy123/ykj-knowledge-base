# 中药图鉴管理流程

> 文档版本 2026-07-04 | 面向读者：内容运营人员

## 概述

管理员在后台维护中药图鉴百科内容，包括中药的基本信息、药味、适应症、功效作用及四个维度的详细说明（基本用法、药物作用、用药方法、注意事项）。内容展示在患者端"中医百科-中药图鉴"模块。

## 触发条件

- 入口路径：管理后台 → 运营管理 → 医学百科 → 中药图鉴管理
- 适用角色：拥有医学百科管理权限的管理员

## 步骤详解

### 第一步：查看中药图鉴列表

进入页面后，系统展示中药图鉴列表，每条记录包含：
- **封面图**：中药的展示缩略图
- **中药名称**：如"人参""黄芪""当归"等
- **药味**：苦、甘、辛等（可多个，用顿号分隔）
- **适应症**：该中药适用的症状（可多个）
- **最后修改时间**：最近编辑时间

列表支持按中药名称（关键词）、药味、适应症三个维度筛选。筛选选项来自数据字典。

### 第二步：新增中药图鉴

点击「新增」按钮，跳转到编辑页面，需要填写：

**基本信息区：**
- **中药名称**：必填，最多20字
- **药味**：必填，可从下拉多选（如酸、苦、甘、辛、咸等）
- **适应症**：可选多个症状
- **功效作用**：必填，最多100字，简要说明药理功效
- **展示封面图**：可选，建议上传 200×200px 的 1:1 比例图片

**中药详情区（四个标签页，均使用富文本编辑器）：**
- **基本用法**：中药的基本使用方法（必填）
- **药物作用**：药物的治疗机理（必填）
- **用药方法**：具体的用药指导（必填）
- **注意事项**：使用禁忌和提醒（必填）

> ⚠️ 四个详情标签页均不能为空。提交时会逐个检查，哪个未填就自动跳转到哪个标签页并给出提示。

### 第三步：编辑中药图鉴

点击「编辑」按钮，跳转到编辑页面，系统回填已有信息。修改后点击「确定」保存。

### 第四步：删除中药图鉴

- **单个删除**：点击「删除」按钮，二次确认后删除
- **批量删除**：勾选多条后点击「批量删除」

删除后数据不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 中药名称为空 | 前端提示"请填写中药名称" |
| 药味未选择 | 前端提示"请选择药味" |
| 功效作用未填写 | 前端提示"请填写功效作用" |
| 四个详情标签页未填写 | 提交时自动切换到未填标签页，提示对应文案 |
| 批量删除未勾选 | 静默无提示 |

## 关联功能

- **患者端中药图鉴浏览**：小程序「中医百科 → 中药图鉴」展示此处维护的内容
- **数据字典**：药味（zytj-yw）、适应症（zytj-syz）的下拉选项来自字典
