# 医生收益结算管理流程

> 文档版本：2026-07-04 | 面向读者：财务人员、运营人员

## 概述

管理平台中所有医生的累计收益，支持查看医生收益明细、批量结算医生收益、查看结算记录，并提供导出功能。该功能是平台与医生之间财务结算的核心环节。

## 触发条件

- **前置条件**：拥有财务模块访问权限、医生收益管理权限
- **入口路径**：管理后台 → 财务统计 → 收益管理

## 页面结构

页面顶部有标签页切换，分为两个视图：

### 收益记录（默认视图）
以表格形式展示所有医生的收益汇总，每行代表一位医生，列包括：
- 医生编号
- 医生姓名
- 所属团队（仅在开通专家团队时显示）
- 联系电话（默认脱敏显示，点击眼睛图标可查看完整号码）
- 银行卡号（默认脱敏显示，点击眼睛图标可查看完整卡号）
- 总共收益（元）
- 操作：结算按钮

表格支持多选（checkbox），可勾选多个医生后使用"批量结算"按钮。

### 结算记录（第二视图）
以表格形式展示历史结算记录，列包括：
- 医生编号
- 医生姓名
- 银行卡号（默认脱敏显示）
- 结算金额（元）
- 结算日期

## 步骤详解

### 第一步：查看收益记录

进入页面后默认显示"收益记录"标签页：
- 可通过"医生姓名"输入框模糊搜索
- 可通过"所属团队"下拉筛选（仅开通专家团队时可用）
- 表格展示每位医生的收益汇总和联系方式

点击联系电话或银行卡号旁的眼睛图标，可查看脱敏前完整信息。点击一次显示明文，再次点击恢复脱敏。

### 第二步：结算医生收益

**单个医生结算：**
1. 在收益记录表格中，点击某位医生行末的"结算"按钮
2. 如果该医生收益为0，弹出提示"该医生收益为0"，无法结算
3. 如果收益大于0，弹出结算确认窗口，显示医生姓名、银行卡号、银行名称、开户行名称、卡主名称、结算金额
4. 确认无误后点击"确定"，系统执行结算
5. 结算成功后弹出成功提示，收益记录刷新

**批量结算：**
1. 在表格中勾选需要结算的医生（可多选）
2. 点击表格上方的"批量结算"按钮
3. 如果未选择任何医生，提示"请至少选择一条数据"
4. 弹出结算确认窗口，展示所有被选中医生的结算信息
5. 确认后点击"确定"，批量执行结算

### 第三步：查看结算记录

切换到"结算记录"标签页：
- 可按日期范围筛选历史结算记录
- 查看结算金额和结算日期
- 支持导出结算记录为 Excel

### 第四步：导出数据

**收益记录导出**：在收益记录页面，点击"导出"按钮，导出当前筛选条件下的医生收益汇总。

**结算记录导出**：在结算记录页面，点击"导出"按钮，导出当前筛选条件下的结算记录。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 医生收益为0 | 点击结算时弹出"该医生收益为0"提示，无法结算 |
| 未选择任何医生点击批量结算 | 弹出"请至少选择一条数据"提示 |
| 未开通专家团队 | 所属团队列和筛选项自动隐藏 |
| 银行卡号为空 | 显示"暂无银行卡" |
| 联系电话被脱敏 | 默认显示为星号部分隐藏的号码，需点击眼睛图标查看完整信息 |

## 安全措施

- 联系电话和银行卡号默认脱敏显示，防止敏感信息泄露
- 查看完整信息（手机号、银行卡号）需要调用专门的解密接口
- 结算操作需要"医生收益结算"权限（`hosp:doctor-incomes:settle`）
- 导出操作需要"医生收益导出"权限（`hosp:doctor-incomes:export`）

## 关联功能

- **交易流水查询**：追溯每笔收益对应的原始交易
- **医生信息管理**：查看医生的基本信息、银行卡绑定状态
- **药品订单统计**：了解医生开药产生的收益构成
