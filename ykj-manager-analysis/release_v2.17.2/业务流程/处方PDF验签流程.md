# 处方PDF验签流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、药房管理人员

## 概述

验证电子处方PDF文件的CA（数字证书）签名真实性。当患者或药房对处方的真实性存疑时，管理员可以通过上传处方PDF文件来验证其CA签名是否有效。

## 触发条件

- **入口路径**：诊疗管理 → 处方管理 → 处方验签
- **前置条件**：用户具有处方管理权限；需要验证的处方PDF文件

## 步骤详解

### 第一步：上传处方PDF

进入页面后，将电子处方PDF文件拖拽到上传区域，或点击「点击上传」选择文件。

上传限制：
- 仅支持 **PDF格式** 文件
- 单文件大小不超过 **2MB**
- 只能上传1个文件

如果上传非PDF文件或文件过大，系统会立即给出错误提示。

### 第二步：点击验证

文件上传后，点击「验证」按钮。系统会将PDF文件发送给CA签章服务商，由服务商验证文件中电子签名的真实性和有效性。

**注意**：未上传文件时点击「验证」，系统提示"请先上传pdf文件！"

### 第三步：查看结果

验证完成后，系统显示结果：

- **验证通过**：提示"验证通过！"，说明该处方的CA签名是真实有效的
- **验证不通过**：提示"验证不通过！"，说明签名无效、被篡改或不是本平台开具的处方

验证完成后（无论通过与否），上传的文件会被自动清空，方便进行下一次验证。

### 验签说明

页面右侧有验签说明：
1. 在线验签是将上传的电子处方PDF文件通过CA厂家提供的验证PDF签名结果接口，验证签名的真实性
2. 仅支持本互联网医院开具的电子处方验签

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 上传非PDF文件 | 提示"只能上传pdf文件！"，阻止上传 |
| PDF文件超过2MB | 提示"上传pdf文件大小不能超过 2MB!"，阻止上传 |
| 未上传文件就点验证 | 提示"请先上传pdf文件！" |
| CA签章服务商接口失败 | 显示"验证不通过！" |
| 处方不是本平台开具 | 显示"验证不通过！" |
| 处方签章已被篡改 | 显示"验证不通过！" |

## 关联功能

- **处方列表管理**：查看和管理所有处方
- **处方药师审核与电子签名**：处方开具时的CA签章流程
- **CA签章服务商**：依赖第三方CA签名验证接口
