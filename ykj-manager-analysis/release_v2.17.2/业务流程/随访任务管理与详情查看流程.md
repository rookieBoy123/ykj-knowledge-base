# 随访任务管理与详情查看流程

> 文档版本 v1.0 | 面向读者：运营人员、随访管理人员

## 概述

管理员在后台查看所有医生创建的随访任务，支持按状态、创建人、患者等多维度筛选。点击"查看详情"可查看任务的完整信息，包括患者信息、随访内容（患教资料/提醒/问卷）的各项完成情况。

## 触发条件

- **前置条件**：管理员已登录管理后台，拥有随访任务查看权限（`fup:tasks:read`）
- **入口路径**：诊疗管理 → 随访管理 → 随访任务

## 步骤详解

### 第一步：查看随访任务列表

进入页面后，系统默认查询当月（本月1日到当天）的随访任务列表，每页10条。

列表展示以下信息：

| 列名 | 说明 |
|------|------|
| 患者 | 患者姓名 |
| 手机号 | 患者手机号 |
| 随访任务 | 该任务包含的所有内容标题（多个用逗号分隔） |
| 随访状态 | 待随访 / 已随访 / 已完成 / 已取消 |
| 创建人 | 创建该随访任务的医生姓名 |
| 创建时间 | 任务创建时间 |
| 随访日期 | 计划发送随访的日期 |
| 操作 | 「查看详情」按钮 |

### 第二步：按状态筛选

列表顶部提供Tab快捷筛选，支持以下状态：

| Tab | 含义 | 说明 |
|-----|------|------|
| 全部 | 不筛选状态 | 默认选中 |
| 待随访 | 任务已创建但尚未开始随访 | 患者还未收到任何随访内容 |
| 已随访 | 随访内容已发送给患者 | 患者正在进行中 |
| 已完成 | 所有随访内容已完成 | 全部待办事项已处理 |
| 已取消 | 任务被取消 | 不再执行 |

### 第三步：按条件搜索

除了状态筛选外，还支持以下搜索条件：

- **创建时间**（日期范围）：筛选在指定时间段内创建的任务
- **随访日期**（日期范围）：筛选计划在指定时间段内发送的任务，默认范围为当月1日至当天
- **创建人**：通过远程搜索医生姓名来选择（支持模糊搜索，最多返回500条匹配医生）
- **患者姓名**：精确匹配患者姓名（最多5个字）
- **手机号码**：精确匹配手机号（最多11位）

### 第四步：查看随访任务详情

点击「查看详情」按钮，弹出详情弹窗，包含两大信息区域：

#### 患者信息区

- **患者头像**（圆形显示）
- **患者姓名** + **性别** + **年龄**
- **手机号码**

#### 随访信息区

- **随访状态标签**（颜色区分：待随访/已随访=橙色warning，已完成=绿色success，已取消=红色danger）
- **随访任务内容列表**：逐条展示该任务包含的所有内容，格式为"类型：内容标题"：

| 类型编号 | 类型名称 | 显示方式 |
|---------|---------|---------|
| 0 | 患教资料 | 蓝色文字可点击，点击弹出患教详情弹窗 |
| 1 | 提醒 | 普通文字，不可点击 |
| 2 | 问卷 | 蓝色文字可点击，点击弹出问卷详情弹窗 |

- **每条内容的完成状态**：在已随访或已完成状态下，每条内容右侧显示标签：
  - 「已完成」（绿色）：该内容已被患者完成
  - 「待办」（橙色）：该内容尚未完成
- **禁用标识**：如果某条内容（患教或问卷）在任务发送前被禁用，且任务尚未到达发送时间，显示红色「已禁用」标签
- **创建人**、**创建时间**、**随访日期**：基本信息
- **随访小结**：仅在任务状态为"已完成"且内容中包含问卷类型时显示
- **小结时间**：填写小结的时间

### 第五步：查看内容详情

#### 查看患教资料详情
点击患教资料类内容的蓝色标题，弹出患教详情弹窗，显示该患教资料的标题和正文内容。

#### 查看问卷详情
点击问卷类内容的蓝色标题，弹出问卷详情弹窗，显示该问卷的完整题目、选项和患者的作答情况。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未选择任何搜索条件 | 默认查询当月数据 |
| 没有匹配的随访任务 | 列表显示为空 |
| 创建人搜索无匹配 | 下拉选项为空 |
| 任务内容被禁用（发送前） | 显示红色"已禁用"标签 |
| 任务内容被禁用（已发送后） | 不显示禁用标签（内容已发出，禁用不影响已发送的任务） |
| 任务状态为待随访/已取消 | 不显示内容的完成/待办标签 |
| 任务不包含问卷类内容 | 不显示随访小结区域 |

## 关联功能

- 患教详情弹窗 → 查看患教资料完整内容
- 问卷详情弹窗 → 查看问卷题目和患者答案
- 随访患教课程管理 → 创建和编辑患教内容
- 随访问卷管理 → 创建和编辑问卷内容
- 随访提醒模板管理 → 管理提醒内容
