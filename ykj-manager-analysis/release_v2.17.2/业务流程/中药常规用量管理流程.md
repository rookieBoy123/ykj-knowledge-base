# 中药常规用量管理流程

> 文档版本 1.0 | 面向读者：运营管理员、药房管理员

## 概述
管理人员可以为中药设置"常规最大用量"（以克为单位），用于在医生开具处方时进行用量上限校验。当处方中某味中药的用量超过设置的常规最大用量时，系统会给出提示。

## 触发条件
- 运营管理员需要维护中药的安全用量标准
- 入口：管理后台 → 医院管理 → 药品信息 → 中药常规用量

## 步骤详解

### 第一步：查看用量列表
进入页面后看到中药常规用量列表，显示：
- 序号
- 药品通用名
- 常规最大用量（显示数值+单位"g"）
- 更新时间

顶部有搜索框，可按"药品名称"搜索特定的中药。

### 第二步：新增常规用量
点击"新增"按钮，弹出表单填写：
- **药品通用名**：必填，最多20个字符
- **常规最大用量(g)**：必填，使用数字输入框，精度为0.1g，步长为0.1

校验规则：
- 药品名称不能为纯空格
- 常规最大用量必须大于0

保存后返回列表，新增记录出现在列表末尾。

### 第三步：编辑用量配置
点击某条记录的"编辑"按钮，弹出编辑表单。药品通用名在编辑时不可修改（置灰），只能修改常规最大用量。

### 第四步：删除用量配置
点击"删除"按钮，系统弹出确认提示"是否确认删除？"，点击确定后删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药品名称为空格 | 提示"药品名称不能为空格"，不允许提交 |
| 常规最大用量 ≤ 0 | 提示"常规最大用量需大于0"，不允许提交 |
| 删除时未选择 | 无需选择，每条记录独立删除 |
| 搜索不到药品 | 列表显示空，无特殊提示 |

## 关联功能
- **处方开具**：医生在小程序开处方时，系统会根据此处设置的用量上限进行校验
- **药品管理**：此处配置的是"药品通用名"，与药品库中的通用名对应
