# 问诊诊室（医生工作站）工作流程

> 文档版本 2026-07-06 | 面向管理员/运营人员

## 概述

问诊诊室是管理后台为医生提供的在线接诊工作站。医生在此可以查看待处理的问诊单、与患者在线沟通、查看病历资料、书写病历和开具处方，是三栏式布局的综合工作台。

## 触发条件

- **前置条件**：使用医生角色账号登录管理后台
- **入口路径**：管理后台 → 诊疗管理 → 问诊诊室
- **角色限制**：仅医生角色可访问，非医生角色会看到"请登录医生角色查看!"的提示

## 页面布局

页面采用左中右三栏布局：

| 区域 | 内容 | 说明 |
|------|------|------|
| 左侧（360px） | 问诊单列表 | 分类展示待处理/进行中/已完成的问诊单 |
| 中间（弹性） | 聊天区 | 与患者的即时通讯（IM），支持文字聊天和视频通话 |
| 右侧（360px） | 问诊资料 | 问诊详情、病历记录、处方记录的折叠面板 |

## 步骤详解

### 第一步：进入问诊诊室

1. 系统检查当前登录角色是否为"医生"
2. **非医生角色**：页面显示"请登录医生角色查看!"提示，无法使用
3. **医生角色**：进入三栏工作台，默认未选中任何问诊单，中间区域提示"选择患者查看详细信息"

### 第二步：选择问诊单

1. 在左侧问诊单列表中切换选项卡（待接诊/进行中/已完成）
2. 点击某个问诊单，系统自动加载该患者的问诊详情
3. 中间聊天区显示与该患者的对话记录
4. 右侧面板显示问诊资料（患者信息、病历、处方等）

### 第三步：接诊

1. 在聊天区点击"接诊"按钮
2. 系统调用接诊接口，提示"接诊成功"
3. 问诊单自动从"待接诊"移动到"进行中"选项卡

### 第四步：问诊中操作

医生可以在问诊过程中进行以下操作：

- **文字聊天**：通过中间聊天区与患者沟通
- **视频通话**：点击视频按钮发起视频问诊，系统向患者发送视频邀请卡片（含"加入"按钮），同时获取TRTC房间参数建立视频连接
- **查看历史病历**：点击查看患者的历史就诊记录
- **书写病历**：点击"写病历"按钮，右侧面板切换为病历编辑界面，填写完成后提交保存
- **开具医嘱**：点击"开医嘱"按钮，系统先检查是否已有病历，未写病历时提示"请先写病历再开医嘱!"；已有病历时打开医嘱编辑器

### 第五步：结束问诊

1. 在聊天区点击"结束问诊"按钮
2. 系统弹出确认弹窗"确定结束问诊吗？"
3. 确认后调用结束接口，提示"问诊结束"
4. 问诊单移动到"已完成"选项卡

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 非医生角色登录 | 显示"请登录医生角色查看!"提示页 |
| 未选择任何问诊单 | 中间区域显示"选择患者查看详细信息"占位提示 |
| 点击开医嘱但未写病历 | 弹出提示"请先写病历再开医嘱!"，阻止操作 |
| 视频通话获取房间号失败 | 提示"响应错误，请联系管理人员" |
| 取消结束问诊 | 提示"操作取消"，不改变问诊状态 |

## 关联功能

- **问诊单列表**：医生问诊记录查看流程
- **病历管理**：病历创建与编辑流程
- **处方管理**：处方开具流程
- **视频通话**：基于TRTC的视频问诊功能
