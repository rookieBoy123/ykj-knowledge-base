# 视频内容管理流程

> 文档版本 2026-07-04 | 面向运营人员/内容管理员

## 概述
运营人员通过此功能管理小程序患者端展示的医疗科普视频。支持按分类组织视频，可控制每个视频的上架/下架状态、推荐标记，并查看播放次数。

## 触发条件
- 入口：运营管理 → 内容管理 → 视频管理
- 使用场景：上传新的医疗科普视频、管理视频的分类和展示状态、将热门视频推荐到患者端首页

## 页面布局
页面左右分栏：左侧为视频分类树，右侧为视频列表。

## 步骤详解

### 第一步：管理视频分类（左侧）
左侧分类树展示所有视频分类。每个分类支持以下操作：
- **新增分类**：点击底部"新增"按钮，输入分类名称后保存
- **编辑分类名称**：点击分类右侧的编辑图标，修改后保存
- **删除分类**：点击删除图标，系统会弹出二次确认（"删除后数据不可恢复，您确定要删除吗？"），确认后删除
- 删除分类后，如果当前正在查看该分类下的视频，会自动切换回"全部"

第一个分类固定为"全部"，表示不按分类过滤，显示所有视频。

### 第二步：查看视频列表（右侧）
选择左侧某个分类后，右侧显示该分类下的视频列表。每条视频展示以下信息：

| 信息 | 说明 |
|------|------|
| 编号 | 视频的序号 |
| 视频分类 | 视频所属的分类名称 |
| 视频名称 | 视频的标题 |
| 封面图片 | 视频的缩略封面图 |
| 关联医生 | 视频所关联的医生姓名 |
| 状态 | 上架/下架开关 |
| 播放次数 | 该视频在患者端的播放量 |
| 上传时间 | 视频上传的时间 |
| 是否推荐 | 是否推荐到首页 |
| 操作 | 编辑、删除 |

### 第三步：筛选视频
右侧顶部提供两个筛选条件：
- **视频名称**：输入视频名称进行模糊搜索，最多输入30个字符
- **状态**：下拉选择"上架"或"下架"进行过滤

### 第四步：新增视频
点击列表上方"新增"按钮，弹出视频新增窗口（包含分类选择、视频名称、封面图片上传、关联医生选择等字段），填写完成后保存。

### 第五步：编辑/删除视频
- **编辑**：点击操作列的"编辑"按钮，弹出编辑窗口，可修改视频信息
- **删除**：点击"删除"按钮，弹出二次确认（"删除后数据不可恢复，您确定要删除该视频信息吗？"），确认后删除

### 第六步：控制视频上架/下架
列表中"状态"列为开关控件：
- **开启（上架）**：患者在微信小程序中可以看到该视频
- **关闭（下架）**：系统确认提示"视频下架后患者端将不可见，确认要下架吗？"，确认后下架

### 第七步：设置推荐
列表中"是否推荐"列为开关控件：
- **开启（是）**：视频在患者端首页推荐位展示
- **关闭（否）**：系统确认提示"设置是否推荐为否后患者端首页将不显示该视频，确认要修改吗？"，确认后取消推荐

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 删除分类时有视频属于该分类 | 系统允许删除分类（不会拦截），分类删除后该分类下的视频仍存在，只是列表中没有该分类可选 |
| 搜索结果为空 | 列表显示空，可以调整搜索条件重新搜索 |
| 最后一页数据被全部删除 | 自动回退到上一页，避免显示空页 |
| 视频上架/下架或推荐/取消推荐 | 操作即时生效，用提示消息告知操作结果 |

## 关联功能
- **医生管理**：视频的"关联医生"字段需要从医生信息中选择
- **患者端视频展示**：上架的视频在微信小程序患者端展示，推荐视频在首页露出
- **分类管理**：视频分类通过数据字典维护
