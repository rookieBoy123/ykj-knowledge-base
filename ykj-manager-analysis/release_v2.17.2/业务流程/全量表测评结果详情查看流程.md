# 全量表测评结果详情查看流程

> 文档版本 2026-07-04 | 面向管理员

## 概述

管理员在后台查看患者某个量表测评的完整结果。页面为左右两栏布局：左侧显示测评结果概览（含雷达图）和题目明细，右侧为锚点导航菜单，支持点击快速定位到对应区域。如果测评关联了问诊记录，还可以查看问诊对话记录。

## 触发条件

- 从患者详情页或测评列表页点击某个测评记录进入
- 页面通过路由参数 `id` 传入测评记录的唯一标识
- 入口：后台 → 患者管理 → 患者详情 → 点击某次量表测评

## 步骤详解

### 第一步：查看测评结果概览

页面加载后自动获取测评详情数据。左侧顶部"测评结果"卡片展示：

- **测评量表**：量表名称（如"SDS抑郁自评量表"）
- **测评总分**：以"XX分"形式展示
- **测评结果**：结果文字描述
- **结果说明**：详细解释说明
- **雷达图**：当测评的维度分数有3个及以上维度时，自动展示雷达图可视化
- **测评人**：患者姓名、性别、年龄
- **测评时间**：测评完成时间

如果该测评关联了问诊记录（有问诊ID），还会额外显示：
- **问诊医生**：医生姓名、科室、职称、医院
- **问诊时间**：问诊发生时间
- 右上角出现「问诊对话记录」按钮，点击可展开对话记录组件

### 第二步：查看题目明细

左侧"题目明细"卡片逐题展示测评的所有题目和患者的作答：

- 题目序号和题目名称
- **单选题（type=0）**：以只读单选按钮组展示，选中患者所选选项
- **多选题（type=1）**：以只读多选框组展示，选中患者所选的所有选项

所有选项均为禁用状态（只读），不可修改。

### 第三步：使用右侧导航

右侧固定导航菜单包含两个锚点：
- 「测评结果」：点击滚动到测评结果区域
- 「题目明细」：点击滚动到题目明细区域

滚动动画为平滑过渡，当前所在区域的导航项高亮显示。手动滚动页面时，导航高亮也会自动跟随变化。

### 第四步：查看问诊对话记录

如果测评结果卡片右上角有「问诊对话记录」按钮，点击后页面下方会展开问诊对话记录组件，显示该次问诊中的医患聊天记录。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 测评没有多维度的分数数据 | 不显示雷达图（dimensionScores的键少于3个） |
| 测评未关联问诊（无问诊ID） | 不显示问诊医生/时间信息，也没有"问诊对话记录"按钮 |
| 题目选项数据为JSON格式 | 前端自动解析JSON并渲染选项列表 |
| 页面刷新后锚点定位不准 | 系统检测到手动刷新后，使用不同的锚点偏移计算方式 |

## 关联功能

- **患者详情页**：从患者详情进入此页
- **问诊对话记录**：可展开查看关联的医患对话
