# 机构管理流程

> 文档版本：v1.0 | 面向读者：平台管理员、技术管理员

## 概述

机构管理用于管理接入平台的第三方合作机构。每个机构拥有独立的机构ID（AppID）、机构密钥和应用密钥。机构可以被启用或禁用，用于控制第三方对接权限。

## 触发条件

- **前置条件**：已登录管理后台，具备系统管理权限
- **入口路径**：左侧菜单 → 运营管理 → 机构管理

## 步骤详解

### 第一步：查看机构列表

进入页面后看到机构列表表格：

| 列名 | 说明 |
|------|------|
| 机构ID | 系统自动生成的唯一标识（AppID） |
| 机构名称 | - |
| 机构密匙 | 用于API鉴权，右键可自动复制 |
| 回调秘钥 | 用于回调数据签名，右键可自动复制 |
| 创建时间 | - |
| 机构状态 | 启用/禁用开关 |
| 操作 | 编辑 |

### 第二步：右键复制密钥

表格中的"机构密匙"和"回调秘钥"列支持右键自动复制功能——在对应单元格上右键点击，系统自动将值复制到剪贴板并提示"机构密匙复制成功"或"回调秘钥复制成功"。

### 第三步：新增机构

点击"新增机构"按钮，弹出表单弹窗：

| 字段 | 说明 |
|------|------|
| 机构ID | 系统自动生成，不可修改 |
| 机构名称 | 必填，最多30字 |
| 机构密钥 | 系统自动生成UUID格式的密钥，不可修改，但支持"重置"生成新密钥 |
| 回调密钥 | 系统自动生成，不可修改，支持"重置" |

新增时默认设置：
- 审方类型为0
- 机构状态为启用
- CA认证关闭
- 无需药品订单
- 未选择药师
- 回调地址和业务模块为空

### 第四步：编辑机构

点击"编辑"按钮，系统先通过接口获取机构完整信息（包括已选择的药师列表、业务模块等），然后将数据回填到表单中。

### 第五步：启用/禁用机构

在列表中直接点击状态开关，切换机构的启用/禁用状态。切换前会弹出确认弹窗"是否确认禁用？"或"是否确认启用？"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 新增/编辑时未填写机构名称 | 表单校验拦截，提示"请输入机构名称" |
| 选择审方设置但未选择药师 | 点击确定时提示"请选择药师" |
| 启用/禁用操作 | 弹出二次确认弹窗 |
| 密钥泄露需要更换 | 支持对密钥和回调密钥分别"重置"生成新值 |

## 关联功能

- **API对接**：第三方机构使用机构ID+密钥接入平台API
- **药师管理**：机构可以关联指定的药师进行审方
- **医生管理**：开启了机构功能的系统，医生列表中会显示"所属机构"列和筛选项
