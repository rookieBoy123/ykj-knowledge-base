# 医保药品标准目录管理流程

> 文档版本 2026-07-04 | 面向运营/管理员

## 概述
医保药品标准目录是来自医保基础数据映射的药品标准库。管理员在这里管理"成药（西药/中成药）"和"中药饮片"两类药品的标准目录数据。这些数据用于医保结算时判断药品的收费等级（甲类/乙类/丙类）和医保支付比例，是整个医保结算流程的基础数据。

## 触发条件
- 菜单路径：系统管理 → 医保管理 → 标准目录
- 需要权限：查看需 `med:medicare-medicine:read`，导入需 `med:medicare-medicine:import`，导出需 `med:medicare-medicine:export`
- 使用场景：每年医保政策调整后需要更新药品目录，或新增/下架药品时批量导入

## 页面布局
页面顶部有两个标签页切换："成药类型"和"中药饮片"。两个标签页的数据和列表字段完全不同，切换标签会重置搜索条件和列表。

## 步骤详解

### 第一步：选择药品类型
1. 页面默认展示"成药类型"标签（西药/中成药）
2. 点击"中药饮片"切换到中药类型
3. 切换类型后：搜索条件、列表字段、下载的导入模板都会相应变化

### 第二步：搜索筛选
**成药类型**支持以下搜索条件：
- 收费等级：下拉选择"甲类/乙类/丙类"（选项从系统数据字典 `ybypsfxmdj` 加载）
- 注册名称：输入药品的商品名
- 通用名称：输入药品通用名
- 药品代码：输入医保系统分配的药品代码
- 药品本位码：输入国家药品本位码

**中药饮片类型**支持以下搜索条件：
- 收费等级：同上
- 中药名称：输入中药饮片名称
- 药材名称：输入药材名
- 中药代码：输入医保系统分配的中药代码

### 第三步：浏览列表
**成药类型列表**字段包括：药品代码、注册名称、商品名称、注册剂型、实际剂型、注册规格、实际规格、包装材质、最小包装数量、最小包装单位、最小剂量单位、药品企业、批准文号、药品本位码、分包装企业、上市状态、通用名称、收费项目等级、城职医保先支付比例、城乡医保先支付比例、操作时间。

**中药饮片列表**字段包括：中药饮片名称、中药饮片代码、药材名称、炮制方法、功效分类、科（族）、药材种来源、药用部位、性味与归经、功能与主治、用法与用量、医保支付政策、标准名称、标准页码、收费项目等级、城职医保先支付比例、城乡医保先支付比例、操作时间。

收费项目等级显示为"甲类/乙类/丙类"中文标签，由系统自动转译。

### 第四步：下载模板
1. 点击列表上方"下载模板"按钮
2. 根据当前标签页类型，下载对应的 Excel 模板：
   - 成药类型：`医保西药导入模板.xlsx`
   - 中药饮片：`医保中药导入模板.xlsx`
3. 模板存储在华为云 OBS 上，点击后浏览器直接下载

### 第五步：导入数据
1. 准备好按模板填写的 Excel 文件
2. 点击"导入"按钮，选择文件上传
3. 上传成功后，页面显示"正在导入数据"的加载遮罩
4. 系统每 5 秒轮询一次导入进度，直到导入完成
5. 导入成功后自动刷新列表；导入失败则弹出错误提示（显示失败原因）

### 第六步：导出数据
1. 在当前搜索条件下，点击"导出"按钮
2. 页面显示"加载中"提示
3. 导出的是当前搜索条件筛选后的全部数据（不受分页限制）

### 第七步：删除单条药品
1. 点击某条药品的操作列"删除"按钮
2. 弹出确认框："此操作将永久删除该项，是否继续？"
3. 确认后删除，列表自动刷新

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 导入文件格式不对 | 显示具体错误信息（resultMsg），关闭加载遮罩 |
| 导入过程中用户切换了标签 | 不影响当前导入任务（importVersion 保存在 sessionStorage） |
| 页面刷新后导入未完成 | 重新进入页面时自动检测是否有未完成的导入任务，继续轮询进度 |
| 搜索条件切换药品类型 | 搜索条件列表动态变化，匹配当前类型 |
| 删除时取消操作 | 提示"已取消删除"，不执行操作 |

## 关联功能
- **医保结算数据上传**：药品标准目录中的收费等级和医保支付比例直接影响结算金额
- **医保慢性病种管理**：慢性病用药范围参考此目录
- **数据字典**：收费等级选项从字典分类 `ybypsfxmdj` 加载
- **HIS 药品管理**：医院药品需要与标准目录做对照映射
