# 问诊服务订单管理流程

> 文档版本：2026-07-04 | 面向读者：运营人员、客服人员

## 概述

管理员在此页面查看和管理所有门诊服务订单的详细信息，包括问诊状态、对话记录、视频回放、知情同意书确认信息等，并可以对进行中的问诊进行"结束问诊"操作。

## 触发条件

- 前置条件：具有问诊管理权限的管理员账号登录
- 入口路径：「诊疗管理 → 门诊服务订单 → 订单列表」

## 筛选条件

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 问诊编号 | 文本输入 | 模糊搜索，最长30字符 |
| 服务类型 | 多选下拉 | 图文问诊/视频问诊/咨询服务/快速问诊/快速续方/医生复诊/商城问诊/AI图文问诊（AI需开关开启） |
| 问诊流程 | 多选下拉 | 在线问诊/线上复诊/慢病医保/预约挂号/快速开方/AI问诊 |
| 问诊状态 | 下拉选择 | 已预约/问诊中/问诊结束/取消问诊 |
| 支付时间 | 日期范围 | 时间范围筛选 |
| 所属机构 | 文本输入 | 仅机构管理功能开启时显示 |
| 问诊时间 | 日期范围 | 问诊发生的时间范围 |
| 就诊时间 | 日期范围 | 实际就诊的时间范围 |
| 有无病历 | 下拉选择 | 有/无，判断该问诊是否包含病历记录 |
| 医生姓名 | 文本输入 | 最多5字符 |
| 患者姓名 | 文本输入 | 最多5字符 |
| 所属团队 | 下拉选择 | 仅专家团队功能开启时显示 |

## 操作功能

### 查看详情
点击"查看详情"按钮，弹出对话框显示问诊完整记录：

**问诊对话记录**（图文/语音消息）：
- 显示医生和患者的完整对话历史
- 支持语音消息播放
- 配有水印（用户名@系统名称）防截图泄露
- 支持滚动加载更多历史消息
- 群组问诊（IM模式）显示多方消息，包含群组成员头像和名称

**问诊视频记录**：
- 仅视频问诊类型（serviceType=2）显示"视频记录"按钮
- 点击后展示该问诊的所有视频片段列表，每段显示创建时间和"播放"按钮
- 点击播放在内置播放器中观看

**AI问诊对话记录**（AI图文问诊特有）：
- 显示AI助手与患者的对话
- AI消息含普通文本消息和结构化卡片消息（包含标题、字段、按钮）
- 患者消息含文本和图片附件
- 点击患者图片可全屏预览
- AI卡片中的按钮可触发查看"AI预问诊对话记录"或"AI问诊报告"

**AI问诊报告**：
包含以下结构化分节：
- 主诉
- 病史信息（现病史、既往史、过敏史、家族史、个人史、月经史、婚育史）
- 体检（辅助检查）
- AI诊断（含标签列表和"仅供参考，具体以真实医生诊断为准"提示）

### 结束问诊
仅对"问诊中"状态（flowStatus=20）的问诊显示。点击后弹出二次确认对话框，确认后调用服务结束该问诊。

### 知情同意书确认信息
当问诊包含知情同意书（inquiryIcfId 存在时）显示。点击弹出对话框展示：
- 确认人姓名
- 网络IP
- 设备信息
- 微信用户openId
- 同意书签署时间

## 状态说明

| 状态值 | 含义 |
|--------|------|
| 已预约 | flowStatus=20，未签到（isCheckIn=0） |
| 问诊中 | flowStatus=20，已签到（isCheckIn=1） |
| 问诊结束 | flowStatus=30 |
| 取消问诊 | flowStatus=40 |

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 不满足结束条件（非20状态） | "结束问诊"按钮置灰不可点击 |
| 非视频问诊（serviceType≠2） | "视频记录"按钮置灰不可点击 |
| 无知情同意书（inquiryIcfId为空） | "知情同意书确认信息"按钮置灰不可点击 |
| 聊天记录不存在 | 弹出错误提示"没有找到相关问诊对话记录!" |
| 电话号码脱敏查看 | 需验证操作者身份后返回明文，同一时刻仅显示一条明文 |

## 关联功能

- 跳转到问诊单查询页面（含订单号和支付信息）
- 关联服务项目配置和医生信息
