# 会员权益使用与管理流程

> 文档版本：2026-07-05 | 面向读者：运营管理员、客服

## 概述

查看和管理会员持有的健康服务包权益，包括查看每个会员购买的服务包具体权益（有效期、服务条件、使用次数），并支持手动扣减服务次数。用于处理会员的服务权益核对和线下服务核销场景。

## 触发条件

- 前置条件：拥有服务包权益管理权限（`inq:service-package:benefit`）
- 入口路径：管理后台 → 运营管理 → 健康服务包 → 会员权益管理

## 步骤详解

### 第一步：筛选权益记录

页面顶部有两个标签页：
- **生效中**：默认选中，查看当前有效的服务权益
- **已失效**：查看已过期的服务权益

同时支持按"会员电话"（11位手机号）和"服务项目"名称模糊搜索。

### 第二步：查看权益列表

列表展示以下信息（以生效中标签为例）：

- **会员编号**：系统生成的唯一会员标识
- **会员昵称**
- **联系电话**：默认脱敏显示（中间4位星号遮盖）。点击"查看"按钮后调用脱敏接口获取完整手机号，同一时间只有一行显示完整号码（点击另一行的查看会隐藏上一行的完整号码）
- **服务项目**：服务包的具体权益名称（如"图文问诊"、"视频问诊"）
- **服务有效期**：起止日期格式化显示（如"2026年07月01日-2027年07月01日"）
- **服务条件**：适用条件说明
- **服务次数**：不限次数显示"不限次数"，限制次数显示"X次/年"
- **已使用次数**：不限次数显示"-"，其他显示具体数字
- **剩余服务次数**：不限次数显示"-"
- **备注**
- **操作**："扣减一次"按钮（仅对限制次数且仍有剩余次数的权益可用）

### 第三步：扣减服务次数

当会员在实体机构使用权益（如线下就诊）需要核销次数时：

1. 点击对应权益行的"扣减一次"按钮
2. 弹出确认框："将扣减服务次数1次，该操作不可逆，确认扣减吗？"
3. 确认后调用接口扣减，成功提示"操作成功"
4. 列表自动刷新，剩余次数-1

**按钮禁用条件**：
- 不限次数（times=-1）的权益不可扣减
- 剩余次数已用完（remainTimes≤0）的权益不可扣减

### 第四步：查看已失效权益

切换到"已失效"标签页，查看已过期或已用完的权益记录。此时"扣减一次"操作列会被隐藏（已失效权益不能扣减）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 已失效权益尝试扣减 | 操作列直接隐藏，无法操作 |
| 不限次数权益尝试扣减 | 扣减按钮置灰不可点击 |
| 剩余次数为0的权益尝试扣减 | 扣减按钮置灰不可点击 |
| 输入非法手机号码 | 输入框限制最长11位数字，非11位无效 |
| 会员无任何权益 | 列表为空，按条件搜索无结果 |

## 关联功能

- **健康服务包订单管理**：会员购买服务包产生订单，订单完成后生成权益
- **会员信息查询**：可以交叉查询会员的基本信息和购买记录
- **服务包项目管理**：配置服务包包含哪些服务项目/权益
