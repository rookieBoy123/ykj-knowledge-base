# 会诊服务管理流程

> 文档版本 2026-07-04 | 面向远程医疗管理员

## 概述

远程医疗管理员在受邀机构下配置该机构提供的远程会诊服务项目，包括服务名称、类型、是否点名、职称等级要求、单价和单位等信息。

## 触发条件

- **前置条件**：当前登录账号具有"远程会诊管理员"角色；已从受邀机构列表页进入
- **入口路径**：诊疗管理 → 远程医疗中心 → 受邀机构管理 → 点击某个机构的"管理服务"

## 步骤详解

### 页面加载

进入页面后，顶部显示当前机构名称，下方展示该机构下所有会诊服务的列表。列表包含：服务编号、服务名称、服务类型、是否点名、职称等级要求、单价（元）、单位、状态和操作按钮。

页面必须携带机构 ID 参数，缺少时会提示"缺少机构id"。

### 新增服务

1. 点击列表上方的「新增」按钮，弹出新增表单窗口
2. 填写以下信息：
   - **服务名称**：最多 200 个字符
   - **服务类型**：从数据字典下拉列表中选择（由 `jgfwlx` 类别提供）
   - **是否点名**：三个选项 —"是""否""不涉及"。默认为"不涉及"
   - **职称等级要求**：三个选项 —"正高""副高""不涉及"。默认为"不涉及"
   - **单价（元）**：数字输入，服务的价格
   - **单位**：从数据字典下拉列表中选择（由 `jgfwlxdw` 类别提供）
   - **状态**：默认"启用"，可选"禁用"
3. 所有字段均为必填，点击「确定」提交
4. 新增成功后列表自动刷新并提示"新增成功"

### 编辑服务

1. 点击目标服务的「编辑」按钮，弹出编辑窗口
2. 表单自动填入现有信息，修改后点击「确定」
3. 编辑成功后窗口关闭，列表自动刷新

### 删除服务

1. 点击目标服务的「删除」按钮
2. 弹出确认提示："删除后数据不可恢复，确认删除服务【服务名称】吗？"
3. 确认后执行删除，列表刷新

## "是否点名"说明

"是否点名"决定了申请会诊时是否需要指定具体医生：
- **是**：申请会诊时必须指定一位医生
- **否**：申请会诊时不需要指定医生，由机构自行分配
- **不涉及**：该服务不需要考虑点名机制

## "职称等级要求"说明

"职称等级要求"用于限定执行该服务的医生最低职称：
- **正高**：只有正高级职称的医生可执行
- **副高**：副高及以上职称的医生可执行
- **不涉及**：对职称等级没有限制

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 页面未携带机构ID | 提示"缺少机构id"，页面空白 |
| 未填写必填字段 | 表单校验提示具体缺失项，阻止提交 |
| 服务存在申请记录时删除 | 服务端拦截，提示"服务【名称】存在申请记录，不可删除！" |
| 删除时二次确认取消 | 不执行删除，停留在当前页面 |

## 关联功能

- **受邀机构管理**：入口来源
- **会诊医生管理**：同级别的机构子功能
- **会诊申请**：服务删除的约束来源（已有关联申请的不可删）
