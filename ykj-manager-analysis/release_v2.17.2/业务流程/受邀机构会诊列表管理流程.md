# 受邀机构会诊列表管理流程

> 文档版本 2026-07-04 | 面向受邀机构管理员

## 概述

被邀请参与远程会诊的机构管理员，在后台查看和管理所有发给本机构的会诊申请。列表按会诊流程状态分期管理，支持筛选查询、应答邀请、进入会诊室、补充会诊资料和查看详情等操作。页面有严格角色限制，仅受邀机构管理员可访问。

## 触发条件

- 受邀机构管理员需要查看和管理本机构收到的会诊申请
- 入口：后台 → 诊疗管理 → 远程医疗中心 → 会诊列表

## 步骤详解

### 第一步：角色校验

页面加载时首先检查当前登录用户的角色。如果角色代码不是 `invitedOrgManager`，页面显示提示："当前页面仅供受邀机构管理员使用！"，其余内容不显示。校验通过后才加载会诊列表。

### 第二步：筛选会诊申请

搜索栏支持以下筛选条件：

- **会诊单号**（输入框，最多20字）
- **申请科室**（输入框，最多20字）
- **申请医师**（输入框，最多5字）
- **患者姓名**（输入框，最多5字）
- **状态**（下拉选择）：全部、待应答、已拒绝、待收费、待开始、进行中、已结束、已取消、已归档

筛选后列表刷新。

### 第三步：查看会诊列表

列表展示会诊关键信息：

- 会诊单号
- 申请科室
- 主治医师
- 患者姓名、性别、年龄
- 拟申请会诊机构
- 拟申请会诊服务
- 拟申请会诊科室
- 拟申请会诊医生
- 拟申请会诊时间
- 当前状态（如"待应答""进行中"等中文显示）
- 会诊时间

### 第四步：处理会诊申请

每行操作按钮根据会诊状态动态显示：

| 状态 | 操作按钮 | 说明 |
|------|---------|------|
| 待应答（flowStatus=2） | 「应答」 | 打开会诊邀请应答弹窗，决定接受或拒绝 |
| 进行中（flowStatus=5） | 「进入会诊」 | 跳转到远程会诊室页面，参与在线会诊 |
| 已结束（flowStatus=6） | 「补充资料」 | 打开补充资料弹窗，上传会诊补充文件 |
| 所有状态 | 「查看详情」 | 打开详情弹窗，查看会诊完整信息 |

### 第五步：应答会诊邀请

点击「应答」，弹出应答弹窗（由子组件 invitationResp 实现），受邀机构可以对会诊邀请做出应答（接受或拒绝）。

### 第六步：补充会诊资料

会诊结束后，受邀机构可上传补充资料：

- 点击「补充资料」打开弹窗
- 如已有上传过的资料，自动加载显示
- 点击「上传」选择文件：
  - 支持格式：PDF、JPG、PNG、JPEG
  - 单个文件不超过 10MB
  - 提示："将纸质文档扫描成图片或pdf上传"
- 上传后文件出现在列表中
- 点击「保存」提交所有文件

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 非受邀机构管理员角色访问 | 页面显示"当前页面仅供受邀机构管理员使用！"，无其他内容 |
| 补充资料未选择文件就保存 | 提示"请选择文件后再保存" |
| 列表翻页到最后一页数据为空时 | 自动回退到上一页 |

## 关联功能

- **远程会诊室**：进行中的会诊通过"进入会诊"跳转到会诊室
- **会诊详情**：查看会诊完整信息
- **会诊邀请应答**：对待应答会诊做出应答
