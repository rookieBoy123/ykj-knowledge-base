# 服务项目管理

> 文档版本：v1.0 | 最后更新：2026-07-02 | 面向读者：运营人员

## 概述

服务项目管理用于创建和配置医院提供的门诊服务项目。管理人员可以定义服务的名称、类型、适用人员、服务方式（自行到院/上门服务）、排班时间、服务内容和须知，并在小程序预约页面向患者展示。

## 触发条件

- **前置条件**：拥有管理后台登录权限
- **入口路径**：管理后台 → 诊疗管理 → 服务项目管理 → 项目列表 → 新增/编辑

## 步骤详解

### 第一步：基本信息

| 字段 | 说明 | 限制 |
|------|------|------|
| 服务名称 | 服务项目的名称 | 2-20字；编辑时不可修改 |
| 服务类型 | 下拉选择服务分类 | 编辑时不可修改 |
| 支持人员类型 | 勾选个人/团体企业 | 默认勾选个人 |
| 图片 | 服务展示图片 | 建议750×750比例1:1，显示在小程序预约页 |

### 第二步：人员类型与价格

| 人员类型 | 配置 |
|----------|------|
| 个人 | 价格（元），精确到分 |
| 团体/企业 | 定金（元），精确到分 |

勾选团体或企业需填写定金。勾选后该服务显示在团体/企业预约服务中。

### 第三步：项目明细

如开启自动开票功能，可设置服务包含的项目清单：
- 选择收费项目（下拉选择）
- 设置数量
- 自动计算金额
- 支持新增/删除费用明细行
- 开单科室：树形下拉选择科室

### 第四步：服务方式

支持多选以下服务方式：

| 服务方式 | 配置项 |
|----------|--------|
| 自行到院 | 医院名称（最多256字）、地址（最多256字，支持地图搜索）、经纬度 |
| 上门服务 | 上门服务费（元，精确到分） |

选择单一服务方式则患者端默认选中该方式；选择上门服务时，费用将统计于总费用中。

地址输入框旁有"搜索"按钮，可打开地图定位获取经纬度。

### 第五步：服务排班

勾选可预约的星期（周一至周日），该排班按照每月对应日期显示，默认显示近2个月的排班。

### 第六步：开放时间段

可设置两种时间段：

**工作时间段**（勾选后启用）：
- 支持添加多个时段（如08:00-12:00, 14:00-18:00）
- 时间选择精确到5分钟（06:00-23:59）
- 每个时段可独立删除
- 可动态添加新时段

**非工作时间段**（勾选后启用）：
- 设置一个时间段（起始时间-结束时间）
- 可设置非工作时间段费用（选填），费用自动统计在最终费用中

### 第七步：状态设置

- **启用**：服务在小程序端显示，患者可预约
- **禁用**：服务不在小程序显示，可后续重新编辑启用

### 第八步：富文本内容

- **包含项目**：使用富文本编辑器编写服务包含的具体项目内容
- **服务须知**：使用富文本编辑器编写服务须知信息

### 第九步：客服电话

填写服务相关的客服联系电话（最多16位）。

### 第十步：提交

- 点击"确认提交"保存服务配置
- 点击"取消"返回上一页

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 服务名称为空或不符合长度限制 | 表单校验失败，无法提交 |
| 未选择服务类型 | 表单校验失败 |
| 未设置任何服务方式 | 表单校验失败 |
| 未上传图片 | 表单校验失败 |
| 未勾选服务排班 | 表单校验失败 |
| 工作时间段起止时间冲突 | 系统自动限制结束时间必须大于起始时间 |
| 多个时段时间重叠 | 通过系统的时间范围逻辑避免，起始时间最小值受上一时段结束时间限制 |

## 关联功能

- **项目列表**：查看和管理所有服务项目
- **项目详情**：查看服务项目的完整配置
- **医生管理**：医生开启服务时引用服务项目配置
- **小程序预约页**：患者端展示服务项目信息
