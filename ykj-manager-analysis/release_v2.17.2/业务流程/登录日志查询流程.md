# 登录日志查询流程

> 文档版本 2026-07-03 | 面向系统管理员/安全审计

## 概述
登录日志分为两大类：**管理员登录日志**（后台管理员登录管理后台的记录）和**小程序登录日志**（患者在微信小程序登录的记录）。两个页面结构几乎相同，区别在于小程序登录日志固定过滤 `sysClient='app'`。日志记录了每次登录的账号、IP、操作系统、浏览器、登录方式、成功/失败状态和登录时间。

## 触发条件
- 后台管理员登录日志：系统管理 → 系统管理 → 管理员登录日志
- 小程序登录日志：系统管理 → 系统管理 → 小程序登录日志
- 使用场景：安全审计、排查异常登录、分析用户活跃情况

## 页面功能

### 筛选条件（两个页面完全相同）
| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 用户搜索 | 文本输入 | 输入登录账号进行精确匹配，最长30字符 |
| 登录状态 | 下拉选择 | 全部 / 正常（result=true）/ 失败（result=false） |
| 时间选择 | 日期范围 | 按登录时间范围筛选 |

### 列表字段
| 字段 | 说明 |
|------|------|
| ID | 日志记录序号 |
| 用户账号 | 登录的账号名 |
| 登录IP | 登录时的客户端IP地址 |
| 操作系统 | 如Windows、macOS、iOS、Android |
| 浏览器 | 如Chrome、Safari、微信内置浏览器 |
| 登录方式 | 如"账号密码""手机验证码""微信授权"等 |
| 登录状态 | 正常（蓝色标签）或异常（红色标签） |
| 操作结果 | 成功（绿色）或失败（红色），与登录状态同步显示 |
| 登录时间 | 登录发生的具体时间 |

## 两个页面的区别
| 对比项 | 管理员登录日志 | 小程序登录日志 |
|--------|---------------|---------------|
| 菜单路径 | 管理员登录日志 | 小程序登录日志 |
| 后台过滤 | 无额外过滤 | 固定传 `sysClient='app'` |
| 数据来源 | 同一张日志表 | 同一张日志表（加sysClient过滤） |

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 登录失败 | 登录状态显示红色"异常"，操作结果显示红色"失败" |
| 查询无结果 | 列表显示空 |
| 用户搜索无匹配 | 输入不存在的账号后无数据返回 |

## 关联功能
- **后台操作日志**：记录管理员登录后的操作行为，而登录日志只记录登录行为本身
- **用户管理**：管理员的账号在用户管理页面中维护
- 服务端：`MgmtSystemLoginLogController`
