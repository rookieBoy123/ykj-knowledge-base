# 药品库信息管理流程

> 文档版本 2026-07-05 | 面向系统管理员、药房管理人员

## 概述

药品库信息管理是药房管理模块中用于新增和查看药品信息的功能。系统支持两种药品类型——成药（西药/中成药）和中药，两种类型的填写字段和校验规则不同。管理员可以新增药品或查看已有药品的详细信息。

## 触发条件

- **前置条件**：拥有药房管理后台的访问权限
- **入口路径**：药房管理 → 药品管理 → 药品库 → 新增药品 / 查看药品

## 步骤详解

### 第一步：选择药品类型

进入页面后，首先需要选择药品类型：

- **成药类型**：适用于西药、中成药等常规药品
- **中药类型**：适用于中药饮片等

页面顶部有红色提示文字："按规定：互联网医院不得在网上开具麻醉、精神类药品处方，严禁违规上传！"

### 第二步：填写基本信息（成药类型）

选择"成药类型"后，基本信息区包含以下字段：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 药品监管ID | 国家药品监管唯一标识，最多255字符 | 是（新增时） |
| 通用名 | 药品的通用名称，最多255字符 | 是 |
| 商品名称 | 药品的商品名，最多255字符 | 是 |
| 药品剂型 | 从剂型字典中选择（如片剂、胶囊等） | 是 |
| 药品类型 | OTC（非处方药）或RX（处方药） | 是 |
| 厂家或产地 | 支持输入或从已有厂家中选择 | 是 |
| 抗菌药物 | 仅上海适应地区显示，选择"是"或"否" | 上海地区必填 |

### 第三步：填写规格信息（仅成药类型）

规格信息区包含：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 剂量 | 数字格式，如100、0.5 | 否 |
| 剂量单位 | 从剂量单位字典中选择（如mg、g等） | 否 |
| 制剂数量 | 一盒包含的片/粒/支数量，数字格式 | 是 |
| 制剂单位（用量单位） | 从成药用量单位字典中选择（如片、粒、支等） | 是 |
| 包装单位 | 从包装单位字典中选择（如盒、瓶等） | 是 |
| 药品规格 | 可手动输入或勾选"自动生成"由系统拼接 | 否 |

**自动生成规格**：当勾选"自动生成"复选框时，系统会根据剂量+剂量单位、制剂数量+制剂单位、包装单位自动拼接成规格字符串。格式为：`（剂量）（剂量单位）*（制剂数量）（制剂单位）/（包装单位）`。

> **注意**：自动生成仅在同时填写了制剂数量、制剂单位和包装单位时才可用。缺少任一字段时，复选框为灰色不可选。

### 第四步：填写医保支付信息（仅成药类型）

**药品医保编码**：通过远程搜索查询国家医保药品目录：

- 输入药品注册名称进行搜索
- 搜索结果展示：药品名称、规格、厂家、医保编码
- 选中后自动填充医保编码
- 如需使用医保支付，此为必填项

> **注意**：上海适应地区下，医保编码和抗菌药物字段均为必填。

### 第五步：填写基本信息（中药类型）

选择"中药类型"后，字段有所不同：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 通用名 | 药品通用名称，最多255字符 | 是 |
| 商品名称 | 药品商品名，最多255字符 | 是 |
| 拼音编码 | 输入通用名后自动通过拼音接口生成 | 否（自动生成） |
| 药品剂型 | 从剂型字典中选择 | 是 |
| 计量计价单位 | 医生开方时使用的单位（如克、枚等） | 是 |
| 库存单位 | 药房库存管理使用的单位 | 是 |
| 规格转换比 | 库存单位与计量计价单位的转换比例 | 二者不同时必填 |
| 医保代码 | 国家医保药品代码，最多100字符 | 否 |

> **规格转换比说明**：如果库存单位是"公斤"，计量计价单位是"克"，则转换比为1000（1公斤=1000克）。医生开方时按计量计价单位开具，而库存计算按库存单位进行，配置错误将导致库存计算错误。

### 第六步：填写药品说明书

两种药品类型都需要填写药品说明书，使用富文本编辑器：

- 支持图片、文字排版
- 查看模式下以HTML渲染展示

### 第七步：保存药品

点击"保存"按钮提交：

- **新增成药**：调用成药新增接口，提交前会校验药品监管ID是否重复、药品通用名是否为精麻类药品
- **新增中药**：调用中药新增接口，计量计价单位默认为"克"
- **编辑**：调用药品编辑接口更新

保存成功后返回上一页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 通用名为精麻类药品 | 提示"（药品名）为精麻类药品无法添加"，无法保存 |
| 药品监管ID重复 | 提示"（监管ID）为重复药品监管id不能添加" |
| 规格自动生成不可用 | 复选框显示为灰色，需先填写制剂数量、制剂单位和包装单位 |
| 切换药品类型 | 清空已填写的所有表单数据，需重新填写 |
| 上海地区抗菌药物未选 | 提交时校验提示"请选择抗菌药物" |
| 上海地区医保编码未填 | 提交时校验提示"请输入医保编码" |
| 中药库存单位与计量计价单位不同但未填转换比 | 提交时校验提示"请输入转换比例" |
| 查看模式 | 所有字段变为只读，底部按钮改为"返回" |

## 关联功能

- **药品管理列表**：药品库的主列表页，可对已有药品进行编辑
- **药品黑名单**：通用名校验时会调用黑名单接口检查是否为精麻类药品
- **厂家管理**：厂家或产地字段从已有的药厂列表中筛选
- **医保药品目录**：医保编码通过远程查询国家医保标准目录
