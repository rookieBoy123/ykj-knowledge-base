# 评价审核管理流程

> 文档版本 2026-07-05 | 面向运营人员

## 概述

评价审核管理用于管理患者对问诊服务的评价内容。所有评价提交后默认进入"待审核"状态，运营人员可以审核通过、查看详情、控制评价是否在前端显示、或删除不当评价。

## 触发条件

- **入口路径**：管理后台 → 门诊服务 → 评价列表
- **使用角色**：拥有评价管理权限的运营人员
- **数据来源**：患者在小程序端完成问诊后提交的评价

## 步骤详解

### 第一步：查看评价列表

页面默认按分页展示所有评价记录（每页 10 条）。每条记录显示：
- 会员名称
- 问诊类型（图文问诊/视频问诊等）
- 接诊医生
- 评分（满意度分数）
- 评价内容
- 评价时间
- 显示状态（显示/隐藏开关）
- 审核状态（待审核/审核通过/审核不通过）

### 第二步：筛选评价

可通过以下条件筛选：
- **关键字**：搜索评价内容中的关键词
- **评价时间**：时间范围筛选
- **状态**：按审核状态筛选（待审核/审核通过/审核不通过）

### 第三步：审核评价

**快速通过**：
- 待审核的评价在操作栏显示"通过"按钮
- 点击"通过"直接将评价状态改为"审核通过"，同时自动将"显示"开关打开

**查看详情后通过**：
- 点击"详情"打开评价详情弹窗，可查看完整信息：会员名称、问诊类型、接诊医生、评价日期、分数、所选标签、评价内容
- 在详情弹窗中可手动调整"是否显示"开关，点击"通过"按钮提交

**已审核的评价**（审核通过/审核不通过）点击详情弹窗后，按钮文字变为"确定"，仅用于修改显示开关，不再改变审核状态。

### 第四步：控制评价显示/隐藏

列表中的每条评价都有一个"显示/隐藏"开关：
- 开启（显示）：评价会在小程序端的医生主页或服务评价区域展示
- 关闭（隐藏）：评价不会在前端显示，但记录保留在后台

这是一个独立的操作，不受审核状态限制。已通过审核的评价可以手动隐藏；审核不通过的评价也可以手动显示。

### 第五步：删除评价

点击"删除"按钮，弹出确认提示"此操作将永久删除该文件，是否继续？"，确认后删除评价记录。删除操作不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 删除时取消操作 | 提示"已取消删除"，不做任何修改 |
| 分页超出范围（当前页无数据） | 自动回退到上一页重新加载 |
| 审核通过时自动打开显示开关 | 无需手动操作，通过审核即显示 |

## 状态流转

```
患者提交评价 → 待审核(10)
                ├── 审核通过(20) → 可显示/隐藏
                └── 审核不通过(30) → 可显示/隐藏
```

## 关联功能

- **问诊服务**：评价来自患者完成问诊后的反馈
- **医生主页**：审核通过且设为显示的评价会展示在医生主页前端
