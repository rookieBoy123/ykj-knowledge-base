# 专科实践管理流程

> 文档版本 2026-07-05 | 面向运营人员

## 概述

专科实践（诊疗科目配置）用于管理互联网医院的执业许可科目。每个科目需要关联国家标准的科室编码，并登记执业许可的三个关键日期：发证日期、生效日期和到期日期。这是互联网医院合规运营的基础配置。

## 触发条件

- **入口路径**：管理后台 → 医院管理 → 专科实践
- **使用角色**：拥有"医疗科目配置"写入权限的管理员（权限标识：`hosp:medical-subjects:write`）
- **前置条件**：系统字典中已配置"科室编码"（ksbm）数据

## 步骤详解

### 第一步：查看科目列表

页面默认展示所有已配置的诊疗科目，每页 10 条。列表字段包括：
- 科目编码（来自国家标准的科室编码）
- 科目名称（对应科室编码的中文名称）
- 执业许可发证日期
- 执业许可生效日期
- 执业许可到期日期
- 创建时间

支持按"科目名称"进行模糊搜索。

### 第二步：新增科目

点击"新增"按钮（仅拥有写入权限的用户可见），打开新增对话框，需要填写：

1. **科目**：下拉选择框，选项来自字典"科室编码"（ksbm），支持筛选搜索。新增时可选，但保存后不可修改。
2. **执业许可发证日期**：必填，日期选择器
3. **执业许可生效日期**：必填，日期选择器
4. **执业许可到期日期**：必填，日期选择器

保存时系统自动将科目编码转换为科目名称一并存储。

### 第三步：编辑科目

点击操作栏的"编辑"按钮，打开编辑对话框。与新增不同：
- **科目**字段在编辑模式下为禁用状态（不可修改），因为科目编码一旦确定不应更改
- 三个执业许可日期可以修改

保存后科目名称会自动根据当前编码更新（从字典中查找对应的最新名称）。

### 第四步：删除科目

点击"删除"按钮，弹出确认提示"确认删除此科目吗？"，确认后删除。注意：删除操作不会检查该科目是否被医生或其他功能引用，直接删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 科室编码字典为空 | 科目下拉框无选项，无法新增 |
| 新增/编辑时必填项为空 | 表单校验拦截，提示"请输入科目""请选择执业许可发证日期"等 |
| 删除时取消操作 | 提示"已取消删除" |
| 科目名称在字典中更新后 | 编辑保存时自动同步最新名称，无需手动修正 |

## 关联功能

- **科室编码字典**：科目下拉框的选项来源于字典"ksbm"，需先维护字典数据
- **医生管理**：医生注册时选择的执业科目与此配置关联
- **执业许可管理**：到期日期可用于监控许可证有效期
