# 医生药房授权管理流程

> 文档版本 2026-07-05 | 面向运营管理员

## 概述

医生药房授权管理用于控制医生可以在哪些药房开具处方。只有经过授权的医生-药房组合，医生端小程序中才能选择对应的药房进行处方开药。

## 触发条件

- **前置条件**：操作员拥有"医生药房授权"写入权限（`med:pharmacy-doctors:write`）
- **入口路径**：管理后台 → 医院管理 → 药房管理 → 许可证管理

## 页面布局

页面分为两个区域：

- **授权列表区**：展示已有的医生-药房授权记录，包含医生姓名、药房名称、创建时间，每条记录可删除
- **搜索区**：支持按医生姓名（最长5个字）和药房名称（最长20个字）搜索已有授权

## 步骤详解

### 第一步：查看已有授权

打开页面后，系统自动加载已有的医生-药房授权列表，分页展示（每页10条）。可以按医生姓名或药房名称搜索筛选。

### 第二步：新增授权（两步穿梭框）

点击"新增"按钮，弹出授权药房对话框，分两步操作：

**第1步：选择医生**

左侧显示待选择的医生列表（只显示审核通过的医生，即入驻状态为"已通过"的医生），右侧显示已选择的医生。支持按科室名称和医生姓名搜索。

- 在左侧勾选医生 → 点击右箭头 → 医生移入右侧"已选择"列表
- 在右侧勾选医生 → 点击左箭头 → 医生从"已选择"列表移回左侧

**第2步：选择药房**

与选择医生类似，左侧显示药房列表，右侧显示已选择的药房。支持按药房名称搜索。

- 在严格模式下，药房列表会隐藏"药房类型"列（不再区分线上/线下药房）

### 第三步：确认授权

选择完医生和药房后，点击"确认"按钮：

- 如果未选择医生，提示"请选择医生"
- 如果未选择药房，提示"请选择药房"
- 如果两者均已选择，系统会将每个选中医生与每个选中药房进行笛卡尔积组合（例如选了2个医生+3个药房=6条授权记录），批量保存

授权成功后，对话框关闭并刷新授权列表。

### 第四步：删除授权

在授权列表中，每条记录都有"删除"按钮。点击后会弹出二次确认对话框，确认后删除该授权记录。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未选择医生就点击确认 | 提示"请选择医生" |
| 未选择药房就点击确认 | 提示"请选择药房" |
| 删除时取消 | 提示"已取消删除" |
| 删除最后一页的最后一条 | 自动跳转到前一页 |
| 搜索无结果 | 列表显示为空 |

## 关联功能

- **医生管理**：授权列表中只显示审核通过的医生（来源于人员审核管理）
- **药房管理**：授权列表中的药房来源于药房药品库
- **医生端处方开药**：医生只能选择已授权药房进行处方开药
