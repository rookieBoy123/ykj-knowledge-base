# 医生药房授权管理流程

> 文档版本：2026-07-06 | 面向读者：运营/管理人员

## 概述

管理人员在后台管理医生与药房的授权关系，即指定哪些医生可以在哪些药房开方。授权后的医生才能在对应的药房为患者开具处方药品。

## 触发条件

- **前置条件**：管理人员已登录后台，拥有授权管理权限（`med:pharmacy-doctors:write`）
- **入口路径**：后台 → 医院管理 → 授权管理（药房授权）

## 步骤详解

### 第一步：查看授权列表

进入页面自动加载已有的医生-药房授权记录。列表展示：

| 列名 | 说明 |
|------|------|
| 医生姓名 | 被授权的医生 |
| 药房名称 | 授权的药房 |
| 创建时间 | 授权创建时间 |

支持按医生姓名或药房名称搜索。

### 第二步：新增授权

点击「新增」按钮，弹出授权药房对话框（宽1300px）。

对话框分为左右两部分：
- **左侧：医生列表**（分页展示）
  - 自动筛选"审核通过"（flowStatus=20）的医生
  - 支持按医生姓名搜索
  - 可勾选一个或多个医生

- **右侧：药房列表**（分页展示）
  - 支持按药房名称搜索
  - 可勾选一个或多个药房

选择医生和药房后，确认授权。系统将所选医生与所选药房建立关联关系。

### 第三步：删除授权

点击某条记录的「删除」按钮，弹出确认对话框"是否确认删除该授权?"，确认后执行删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 无搜索结果 | 列表显示空 |
| 取消删除操作 | 提示"已取消删除" |
| 删除操作失败 | 保持原状态 |

## 关联功能

- **药房渠道管理**：授权的药房来自药房渠道列表
- **医生管理**：授权的医生来自医生列表（仅审核通过的医生）
- **处方开方**：医生只有在授权药房内才能为患者开方
