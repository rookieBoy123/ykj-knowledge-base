# 问诊室医生工作台流程

> 文档版本 2026-07-04 | 面向读者：运营人员、产品经理

## 概述

问诊室是管理后台中医生进行在线问诊的核心工作台。采用三栏式布局：左侧问诊单列表、中间对话聊天区域、右侧问诊资料面板（含病历、处方等）。医生可以在这里完成接诊、问诊对话、书写病历、开具处方、发起视频通话、结束问诊等全流程操作。

## 触发条件

- 医生登录管理后台后进入问诊室处理待接诊/进行中的问诊
- 需要查看患者问诊资料、病历、处方时
- 需要为患者开具处方时
- 需要发起视频问诊时

## 前置条件

- 当前登录账号必须是「医生」角色，非医生角色进入时页面显示"请登录医生角色查看"
- 至少有一条问诊单（待接诊/进行中/已结束）

## 步骤详解

### 第一步：进入问诊室

医生登录后进入「诊疗管理 → 问诊室」页面。页面初始状态显示三栏布局：左侧问诊列表，中间显示"选择患者查看详细信息"的空状态提示，右侧空白。

### 第二步：选择问诊单

左侧问诊单列表按标签页组织，包含三类问诊：
- **待接诊**：患者已发起但医生尚未接受的问诊
- **进行中**：医生已接诊、正在对话的问诊
- **已结束**：已完成的问诊

点击任意一条问诊单，系统自动查询问诊详情并在中间和右侧面板展示对应内容。

### 第三步：接诊操作

在待接诊列表中点击某条记录后，中间聊天区域会显示「接诊」按钮。点击后调用接诊接口，提示"接诊成功"，1秒后自动切换到「进行中」标签页并刷新当前问诊详情。

### 第四步：问诊对话

接诊后中间区域进入聊天界面，医生和患者可以进行图文对话交流（基于即时通讯IM）。

### 第五步：查看问诊资料

右侧面板展示当前问诊的详细资料，包括：
- 患者基本信息
- 问诊申请信息
- 问卷填写内容
- 历史病历记录
- 已开具的处方记录

### 第六步：书写病历

在聊天区域点击「写病历」按钮，右侧面板切换为病历编辑界面。医生填写病历内容后可执行：
- **提交病历**：保存新病历，提示"病历保存成功"
- **更新病历**：修改已有病历，提示"病历更新成功"
- **取消**：放弃编辑，返回问诊资料视图

### 第七步：开具处方

点击「开医嘱」按钮前，系统会检查是否已有病历记录：
- **有病历**：读取病历中的诊断信息，打开处方编辑器
- **无病历**：弹窗提示"请先写病历再开医嘱！"，强制要求先完成病历

处方编辑器以弹窗形式打开，医生选择药品和用法用量后提交。

### 第八步：视频问诊

医生可以发起视频通话邀请患者。点击后系统自动：
1. 调用接口获取视频通话房间号
2. 通过IM向患者发送视频问诊邀请卡片（包含医生姓名和职称）
3. 获取TRTC（腾讯实时音视频）签名参数
4. 打开视频通话界面

患者收到邀请后点击「加入」即可进入视频问诊。

### 第九步：结束问诊

点击「结束问诊」按钮，系统弹出确认对话框"确定结束问诊吗？"，医生确认后：
- 调用结束接口
- 提示"问诊结束"
- 自动切换到「已结束」标签页
- 刷新当前问诊详情

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 非医生角色访问 | 页面显示"请登录医生角色查看"，阻止所有操作 |
| 未选择问诊单 | 中间区域显示"选择患者查看详细信息"空状态 |
| 无病历开处方 | 弹窗"请先写病历再开医嘱！"阻止操作 |
| 结束问诊取消 | 用户取消时提示"操作取消" |
| 视频房间获取失败 | 提示"响应错误，请联系管理人员" |
| 问诊详情加载 | 每次点击问诊单都重新请求详情，不依赖缓存 |

## 关联功能

- **问诊单查询**：统计视角查看所有问诊记录
- **处方审核**：药师审核医生开具的处方
- **病历管理**：病历的归档和查询
- **即时通讯**：医生患者对话依赖腾讯IM
