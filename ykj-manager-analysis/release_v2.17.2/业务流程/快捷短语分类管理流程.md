# 快捷短语分类管理流程

> 文档版本 v1.0 | 面向读者：运营人员

## 概述

快捷短语用于医生在问诊聊天中快速发送常用回复。管理员可以按分类管理快捷短语，支持分类切换、内容的增删改查和批量删除，并对内容名称做重名校验。

## 触发入口

管理后台 → 运营管理 → 快捷短语设置

---

## 步骤详解

### 第一步：选择分类

页面顶部以标签（Tab）形式展示所有快捷短语分类（如"问候语""常用诊断""用药建议"等）。默认选中第一个分类，点击不同标签切换显示对应分类下的短语列表。

分类列表来自接口，仅加载标记为「公共」（isCommon=1）的分类。

### 第二步：查看短语列表

选中分类后，右侧表格展示该分类下的所有短语内容：

| 列名 | 说明 |
|------|------|
| 复选框 | 支持多选，用于批量删除 |
| 内容 | 快捷短语文字内容 |
| 创建时间 | 短语的创建时间 |
| 修改时间 | 短语的最后修改时间 |
| 操作 | 编辑、删除按钮 |

表格上方有搜索框，支持按关键词过滤当前分类下的短语内容。

### 第三步：新增短语

点击「新增」按钮弹出表单：

- **内容**（必填）：输入短语文字，最多255字
- **备注**（可选）：输入备注说明，最多500字
- 保存前系统自动校验：同一分类下内容名称不能重复，重复时提示"内容名称已存在，请重新输入"

短语自动归属到当前选中的分类下。

### 第四步：编辑短语

点击某条记录的「编辑」按钮，弹出表单回填已有内容和备注。修改时同样触发重名校验：
- 如果修改后的内容名称与修改前不同，校验重复
- 如果未做修改，跳过重名校验

### 第五步：删除短语

- **单个删除**：点击某条记录的「删除」按钮，确认后删除
- **批量删除**：勾选多条记录后，点击「批量删除」按钮
- 未勾选任何记录时，提示"请至少选择一条数据"
- 确认框点击取消则提示"已取消删除"

---

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 内容名称为空 | 表单校验提示"请输入内容" |
| 新增时内容名称与已有重复 | 提示"内容名称已存在，请重新输入" |
| 编辑时修改了内容名称且重复 | 提示"内容名称已存在，请重新输入" |
| 批量删除时未勾选记录 | 提示"请至少选择一条数据" |
| 分类列表为空 | Tab栏不显示任何标签 |

## 关联功能

- **问诊聊天**：医生在IM聊天窗口中可以选择快捷短语快速发送
- **数据字典**：快捷短语分类通过字典体系管理
