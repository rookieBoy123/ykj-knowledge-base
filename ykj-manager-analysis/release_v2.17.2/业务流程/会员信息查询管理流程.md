# 会员信息查询管理流程

> 文档版本 v1.0 | 面向读者：运营人员

## 概述

会员管理页面用于查询和浏览平台注册会员的基本信息，包括会员编号、名称、手机号、会员类型、引流医生和注册时间。手机号默认脱敏显示，支持点击查看明文。

## 触发入口

管理后台 → 运营管理 → 会员管理

---

## 步骤详解

### 第一步：会员列表查询

页面支持以下筛选条件：

| 筛选项 | 说明 |
|--------|------|
| 会员编号 | 精确输入，最多20个字符 |
| 会员名称 | 模糊输入（昵称），最多100个字符 |
| 会员类型 | 下拉选择：普通会员 / 特殊会员 |
| 引流医生 | 模糊输入，最多5个字符 |
| 手机号码 | 精确输入，最多11个字符 |

列表展示：会员编号、会员名称、手机号码（脱敏）、会员类型、引流医生、创建时间。

### 第二步：查看完整手机号

手机号列默认显示脱敏格式（如 138****1234），点击脱敏图标后：
- 系统调用接口获取明文手机号
- 明文显示在当前行
- 同时点击其他行时，之前的明文会自动恢复脱敏状态
- 同一时间只有一行显示明文

### 第三步：会员类型说明

| 类型值 | 显示名称 | 说明 |
|--------|---------|------|
| 0 | 普通会员 | 默认类型 |
| 1 | 特殊会员 | 标记为特殊身份 |

---

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 查询手机号明文接口失败 | 保持脱敏状态，不显示明文 |
| 列表为空 | 表格显示空数据 |
| 会员类型值为0/1以外的值 | 显示"未定义" |

## 关联功能

- **患者管理**：会员与患者（就诊人）通过 memberId 关联
- **引流医生**：记录会员是通过哪个医生引流注册的
