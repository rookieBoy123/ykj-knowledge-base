# 中药剂量管理流程

> 文档版本 | 面向读者：运营管理员

## 概述
管理员可以为每一种中药（中药饮片）设置"上限剂量"，即该药品在单次处方中允许使用的最大克数。当医生在处方中开出超过上限剂量的中药时，系统会进行提醒或拦截，防止超量用药风险。

## 触发条件
- **前置条件**：拥有药品管理权限的管理员账号
- **入口路径**：管理后台 → 医院管理 → 药品信息 → 中药剂量管理

## 步骤详解

### 第一步：查看剂量列表
进入页面后，系统展示所有已配置的中药剂量限制列表。每条记录显示：
- **序号**：自动编号
- **药品名称**：中药的名称
- **药品上限剂量**：以"克（g）"为单位显示
- **更新时间**：最后修改时间

支持按药品名称搜索。分页显示，默认每页10条。

### 第二步：新增加剂量限制
1. 点击「新增」按钮，弹出编辑窗口
2. 输入**药品名称**（最多30个字）
3. 输入**上限剂量**（必须为数字，最小值为0.01g）
4. 点击「确定」保存

新增后系统提示"添加成功"，列表自动刷新。注意：上限剂量必须大于等于0.01g，否则会提示"上限剂量最小值为0.01"。

### 第三步：编辑剂量限制
1. 点击列表中某条记录的「编辑」按钮
2. 系统先查询该药品的当前详情，然后将当前数据填充到编辑窗口
3. 修改药品名称或上限剂量后保存

### 第四步：删除剂量限制
点击「删除」按钮，弹出确认提示"此操作将永久删除该药品，是否继续？"，确认后删除。如果取消操作会提示"已取消删除"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 剂量输入值小于0.01 | 表单校验提示"上限剂量最小值为0.01"，无法提交 |
| 药品名称为空 | 表单校验提示"请输入药品名称"，无法提交 |
| 剂量为空 | 表单校验提示"请输入剂量"，无法提交 |

## 业务影响
设置上限剂量后，当医生在开处方时使用该中药且用量超过上限，系统会进行拦截或警告，确保用药安全。

## 关联功能
- **处方开具**（医生端）：剂量限制直接影响医生是否能开出该剂量
- **药品管理**：与药品主数据联动
