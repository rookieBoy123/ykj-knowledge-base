# 排班模板管理流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、系统管理员

## 概述

排班模板是医生排班的基础框架。管理员可以创建一个通用的排班模板，定义每天的上下午时段和每个时段的看诊号源数量。后续为每位医生设置排班时，系统会默认使用此模板，减少重复配置工作。

## 触发条件

- **入口路径**：医院管理 → 医生管理 → 排班模板
- **前置条件**：用户具有医生排班管理权限

## 步骤详解

### 第一步：查看当前模板

进入页面后，系统自动加载当前已保存的排班模板。模板按星期展示（周一至周日），每天包含上午和下午两个默认时段。

每个时段显示的信息：
- 时段名称（如"上午"、"下午"）
- 开始时间和结束时间
- 号源数量（该时段可预约的患者人数上限）
- 启用开关（是否开启该时段）

### 第二步：编辑模板

管理员可以修改任意时段的以下内容：
- **时段名称**：修改该时段的显示名称
- **开始/结束时间**：设置该时段的起止时间
- **号源数量**：设置该时段可接诊的患者数量上限
- **启用状态**：开关控制该时段是否可用

### 第三步：保存模板

修改后点击底部「保存」按钮提交。系统会校验所有时段数据（如时间是否合法、号源数量是否为正整数），校验通过后保存到服务器。

**重要提示**：如果在未保存的情况下离开页面（点击其他菜单或关闭页面），系统会弹出确认框"排班信息未保存，是否保存？"，提供两个选项：
- 「保存并退出」：先保存再离开
- 「直接退出」：放弃修改直接离开

### 第四步：生效范围

保存后的排班模板会作为默认模板生效于所有医生。当管理员为某位医生配置排班时，系统会以此模板为基础，管理员可以在模板基础上进行调整。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 时段数据填写不合法（如时间为空） | 保存按钮不可点击，表单校验不通过 |
| 模板数据未修改就离开页面 | 不弹确认框，直接离开 |
| 模板数据已修改但未保存就离开 | 弹出确认框让用户选择保存或放弃 |
| 保存失败（服务器异常） | 显示错误提示，保留当前编辑内容不丢失 |
| 号源数量输入负数或0 | 组件内部校验，不允许提交 |

## 关联功能

- **医生排班周视图管理**：使用此模板为基础，为每个医生设置具体排班
- **问诊预约**：患者端预约时，系统根据医生排班和号源数量判断是否可约
