# 服务评价审核管理流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、客服人员

## 概述

服务评价管理页面用于查看和审核患者对问诊服务的评价。所有评价需要经过审核才能在小程序中公开展示，管理员可审核通过/不通过、控制评价的显示/隐藏状态。

## 触发条件

- 运营需要审核患者提交的服务评价
- 需要下架不当评价内容
- 需要查看评价统计数据

## 步骤详解

### 第一步：筛选评价记录

进入页面后，可通过以下条件筛选：

| 筛选项 | 说明 |
|--------|------|
| 关键字 | 搜索评价内容中的关键词（最多255字） |
| 评价时间 | 时间范围筛选 |
| 状态 | 待审核、审核通过、审核不通过 |

点击搜索后列表刷新。

### 第二步：查看评价列表

列表展示以下信息：

- **会员名称**：评价人
- **问诊类型**：服务类型（图文问诊、视频问诊等）
- **接诊医生**：被评价的医生
- **分数**：患者给出的满意度评分
- **评价内容**：支持悬停查看完整内容
- **评价时间**：评价提交时间
- **状态**：显示/隐藏开关
- **操作**：通过、详情、删除

### 第三步：审核评价（待审核状态）

对于待审核（flowStatus=10）的评价，可直接在列表中点击「通过」按钮，评价状态变为审核通过。也可以通过点击「详情」进入详情弹窗进行审核。

**审核详情弹窗**展示：
- 会员名称
- 问诊类型
- 接诊医生
- 评价日期
- 分数
- 所选标签（标签形式展示）
- 评价内容
- 是否显示开关

在弹窗中点击「通过」可将待审核评价设置为审核通过（flowStatus=20）并自动设置为显示状态（status=1）。

### 第四步：显示/隐藏切换

已审核通过的评价可通过列表中的开关控制是否在小程序中公开展示：
- **显示**（status=1）：评价在小程序医生主页可见
- **隐藏**（status=0）：评价不公开显示

切换操作即时生效，调用不同的接口（显示调用 evaluateShow，隐藏调用 evaluateHide）。

### 第五步：删除评价

点击「删除」按钮，系统弹出二次确认"此操作将永久删除该文件, 是否继续?"，确认后删除评价。删除操作不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 列表最后一页数据为空（如删除后） | 自动回退到上一页 |
| 取消删除操作 | 提示"已取消删除" |
| 显示/隐藏切换 | 即时生效，提示"操作成功" |

## 关联功能

- 医生主页展示已审核通过的评价
- 医生收益与评价关联
