# 标准药品信息管理流程

> 文档版本：2026-07-04 | 面向读者：药品管理员、运营人员

## 概述

管理药品的基础标准信息库，包括药品药监ID、本位码、通用名、规格、生产厂家、说明书等核心数据。支持新增、编辑、查询和列表查看，为药房上架和处方开具提供标准化的药品数据基础。

## 触发条件

- **前置条件**：需要拥有标准药品管理权限（`std:med:read` 查看、`std:med:write` 编辑）
- **入口路径**：医院管理 → 标准管理 → 标准药品信息

## 步骤详解

### 第一步：进入标准药品列表页

页面展示所有已录入的标准药品信息列表，每页 10 条。列表包含以下信息：

| 列名 | 说明 |
|------|------|
| 药品药监ID | 12 位药品监管唯一标识 |
| 国家医保药品编码 | 医保目录中的药品编码 |
| 药品本位码 | 14 位药品标准编码 |
| 药品通用名 | 药品的通用名称 |
| 商品名称 | 药品的商品品牌名 |
| 生产企业名称 | 药品的生产厂商 |
| 批准文号 | 药品的批准文号 |
| 药品规格 | 如「10mg*7片/盒」 |
| 剂量 | 每单位的剂量数值 |
| 剂量单位 | 剂量的计量单位 |
| 制剂数 | 每包装内含的制剂数量 |
| 制剂单位 | 制剂的数量单位 |
| 包装单位 | 如盒、瓶、袋等 |

### 第二步：搜索药品

页面提供两个搜索条件：

- **药品通用名**：按药品通用名模糊搜索，最多 255 个字符
- **商品名称**：按商品品牌名搜索，最多 255 个字符

输入条件后点击查询刷新列表。

### 第三步：新增药品

点击列表上方的「新增」按钮，进入标准药品添加页面。填写以下信息区：

**基本信息区**：
- **药品药监ID**（必填）：12 位字母数字组合，符合正则校验。旁边提供「查询药品药监ID」链接，可跳转到药监局官网查询
- **药品本位码**（选填）：14 位纯数字
- **国家医保药品编码**（选填）
- **通用名**（必填）
- **商品名称**（必填）
- **生产企业名称**（必填）
- **批准文号**（选填）

**规格信息区**：
- **剂量**（必填）：只能输入数字
- **剂量单位**（必填）：从字典下拉选择（如 g、mg、ml 等）
- **制剂数量**（必填）：只能输入数字
- **制剂单位（用量单位）**（必填）：从字典下拉选择
- **包装单位**（必填）：从字典下拉选择
- **药品规格**（必填）：可手动输入，也可勾选「自动生成」后由系统根据剂量、制剂数量、制剂单位、包装单位自动拼接生成。自动生成的格式为：`{剂量}{剂量单位}*{制剂数量}{制剂单位}/{包装单位}`（如「10mg*7片/盒」）

**自动生成规则**：
- 必须同时填写了「制剂数量」「制剂单位」「包装单位」后才能勾选「自动生成」
- 如果三项中任意一项被清空，自动生成勾选自动取消
- 规格自动拼接后仍可手动修改

**建议用法用量区**：
- **每次用量**：输入数字，如 1
- **使用次数**：从字典下拉选择使用频次（如每日一次、每日两次等）
- **使用方式**：从字典下拉选择（如口服、外用等）
- **建议使用天数**：输入天数，范围 1-999

**药品说明书区**：
- **品牌**：最多 20 字
- **功能主治**：文本框，最多 200 字
- **成分**：最多 200 字
- **性状**：最多 200 字
- **贮藏**：最多 200 字
- **不良反应**：最多 200 字
- **建议用法用量**：最多 200 字
- **禁忌**：最多 200 字
- **注意事项**：最多 500 字

**药品图片区**：
- 支持上传药品图片，最多 10 张，单张不超过 10M
- 页面提示建议图片宽度 670px，字号 12pt

### 第四步：编辑药品

在列表页点击某条药品的「编辑」按钮，进入编辑页面。页面会加载该药品的所有已有信息，可修改后保存。

### 第五步：保存提交

填写完成后点击页面底部「保存」按钮。系统会校验所有必填字段，校验通过后提交保存。保存成功后自动返回列表页，列表自动刷新。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药品药监ID为空 | 提示「药品药监ID不能为空」 |
| 药品药监ID长度不是 12 位 | 提示「只能输入长度为12位的数字、字母」 |
| 药品药监ID格式不对 | 提示「格式不对，请重新输入」 |
| 药品本位码长度不是 14 位 | 提示「只能输入长度为14位的数字」 |
| 药品本位码含非数字 | 提示「只能输入长度为14位的数字」 |
| 制剂数量/剂量为空 | 提示对应字段必填 |
| 未选择剂量单位/制剂单位/包装单位 | 提示对应字段必填 |
| 药品规格为空 | 提示「请输入药品规格」 |
| 点击取消 | 弹出确认框「是否确认取消?」 |

## 关联功能

- **药品库管理**：基于标准药品信息上架具体药品
- **药房管理**：药房引用标准药品数据进行药品维护
- **医保药品目录管理**：医保目录中引用标准药品编码
