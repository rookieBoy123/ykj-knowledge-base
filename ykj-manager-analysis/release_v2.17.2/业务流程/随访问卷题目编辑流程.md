# 随访问卷题目编辑流程

> 文档版本 1.0 | 面向读者：随访管理员、运营人员

## 概述
在创建/编辑随访问卷时，管理员可以为问卷添加题目。题目支持四种类型：单选题、多选题、单项填空题、图片上传题。题目可以上下移动调整顺序，也可以删除。

## 触发条件
- 从随访问卷列表点击"添加问卷"或编辑已有问卷时进入
- 入口：管理后台 → 诊疗管理 → 随访管理 → 随访问卷 → 添加/编辑问卷

## 步骤详解

### 第一步：填写问卷基本信息
进入页面后默认显示"基本信息"标签页（左侧栏第一个）。需要填写：
- **随访标题**：必填，最多30个字符
- **随访说明**：选填，最多200个字符，用于向患者说明问卷目的

点击"保存"后，如果是新建问卷，会自动生成问卷ID并跳转到第二步。

### 第二步：管理题目列表
保存基本信息后，左侧栏出现问卷标题和可展开的题目列表。展开后显示所有已添加的题目，每道题显示题号（如"1.您的症状"），点击即可在右侧编辑该题目。

题目列表支持：
- **新增**：点击"新增"按钮清空右侧表单，准备添加新题目
- **删除**：点击题目右侧的删除图标，确认后删除
- **上移/下移**：第一道题的"上移"按钮和最后一道题的"下移"按钮为禁用状态
- **预览**：点击右上角"预览"按钮可查看问卷最终呈现效果

### 第三步：编辑单个题目
右侧为题目编辑区，可配置：
- **题目名称**：必填，最多30个字符
- **题目类型**：必填，从下拉选择（单选/多选/单项填空/图片上传）

#### 当题目类型为"单选"或"多选"时：
需要添加至少2个、最多10个选项。每个选项可以：
- 修改选项文字
- 上移/下移调整顺序
- 删除选项
- 点击"添加选项"增加新选项（选项达到10个后隐藏此按钮）

选项名称自动分配序号（A、B、C、D……），校验规则：
- 选项名称必填
- 选项名称不能重复

#### 当题目类型为"单项填空"时：
无需配置选项，患者作答时可自由输入文字。

#### 当题目类型为"图片上传"时：
无需配置选项，患者作答时上传图片，最多9张。

### 第四步：切换题目类型
修改题目类型时，已配置的选项列表会清空，需重新配置。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未填写随访标题 | 提示"请输入随访标题"，无法保存 |
| 单选/多选题目少于2个选项 | 提示"请添加2至10个选项" |
| 选项名称为空 | 提示"选项名称为必填" |
| 选项名称重复 | 提示"选项名称不能重复" |
| 删除正在编辑的题目 | 自动切换到"新增题目"状态 |
| 问卷无题目时 | 自动展开题目列表并切换到题目编辑页 |
| 未选择题目类型 | 提示"请选择题目类型" |

## 关联功能
- **随访问卷列表**：发布后的问卷可在随访任务中引用
- **患者端小程序**：患者收到随访任务后按问卷作答
