# 医生人员审核详情流程

> 文档版本：2026-07-04 | 面向读者：运营人员、系统管理员

## 概述

管理员在人员审核列表点击某条待审核记录后，进入此页面查看医生的详细资料，包括基础信息、从业资料、证件证书、服务配置等，并决定审核通过或驳回。

## 触发条件

- 前置条件：具有医生管理权限的管理员账号登录
- 入口路径：「医院管理 → 医生管理 → 人员审核 → 查看详情」

## 页面结构说明

该页面将医生信息分为以下区域：

### 一、基础信息区

显示医生的基本信息，包括：

| 字段 | 说明 |
|------|------|
| 头像 | 可上传200×200px或1:1比例图片 |
| 医生姓名 | 审核通过后不可修改（除非处于待审核状态即 flowStatus=10） |
| 性别 | 男/女选择 |
| 手机号码 | 11位手机号 |
| 出生日期 | 日期选择器 |
| 所在科室 | 树形下拉选择，从医院科室树中选择 |
| 医生职务 | 主任医师/副主任医师/主治医师/医师/医士（字典可配） |
| 民族 | 仅山东地区显示，字典选择 |
| 擅长疾病 | 文本域，最长500字 |
| 医生简介 | 文本域，最长500字 |
| 对症询医 | 仅当系统启用 doctorDiseaseEnabled 时显示，可多选最多10个疾病分类 |

### 二、从业资料区

| 字段 | 说明 |
|------|------|
| 主要执业医院 | 远程搜索选择，输入医院名称搜索 |
| 职务类别 | 字典选择 |
| 全科医生 | 是/否单选 |
| 开始执业日期 | 日期选择 |

**上海地区额外字段**：
- 参加工作日期
- 执业机构（多选，远程搜索医院）
- 执业地点（文本输入，最长50字）

**全日制最高所学专业情况**（仅山东和上海地区显示）：
- 学历（字典选择）
- 学历取得时间（仅山东地区）

### 三、医保相关

- **医保医师编码**：输入框，最长32位。旁边提供"医保医师代码数据库"链接（https://code.nhsa.gov.cn/），提示：该编码由国家医保医师代码数据库动态维护，确认填写后可开具医保处方。

### 四、证件/证书资料区

需要上传三类证件：

| 证件类型 | 上传要求 |
|---------|---------|
| 证件类型（身份证/护照/其他） | 身份证需上传人像页+国徽页，护照只需主页，其他需正面+背面 |
| 资格证书 | 上传证书正面+背面，需选择资质类别名称，填写资格证书编号和获取时间 |
| 执业证书 | 上传证书正面+背面，需填写执业证书编号、发证日期，选择执业范围（树形选择） |

额外可上传最多5张"其他相关证明"图片。

### 五、CA认证状态区（条件显示）

如系统启用了CA签章功能（caflagPage），显示医生的CA认证状态。状态包括：未认证、认证中、已认证等。

如当前审核状态为"驳回"（flowStatus=30），显示驳回原因。

### 六、所属机构区（条件显示）

如系统启用了机构管理（organizationEnabled），可多选"所属机构"，支持远程搜索。

### 七、工作类型选择区

| 类型 | 说明 |
|------|------|
| 医生 | 勾选后显示后续服务配置 |
| 医助 | 辅助医生接诊患者做诊前诊后服务 |

如勾选了"医生"工作类型：
- **测试账号**：勾选后仅可用于测试，不对信息进行备案，诊疗数据不上传
- **全平台不展示**（编辑时显示）：勾选后医生不在患者端全平台展示，无法被搜索到，只有扫医生邀请码的患者才能看到医生

### 八、开启服务项目（仅医生工作类型且在编辑模式下显示）

可配置以下服务类型的开启状态、价格和时长：

| 服务类型 | 配置项 | 备注 |
|---------|--------|------|
| 图文问诊 | 开关、价格(元)、时长(小时/次) | 时长最小值由系统配置 imageTextMinDuration 决定 |
| 视频问诊 | 开关、价格(元)、时长(分钟/次) | 时长最小值由 videoMinDuration 决定 |
| 咨询服务 | 开关、价格(元)、时长 | 时长最小值由 consultMinDuration 决定 |
| 快速续方 | 开关、续方诊金(元) | 续方诊金与处方订单一起收取 |
| 医生复诊 | 开关、复诊诊金(元) | 复诊诊金与处方订单一起收取 |
| 会诊服务 | 开关、会诊诊金(元) | 价格有区间限制 |

**服务附加选项**：
- **医助辅助接诊**：启用后系统自动匹配医助接诊患者做诊前诊后服务
- **AI预问诊**（需 ipcEnabled 开关开启）：启用后系统为医生配置AI助手辅助预问诊收集病史信息
- **隐藏诊费设置**：勾选后医生不可修改服务开启状态和诊费设置

**平台服务**（特殊类型，医生无法自主取消，仅后台操作）：
- **快速问诊**：免费，24小时，可开处方，系统按服务量自动分配；开启后可附带禁止开处方功能
- **商城问诊**（需 mallEnabled 开关开启）：用户在商城购买处方药时匹配该医生，诊金免费，24小时

### 九、接诊模式选择（仅医生工作类型且编辑模式显示，需 registrationEnabled）

| 模式 | 说明 |
|------|------|
| 实时接诊模式 | 医生设置在线即可接诊，离线则提醒用户是否愿意等待 |
| 预约接诊模式 | 医生需设置排班，用户需预约挂号 |

预约接诊模式下额外显示：
- **过号是否接诊**：选择"是"，超过预约时间患者可签到，医生可继续接诊；选择"否"，患者不可签到，问诊自动取消。

## 审核操作

页面底部操作按钮根据场景显示：

| 场景 | 可用按钮 |
|------|---------|
| 查看模式（readOnly=true） | "返回" |
| 编辑模式 + 待审核状态 | "驳回"、"确定" |
| 编辑模式 + 非待审核状态 | "确定" |

**驳回操作**：点击"驳回"弹出对话框，需填写驳回原因（最长255字），确认后向服务端提交驳回请求。

**确定操作**：保存对医生信息的修改（包括审核通过）。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 医生姓名超5个字符 | 输入框限制5字，前端校验拦截 |
| 未上传必需证件图片 | 表单校验拦截，提示对应项为必填 |
| 会诊诊金超出价格区间 | 输入框 min/max 限制，无法输入超出值 |
| 诊费设置为负值 | 输入框 min=0 限制 |
| 图片比例为非1:1 | 头像区文字提示"请上传1:1比例图片" |
| 证件类型切换 | 护照只显示正面上传框，身份证显示正反面 |

## 关联功能

- 跳转来源：人员审核列表页
- 关联配置：诊费设置、排班设置、CA签章管理
