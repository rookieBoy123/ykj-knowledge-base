# 药房订单管理流程（管理后台）

> 文档版本：release/v2.17.2 | 面向读者：运营、药房管理员

---

## 一、模块入口

路径：药房管理 → 药房订单 → 订单列表。列表支持缓存策略：从详情返回用缓存，从其他页面进入重新请求。

---

## 二、订单操作

| 操作 | 说明 |
|------|------|
| 确认发货 | 填写快递单号和公司，记录发货人 |
| 确认收货 | 管理员代替患者确认 |
| 取货码取货 | 输入取货码，验证后取货 |
| 修改快递单号 | 发货后可修正 |
| 修改收货地址 | 发货前可修改 |

---

## 三、关联模块

- 药品管理：药品上架、编辑、库存
- 药师管理：药师列表、账号状态
- 处方审核：药师审核处方
- 销售统计：药品销量和金额统计

---

## 四、关键规则

1. 操作权限：order:orders:write
2. 操作审计：发货操作记录到审计日志
3. 取货码唯一且取货后不可再用
4. 订单状态不可逆
