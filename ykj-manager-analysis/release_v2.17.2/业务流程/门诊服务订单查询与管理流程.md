# 门诊服务订单查询与管理流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、客服人员

## 概述

门诊服务订单管理页面用于查询和管理所有线下门诊服务的问诊预约记录。管理员可以按多种条件筛选问诊记录，查看问诊详情、对话记录和视频回放，必要时可以手动结束问诊。

## 触发条件

- 管理员需要查看某个患者的问诊情况
- 客服需要核实问诊状态或处理投诉
- 运营需要统计门诊服务的使用情况

## 步骤详解

### 第一步：筛选门诊服务订单

进入页面后，可通过以下条件筛选：

| 筛选项 | 说明 |
|--------|------|
| 问诊编号 | 输入精确的问诊编号查询 |
| 服务类型 | 多选：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊 |
| 问诊流程 | 多选：在线问诊、线上复诊、慢病医保、预约挂号、快速开方（开启AI医生后增加AI问诊选项） |
| 问诊状态 | 已预约（签到前）、问诊中（签到后）、问诊结束、取消问诊 |
| 支付时间 | 选择时间范围筛选已支付的订单 |
| 所属机构 | 输入机构名称（仅开启机构管理时显示） |
| 问诊时间 | 问诊发起的时间范围 |
| 就诊时间 | 实际就诊的时间范围 |
| 有无病历 | 筛选是否已有病历记录（有/无） |
| 医生姓名 | 输入医生姓名（最多5字） |
| 患者姓名 | 输入患者姓名（最多5字） |
| 所属团队 | 选择专家团队（仅开启专家团队时显示） |

点击搜索按钮后，列表刷新。

### 第二步：查看问诊列表

列表展示以下信息：
- **问诊编号**：唯一标识，支持悬停查看完整编号
- **服务类型**：如"图文问诊"、"视频问诊"
- **问诊流程**：如"在线问诊"、"线上复诊"
- **医生姓名**：接诊医生
- **支付时间**：订单支付的时间
- **诊费**：实际支付金额
- **患者姓名**：脱敏显示（姓+星号），点击可查看完整姓名
- **患者联系方式**：脱敏显示，点击可查看明文手机号（需调用脱敏接口，一次只显示一行明文）
- **所属机构**：仅开启机构管理时显示
- **所属团队**：仅开启专家团队时显示
- **问诊时间**：问诊发起时间
- **就诊时间**：实际就诊时间
- **状态**：已预约/问诊中/问诊结束/取消
- **有无病历**：有/无
- **操作**：查看详情、结束问诊、视频记录、知情同意书

### 第三步：查看问诊详情（问诊对话记录）

点击「查看详情」，弹窗显示问诊对话记录：

- **普通问诊**：展示IM即时通讯消息记录，支持滚动加载历史消息（每次加载20条），支持语音消息播放
- **AI图文问诊**：展示AI问诊对话记录，分为AI角色（左侧蓝色标识）和患者角色（右侧），支持文本消息、卡片消息、图片消息的展示

在AI问诊记录弹窗中，还可查看「AI问诊报告」，包含：
- 主诉
- 病史信息（现病史、既往史、过敏史、家族史、个人史、月经史、婚育史）
- 体检（辅助检查）
- AI诊断（标签展示，附"仅供参考"提示）

弹窗中带有水印保护（格式：管理员用户名@系统名称）。

### 第四步：结束问诊

对于问诊中的记录（flowStatus=20），管理员可点击「结束问诊」按钮。系统会弹出二次确认"确认是否结束所选问诊?"，确认后调用接口结束问诊，刷新列表。

### 第五步：查看视频记录

对于视频问诊类型的记录，点击「视频记录」按钮，弹窗列出门诊过程中录制的视频列表。每条视频显示创建时间，可点击「播放」在线观看视频回放。

### 第六步：查看知情同意书

对于已签署知情同意书的记录，点击「知情同意书确认信息」按钮，弹窗显示：
- 知情同意书声明内容
- 确认人姓名
- 网络IP
- 设备信息
- 微信用户openId
- 同意书签署时间

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 查看详情时该问诊无对话记录 | 提示"没有找到相关问诊对话记录!" |
| 结束问诊时问诊编号为空 | 提示"问诊编号为空！" |
| 点击其他行的手机号脱敏查看 | 自动恢复之前显示明文的行，同一时间只显示一行明文 |
| 列表最后一页数据为空 | 自动回退到上一页 |

## 关联功能

- 跳转到问诊室医生工作台处理问诊
- 跳转到处方列表查看该问诊关联的处方
- 跳转到订单详情查看支付信息
