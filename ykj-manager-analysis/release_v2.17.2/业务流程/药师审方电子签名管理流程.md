# 药师审方电子签名管理流程

> 文档版本 2026-07-06 | 面向管理员/运营人员

## 概述

药师在审核处方并确认发药量后，需要上传个人电子签名。该功能允许药师查看个人信息、上传手写签名图片（支持裁剪），并将签名存储到系统中用于处方审核记录的电子签署。

## 触发条件

- **前置条件**：以药师身份登录管理后台
- **入口路径**：管理后台 → 诊疗管理 → 处方管理 → 药师审方 → 录入药量 → 签名上传

## 页面结构

页面分为三个信息区：

| 区域 | 内容 |
|------|------|
| 基本信息区 | 药师姓名、性别、手机号码、邮箱地址 |
| 相关证书区 | 身份证照片、资格证书照片（图片预览） |
| 电子签名区 | 当前签名预览、上传新签名（含裁剪工具）、提交按钮 |

## 步骤详解

### 第一步：查看药师信息

1. 进入页面后，系统自动加载当前登录药师的信息
2. 基本信息区展示：药师姓名、性别、手机号码、邮箱
3. 证书区展示：身份证正反面照片、资格证书照片
4. 若药师信息为空（无药师身份），页面显示"暂无数据"

### 第二步：上传电子签名

1. 点击"点击上传签名"按钮，选择本地图片文件
2. 上传前校验：
   - 仅支持 JPG/PNG 格式，不符合时提示"上传图片只能是jpg/png格式!"
   - 文件大小不超过 2M，超限时提示"上传图片大小不能超过 2M!"
3. 图片通过校验后，弹出图片裁剪对话框
4. 裁剪工具功能：
   - 鼠标滚轮可放大缩小图片
   - 可拖动图片选择裁剪区域
   - 裁剪框宽高比固定为 2:1
   - 默认输出 PNG 格式，质量为 80%
5. 点击"确认"完成裁剪，系统上传裁剪后的签名图片

### 第三步：提交签名

1. 确认裁剪后显示签名预览图
2. 点击"提交"按钮保存签名
3. **校验**：必须先上传签名图片才能提交，未上传时提示"签名必须上传!"
4. 系统调用更新接口保存签名图片ID
5. 成功后提示"操作成功"

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未上传签名就提交 | 提示"签名必须上传!"，阻止提交 |
| 上传非 JPG/PNG 格式图片 | 提示"上传图片只能是jpg/png格式!" |
| 上传超过 2M 的图片 | 提示"上传图片大小不能超过 2M!" |
| 无药师身份/药师信息为空 | 页面显示"暂无数据"占位提示 |

## 关联功能

- **药师列表管理**：药师基本信息的增删改查
- **处方审核管理**：药师审核处方流程
- **药量录入**：处方药品发药量的录入确认
