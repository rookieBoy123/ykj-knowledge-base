# 标准医疗信息管理流程

> 文档版本 2026-07-05 | 面向管理后台运营人员

## 概述

标准医疗信息库是平台药品数据的"源头"——管理员在这里录入和维护每款药品的完整信息（包括药监编码、规格、说明书、建议用法用量等），供医生开方时查询和使用。

## 触发条件

- **前置条件**：管理员拥有"标准药品"的读写权限（`std:med:read` / `std:med:write`）
- **入口路径**：管理后台 → 医院管理 → 标准管理 → 标准医疗信息列表

## 步骤详解

### 第一步：查看药品列表

进入页面后，看到所有已录入的标准药品信息列表。列表展示了每款药品的关键字段：

- 药品药监ID（YPID，国家药品监督管理局统一编码）
- 国家医保药品编码
- 药品本位码
- 药品通用名、商品名称、生产企业名称
- 批准文号、药品规格、剂量及单位、制剂数及制剂单位、包装单位

列表支持按**药品通用名**和**商品名称**搜索。点击列表右上角的"新增"按钮进入新增页面。

### 第二步：新增药品信息（三个表单区域）

新增页面分为三个信息区域，每个区域包含多个字段：

**① 基本信息区**（10个字段）：
- **药品药监ID**：必填，12位字母数字组合。页面提供查询链接（跳转到国家卫健委查询网站），方便管理员查证
- **药品本位码**：选填，14位纯数字。填写后会进行格式校验
- **国家医保药品编码**：选填
- **通用名**：必填
- **商品名称**：必填
- **生产企业名称**：必填
- **批准文号**：选填

**② 规格信息区**（6个字段，含自动生成功能）：
- **剂量**：必填，整数或小数（只能输入数字和小数点）
- **剂量单位**：必填，下拉选择（从字典中加载剂量单位列表）
- **制剂数量**：必填，整数或小数
- **制剂单位（用量单位）**：必填，下拉选择
- **包装单位**：必填，下拉选择
- **药品规格**：必填。管理员可以手动输入，也可以勾选"自动生成"复选框让系统自动拼接。自动生成的规则是：如果有剂量和剂量单位，格式为「剂量+剂量单位 * 制剂数量+制剂单位/包装单位」；如果没有剂量，则只显示「制剂数量+制剂单位/包装单位」

> ⚠️ 注意："自动生成"复选框只有在三个字段都填写后才会启用——制剂数量、制剂单位、包装单位缺一不可。如果其中任一字段被清空，自动生成开关会自动关闭。

**③ 建议用法用量区**（4个字段）：
- **每次用量**：选填，整数或小数
- **使用次数**：选填，下拉选择频次（由后端接口返回使用频率列表）
- **使用方式**：选填，下拉选择（从字典中加载用药途径列表）
- **建议使用天数**：选填，1-999天

### 第三步：填写药品说明书（可选）

说明书区域包含以下字段，全部为选填：
- **品牌**：最多20字
- **功能主治**：最多200字
- **成分、性状、贮藏、不良反应、建议用法用量、禁忌**：各最多200字
- **注意事项**：最多500字

页面上提示建议字号12pt，上传图片宽度670px。这些是给后续生成药品详情页的设计参考。

### 第四步：上传药品图片（可选）

支持上传最多10张药品图片，每张不超过10M。图片上传后以缩略图列表形式展示在页面底部，可以预览和删除。

### 第五步：保存

点击底部"保存"按钮，系统先校验所有必填字段是否完整。校验通过后提交保存。如果有任何必填项未填写或格式不对（如药品药监ID不是12位），页面顶部会显示红色的错误提示，需要修正后才能保存。

保存成功后自动返回列表页。新增的药品会出现在列表中。

### 编辑已有药品

从列表页点击某款药品的"编辑"按钮，进入同样的表单页面，所有已有数据会预填充。编辑时药品药监ID可以修改（与新录入时一样的校验规则）。修改后点保存即可。

> ⚠️ 编辑时如果只修改了基本信息而不修改说明书或用法用量，说明书和用法用量的原有数据不会丢失——系统使用 saveOrUpdate 策略，有值才更新。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 药品药监ID未填写 | 提示"药品药监ID不能为空" |
| 药品药监ID长度不是12位 | 提示"只能输入长度为12位的数字、字母" |
| 药品药监ID格式不符合（如含特殊符号） | 提示"格式不对，请重新输入" |
| 药品本位码填写但长度不是14位 | 提示"只能输入长度为14位的数字" |
| 药品本位码包含非数字字符 | 提示"只能输入长度为14位的数字" |
| 必填字段未填写（通用名/商品名/生产企业/剂量/规格等） | 提交时提示"请完善资料"，对应字段下方显示红色错误文字 |
| 制剂数量/单位/包装单位任一为空时勾选"自动生成" | 自动生成复选框被置灰禁用，无法开启 |
| 无添加/编辑权限 | 对应的"新增""编辑"按钮不显示 |

## 关联功能

- 列表页编辑后会跳转到详情页，返回列表页时有路由缓存策略：从详情页返回时保留之前的搜索条件和页码，从其他页面进入时重置搜索条件
- 服务端有独立的药品说明书导入接口（Excel导入），可在后端直接批量导入说明书数据
- 标准医疗信息库中的数据会同步到开放平台接口，供合作机构查询
