# 可视化页面设计管理流程

> 文档版本 1.0 | 面向读者：运营人员、产品经理

## 概述

管理后台提供了一套可视化拖拽页面搭建工具，管理员无需开发技能即可通过拖拽组件、调整配置来搭建小程序首页及其他页面的布局和内容。搭建完成后可实时保存、另存为模板、手机扫码预览效果。

## 触发条件

- **入口路径**：运营管理 → 方案设计 → 选中某个页面进入设计器
- **操作模式**：新增（全新搭建）、编辑（修改已有页面）、复制（基于已有页面创建副本）
- **前置条件**：需拥有页面设计的管理权限

## 步骤详解

### 第一步：进入设计器

页面进入后呈现**三栏布局**：
- **左侧**：组件库面板，分类展示可使用的页面组件（如搜索框、轮播图、导航栏等），点击展开/折叠分类
- **中间**：手机模拟预览区，实时展示当前页面的效果，可在其中拖拽排序、复制、删除组件
- **右侧**：配置面板，选中某个组件后在此处编辑该组件的详细参数

顶部工具栏显示当前页面名称（可点击编辑图标修改）、"手机预览"按钮和"保存修改"按钮。

如果是新增页面（无历史数据），系统会自动弹出页面命名框，页面名称限制15个字以内。

### 第二步：添加页面组件

从左侧组件库中，有两种方式添加组件到页面：
1. **点击添加**：直接点击组件，将其添加到页面最底部（搜索框默认添加在顶部，底部导航固定在最下方）
2. **拖拽添加**：按住组件拖到中间预览区释放

部分组件有数量限制：
- 搜索框和底部导航栏：整个页面只能添加一次，重复添加会提示"该组件只能添加一次"
- 其他组件可重复添加

### 第三步：配置组件内容

点击中间预览区的任一组件，右侧配置面板会显示该组件所有可编辑的参数。不同组件配置项各不相同，例如：
- 轮播图：可上传图片、设置跳转链接、调整高度
- 商品列表：可选择展示哪些商品分类、排列方式
- 导航栏：配置图标、名称、跳转目标

### 第四步：调整组件排序

- **拖拽排序**：在中间预览区直接拖动组件调整顺序（搜索框和底部导航禁止拖动）
- **上移/下移按钮**：点击组件右上角的上/下箭头逐位移动（搜索框和底部导航禁止移动）
- **复制组件**：点击右上角的复制图标，在当前组件下方插入一个副本
- **删除组件**：点击右上角的删除图标移除组件

### 第五步：设置页面全局属性

点击中间区域底部的"页面设置"按钮，可配置整个页面的通用属性：
- **页面标题**：页面顶部显示的文字
- **标题颜色**：标题文字颜色和背景色
- **背景**：可设置纯色背景或图片背景，图片支持平铺/拉伸/不重复三种模式
- **页面可见性**：控制该页面在小程序端是否展示

### 第六步：保存与发布

- **保存修改**：点击顶部"保存修改"按钮（或使用快捷键 Ctrl+S），将所有组件配置和页面属性保存
- 保存时系统会校验页面名称不能为空
- 新增页面首次保存时调用新建接口，后续保存调用编辑接口
- 保存成功后页面ID会更新，后续操作基于该ID

### 第七步：其他操作

- **另存为模板**：将当前页面复制为一个新模板，需输入新模板名称（不超过15字）
- **页面重置**：清空当前页面所有组件和配置，恢复为初始状态（需要二次确认）
- **手机预览**：点击"手机预览"按钮弹出二维码，用小程序扫码可在手机上看到真实效果

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 保存时页面名称为空 | 提示"模板名称不能为空"，阻止保存 |
| 重复添加搜索框或底部导航 | 提示"该组件只能添加一次" |
| 重复添加医生视频列表（功能未开启） | 该组件不在左侧列表中显示 |
| 重复添加健康服务包（功能未开启） | 该组件不在左侧列表中显示 |
| 离开页面未保存 | 弹窗提示"离开后系统不会保存已修改内容"，需用户确认 |
| 拖拽搜索框或底部导航 | 提示"该组件禁止拖拽/移动" |
| 新建页面时未设置名称 | 自动弹出命名输入框 |

## 关联功能

- **方案设计列表**：管理所有DIY页面的增删改查入口
- **小程序首页**：搭建的页面发布后在小程序端展示
- **功能开关**：医生视频列表和健康服务包组件受系统功能开关控制，关闭时在设计器中不可见
