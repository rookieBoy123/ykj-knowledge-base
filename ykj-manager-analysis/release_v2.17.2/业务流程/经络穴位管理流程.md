# 经络穴位管理流程

> 文档版本 2026-07-04 | 面向读者：内容运营人员

## 概述

管理员在后台维护中医经络穴位百科内容，包括穴位的名称、归属经络、身体部位、功效作用及穴位的详细说明（富文本）。内容展示在患者端"中医百科-经络穴位"模块。

## 触发条件

- 入口路径：管理后台 → 运营管理 → 医学百科 → 经络穴位管理
- 适用角色：拥有医学百科管理权限的管理员

## 步骤详解

### 第一步：查看经络穴位列表

进入页面后，系统展示穴位列表，每条记录包含：
- **封面图**：穴位的展示缩略图
- **穴位名称**：如"合谷""足三里""涌泉"等
- **归属经络**：如手太阴肺经、足阳明胃经等
- **身体部位**：如头部、上肢、下肢等
- **功效作用**：该穴位的治疗功效
- **最后修改时间**：最近编辑时间

列表支持按穴位名称（关键词）、归属经络、身体部位三个维度筛选。筛选选项来自数据字典。

### 第二步：新增穴位

点击「新增」按钮，弹出表单弹窗（而非跳转新页面），需要填写：

- **穴位名称**：必填，最多10字
- **展示封面图**：必填（与经典名方/中药图鉴不同），建议 200×200px 1:1 比例
- **归属经络**：必填，下拉选择（如手太阴肺经、手阳明大肠经等）
- **身体部位**：必填，下拉选择（如头部、上肢、下肢等）
- **功效作用**：必填，最多100字
- **穴位详情**：必填，富文本编辑器，可插入图片

> ⚠️ 与经典名方和中药图鉴不同，经络穴位的封面图为必填项。

### 第三步：编辑穴位

点击「编辑」按钮，弹出表单弹窗，系统回填已有信息。穴位详情通过 `setContent()` 方法回填富文本内容。

### 第四步：删除穴位

- **单个删除**：点击「删除」按钮，二次确认后删除
- **批量删除**：勾选多条后点击「批量删除」

删除后数据不可恢复。

### 第五步：新增/编辑后刷新

新增成功后，列表自动回到第一页刷新（无论原来在第几页）。编辑成功后，停留在当前页刷新。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 穴位名称为空 | 前端提示"请填写穴位名称" |
| 封面图未上传 | 前端提示"请上传封面图"（与经典名方不同，此项必填） |
| 归属经络未选择 | 前端提示"请选择归属经络" |
| 身体部位未选择 | 前端提示"请选择身体部位" |
| 功效作用未填写 | 前端提示"请填写功效作用" |
| 穴位详情未填写 | 前端提示"请填写穴位详情" |

## 关联功能

- **患者端经络穴位浏览**：小程序「中医百科 → 经络穴位」展示此处维护的内容
- **数据字典**：归属经络（jlxw-jl）、身体部位（jlxw-bw）的下拉选项来自字典
