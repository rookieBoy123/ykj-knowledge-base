# 问诊订单查询与管理流程

> 文档版本：2026-07-04 | 面向读者：运营管理员、财务人员

## 概述

问诊订单查询页面用于查询所有线上问诊的订单记录。与门诊服务订单不同，这里查询的是线上问诊（图文、视频、快速问诊等）产生的订单，包含完整的支付信息和收益数据。

## 触发条件

- 财务人员需要核对问诊订单的支付情况
- 运营需要统计医生的问诊收益
- 客服需要帮患者查询订单状态

## 步骤详解

### 第一步：筛选问诊订单

页面默认查询最近30天内的订单。支持以下筛选条件：

| 筛选项 | 说明 |
|--------|------|
| 订单编号 | 精确查询（最多20字） |
| 创建时间 | 默认最近30天 |
| 服务类型 | 多选：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊（开启AI医生后增加AI图文问诊） |
| 问诊流程 | 多选：在线问诊、线上复诊、慢病医保、预约挂号、快速开方（开启AI医生后增加AI问诊） |
| 问诊状态 | 已预约、问诊中、问诊结束、取消问诊 |
| 支付状态 | 未支付、已支付 |
| 医生姓名 | 最多5字 |
| 患者姓名 | 最多5字 |
| 支付时间 | 时间范围筛选 |
| 所属团队 | 选择专家团队（仅开启专家团队功能时显示） |
| 问诊单编号 | 关联的问诊单编号（最多30字） |

### 第二步：查看订单列表

列表展示以下信息：

- **订单号**：订单唯一编号
- **问诊单编号**：关联的问诊单编号
- **服务类型**：如"图文问诊"、"视频问诊"
- **问诊流程**：如"在线问诊"、"线上复诊"
- **医生姓名**：接诊医生
- **所属团队**：专家团队名称（仅开启专家团队时显示）
- **患者姓名**：脱敏显示（姓+星号），点击可查看完整姓名
- **联系方式**：脱敏显示，点击调用脱敏接口查看明文，同一时间只显示一行明文
- **创建时间**：订单创建时间
- **订单支付时间**：仅已支付订单显示支付时间
- **诊费（元）**：支付金额
- **平台服务费**：仅已支付且问诊结束的订单显示
- **医生收益（元）**：仅已支付且问诊结束的订单显示
- **问诊状态**：已预约/问诊中/问诊结束/取消
- **支付状态**：已支付/未支付

> **注意**：平台服务费和医生收益仅在问诊结束（flowStatus=30）且已支付（isPaid=1）时才显示具体数值，否则显示"-"。

### 第三步：导出订单

点击「导出」按钮可将当前筛选条件下的订单数据导出为Excel文件（.xls格式）。

**导出限制**：创建时间和支付时间的时间跨度均不能超过三个月，否则提示"时间间隔不能超过三个月"。

### 第四步：数据脱敏机制

患者姓名和联系方式默认脱敏显示：
- **患者姓名**：显示姓 + 星号（如"张**"），点击眼睛图标可切换显示完整姓名
- **联系方式**：显示脱敏手机号，点击眼睛图标后通过 `memberPhone` 接口获取明文，同时自动恢复之前显示明文的行

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 导出时时间跨度超过3个月 | 提示"时间间隔不能超过三个月" |
| 列表最后一页数据为空 | 自动回退到上一页 |
| 切换手机号明文查看 | 自动关闭之前显示的明文，同时只显示一行明文 |
| 未支付订单查看支付时间 | 显示"-" |

## 关联功能

- 跳转到问诊详情查看问诊记录
- 关联退款审核（已支付订单的退款处理）
- 关联医生收益结算
