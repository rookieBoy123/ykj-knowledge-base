# 问诊订单查询与管理流程

> 文档版本 1.0 | 面向角色：运营管理员、财务管理员

## 概述

问诊订单查询页面是管理后台用于统一查询所有问诊订单的管理工具。运营人员可以通过多维度筛选条件（订单编号、服务类型、问诊流程、问诊状态、支付状态等）快速检索问诊记录，并支持数据导出。

## 触发条件

- **前置条件**：登录管理后台，拥有问诊订单查询权限
- **入口路径**：管理后台 → 诊疗管理 → 门诊服务订单 → 问诊列表

## 步骤详解

### 第一步：进入问诊列表页

页面顶部是筛选区域，下方是问诊订单列表。页面加载时会自动设置默认时间范围为"最近30天"。

### 第二步：使用筛选条件查询

系统提供了以下筛选维度，可以单独或组合使用：

| 筛选条件 | 说明 |
|---------|------|
| 订单编号 | 精确搜索订单号，最多20位 |
| 创建时间 | 选择日期范围，默认最近30天 |
| 服务类型 | 多选：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊。如果系统开启了AI医生功能，还会额外显示"AI图文问诊"选项 |
| 问诊流程 | 多选：在线问诊、线上复诊、慢病医保、预约挂号、快速开方。如果开启了AI医生，还会额外显示"AI问诊" |
| 问诊状态 | 单选：已预约（含已预约和已签到）、问诊中、问诊结束、取消问诊 |
| 支付状态 | 单选：未支付、已支付 |
| 医生姓名 | 模糊搜索医生姓名，最多5个字符 |
| 患者姓名 | 模糊搜索患者姓名，最多5个字符 |
| 支付时间 | 选择日期范围 |
| 所属团队 | 下拉选择专家团队（仅当系统开启了专家团队功能时显示） |
| 问诊单编号 | 精确搜索问诊单编号，最多30位 |

### 第三步：查看问诊订单列表

查询结果以表格形式展示，包含以下信息：

| 列名 | 说明 |
|------|------|
| 订单号 | 订单唯一编号 |
| 问诊单编号 | 问诊单唯一编号 |
| 服务类型 | 如"图文问诊"、"视频问诊"等 |
| 问诊流程 | 如"在线问诊"、"线上复诊"等 |
| 医生姓名 | 接诊医生的姓名 |
| 所属团队 | 医生所在的专家团队名称（仅开启专家团队时显示） |
| 患者姓名 | 脱敏显示（如 张**），可点击查看完整姓名 |
| 联系方式 | 脱敏显示，可点击查看完整手机号 |
| 创建时间 | 订单创建时间 |
| 订单支付时间 | 如未支付则显示"-" |
| 诊费(元) | 问诊费用 |
| 平台服务费 | 平台抽取的服务费（仅已支付且问诊结束的订单显示，否则显示"-"） |
| 医生收益(元) | 医生实际所得（仅已支付且问诊结束的订单显示，否则显示"-"） |
| 问诊状态 | 草稿/问诊中/问诊结束/取消问诊 |
| 支付状态 | 已支付/未支付 |

**数据脱敏**：患者姓名和联系方式默认脱敏显示。管理员点击后可以查看完整信息，但同一时间只能查看一条记录的完整信息（点击新记录会自动隐藏之前显示的）。

**问诊状态的复合显示**：列表中的"已预约"状态实际是"已预约+未签到"（对应 flowStatus=20, isCheckIn=0），"问诊中"是"已预约+已签到"（对应 flowStatus=20, isCheckIn=1）。这两个子状态在筛选时通过相同的 flowStatus=20 和不同的 isCheckIn 值区分。

### 第四步：导出数据

点击「导出」按钮，将当前筛选条件下的所有数据导出为Excel文件（.xls格式）。

**导出限制**：创建时间范围和支付时间范围均不能超过3个月。如果超出，会提示"时间间隔不能超过三个月"。

导出过程中会显示加载提示，导出完成后自动关闭。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 导出时时间范围超过3个月 | 提示"时间间隔不能超过三个月"，拒绝导出 |
| 搜索无结果 | 列表显示为空 |
| 患者姓名/联系方式查看 | 默认脱敏，点击后需服务端验证才能查看完整信息 |

## 关联功能

- **系统功能开关**：专家团队开关控制"所属团队"筛选项和列的显示；AI医生开关控制"AI问诊"流程和"AI图文问诊"服务类型的显示
- **订单管理**：此页面查询的问诊数据来自问诊订单服务
