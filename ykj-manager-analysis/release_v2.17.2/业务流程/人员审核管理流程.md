# 人员审核管理流程

> 文档版本 2026-07-04 | 面向读者：运营管理员、医务管理人员

## 概述

管理员审核医生入驻申请，查看医生提交的个人资料、资质证书，决定通过或不通过该医生的入驻申请。支持按审核状态分标签页查看、按姓名/科室/医院/职称等多维度筛选。

## 触发条件

- **前置条件**：医生通过小程序提交了入驻申请（或管理员手动新增医生）
- **入口路径**：管理后台 → 医院管理 → 医生管理 → 人员审核

## 步骤详解

### 第一步：进入审核列表

页面默认展示「待审核」标签页，列出所有待审核的医生申请。页面顶部显示三个标签页：
- **待审核（N）**：显示待审核的申请数量
- **不通过（N）**：显示已驳回的申请数量
- **已通过（N）**：显示已审核通过的申请数量

每个标签页括号内的数字是系统实时统计的真实数量。

### 第二步：筛选查找

管理员可以通过以下条件快速定位需要审核的医生：

- **医生姓名**：输入医生姓名搜索（最多5个字）
- **科室**：树形下拉选择科室，支持模糊搜索
- **所属医院**：远程搜索选择医院（输入关键词后从系统拉取匹配医院）
- **职称**：下拉选择（主任医师、副主任医师等）
- **所属机构**：如果系统开启了机构管理模式，可筛选所属机构

列表展示字段包括：编号、医生姓名、所属医院、科室、职称、电话号码（脱敏显示）、所属机构、邀请人、申请时间。

> 电话号码默认为脱敏状态，管理员点击后可查看完整号码（系统记录脱敏查看日志）。

### 第三步：审核医生申请

列表每行操作列根据不同状态显示不同按钮：
- **待审核状态**：显示「去审核」按钮
- **不通过状态**：显示「查看详情」按钮
- **已通过状态**：显示「查看详情」按钮

点击「去审核」进入医生详情审核页面，包含以下信息区：

#### 基础信息区
- 头像（200×200像素要求）
- 医生姓名、性别、手机号码、出生日期
- 所在科室（树形选择）、医生职务、擅长疾病、医生简介
- 对症询医（选择疾病分类关联，最多10个）

#### 从业资料区
- 主要执业医院（远程搜索选择）
- 职务类别、全科医生标识
- 开始执业日期
- 执业机构/地点（上海地区特有）
- 学历信息（全日制最高学历）
- 医保医师编码（附医保编码数据库链接）

#### 证件/证书资料区
- 证件类型（身份证正面+反面照片，支持多种证件类型）
- 身份证号
- 资格证书（正面+背面照片，资质类别、编号、获取时间）
- 执业证书（正面+背面照片，执业证书编号）

> 审核页面支持三种证书上传和对应编号填写，所有字段均有校验规则。

### 第四步：审核决定

在详情页面底部，管理员可以：
- **通过**：点击通过按钮，医生状态变为「已通过」，医生可以在小程序接诊
- **不通过**：点击不通过按钮，医生状态变为「不通过」，医生需要重新提交资料

审核数据结构：
- 审核前（flowStatus=10）可以修改医生所有信息
- 审核通过后（flowStatus≠10），部分字段（如医生姓名）变为只读

### 第五步：HIS系统同步（严格模式）

如果系统配置了互联网医院模式（internetDiagnosis），页面顶部会显示「同步HIS医生」按钮（替代「新增」按钮），点击后弹窗从HIS系统同步医生信息到本系统。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 搜索无结果 | 列表显示空状态，可调整筛选条件重新搜索 |
| 最后一页数据删除后 | 自动回退到上一页，防止空白页 |
| 电话号码查看 | 点击脱敏字段触发验证，获取完整号码后展示；再次点击恢复脱敏 |
| 保存时字段校验失败 | 表单标红提示对应字段错误信息 |
| HIS同步医生信息异常 | HIS弹窗组件处理异常，同步失败需联系技术人员 |

## 关联功能

- **医生列表管理**：审核通过后医生出现在医生列表中
- **医生排班管理**：审核通过的医生可进行排班
- **机构管理**：如果开启了机构模式，医生与机构关联
- **商务邀请医生**：商务人员邀请的医生在此处审核
