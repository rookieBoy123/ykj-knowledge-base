# 人员审核管理流程

> 文档版本 v1.0 | 面向读者：运营管理员

## 概述

管理员在后台查看所有申请入驻的医生，根据提交的资质材料进行审核，审核通过后医生方可上线提供服务，不通过则需填写驳回原因。

## 触发条件

- 前置条件：拥有后台管理权限的管理员账号
- 入口路径：诊疗管理 → 医生管理 → 人员审核

## 步骤详解

### 第一步：查看审核列表

进入页面后，默认展示「待审核」状态的医生列表。页面顶部有三个状态标签页：

| 标签 | 含义 | 对应状态值 |
|------|------|-----------|
| 待审核 | 医生已提交入驻申请，等待审核 | 10 |
| 已通过 | 审核通过，医生可正常服务 | 20 |
| 不通过 | 审核驳回，需填写驳回原因 | 30 |

每个标签上显示对应状态的医生数量。列表展示医生的编号、姓名、所属医院、科室、职称、电话号码、邀请人和申请时间等信息。

**筛选条件**：
- 医生姓名（模糊搜索，最多5个字）
- 科室（树形下拉选择）
- 所属医院（远程模糊搜索）
- 职称（下拉选择）
- 所属机构（功能开关控制是否显示）

电话号码默认脱敏显示，管理员需点击后才能查看完整号码。

### 第二步：审核医生资料

点击待审核医生的「去审核」按钮，进入医生详情页。详情页分为以下几个信息区：

**① 基础信息区**
- 头像图片（支持上传，建议200×200px或1:1比例）
- 医生姓名（最多5字）
- 性别（男/女）
- 手机号码（11位）
- 出生日期
- 所在科室（树形选择）
- 医生职务（下拉选择）
- 擅长疾病（最多500字）
- 医生简介（最多500字）
- 对症询医相关疾病分类（最多选10个，功能开关控制）

**特殊地域差异**：
- 山东地区：额外显示「民族」字段
- 上海地区：额外显示「参加工作日期」「执业机构」「执业地点」

**② 从业资料区**
- 主要执业医院（远程搜索）
- 职务类别（下拉选择）
- 全科医生（是/否）
- 开始执业日期
- 学历（上海/山东地区显示）
- 医保医师编码（32位以内，附链接到国家医保数据库）

**③ 证件/证书资料区（三证上传）**
1. **证件类型**：支持身份证（正反面）、护照（仅正面）等。管理员可选择证件类型
2. **资格证书**：需上传正反面照片，填写资质类别名称、资格证书编号、资格获取时间
3. **执业证书**：需上传正反面照片，填写执业证书编号、发证日期、执业范围（树形选择）
4. **其他相关证明**：最多上传5张

**④ CA认证状态**（功能开关控制显示）

### 第三步：审核决定

管理员在详情页底部有以下操作按钮：

**审核通过**：点击「确定」按钮，系统将自动保存所有资料修改（包括服务项目设置），并将医生状态从「待审核(10)」改为「已通过(20)」。医生即可在小程序端正常提供服务。

**驳回**：点击「驳回」按钮，弹出驳回原因输入框。管理员必须填写驳回原因（支持最多255字），**不能为空或纯空格**。提交后医生状态变为「不通过(30)」，驳回原因会展示在医生详情页的「审核结果」区域。

### 第四步（已通过医生）：修改资料

对于已审核通过的医生（flowStatus=20），管理员再次进入详情页可以修改资料：
- 如果医生已经接诊过（isClear > 0）或当前在线（onlineStatus=1），部分字段会被锁定不可编辑
- 修改完成后走「编辑资料」接口保存

### 服务项目管理（编辑模式）

审核页面同时支持配置医生的服务项目：

| 服务类型 | 说明 | 可控项 |
|---------|------|--------|
| 图文问诊 | type=1 | 开关、价格（元）、时长（小时/次） |
| 视频问诊 | type=2 | 开关、价格（元）、时长 |
| 咨询服务 | type=3 | 开关、价格（元）、时长 |
| 快速问诊（平台服务） | type=4 | 仅开关 |
| 快速续方 | type=5 | 开关、续方诊金（元） |
| 会诊服务 | type=6 | 开关、会诊诊金（元），有价格区间限制 |
| 医生复诊 | type=7 | 开关、复诊诊金（元），与处方一起收取 |
| 商城问诊（平台服务） | type=8 | 仅开关 |

**约束规则**：开启快速续方(type=5)时必须同时开启图文问诊(type=1)，否则系统提示"开启快速续方的同时请开启图文问诊"。

**医助/AI辅助功能**（预约模式下可配置）：
- 医助辅助接诊：自动匹配医助做诊前诊后服务
- AI预问诊：为医生配置AI助手预先收集病史信息

**其他设置**：
- 测试账号：勾选后仅用于测试，不备案、不上传诊疗数据
- 隐藏平台展示：勾选后医生不在全平台展示，只能通过邀请码找到
- 隐藏诊费设置：勾选后医生不可自行修改服务开启状态和诊费

### 接诊模式选择（预约功能开关控制）

- **实时接诊模式**：医生设置在线即可接诊，离线则提醒用户等待
- **预约接诊模式**：医生需设置排班，用户问诊需预约挂号
  - 过号是否接诊：是→超时仍可签到接诊；否→超时自动取消

### 工作类型选择

- 医生（type=1）
- 医助（type=2）：辅助医生接诊患者做诊前诊后服务

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 驳回时未填写原因或仅输入空格 | 弹窗提示"驳回原因不能为空且不能为空格"，不允许提交 |
| 表单验证不通过 | 提示"请完善资料" |
| 开启快速续方但未开启图文问诊 | 提示"开启快速续方的同时请开启图文问诊" |
| 已接诊/在线的医生 | 部分字段被锁定，不允许编辑 |
| 电话号码查看 | 默认脱敏显示，点击后调用接口获取完整号码（单次有效） |
| 编辑时发现医生flowStatus>10且已接诊/在线 | 姓名等部分字段不可编辑 |

## 关联功能

- 医生列表管理 → 审核通过后医生出现在医生列表中
- 医生排班管理 → 预约模式医生需在此配置排班
- 互联网医院信息 → 所属医院来源
- 科室管理 → 所在科室数据来源
- CA签章管理 → CA认证状态查看
