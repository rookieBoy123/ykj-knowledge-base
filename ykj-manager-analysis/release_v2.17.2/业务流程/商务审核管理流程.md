# 商务审核管理流程

> 文档版本 2026-07-04 | 面向运营管理员

## 概述

管理员查看待审核的商务人员资料并进行审批（通过或驳回）。商务人员是负责邀请医生入驻平台的业务人员。

## 触发条件

- **前置条件**：商务人员在小程序中提交了个人资料（姓名、手机号、身份证、银行卡等）
- **入口路径**：运营管理 → 商务管理 → 商务审核列表

## 步骤详解

### 第一步：查看待审核列表

进入页面后，系统自动加载所有"待审核"状态的商务申请。列表展示以下信息：
- 商务姓名
- 性别
- 手机号码（默认脱敏显示，点击后可查看完整号码）
- 提交时间
- 审核状态（待审核/驳回/审核通过）

可以通过商务姓名搜索过滤。

### 第二步：审核商务资料

点击某条记录的"审核"按钮，弹出审核窗口，展示该商务的全部提交资料：
- **基本信息**：商务姓名、性别、手机号码、身份证号码
- **银行信息**：开户行、银行卡号
- **联系地址**

底部审核区域包含：
- **审核结果**：通过 / 不通过（默认选中"不通过"）
- **备注**：必填，最多255字

### 第三步：提交审核结果

点击"保存"按钮提交审核。系统会：
- 如果审核通过，商务状态变为"审核通过"，该商务可以正常邀请医生
- 如果审核不通过，商务状态变为"驳回"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 列表为空 | 显示空列表（没有待审核申请） |
| 未选择审核结果就保存 | 默认值为"不通过"（flowStatus='30'），会直接驳回 |
| 手机号码脱敏 | 列表中手机号默认隐藏部分数字，点击后可查看完整号码（需调用解密接口） |

## 关联功能

- **商务管理**：已通过审核的商务在"商务列表"中管理，可进行增删改查
- **邀请医生管理**：审核通过后，商务可邀请医生入驻，其邀请的医生在"邀请医生管理"中查看
