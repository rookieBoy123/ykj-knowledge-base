# 医生列表管理流程

> 文档版本：v1.0 | 面向读者：运营管理员、平台管理员

## 概述

医生列表管理是管理后台中管理所有平台医生的核心功能。管理员可以在这里查看、搜索、管理所有医生的信息，包括账号启用/禁用、医生二维码邀请码管理、手机号变更、CA认证同步、会诊管理员设置等操作。

## 触发条件

- **前置条件**：已登录管理后台，具备医生管理权限
- **入口路径**：左侧菜单 → 医院管理 → 医生管理 → 医生列表

## 步骤详解

### 第一步：搜索筛选医生

管理员进入页面后，看到医生列表表格和搜索区域。支持以下搜索筛选维度：

- **医生姓名**：支持模糊搜索，最多输入5个字
- **工作类型**：可筛选"医生"或"医助"
- **科室**：树形下拉选择科室，支持搜索
- **所属医院**：远程搜索，输入关键词动态加载医院列表
- **职称**：从字典数据加载职称列表（如主任医师、副主任医师等）
- **账号状态**：启用/禁用
- **所属机构**：仅在开启了"机构管理"功能开关时显示
- **开启服务**：多选，可按开通的服务类型筛选（图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊）
- **是否会诊管理员**：是/否
- **关联商务**：仅在非严格模式下显示，用于搜索某商务关联的医生

### 第二步：查看医生列表

表格展示了每位医生的关键信息：

| 列名 | 说明 |
|------|------|
| 编号 | 医生ID |
| 医生姓名 | - |
| 所属医院 | - |
| 科室 | - |
| 职称 | - |
| 联系电话 | 默认脱敏显示，点击可查看完整号码（需二次验证权限） |
| 工作类型 | 医生/医助 |
| 开启服务 | 该医生已开通的服务类型，未开启的显示"未开启" |
| 所属机构 | 仅在机构功能开启时显示 |
| 关联商务 | 仅在非严格模式下显示 |
| 关注量 | 患者关注该医生的数量 |
| 在线状态 | 在线/离线 |
| 状态 | 启用/禁用开关 |
| CA认证状态 | 仅在CA功能开启时显示（未认证/认证中/已认证/认证失败） |
| CA用户同步状态 | CA的详细同步步骤状态 |
| 注册时间 | - |

### 第三步：对医生进行操作

每条医生记录的操作列提供以下功能：

1. **编辑**：跳转到医生信息详情页，编辑医生的完整资料
2. **邀请码**：弹窗显示医生的患者端邀请二维码，可点击"刷新二维码"生成新码
3. **变更手机号**：弹窗输入新手机号（11位），系统会实时校验手机号格式且不能与已有医生/药师重复。修改成功后刷新列表
4. **同步资料**：将医生信息同步到CA（电子签章）系统。仅CA功能开启时显示，且医生CA状态为"未同步"或"认证失败"时才可用
5. **去认证**：引导医生进行CA认证。仅CA功能开启时显示，且在医生CA状态为"认证中"时显示
6. **设为会诊管理员**：将医生设为远程会诊管理中心的管理员，该医生可以管理会诊相关事务
7. **取消会诊管理员**：撤销医生的会诊管理员权限，操作前会有二次确认弹窗

### 第四步：启用/禁用医生账号

在列表的"状态"列，管理员可以直接点击开关切换医生的账号状态：
- **启用**（状态值=10）：医生可以正常使用系统
- **禁用**（状态值=20）：禁用后该医生账号不能正常使用，切换前系统会弹出确认提示

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 变更手机号时输入已有号码 | 前端实时调接口校验"手机号码已存在，请重新输入"，无法提交 |
| 变更手机号时输入格式错误 | 提示"手机号格式错误"，无法提交 |
| 手机号为空 | 提示"手机号不能为空" |
| 获取医生二维码失败 | 提示"无法获取医生二维码，请联系管理员" |
| 取消会诊管理员时的误操作 | 弹出二次确认弹窗"你确认要取消该医生的会诊管理员权限吗？" |
| 禁用医生账号 | 弹窗提示"是否确认禁用该医生账号，禁用后该医生账号不能正常使用" |
| CA同步资料按钮不可用 | 仅当CA功能开启 且 医生CA状态为"未同步(1)"或"认证失败(4)"时才可点击 |
| 联系电话隐私保护 | 默认脱敏显示，点击后通过后台接口获取完整号码，每次切换会清除上一行的明文显示 |

## 关联功能

- **医生信息详情**：点击"编辑"跳转到医生信息编辑页，可编辑医生完整资料
- **CA认证系统**：通过"同步资料"和"去认证"操作联动
- **会诊管理**：设为/取消会诊管理员直接影响远程会诊的权限控制
- **科室管理**：科室筛选依赖科室树数据
- **机构管理**：机构筛选依赖机构功能开关
