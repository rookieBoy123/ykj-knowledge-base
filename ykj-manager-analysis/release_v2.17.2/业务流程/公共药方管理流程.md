# 公共药方管理流程

> 文档版本 2026-07-04 | 面向读者：药房管理员、运营人员

## 概述

管理中医公共药方模板，定义标准化的中药处方清单（药方名称+药品清单+剂量），供医生在开处方时快捷调用。支持药方的新增、编辑、查看详情和删除。

## 触发条件

- **前置条件**：具有药方管理权限（`med:prescriptions:write`）
- **入口路径**：管理后台 → 医院管理 → 药品信息 → 公共药方

## 步骤详解

### 第一步：查看药方列表

进入页面后展示所有已创建的公共药方，每条记录显示：
- **药方编码**：系统自动生成的唯一编号
- **药方名称**
- **药品**：药方中包含的药品名称列表（逗号分隔）
- **更新时间**
- **操作按钮**：详情、编辑、删除

列表支持按药方名称和药方编码筛选。

### 第二步：新增公共药方

点击「新增」按钮，弹出新增药方弹窗（960px宽），需要填写：

1. **关联药房**（可选）：下拉选择药房（系统预加载所有药房列表），药方与药房关联
2. **药方名称**（必填）：输入药方名称
3. **药方说明**（可选）：药方的用途和注意事项说明
4. **药品清单**（必填，至少添加一种药品）：
   - 从药品库中选择药品
   - 设置每种药品的用量（数量+单位）
   - 支持添加多种药品

> 新增和编辑共用同一个子组件 `editprescription.vue`，根据操作类型（add/edit）区分行为。

### 第三步：编辑公共药方

点击列表行的「编辑」按钮：
- 系统先查询药方详情（包含药品清单和单位信息）
- 弹窗回填现有数据，允许修改药方名称、药品清单、说明
- 药方编码不可修改（系统自动生成）

### 第四步：查看药方详情

点击「详情」按钮，弹出详情弹窗（700px宽），展示：
- **药方信息**：药方名称、药方类型（固定显示「中药类型」）、创建时间、药方说明
- **药方清单**：逐条展示药品名称和用量（格式：药品名 × 数量+单位）
- **单剂重量提示**：显示「单剂重量X克（非克药材和辅料未计入）」，只统计单位为「克」的药材

### 第五步：删除公共药方

点击「删除」按钮，弹出确认框「是否确认删除该公共药方?」，确认后执行删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 药方名称为空 | 提示「请输入药方名称」 |
| 药品清单为空 | 提示「请添加药品」，不允许保存 |
| 最后一页数据删除后 | 自动回退到上一页，避免显示空白页 |
| 删除确认取消 | 提示「已取消删除」，不执行删除 |
| 药方详情查看 | 单剂重量仅统计单位为「克」的药材，非克单位和辅料不计入 |

## 关联功能

- **处方管理**：医生开处方时可选择公共药方快速填入药品
- **药品库管理**：药方中的药品来自药品库，需先在药品库中录入
- **药房渠道管理**：药方关联到具体药房渠道
