# 医生信息管理

> 文档版本：v1.0 | 最后更新：2026-07-02 | 面向读者：运营人员、医院管理员

## 概述

管理后台支持完整的医生信息管理流程，包括新增医生、编辑医生资料、审核医生入驻申请（通过/驳回）、管理医生的服务项目配置（图文问诊、视频问诊、咨询服务、快速续方、医生复诊、会诊服务、快速问诊、商城问诊）、CA电子签名认证、以及工作类型和接诊模式设置。

## 触发条件

- **前置条件**：拥有管理后台登录权限，具有医生管理权限
- **入口路径**：
  - 新增医生：管理后台 → 医院管理 → 医生管理 → 医生列表 → 新增
  - 编辑医生：管理后台 → 医院管理 → 医生管理 → 医生列表 → 点击某个医生
  - 审核医生：管理后台 → 医院管理 → 人员审核 → 点击审核

## 步骤详解

### 一、基础信息

管理以下医生基础资料：

| 字段 | 说明 | 限制 |
|------|------|------|
| 头像图片 | 医生头像照片 | 建议200×200像素，1:1比例 |
| 医生姓名 | 真实姓名 | 最多5个字；编辑模式下审核通过后不可修改 |
| 性别 | 男/女 | 下拉选择 |
| 手机号码 | 联系电话 | 11位手机号 |
| 出生日期 | 出生年月日 | 日期选择器 |
| 所在科室 | 所属科室 | 树形下拉选择 |
| 医生职务 | 职称级别 | 下拉选择（主任医师/副主任医师/主治医师/住院医师/医师/医士） |
| 民族 | 民族信息 | 仅山东地区显示 |
| 擅长疾病 | 专业擅长描述 | 最多500字 |
| 医生简介 | 个人介绍 | 最多500字 |
| 对症询医 | 关联的疾病分类 | 最多10个，支持远程搜索 |

### 二、从业资料

| 字段 | 说明 |
|------|------|
| 主要执业医院 | 医生执业的主医院，支持远程搜索 |
| 职务类别 | 下拉选择职务类别 |
| 全科医生 | 是/否 |
| 开始执业日期 | 执业起始日期 |

**上海地区额外字段**：
- 参加工作日期
- 执业机构（可多选）
- 执业地点（最多50字）

**学历信息**（山东和上海地区显示）：
- 学历：下拉选择
- 学历取得时间（仅山东地区）

### 三、医保相关

- **医保医师编码**：填写国家医保医师代码，用于开具医保处方。该编码由国家医保医师代码数据库动态维护。页面提供跳转链接到医保代码数据库官网。

### 四、证件/证书资料

需要上传四类证件：

| 证件类型 | 上传要求 | 关联信息 |
|----------|----------|---------|
| 身份证件 | 正面+背面（护照仅正面） | 证件类型选择（身份证/护照/其他）+ 证件号码 |
| 资格证书 | 正面+背面 | 资质类别名称 + 资格证书编号 + 资格获取时间 |
| 执业证书 | 正面+背面 | 执业证书编号 + 发证日期 + 执业范围（树形选择） |
| 其他相关证明 | 最多5张 | 无额外字段 |

### 五、CA认证状态

显示医生的CA电子签名认证状态：
- **未认证**：无CA用户编号或认证中
- **认证中**：已提交认证申请待审核
- **已认证**：认证通过
- **已拒绝**：认证被驳回

操作按钮根据状态动态显示：
- 无CA用户编号或未认证/已拒绝 → "同步资料"按钮（同步医生资料到CA系统）
- 认证中 → "去认证"按钮（跳转进行认证操作）
- 已认证 → 无按钮（已完成）

### 六、审核结果

当医生申请被驳回（flowStatus=30）时，显示驳回原因。驳回操作时需填写驳回原因（最多255字）。

### 七、所属机构

如开启机构功能，可为医生分配所属机构（多选，支持远程搜索）。

### 八、工作类型选择

| 工作类型 | 说明 |
|----------|------|
| 医生 | 正常接诊医生 |
| 医助 | 辅助医生接诊患者，做诊前诊后服务 |

**医生类型的额外选项**：
- **测试账号**：勾选后仅可用于测试系统使用，不对信息进行备案，诊疗数据不上传
- **全平台不展示**：勾选后医生不在患者端全平台展示，也无法被搜索到，只有扫医生邀请码的患者才能显示医生资料并进行问诊

### 九、开启服务项目

为医生配置可提供的服务类型，每种服务独立开关和定价：

| 服务类型 | 配置项 | 说明 |
|----------|--------|------|
| 图文问诊 | 开关 + 价格（元）+ 时长（小时/次） | 通过文字+图片进行在线问诊 |
| 视频问诊 | 开关 + 价格（元）+ 时长 | 通过视频通话进行在线问诊 |
| 咨询服务 | 开关 + 价格（元）+ 时长 | 提供健康咨询 |
| 快速续方 | 开关 + 续方诊金（元） | 快速延续已有处方 |
| 医生复诊 | 开关 + 复诊诊金（元） | 复诊诊金与处方订单一起收取 |
| 会诊服务 | 开关 + 会诊诊金（元） | 有价格区间限制（系统配置） |

**平台服务**（无需医生自主设置价格）：
- **快速问诊**：系统自动匹配，免费，24小时服务时间，可开处方。医生加入后无法自主取消，仅后台可操作。开启"禁止开处方"功能后医生只能咨询服务不能开处方。
- **商城问诊**：用户在商城购买处方药时匹配该医生，诊金免费，24小时服务时间。医生无法自主取消，仅后台操作。

**辅助功能开关**：
- **医助辅助接诊**：启用后系统自动匹配医助接诊患者做诊前诊后服务
- **AI预问诊**：启用后系统自动为医生配置AI助手，辅助预问诊收集病史信息
- **隐藏诊费设置**：医生端不可修改服务开启状态和诊费设置

### 十、接诊模式选择

如开启预约挂号功能，可选择医生的接诊模式：

| 接诊模式 | 说明 |
|----------|------|
| 实时接诊 | 医生设置在线即可接诊，离线时提醒用户是否愿意等待 |
| 预约接诊 | 医生需设置排班，用户问诊需预约挂号 |

预约接诊模式下的额外设置：
- **过号是否接诊**：选择"是"——超过预约时间患者可签到，医生可继续接诊；选择"否"——患者不可签到，问诊将自动取消

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 审核医生时填写驳回原因 | 必须填写（最多255字），否则无法提交 |
| 医生姓名编辑限制 | 审核通过后（flowStatus ≠ 10）医生姓名不可修改（只读） |
| 身份证件类型为护照 | 仅需上传正面照片（隐藏背面上传入口） |
| 未开启远程会诊功能 | 不显示会诊服务配置 |
| 未开启预约挂号功能 | 不显示接诊模式选择 |
| 未开启对症询医功能 | 不显示疾病分类选择 |
| 未开启AI预问诊功能 | 不显示AI预问诊开关 |
| 未开启快速续方功能 | 不显示快速续方服务配置 |
| 非山东/上海地区 | 不显示学历、民族、执业地点等地区特有字段 |

## 关联功能

- **医生列表**：管理全院医生
- **人员审核列表**：审核医生入驻申请
- **医生排班**：设置医生出诊排班
- **排班模板**：管理排班模板
- **服务项目管理**：管理门诊服务项目类型
