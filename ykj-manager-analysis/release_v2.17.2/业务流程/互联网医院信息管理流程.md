# 互联网医院信息管理流程

> 文档版本：v1.0 | 面向读者：平台管理员

## 概述

互联网医院信息管理是配置平台所属互联网医院基础档案信息的页面。这些信息用于向监管部门申报，涵盖了互联网医院和对应实体医院的全部注册和许可信息，字段多达30+项，全部为必须填写项。

## 触发条件

- **前置条件**：已登录管理后台，具备系统配置权限
- **入口路径**：左侧菜单 → 医院管理 → 互联网医院信息

## 步骤详解

### 第一步：进入信息管理页面

管理员进入页面后，系统自动加载当前已保存的互联网医院信息（如果已填写过）。页面分为两大区域：
- **互联网医院信息**：填写互联网医院本身的注册信息
- **实体医院信息**：填写互联网医院对应的线下实体医疗机构信息

### 第二步：填写互联网医院信息

需要填写的核心字段包括（全部必填）：

| 字段 | 说明 | 字符限制 |
|------|------|---------|
| 机构标识 | 医疗机构唯一标识码 | 30字 |
| 机构名称 | 互联网医院全称 | 500字 |
| 单位隶属关系 | 下拉选择，如"中央属""省属"等 | - |
| 信息安全等级保护证书编号 | 等保证书编号 | 18字 |
| 机构类型 | 下拉选择 | - |
| 机构成立日期 | 日期选择 | - |
| 机构分类 | 下拉选择 | - |
| 经济类型 | 从字典加载，如国有、民营等 | - |
| 信息安全等级保护 | 下拉选择等级 | - |
| 机构分类管理类别 | 下拉选择 | - |
| 是否分支机构 | 是/否 | - |
| 机构所在地民族自治地方标志 | 是/否 | - |
| 部署平台 | 下拉选择如公有云、私有云等 | - |
| 是否具备双路供电或紧急发电设施 | 是/否 | - |
| 地址 | 三级联动省市区选择 + 详细地址输入 | 2000字 |
| 互联网医院网址 | 医院网站地址 | 200字 |
| 第二机构名称 | 别名/第二名称 | 500字 |
| 邮政编码 | 6位 | 6字 |
| E-mail | 邮箱地址 | 50字 |
| 电话号码 | 总机/查询台 | 50字 |
| 机房面积 | 数字输入 | 8字 |
| 第三方机构统一社会信用代码 | - | 100字 |
| 第三机构名称 | - | 500字 |
| 协议有效期 | 日期选择 | - |
| 机构描述 | 机构简介 | 5000字 |
| 备注 | 补充信息 | 500字 |

### 第三步：填写实体医院信息

| 字段 | 说明 | 限制 |
|------|------|-----|
| 统一社会信用代码 | 实体医院的统一社会信用代码 | 30字 |
| 实体医院名称 | 线下的实体医院名称 | 70字 |
| 许可证号码 | 医疗机构执业许可证号 | 50字 |
| 实体医院机构等级 | 下拉选择，如三级甲等 | - |
| 实体医疗机构级别 | 下拉选择 | - |
| 开办资金金额数 | 万元为单位 | 12字 |
| 法人代表/负责人 | 法人姓名 | 50字 |
| 许可项目名称 | 诊疗科目等 | 1000字 |
| 许可证有效期 | 日期范围选择（开始-结束） | - |

### 第四步：保存

点击"保存"按钮，系统校验所有必填项，验证通过后将数据提交保存。保存成功提示"保存成功"。

地址字段有特殊校验：必须同时选择了省市区（级联选择器）并填写了详细地址，否则会提示"请选择地址"或"请完善地址信息"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 有必填字段未填写 | 点击保存时表单校验高亮红色提示，无法提交 |
| 未选择省市区地址 | 校验规则拦截："请选择地址" |
| 未填写详细地址 | 校验规则拦截："请完善地址信息" |
| 未选择许可证有效期 | 校验规则拦截："请选择日期" |
| 点击"重置"按钮 | 弹出确认弹窗"是否确认重置？"，确认后重新加载服务器上的原始数据 |
| 互联网医院网址未填写 | 提示"请输入互联网医院网址" |

## 关联功能

- **字典管理**：机构类型、经济类型、信息安全等级等下拉选项来自系统字典
- **区域数据**：省市区三级联动数据来自系统内置行政区划数据
- **医保申报**：填写的机构标识和实体医院信息可能用于医保系统申报
