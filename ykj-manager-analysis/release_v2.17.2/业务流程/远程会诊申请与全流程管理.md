# 远程会诊申请与全流程管理

> 文档版本 v1.0 | 面向读者：运营人员、远程医疗管理员

## 概述

远程会诊管理用于协调不同医疗机构之间的医生联合会诊。管理员可以新增会诊申请、编辑草稿状态的申请、跟踪整个会诊流程（从申请到归档），并在流程中执行收费、取消、撤回、开始会诊、结束会诊、补充资料、归档等操作。

## 前置条件

- 操作者必须是「远程医疗管理员」角色，否则页面显示"当前页面仅供远程医疗管理员使用"
- 系统已配置会诊机构、会诊服务、科室字典

## 触发入口

管理后台 → 诊疗管理 → 远程医疗中心 → 远程会诊

---

## 步骤详解

### 第一步：会诊申请列表

页面展示所有会诊申请的列表，支持以下筛选条件：

| 筛选项 | 说明 |
|--------|------|
| 会诊单号 | 精确输入，最多20个字符 |
| 申请科室 | 模糊输入 |
| 申请医师 | 模糊输入 |
| 患者姓名 | 模糊输入 |
| 拟申请会诊机构 | 下拉选择 |
| 状态 | 下拉选择（见下方状态流转） |

列表展示：会诊单号、申请科室、申请医师、患者姓名/性别/年龄、拟申请会诊机构/服务/科室/医生、拟申请会诊时间、状态、会诊时间。

列表顶部有「新增」按钮。

### 第二步：新增会诊申请

点击「新增」弹出申请表单，需填写：

**患者信息区域：**
- 输入患者身份证号后点击「查询」，系统自动匹配患者姓名、性别、年龄
- 若无匹配记录，可点击「快速建档」跳转到患者建档页面
- 匹配成功后显示患者详情链接

**病情信息：**
- 病史摘要（必填，最多2000字）— 包括主诉、现病史、既往史、检验检查、治疗和服药情况
- 目前诊断（必填，最多200字）
- 会诊目的（多选）：从字典获取选项，如指导治疗、明确诊断等

**申请会诊信息：**
- 申请科室（树形选择，从科室字典加载）
- 主治医师（手动输入，最多5字）
- 拟申请会诊时间（日期时间选择器）
- 拟申请会诊科室（多选）
- 拟申请会诊机构（下拉，从机构列表加载）

**会诊服务与费用：**
- 选择机构后，自动加载该机构的会诊服务列表
- 选择服务后，显示服务单价和计费单位（如"100元/次"）
- 如果服务设置了「是否点名=是」，则显示「拟申请会诊医生」选择框，可指定具体医生

**文件上传：**
- 影像文件（支持rar/zip压缩包，最大200MB）
- 其他文件（支持jpg/png/pdf，最大10MB）

填写完成后，可点击「保存草稿」暂存，或「提交申请」正式发起。

### 第三步：编辑与删除

- **编辑**：仅状态为「草稿」「已撤回」「已拒绝」的记录可编辑
- **删除**：仅状态为「草稿」「已撤回」的记录可删除，删除后不可恢复

### 第四步：状态流转操作

会诊申请提交后进入以下状态流转，每个状态对应不同的操作按钮：

| 当前状态 | 可执行操作 | 触发条件 |
|----------|-----------|---------|
| 草稿（1） | 编辑、删除 | — |
| 待应答（2） | 撤回 | 申请已提交，等待被申请方回应 |
| 待收费（3） | 收费、取消 | 被申请方同意会诊 |
| 待开始（4） | 开始会诊、取消 | 已完成收费 |
| 进行中（5） | 进入会诊、结束会诊 | 会诊已开始 |
| 已结束（6） | 补充资料、归档 | 会诊结束 |
| 已归档（7） | 查看详情 | 资料已补全并归档 |
| 已取消（8） | 查看详情 | 管理员取消 |
| 已撤回（9） | 编辑、删除 | 待应答时撤回 |
| 已拒绝（10） | 编辑、取消 | 被申请方拒绝 |

**关键操作说明：**

- **收费**：点击后弹出确认框，确认后状态变更为「待开始」
- **取消**：需填写取消原因（必填，最多500字）
- **撤回**：待应答状态时可撤回申请，状态回到「已撤回」
- **开始会诊**：点击后系统自动调用接口，成功后跳转到会诊中心页面进行实时会诊
- **结束会诊**：确认后状态变为「已结束」
- **补充资料**：支持上传jpg/png/pdf/doc/xls等文件（单文件最大10MB），已上传的可继续追加
- **归档**：确认后状态变为「已归档」，归档后不可再补充资料

### 第五步：查看详情

点击「查看详情」弹出详情弹窗，展示会诊的完整信息，包括：
- 患者基本信息（姓名、性别、年龄、身份证号）
- 病情描述（病史摘要、目前诊断、会诊目的）
- 申请信息（科室、医师、机构、服务、费用）
- 流程日志（各状态变更的时间和操作人）
- 所有上传文件列表

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 非远程医疗管理员角色访问 | 页面显示"当前页面仅供远程医疗管理员使用"，隐藏所有功能 |
| 删除非草稿/已撤回状态的记录 | 操作按钮不显示 |
| 编辑非草稿/已撤回/已拒绝状态的记录 | 操作按钮不显示 |
| 取消会诊未填写原因 | 提交时校验报错"请输入取消会诊原因" |
| 补充资料未选择文件 | 提交时校验报错"请选择文件后再保存" |
| 归档前未上传全部资料 | 提示"归档后不再支持补充上传资料"，需确认后继续 |
| 影像文件超过200MB | 前端限制，超出无法上传 |
| 患者身份证号查询无结果 | 显示空状态，可点击快速建档 |

## 关联功能

- **会诊中心**：点击「进入会诊」跳转到远程会诊中心页面进行实时会诊
- **患者管理**：快速建档跳转到患者建档页
- **机构管理**：会诊机构列表依赖机构管理模块
- **会诊服务管理**：会诊服务列表依赖服务管理模块（服务名称、价格、是否点名等）
