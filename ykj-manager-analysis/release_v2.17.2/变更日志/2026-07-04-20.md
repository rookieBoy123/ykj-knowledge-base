# 变更日志 - 2026-07-04（第20轮）

## 新增业务流程

### 1. 人员审核管理流程
- 文档：`业务流程/人员审核管理流程.md`
- 概述：管理员审核申请入驻的医生资质资料，覆盖待审核/已通过/不通过三种状态的完整审核流程，包括三证上传审核、服务项目配置、工作类型选择、接诊模式设置等
- 覆盖页面：`hosp/doctormanage/personneAudit/index.vue` (463行) + `personneAudit/details/index.vue` (1731行)
- 关键发现：
  - 三状态流转：待审核(10) → 审核通过(20) / 驳回(30)
  - 驳回原因必填且不能为纯空格，255字上限
  - 已接诊/在线医生部分字段锁定不可编辑
  - 八种服务类型：图文问诊/视频问诊/咨询/快速问诊/快速续方/会诊/复诊/商城问诊
  - 快速续方依赖图文问诊开启（硬校验）
  - 支持两种接诊模式：实时接诊 vs 预约接诊（含过号处理）
  - 地域差异：山东（民族）、上海（工作日期/执业机构/执业地点）
  - 医保医师编码关联国家医保数据库
  - 电话号码脱敏显示，单次查看后重置

### 2. 业务开展情况管理流程
- 文档：`业务流程/业务开展情况管理流程.md`
- 概述：管理员按月记录互联网医院的15项经营指标数据，支持新增/编辑/删除，月份唯一性校验
- 覆盖页面：`hosp/businessinfo/index.vue` (438行)
- 关键发现：
  - 15项指标全部必填：人员3项 + 财务9项（万元）+ 其他3项
  - 月份唯一性校验：编辑时月份未变跳过校验，月份变更需重新校验
  - 删除有二次确认弹窗

### 3. 医生排班周视图管理流程
- 文档：`业务流程/医生排班周视图管理流程.md`
- 概述：管理员以周视图表格查看医生排班，支持上周/本周/下周快捷切换和自定义日期范围筛选，一键发布排班
- 覆盖页面：`hosp/doctormanage/doctorSchedule/index.vue` (351行)
- 关键发现：
  - 周视图表格：行=医生，列=周一至周日（今天标注"今天"）
  - 三种周切换方式：本周/上周/下周按钮 + 自定义日期范围
  - 发布后系统自动日均排15天后的班
  - 已发布（绿勾）/待发布（橙叹号）状态图例
