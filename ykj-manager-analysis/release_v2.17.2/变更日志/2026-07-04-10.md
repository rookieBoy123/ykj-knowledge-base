# 变更日志 - 2026-07-04（第10轮）

## 新增业务流程

### 1. 人员审核管理流程
- 文档：`业务流程/人员审核管理流程.md`
- 概述：管理员审核医生入驻申请（基础信息+从业资料+证件证书），通过或不通过。支持三标签页（待审核/不通过/已通过）、HIS同步医生、手机号脱敏保护
- 覆盖页面：`hosp/doctormanage/personneAudit/index.vue` (463行) + `personneAudit/details/index.vue` (1731行)
- 关键发现：三状态流转10→20/30；已审核后医生姓名变为只读；HIS模式下新增按钮变为「同步HIS医生」；手机号脱敏需调用memberPhone接口验证

### 2. 业务开展情况管理流程
- 文档：`业务流程/业务开展情况管理流程.md`
- 概述：按月填报互联网医院经营指标（职工数/资产/收支等15项），支持新增/编辑/删除
- 覆盖页面：`hosp/businessinfo/index.vue` (438行)
- 关键发现：月份唯一性实时校验（后端接口_nfExists）；金额单位均为万元；编辑时月份不再触发重复校验；多字段必填验证

### 3. 公共药方管理流程
- 文档：`业务流程/公共药方管理流程.md`
- 概述：管理中医公共药方模板（药方名称+药品清单+剂量），供医生开处方快捷调用。支持新增/编辑/详情/删除
- 覆盖页面：`hosp/druginfo/publicprescription/index.vue` (411行)
- 关键发现：药方编码系统自动生成；关联药房渠道；单剂重量仅统计单位为「克」的药材；详情展示所有药品名称和剂量；新增/编辑共用子组件

### 4. 医生排班管理流程
- 文档：`业务流程/医生排班管理流程.md`
- 概述：以周视图管理医生每日排班（上午/下午/晚班），支持上周/本周/下周切换、新增/编辑/停诊、发布到小程序端
- 覆盖页面：`hosp/doctormanage/doctorSchedule/index.vue` (351行) + `components/scheduleItem.vue` (443行)
- 关键发现：发布后系统每天自动排15天后的班；当天已过时间段不可修改；停诊分全天和指定班次；未保存关闭弹窗触发二次确认

### 5. AI医生管理流程
- 文档：`业务流程/AI医生管理流程.md`
- 概述：基于模板医生创建AI问诊机器人分身，管理AI医生的基本信息、问诊价格、启用/禁用和主页二维码
- 覆盖页面：`hosp/doctormanage/aidoctorlist/index.vue` (313行，TypeScript Composition API)
- 关键发现：AI医生关联「模板医生」；启用/禁用独立API和权限；主页二维码可刷新（旧码失效）；手机号脱敏调用memberPhone接口
