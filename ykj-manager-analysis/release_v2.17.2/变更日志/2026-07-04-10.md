# 变更日志 - 2026-07-04（第10轮）

## 新增业务流程

### 1. 疾病分类管理流程
- 文档：`业务流程/疾病分类管理流程.md`
- 概述：管理员维护患者端"对症询医"的症状分类，支持增删改查、排序（上移/下移/置顶/置底）、显示/隐藏切换
- 覆盖页面：`operate/article/diseaseCategory/index.vue` (265行) + `components/form.vue` (192行)
- 关键发现：症状名称最多10字；新增自动排最后；患者端最多显示8条；删除时级联清除医生擅长病种关联；排序通过sort整数值实现

### 2. 经典名方管理流程
- 文档：`业务流程/经典名方管理流程.md`
- 概述：管理员维护中医经典名方百科内容，四个富文本详情标签页（基本用法/药物作用/用药方法/注意事项），支持增删改查和批量删除
- 覆盖页面：`medicalEncyclopedia/classicFamousPrescriptions/index.vue` (283行) + `addOrEdit/index.vue` (269行)
- 关键发现：字典驱动下拉（处方类型/适应症/书籍出处来自数据字典）；四个详情标签页全部必填；提交时自动切换未填标签页；拼音自动生成

### 3. 中药图鉴管理流程
- 文档：`业务流程/中药图鉴管理流程.md`
- 概述：管理员维护中药图鉴百科内容，结构与经典名方类似但字段有差异（药味多选、无书籍出处）
- 覆盖页面：`medicalEncyclopedia/chineseMedicineIllustration/index.vue` (270行) + `addOrEdit/index.vue` (250行)
- 关键发现：药味多选（酸苦甘辛咸）；字典驱动（zytj-yw/zytj-syz）；封面图可选（与经络穴位不同）

### 4. 经络穴位管理流程
- 文档：`业务流程/经络穴位管理流程.md`
- 概述：管理员维护经络穴位百科内容，使用弹窗表单（非页面跳转），封面图必填
- 覆盖页面：`medicalEncyclopedia/meridiansAcupoints/index.vue` (283行) + `AddOrEdit.vue` (179行)
- 关键发现：封面图必填（与经典名方/中药图鉴不同）；弹窗模式；新增后回到首页，编辑后保持当前页

### 5. 任务执行日志查询流程
- 文档：`业务流程/任务执行日志查询流程.md`
- 概述：查看所有定时任务的执行历史，包括计划时间、实际时间、耗时、成功/失败状态，支持按任务和状态筛选
- 覆盖页面：`systemmanage/scheduler/logs/index.vue` (169行)
- 关键发现：失败记录悬停可查看错误原因；可从定时任务管理页跳转携带筛选参数；日志独立存储不依赖任务定义
