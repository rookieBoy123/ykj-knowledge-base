# 变更日志 2026-07-05（第6轮）

## 新增业务流程（5篇）

### 1. 问诊列表查询流程
- **文件**：`业务流程/问诊列表查询流程.md`
- **来源**：`therapy/outpatientserviceorder/inquiryListQuery/index.vue`（647行）
- **概要**：问诊订单的多维筛选与数据导出工具。覆盖10+种筛选维度（服务类型/问诊流程/问诊状态/支付状态/医生姓名/患者姓名/所属团队等），支持导出Excel（限制三个月跨度），患者姓名和手机号采用单次脱敏查看机制，平台服务费和医生收益仅在问诊结束+已支付时显示。

### 2. 评价审核管理流程
- **文件**：`业务流程/评价审核管理流程.md`
- **来源**：`therapy/outpatientserviceorder/commentlist/index.vue`（388行）
- **概要**：患者评价的审核与管理。三状态流转（待审核→审核通过/不通过），显示开关独立控制前端展示，快速通过自动开启显示，详情弹窗支持查看评分数/标签/内容，支持删除操作。

### 3. 专科实践管理流程
- **文件**：`业务流程/专科实践管理流程.md`
- **来源**：`hosp/specialtyPractice/index.vue`（318行）
- **概要**：互联网医院执业许可科目配置。关联国家标准科室编码字典（ksbm），登记执业许可发证/生效/到期三个关键日期，科目编码新增后不可修改。

### 4. 药房药品库管理流程
- **文件**：`业务流程/药房药品库管理流程.md`
- **来源**：`hosp/pharmacy/drugmanage/index.vue`（455行）
- **概要**：药房侧药品信息管理，区分成药/中药两种类型（字段体系和操作不同），支持Excel批量导入导出（含失败详情逐条展示），可从标准药品数据库同步更新，路由守卫实现从编辑页返回时保留搜索状态。

### 5. 标准科室诊断对照管理流程
- **文件**：`业务流程/标准科室诊断对照管理流程.md`
- **来源**：`hosp/stdManage/stdDeptDiagList/index.vue`（274行）+ `hosp/stdManage/stdMedDiagList/index.vue`（316行）
- **概要**：科室-诊断对照（科室关联ICD诊断编码）和药品-诊断对照（药品通用名关联诊断）两个子功能。诊断下拉支持远程搜索，药品新增时处理重名合并逻辑（提示用户选择合并或放弃）。
