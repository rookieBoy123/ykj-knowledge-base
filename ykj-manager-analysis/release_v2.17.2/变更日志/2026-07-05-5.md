# 变更日志 - 2026-07-05（第5轮）

## 新增业务流程（5篇）

### 1. 开放平台机构管理流程
- **概述**：管理外部合作机构的注册、密钥管理、启停控制。含机构ID自动生成、审方配置冲突校验、密码右键复制等。
- **来源文件**：`operate/organizationManage/index.vue`（384行）
- **关键发现**：
  - 机构密钥和回调密钥列支持右键自动复制，无需手动选中文本
  - 自动审方必须配置审方药师（否则提交失败）
  - 自动审方与CA签章互斥（CA必须用平台流程审方）
  - 需要药品下单时必须选择药房
  - 表单中有11个配置项，涵盖审方设置、CA签章、药品下单、回调地址等

### 2. 会员权益使用与管理流程
- **概述**：查看和管理会员持有的健康服务包权益，支持按生效/失效状态切换、手机号单次脱敏查看、手动扣减服务次数。
- **来源文件**：`healthServicePackage/memberBenefits/index.vue`（272行）
- **关键发现**：
  - 手机号脱敏显示，点击"查看"仅显示当前行完整号码（其他行自动隐藏）
  - 不限次数权益（times=-1）扣减按钮置灰
  - 剩余次数=0时扣减按钮置灰
  - 已失效权益标签页隐藏操作列
  - 扣减操作不可逆，需二次确认

### 3. 药师电子签名上传流程
- **概述**：药师在审方工作台上传个人电子签名图片，含图片格式/大小校验、2:1固定比例裁剪、裁剪后自动上传。
- **来源文件**：`recipemanage/recipecheckpharmacist/enteringqm/index.vue`（403行）
- **关键发现**：
  - 图片格式仅支持JPG/PNG，大小限制2MB
  - 裁剪框固定2:1宽高比（不可调整），鼠标滚轮缩放图片
  - 未上传签名时提交会提示"签名必须上传！"
  - 药师信息为空时显示"暂无数据"空状态
  - 展示药师基本信息（姓名/性别/手机/邮箱）+ 身份证和资格证书照片

### 4. 问答题库管理流程
- **概述**：独立于问卷的题目库管理，支持单选/多选/问答/上传图片四种题型，含完整CRUD和批量删除。
- **来源文件**：`questionnaireManage/questionLib/index.vue`（285行）+ 子组件 `addQuestion.vue`（81行）+ `editQuestion.vue`（94行）
- **关键发现**：
  - 题目编码自动生成：TM+年月日时分秒毫秒
  - 题目名称唯一（重名提示"已存在"），最长100字
  - 单选/多选至少2个选项，问答/上传图片不需要选项
  - 编辑时题目类型不可修改
  - 批量删除未选择时提示"请至少选择一条数据"
  - 翻页超出范围自动回退上一页

### 5. 远程会诊审核管理流程
- **概述**：审核合作机构提交的专家会诊申请，双栏布局（左侧列表+右侧详情），审核后自动切换Tab。
- **来源文件**：`telemedicine/telemedicinecheck/index.vue`（252行）
- **关键发现**：
  - 三状态标签页：待审核（蓝）/已通过（绿）/已拒绝（红）
  - "待签名"中间状态：审核通过但签章未完成
  - 左侧列表无限滚动加载（v-infinite-scroll）
  - 审核后自动切换Tab并刷新列表
  - 医生姓名搜索限制5个字
