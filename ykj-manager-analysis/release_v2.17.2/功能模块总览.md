# 管理后台功能模块总览

> 文档版本：v1.0 | 最后更新：2026-07-02 | 面向读者：运营人员、产品经理

## 系统概述

管理后台是互联网医院运营管理平台，供医院管理员、运营人员使用，支持对医生、患者、订单、处方、问诊、会诊、药品、财务等进行全面管理。

---

## 一、医院管理（hosp）

管理医院基础信息和医生资源。

### 1.1 医生管理（doctormanage）
- **医生列表**：查看和管理全院医生信息，支持新增、编辑、审核
- **人员审核**：审核医生入驻申请，通过/驳回并填写原因，支持查看身份证、资格证书、执业证书等证件
- **AI医生列表**：管理AI辅助医生
- **医生排班**：设置医生出诊时间和号源
- **排班模板**：创建和管理排班模板

### 1.2 患者管理（patient）
- **患者列表**：查看和管理患者信息
- **患者详情**：查看患者基本信息、病历、处方记录
- **病历详情**：查看患者的完整病历、病史信息、处方建议、会诊记录
- **量表评估详情**：查看患者的量表测评结果

### 1.3 药品管理（druginfo / pharmacy）
- **药品信息**：管理药品基础信息和分类
- **药品管理**：药品上架/下架、编辑药品详情
- **黑名单**：管理禁售药品
- **中药剂量**：管理中药剂量配置
- **中药煎法**：管理中药常规煎法
- **特殊煎法**：管理药品特殊煎法
- **药厂列表**：管理药品生产厂家
- **渠道管理**：管理药品配送渠道
- **公共处方**：管理公共处方模板

### 1.4 其他医院管理
- **机构管理**：管理互联网医院机构信息
- **设备信息**：管理医院设备
- **科室管理**：管理科室和部门配置
- **药师列表**：管理审方药师
- **医院配置**：系统级医院参数配置
- **地址管理**：管理医院地址信息
- **单位配置**：管理计量单位

---

## 二、诊疗管理（therapy）

管理在线诊疗业务。

### 2.1 门诊服务订单（outpatientserviceorder）
- **订单列表**：查看所有问诊订单，支持按问诊编号、服务类型、问诊流程、状态、支付时间等筛选
- **问诊列表查询**：专门的问诊订单查询页面
- **评价列表**：查看患者对问诊服务的评价

### 2.2 处方管理（recipemanage）
- **处方列表**：查看和管理所有处方
- **药师审方**：药师审核处方，确认药品合理性

### 2.3 远程会诊（remoteMedicalCenter）
- **会诊列表**：查看和管理远程会诊申请
- **远程会诊**：远程会诊管理中心（仅远程会诊管理员可用）
- **会诊中心**：进入会诊的实时操作页面
- **受邀机构管理**：管理可参与远程会诊的医疗机构
- **医生管理**：管理可参与会诊的医生
- **服务管理**：管理会诊服务配置

### 2.4 问诊科室（inquiryOffice）
管理问诊科室的配置

### 2.5 其他诊疗管理
- **服务项目管理**：创建和管理门诊服务项目（含排班、价格、服务方式等）
- **量表管理**：创建和管理医学量表/测评问卷
- **问卷管理**：创建和管理调查问卷
- **随访管理**：管理患者随访计划和执行
- **检查报告**：管理检查/检验报告

---

## 三、财务管理（financemanage）

管理平台的财务收支。

- **财务面板**：财务数据总览仪表盘
- **收款列表**：查看所有收款记录
- **发票管理**：开具和管理发票
- **发票查询**：查询已开发票
- **医保结算管理**：管理医保结算流程
- **报销列表**：管理报销申请和审核
- **药品销售统计**：药品销售数据分析
- **药品订单统计**：药品订单数据统计
- **收益管理**：医生/平台收益统计

---

## 四、运营管理（operate）

管理平台日常运营。

### 4.1 内容管理（article）
- **文章列表**：管理健康科普文章
- **Banner管理**：管理首页轮播图
- **推荐医生**：管理首页推荐医生
- **视频管理**：管理健康视频内容
- **健康咨询**：管理健康资讯内容
- **协议管理**：管理用户协议/隐私政策

### 4.2 业务管理（businessmanage）
- **业务列表**：管理平台开通的业务线
- **业务审核列表**：审核机构申请开通的业务
- **业务邀请医生**：邀请医生加入业务线

### 4.3 其他运营功能
- **专家团队**：管理专家团队和成员
- **反馈管理**：查看和处理用户反馈
- **会员管理**：管理平台会员
- **运费管理**：设置药品配送运费规则
- **健康服务包**：管理健康服务套餐
- **不良事件**：记录和管理不良事件
- **医学科普**：管理医学科普/食疗内容
- **费用项目管理**：管理收费项目
- **方案设计**：创建和管理运营方案
- **话术设置**：管理客服话术
- **机构管理**：管理合作机构

---

## 五、系统管理（systemmanage）

管理系统配置和运行参数。

- **字典设置**：管理系统数据字典（下拉选项等）
- **文件日志**：查看文件操作日志
- **医保配置**：配置医保相关参数
- **调度管理**：管理定时任务
- **系统设置**：系统级参数配置

---

## 核心概念说明

| 概念 | 说明 |
|------|------|
| 图文问诊 | 医生通过文字+图片方式为患者提供在线问诊服务 |
| 视频问诊 | 医生通过视频通话方式为患者提供在线问诊服务 |
| 快速问诊 | 系统自动匹配医生，免费24小时问诊服务 |
| 快速续方 | 患者快速申请延续已有处方 |
| 医生复诊 | 患者预约特定医生进行复诊 |
| 远程会诊 | 多个医疗机构的医生共同为患者会诊 |
| 慢病医保 | 针对慢性病患者的医保复诊服务 |
| 预约挂号 | 患者预约医生的排班时段进行问诊 |
| CA认证 | 医生电子签名认证，用于开具电子处方 |
| AI预问诊 | AI助手在医生接诊前收集患者病史信息 |
