# 源康健互联网医院 管理后台 — 常见管理操作

> 版本：v2.17.2 | 面向读者：客服 / 运营 / 产品经理

以下是后台管理员最常遇到的 10 个操作场景及其操作流程。

---

## 1. 新增后台管理账号

**适用角色**：超级管理员 / 系统管理员

**操作路径**：系统管理 → 用户管理

**操作步骤**：
1. 进入「系统管理」→「用户管理」页面
2. 点击「新增」按钮
3. 填写新用户的基本信息（姓名、手机号、登录密码等）
4. 为新用户分配角色（可分配多个角色，如同时是药师和财务）
5. 点击「保存」，账号即创建成功
6. 通知该用户使用手机号登录对应端（管理端或药房端）

**注意事项**：
- 密码需包含大写字母、小写字母和数字，长度8-16位
- 管理端和药房端的登录入口不同，确保告知用户正确的登录地址

---

## 2. 修改平台运行参数（系统配置）

**适用角色**：超级管理员 / 系统管理员

**操作路径**：系统管理 → 系统配置

**操作步骤**：
1. 进入「系统管理」→「系统配置」页面
2. 页面按分类展示所有可配置项（如"问诊设置""药房设置""微信小程序""医保配置"等），通过两级选项卡切换
3. 找到需要修改的配置项，直接修改数值或切换开关
4. 点击页面顶部的「保存」按钮
5. 系统保存后立即生效

**常用配置示例**：
- 问诊设置：问诊开放时间、默认问诊时长、图文/视频问诊最低时长
- 功能开关：开启/关闭在线商城、随访功能、量表功能、远程会诊等
- 药房设置：默认药房、是否启用第三方药房

**回滚操作**：
- 点击「修改历史」按钮，可查看所有历史修改记录
- 每条记录显示修改时间、操作人、修改前后的值
- 点击「还原」可回滚到历史版本

---

## 3. 查看和审核医生注册申请

**适用角色**：运营人员（有医生管理权限的角色）

**操作路径**：医院管理 → 医生管理 → 人员审核

**操作步骤**：
1. 进入「医院管理」→「医生管理」→「人员审核」页面
2. 查看待审核的医生列表
3. 点击某个医生，查看其提交的详细信息（执业证照、职称、科室、所属医院等）
4. 审核通过：点击「通过」按钮
5. 审核不通过：点击「驳回」，填写驳回原因
6. 审核通过后，该医生即可在平台上线接诊

**注意事项**：
- 审核时需要重点核对医生的执业资质证照是否真实有效
- 驳回时应写明具体原因，方便医生修改后重新提交

---

## 4. 查看患者信息和病历

**适用角色**：运营人员、客服

**操作路径**：医院管理 → 患者管理

**操作步骤**：
1. 进入「医院管理」→「患者管理」页面
2. 左侧显示患者列表，支持按姓名、标签、性别筛选
3. 点击某个患者行，右侧展示该患者详细信息
4. 患者详情中包含多个标签页：
   - **基本信息**：姓名、性别、年龄、手机号、标签
   - **病历记录**：该患者的线上问诊病历
   - **检查报告**：上传的检查/检验报告
   - **随访记录**：随访任务完成情况
   - **消息沟通**：与医生的历史聊天记录
   - **量表评估**：填写的量表评估结果
5. 还可以查看患者的处方记录、报销记录等

**特别说明**：
- 后台还能通过"病历查看"功能（独立页面），在线下场景中快速调取患者线上病历

---

## 5. 查看和处理用户意见反馈

**适用角色**：运营人员

**操作路径**：运营管理 → 意见反馈

**操作步骤**：
1. 进入「运营管理」→「意见反馈」页面
2. 查看用户提交的反馈列表（含反馈内容、提交时间、用户信息）
3. 点击某条反馈，进入详情页
4. 查看用户描述的问题和建议
5. 在"处理反馈"区域填写处理意见和结果
6. 点击「提交」，完成反馈处理

---

## 6. 管理C端首页装修

**适用角色**：运营人员

**操作路径**：运营管理 → 装修管理

**操作步骤**：
1. 进入「运营管理」→「装修管理」页面
2. 可创建新的装修页面或编辑已有页面
3. 在可视化编辑器中，通过拖拽方式添加首页组件：
   - Banner轮播图（上传图片、设置跳转链接）
   - 医生列表（选择展示哪些医生）
   - 视频模块（添加健康科普视频）
   - 文章推荐（选择健康资讯文章）
   - 菜单导航（配置快捷入口图标）
   - 底部Tab导航
   - 服务包推荐等
4. 每个组件可单独配置样式、内容、是否显示
5. 配置完成后保存，C端小程序/APP首页即更新

---

## 7. 管理药品黑名单

**适用角色**：管理员、药房管理员

**操作路径**：医院管理 → 药品信息 → 药品黑名单

**操作步骤**：
1. 进入「医院管理」→「药品信息」→「药品黑名单」页面
2. 查看当前黑名单列表
3. 添加药品到黑名单：选择药品、填写拉黑原因
4. 列入黑名单的药品，医生在开处方时将无法选择
5. 也可将药品从黑名单中移除

**典型场景**：某药品出现质量问题、被药监局通报、库存不足需要暂停使用等情况。

---

## 8. 药师审核处方

**适用角色**：药师（管理端） / 药房药师（药房端）

**操作路径**：
- 管理端：诊疗服务管理 → 处方管理 → 药师审方
- 药房端：药房管理 → 处方审核

**操作步骤**：
1. 进入审方页面，查看待审核的处方列表
2. 点击某个处方，查看处方详情（患者信息、诊断、药品清单、用法用量等）
3. 药师逐项核对：
   - 诊断与药品是否匹配
   - 药品剂量是否合理
   - 是否有药物相互作用风险
   - 是否有禁忌症
4. 审方结果：
   - **通过**：点击「审核通过」，处方进入调配环节
   - **不通过**：填写审核意见，处方退回给医生修改
5. 如果药房端配置了药师签名，审核通过的处方会自动附加药师电子签名

---

## 9. 处理在线问诊（医生接诊）

**适用角色**：医生（通过问诊工作台操作）

**操作路径**：诊疗服务管理 → 问诊工作台

**操作步骤**：
1. 进入「问诊工作台」页面
2. 左侧问诊单列表显示待接诊的患者（按状态分类：待接诊、进行中、已完成等）
3. 点击某个问诊单，中间聊天区域加载与患者的沟通记录
4. 可与患者进行图文、语音、视频沟通
5. 右侧展示问诊详情、历史病历、历史处方
6. 诊断后点击「写病历」填写电子病历
7. 需要开药时点击「开处方」，弹出处方编辑窗口：
   - 选择药品（中药/西药）
   - 填写用法用量、天数
   - 可使用处方模板快速填充
8. 提交处方后，处方进入药师审核流程
9. 问诊完成后点击「结束问诊」

---

## 10. 导出数据报表

**适用角色**：管理员、财务、运营

**操作路径**：各数据列表页面

**操作步骤**：
1. 在任意有表格的页面（如患者列表、订单列表、销售统计等），通常页面顶部或表格上方有「导出」或「下载」按钮
2. 可使用筛选条件先过滤出需要的数据范围
3. 点击「导出」按钮
4. 系统生成Excel文件并自动下载到本地
5. 用Excel打开即可查看和分析数据

**支持导出的常见数据**：
- 患者信息列表
- 订单数据
- 药品销售统计
- 财务流水
- 处方记录
