# 专家团队管理流程

> 文档版本 1.0 | 面向运营人员

## 概述
运营人员在后台创建、维护专家团队信息，并将医生纳入团队，用于在小程序端展示专家团队形象，吸引患者预约。团队可配置分佣比例，团队成员按比例获得诊费和药品订单分成。

## 前置条件
- 已登录管理后台
- 拥有专家团队管理权限
- 已有注册医生可加入团队

## 步骤详解

### 第一步：进入专家团队列表
从运营菜单进入"专家团队"页面。页面展示所有已创建的团队列表，每条记录包含：编号、团队头像、团队名称、首席专家、团队简介、展示状态（显示/隐藏开关）、修改时间。

支持按"团队名称"和"展示状态"两维筛选查询，每页显示10条。

### 第二步：新增专家团队
点击"新增"按钮，进入团队详情编辑页。填写以下内容：
- **团队头像**：上传172×200px的图片（不超过2MB），未上传时显示默认头像
- **团队名称**：必填，不超过20字
- **首席专家**：从医生列表中选择，支持远程搜索
- **团队简介**：必填，不超过300字
- **服务内容**：使用富文本编辑器编辑，描述团队提供的服务
- **分佣设置**：
  - 诊费分佣(%)：设置每笔诊费分给团队成员的佣金比例，0-100%，支持两位小数
  - 药品订单分佣(%)：设置每笔药品订单分给团队成员的佣金比例，0-100%，支持两位小数
  - 两项分佣比例系统默认值从全局配置读取，管理员可按需求调整
- **显示状态**：仅新增时可设置——显示（患者端可见）或隐藏（患者端不可见）

点击"保存"，校验所有必填项通过后创建团队。

### 第三步：管理团队成员
在团队列表点击"管理成员"，进入成员管理页。页面顶部展示团队头像、名称和简介。列表展示团队当前所有医生成员。

**添加成员**：点击"新增"按钮，弹出选择医生弹窗，选中并确认后医生加入团队。加入后自动排在列表末尾。

**排序**：支持四种排序操作——排至最前、上移、下移、排至最后。排序影响小程序端展示顺序。

**查看详情**：点击"编辑"跳转到医生详情页查看完整信息。联系电话默认脱敏显示（中间四位星号），点击眼睛图标可查看完整号码。

### 第四步：展示状态管理
在团队列表可以直接切换"展示状态"开关：
- **显示→隐藏**：弹出确认提示"设置展示状态为'隐藏'后患者端将不显示该专家团队信息"，确认后隐藏
- **隐藏→显示**：直接切换，患者端重新可见

### 第五步：编辑与删除
- **编辑**：点击"编辑"进入详情页修改团队信息（不包含显示状态）
- **删除**：弹出二次确认弹窗"删除后数据不可恢复，您确认要删除该专家团队吗？"，确认后删除

## 权限控制
- 专家团队管理功能需后台授予权限
- 编辑医生详情需要跳转到医生管理模块，需要对应权限

## 异常处理
| 情况 | 系统处理方式 |
|------|------------|
| 未上传团队头像 | 自动显示默认头像 |
| 分佣比例输入非数字 | 前端校验拦截，提示"必须为数字" |
| 新增时首席专家列表为空 | 下拉框无选项，提示医院需先注册医生 |
| 删除已显示团队 | 二次确认弹窗防止误操作 |

## 关联功能
- 小程序患者端：首页查看专家团队、预约医生
- 医生管理：医生详情页查看团队归属
