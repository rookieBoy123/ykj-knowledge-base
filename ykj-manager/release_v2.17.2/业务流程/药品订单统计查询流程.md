# 药品订单统计查询流程

> 文档版本 2026-07-02 | 面向产品运营、客服

## 概述

药品订单统计页面为运营管理人员提供药品销售数据的统计分析功能，可按时间维度和机构维度查询订单总量、金额汇总等经营指标，帮助运营团队掌握药品销售趋势。

## 触发条件

- 运营管理人员登录后台
- 前置条件：拥有相应数据查看权限
- 入口路径：后台左侧菜单 → 财务管理 → 药品订单统计

## 步骤详解

### 第一步：选择筛选条件

支持以下筛选条件：

| 筛选项 | 说明 |
|--------|------|
| 日期范围 | 按订单创建时间筛选，支持时间段选择 |
| 机构/药房 | 按药房维度筛选 |

### 第二步：查看统计数据

系统根据筛选条件展示统计结果，包含：
- 订单数量
- 销售金额汇总
- 可按药房维度查看各渠道销售情况

### 第三步：导出数据（如支持）

如需进一步分析，可将统计数据导出为Excel文件。

## 关联功能

- 药品订单统计 → 交易明细查询（查看具体订单）
- 药品订单统计 → 药品销售统计（按药品维度分析）
