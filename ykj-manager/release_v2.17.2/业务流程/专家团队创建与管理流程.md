# 专家团队创建与管理流程

> 文档版本：2026-07-01 | 面向读者：运营管理员

## 概述

管理员通过后台创建和管理专家团队，包括设置团队基本信息、分佣比例、管理团队成员和排序展示。专家团队在小程序端以团队为单位向患者展示。

## 触发条件

- **前置条件**：登录管理后台，系统配置中开启了专家团队功能（expertTeamEnabled）
- **入口路径**：运营管理 → 专家团队

## 步骤详解

### 第一步：查看专家团队列表

进入页面后，系统加载所有专家团队记录，每页显示10条。列表展示：

| 列名 | 说明 |
|------|------|
| 编号 | 团队ID |
| 团队头像 | 圆形头像（自定义头像或默认头像） |
| 团队名称 | 团队的名称 |
| 首席专家 | 团队的负责人医生 |
| 团队简介 | 团队的简要介绍文字 |
| 展示状态 | 显示/隐藏开关，控制小程序端是否可见 |
| 修改时间 | 最后更新时间 |

### 第二步：筛选团队

- **团队名称**：按名称模糊搜索，最多30字
- **展示状态**：筛选「显示」或「隐藏」的团队

### 第三步：新增专家团队

点击「新增」按钮进入详情页，填写以下内容：

**基本信息：**
- 团队头像：上传图片（建议172×200px，最大2MB）
- 团队名称：文本输入，最多20字（必填）
- 首席专家：从医生列表中搜索选择（必填）
- 团队简介：文本域，最多300字（必填）
- 显示状态：创建时可选择「显示」或「隐藏」（编辑时不显示此选项）

**服务内容：** 使用富文本编辑器填写，支持图文混排。

**分佣设置：**
- 诊费分佣（%）：设置每笔诊费分给团队成员的佣金比例，范围0～100，支持两位小数
- 药品订单分佣（%）：设置每笔药品订单分给团队成员的佣金比例，范围0～100，支持两位小数

分佣比例的默认值从系统配置中读取（teamInquiryRatio和teamMedicineRatio）。

### 第四步：编辑专家团队

点击列表的「编辑」进入详情页，可修改除显示状态外的所有信息。保存后即时生效。

### 第五步：管理团队成员

点击列表的「管理成员」进入成员管理页，页面展示团队头像、名称和简介，下方列出所有团队成员。

**成员列表展示：**
- 医生姓名、所属医院、科室、职称
- 联系电话（脱敏，点击可查看明文）
- 开启服务（医生提供的服务类型）
- 状态（启用/禁用）

**添加成员：** 点击「新增」按钮打开弹窗，从医生库中选择医生加入团队。

**移除成员：** 点击「删除」按钮，二次确认后移除。注意：删除的成员不可恢复。

**排序调整：** 每个成员和团队本身都支持四种排序操作：
- 排至最前：将该项移到列表第一位
- 上移：向上移动一个位置
- 下移：向下移动一个位置
- 排至最后：将该项移到列表最后一位

### 第六步：显示/隐藏团队

在列表中的「展示状态」开关：
- 开启（显示）：团队在小程序端对患者可见
- 关闭（隐藏）：团队在小程序端不显示

切换隐藏时有二次确认弹窗：「设置展示状态为"隐藏"后患者端将不显示该专家团队信息，您确认要修改吗？」

### 第七步：删除团队

点击「删除」按钮，二次确认后删除。注意：删除后不可恢复。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 首席专家被禁用 | 团队列表中成员状态显示为「禁用」，但不影响团队本身的展示。建议更换首席专家 |
| 删除仍有成员的团队 | 删除前需先移除团队中的所有成员，否则可能产生数据不一致 |
| 分佣比例未设置 | 分佣比例为必填项（数字类型），默认从系统配置读取，管理员可根据实际需要调整 |
| 团队头像未上传 | 使用系统默认头像（team_default_avatar.png）替代 |
| 团队功能未开启 | 如果系统配置中expertTeamEnabled为false，专家团队菜单和相关列不会显示 |

## 关联功能

- **医生管理**：团队成员从医生库中选择，团队的首席专家也必须是平台已入驻的医生
- **小程序端展示**：患者在小程序中可查看专家团队信息（如启用了专家团队功能）
- **医生收益**：专家的分佣比例影响医生的收益计算
