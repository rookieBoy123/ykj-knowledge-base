# 健康服务包配置管理流程

> 文档版本 2026-07-02 | 面向读者：运营、产品

## 概述

运营人员在管理后台配置「健康服务包」（特色疗法套餐），包括设置服务包名称、包含的服务项目、销售价格，以及控制服务包的启用/禁用和首页推荐状态。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 健康服务包 → 服务包管理
- **前置条件**：具有运营管理权限

## 步骤详解

### 一、服务包列表查询

进入页面默认加载服务包列表，每页10条，支持分页浏览。

**筛选条件**：
- **服务包名称**：支持模糊搜索，最多30字
- **状态**：启用的服务包 / 禁用的服务包

**列表字段**：

| 字段 | 说明 |
|------|------|
| 服务包 | 服务包名称 |
| 服务项目 | 包含的服务项目名称串 |
| 销售价格（元） | 打包售价 |
| 状态 | 启用/禁用开关 |
| 是否推荐 | 是/否推荐开关 |
| 最后修改时间 | 最近一次编辑的时间 |
| 操作 | 编辑、删除 |

### 二、新增/编辑服务包

点击「新增」按钮或列表中的「编辑」跳转到新增/编辑页面（独立页面），可配置：
- 服务包名称
- 封面图片
- 包含的服务项目（多选，从已有服务项目库中选择）
- 销售价格
- 其他展示信息

### 三、启用与禁用

列表中的「状态」列为开关控件：
- **开启（启用）**：服务包在患者端小程序可见可购买
- **关闭（禁用）**：服务包在患者端不显示

切换为禁用时弹出确认提示：「禁用服务包后将在患者端不显示，您确认要禁用吗？」

### 四、推荐设置

「是否推荐」列为开关控件：
- **是（推荐）**：服务包在患者端首页展示
- **否（不推荐）**：首页不展示，但可通过搜索或分类找到

切换为不推荐时弹出确认提示：「设置是否推荐为否后患者端首页将不显示该服务包，确认要修改吗？」

### 五、删除服务包

点击「删除」弹出二次确认：「数据删除后不可恢复，您确认要删除吗？」确认后删除，列表刷新。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 删除操作 | 二次确认弹窗，防止误删 |
| 禁用服务包 | 二次确认弹窗，提示对患者端的影响 |
| 取消推荐 | 二次确认弹窗，提示首页不再展示 |
| 列表无数据 | 显示空列表 |

## 关联功能

- **服务项目管理**：服务包由多个服务项目组成，服务项目需先在其他菜单中配置
- **服务包订单管理**：查看和管理患者购买服务包的订单
- **患者端展示**：启用的服务包在患者端特色疗法列表展示
