# 消息通知模板管理流程

> 文档版本 2026-07-02 | 面向读者：运营人员

## 概述

管理员在后台统一管理所有自动通知消息的发送渠道和模板。系统支持对四类人群（患者、医生、商务、药师）通过三种不同渠道（小程序订阅消息、短信、企微机器人）发送通知。管理员可以针对每个通知场景独立开启或关闭某种渠道，并自定义消息模板内容。

## 触发条件

- **前置条件**：拥有后台管理权限（系统管理模块访问权限）
- **入口路径**：系统管理 → 消息通知管理

## 步骤详解

### 第一步：选择通知对象

页面顶部有四个标签页切换：
- **通知患者**：针对患者的消息通知（如问诊提醒、处方通知、支付结果等）
- **通知医生**：针对医生的消息通知（如接诊提醒、收益到账等）
- **通知商务**：针对商务人员的消息通知
- **通知药师**：针对药师的消息通知（如审核提醒）

### 第二步：查看通知列表

列表展示所有通知场景，每行包含：
- **通知名称**：如"问诊支付成功通知""处方审核通过通知"
- **场景描述**：说明该通知在什么情况下触发
- **渠道开关**：每个渠道有独立的启用/禁用开关
  - 小程序订阅：显示开关的条件是该场景支持订阅消息
  - 短信：显示开关的条件是该场景支持短信发送
  - 企微机器人通知：显示开关条件是场景支持企微通知

点击开关可即时启用或禁用某个渠道的通知，无需二次确认。

### 第三步：配置通知详情

点击某条通知的"配置"按钮，弹出配置窗口：

**渠道切换**：顶部标签可切换查看/编辑不同渠道的配置（小程序订阅/短信/企微机器人）。注意：只有该通知场景支持的渠道才会显示。

**通用配置**：
- 是否通知：开启/关闭该渠道的通知
- 自定义模板（仅短信渠道）：开启后可使用自定义模板ID
- 模板内容：预览当前使用的消息模板（不可直接编辑）

**渠道专属配置**：
- **小程序订阅消息**：显示跳转URL（含参数占位符说明，如`#{uniqueId}`代表业务唯一标识）、模板参数值
- **短信**：可开启自定义模板并输入模板ID、模板内容中的变量可编辑
- **企微机器人**：需填写webhook地址（500字限制）

### 第四步：同步小程序订阅消息

点击"同步小程序订阅消息"按钮，系统会从微信小程序平台拉取最新的订阅消息模板，同步到本地。同步成功后列表中的模板内容会自动更新。

### 第五步：保存配置

点击"确定"保存当前渠道的配置修改。页面上方的开关也可以直接修改即时生效。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 同步订阅消息失败 | 弹窗显示失败原因（如"没有合适的模板"） |
| 模板ID为空 | 提交时校验："请输入模板ID" |
| 模板内容为空 | 提交时校验："请输入模板内容" |
| 未选择通知状态 | 提交时校验："请选择状态" |
| 修改模板内容后自动提取参数 | 内容变化时自动解析`#{xxx}`格式的变量，生成对应的参数输入框 |

## 关联功能

- **系统配置**：小程序经营类目信息在页面顶部展示，影响订阅消息的可用模板
- **问诊流程**：支付成功、接诊、处方审核等节点自动触发对应的消息推送
- **微信小程序平台**：订阅消息模板需先在微信公众平台申请，再通过同步功能拉取
