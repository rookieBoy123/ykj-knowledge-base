# 门诊服务订单综合管理流程

> 文档版本 2026-07-02 | 面向产品运营、客服

## 概述

门诊服务订单管理是后台运营人员统一管理所有问诊服务订单的页面。支持多维度筛选查询、问诊对话记录（含IM消息和语音）查看、问诊视频回放、AI预问诊对话记录浏览、AI问诊报告查看以及知情同意书信息确认。

## 触发条件

- 运营管理人员登录后台
- 前置条件：拥有门诊服务订单查看权限
- 入口路径：后台左侧菜单 → 诊疗管理 → 门诊服务订单

## 步骤详解

### 第一步：多维度筛选查询

页面支持以下筛选条件（可组合使用）：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 问诊编号 | 文本输入 | 精确搜索，最多30字 |
| 服务类型 | 多选下拉 | 图文问诊/视频问诊/咨询服务/快速问诊/快速续方/医生复诊/商城问诊 |
| 问诊流程 | 多选下拉 | 在线问诊/线上复诊/慢病医保/预约挂号/快速开方 |
| 问诊状态 | 单选下拉 | 已预约/问诊中/问诊结束/取消问诊 |
| 支付时间 | 日期范围 | 按支付时间筛选 |
| 所属机构 | 文本输入 | 机构名称搜索（条件显示，开启组织架构后可见） |
| 问诊时间 | 日期范围 | 按问诊创建时间筛选 |
| 就诊时间 | 日期范围 | 按就诊时间筛选 |
| 有无病历 | 单选下拉 | 有/无 |

### 第二步：查看订单列表

列表展示所有符合条件的问诊订单，支持分页浏览。

### 第三步：查看问诊对话记录

点击某条订单的对话记录入口，打开弹窗（700px宽）：

**IM消息模式**（新一代问诊）：
- 展示完整的医患对话记录
- 支持语音消息播放（点击播放，支持停止）
- 消息按时间顺序排列
- 底部有水印保护（含操作人信息）

**普通消息模式**（老版问诊）：
- 展示图文问诊对话
- 支持语音消息播放

对话记录支持滚动加载历史消息。

### 第四步：查看问诊视频

切换到「问诊视频」标签页：
- 展示视频问诊的录制视频列表
- 每条视频显示：视频图标 + "问诊视频"标题 + 创建时间
- 点击「播放」打开视频播放器，支持在线播放

如果该问诊没有视频记录，显示空状态。

### 第五步：查看AI预问诊对话记录

如果问诊经过AI预问诊，可查看完整的人机对话记录（700px弹窗）：
- **AI消息**（左侧）：显示AI医生头像（带AI角标），支持普通文本和卡片两种格式
  - 卡片格式：展示标题、结构化字段信息（字段名+值）
  - 卡片底部可能有快捷按钮（如"在线复诊"引导按钮）
- **患者消息**（右侧）：显示患者头像，支持文本和图片（可点击放大）

### 第六步：查看AI问诊报告

AI完成的问诊报告包含以下区块：
- **主诉**：患者主要病情描述
- **病史信息**：现病史、既往史、过敏史、家族史、个人史、月经史、婚育史（仅显示有内容的字段）
- **体检**：辅助检查信息
- **AI诊断**：蓝色标签展示AI推荐的诊断名称，附带提示"仅供参考，具体以真实医生诊断为准"

### 第七步：查看知情同意书确认信息

点击查看患者在问诊前的知情同意书签署记录，弹窗展示：
- 确认人姓名
- 网络IP地址
- 设备信息
- 微信用户openId
- 同意书签署时间

底部提供「关闭」按钮。

## 异常处理

| 情况 | 系统处理方式 |
|-----|------------|
| 无视频记录 | 显示空状态图标 |
| 无AI预问诊记录 | 不显示对应操作入口 |
| 无知情同意书记录 | 不显示对应操作入口 |
| 对话记录加载中 | 显示loading状态 |

## 关联功能

- 门诊服务订单 → 问诊对话记录查看
- 门诊服务订单 → 问诊视频回放
- 门诊服务订单 → AI预问诊记录查看
- 门诊服务订单 → AI问诊报告查看
- 门诊服务订单 → 知情同意书查看
