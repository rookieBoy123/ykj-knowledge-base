# 药师电子签名录入与管理流程

> 文档版本 v1.0 | 面向读者：药师

## 概述
药师在审方时需要在处方上签署电子签名，该流程让药师可以查看自己的基本信息和相关证书，并上传手写签名图片。上传的签名会自动裁剪处理后保存，后续用于处方审核时的电子签章。

## 触发条件
- **前置条件**：已登录的药师账号
- **入口路径**：处方管理 → 药师审方 → 录入签名

## 步骤详解

### 第一步：查看基本信息
进入页面后，顶部卡片展示药师基本信息：
- 药师姓名
- 性别
- 手机号码
- 邮箱地址

若当前登录账号未关联药师信息，页面显示"暂无数据"的空状态提示。

### 第二步：查看相关证书
页面展示药师已上传的证书图片：
- **身份证**：显示身份证正反面图片
- **资格证书**：显示药师资格证书图片

证书图片以列表形式展示，可滚动查看。

### 第三步：上传电子签名
- 点击"点击上传签名"按钮，从本地选择图片文件
- **格式限制**：仅支持JPG/PNG格式，文件不超过2MB
- 不符合条件的文件会被拦截并提示"上传图片只能是jpg/png格式!"或"上传图片大小不能超过 2M!"
- 选择图片后进入**图片裁剪页面**：
  - 支持鼠标滚轮缩放图片
  - 支持拖动调整裁剪范围
  - 固定裁剪框比例为2:1（宽:高）
  - 裁剪完成后点击"确认"提交

### 第四步：提交签名
- 裁剪后的签名图片预览显示在页面上
- 签名上传是**必须操作**，未上传签名点击"提交"会提示"签名必须上传!"
- 确认无误后点击"提交"按钮保存

## 异常处理
| 情况 | 系统处理方式 |
|------|-------------|
| 未上传签名就提交 | 弹出黄色警告"签名必须上传!"，阻止提交 |
| 上传非图片格式 | 弹出红色错误"上传图片只能是jpg/png格式!" |
| 文件超过2MB | 弹出红色错误"上传图片大小不能超过 2M!" |
| 当前账号未绑定药师 | 页面显示空状态，无药师信息卡片展示 |
| 裁剪图片失败 | 裁剪弹窗可重新操作，取消后回到上传页面 |

## 关联功能
- **处方审核**：审核处方时自动附上该药师电子签名
- **处方列表**：查看处方时可看到药师签名信息
- **药师管理**：管理员可在药师列表中查看签名状态
