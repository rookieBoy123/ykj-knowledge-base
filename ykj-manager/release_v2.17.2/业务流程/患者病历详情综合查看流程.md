# 患者病历详情综合查看流程

> 文档版本 2026-07-03 | 面向读者：运营管理员

## 概述
这是管理后台中最完整的患者病历查看页面，从患者管理列表点击进入后，按分区展示患者信息、病史信息、就诊信息、会诊记录、处方建议，并可查看问诊对话记录。页面右侧提供导航菜单，支持快速跳转到各信息区域。

## 触发条件
- 管理员在患者管理中点击某个患者的"病历详情"入口

## 步骤详解

### 第一步：页面加载
页面加载后展示左右双栏布局：
- **左侧**：内容区域，展示各信息模块
- **右侧**：固定导航菜单（圆点+文字），支持点击跳转到对应区域

### 第二步：查看患者信息
**信息卡片**包含：
- 就诊卡号（HIS系统患者ID）
- 姓名、性别、出生日期、年龄（岁/月格式）
- 证件号码
- 监护人信息（如有）：姓名、关系、身份证号（已校验标签）
- 门慢医保复诊标签（如有）

### 第三步：查看病史信息
展示该患者在该次就诊中填写的全部病史：
- 主诉、现病史
- 既往史、家族史、个人史、过敏史
- 月经史、婚育史
- 中医四诊（脉象、舌象）——仅中医病历
- 治法——仅中医病历
- 治疗意见——仅西医病历
- 常见体征
- 辅助检查

### 第四步：查看就诊信息
- 就诊方式：线上就诊
- 首诊机构、首诊时间
- 门诊号
- 问诊时间、服务类型（图文/视频/咨询等）
- 科室名称
- 病情描述
- 病历资料（患者上传的图片，可点击放大预览；无资料显示"记录遗失/不在身边"）
- 主治医生：姓名 + 科室 + 职称 + 医院 + 三甲标记
- "问诊对话记录"按钮——点击弹出聊天记录窗口

### 第五步：查看问诊对话记录
点击"问诊对话记录"按钮弹出对话框：
- **聊天记录模式**：展示医生和患者的完整对话，支持文字、图片、语音消息
- **IM即时通讯模式**：对于使用腾讯IM的问诊，展示IM格式的对话
- 支持消息搜索功能
- 语音消息可点击播放

### 第六步：查看会诊记录
如果该就诊有关联的远程会诊：
- 列表展示每次会诊记录（会诊时间+状态标签）
- 会诊状态包括：会诊申请、会诊待备案、会诊待患者确认、会诊待开始、会诊进行中、待会诊报告、已完成
- 点击某条会诊记录可查看会诊详情

### 第七步：查看处方建议
展示该次就诊医生开具的处方：

**成药处方**：
- 诊断、辨病、辨证
- 药品明细：药品名称+规格×数量，规格、用法用量
- 处方医生
- 查看原始处方按钮
- 查看对应订单按钮（如有订单号）

**中药处方**：
- 诊断、辨病、辨证
- 药品：药材名称和用量列表
- 处方医生
- 医嘱禁忌：用药说明、用药时间、用药禁忌

### 第八步：右侧导航
右侧固定菜单提供快速跳转：
- 点击对应圆点+名称，页面自动滚动到目标区域
- 当前所在区域圆点高亮
- 随页面滚动自动更新当前高亮位置

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 某项病史未填写 | 显示"未填写" |
| 会诊记录为空 | 显示"暂无数据" |
| 处方建议为空 | 显示空状态图 |
| 病历资料为空 | 显示"记录遗失/不在身边" |
| 问诊对话记录为空 | 显示空列表 |
| 年龄为0且月龄为0 | 显示"未知" |
| 监护人无身份证号 | 不显示"已校验"标签 |

## 关联功能
- 患者管理——患者列表入口
- 门诊问诊订单管理——可跳转查看对应订单
- 远程会诊管理——会诊详情查看
- PDF阅读器——原始处方查看
