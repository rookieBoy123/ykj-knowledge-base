# 量表新增与发布管理流程

> 文档版本 1.0 | 面向运营/管理人员

## 概述
管理人员在后台创建和维护医学量表（评估问卷），用于医生在接诊或随访时让患者填写，系统根据患者的回答自动计算评分结果。量表支持草稿→发布→禁用的完整生命周期。

## 前置条件
- 已登录管理后台
- 拥有量表管理权限
- 已创建量表分类（如焦虑评估、抑郁评估等）

## 步骤详解

### 第一步：进入量表列表
从诊疗菜单进入"量表管理"页面。页面展示所有量表列表，支持三条件筛选：
- 量表分类：下拉选择已创建的分类
- 量表名称：输入搜索，不超过128字
- 创建时间：选择日期范围

列表展示每行：量表名称、所属分类、创建时间、启用状态（草稿/发布/禁用）、操作按钮。

### 第二步：新增量表
点击"新增量表"按钮，进入量表编辑页，分三个步骤填写：
1. **基本信息**：量表名称、分类、描述等基础信息
2. **题目信息**：添加题目（单选/多选/填空/上传等题型）、设置选项和分值
3. **评分标准**：设置评分区间和对应结果说明

### 第三步：发布量表
量表初建时为"草稿"状态。点击"发布"会先进行完整性校验：
- 系统检查题目列表和评分标准列表是否完整
- **不完整时**：弹窗提示"该量表有内容未填写，请确认填写完整再发布"，提供"完善量表"按钮跳转到编辑页，或"取消"返回列表
- **完整时**：直接发布，状态变为"发布"，量表在小程序端医生可见

### 第四步：启用/禁用量表
- **发布→禁用**：已发布的量表可切换为禁用状态，患者端不再可见，但不影响已完成的测评记录
- **禁用→启用**：禁用的量表可重新启用（注意：启用按钮点击后同样走发布流程，即完整性校验）
- **草稿→发布**：仅草稿状态可发布，同样需要完整性校验

### 第五步：排序
非草稿状态的量表支持排序操作（草稿不可排序）：
- **上移**：仅非第一条可操作（列表首条或非首页首条不可上移）
- **下移**：仅非最后一条可操作（列表末条不可下移）

### 第六步：编辑限制
- 仅"草稿"状态的量表可编辑（点击"编辑"按钮可见）
- 已发布和已禁用的量表不可编辑（编辑按钮隐藏）

## 缓存策略
从量表编辑页返回列表时：
- **从编辑页返回**：保留之前的搜索条件和页码
- **从其他页面跳入**：重置搜索条件并重新加载

## 异常处理
| 情况 | 系统处理方式 |
|------|------------|
| 发布时题目/评分标准未填写 | 弹窗引导前往完善，不执行发布 |
| 草稿状态的量表未编辑 | 编辑按钮正常显示，可继续编辑 |
| 已发布量表尝试编辑 | 编辑按钮隐藏，无法编辑 |
| 量表分类列表为空 | 分类筛选下拉框无选项 |

## 关联功能
- 量表分类管理：创建量表所属分类
- 医生端量表发送：医生在问诊/随访中选择量表
- 患者端量表填写：患者答题并查看结果
- 量表记录查看：患者档案中查看历史测评记录
