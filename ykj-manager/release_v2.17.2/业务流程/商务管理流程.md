# 商务管理流程

> 文档版本 2026-07-01 | 面向读者：后台管理员、运营人员

## 概述

管理员在后台管理平台的商务人员（地推/BD），包括新增、编辑、启用禁用、生成专属二维码、管理其邀请的医生，以及变更手机号等操作。商务人员通过自己的专属二维码邀请医生入驻平台。

## 触发条件

- 运营管理→商务管理菜单进入
- 需要添加新的商务推广人员
- 需要查看商务邀请医生的业绩

## 步骤详解

### 第一步：浏览商务列表

列表支持三维筛选：
- **商务姓名**：文本模糊搜索，最多20字
- **状态**：启用/禁用下拉选择
- **注册时间**：日期范围选择

列表展示字段包括：编号、商务姓名、性别、手机号码（脱敏显示）、身份证号、开户行、银行账号、详细地址、状态（启用/禁用开关）、注册时间。列表每页10条分页加载。

### 第二步：新增/编辑商务

点击"新增商务"按钮打开弹窗表单，包含8个字段：

| 字段 | 校验规则 |
|------|---------|
| 商务姓名 | 最多20字，必填 |
| 性别 | 下拉选择（字典：性别） |
| 手机号码 | 11位手机号格式校验，必填 |
| 身份证号码 | 最多18位 |
| 开户行 | 最多100字 |
| 银行账号 | 最多100字 |
| 详细地址 | 最多255字 |
| 状态 | 启用/禁用的单选开关 |

编辑模式下弹出相同的表单，字段自动回填当前数据。

### 第三步：启用/禁用

列表中的状态开关可即时切换商务的启用/禁用状态。禁用后该商务的邀请码将失效。

### 第四步：二维码管理

点击列表中的二维码按钮，弹出该商务的专属邀请二维码。二维码包含商务ID信息，医生扫码注册时会自动关联该商务。支持"刷新二维码"按钮重新生成。

### 第五步：邀请医生管理

点击"邀请医生"按钮，打开一个独立窗口，展示该商务邀请的所有医生列表。支持：
- 按医生姓名搜索
- 分页浏览
- 查看每位医生的药品订单销量统计

### 第六步：变更手机号

点击"变更手机号"按钮，弹出手机号编辑框。输入新手机号（11位格式校验），确认后立即生效。

### 第七步：删除商务

列表中的删除按钮可删除商务记录（不可恢复）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号格式不正确 | 表单校验提示"请正确填写电话号码" |
| 手机号已存在 | 保存时后端返回错误提示 |
| 未选择任何筛选条件 | 默认查询全部商务 |
| 二维码生成失败 | 显示错误提示 |

## 关联功能

- 关联：医生管理（商务邀请的医生列表）
- 关联：药品订单统计（商务名下医生的销量）
- 小程序端：医生扫码注册自动关联商务
