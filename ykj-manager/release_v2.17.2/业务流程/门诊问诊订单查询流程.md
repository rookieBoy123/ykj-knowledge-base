# 门诊问诊订单查询流程

> 文档版本 v1.0 | 面向读者：运营、客服、财务

## 概述

管理员在后台查询所有门诊问诊订单的完整信息，包括订单号、问诊单编号、服务类型、医生患者信息、费用明细、问诊和支付状态，支持多维度筛选和数据导出。

## 触发条件

- **前置条件**：拥有门诊服务订单查看权限
- **入口路径**：管理后台 → 诊疗管理 → 门诊服务订单 → 问诊列表查询

## 步骤详解

### 第一步：进入查询页

页面加载时自动完成：

- **默认时间范围**：自动设置为最近30天（当天往前推30天）
- **字典初始化**：加载专家团队列表（如已启用专家团队功能）
- **AI问诊适配**：如果系统开启了AI医生功能，筛选条件中会自动增加"AI问诊"流程选项和"AI图文问诊"服务类型选项
- **功能联动**：专家团队功能未开启时，隐藏"所属团队"筛选和列

### 第二步：多维度筛选

支持以下丰富的筛选条件：

- **订单编号**：精确输入，最多20字
- **创建时间**：日期范围选择器，默认最近30天
- **服务类型**：多选（图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊、AI图文问诊）
- **问诊流程**：多选（在线问诊、线上复诊、慢病医保、预约挂号、快速开方、AI问诊）
- **问诊状态**：单选（已预约、问诊中、问诊结束、取消问诊）
- **支付状态**：单选（未支付、已支付）
- **医生姓名**：模糊搜索，最多5字
- **患者姓名**：模糊搜索，最多5字
- **支付时间**：日期范围选择器
- **所属团队**：下拉选择专家团队（功能开关控制显隐）
- **问诊单编号**：精确输入，最多30字

### 第三步：查看订单列表

列表展示以下字段：

| 字段 | 说明 |
|------|------|
| 订单号 | 18位，可复制 |
| 问诊单编号 | 18位，可复制 |
| 服务类型 | 图文/视频/咨询/快速等 |
| 问诊流程 | 在线问诊/线上复诊/慢病医保等 |
| 医生姓名 | 接诊医生 |
| 所属团队 | 专家团队名称（功能开关控制显隐） |
| 患者姓名 | 默认脱敏显示（姓+*号），点击可逐条查看明文 |
| 联系方式 | 默认脱敏，点击可逐条查看明文（每次只显示当前行） |
| 创建时间 | 订单创建时间 |
| 订单支付时间 | 支付时间（未支付显示"-"） |
| 诊费(元) | 问诊费用 |
| 平台服务费 | 平台抽成金额 |
| 医生收益(元) | 医生实际收益 |
| 问诊状态 | 草稿/问诊中/问诊结束/取消问诊 |
| 支付状态 | 已支付/未支付 |

### 第四步：数据导出

- **导出功能**：点击"导出"按钮，将当前筛选条件下的所有结果导出为Excel文件
- **导出范围**：基于当前筛选条件的所有数据，不限于当前页

## 隐私保护

- **患者姓名脱敏**：默认只显示姓+星号（如"张**"），点击可查看当前行的明文
- **手机号脱敏**：默认隐藏，点击后逐条解密显示，同时清除其他行的明文（每次只显示一条）
- **访问控制**：页面加载时根据系统配置控制专家团队相关功能的可见性
