# 药品订单销售统计流程

> 文档版本 v1.0 | 面向读者：财务人员、运营管理者

## 概述

药品订单销售统计页用于查看和分析药品订单的销售数据，提供多维度筛选和金额汇总功能。页面顶部展示按支付渠道分类的汇总金额，列表展示每笔订单的详细收支信息（含微信支付、医保统筹、医保个账、线下支付、退款等分项）。

## 触发条件

- 前置条件：已登录管理后台，拥有相关数据查看权限
- 入口路径：左侧菜单 → 财务管理 → 药品订单统计

## 步骤详解

### 第一步：查看金额汇总

页面顶部自动显示当前筛选条件下的金额总览：

- 合计收入
- 微信支付
- 医保统筹（仅开启医保功能时显示）
- 医保个帐（仅开启医保功能时显示）
- 医保支付-微信支付（仅开启医保功能时显示）
- 线下支付
- 退款

所有金额单位为"元"，显示精确到分。

### 第二步：筛选查询

支持以下筛选条件：

- **订单支付日期**：日期范围选择器，默认最近 30 天
- **订单编号**：精确搜索，最多 30 个字符
- **订单类型**：下拉选择（选项来自字典数据 `ddlx`）
- **医生**：远程搜索下拉框，支持输入医生姓名模糊搜索，仅搜索已审核通过的医生（flowStatus=20）
- **所属团队**：下拉选择（仅当系统配置 `expertTeamEnabled` 开启时显示，关闭时该筛选项和列表列均隐藏）

### 第三步：查看订单明细列表

列表展示字段：

- 订单编号（200px 宽度）
- 订单类型
- 订单支付时间
- 医生姓名
- 所属团队（仅专家团队功能开启时显示）
- 患者姓名
- 订单总金额（元）
- 订单实付金额（元）
- 微信支付（元）
- 医保统筹（元，仅医保开启时显示）
- 医保个账（元，仅医保开启时显示）
- 医保支付-微信支付（元，仅医保开启时显示）
- 线下支付（元）
- 退款（元）

### 第四步：导出报表

点击列表上方"导出"按钮，弹出加载提示，生成 Excel 文件（文件名：订单销售统计.xls）后自动下载。导出数据不受分页限制，导出全部符合条件的记录。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 筛选无结果 | 列表为空，汇总金额显示为 0 |
| 医保功能未开启 | 列表不显示医保相关金额列，汇总不显示医保维度 |
| 专家团队功能未开启 | 不显示"所属团队"筛选和列 |
| 导出请求失败 | 关闭加载提示，用户可重试 |
| 医生输入无匹配 | 下拉列表为空 |

## 关联功能

- **财务看板**：看板柱状图展示每月交易总额，统计页提供明细级数据
- **交易流水采集**：可查看每笔交易的具体流水记录
- **药品管理**：可追溯到药品订单具体处方信息
