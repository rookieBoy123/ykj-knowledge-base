# 商务（业务员）管理与邀请医生流程

> 文档版本 v1.0 | 面向读者：后台运营人员

## 概述

后台运营人员可以管理商务推广人员（业务员）的账号，包括新增、编辑、启用/禁用、删除，以及查看业务员邀请的医生列表和销售数据。每位业务员有专属的邀请二维码，医生扫码注册后自动关联到该业务员名下。

## 触发条件

- **前置条件**：已登录管理后台，有业务员管理权限
- **入口路径**：后台菜单 → 运营管理 → 商务管理 → 商务列表

## 步骤详解

### 第一步：查看业务员列表

列表支持搜索筛选：
- **商务姓名**：模糊搜索
- **状态**：启用/禁用
- **注册时间**：日期范围筛选

表格展示：
- 商务编号、商务姓名、性别、联系电话（脱敏+可查看明文）
- 审核时间、审核状态（待审核/审核通过/审核不通过）
- 状态开关（启用/禁用，切换时二次确认）
- 邀请按钮（查看邀请医生数量和销量）
- 操作按钮（编辑/修改资料/邀请医生码/变更手机号/删除）

### 第二步：新增/编辑业务员

点击"新增"或"编辑"弹出表单对话框：

**必填字段**：
- **商务姓名**（最多20字）
- **性别**：从字典加载（男/女）
- **手机号码**：11位，正则校验 `^1[23456789]\d{9}$`
- **身份证号码**（最多18位）
- **开户行**（最多100位）
- **银行账号**（最多100位）
- **详细地址**（最多255位）
- **状态**：启用/禁用

**编辑模式特殊规则**：
- 审核通过（flowStatus=20）显示"编辑"按钮
- 审核不通过（flowStatus=30）显示"修改资料"按钮
- 关闭对话框时二次确认"确认关闭？"

### 第三步：启用/禁用业务员

列表中的状态开关：
- 切换为"启用"：确认框"是否确认启用该商务账号"
- 切换为"禁用"：确认框"是否确认禁用该商务账号，禁用后该商务账号不能正常使用"
- 切换后列表自动刷新

### 第四步：邀请医生码

点击"邀请医生码"操作：
- 弹窗显示业务员姓名和专属二维码
- 二维码图片支持缩放
- 点击"刷新二维码"可重新生成一个新的二维码
- 医生扫码注册时会携带业务员ID

### 第五步：查看邀请医生与销量

点击列表中的"查看"按钮（邀请列）：
- 弹窗标题为"邀请医生"
- 左侧显示邀请的医生列表（分页），包含医生姓名、科室等
- 顶部显示药品销售统计数量
- 支持按条件搜索医生
- 医生列表分页加载

### 第六步：变更手机号

点击"变更手机号"：
- 弹出独立对话框
- 输入新的11位手机号（格式校验）
- 确认后通过后端接口更新手机号

### 第七步：删除业务员

点击"删除"操作按钮删除业务员账号。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号格式错误 | 表单校验不通过，红色提示"请正确填写电话号码" |
| 姓名为空 | 提示"请输入名称" |
| 二维码获取失败 | 提示"无法获取商务二维码，请联系管理员" |
| 关闭编辑框 | 二次确认"确认关闭？" |
| 启用/禁用 | 二次确认后才执行 |

## 关联功能

- **医生审核管理**（`personneAudit/index.vue`）：审核业务员邀请的医生注册申请
- **业务员邀请医生列表**（`businessInvitationDoctor/index.vue`）：独立查看邀请医生
- **审核列表**（`businesschecklist/index.vue`）：审核业务员的资质
