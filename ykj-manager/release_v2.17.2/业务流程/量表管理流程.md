# 量表管理流程（后台）

> 文档版本 2026-07-02 | 面向使用者：运营人员

## 概述

运营人员在后台创建和管理医学量表（如焦虑量表、抑郁量表等），量表可供医生在问诊中分配给患者填写，或患者自行测评。量表有草稿、已发布、禁用三种状态。

## 触发条件

- **前置条件**：运营人员具有量表管理权限
- **入口路径**：后台 → 诊疗管理 → 量表管理 → 量表列表

## 步骤详解

### 第一步：查看量表列表

进入量表列表页：
- **筛选条件**：
  - 量表分类：下拉选择（从量表分类接口获取选项）
  - 量表名称：模糊搜索，最多 128 字符
  - 创建时间：日期范围选择
- **列表字段**：量表名称、量表分类、创建时间、启用状态、操作按钮
- 分页显示，每页 10 条

### 第二步：新增量表

点击「新增量表」按钮跳转到量表编辑页面：
- 量表编辑页分步骤填写：基本信息 → 题目 → 评分规则
- 编辑过程中自动保存为草稿状态（status=0）

### 第三步：管理量表状态

每条量表有三种状态和对应操作：

**草稿（status=0）**：
- **编辑**：继续编辑量表内容
- **发布**：发布前校验题目列表和评分规则列表均不为空
  - 如果缺少内容 → 弹窗提示「该量表有内容未填写，请确认填写完整再发布」，可点击「完善量表」跳转编辑页
  - 如果内容完整 → 直接发布，提示「发布成功！」

**已发布（status=1）**：
- **禁用**：点击禁用按钮，提示「操作成功！」

**禁用（status=2）**：
- **启用**：点击启用按钮，提示「操作成功！」

### 第四步：调整排序

已发布或禁用的量表支持排序调整：
- **上移**：列表第一页第一个位置不显示上移，其他均显示
- **下移**：列表最后一页最后一个位置不显示下移，其他均显示
- 通过 `scaleSortUp` / `scaleSortDown` 接口调整排序权重

### 第五步：编辑量表

仅草稿状态的量表可编辑：
- 点击「编辑」跳转量表编辑页面
- 传递量表 ID 参数 `scaleId`
- 从编辑页返回时保留列表筛选状态（`status=cache`），否则重置搜索条件

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 量表内容不完整时发布 | 弹窗提示并引导完善，不可发布 |
| 量表分类无数据 | 分类下拉为空 |
| 列表为空 | 表格显示空数据行 |

## 关联功能

- **量表分类管理**：量表分类的创建和编辑（独立页面）
- **量表编辑**：量表基本信息、题目、评分规则的详细编辑
