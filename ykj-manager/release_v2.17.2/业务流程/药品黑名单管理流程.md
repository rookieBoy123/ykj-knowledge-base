# 药品黑名单管理流程

> 文档版本 v1.0 | 面向读者：运营管理人员

## 概述

运营人员在后台维护药品通用名黑名单，被列入黑名单的药品在医生开方时将受到限制。支持单个新增/编辑/删除、批量删除、Excel模板下载和批量导入。

## 触发条件

- 前置条件：拥有黑名单管理权限（med:blacklists:write / med:blacklists:import）
- 入口路径：管理后台 → 医院管理 → 药品信息 → 限制类药品（黑名单）

## 步骤详解

### 第一步：查看黑名单列表

进入页面展示黑名单列表，包含：
- 复选框（支持多选）
- 通用名（药品通用名称）
- 更新时间

支持按通用名进行模糊搜索（最长255字）。

### 第二步：新增黑名单药品

点击"新增"按钮 → 弹出新增窗口（450px宽）→ 输入药品通用名（必填，最长255字，带字数统计）→ 点击"确定"保存 → 提示"新增成功"

**校验规则**：通用名不能为空，否则提示"请输入通用名"

### 第三步：编辑黑名单药品

点击列表中的"编辑"按钮 → 弹出编辑窗口 → 修改通用名 → 点击"确定"保存 → 提示"编辑成功"

### 第四步：删除黑名单药品

- **单个删除**：点击某行的"删除"按钮 → 确认弹窗"此操作将永久删除该项, 是否继续?" → 确认后删除
- **批量删除**：勾选多条记录 → 点击"批量删除" → 确认弹窗"此操作将永久删除选中项, 是否继续?" → 确认后删除
- 未选择数据时点击批量删除，提示"请至少选择一条数据"

### 第五步：Excel批量导入

点击"下载模板" → 下载名为"黑名单导入模板.xlsx"的Excel模板（表头：通用名）→ 填写药品通用名 → 点击"导入"按钮选择文件上传 → 提示"导入成功"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 通用名为空 | 提示"请输入通用名" |
| 未选择任何数据就批量删除 | 提示"请至少选择一条数据" |
| 删除确认 | 提示"此操作将永久删除该项/选中项, 是否继续?" → 取消提示"已取消删除" |

## 关键规则

- **按通用名限制**：黑名单基于药品的"通用名"进行匹配，不区分厂家和规格
- **批量导入**：支持Excel导入，模板仅需填写通用名一列
- **权限控制**：新增/编辑/批量删除需要 med:blacklists:write 权限；下载模板/导入需要 med:blacklists:import 权限
- **删除后不可恢复**：删除操作有确认弹窗，确认后永久删除

## 关联功能
