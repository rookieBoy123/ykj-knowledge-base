# 线下病历详情查看流程

> 文档版本 1.0 | 面向非技术读者 | 2026-07-02

## 概述

后台管理人员在管理系统中查看从HIS（医院信息系统）同步过来的线下就诊病历。这些病历记录了患者在线下医院就诊的完整信息，包括患者基本信息、监护人信息、病史信息（主诉、现病史、既往史等十一项）、体格检查数据、西医诊断、中医四诊信息、以及检查/检验报告和处方信息。

## 触发条件

- 入口：后台管理系统 → 医院管理 → 患者管理 → 患者详情 → 线下病历tab
- 前提：患者已在合作医院的HIS系统中产生就诊记录，数据已同步至平台

## 步骤详解

### 第一步：查看患者基本信息

页面顶部展示患者的基本身份信息：
- 就诊卡号（HIS系统中的患者ID）
- 姓名
- 性别
- 出生日期
- 年龄（大于1岁显示"X岁"，小于1岁显示"X月"）
- 身份证号（已校验标记）

### 第二步：查看监护人信息（如有）

如果患者有监护人（hadGuardian=1），显示监护人信息：
- 监护人姓名
- 与患者关系（父母/配偶/子女/亲属/其他）
- 监护人身份证号（已校验标记）

### 第三步：查看病史信息

展示就诊时的完整病史记录：

| 信息项 | 说明 |
|-------|------|
| 接诊医生 | 线下来院就诊医生姓名 |
| 就诊科室 | 医生所在科室 |
| 就诊时间 | 线下就诊的具体时间 |
| 主诉 | 患者的主要症状描述 |
| 现病史 | 发病过程、诊治经过 |
| 既往史 | 过去的疾病和手术史 |
| 个人史 | 生活习惯、职业等 |
| 家族史 | 家族遗传病史 |
| 婚育史 | 婚姻和生育情况 |
| 月经史 | （女性）月经相关记录 |
| 过敏史 | 药物、食物等过敏情况 |

所有病史字段如无数据则显示"无"。

### 第四步：查看体格检查

展示患者的体格检查数据：
- 体温（℃）
- 脉率（次/min）
- 呼吸频率（次/min）
- 血压（收缩压/舒张压，mmHg）
- 身高（cm）
- 体重（kg）
- BMI（身体质量指数）

### 第五步：查看中医四诊信息

显示中医望闻问切四诊内容，如无数据则显示"无"。

### 第六步：查看西医诊断

展示就诊时的西医诊断信息，包括：
- 诊断名称（主要诊断/次要诊断）
- 诊断编码

### 第七步：查看检查检验报告

展示该次就诊关联的检查报告和检验报告列表，报告来自合作医院HIS系统的数据同步。

### 第八步：查看处方信息

展示医生开具的处方详情，包括药品名称、用法用量等，与药品管理模块关联。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 患者无线下病历记录 | 不显示该Tab或无数据提示 |
| 病史某字段为空 | 显示"无" |
| 监护人信息不存在 | 不显示监护人区域 |
| 检查/检验报告为空 | 报告列表为空 |

## 关联功能

- **患者管理**：从患者列表进入患者详情
- **检查报告管理**：查看和下载检查报告
- **处方管理**：查看关联的处方信息
