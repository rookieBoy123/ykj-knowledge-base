# 医保慢性病种管理流程

> 文档版本：2026-07-03 | 面向读者：运营、管理员

## 概述

医保慢性病种管理用于维护医保系统中的慢性病种类数据。每个慢性病种包含病种名称、病种编码、关联科室等信息，并可控制启用/禁用状态。这些数据用于医保结算、医生开方时选择慢性病诊断等场景。

## 触发条件

- 医保政策调整，需要新增或修改慢性病种类
- 需要调整某慢性病种的关联科室
- 某慢性病种不再适用，需要禁用

## 步骤详解

### 第一步：浏览慢性病种列表

进入"慢性病种管理"页面，以表格形式展示所有慢性病种：
- **序号**：自动编号
- **病种编码**：医保系统中的编码
- **病种名称**：慢性病的名称
- **关联科室**：该病种关联的医院科室列表（多个科室以逗号分隔显示）
- **状态**：启用/禁用开关
- **操作**：编辑按钮

支持按"病种名称"和"病种编码"进行搜索筛选。

### 第二步：编辑慢性病种

点击"编辑"按钮，弹出编辑窗口，显示：
- **病种名称**：仅展示，不可修改
- **病种编码**：仅展示，不可修改
- **关联科室**：从医院科室树形结构中选择（支持多选、搜索），修改后保存
- **状态**：启用或禁用（单选切换）

**校验规则**：关联科室为必填项，未选择时提示"请选择关联科室"。

### 第三步：启用/禁用

在列表页可以直接切换某病种的启用/禁用状态：
- **启用**：病种正常使用，医生开方时可选择
- **禁用**：病种不再使用，前端不再展示该选项
- 切换操作即时生效，无需二次确认

### 第四步：删除慢性病种

点击删除按钮：
1. 弹出确认提示"此操作将永久删除该项，是否继续？"
2. 确认后调用删除接口
3. 删除成功后刷新列表

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 编辑时未选择关联科室 | 提示"请选择关联科室" |
| 删除操作取消 | 无提示，关闭确认框 |
| 编辑保存失败 | 弹出接口返回的错误提示 |
| 删除失败 | 弹出接口返回的错误提示 |
| 启用/禁用失败 | 操作无反馈（静默），状态可能回滚 |

## 关联功能

- **科室管理**：关联科室选项来自医院科室树形数据
- **医生开方**：医生选择慢性病诊断时，选项来源于此处的慢性病种数据
- **医保结算**：医保结算申报时使用慢性病种数据
