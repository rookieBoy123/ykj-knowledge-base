# 财务看板流程

> 文档版本 v1.0 | 面向读者：运营人员、财务人员

## 概述

财务看板是后台首页的数据仪表盘，用图表形式展示平台的交易趋势和订单来源构成。运营人员可切换不同年份查看数据变化，无需手动导出报表即可快速了解平台整体营收情况。

## 触发条件

- 前置条件：已登录管理后台，拥有相关数据查看权限
- 入口路径：左侧菜单 → 财务管理 → 看板（或登录后默认首页）

## 步骤详解

### 第一步：查看交易数据柱状图

页面加载后，上方默认展示当前年份的月度交易数据柱状图：

- 横轴为 1-12 月，纵轴为交易金额（单位：元）
- 鼠标悬停到柱子可查看具体月份的交易金额
- 图表右上角有年份选择器，默认显示当前年份
- 切换年份后自动刷新数据

### 第二步：查看订单来源饼图

下方展示订单来源分布饼图：

- 按订单类型（药品订单/问诊订单/会诊订单/预约订单/服务包订单）显示占比
- 每个扇形标注类型名称和百分比
- 同样支持切换年份查看不同年度的来源构成
- 图例在饼图右侧垂直排列

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 选择年份后数据为空 | 图表显示为空（柱状图无数据，饼图为空饼），无特殊提示 |
| 某月份无交易数据 | 该月份柱子高度为 0（金额为 0 元） |
| API 请求失败 | 图表区域保持空白，无 loading 或错误提示（待改进） |

## 关联功能

- **交易流水采集**：点击具体订单信息跳转查看明细（当前版本详情按钮逻辑已注释，仅展示汇总数据）
- **药品订单统计**：在另一页面查看更详细的订单销售统计

## 数据说明

- 交易数据金额在后端以"分"为单位存储，前端除以 100 转换为"元"
- 订单来源映射：1-药品订单、2-问诊订单、3-会诊订单、4-预约订单、6-服务包订单
- 两个图表的数据来源独立：柱状图按月统计总额，饼图按订单类型统计次数
