# 问诊订单综合查询与导出流程

> 文档版本 v1.0 | 面向读者：后台运营人员

## 概述

后台运营人员可以在问诊订单查询页面按多维度筛选查询所有问诊订单，查看订单详情（包括脱敏的患者姓名和联系方式），并导出查询结果为Excel文件。

## 触发条件

- **前置条件**：已登录管理后台，有问诊订单查看权限
- **入口路径**：后台菜单 → 诊疗管理 → 门诊服务订单 → 问诊列表查询

## 步骤详解

### 第一步：设置筛选条件

搜索区域提供丰富的筛选维度：

- **订单编号**：精确搜索，最多20字符
- **创建时间**：日期范围选择，默认最近30天（从30天前到今天23:59:59）
- **服务类型**（多选）：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊；如开启AI医生功能则增加"AI图文问诊"
- **问诊流程**（多选）：在线问诊、线上复诊、慢病医保、预约挂号、快速开方；如开启AI则增加"AI问诊"；选"全部"时清空流程筛选
- **问诊状态**：已预约（含签到状态）、问诊中（含签到状态）、问诊结束、取消问诊
- **支付状态**：未支付、已支付
- **医生姓名**：模糊搜索，最多5字符
- **患者姓名**：模糊搜索，最多5字符
- **支付时间**：日期范围
- **所属团队**：下拉选择（仅开启专家团队功能时显示）
- **问诊单编号**：精确搜索，最多30字符

### 第二步：查看列表

表格列展示：

| 列名 | 说明 |
|------|------|
| 订单号 | 订单唯一编号 |
| 问诊单编号 | 问诊单编号 |
| 服务类型 | 如"图文问诊" |
| 问诊流程 | 如"在线问诊" |
| 医生姓名 | 接诊医生 |
| 所属团队 | 专家团队名称（仅开启专家团队时显示） |
| 患者姓名 | 脱敏显示（姓+*号），点击可查看完整姓名 |
| 联系方式 | 脱敏显示，点击可查看完整手机号（同一时间只能查看一个） |
| 创建时间 | 订单创建时间 |
| 订单支付时间 | 支付时间（含格式化处理） |
| 诊费(元) | 患者支付金额 |
| 平台服务费 | 平台抽取的服务费 |
| 医生收益(元) | 医生实际收益 |
| 问诊状态 | 草稿/问诊中/问诊结束/取消问诊 |
| 支付状态 | 已支付/未支付 |

**表格操作**：
- 支持分页（默认每页10条）
- 支持点击列头排序
- 点击"重置"可清空所有筛选条件

### 第三步：导出数据

点击"导出"按钮：
1. 校验时间范围：创建时间和支付时间间隔均不能超过3个月（超过则提示"时间间隔不能超过三个月"）
2. 显示"加载中"提示
3. 按当前筛选条件导出为Excel文件（`.xls`格式）
4. 导出时自动拆分逗号分隔的多选参数（如服务类型、问诊流程）

### 手机号脱敏机制

- 患者姓名和手机号默认脱敏显示（姓+星号）
- 点击脱敏字段触发查看完整信息（需权限验证）
- **同一时间只能查看一条记录**的完整信息——查看第二条时，第一条自动恢复脱敏
- 手机号查看为后端解密后返回

### 问诊状态复合筛选

"已预约"和"问诊中"状态实际是复合筛选：
- "已预约"：flowStatus=20 且 isCheckIn=0（问诊中状态但未签到）
- "问诊中"：flowStatus=20 且 isCheckIn=1（问诊中状态且已签到）

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 导出时间范围超过3个月 | 提示"时间间隔不能超过三个月"，不执行导出 |
| 导出数据为空 | 导出空的Excel文件 |
| 手机号解密失败 | 不显示明文，保持脱敏状态 |
| 问诊流程选"全部" | 自动清空问诊流程筛选参数 |

## 关联功能

- **订单详情**（`orderlistDetails/index.vue`）：查看单笔问诊订单的详细信息
- **问诊详情**（`inquiryOffice/index.vue`）：查看问诊聊天记录和病历
- **退款审核**（`reimburselist/index.vue`）：对已支付订单进行退款审核
