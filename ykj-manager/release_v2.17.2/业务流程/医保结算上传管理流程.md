# 医保结算上传管理流程

> 文档版本 2026-07-02 | 面向读者：运营、产品

## 概述

管理后台提供医保结算数据的管理功能。运营人员可查询患者的医保结算记录，按状态筛选（待上传/已上传/上传失败），支持单条上传和批量上传操作，并上传时需指定申报时间。

## 触发条件

- **前置条件**：具有医保结算管理权限的后台用户
- **入口路径**：管理后台 → 财务管理 → 医保结算上传管理
- **默认时间范围**：进入页面默认查询近30天的结算记录

## 步骤详解

### 第一步：查询结算记录

页面提供丰富的筛选条件：

| 筛选条件 | 说明 |
|---------|------|
| **状态Tab** | 待上传（默认）/ 已上传 / 上传失败 |
| **病人名称** | 支持模糊搜索 |
| **结算日期** | 日期范围选择 |
| **人员编号** | 精确搜索 |
| **病种** | 下拉选择，支持远程搜索，数据来源于慢性病种管理 |
| **身份证号** | 精确搜索 |
| **科室** | 树形选择，按科室层级筛选 |

**Tab切换行为**：
- 切换到"已上传"时：显示申报时间列和筛选条件，隐藏复选框和批量上传按钮
- 切换到"待上传"或"上传失败"时：显示复选框和批量上传按钮

### 第二步：查看列表

列表展示字段：

| 字段 | 说明 |
|------|------|
| 复选框 | 仅在"待上传"/"上传失败"状态下显示 |
| 人员编号 | 医保人员唯一编号 |
| 患者姓名 | 就诊人姓名 |
| 慢性诊断科室 | 诊断对应的科室 |
| 病种 | 慢性病病种名称 |
| 证件类型 | 固定显示"居民身份证（户口簿）" |
| 证件号 | 身份证号码 |
| 性别 | 男/女 |
| 年龄 | 患者年龄 |
| 结算时间 | 医保结算的具体时间 |
| 申报时间 | 仅在"已上传"状态下显示 |
| 操作 | 上传/重新上传/查看详情 |

### 第三步：单条上传

点击某条记录的"上传"按钮（仅在"待上传"状态下显示）：

1. 弹出对话框，选择**申报时间**（日期选择器，不能选择未来日期）
2. 点击确认，调用上传接口（declaresUpload）
3. 上传成功后提示"上传成功"，刷新列表
4. 未选择申报时间提示"请选择申报时间"

### 第四步：批量上传

1. 勾选需要上传的多条记录（必须至少选一条）
2. 点击顶部"批量上传"按钮
3. 未选择任何记录时提示"请至少选择一条数据"
4. 弹出申报时间选择对话框
5. 确认后批量上传

### 第五步：重新上传

对于"上传失败"状态的记录，点击"重新上传"按钮，流程与单条上传相同。

### 第六步：查看详情

点击任意状态记录的"查看详情"，弹出详情对话框查看该条医保结算的完整信息。

## 三种状态说明

| 状态 | 含义 | 可执行操作 |
|------|------|-----------|
| **待上传**（status=1） | 医保结算已完成，数据待上传到医保系统 | 上传（单选/批量） |
| **已上传**（status=2） | 数据已成功上传到医保系统 | 查看详情 |
| **上传失败**（status=3） | 上次上传失败，需要重新上传 | 重新上传 |

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 批量上传未选择记录 | 提示"请至少选择一条数据" |
| 未选择申报时间 | 提示"请选择申报时间" |
| 列表无匹配数据 | 显示空列表 |
| 上传接口失败 | 后端返回错误信息，前端展示 |

## 关联功能

- **慢性病种管理**（chronicdiseasemanage）：配置可选病种
- **科室管理**（tDepartment）：科室树形数据
- **字典管理**：订单类型等基础数据
