# 快捷短语分类及内容管理流程

> 文档版本 v1.0 | 面向读者：运营管理人员

## 概述

运营人员在后台管理医生问诊时使用的快捷短语，按分类组织内容，方便医生在问诊对话中快速发送常用话术。支持分类切换、内容新增、编辑、删除和批量删除。

## 触发条件

- 前置条件：拥有快捷短语管理权限
- 入口路径：管理后台 → 运营管理 → 快捷短语设置

## 步骤详解

### 第一步：查看分类和内容

进入页面后顶部显示分类标签栏（Tab切换），默认选中第一个分类。标签下方展示该分类下的短语内容列表，包含：
- 复选框（支持多选）
- 内容（短语文字）
- 创建时间
- 修改时间

分类通过后台接口自动获取（isCommon=1的公共分类），每个分类有唯一的分类编码和分类名称。

### 第二步：切换分类

点击顶部分类标签 → 切换显示对应分类下的短语内容 → 页码重置为第一页。

### 第三步：新增短语内容

点击"新增"按钮 → 弹出新增窗口 → 填写内容（必填，最长255字，带字数统计）和备注（选填，最长500字）→ 点击"保存" → 提示"操作成功"

**校验规则**：
- 内容不能为空："请输入内容"
- 内容名称唯一性校验：若同分类下存在同名内容，提示"内容名称已存在，请重新输入"

### 第四步：编辑短语内容

点击"编辑"按钮 → 弹出编辑窗口 → 修改内容和备注 → 保存时同样校验唯一性

### 第五步：删除短语内容

- **单个删除**：点击某行的"删除"按钮 → 确认弹窗"确认删除吗？" → 确认后删除 → 提示"操作成功"
- **批量删除**：勾选多条记录 → 点击"批量删除" → 确认弹窗 → 确认后删除
- 未选择数据时提示"请至少选择一条数据"

### 第六步：搜索过滤

在搜索框中输入关键字 → 点击"查询"筛选匹配的内容 → 点击"重置"清空搜索条件。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 内容名称为空 | 触发校验提示"请输入内容" |
| 同分类下内容名称重复 | 提示"内容名称已存在，请重新输入" |
| 未选择数据就批量删除 | 提示"请至少选择一条数据" |
| 删除确认 | 提示"确认删除吗？" → 取消提示"已取消删除" |
| 关闭编辑窗口未保存 | 提示"确认关闭？" |

## 关键规则

- **分类维度**：快捷短语按分类组织，每个分类下可有多个内容条目
- **公共分类**：仅加载 isCommon=1 的分类
- **唯一性校验**：同一分类下内容名称不允许重复，修改时如名称未变化则跳过重复检查
- **删除确认**：所有删除操作都有二次确认弹窗
