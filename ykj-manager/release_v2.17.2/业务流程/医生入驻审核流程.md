# 医生入驻审核流程

> 文档版本：2026-07-02 | 面向读者：产品运营、管理员

## 概述

医生通过小程序注册提交资料后，后台运营人员需要对医生资料进行审核。审核通过后医生才能正常使用平台服务。支持按审核状态分Tab查看和批量操作。

## 触发条件

- **前置条件**：具有医生审核权限的后台账号
- **入口路径**：左侧菜单 → 医院管理 → 医生管理 → 人员审核

## 步骤详解

### 第一步：按审核状态筛选

页面顶部有三个Tab，每个Tab显示对应状态的医生数量和：
- **待审核**：显示待审核的医生数量
- **不通过**：显示审核不通过的医生数量
- **已通过**：显示审核通过的医生数量

默认进入「待审核」Tab。

### 第二步：筛选查询

在每个Tab下支持多维度筛选：
- 医生姓名（最多5字）
- 科室（树形选择）
- 所属医院（远程搜索下拉）
- 职称（字典下拉）
- 所属机构（机构模式时显示）

### 第三步：查看审核列表

列表展示以下信息：

| 列名 | 说明 |
|------|------|
| 编号 | 医生ID |
| 医生姓名 | 注册时填写的姓名 |
| 所属医院 | 所属医院名称 |
| 科室 | 科室名称 |
| 职称 | 职称名称 |
| 电话号码 | 脱敏状态，点击可查看完整号码 |
| 所属机构 | 机构名称（机构模式时显示） |
| 邀请人 | 邀请医生注册的商务人员（非严格模式时显示） |
| 申请时间 | 医生提交资料的时间 |

### 第四步：审核操作

根据医生当前审核状态，操作列显示不同按钮：

- **待审核状态**：显示「去审核」按钮
  - 点击跳转到审核详情页，可查看医生提交的全部资料
  - 包括：基本信息、专业信息、上传的证件照片
  - 可操作：审核通过 / 审核不通过（需填写不通过原因）

- **已通过/不通过状态**：显示「查看详情」按钮
  - 点击查看医生资料详情（只读模式）

### 第五步：新增医生（非互联网诊疗模式）

在非互联网诊疗模式下，页面支持直接新增医生：
- 点击「新增」按钮跳转到医生详情编辑页
- 可手动录入医生信息

### 第六步：同步HIS医生（互联网诊疗模式）

在互联网诊疗模式下，支持从HIS系统同步医生：
- 点击「同步HIS医生」按钮
- 弹出HIS同步弹窗，选择要同步的医生
- 同步完成后自动刷新列表

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 从详情页返回 | 保留之前的筛选条件和Tab状态（缓存模式） |
| 从其他页面进入 | 重置搜索条件 |
| 分页数据为空且不在第一页 | 自动回退到上一页 |
| 严格模式 | 隐藏邀请人列 |
