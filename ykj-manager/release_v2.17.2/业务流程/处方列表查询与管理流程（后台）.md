# 处方列表查询与管理流程（后台）

> 文档版本 2026-07-02 | 面向运营管理人员

## 概述

运营人员通过处方管理页面可以查看平台上所有医生开具的处方记录，支持按医生、科室、处方类型、状态等多个维度筛选，并可查看处方的电子版PDF。

## 触发条件

- 登录管理后台，拥有处方查询权限
- 从诊疗管理 → 处方管理进入"处方列表"

## 步骤详解

### 第一步：条件筛选
支持以下多维度筛选条件：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 医生姓名 | 文本输入 | 按处方医生姓名模糊搜索，最多5个字 |
| 科室 | 树形下拉 | 按医生所属科室筛选，支持科室树结构 |
| 处方类型 | 下拉选择 | 中药处方(2) / 成药处方(1) |
| 处方编号 | 文本输入 | 精确搜索某张处方 |
| 开具时间 | 日期范围 | 选择起止日期 |
| 状态 | 下拉选择 | 开方中/待审核/审核中/审核通过/审核不通过/已失效 |
| 所属团队 | 下拉选择 | 按专家团队筛选（仅当系统启用专家团队功能时显示） |

### 第二步：查看处方列表
列表展示字段：

| 字段 | 说明 |
|------|------|
| 处方编号 | 系统生成的唯一处方号 |
| 处方类型 | 成药处方 / 中药处方 |
| 处方医生 | 开具处方的医生姓名 |
| 所属团队 | 医生所属专家团队（启用专家团队时显示） |
| 科室 | 医生所在科室 |
| 开具时间 | 处方创建时间 |
| 诊断 | 临床诊断内容 |
| 患者姓名 | 脱敏展示（首字+*号，点击可查看全名） |
| 状态 | 处方当前生命周期状态 |

### 第三步：患者姓名脱敏
- 默认显示：姓名的第一个字 + 对应长度的星号（如"张**"）
- 点击可查看完整的患者姓名（仅当次操作有效）
- 用于保护患者隐私的同时满足运营查询需求

### 第四步：查看电子处方
- 操作列"查看电子处方"按钮仅在有电子处方文件时显示
- PDF格式：在新窗口打开PDF预览器
- 图片格式：使用图片预览弹窗查看

## 处方状态说明

| 状态码 | 状态名称 | 说明 |
|--------|---------|------|
| 10 | 开方中 | 医生正在编辑处方 |
| 20 | 待审核 | 处方已提交，等待审核 |
| 25 | 审核中 | 药师正在审核处方 |
| 30 | 审核通过 | 审核已通过 |
| 40 | 审核不通过 | 审核被驳回 |
| 50 | 已失效 | 处方超过有效期或主动作废 |

## 关联功能

- 【查看电子处方】：在线预览处方的正式PDF文件
- 【专家团队筛选】：当系统启用专家团队功能时，可按团队维度管理
