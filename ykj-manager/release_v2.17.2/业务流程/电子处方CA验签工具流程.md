# 电子处方CA验签工具流程

> 文档版本 v1.0 | 面向读者：运营管理员、质控人员

## 概述

管理员可通过上传电子处方的PDF文件，利用CA（电子认证）厂商提供的验签接口验证处方签名的真实性。用于核实电子处方是否由合法CA证书签署，防止处方伪造或篡改。

## 触发条件

- 前置条件：具备CA厂商的验签能力
- 入口路径：管理后台 → 诊疗管理 → 处方管理 → 处方验签

## 步骤详解

### 第一步：上传处方PDF文件

页面提供拖拽上传区域，支持以下操作：
- 将PDF文件拖拽到上传区域
- 点击「点击上传」文字选择本地文件
- 只能上传PDF格式文件
- 单次只能上传一个文件
- 文件大小不能超过2MB

### 第二步：点击验证

上传文件后点击「验证」按钮，系统将PDF文件提交给CA厂商的验签接口。验证过程中：
- 如果未上传文件就点击验证，提示「请先上传pdf文件！」
- 验证通过时提示「验证通过！」
- 验证不通过时提示「验证不通过！」

验证完成后，已上传的文件会被自动清空，方便进行下一次验签。

### 第三步：了解验签说明

页面右侧展示验签说明，帮助管理员理解验签的功能和限制：
- 在线验签是通过CA厂家提供的验证接口验证PDF签名的真实性
- 仅支持本互联网医院开具的电子处方验签

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 上传非PDF文件 | 提示「只能上传pdf文件！」并拒绝上传 |
| PDF文件超过2MB | 提示「上传pdf文件大小不能超过 2MB!」并拒绝上传 |
| 未上传文件点击验证 | 提示「请先上传pdf文件！」 |
| 验签接口返回失败 | 提示「验证不通过！」 |
| 上传多个文件 | 限制只能上传1个文件，超出限制不会添加 |

## 关联功能

- **处方列表管理**：查看所有处方的列表
- **药师审方**：药师审核处方时依赖处方真实性
- **CA签章管理**：医生的电子签章配置管理
