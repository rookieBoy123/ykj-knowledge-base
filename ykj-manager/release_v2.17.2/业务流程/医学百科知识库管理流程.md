# 医学百科知识库管理流程

> 文档版本 1.0 | 面向运营/内容编辑人员

## 概述
运营人员在后台统一管理小程序"医学百科"模块展示的各类中医药知识内容，包括疾病知识、经络穴位、药膳食疗、中草药图解、经典名方共五大类知识库。每类知识库独立管理，支持新增、编辑、删除和批量删除操作，所有筛选条件基于系统字典配置，确保数据标准化。

## 前置条件
- 已登录管理后台
- 拥有医学百科管理权限
- 相关字典数据已配置（如疾病分类、经络类型、身体部位等）

## 步骤详解

### 一、疾病知识管理

#### 进入疾病知识列表
从运营菜单 "医学百科 → 疾病知识" 进入。支持筛选：
- 疾病名称：输入搜索（不超过20字）
- 疾病分类：下拉选择（来源于字典"jbzs-jbfl"）

列表展示：疾病名称、疾病分类、最后修改时间。

#### 新增/编辑疾病
点击"新增"/"编辑"弹窗，通过富文本编辑器录入疾病知识详情。编辑时默认回填已有内容。

#### 删除
支持单个删除和批量删除（选中多条→批量删除），均弹出二次确认"数据删除后不可恢复，您确认要删除吗？"

### 二、经络穴位管理

#### 进入穴位列表
从运营菜单 "医学百科 → 经络穴位" 进入。支持三维筛选：
- 穴位名称：输入搜索（不超过10字）
- 归属经络：下拉选择（来源于字典"jlxw-jl"）
- 身体部位：下拉选择（来源于字典"jlxw-bw"）

列表展示：封面图（缩略图）、穴位名称、归属经络、身体部位、功效作用、最后修改时间。

#### 新增/编辑穴位
弹窗编辑，包含封面图上传、穴位名称、归属经络、身体部位、功效作用等字段，所有分类字段基于字典选项。

### 三、药膳食疗管理

药膳食疗知识库同样支持列表查询、新增/编辑、单条删除和批量删除的基本操作，分类基于对应字典配置。

### 四、中草药图解管理

中草药图解知识库维护中药的图解详情信息，支持列表管理和增删改操作。

### 五、经典名方管理

经典名方知识库维护中医经典处方信息，同样支持完整的CRUD操作。

## 公共特性

### 分类标准化
所有分类字段的选项均来自系统字典配置，确保：
- 疾病分类（jbzs-jbfl）、经络归属（jlxw-jl）、身体部位（jlxw-bw）等统一管理
- 新增知识内容时下拉选项自动同步最新字典
- 修改字典后知识库自动生效新分类

### 删除确认机制
所有删除操作（单条/批量）均弹出二次确认弹窗，防止误操作。

### 分页机制
默认每页10条，支持切换每页显示数量。

## 异常处理
| 情况 | 系统处理方式 |
|------|------------|
| 批量删除未选中任何条目 | 静默不执行，无提示 |
| 删除被患者收藏的内容 | 二次确认弹窗，删除后患者端不再展示 |
| 字典分类为空 | 对应筛选下拉框无选项 |

## 关联功能
- 小程序患者端：医学百科模块查看各类知识
- 字典配置：统一管理分类数据
