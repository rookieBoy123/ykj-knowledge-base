# 慢病病种管理流程

> 文档版本 2026-07-01 | 面向读者：后台管理员、医保运营人员

## 概述

管理员在后台管理医保慢病（门诊慢特病）的病种字典，包括病种编码、病种名称、关联科室的配置，以及启用/禁用控制。这些配置直接影响患者在"慢病医保"入口看到哪些病种可选。

## 触发条件

- 系统管理→医保管理→慢病病种管理菜单进入
- 医保政策调整需要新增或停用某个病种
- 需要调整病种关联的科室

## 步骤详解

### 第一步：浏览病种列表

列表支持二维筛选：
- **病种名称**：文本搜索
- **病种编码**：文本搜索

列表展示字段：序号、病种编码、病种名称、关联科室（多个科室名称拼接显示）、启用/禁用开关、编辑按钮。每页10条分页加载。

### 第二步：新增/编辑病种

点击编辑按钮打开弹窗表单：

| 字段 | 说明 |
|------|------|
| 病种名称 | 只读显示（由医保编码决定，不可修改） |
| 病种编码 | 只读显示（由医保编码决定，不可修改） |
| 关联科室 | 树形选择器多选（科室树结构，可选择多个科室，必填） |
| 状态 | 启用/禁用单选 |

**关键逻辑**：病种的名称和编码由医保标准决定不可修改，管理员只能配置关联哪些科室和启用/禁用。

### 第三步：启用/禁用

列表中的状态开关即时生效，无需二次确认。禁用的病种在患者端的"慢病医保"入口不会出现。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未选择关联科室 | 表单校验提示"请选择关联科室" |
| 病种编码重复 | 保存时后端返回错误提示 |
| 科室树加载失败 | 下拉为空，影响新增/编辑操作 |

## 关联功能

- 患者端：慢病医保入口（根据病种+科室匹配可选的医生）
- 关联：医保结算上传管理（按病种筛选结算清单）
- 关联：科室管理
