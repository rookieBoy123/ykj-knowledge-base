# 随访模板题目编辑流程（后台）

> 文档版本 2026-07-02 | 面向使用者：运营人员

## 概述

运营人员在后台创建和编辑随访模板时，可以为模板添加题目。题目是随访问卷的基本组成单元，每个题目包含题目文本和选项设置。支持预览模板以查看患者端最终效果。

## 触发条件

- **前置条件**：具有后台操作权限
- **入口路径**：后台 → 诊疗管理 → 随访管理 → 问卷

## 步骤详解

### 第一步：进入模板编辑页

页面左侧为导航面板，右侧为编辑区域：

- **基本信息 Tab**（默认选中）：编辑模板标题、说明等基本信息，保存后获取模板 ID
- **题目列表**（编辑模板时出现）：展示所有已添加的题目，可展开/折叠

### 第二步：添加基本信息

在「基本信息」Tab 下：
- 填写问卷标题和说明
- 保存后返回模板 ID（`followupId`）
- 系统自动跳转到题目编辑步骤

### 第三步：管理题目列表

左侧题目列表展示：
- 题目的序号和名称（如「1. 您最近一周的睡眠质量如何？」）
- 可操作：删除、上移、下移
- 当前编辑的题目高亮显示

**新增题目**：
- 点击「新增」按钮，右侧切换到空白题目编辑表单
- 如果是第一条题目，列表自动展开

**编辑题目**：
- 点击题目名称，右侧显示该题目的编辑表单
- 修改后保存

**删除题目**：
- 点击删除按钮 → 二次确认弹窗 → 确定后删除
- 如果正在编辑该题目，删除后自动切换到新增模式

**排序调整**：
- 上移：非首位的题目可上移（调用 `editItemNumber` 接口，参数 `upOrDown=-1`）
- 下移：非末位的题目可下移（`upOrDown=1`）

### 第四步：预览问卷

页面右上角有「预览」按钮：
- **编辑基本信息时预览**：使用当前填写的基本信息 + 所有已保存题目
- **编辑题目时预览**：如果修改了当前题目但未保存，预览时同时展示修改后的内容作为预览效果
- **新增题目预览**：未保存的新题目也参与预览
- 预览以弹窗组件展示，模拟患者端视图

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 删除题目 | 二次确认弹窗「是否确认删除？」 |
| 首条题目不可上移 | 上移按钮置灰 |
| 末条题目不可下移 | 下移按钮置灰 |
| 预览时无题目 | 预览弹窗展示空问卷 |

## 关联功能

- **随访模板列表**：问卷模板的创建和管理列表
- **随访计划**：医生选择模板创建随访计划
- **随访任务**：患者端查看和填写问卷
