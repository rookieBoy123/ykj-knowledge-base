# 病历查看流程

> 文档版本：v2.17.2 | 面向读者：运营 / 客服

## 概述

管理员通过后台查看指定患者的完整线上就诊病历记录，包括病历详情和关联的处方信息。

## 触发条件

- 管理员在患者详情页点击跳转进入
- 需携带患者身份证号（idCard）和加密令牌（token）参数

## 步骤详解

### 第一步：进入病历查看页

页面加载时需要两个必填参数：
- **idCard**：患者身份证号
- **token**：加密的访问令牌

如果缺少任一参数，页面不会加载任何数据。进入后，系统先获取医院名称作为页面标题，再通过身份证号查询患者ID。

### 第二步：浏览病历时间线

页面采用左右分栏布局：

**左侧（宽400px）**：病历时间线
- 以时间轴形式展示该患者的所有线上就诊记录
- 每条记录显示门诊号（带「复诊」标签）、接诊医生（姓名+科室+职称）、主诉
- 时间线包含西医诊断、中医辨病、中医辨证（按诊断类型分区显示）
- 如果有电子病历PDF，卡片底部显示「电子病历」按钮，点击在新窗口打开PDF
- 点击任意一条记录，右侧自动加载完整病历详情
- 当前选中的记录卡片背景变为灰色高亮

**右侧**：病历详情区
- 标题固定为「病历详情」
- 分为两个大块：病历信息和处方信息

### 第三步：查看病历信息

右侧病历信息区展示以下内容：

| 信息分类 | 具体内容 |
|---------|---------|
| 就诊人基本信息 | 姓名、性别（男/女）、年龄（岁或月）、门诊号、初诊标志（固定显示"复诊"） |
| 监护人信息 | 仅当患者有监护人时显示：监护人姓名、监护人身份证号 |
| 就诊信息 | 就诊类型（图文/视频等）、就诊机构、就诊时间、就诊科室、接诊医生 |
| 病情描述 | 主诉、现病史、既往史、过敏史、个人史、家族史 |
| 特殊病史 | 月经史、婚育史（女患者） |
| 中医诊断 | 中医四诊（脉象+舌象）、治法（仅中医类型病历时显示） |
| 西医诊断 | 西医诊断名称、治疗意见（仅西医类型病历时显示） |
| 体格相关 | 常见体征、辅助检查 |
| 病历资料 | 患者上传的病历图片（可点击放大预览，空时显示"记录遗失/不在身边"） |

所有未填写的字段统一显示「未填写」。病历类型区分为中医和西医：中医病历额外显示中医辨病、中医辨证、中医四诊、治法；西医病历额外显示治疗意见。

### 第四步：查看处方信息

如果该次就诊开具了处方，病历详情下方展示处方信息区：

- **西药处方（type=1）**：以表格形式展示药品清单，包括序号、药品名称、规格、剂型、每次用量、用药途径、用药频次、数量
- **中药处方（type=2）**：展示药材清单、剂数、用药方法（内服/外用）、服药频率（几天几剂·一剂分几次服用）、用药说明（煎煮方法）
- 每条处方显示处方类型、处方编号、开具时间
- 显示处方医生和审核药师姓名
- 如果有电子处方PDF（treatmentOrderUrl非空），显示「电子处方」按钮，点击在新窗口打开

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 缺少idCard或token参数 | 页面不发起任何请求，一片空白 |
| 身份证号查不到患者 | 提示「没有该患者相关信息」 |
| 该患者无线上就诊记录 | 显示「暂无线上就诊记录!」 |
| 处方中没有审核药师 | 审核药师栏显示短横线"-" |
| 电子病历/处方PDF不存在 | 不显示对应的「电子病历」/「电子处方」按钮 |

## 关联功能

- 从「患者列表 → 患者详情」进入此页面
- 电子病历/处方PDF在新标签页中通过PDF查看器打开
- 病历图片支持点击放大预览（多图轮播）
