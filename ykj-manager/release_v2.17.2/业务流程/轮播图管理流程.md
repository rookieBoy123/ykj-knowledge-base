# 轮播图管理流程

> 文档版本 2026-07-02 | 面向读者：产品、运营

## 概述

管理员在后台管理小程序首页轮播图，支持新增、编辑、删除、排序和显示/隐藏切换。轮播图可配置跳转链接（内部小程序页面、外部小程序、H5网页），用于活动推广和功能入口引导。

## 触发条件

- 管理员在后台导航菜单点击"轮播图管理"
- 需要更新首页Banner内容

## 步骤详解

### 第一步：查看轮播图列表

列表展示以下字段：

| 列表列 | 说明 |
|--------|------|
| 图片 | 缩略图展示 |
| 轮播标题 | 悬停显示完整标题文字 |
| 链接类型 | 内部小程序 / 外部小程序 / H5 |
| 状态 | 显示/隐藏开关（即时切换） |
| 发布时间 | 创建时间 |
| 操作 | 编辑、删除、上移、下移 |

每页展示10条，支持翻页；翻页越界时自动回退到上一页。

### 第二步：筛选轮播图

支持两个筛选条件：

- **轮播标题**（模糊搜索，最多255字）
- **状态**：显示 / 隐藏 下拉选择

### 第三步：新增轮播图

点击"新增"按钮弹出编辑弹窗，需填写：

1. **轮播标题**（必填，最多255字）
2. **轮播图片**（必填）：点击上传区域选择图片，仅支持 jpg/png 格式，大小不超过 2MB，建议尺寸 1078×330px
3. **链接类型**（必填）：内部小程序 / 外部小程序 / H5
4. **小程序ID**（仅外部小程序需填，最多64字）：若选择"外部小程序"，需填写目标小程序的 AppID
5. **跳转路径**（必填，最多255字）：页面路径，外部小程序若留空则默认跳转首页
6. **是否显示**（默认"隐藏"）：显示/隐藏 单选

### 第四步：编辑轮播图

点击"编辑"弹出弹窗（复用新增弹窗），编辑模式下图片为非必填（保留原图）。提交时追加校验：若填写了跳转路径但未选择链接类型，提示"请选择链接类型"。

### 第五步：显示/隐藏切换

列表中每行有一个开关按钮，点击可即时切换轮播图的显示/隐藏状态。切换调用不同接口（bannerShow / bannerHide），无二次确认。

### 第六步：排序调整

每行支持"上移"和"下移"操作：

- **上移**：向上一位置，若已是第一条提示"第一条不能上移！"
- **下移**：向下一位置，若已是最后一条提示"最后一条不能下移！"

排序在筛选条件下也生效（排序请求会携带当前筛选参数）。

### 第七步：删除轮播图

单条删除：点击"删除" → 二次确认弹窗"此操作将永久删除该文件，是否继续？" → 确认后删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 图片格式不正确 | 提示"上传头像图片只能是jpg/png格式！" |
| 图片超过2MB | 提示"上传头像图片大小不能超过 2MB！" |
| 标题未填写 | 提示"请输入轮播标题" |
| 链接类型未选 | 提示"请选择类型" |
| 图片未上传 | 提示"请上传轮播图片" |
| 已填写路径但未选链接类型 | 提示"请选择链接类型" |
| 外部小程序未填AppID | 提示"请填写Appid" |
| 上移到第一条 | 提示"第一条不能上移！" |
| 下移到最后一条 | 提示"最后一条不能下移！" |
| 筛选结果为空时翻页 | 自动回退页码重新查询 |

## 链接类型对比

| 链接类型 | 需填AppID | 跳转路径示例 | 说明 |
|---------|----------|-------------|------|
| 内部小程序 | 否 | /pages/xxx/xxx | 跳转本小程序内的页面 |
| 外部小程序 | 是 | pages/xxx/xxx | 跳转其他小程序，需填写目标AppID |
| H5 | 否 | https://... | 跳转外部网页 |

## 关联功能

- 小程序首页（轮播图展示）
- 医生端首页（可选轮播图展示位置）
- 图片上传（对象存储）
