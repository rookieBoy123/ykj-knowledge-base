# 评价审核管理流程（管理后台）

> 文档版本：release/v2.17.2 | 面向读者：运营

## 概述

运营人员在管理后台查看和管理患者对医生的服务评价。支持按评价内容和评价时间搜索、按审核状态筛选。每条评价可以查看详情、审核通过/不通过、显示/隐藏、删除。

## 触发条件

### 前置条件
- 运营人员已登录管理后台并拥有评价管理权限

### 入口路径
- **管理后台** → 诊疗服务管理 → 评价列表

## 步骤详解

### 第一步：查看评价列表

页面默认加载所有评价记录，以分页表格形式展示：

| 列名 | 说明 |
|------|------|
| 会员名称 | 评价者的姓名 |
| 问诊类型 | 图文问诊/视频问诊等 |
| 接诊医生 | 被评价的医生姓名 |
| 分数 | 1-5星的满意度分数 |
| 评价内容 | 患者填写的文字评价（过长时悬浮提示全文） |
| 评价时间 | 评价提交的日期时间 |
| 状态 | 显示/隐藏开关 |
| 操作 | 详情、通过（待审核时）、删除 |

### 第二步：搜索和筛选

支持三种筛选条件组合：
1. **关键字**：在评价内容中搜索，最大255字
2. **评价时间**：日期范围选择器，起始日期~结束日期
3. **状态**：下拉选择——待审核（10）/审核通过（20）/审核不通过（30）

### 第三步：审核评价

**「通过」操作**（仅待审核状态显示）：
- 点击后直接通过，评价状态变为审核通过（flowStatus=20），同时显示状态自动设为显示

**「详情」弹窗**：
- 点击详情弹窗查看完整评价信息：
  - 会员名称、问诊类型、接诊医生、评价日期
  - 分数（星级）、所选标签（标签列表）
  - 评价内容全文
  - 是否显示开关
- 在弹窗中可直接修改「是否显示」状态
- 弹窗底部按钮根据flowStatus变化：
  - 待审核（flowStatus=10）：按钮为「通过」
  - 其他状态：按钮为「确定」
- 修改后点确定保存

### 第四步：显示/隐藏评价

表格中每条评价都有一个显示/隐藏的即时开关：
- 切换为「显示」：评价在小程序医生主页中可见
- 切换为「隐藏」：评价在小程序端不可见

### 第五步：删除评价

点击「删除」触发二次确认弹窗「此操作将永久删除该文件，是否继续？」：
- 确认后永久删除
- 取消则不执行删除

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 分页查询超出范围 | 自动回退到上一页重新查询 |
| 删除确认取消 | 提示「已取消删除」 |
| 删除失败 | 静默失败（待确认是否提示） |

## 关联功能

- 医生管理：评价影响医生主页的展示
- 门诊服务订单：评价关联具体的问诊订单
