# 远程会诊服务项目配置流程

> 文档版本 v1.0 | 面向读者：远程医疗管理员

## 概述
远程会诊服务项目是机构间开展远程会诊的前提配置。每个参与远程会诊的机构需要配置其可提供的会诊服务项目，包括服务名称、服务类型、是否需要点名会诊、对会诊医生的职称等级要求、服务单价等。管理员可为每个机构新增、编辑、删除会诊服务项目。

## 触发条件
- **前置条件**：拥有机构服务管理权限的远程医疗管理员，且已选择具体机构
- **入口路径**：远程医疗中心 → 机构管理 → 选择机构 → 管理服务

## 步骤详解

### 第一步：进入机构服务管理页面
从机构列表中选择一个机构，点击"管理服务"进入该机构的服务项目配置页面。页面顶部显示当前机构名称，确认操作的是正确的机构。

### 第二步：查看已有服务列表
服务列表展示每个服务项目的详细信息：
- **服务编号**：系统自动生成的唯一ID
- **服务名称**：如"远程影像会诊"、"远程病理会诊"等
- **服务类型**：从字典中加载的服务类型分类
- **是否点名**：是/否/不涉及（决定患者是否可以指定会诊医生）
- **职称等级要求**：正高/副高/不涉及（对参与会诊医生的职称要求）
- **单价**：每次会诊的收费标准（元）
- **单位**：从字典加载（如"次"、"小时"）
- **状态**：启用/禁用
- **操作**：编辑、删除按钮

### 第三步：新增服务项目
- 点击"新增"按钮打开表单弹窗
- 填写以下必填项：
  - **服务名称**：最多200字，有字数统计
  - **服务类型**：下拉选择（从字典"jgfwlx"加载）
  - **是否点名**：单选（是/否/不涉及）
  - **职称等级要求**：单选（正高/副高/不涉及）
  - **单价**：输入数字
  - **单位**：下拉选择（从字典"jgfwlxdw"加载）
  - **状态**：启用/禁用
- 所有字段必填，未填写时表单校验会提示对应错误信息

### 第四步：编辑服务项目
- 点击某条服务的"编辑"按钮，弹出预填好已有数据的表单弹窗
- 修改任意字段后确认保存

### 第五步：删除服务项目
- 点击某条服务的"删除"按钮
- 弹出确认对话框："删除后数据不可恢复，确认删除服务【服务名称】吗？"
- 确认后删除，取消则不做任何操作

## 异常处理
| 情况 | 系统处理方式 |
|------|-------------|
| 缺少机构ID参数 | 页面提示"缺少机构id"，无法继续操作 |
| 新增/编辑时表单校验不通过 | 对应字段下方显示红色错误提示，阻止提交 |
| 删除已有关联会诊记录的服务 | 可能影响已有会诊流程，需谨慎操作 |
| 机构信息加载失败 | 页面顶部机构名称为空 |
| 字典数据加载失败 | 服务类型和单位下拉框为空 |

## 关联功能
- **远程会诊管理**：发起会诊时需要选择机构和会诊服务
- **机构管理**：此处从机构管理进入，服务与机构一一对应
- **医生排班**：会诊服务的职称等级要求影响可参与会诊的医生筛选
