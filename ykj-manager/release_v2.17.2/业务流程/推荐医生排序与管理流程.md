# 推荐医生排序与管理流程

> 文档版本 2026-07-03 | 面向运营管理员

## 概述

管理在患者端和医生端展示的"推荐医生"列表。运营管理员可以选择哪些审核通过的医生作为推荐医生展示，并控制展示顺序。患者端按推荐排序展示医生，帮助患者快速找到优质医生。

## 触发条件

- 运营管理员需要调整推荐医生的展示名单或顺序
- 菜单路径：运营管理 → 资讯文章 → 推荐医生

## 步骤详解

### 第一步：查看推荐医生列表

进入页面后展示当前已设为"推荐医生"的列表，包含以下信息：
- **编号**：医生ID
- **医生姓名**
- **所属医院**
- **科室**
- **职称**：如主任医师、副主任医师等
- **电话号码**
- **创建时间**：设为推荐的时间

支持按"医生姓名"和"医生职位"搜索。

### 第二步：添加推荐医生

点击"添加"按钮，弹出"选择推荐医生"对话框。对话框中分为左右两个区域：

**左侧（可选医生列表）**：
- 显示所有审核通过的医生（状态为10）
- 支持按医生姓名、所属医院搜索
- 支持分页浏览
- 勾选要添加的医生

**右侧（确认操作）**：
- 勾选至少一位医生后，点击"确定"按钮
- 选中的医生会被添加到推荐医生列表末尾

### 第三步：排序调整

每位推荐医生支持四种排序操作：

- **排至最前**：将该医生移到推荐列表第一位
- **上移**：将该医生向上移动一位（已在第一位时提示"第一条不能上移！"）
- **下移**：将该医生向下移动一位（已在最后一位时提示"最后一条不能下移！"）
- **排至最后**：将该医生移到推荐列表最后

排序决定患者端推荐医生的展示顺序。排在前面的医生在小程序中优先展示。

### 第四步：查看医生详情

点击"医生详情"按钮，跳转到该医生的详细信息页面，可查看医生的完整资料（包括执业信息、职称、擅长领域等）。在详情页返回后，推荐列表会保持之前的搜索条件和页码（缓存状态）。

### 第五步：移除推荐医生

点击某位医生的"删除"按钮，弹出确认框"是否确认删除？"。确认后该医生从推荐列表移除（不会影响医生的正常执业状态，只是不再作为推荐医生展示）。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 添加时未勾选任何医生 | 提示"请至少选择一条数据" |
| 第一条执行上移 | 提示"第一条不能上移！" |
| 最后一条执行下移 | 提示"最后一条不能下移！" |
| 删除操作 | 弹出二次确认框，可取消 |
| 取消删除 | 提示"已取消删除" |

## 关联功能

- **医生列表管理**：推荐医生来自审核通过的医生池
- **医生入驻审核**：只有审核通过的医生（状态=10）才能被选为推荐医生
- **患者端医生列表**：小程序中按推荐排序展示医生
- **医生详情**：跳转到医生信息详情页查看完整资料
- **所属医院列表**：添加医生时的医院搜索选项来自医院管理
