# 医生收益结算管理流程

## 概述
管理后台运营人员查看平台所有医生的问诊和开方收益，对医生进行批量或单个收益结算，将累计收益支付给医生。

## 触发条件
运营人员（财务角色）需要核对和结算医生收益时进入「财务管理 → 收益管理」页面。

## 前置条件
- 登录账号具有财务管理相关权限
- 平台有已产生收益的医生

## 步骤详解

### 步骤一：查看收益记录
进入页面默认显示「收益记录」选项卡。列表展示每位医生的累计收益情况，包括：
- 医生编号和医生姓名
- 所属团队（需开启专家团队功能后显示）
- 联系电话（默认脱敏，点击可查看完整号码，需后端权限校验）
- 银行卡号（默认脱敏，点击可查看完整卡号，需后端权限校验）
- 总收益金额（单位为元，支持两位小数显示）

支持按医生姓名（最多5个字）和所属团队两个条件搜索。每页10条记录。

### 步骤二：单个结算
点击某位医生所在行的「结算」按钮，系统会校验该医生的收益：
- 收益金额大于0：弹出「医生收益结算」对话框，确认该医生的姓名、银行卡号、开户银行、卡主名称及结算金额
- 收益金额为0：提示「该医生收益为0」，无法结算

确认后点击「确定」提交结算，结算成功后弹窗关闭，收益列表自动刷新。

### 步骤三：批量结算
勾选列表中多位医生的复选框，点击列表上方的「批量结算」按钮：
- 至少选择一位医生，否则提示「请至少选择一条数据」
- 弹出结算确认对话框，展示所有选中医生的银行卡信息和结算金额
- 确认后一次性提交结算

### 步骤四：查看结算记录
切换到「结算记录」选项卡，查看已完成的结算历史。列表展示：
- 医生编号和医生姓名
- 银行卡号（脱敏显示）
- 结算金额
- 结算日期

支持按日期范围筛选结算记录。每页10条。

### 步骤五：导出数据
- 收益记录页面：点击「导出」按钮可将当前搜索结果导出为Excel文件（需导出权限）
- 结算记录页面：同样支持按日期范围导出结算数据

## 权限控制

| 操作 | 所需权限 |
|------|---------|
| 查看收益/结算列表 | 财务管理模块访问权限 |
| 批量结算 | `hosp:doctor-incomes:settle` |
| 导出收益数据 | `hosp:doctor-incomes:export` |
| 查看完整手机号 | 后端校验权限 |
| 查看完整银行卡号 | 后端校验权限 |

## 异常处理
- 收益为0：不弹出结算对话框，提示「该医生收益为0」
- 未选中数据批量结算：提示「请至少选择一条数据」
- 银行卡号/手机号脱敏：列表默认显示脱敏数据，点击眼睛图标可查看明文，明文权限由后端校验

## 关联功能
- 医生管理（医生列表）：医生基本信息管理
- 发票查询：订单支付与发票关联
- 小程序医生端「收益」页面：医生自行查看收益和结算记录
