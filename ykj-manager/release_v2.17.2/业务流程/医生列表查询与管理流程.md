# 医生列表查询与管理流程

> 文档版本：v1.0 | 面向读者：管理员

## 概述

管理员在后台管理所有注册医生的基本信息、服务状态、账号启用/禁用、CA签章认证以及会诊管理员权限分配。

## 触发条件

- **前置条件**：以管理员角色登录管理后台
- **入口路径**：医院管理 → 医生管理 → 医生列表

## 步骤详解

### 第一步：多维度筛选医生

页面顶部提供最多10个搜索条件：

1. **医生姓名**：输入框，最多5个字符
2. **工作类型**：下拉选择（全部/医生/医助）
3. **科室**：树形选择器，从HIS同步的科室树动态加载
4. **所属医院**：远程搜索下拉（输入关键字后端搜索）
5. **职称**：下拉选择，选项来自字典`positionCategory`
6. **账号状态**：全部/启用(status=10)/禁用(status=20)
7. **所属机构**：仅在机构功能开启时显示
8. **开启服务**：多选，可选图文问诊/视频问诊/咨询/快速问诊/快速续方/医生复诊/商城问诊
9. **关联商务**：仅在非严格模式下显示，输入框20字符限制
10. **是否会诊管理员**：是/否

以上条件可组合使用，点击搜索后重置到第一页。

### 第二步：浏览医生列表

列表支持每页10条分页，展示以下字段：

| 列名 | 说明 |
|------|------|
| 编号 | 医生ID |
| 医生姓名 | - |
| 所属医院 | - |
| 科室 | - |
| 职称 | positionCategoryName |
| 联系电话 | 默认脱敏，点击眼睛图标逐条解密查看完整号码 |
| 工作类型 | 医生/医助 |
| 开启服务 | 多个服务以顿号分隔 |
| 所属机构 | 机构功能开启时显示 |
| 关联商务 | 严格模式下隐藏 |
| 关注量 | 患者关注数量 |
| 在线状态 | 绿色"在线"或灰色"离线" |
| 状态 | 启用/禁用开关 |
| CA认证状态 | authStatus_enumName |
| CA用户同步状态 | caSubStatusText（-1/0/1/2/3/4/5） |
| 注册时间 | - |

### 第三步：操作医生账号

每行右侧有操作按钮：

1. **编辑**：跳转到医生信息编辑页（info页面）
2. **邀请码**：打开二维码弹窗，展示医生的专属邀请二维码，可刷新
3. **变更手机号**：弹出表单输入新手机号（11位数字，校验格式+唯一性），提交后同时重置微信绑定
4. **同步资料**（CA开启时显示）：仅在authStatus为"未认证"或"认证失败"时可用，将医生信息同步到签章平台
5. **去认证**（CA开启时显示）：仅在authStatus为"认证中"时可用，跳转继续认证流程
6. **设为/取消会诊管理员**：根据当前状态切换

### 第四步：启用/禁用医生账号

列表中的开关控制医生账号状态：

- **启用（status=10）→禁用（status=20）**：弹出确认"是否确认禁用该医生账号，禁用后该医生账号不能正常使用"。
- **禁用→启用**：弹出确认"是否确认启用该医生账号"。

操作即时生效，影响小程序端的可见性和服务能力。

### 第五步：CA签章状态管理

CA列展示两个状态：

1. **CA认证状态**（authStatus）：1-未认证、2-认证中、3-认证成功、4-认证失败
2. **CA用户同步状态**（caSubStatus）：仅在医网信/源康健PaaS平台有效，展示从"未审核"到"用户停用"的7种状态

管理员可通过「同步资料」主动将医生信息同步到签章平台，或通过「去认证」协助医生完成认证。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 变更手机号时新号码已存在 | 实时校验提示"手机号码已存在，请重新输入" |
| 变更手机号格式错误 | 提示"手机号格式错误" |
| 变更手机号为空 | 提示"手机号不能为空" |
| 同步资料时认证状态不符 | 按钮置灰不可点击 |
| 搜索条件中科室树加载失败 | 科室下拉为空 |
| 手机号脱敏查看后点击页面任意处 | 自动恢复脱敏显示 |

## 关联功能

- **医生信息编辑**：跳转到doctorlist/info页面
- **患者管理**：通过患者标签关联
- **CA签章管理**：签章平台对接
- **会诊管理**：会诊管理员权限控制
