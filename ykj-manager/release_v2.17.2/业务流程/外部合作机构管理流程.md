# 外部合作机构管理流程

> 文档版本 2026-07-02 | 面向读者：产品、运营

## 概述

平台支持与外部医疗机构合作对接，允许第三方系统通过开放接口与平台进行数据交互（如外部医院推送患者、对接问诊等）。运营人员在后台管理"外部合作机构"的入驻信息，包括机构名称、接口密钥、回调密钥等，并可随时启用/禁用机构。

## 触发条件

- 有新的合作机构需要对接平台
- 需要管理已有机构的接口密钥和状态

---

## 步骤详解

### 第一步：查看机构列表

进入"外部机构管理"页面，系统展示所有已注册的合作机构列表：

| 列名 | 说明 |
|------|------|
| 机构ID | 系统自动生成，全局唯一标识 |
| 机构名称 | 机构中文名称（如"某某市人民医院"），最长30字 |
| 机构密钥 | 用于接口调用认证，右键自动复制 |
| 回调密钥 | 用于回调数据签名验证，右键自动复制 |
| 创建时间 | 机构入驻时间 |
| 机构状态 | 启用/禁用，可在列表中直接切换开关 |
| 操作 | 编辑按钮 |

### 第二步：筛选查询

支持按以下条件筛选：
- **机构名称**：模糊搜索，输入机构名称关键字
- **状态**：全部/启用/禁用，下拉选择

### 第三步：新增机构

点击"新增机构"按钮，弹出新增表单：

**表单字段**：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 机构ID | 系统自动生成，不可编辑 | 自动 |
| 机构名称 | 输入机构中文名称，最长30字 | 是 |
| 机构密钥 | 系统自动生成随机密钥，不可编辑，可重置 | 自动 |
| 回调密钥 | 系统自动生成随机密钥，不可编辑，可重置 | 自动 |
| 默认值 | 状态默认启用 | - |

新增时机构密钥和回调密钥由系统自动生成全局唯一标识（GUID），确保安全性。

### 第四步：编辑机构

点击列表中的"编辑"按钮，弹出编辑表单：
- 可修改机构名称
- 机构密钥和回调密钥不可直接编辑，但可以"重置"生成新的密钥
- 保存后立即生效

### 第五步：启用/禁用机构

列表中有启用/禁用的开关按钮，点击即可切换：
- **启用**：该机构可以正常调用平台接口
- **禁用**：该机构无法调用接口，正在进行的对接将中断

操作即时生效，无需二次确认。

### 第六步：复制密钥

列表中的"机构密钥"和"回调密钥"列支持右键点击自动复制到剪贴板：
- 右键点击密钥列 → 自动复制
- 复制成功提示"机构密匙复制成功"或"回调秘钥复制成功"

---

## 数据校验

| 校验项 | 规则 | 提示 |
|--------|------|------|
| 机构名称 | 必填，最长30字 | "请输入机构名称" |
| 机构密钥 | 必填，新增时自动生成 | - |
| 回调密钥 | 必填，新增时自动生成 | - |

---

## 异常处理

| 情况 | 处理方式 |
|------|---------|
| 机构名称为空 | 前端校验拦截，提示"请输入机构名称" |
| 接口调用失败 | 前端提示错误信息，不关闭弹窗 |

## 关联功能

- 开放平台接口：机构的密钥用于调用平台开放接口时认证
- 处方对接：外部机构通过此通道上传处方和患者信息
- 问诊对接：外部机构通过此通道创建问诊单
