# 门诊服务订单管理流程（管理后台）

> 文档版本：release/v2.17.2 | 面向读者：运营

## 概述

运营人员在管理后台查看和管理所有门诊服务订单。支持按问诊编号、服务类型两个维度搜索筛选，分页浏览订单列表。核心功能包括：查看图文问诊的聊天对话记录（支持两种消息类型——旧版worn格式和新版IM格式）、查看视频问诊的回放视频列表并播放、查看知情同意书确认信息、查看AI问诊对话记录、查看AI问诊报告。

## 触发条件

### 前置条件
- 运营人员已登录管理后台
- 有相应模块的查看权限

### 入口路径
- **管理后台** → 诊疗服务管理 → 门诊服务订单列表

## 步骤详解

### 第一步：搜索筛选订单

页面提供两种搜索条件：

1. **问诊编号**：输入框，支持精确搜索问诊单号，最多30字
2. **服务类型**：多选下拉框，支持同时选择多个类型
   - 图文问诊（1）、视频问诊（2）、咨询服务（3）、快速问诊（4）
   - 快速续方（5）、医生复诊（7）、商城问诊（8）

点击查询后从第1页开始加载，支持分页浏览。

### 第二步：查看图文问诊聊天记录

点击订单列表中的「聊天记录」按钮，弹出对话框。对话框内根据消息类型分两种展示模式：

**旧版消息（worn类型）**：
- 展示文字、图片、语音等消息
- 语音消息可以播放/停止
- 每条消息显示发送者和时间

**新版IM消息**：
- 支持富文本消息、自定义处方消息、图片消息等
- 群聊场景下显示群成员信息
- 自定义消息支持交互按钮（如「确认领取」「去支付」）

### 第三步：查看视频问诊回放

点击「视频」按钮，弹出视频列表对话框：
- 每条记录显示视频缩略图、标题「问诊视频」、创建时间
- 点击「播放」按钮在另一个弹窗中播放视频
- 如果视频列表为空，显示「暂无数据」空状态

### 第四步：查看知情同意书

点击相关按钮弹出「知情同意书确认信息」弹窗，展示：
- 同意书文本内容
- 确认人姓名
- 网络IP
- 设备信息
- 微信用户openId
- 同意书签署时间

仅「关闭」按钮。

### 第五步：查看AI问诊记录

点击「AI记录」按钮，弹出AI问诊对话弹窗：
- 对话气泡左右分布：AI（左，带头像+AI标识）vs 患者（右，带头像）
- AI消息支持普通文本和结构化卡片两种类型
- 卡片类型展示标题+键值对信息+交互按钮
- 患者消息支持文本、图片展示
- 图片可点击预览大图

### 第六步：查看AI问诊报告

点击「报告」按钮，弹窗展示AI问诊报告的四部分：
1. **主诉**：蓝色区域，展示chiefComplaint
2. **病史信息**：包括现病史、既往史、过敏史、家族史、个人史、月经史、婚育史，有空则不显示该行
3. **体检**：辅助检查信息
4. **AI诊断**：蓝色标签展示诊断名称，底部黄色免责声明「仅供参考，具体以真实医生诊断为准」

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 视频列表为空 | 显示空状态图标 |
| 聊天记录加载中 | 显示loading动画 |
| 无知情同意书记录 | 不显示对应按钮（待确认） |

## 关联功能

- 问诊工作台（inquiryOffice）：更细粒度的单次问诊管理
- 处方管理、评价管理、订单管理
