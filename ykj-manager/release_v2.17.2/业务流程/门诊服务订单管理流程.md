# 门诊服务订单管理流程

> 文档版本 v1.0 | 面向运营人员和管理员

## 概述

门诊服务订单管理是运营人员在后台查看和管理所有线上问诊订单的功能。支持按多种条件筛选问诊记录，查看问诊对话/视频回放，对进行中的问诊做结束处理等。

## 触发条件

- 运营人员需要在后台查看问诊服务记录
- 需要追溯某个问诊的对话内容或视频记录
- 需要人工结束某个进行中的问诊
- 需要核实患者的知情同意书签署信息

## 筛选条件

### 基础搜索字段
| 筛选项 | 说明 |
|--------|------|
| 问诊编号 | 精确搜索，最多输入30个字符 |
| 服务类型 | 多选：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊、AI图文问诊 |
| 问诊流程 | 多选：在线问诊、线上复诊、慢病医保、预约挂号、快速开方、AI问诊 |
| 问诊状态 | 单选：已预约、问诊中、问诊结束、取消问诊。已预约和问诊中的区分通过签到状态决定 |
| 支付时间 | 日期范围选择 |
| 问诊时间 | 日期范围选择 |
| 就诊时间 | 日期范围选择（病人实际就诊的预约时间段） |
| 有无病历 | 单选：有/无 |
| 医生姓名 | 精确搜索，最多5个字符 |
| 患者姓名 | 精确搜索，最多5个字符 |
| 所属机构 | 仅当系统开启多机构模式时显示 |
| 所属团队 | 仅当系统开启专家团队功能时显示 |

### 特别说明
- "AI问诊"流程选项仅在系统开启AI医生功能后出现
- "AI图文问诊"服务类型选项同样受AI医生开关控制
- 问诊状态使用复合值（如"20,0"表示已预约未签到，"20,1"表示问诊中已签到）。选择"全部"时清空该条件

### 操作按钮
| 按钮 | 显示条件 | 作用 |
|------|---------|------|
| 查看详情 | 始终显示 | 查看问诊对话记录（普通IM消息或AI预问诊对话） |
| 结束问诊 | flowStatus=20（问诊中） | 人工结束进行中的问诊，操作前二次确认 |
| 视频记录 | serviceType=2（视频问诊） | 查看视频问诊的录制回放列表 |
| 知情同意书确认信息 | inquiryIcfId不为空 | 查看患者的知情同意书签署详情 |

## 操作详解

### 查看详情（问诊对话记录）
点击"查看详情"后弹出对话记录窗口：

1. **普通问诊**（非AI问诊）：加载IM即时通讯的对话记录。首次进入加载最新20条消息，向上滚动时自动加载更早的消息（滚动到顶部触发加载上一页）。支持播放语音消息。

2. **AI图文问诊**（serviceType=9）：加载AI预问诊的对话记录，展示用户与AI的问答过程。包括：
   - AI端：显示医生头像+AI标识角标
   - 用户端：显示患者头像
   - 卡片类型消息：如"病情卡片"展示主诉、病史等信息
   - 底部按钮：支持"查看报告"等自定义操作

3. **AI问诊报告**：AI预问诊结束后可查看报告，包括主诉、病史信息（现病史/既往史/过敏史/家族史/个人史/月经史/婚育史）、体检（辅助检查）、AI诊断结果。AI诊断结果下方有"仅供参考"的免责提示。

### 查看知情同意书
弹出"知情同意书确认信息"窗口，显示：
- 确认人姓名
- 网络IP地址
- 设备信息
- 微信用户openId
- 同意书签署时间

### 结束问诊
仅问诊中（flowStatus=20）的订单可结束。操作前弹出确认框，确认后调用结束接口。

### 视频记录查看
仅视频问诊（serviceType=2）显示。弹出后展示该问诊的所有视频录制列表，每条显示录制时间。点击"播放"在新弹窗中播放视频。

## 异常处理

| 情况 | 处理方式 |
|------|---------|
| 查询结果为空 | 列表显示空数据 |
| 查询结果超出当前页 | 自动回退到上一页重新查询 |
| 问诊编号为空 | 提示"问诊编号为空！" |
| 没有找到对话记录 | 提示"没有找到相关问诊对话记录!" |
| 患者数据脱敏 | 患者姓名只显示首字+星号，联系方式需要点击获取明文（每次只能查看一条的明文） |
| 视频列表为空 | 显示空状态图标 |

## 关联功能

- 关联问诊记录中的处方信息
- 关联知情同意书签署记录
