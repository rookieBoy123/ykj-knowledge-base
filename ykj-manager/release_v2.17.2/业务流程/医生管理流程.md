# 医生管理流程（管理后台）

> 文档版本：release/v2.17.2 | 面向读者：运营、客服、产品经理

---

## 概述

管理后台运营人员管理已审核通过的医生，包括查看医生信息、启用/禁用医生、变更手机号码、生成医生二维码、管理医生排班等。

## 触发条件

- 运营人员需要查看和管理已上线的医生
- 医生反馈账号问题需要后台处理
- 需要为医生生成推广二维码

## 前置条件

- 运营人员拥有医生管理权限
- 医生已完成审核（状态为"已通过"）

## 步骤详解

### 步骤一：进入医生列表

**入口路径：** 左侧菜单 → 医院管理 → 医生列表

**搜索条件：**
- 医生姓名（输入框，最多5个字）
- 工作类型（全部/医生/医助）
- 科室（树形下拉选择）
- 所属医院（远程搜索下拉）
- 职称（下拉选择）
- 账号状态（全部/启用/禁用）
- 所属机构（如有机构功能则显示）
- 开启服务（多选）：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊

### 步骤二：查看医生信息

表格展示已审核通过的医生信息，特色列：

- **电话号码**：脱敏显示，点击可查看完整号码
- **账号状态**：启用（状态10）/ 禁用（状态20）
- **开启服务**：展示医生当前开启的问诊服务类型

### 步骤三：管理医生状态

**启用/禁用操作（在医生详情页中）：**
- **启用医生**（status=10）：医生正常使用，患者可搜索到
- **禁用医生**（status=20）：医生无法登录或使用，患者搜索不到
- 禁用后已有的问诊和处方不受影响

### 步骤四：生成推广二维码

点击列表中的二维码按钮：
1. 弹出二维码展示窗口
2. 显示医生的专属推广二维码
3. 支持"刷新二维码"功能重新生成
4. 患者扫描二维码可直接跳转到该医生的详情页

### 步骤五：变更手机号码

如需修改医生的绑定手机号：
1. 点击"变更手机号码"按钮
2. 在弹出的表单中输入新的手机号码（11位）
3. 系统自动校验：
   - 手机号格式是否正确（1开头的11位数字）
   - 手机号是否已被其他医生或药师使用（唯一性校验）
4. 确认后完成变更

### 步骤六：管理医生排班

**医生排班页面**支持：
- 为医生设置出诊时间（按周设置，如周一上午、周三下午等）
- 支持排班模板：可创建排班模板，批量应用到多位医生

**排班模板功能：**
- 创建排班模板：设置通用的出诊时间段
- 应用模板：选择一个排班模板应用到指定医生
- 管理模板：编辑、删除已有模板

## 异常处理

| 异常场景 | 提示 | 处理方式 |
|---------|-----|---------|
| 手机号码已存在 | "手机号码已存在，请重新输入" | 确认号码未被占用或联系原号主解绑 |
| 手机号格式错误 | "手机号格式错误" | 输入正确的11位手机号 |
| 医生禁用后患者仍能看到 | 可能有小程序缓存 | 患者端下拉刷新或重新进入 |
| 二维码过期 | 重新生成 | 点击"刷新二维码"按钮 |
