# 健康服务包订单管理与核销流程

> 文档版本 2026-07-02 | 面向读者：运营、产品

## 概述

运营人员在管理后台管理患者购买的健康服务包订单，包括订单查询、导出，以及线下核销功能。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 健康服务包 → 服务包订单
- **前置条件**：具有运营管理权限

## 步骤详解

### 一、订单列表查询

**Tab快捷筛选**（按订单状态）：

| Tab | 状态值 | 说明 |
|-----|--------|------|
| 全部 | - | 所有状态的订单 |
| 待支付 | 10 | 已下单未付款 |
| 已付款 | 20 | 已完成付款，待核销 |
| 已完成 | 60 | 已核销完成 |
| 已取消 | 70 | 用户主动取消 |
| 退款中 | 89 | 退款处理中 |
| 退款成功 | 90 | 退款已完成 |

**详细筛选条件**：
- 订单编号（最多20字）
- 服务包名称（最多30字）
- 联系电话（最多11位）
- 创建时间（日期范围）

### 二、订单列表字段

| 字段 | 说明 |
|------|------|
| 订单编号 | 唯一订单号 |
| 服务包名称 | 购买的服务包 |
| 服务项目 | 包含的项目名称 |
| 订单金额（元） | 实付金额 |
| 联系电话 | 默认脱敏，点击查看明文 |
| 状态 | 当前订单状态 |
| 创建时间 | 下单时间 |
| 订单支付时间 | 付款时间 |
| 操作 | 查看详情 |

### 三、订单详情

点击「查看详情」跳转订单详情页，可查看完整订单信息。

### 四、订单核销

点击「订单核销」按钮弹出核销弹窗，输入订单号进行核销确认。核销后订单状态从「已付款」变为「已完成」。

### 五、导出订单

点击「导出」按钮，将当前筛选条件下的订单列表导出为Excel文件（.xls格式），导出过程中显示加载提示。

### 六、电话号码脱敏保护

列表中的联系电话默认以脱敏形式显示（如138****1234）。点击脱敏字段后需验证权限，验证通过后才显示完整号码。同时只有一个号码处于明文状态。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 列表无数据 | 显示空列表 |
| 导出数据量大 | 显示加载中提示，完成后自动下载 |
| 电话查看权限 | 需权限验证才能查看明文 |

## 关联功能

- **服务包配置**：服务包的名称、价格来源于服务包管理中的配置
- **患者端订单**：患者在患者端下单后数据同步到后台
