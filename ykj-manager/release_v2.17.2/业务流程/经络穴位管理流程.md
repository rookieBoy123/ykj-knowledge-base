# 经络穴位管理流程

> 文档版本：2026-07-03 | 面向读者：运营人员、内容管理员

## 概述

经络穴位是医学百科下的一个内容管理模块，用于维护人体穴位的信息，包括穴位名称、归属经络、身体部位、功效作用等。每个穴位可附带封面图。管理员可以新增、编辑、删除经络穴位条目。新增和编辑操作通过弹窗完成。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 医学百科 → 经络穴位
- **前置条件**：拥有对应菜单权限的管理员

## 步骤详解

### 第一步：浏览经络穴位列表

进入页面后，系统自动加载经络穴位列表，每页显示10条记录。列表展示以下信息：
- **封面图**：穴位的展示缩略图
- **穴位名称**：穴位的标准名称
- **归属经络**：该穴位所属的经络名称（如手太阴肺经、足阳明胃经等）
- **身体部位**：该穴位在身体上的位置描述
- **功效作用**：穴位的主要功效
- **最后修改时间**：最近编辑时间

管理员可以通过筛选条件过滤数据：
- **穴位名称**：按名称搜索（最多10字）
- **归属经络**：下拉选择，从系统字典 `jlxw-jl` 获取
- **身体部位**：下拉选择，从系统字典 `jlxw-bw` 获取

### 第二步：新增经络穴位

点击「新增」按钮，弹出新增弹窗。弹窗包含以下字段：
- **穴位名称**：必填，最多10字
- **归属经络**：下拉选择（必填）
- **身体部位**：下拉选择（必填）
- **展示封面图**：上传封面图
- **功效作用**：通过富文本编辑器编辑
- **穴位详情**：通过富文本编辑器编辑

### 第三步：编辑和删除

编辑通过弹窗完成，自动回显已有数据。删除支持单条和批量，均有二次确认弹窗。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 穴位名称为空 | 表单校验提示"请填写穴位名称" |
| 归属经络未选择 | 表单校验提示"请选择归属经络" |
| 身体部位未选择 | 表单校验提示"请选择身体部位" |
| 批量删除未勾选 | 静默无响应 |

## 关联功能

- 通过服务端 API 提供给小程序端的经络穴位展示
- 归属经络和身体部位选项来源于系统字典
