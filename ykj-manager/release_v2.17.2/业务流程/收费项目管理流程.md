# 收费项目管理流程

> 文档版本：v2.17.2 | 面向读者：运营

## 概述

管理员在后台管理医院收费项目（如挂号费、检查费、治疗费等），支持新增、编辑、启用/禁用收费项目，用于医保结算和费用计算。

## 触发条件

- 管理员登录后台后，进入「收费项目管理」菜单
- 需要新增服务收费项目或调整费用标准

## 步骤详解

### 第一步：筛选收费项目

管理员可以通过以下条件筛选：

| 筛选条件 | 说明 |
|---------|------|
| 费用编码 | 精确搜索，最多20个字符 |
| 费用名称 | 模糊搜索，最多100个字符 |
| 费用类别 | 下拉选择（选项从字典"fylb"动态加载，如挂号费、检查费、治疗费等） |
| 状态 | 全部/启用/禁用 |

点击「查询」按钮，每页显示10条记录。

### 第二步：浏览收费项目列表

列表展示以下信息：

| 列名 | 说明 |
|------|------|
| 费用编码 | 系统内部编码 |
| 医保编码 | 医保系统对接使用的编码（insuranceCode） |
| 费用名称 | 收费项目的名称 |
| 费用类别 | 所属类别（如挂号费、检查费） |
| 单价（元） | 收费金额 |
| 单位 | 计费单位（如"次"、"项"） |
| 状态 | 启用/禁用的开关按钮 |
| 操作 | 编辑按钮 |

**列表头部有「新增」按钮**，需要「hosp:charge-items:write」权限才可见。

### 第三步：新增收费项目

点击「新增」按钮，弹出编辑表单窗口（通过editItem子组件承载），需要填写：

- 费用编码（必填，唯一）
- 医保编码（用于医保对接）
- 费用名称（必填）
- 费用类别（从字典下拉选择）
- 单价（元）
- 单位
- 状态（启用/禁用）

填写完成后点击保存，列表自动刷新。

### 第四步：编辑收费项目

点击列表中的「编辑」按钮，打开相同的编辑窗口，字段预填当前值，修改后保存即可。

### 第五步：启用/禁用收费项目

列表中的「状态」列为开关按钮：

- **启用 → 禁用**：点击开关关闭，调用chargeDisable接口，同时会影响前端小程序相应服务的定价计算
- **禁用 → 启用**：点击开关打开，调用chargeEnable接口

切换后立即生效，无需额外确认。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 费用编码重复 | 后端校验并返回错误提示 |
| 费用类别字典加载失败 | 费用类别下拉框为空，需联系技术人员排查字典配置 |
| 没有「hosp:charge-items:write」权限 | 新增按钮不显示，只能查看 |

## 关联功能

- 与「医保结算」关联：医保编码字段用于上传医保系统的费用明细
- 与「处方结算」关联：部分服务类收费项目会影响处方费用计算
- 与「字典管理」关联：费用类别选项来自字典"fylb"的维护
