# 医保慢病病种关联科室管理流程

> 文档版本 v1.0 | 面向读者：医保管理员

## 概述
医保慢病（慢性病）病种管理用于配置平台支持的慢性病类型，每个病种可关联医院的科室，用于后续医保结算时匹配病种和对应的诊疗科室。管理员可查看病种列表，编辑病种的关联科室和启用/禁用状态。

## 触发条件
- **前置条件**：拥有慢病管理权限的管理员
- **入口路径**：系统管理 → 医保管理 → 慢病病种管理

## 步骤详解

### 第一步：查看病种列表
进入页面后展示所有已配置的慢病病种列表，包含：
- **序号**：自动编号
- **病种编码**：病种在医保系统中的唯一编码
- **病种名称**：如高血压、糖尿病等
- **关联科室**：该病种关联的医院科室列表，多个科室用逗号分隔显示
- **状态**：启用/禁用开关按钮
- **操作**：编辑按钮

### 第二步：搜索病种
可按以下条件筛选：
- **病种名称**：模糊搜索匹配病种名称
- **病种编码**：精确搜索病种编码

### 第三步：编辑病种关联科室
- 点击某条病种的"编辑"按钮，弹出编辑对话框
- 对话框中显示病种的**名称**和**编码**（只读，不可修改）
- 关键操作是选择**关联科室**：使用树形选择器从医院科室树中勾选科室，支持多选
- 科室数据来源于医院科室管理模块的科室树结构

### 第四步：启用/禁用病种
- 在列表的"状态"列中，每个病种有一个开关按钮
- **启用状态**（绿色）：病种生效，可以在医保结算中使用
- **禁用状态**（灰色）：病种不生效，医保结算中不可选
- 切换状态时系统自动调用接口，操作成功后有"修改成功"提示

## 异常处理
| 情况 | 系统处理方式 |
|------|-------------|
| 编辑时未选择关联科室 | 表单校验提示"请选择关联科室"，阻止保存 |
| 编辑时科室树加载失败 | 科室下拉框为空，无法选择科室 |
| 病种编码被医保系统修改 | 病种名称仍显示，但关联科室和状态可能需重新配置 |
| 网络请求失败 | 列表不更新，停留在上一次成功加载的数据 |

## 关联功能
- **医保结算申报**：选择病种时引用此处的病种列表
- **医保标准药品目录**：药品的收费等级与病种关联
- **科室管理**：此处的科室树来源于科室管理模块
