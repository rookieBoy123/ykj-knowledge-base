# 已入驻医生信息编辑与服务项目配置流程

> 文档版本 v1.0 | 面向运营人员

## 概述

已通过审核正式入驻的医生，其基础档案和服务配置可能需要后续调整。运营人员在管理后台医生列表中点击某位医生进入详情编辑页，可以修改医生的基础信息、从业资料、资格证书、服务项目开启状态与价格、以及接诊模式等配置。

## 触发条件

- **前置条件**：医生已通过审核（flowStatus=20），运营人员有编辑权限
- **入口路径**：医院管理 → 医生管理 → 医生列表 → 点击某位医生进入详情

## 步骤详解

### 第一步：查看与编辑基础信息

页面展示医生的完整信息，分为"基础信息""从业资料""资格证书""执业证书"等板块。

**编辑权限控制**：当医生已有线上接诊记录（isClear > 0 或 onlineStatus = 1）时，大部分字段变为禁用状态（isForbidden=true），包括：性别、出生日期、所在科室、医生职务、全科医生、开始执业日期、执业医院、职务类别。

始终禁用的字段：医生姓名、手机号码。

### 第二步：修改从业资质

证书图片和编号等从业资质信息可在编辑模式下修改。从业资料区域包含：
- 主要执业医院（远程搜索）
- 职务类别（字典选择）
- 全科医生（是/否）
- 开始执业日期
- 上海地区额外字段：参加工作日期、执业机构

### 第三步：配置服务项目

已入驻医生可配置以下8种服务类型，每种服务有独立的开关和价格设置：

| 服务类型 | 价格字段 | 时长字段 | 依赖条件 |
|---------|---------|---------|---------|
| 图文问诊 | fee（元） | duration（小时） | 基础服务 |
| 视频问诊 | fee（元） | duration + durationUnit | — |
| 咨询服务 | fee（元） | duration + durationUnit | — |
| 快速续方 | fee（续方诊金，元） | 无 | 必须同时开启图文问诊 |
| 会诊服务 | fee（元），受职务价格区间限制 | 无 | — |
| 医生复诊 | fee（元），与处方订单合并收取 | 无 | — |
| 快速问诊 | 无（免费） | 无（24小时） | 平台服务，医生不可自行取消 |
| 商城问诊 | 无（免费） | 无（24小时） | 需系统开启商城功能 |

### 第四步：配置辅助功能

- **医助辅助接诊**：开启图文或视频问诊后可勾选，系统自动匹配医助
- **AI预问诊**：需IPC功能开启，AI助手辅助收集病史
- **隐藏诊费设置**：勾选后医生在小程序端无法修改服务状态和费用

### 第五步：设置接诊模式

预约功能开启时显示：
- **实时接诊模式** vs **预约接诊模式**（需设置排班）
- 预约模式下可设"过号是否接诊"

### 第六步：保存修改

点击"确定"按钮后：
1. 先调用 updateDoctorService 保存服务项目配置（图文问诊的时长从小时转换为分钟）
2. 检查快速续方依赖：如果快速续方开启但图文问诊未开启，阻止保存并提示
3. 调用编辑接口保存医生信息
4. 成功后提示"保存成功"并返回上一页

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 表单必填项未填 | 提示"请完善资料" |
| 快速续方开启但图文问诊未开 | 弹出警告："开启快速续方的同时请开启图文问诊"，阻止保存 |
| 已有线上记录的医生 | 基础信息字段置灰不可编辑（isForbidden） |
| 医师姓名/手机号 | 始终禁用不可修改 |
| 会诊诊金超出职务允许范围 | 输入框有 min/max 限制 |
| 快速问诊/商城问诊 | 医生不可自行取消，仅后台可操作 |

## 与审核详情页的区别

| 对比维度 | 审核详情页 | 已入驻编辑页 |
|---------|-----------|------------|
| 场景 | 新增审核/驳回重审 | 已通过审核的医生 |
| 姓名可编辑 | 仅待审核状态可编辑 | 始终禁用 |
| 审核操作 | 有"驳回"按钮 | 无驳回操作 |
| 全平台展示 | 审核编辑时可选 | 已入驻编辑时可改 |
| 测试账号 | 可设置 | 不显示 |
| 接诊模式 | 仅编辑时显示 | 编辑时显示 |

## 关联功能

- **医生列表**：从列表进入详情
- **医生排班管理**：预约接诊模式下关联排班
- **医生收益结算**：服务价格影响收益计算
