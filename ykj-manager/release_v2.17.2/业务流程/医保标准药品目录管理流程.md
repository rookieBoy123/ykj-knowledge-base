# 医保标准药品目录管理流程

> 文档版本 v1.0 | 面向读者：医保管理员、药房管理员

## 概述
医保标准药品目录是平台对接医保结算的基础数据。管理员可在后台查看和管理西药（成药）和中药饮片两类医保标准药品目录，支持按收费等级、药品名称、药品代码、药品本位码等多维度查询，并可下载模板、批量导入和导出数据。

## 触发条件
- **前置条件**：拥有"医保药品导入"或"医保药品导出"权限的管理员
- **入口路径**：系统管理 → 医保管理 → 标准药品目录

## 步骤详解

### 第一步：选择药品类型标签页
进入页面后，顶部显示两个标签页切换：
- **成药类型**：展示西药标准药品目录，包含药品代码、注册名称、商品名称、注册剂型、实际剂型、注册规格、实际规格、包装材质、最小包装数量/单位、最小剂量单位、药品企业、批准文号、药品本位码、分包装企业、上市状态、通用名称、收费项目等级、城职/城乡医保先支付比例等字段
- **中药饮片**：展示中药饮片目录，包含中药饮片名称、代码、药材名称、炮制方法、功效分类、科族、药材种来源、药用部位、性味与归经、功能与主治、用法与用量、医保支付政策等字段

### 第二步：按条件搜索药品
可根据以下条件筛选：
- **收费项目等级**：甲类、乙类、丙类（从字典数据动态加载）
- **注册名称/商品名称**（成药类型）
- **通用名称/标准名称**
- **药品代码/编码**
- **药品本位码**
- **中药名称/药材名称/中药代码**（中药饮片）

### 第三步：下载模板和导入
- 点击"下载模板"按钮，根据当前药品类型（成药/中药）下载对应的Excel模板
- 点击"导入"按钮上传填好的Excel文件，系统会显示"正在导入数据"的状态提示
- 导入成功后，系统自动记录导入版本号到本地存储，用于后续查询导入结果

### 第四步：导出数据
- 点击"导出"按钮，将当前筛选条件下的数据导出为Excel文件

## 数据字段说明
- **收费项目等级**：分为甲类（0）、乙类（1）、丙类（2），影响医保报销比例
- **城职医保先支付比例**：城镇职工医保的自付比例
- **城乡医保先支付比例**：城乡居民医保的自付比例
- 数据来源于医保基础数据映射目录，建议每年手动更新导入一次

## 异常处理
| 情况 | 系统处理方式 |
|------|-------------|
| 导入文件格式不符 | 提示上传失败，需使用下载的标准模板填写 |
| 药品编码重复 | 导入时自动跳过或更新已有记录 |
| 收费等级数据未加载 | 查询条件中的收费等级下拉框为空，页面静默处理 |
| 缺少导入/导出权限 | 对应的"下载模板""导入""导出"按钮不显示 |

## 关联功能
- **医保慢病管理**：慢病病种关联科室配置中使用此处标准药品目录
- **医保结算申报**：医保结算时需要匹配标准药品编码
- **处方审核**：药师审核处方时参考收费等级和医保支付比例
