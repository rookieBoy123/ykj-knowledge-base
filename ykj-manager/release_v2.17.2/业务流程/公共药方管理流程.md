# 公共药方管理流程

> 文档版本 v1.0 | 面向读者：运营管理人员

## 概述

运营人员在后台管理公共中药药方模板，支持新增、编辑、删除和查看药方明细，医生开方时可选择公共药方快速引用。

## 触发条件

- 前置条件：拥有药方管理权限（med:prescriptions:write）
- 入口路径：管理后台 → 医院管理 → 药品信息 → 公共药方

## 步骤详解

### 第一步：查看药方列表

进入页面后展示药方列表，包含以下列：
- 药方编码（系统自动生成）
- 药方名称
- 药品列表（显示所有药品名称，逗号分隔）
- 更新时间

支持按药方名称（最长50字）和药方编码（最长32字）进行搜索。

### 第二步：新增公共药方

点击"新增"按钮 → 弹出新增药方窗口（960px宽）→ 填写信息：
- 药方名称（必填，否则提示"请输入药方名称"）
- 药方说明（选填）
- 药品清单（必填，必须至少添加一种药品，否则提示"请添加药品"）

药品清单从药房的药品库中选择，支持选择对应的药房渠道。

### 第三步：编辑药方

点击列表中的"编辑"按钮 → 弹出编辑窗口 → 加载药方当前信息 → 修改后保存。

### 第四步：查看药方详情

点击"详情"按钮 → 弹出药方明细窗口（700px宽），展示：
- **药方信息**：药方名称、药方类型（中药类型）、创建时间、药方说明
- **药方清单**：每种药品的名称和重量（克），并提示"单剂重量XX克（非克药材和辅料未计入）"

### 第五步：删除药方

点击"删除"按钮 → 确认弹窗"是否确认删除该公共药方?" → 确认后删除 → 提示"操作成功"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药方名称为空 | 提示"请输入药方名称" |
| 未添加药品 | 提示"请添加药品" |
| 删除确认 | 提示"是否确认删除该公共药方?" → 取消提示"已取消删除" |

## 关键规则

- **药方编码**：由系统自动生成，不可手动修改
- **药品来自药品库**：药方中的药品从药房的药品库中选取，必须属于某个药房渠道
- **药方类型固定**：当前仅支持"中药类型"
- **权限控制**：新增/编辑/删除需要 med:prescriptions:write 权限，无权限时按钮隐藏

## 关联功能

- 药房渠道管理（药方需关联药房）
- 医生开方（选取公共药方快速引用）
- 药品库管理
