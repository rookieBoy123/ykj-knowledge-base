# 门诊问诊室（医生工作台）管理流程

> 文档版本 2026-07-03 | 面向角色：后台管理员

## 概述

门诊问诊室是管理后台提供的医生工作台功能，管理员可以以医生身份登录后台，查看问诊单列表，进入问诊室查看和管理患者的问诊——包括接诊、结束问诊、对话沟通、查看病历、撰写病历和开具医嘱。页面采用三栏布局：左侧问诊单列表、中间会话区域、右侧问诊资料。

## 触发条件

- **前置条件**：当前登录的必须是"医生"角色（roleCode=doctor）
- **入口路径**：后台菜单 → 诊疗管理 → 门诊问诊室

## 步骤详解

### 第一步：查看问诊单列表

左侧面板展示三种状态的问诊单：
- **待接诊**（wait）：患者已预约但医生尚未接诊
- **问诊中**（underway）：医生正在处理的问诊
- **已结束**（finish）：已完成的问诊

点击任一问诊单，右侧会加载该问诊的详细信息。

### 第二步：了解问诊详情

中间区域展示问诊基本信息：患者姓名、问诊编号、服务类型等。右侧面板展示问诊资料，包括患者详情、病历记录、处方记录等。

### 第三步：接诊操作

选中待接诊的问诊单后，点击"接诊"按钮，确认后该问诊状态从"待接诊"变为"问诊中"。接诊成功后页面刷新。

### 第四步：问诊中的操作

**对话沟通**：中间区域展示与患者的 IM 聊天界面，可以发送文字、图片消息，与患者实时交流。

**写病历**：点击"写病历"后右侧面板切换为病历编辑模式，可以新增或编辑患者的问诊病历。提交成功后自动刷新问诊详情。

**发起视频**：点击"发起视频"后系统创建视频房间，向患者发送视频邀请卡片消息，患者点击加入后可以进行视频问诊。

**开医嘱**：点击"开医嘱"前必须先完成病历填写，如果病历为空会提示"请先写病历再开医嘱！"。点击确定后弹出医嘱填写对话框。

### 第五步：结束问诊

问诊结束后点击"结束问诊"，系统弹出确认框，确认后问诊状态变为"已结束"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 非医生角色访问 | 页面显示"请登录医生角色查看!" |
| 未选择问诊单 | 中间显示"选择患者查看详细信息" |
| 开医嘱时无病历 | 弹窗提示"请先写病历再开医嘱!" |
| 病历保存失败 | 后台返回错误提示 |
| 视频请求失败 | 提示"响应错误，请联系管理人员" |

## 关联功能

- 门诊服务订单管理（outpatientserviceorder）
- 病历管理（casehistory）
- 处方管理（recipemanage）
- 视频记录查看
