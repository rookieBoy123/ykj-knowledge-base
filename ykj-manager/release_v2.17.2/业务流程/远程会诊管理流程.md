# 远程会诊管理流程

> 文档版本 1.0 | 面向非技术读者 | 2026-07-01

## 概述

管理员（远程医疗管理员角色）在后台管理系统管理跨机构的远程会诊申请。覆盖从新增会诊申请、编辑、提交、撤回、收费、开始会诊、结束会诊到归档的全生命周期操作。仅"远程医疗管理员"角色可访问此页面。

## 触发条件

- 角色要求：必须是"远程医疗管理员"身份
- 入口：后台管理系统 → 诊疗业务 → 远程医疗中心 → 远程会诊

## 步骤详解

### 第一步：会诊列表查询

进入页面后，管理员可以通过以下条件筛选会诊记录：

| 筛选条件 | 说明 |
|---------|------|
| 会诊单号 | 精确输入，最长20字符 |
| 申请科室 | 按科室名称模糊搜索 |
| 申请医师 | 按医生姓名搜索，最长5字 |
| 患者姓名 | 按患者姓名搜索，最长5字 |
| 拟申请会诊机构 | 下拉选择已配置的远程医疗机构 |
| 状态 | 见下方状态说明 |

列表展示14项字段：会诊单号、申请科室、申请医师、患者姓名、患者性别、患者年龄、拟申请会诊机构、拟申请会诊服务、拟申请会诊科室、拟申请会诊医生、拟申请会诊时间、状态、会诊时间、操作。

每页显示10条记录。

### 第二步：状态流转与对应操作

会诊有11个状态，每个状态下可执行不同的操作：

| 状态 | 状态码 | 可执行操作 | 说明 |
|------|--------|-----------|------|
| 草稿 | 1 | 编辑、删除 | 新建未提交的会诊申请 |
| 待应答 | 2 | 撤回 | 已提交，等待对方机构响应 |
| 已撤回 | 9 | 编辑、删除 | 主动撤回的申请 |
| 已拒绝 | 10 | 编辑、取消 | 对方机构拒绝了会诊申请 |
| 待收费 | 3 | 收费、取消 | 对方已同意，等待管理员收费 |
| 待开始 | 4 | 开始会诊、取消 | 已收费，等待会诊时间到达 |
| 进行中 | 5 | 进入会诊、结束会诊 | 会诊正在进行 |
| 已结束 | 6 | 查看详情、补充资料、归档 | 会诊已结束，可补充资料后归档 |
| 已取消 | 8 | — | 会诊已被取消 |
| 已归档 | 7 | — | 会诊已归档，终态 |

### 第三步：新增会诊申请

点击"新增"按钮打开弹窗，管理员填写会诊申请信息：

- 选择申请机构、被邀请机构
- 填写患者信息（姓名、性别、年龄、身份证号等）
- 填写病情描述、会诊目的
- 选择会诊科室和会诊医生
- 设置拟会诊时间
- 上传病历资料（支持图片、PDF、Word、Excel等，单个文件不超过10MB）

提交后申请进入"待应答"状态（flowStatus=2）；也可以保存为"草稿"（flowStatus=1）稍后再编辑。

### 第四步：编辑与删除

- **可编辑状态**：草稿（1）、已撤回（9）、已拒绝（10）
- **可删除状态**：草稿（1）、已撤回（9）
- 编辑时弹窗自动加载已填数据，管理员修改后重新提交

### 第五步：收费操作

当会诊进入"待收费"（flowStatus=3）状态时，管理员可执行收费操作，确认收费后状态变为"待开始"。

### 第六步：开始与进入会诊

- **开始会诊**（仅"待开始"状态可见）：管理员点击后触发会诊正式开始，状态变为"进行中"
- **进入会诊**（仅"进行中"状态可见）：管理员可以进入会诊房间查看/参与

### 第七步：结束会诊与归档

- **结束会诊**（仅"进行中"状态可见）：管理员点击后结束会诊，状态变为"已结束"
- **补充资料**（仅"已结束"状态可见）：管理员可上传补充的治疗方案及效果资料（支持图片、PDF、Word、Excel文件）
- **归档**（仅"已结束"状态可见）：将已结束的会诊标记为"已归档"（终态）

### 第八步：取消会诊

- **可取消状态**：已拒绝（10）、待收费（3）、待开始（4）
- 取消时需要填写取消原因（必填，最长500字）
- 取消后状态变为"已取消"

### 第九步：查看详情

任何状态的会诊记录都可以查看详情（独立的详情弹窗展示完整会诊信息）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 非远程医疗管理员角色访问 | 页面显示"当前页面仅供远程医疗管理员使用！" |
| 取消原因未填写 | 表单校验提示"请输入取消会诊原因" |
| 补充资料未选择文件 | 提示"请选择文件后再保存" |
| 会诊单号输入过长 | 前端限制最多20字符 |
| 医生姓名/患者姓名过长 | 前端限制最多5字符 |

## 关联功能

- **机构管理**：拟申请会诊机构下拉列表从机构管理模块获取
- **医生管理**：拟申请会诊医生从受邀机构的医生列表中选择
- **服务管理**：会诊服务从受邀机构的服务列表中获取
