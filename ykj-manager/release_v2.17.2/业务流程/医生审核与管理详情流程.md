# 医生审核与管理详情流程

> 文档版本 v1.0 | 面向读者：后台运营人员

## 概述
运营人员在后台新增或编辑医生信息，包括基础资料、从业资质、证件证书、服务项目配置、接诊模式设置等。审核通过的医生信息部分字段锁定不可编辑（需先驳回才能修改）。

## 触发条件
- **前置条件**：运营人员有医生管理权限
- **入口路径**：治疗管理 → 医生管理 → 医生列表 → 新增医生 / 点击编辑
- **页面类型**：新增（add）或编辑（edit），编辑时根据流程状态（flowStatus）控制字段可编辑性

## 步骤详解

### 第一步：填写基础信息
顶部区域包含医生的基本个人资料：

- **头像上传**：点击上传 200×200px 或 1:1 比例的图片（支持jpg/png）
- **医生姓名**：最多5个汉字（编辑模式下如非待审核状态则不可修改）
- **性别**：下拉选择男/女
- **手机号码**：11位手机号
- **出生日期**：日期选择器
- **所在科室**：树形下拉选择（从医院科室字典加载），支持搜索
- **医生职务**：下拉选择（从职务字典加载）
- **民族**：仅山东地区显示，下拉选择
- **擅长疾病**：文本域，限500字
- **医生简介**：文本域，限500字
- **对症询医**：多选远程搜索疾病分类（最多选10个），已审核通过的不能修改

### 第二步：填写从业资料
包含医生的执业信息和资质证明：

- **主要执业医院**：远程搜索输入（输入医院名称模糊查询），已审核不可修改
- **职务类别**：下拉选择（从字典加载）
- **全科医生**：单选"是"或"否"
- **开始执业日期**：日期选择器
- **学历**：下拉选择（博士/硕士/本科/大专等）
- **参加工作日期**：仅上海地区显示
- **执业机构**：仅上海地区显示，多选远程搜索
- **执业地点**：仅上海地区显示，文本输入限50字

### 第三步：填写医保信息
- **医保医师编码**：输入框限32位，有国家医保代码库外链供查询
- **提示**：确认填写后可开具医保处方

### 第四步：上传证件证书资料
四大类证件上传和填写：

**1. 身份证**
- 上传正面照片（必传）和反面照片（根据地区配置显示）
- 选择证件类型（居民身份证/护照等）
- 身份证号：限18位，编辑时已审核状态下不可修改

**2. 资格证书**
- 上传证书正面和背面照片
- 资质类别名称：下拉选择（从字典"yszylb"加载）
- 资格证书编号：限255字
- 资格获取时间：日期选择

**3. 执业证书**
- 上传证书正面和背面照片
- 执业证书编号：限255字
- 发证日期：日期选择
- 执业范围：树形下拉选择（从字典"yszyfw"加载），支持搜索

**4. 其他相关证明**
- 最多上传五张图片
- 点击删除按钮可移除已上传图片

### 第五步：配置工作类型与服务
**工作类型选择：**
- 医生：勾选后可配置各项服务
- 医助：辅助医生接诊，做诊前诊后服务

**测试账号标记：**仅用于测试，不进行信息备案，诊疗数据不上传

**全平台可见性：**可在全平台隐藏，只有扫医生邀请码才能看到医生资料

**开启服务项目（6种服务）：**

| 服务类型 | 可配置项 |
|---------|---------|
| 图文问诊 | 开关 + 价格 + 时长（小时/次） |
| 视频问诊 | 开关 + 价格 + 时长 |
| 咨询服务 | 开关 + 价格 + 时长 |
| 快速续方 | 开关 + 续方诊金 |
| 医生复诊 | 开关 + 复诊诊金（与处方订单一起收取） |
| 会诊服务 | 开关 + 会诊诊金（有价格区间限制） |

**医助辅助：**图文或视频问诊开启后可勾选，自动匹配医助做诊前诊后服务

**AI预问诊：**图文或视频问诊开启后如启用IPC则可勾选，系统自动配置AI助手收集病史

**平台服务（被动匹配）：**
- 快速问诊：成为快速问诊队列，免费、24小时、可开方，医生无法自主取消
- 商城问诊：用户购买处方药时匹配该医生，诊金默认免费

**隐藏诊费设置：**勾选后医生端不显示"诊费设置"入口，不可自行修改服务状态和价格

### 第六步：选择接诊模式
- **实时接诊模式**：医生设置在线即可接诊，离线时提醒用户是否等待
- **预约接诊模式**：医生需设置排班，用户问诊需预约挂号

### 第七步：查看审核状态
在编辑模式下可查看：
- **CA认证状态**：未认证/认证中/已认证/认证失败
- **审核驳回**：如 flowStatus=30（审核不通过），显示驳回原因

### 第八步：所属机构绑定
如系统启用了机构功能，可为医生绑定所属机构，支持多选远程搜索。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 医生姓名在待审核状态下修改 | 姓名输入框禁用，不可修改 |
| 头像未上传 | 提交时提示"请上传头像" |
| 执业医院未选择 | 提交时校验"请选择执业医院" |
| 身份证号格式错误 | 提交时提示格式不正确 |
| 会诊价格超出区间 | 输入框有 min/max 限制，超出无法输入 |
| 已审核医生修改关键信息 | 需要先驳回再修改 |
| 对症询医数量超限 | 最多选择10个 |

## 关联功能
- 医生列表管理（列表页进入）
- 医生审核流程（提交审核/通过/驳回）
- CA签章管理（认证状态联动）
- 排班管理（预约接诊模式需配置排班）
