# 药品运费规则配置流程

> 文档版本：v1.0 | 面向读者：运营

## 概述

管理员配置平台药品配送的运费规则。支持按地区设置不同的运费和包邮门槛，支持到付模式，以及医生开方时免运费的灵活控制。

## 触发条件

- **前置条件**：管理员已登录管理后台
- **入口**：左侧菜单 → 运营管理 → 运费设置

## 步骤详解

### 第一步：查看当前规则

页面加载时自动获取已保存的运费规则，展示为多条规则行。规则按优先级排序：前N条为特定地区规则，最后一条为"其他未选择地区"的默认规则。

### 第二步：新增地区规则

点击「新增条件」按钮添加一条新的地区规则行：

- **选择支持地区**：点击「选择地区」→ 弹出省市区三级级联选择器 → 勾选省市 → 确认
- **显示已选地区**：已选地区以标签形式展示（格式：省名+市名），每个标签可单独删除
- **设置运费**：输入金额（元）。设为0即包邮
- **设置包邮门槛**：勾选"订单总价金额满 X 元包邮"，订单金额达到门槛时免运费
- **设置到付**：勾选"到付"则无需设置运费和包邮门槛（运费由收件人支付）

### 第三步：配置默认规则（其他地区）

最后一行规则为"其他未选择地区"的默认配置：

- **统一运费**：对未匹配到特定规则的地区统一收取固定运费
- **统一到付**：所有未匹配地区全部到付
- **包邮门槛**：支持满额包邮（与特定地区规则一致）
- **免运费开关**：开启"可设为免除运费"开关后，医生在开处方时可以手动修改运费为0（包邮）。提示文字："开启后，医生开处方时可修改运费为包邮"

### 第四步：保存规则

- **保存按钮**：页面底部有「保存」按钮
- **离开提醒**：如果离开页面前有未保存的修改，弹出确认框"有更新内容，请先保存"，提供「保存」和「不保存」两个选项

### 第五步：删除规则

点击某条规则行的「删除」按钮，移除该条运费规则。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 航线前离开未保存 | 弹出提示"有更新内容，请先保存" |
| 到付模式开启 | 运费输入框和包邮开关自动禁用 |
| 包邮开关未开启 | 包邮门槛输入框禁用 |
| 删除规则 | 直接删除，无二次确认 |

## 关联功能

- **处方药品下单与配送流程**（小程序患者端）：运费规则影响患者下单时的运费计算
- **处方结算与费用计算流程**（服务端）：后端根据运费规则计算运费
- **药房管理流程**：运费规则按药房维度配置
