# 中药图谱管理流程

> 文档版本：2026-07-03 | 面向读者：运营人员、内容管理员

## 概述

中药图谱是医学百科下的一个内容管理模块，用于维护中药的基本信息、药味、适应症、功效作用及详细的用法说明。管理员可以新增、编辑、删除中药条目，每条中药包含封面图、名称、药味分类、适应症和四个维度的详情介绍（基本用法、药物作用、用药方法、注意事项）。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 医学百科 → 中药图谱
- **前置条件**：拥有对应菜单权限的管理员

## 步骤详解

### 第一步：浏览中药列表

进入页面后，系统自动加载中药图谱列表，每页显示10条记录。列表展示以下信息：
- **封面图**：中药的展示缩略图
- **中药名称**：中药的标准名称
- **药味**：该中药的药味分类（如酸、苦、甘、辛、咸），多个药味以顿号分隔
- **适应症**：该中药适用的病症，多个适应症以顿号分隔
- **最后修改时间**：该条目的最近编辑时间

管理员可以通过顶部的筛选条件过滤数据：
- **中药名称**：按名称搜索（最多输入20个字）
- **药味**：下拉多选，从系统字典中获取药味列表
- **适应症**：下拉多选，从系统字典中获取适应症列表

点击「查询」按钮后，列表会回到第一页并显示筛选结果。

### 第二步：新增中药

点击列表上方的「新增」按钮，跳转到新增页面。新增页面包含以下填写区域：

**基本信息区**：
- **中药名称**：必填，最多20个字
- **药味**：从下拉列表中选择（必填），支持多选和搜索
- **适应症**：从下拉列表中选择（可选），支持多选和搜索
- **展示封面图**：上传封面图片，系统要求上传200×200像素或1:1比例的图片，图片大小不超过2MB
- **功效作用**：必填，最多100字，简要描述该中药的主要功效

**中药详情区（四个标签页，均为必填）**：
- **基本用法**：通过富文本编辑器编辑，支持文字排版和图片插入
- **药物作用**：通过富文本编辑器编辑
- **用药方法**：通过富文本编辑器编辑
- **注意事项**：通过富文本编辑器编辑

填写完成后点击「确定」按钮提交。如果必填字段未填写或详情区有空白标签页，系统会给出相应提示并自动切换到未填写的标签页。

### 第三步：编辑中药

在列表中找到需要修改的中药，点击操作列的「编辑」按钮，跳转到编辑页面。编辑页面会自动回显该中药的所有已有信息，管理员修改后点击「确定」保存。

### 第四步：删除中药

- **单条删除**：点击某条记录操作列的「删除」按钮，系统弹出二次确认弹窗（"数据删除后不可恢复，您确认要删除吗?"），确认后执行删除
- **批量删除**：勾选列表前的复选框选中多条记录，然后点击「批量删除」按钮。如果没有勾选任何记录，批量删除按钮不会触发操作。确认后批量删除

删除成功后，列表自动刷新回到第一页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 中药名称为空 | 表单校验提示"请填写中药名称" |
| 未选择药味 | 表单校验提示"请选择药味" |
| 功效作用为空 | 表单校验提示"请填写功效作用" |
| 基本用法未填写 | 提示"请填写基本用法"，自动切换到基本用法标签页 |
| 药物作用未填写 | 提示"请填写药物作用"，自动切换到药物作用标签页 |
| 用药方法未填写 | 提示"请填写用药方法"，自动切换到用药方法标签页 |
| 注意事项未填写 | 提示"请填写注意事项"，自动切换到注意事项标签页 |
| 删除时未勾选任何记录 | 批量删除按钮无响应（静默忽略） |
| 删除操作取消 | 不执行删除，弹窗关闭 |

## 关联功能

- 该系统数据通过服务端 API 提供给小程序端的「中药图谱」展示功能
- 药味和适应症的下拉选项来源于系统字典管理
