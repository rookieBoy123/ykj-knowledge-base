# 可视化页面设计器拖拽搭建流程

> 文档版本 v1.0 | 面向运营人员

## 概述

可视化页面设计器（也叫"装修页面"）是运营人员在管理后台中通过拖拽方式搭建和编辑小程序首页展示内容的工具。运营人员可以从左侧组件库中拖拽组件到中间预览区，自由排列组合，配置各组件的内容和样式，实时预览效果，最后保存发布到小程序。

## 触发条件

- **前置条件**：运营人员已登录管理后台，拥有页面装修的操作权限
- **入口路径**：运营管理 → 页面装修 → 点击某个已创建的页面进入编辑，或新建页面

## 步骤详解

### 第一步：进入设计器

打开设计器页面后，界面分为三个区域：

- **左侧**：组件库面板，按类别分组展示所有可用的页面组件（如轮播图、搜索框、导航栏、商品列表、医生列表等）
- **中间**：手机模拟预览区，宽375像素，模拟小程序中的真实展示效果。上方显示当前页面标题，可点击编辑图标修改（不超过15个字）
- **右侧**：配置面板，点击中间某个组件后显示该组件的详细配置项（如标题文字、图片、链接、颜色等）

顶部操作栏提供三个按钮：返回、手机预览、保存修改。

### 第二步：从组件库添加组件

在左侧组件库中，各组分为可展开/收起的分类。每个组件以图标+名称的卡片形式展示。

添加组件有两种方式：
1. **点击添加**：直接点击组件卡片，组件会自动添加到预览区末尾
2. **拖拽添加**：按住组件卡片拖到中间预览区，松手后组件插入到对应位置

搜索框组件和底部导航组件（home_footer）每个页面只能添加一次，重复添加会提示"该组件只能添加一次"。

搜索框组件会自动插入到最顶部，底部导航组件自动排列在最底部，不受手动排序影响。

### 第三步：调整组件顺序和删除

选中预览区中的某个组件后，该组件周围出现蓝色虚线边框，同时右侧出现操作工具栏，包含四个按钮：
- **删除**：移除该组件
- **复制**：在当前组件下方复制一个完全相同的组件
- **上移**：将组件向上移动一个位置
- **下移**：将组件向下移动一个位置

搜索框和底部导航组件固定不可移动，尝试移动会提示"该组件禁止移动"。拖拽排序时也禁止拖拽这两个组件。

### 第四步：配置组件内容

点击预览区中的某个组件，右侧配置面板会显示该组件对应的配置表单。不同类型的组件有不同的配置项，常见包括：
- 标题文字和颜色
- 图片上传和链接设置
- 列表数据选择
- 显示/隐藏开关
- 样式设置（背景色、间距等）

配置修改后，中间预览区实时更新。

### 第五步：页面全局设置

点击预览区下方的"页面设置"按钮，或点击页面标题区域，右侧配置面板切换为页面全局配置：
- 页面标题文字
- 页面背景色
- 页面背景图片（可选择平铺或全屏）
- 页面是否显示

### 第六步：保存或另存模板

点击顶部"保存修改"按钮，将当前配置保存到服务器。系统支持三种保存模式：
1. **编辑已有页面**：直接更新当前页面配置
2. **新建页面**：首次保存时创建新页面，并弹出模板名称输入框（必填，不超过15字）
3. **另存模板**：点击"另存模板"按钮，输入新模板名称后保存为一个独立模板

支持快捷键 Ctrl+S 快速保存。

### 第七步：手机预览

点击"手机预览"按钮，弹出二维码弹窗。用手机扫描二维码即可在小程序中预览当前页面的真实效果。注意：只有已保存过的页面（pageId > 0）才能预览。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 模板名称为空 | 保存时提示"模板名称不能为空" |
| 模板名称超过15字 | 输入框限制最大15个字符 |
| 搜索框/底部导航重复添加 | 提示"该组件只能添加一次" |
| 搜索框/底部导航被拖拽移动 | 拖拽操作被阻止，提示"该组件禁止拖拽" |
| 新建页面时未保存就离开 | 弹出确认对话框"当前为离开页面操作，请检查是否已保存修改，离开后系统不会保存已修改内容" |
| 页面重置操作 | 点击"页面重置"弹出确认框，确定后清空所有组件并重新加载原始数据 |
| 页面未保存（pageId=0）时预览 | "手机预览"按钮置灰不可点击 |
| 视频医生列表组件 | 只有系统开启了"医生视频"功能才会在组件库中显示 |
| 健康服务包组件 | 只有系统开启了"服务包"功能才会在组件库中显示 |

## 关联功能

- **小程序首页展示**：设计器搭建的页面最终在小程序首页呈现
- **页面列表管理**：从页面列表进入设计器编辑已有页面
- **二维码预览**：通过二维码在小程序端预览装修效果

## 组件类型

基础组件包含但不限于：轮播图(swiperBg)、搜索框(search_box)、导航栏(nav_bar)、商品列表(goodList)、医生视频列表(doctor_video_list)、健康服务包(health_service_package)、底部导航(home_footer)等。
