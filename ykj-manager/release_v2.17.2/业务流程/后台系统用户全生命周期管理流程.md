# 后台系统用户全生命周期管理流程

> 文档版本：2026-07-03 | 面向读者：系统管理员

## 概述

后台用户管理是系统权限体系的基础模块，用于管理可登录"互联网医院管理后台"和"线下药房管理端"的运营人员账号。管理员可以创建用户、分配角色、启停账号、重置密码，以及查看每个用户的手机号（脱敏展示）和操作时间。

## 触发条件

- 需要新增运营人员（如新同事入职）
- 需要调整某人的角色和权限
- 有人忘记密码需要重置
- 人员离职需要禁用账号
- 需要查看某个账号关联的访问密钥（AccessKey）

入口：管理后台 → 系统管理 → 用户管理

## 步骤详解

### 第一步：用户列表查看与筛选

打开用户管理页面后，看到用户列表，每条记录显示：

- **ID**：系统内唯一编号
- **用户名**：登录账号名（3-20位字母或数字）
- **姓名**：真实姓名
- **手机号码**：默认以脱敏形式显示（如138****1234），点击可查看完整号码（需额外调用接口）
- **邮箱**：用于接收系统通知（如密码重置后的新密码）
- **角色**：显示"互联网医院管理员,药师"等角色名称组合
- **状态**：开关按钮，可直接在列表中启停用户
- **创建时间**：账号创建日期
- **操作**：编辑、修改用户名、重置密码、访问密钥

**筛选条件**：可按用户名、角色、状态（启用/禁用）三个维度筛选用户。

### 第二步：新增用户

点击"新增"按钮，弹出表单填写：

| 字段 | 说明 | 限制 |
|------|------|------|
| **用户名** | 登录账号 | 必填，3-20位纯英文或数字，不可与已有用户名重复 |
| **姓名** | 真实姓名 | 最多50字 |
| **邮箱** | 电子邮箱 | 最多100字，用于接收密码邮件 |
| **手机号** | 11位手机号 | 必填，格式校验（1开头11位），新增时可编辑 |
| **角色** | 级联选择器 | 可从角色列表中选择多个角色分配给用户 |
| **地址** | 联系地址 | 最多1000字 |
| **状态** | 启用/禁用 | 新增时可选 |

**隐藏限制**：角色选择器中，以下角色类型的角色码会被禁用（不可分配给后台用户）：
- 医生（doctor）、患者（patient）、商务（business）、药师（pharmacist）
- 线下药房管理员（offlineAdmin）、线下药房发货员（offlineSender）
- 线下药房药师（offlinePharmacist）、受邀机构管理员（invitedOrgManager）

这些角色的用户管理通过其他模块（如医生入驻审核）完成，不在本页面操作。

### 第三步：编辑用户

点击某条记录的"编辑"按钮，弹出与新增相同的表单，但：

- **用户名**：灰色不可修改
- **手机号**：灰色不可修改（编辑状态下只改内容和角色）
- **角色**：可增删角色（需要至少保留一个角色，否则提交失败提示"角色为必填项"）
- **其他字段**：正常可编辑

保存时系统会向用户邮箱发送通知邮件。如果邮箱发送失败，会弹出黄色提示"邮箱发送失败，请检查邮箱是否正确"，但编辑操作本身不会回滚。

### 第四步：修改用户名

点击"修改用户名"按钮，弹出输入框要求输入新用户名。校验规则与新增时相同（3-20位字母或数字），成功后自动刷新列表。

### 第五步：重置密码

点击"重置密码"，先弹出确认框。确认后系统自动重置该用户的密码，新密码通过邮箱发送给用户。如果邮箱发送失败（用户邮箱未配置或错误），会提示"邮箱发送失败，请检查邮箱是否正确"。

### 第六步：启用/禁用用户

在列表的状态列，直接拨动开关即可开启或关闭用户状态：

- **启用**：调用启用接口，用户可正常登录
- **禁用**：调用禁用接口，用户的当前会话将在下一次请求时被拦截

### 第七步：查看访问密钥

点击"访问密钥"按钮，跳转到访问密钥管理页面，自动筛选该用户的密钥列表。访问密钥用于程序化对接（如开放平台API调用）。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 用户名已存在 | 前端实时校验，输入框下方提示"用户已存在，请重新输入" |
| 用户名格式错误 | 提示"用户名必须是由字母或者数字组成"（编辑用户名时提示"用户名必须是由长度为3至20位的字母或者数字组成"） |
| 手机号格式错误 | 提示"手机号格式错误"（校验规则：1开头11位数字） |
| 编辑用户时未选择角色 | 提交时提示"角色为必填项" |
| 邮箱发送失败 | 操作本身成功，但弹出黄色警告"邮箱发送失败，请检查邮箱是否正确" |
| 手机号脱敏查看 | 手机号默认以星号遮盖中间4位，点击可获取完整号码（调用专门的手机号查询接口） |

## 关联功能

- **访问密钥管理**：用户管理页面可跳转查看用户关联的 AccessKey
- **角色管理**：用户角色来自角色管理模块，角色列表在此页面加载
- **菜单管理**：用户登录后看到的菜单由角色关联的菜单权限决定
- **权限管理**：用户能执行哪些操作由角色关联的权限码决定
- **系统操作日志**：所有用户CRUD操作均有审计日志记录
