# 机构间远程会诊全流程管理

> 文档版本：2026-07-03 第5轮 | 面向读者：平台管理员、远程医疗管理员

## 概述

机构间远程会诊是指一家医院向另一家合作机构发起会诊申请，由受邀机构的医生参与完成患者的远程诊疗。管理员可在后台对会诊从"申请→应答→收费→开始→结束→归档"进行全流程跟踪和操作。

## 触发条件

- **前置条件**：管理员角色必须为"远程医疗管理员"（roleCode = remoteMedicalManager），非该角色无法进入本页面
- **入口路径**：侧边栏 → 诊疗管理 → 远程医疗中心 → 机构间会诊

## 步骤详解

### 第一步：会诊列表查询

管理员进入页面后看到会诊列表，每条记录展示：会诊单号、申请科室、申请医师、患者信息（姓名/性别/年龄）、拟申请的会诊机构/服务/科室/医生、拟申请的会诊时间、当前状态、实际会诊时间。

**搜索筛选**：支持按会诊单号（最多20字）、申请科室（20字）、申请医生（5字）、患者姓名（5字）、拟申请机构（下拉选择）、状态（11种状态可选）进行组合筛选。其中"拟申请会诊机构"的下拉选项从已登记的合作机构列表中动态加载。

### 第二步：新增会诊申请

点击"新增"按钮弹出编辑窗口（宽度1000px），管理员可填写：
- 患者基本信息（姓名、性别、年龄、身份证号、手机号）
- 会诊信息（申请科室、病历摘要、会诊目的）
- 拟邀请信息（会诊机构、会诊服务、会诊科室、会诊医生、拟会诊时间）
- 患者资料上传（支持jpg/png/pdf/doc/excel等格式，单文件不超过10MB）

保存时可选择"暂存为草稿"或"提交申请"。"暂存为草稿"后记录不会发送给受邀方，可随时编辑后重新提交；"提交申请"后记录进入"待应答"状态。

### 第三步：各状态下的操作

根据会诊当前状态，每行记录显示不同的操作按钮：

| 当前状态 | 可用操作 | 说明 |
|---------|---------|------|
| 草稿（状态1） | 编辑、删除 | 可修改会诊信息或删除记录 |
| 待应答（状态2） | 撤回 | 将申请撤回，回退到草稿状态 |
| 已拒绝（状态10） | 编辑、取消 | 受邀方拒绝后可修改重新提交或取消 |
| 已撤回（状态9） | 编辑、删除 | 撤回后同草稿，可重新编辑或删除 |
| 待收费（状态3） | 收费、取消 | 确认患者已支付会诊费用 |
| 待开始（状态4） | 取消、开始会诊 | 费用已收，准备进入会诊 |
| 进行中（状态5） | 进入会诊、结束会诊 | 可进入会诊房间或手动结束 |
| 已结束（状态6） | 补充资料、归档 | 上传补充资料后归档锁定 |

### 第四步：收费确认

当会诊状态为"待收费"时，点击"收费"按钮弹出二次确认"确认已收费吗？"，确认后状态变更为"待开始"。

### 第五步：开始会诊

点击"开始会诊"后自动进入会诊中心页面（remoteConsultationCenter），该页面为患者信息+聊天消息+音视频通话的集成工作台。管理员可在该页面查看患者信息、发送消息、参与视频通话。

### 第六步：进入进行中的会诊

对于"进行中"状态的会诊，点击"进入会诊"跳转到会诊中心页面继续参与。

### 第七步：结束会诊

点击"结束会诊"弹出二次确认，确认后状态变更为"已结束"，此后不可再进入会诊房间。

### 第八步：补充资料

会诊结束后，管理员可为该会诊补充上传治疗方案及效果资料。点击"补充资料"弹出上传窗口，支持图片/pdf/word/excel等格式，单文件不超过10MB。已上传的文件列表会先回显展示（可查看已有资料），然后追加新的文件上传。

如果未选择任何文件就点击保存，会提示"请选择文件后再保存"。

### 第九步：归档

确认所有资料已上传完毕，点击"归档"弹出提示"归档后不再支持补充上传资料，确认相关会诊资料已全部上传吗？"，确认后状态变更为"已归档"，此后该会诊记录锁定，不可再补充资料。

### 第十步：取消会诊

对"已拒绝""待收费""待开始"三种状态可执行取消操作。点击"取消"弹出取消原因输入框（必填，最多500字），提交后状态变更为"已取消"。

### 第十一步：查看详情

所有状态的记录均可点击"查看详情"，弹出详情弹窗展示完整的会诊申请信息、患者信息、受邀方应答信息、签名信息等。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 非远程医疗管理员角色访问 | 页面显示"当前页面仅供远程医疗管理员使用！" |
| 删除操作确认 | 弹出"删除后数据不可恢复，确认删除吗？"二次确认 |
| 结束会诊确认 | 弹出"确认结束会诊吗？"二次确认 |
| 撤回申请确认 | 弹出"确认撤回申请吗？"二次确认 |
| 取消原因未填写 | 提示"请输入取消会诊原因"，不可提交 |
| 补充资料未选择文件 | 提示"请选择文件后再保存"，不可提交 |
| 归档确认 | 弹出"归档后不再支持补充上传资料"警告，需确认 |
| 搜索/筛选后无匹配数据 | 列表显示空状态 |

## 状态流转图

```
草稿(1) ──提交──> 待应答(2) ──受邀方通过──> 待收费(3) ──管理员收费──> 待开始(4) ──开始会诊──> 进行中(5) ──结束会诊──> 已结束(6) ──归档──> 已归档(7)
  │                 │                     │                │
  │                 ├──撤回──> 已撤回(9)    ├──取消──> 已取消(8) 
  │                 └──受邀方拒绝──> 已拒绝(10)
  └──删除
```

## 关联功能

- 机构间会诊中心（进入进行中的会诊时跳转）：患者信息展示、消息通信、视频通话
- 受邀机构管理：管理可接收会诊邀请的合作机构
- 会诊列表查询（consultationList）：查看本机构发起的远程会诊
