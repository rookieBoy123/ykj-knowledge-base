# 微页面装修管理流程

> 文档版本 v1.0 | 面向读者：产品、运营

## 概述

管理员通过可视化装修工具自定义小程序的页面布局和底部导航菜单。支持创建多个自定义页面，每个页面可拖拽配置轮播图、医生列表、视频列表、健康服务包、文章推荐、搜索框、导航菜单等多种模块。底部菜单支持自定义图标、标题和跳转链接。

## 触发条件

- **前置条件**：拥有页面装修管理权限
- **入口路径**：运营管理 → 页面装修 → 页面设计

## 步骤详解

### 第一步：选择管理模式

页面顶部有两个标签页切换：
- **自定义页面**：管理小程序的自定义微页面（列表+新增+设计）
- **自定义底部菜单**：管理小程序底部导航栏的菜单项（图标、标题、链接、排序）

### 第二步：自定义页面管理

#### 页面列表

进入"自定义页面"标签后看到已创建的页面列表，每页10条：

| 列名 | 说明 |
|------|------|
| ID | 页面唯一标识 |
| 页面名称 | 用户设置的页面名称 |
| 创建时间 | 页面创建时间 |
| 操作 | 预览、设计、设为首页、设为医带患首页、复制、删除 |

#### 搜索

顶部支持按**页面名称**搜索（最多输入15个字）。

#### 新增页面

点击"新增页面"按钮创建新页面，进入页面设计器。

#### 设计页面

点击某个页面的"设计"按钮进入可视化设计器（内置页面不可设计）。设计器左右双栏布局：

- **左侧**：手机模拟器预览区，实时展示页面效果
- **右侧**：模块配置面板

**可添加的页面模块**（按类型）：

| 模块 | 说明 |
|------|------|
| 搜索框 | 页面顶部搜索入口 |
| 轮播图 | 多图轮播，可配置跳转链接 |
| 导航菜单 | 宫格式功能入口 |
| 图片魔方 | 多图拼接展示 |
| 标题栏 | 分区标题 |
| 医生列表 | 横向/纵向展示医生卡片 |
| 医生视频列表 | 医生科普视频推荐 |
| 文章列表 | 健康资讯文章推荐 |
| 视频模块 | 视频内容展示 |
| 健康服务包 | 特色诊疗服务包推荐 |
| 辅助空白 | 占位留白，调整间距 |
| 辅助线条 | 分割线 |
| 底部信息 | 页脚展示 |
| 消息滚动 | 公告/消息滚动条 |

每个模块可拖拽排序。点击模块进入右侧配置面板，可设置：标题、展示数量、数据来源（手动选择/按分类自动拉取）、背景色、是否显示等。

#### 页面操作

- **预览**：弹出二维码弹窗，扫码可在手机上预览页面效果
- **设为首页**：将页面设为小程序首页（已设为首页的页面不可重复设置，按钮置灰）
- **设为医带患首页**：将页面设为医生端首页
- **复制**：复制当前页面创建副本
- **删除**：删除页面（首页和内置页面不可删除，按钮置灰）

### 第三步：自定义底部菜单

切换到"自定义底部菜单"标签，进入底部菜单配置页。

**左侧预览区**：手机模拟器展示底部菜单效果

**右侧配置区**：底部菜单项列表，每项可配置：
- **选中图标**：菜单被选中时显示的图标（点击上传图片）
- **未选中图标**：菜单未选中时显示的图标（点击上传图片）
- **标题**：菜单名称（最多4个字）
- **链接**：菜单跳转链接（点击从链接选择器中选择，支持微页面、文章、医生、科室、疾病等多种链接类型）
- **状态开关**：控制该菜单项是否显示

菜单项可**拖拽排序**，调整在底部栏中的位置顺序。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 删除首页 | 首页按钮置灰不可删除 |
| 删除内置页面 | 内置页面不可删除 |
| 设为首页但已是首页 | 按钮置灰不可重复设置 |
| 设为首页但已是医带患首页 | 不可同时设为普通首页和医带患首页 |
| 设计内置页面 | 设计按钮置灰（内置页面不可编辑） |

## 关联功能

- 小程序首页渲染（读取自定义页面配置）
- 医生管理（医生列表模块的数据来源）
- 视频管理（视频模块的数据来源）
- 文章管理（文章模块的数据来源）
- 健康服务包管理（服务包模块的数据来源）
