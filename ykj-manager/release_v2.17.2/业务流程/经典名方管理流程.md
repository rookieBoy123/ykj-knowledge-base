# 经典名方管理流程

> 文档版本：2026-07-03 | 面向读者：运营人员、内容管理员

## 概述

经典名方是医学百科下的一个内容管理模块，用于维护中医经典处方的信息，包括处方名称、处方类型、适应症、书籍出处、功效作用等。经典名方来源于中医经典著作，是传承中医药文化的重要内容。管理员可以新增、编辑、删除经典名方条目。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 医学百科 → 经典名方
- **前置条件**：拥有对应菜单权限的管理员

## 步骤详解

### 第一步：浏览经典名方列表

进入页面后，系统自动加载经典名方列表，每页显示10条记录。列表展示以下信息：
- **处方名称**：经典处方的标准名称
- **处方类型**：处方的分类类型（如汤剂、丸剂、散剂等，由系统字典维护）
- **适应症**：该处方适用的病症
- **书籍出处**：该处方的来源典籍
- **功效作用**：处方的主要功效描述
- **最后修改时间**：该条目的最近编辑时间

管理员可以通过筛选条件过滤数据：
- **处方名称**：按名称搜索（最多20个字）
- **处方类型**：下拉选择，从系统字典 `jtmf-lx` 获取
- **适应症**：下拉选择，从系统字典 `jtmf-syz` 获取
- **书籍出处**：下拉选择，从系统字典 `jtmf-sj` 获取

### 第二步：新增经典名方

点击列表上方的「新增」按钮，跳转到新增/编辑页面。新增页面需要填写以下信息：
- **处方名称**：必填，最多20字
- **处方类型**：从下拉列表选择（必填）
- **适应症**：从下拉列表多选（可选）
- **书籍出处**：从下拉列表选择（可选）
- **功效作用**：通过富文本编辑器编辑
- **处方详情**：通过富文本编辑器编辑，可插入图片
- **展示封面图**：上传封面图片，要求200×200像素或1:1比例，不超过2MB

### 第三步：编辑和删除

编辑操作与中药图谱类似，点击「编辑」进入编辑页面，信息回显后可修改保存。删除支持单条删除和批量删除，均有二次确认弹窗。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 处方名称为空 | 表单校验提示"请填写处方名称" |
| 处方类型未选择 | 表单校验提示"请选择处方类型" |
| 删除未勾选记录 | 批量删除静默无响应 |
| 删除操作取消 | 不执行删除，弹窗关闭 |

## 关联功能

- 通过服务端 API 提供给小程序端的「经典名方」展示功能
- 处方类型、适应症、书籍出处选项来自系统字典
