# 标准药品信息新增与编辑管理流程

> 文档版本 2026-07-03 | 面向管理员

## 概述

标准药品信息是平台药品库的基础数据，定义了每种药品的标准属性（通用名、商品名、规格、生产企业等）以及建议用法用量和药品说明书。管理员可以新增和编辑标准药品信息，这些信息将用于医生开方时的药品检索和患者查看药品详情。

## 触发条件

- **前置条件**：管理员已登录管理后台，拥有药品管理权限
- **入口路径**：管理后台 → 药品管理 → 标准药品信息列表 → 点击"新增"或"编辑"

## 步骤详解

### 第一步：填写基本信息

管理员进入表单页面（新增为空白表单，编辑时加载已有数据），依次填写：

**基本信息**：
- **药品药监ID**（必填）：12位数字和字母组合，可点击链接跳转到国家药品监督管理局网站查询
- **药品本位码**（选填）：14位纯数字
- **国家医保药品编码**（选填）
- **通用名**（必填）：药品的通用名称
- **商品名称**（必填）
- **生产企业名称**（必填）
- **批准文号**（选填）

### 第二步：填写规格信息

规格信息用于完整描述药品的计量属性：

- **剂量**（必填）：数字输入框，只允许输入数字和小数点
- **剂量单位**（必填）：下拉选择，如 mg、g、ml 等
- **制剂数量**（必填）：数字输入框，如每盒多少片/粒
- **制剂单位/用量单位**（必填）：下拉选择，如 片、粒、支
- **包装单位**（必填）：下拉选择，如 盒、瓶、袋
- **药品规格**（必填）：可以手动输入，也可以勾选"自动生成"由系统根据前面填写的剂量、制剂数量、单位等信息自动拼接生成

自动生成规则：`剂量+剂量单位*制剂数量+制剂单位/包装单位`（例如 "500mg*24片/盒"）

⚠️ 勾选"自动生成"需要先填写制剂数量、制剂单位和包装单位，否则复选框不可用。

### 第三步：填写建议用法用量

此部分为非必填，作为标准药品的推荐用法：
- **每次用量**：数字输入
- **使用次数**：下拉选择（如每日1次、每日2次、每日3次等）
- **使用方式**：下拉选择（如口服、外用、注射等）
- **建议使用天数**：数字输入，范围1-999天

### 第四步：填写药品说明书

此部分为非必填，用于完善药品信息展示：
- **品牌**：最多20字
- **功能主治**：最多200字
- **成分**：最多200字
- **性状**：最多200字
- **贮藏**：最多200字
- **不良反应**：最多200字
- **建议用法用量**：最多200字
- **禁忌**：最多200字
- **注意事项**：最多500字

### 第五步：上传药品图片

管理员可以上传药品图片：
- 最多上传10张
- 每张图片大小不超过10M
- 支持列表展示模式
- 图片上传后可以删除

### 第六步：保存提交

点击"保存"按钮后系统进行表单校验：

**必填项校验**：
- 药品药监ID：不能为空，且必须是12位数字字母组合
- 药品本位码：如填写则必须14位纯数字
- 通用名、商品名称、生产企业名称：不能为空
- 剂量、剂量单位、制剂数量、制剂单位、包装单位、药品规格：不能为空

如果校验不通过，页面会在对应字段下方显示红色错误提示。全部通过后，新增调用保存接口，编辑调用更新接口。保存成功后提示"保存成功"并返回列表页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药品药监ID格式不正确 | 提示"只能输入长度为12位的数字、字母" |
| 药品本位码格式不正确 | 提示"只能输入长度为14位的数字" |
| 任一必填项未填写 | 对应字段标红提示，提交时提示"请完善资料" |
| 取消编辑 | 弹出确认框"是否确认取消？"，确认后返回上一页 |
| 编辑时未找到药品数据 | 页面加载空白表单（具体表现取决于接口返回） |

## 关联功能

- **药品信息管理（后台）**：标准药品信息在药品管理列表中使用
- **药品详情查阅**：患者在小程序中查看的药品详情来源于此
- **处方开具**：医生开方时检索标准药品库
- **药品库管理与同步**：标准药品同步到各药房
