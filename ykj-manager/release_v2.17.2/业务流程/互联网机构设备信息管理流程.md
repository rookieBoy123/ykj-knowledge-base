# 互联网机构设备信息管理流程

> 文档版本 2026-07-03 | 面向读者：运营管理员

## 概述
管理员可以在系统中录入和管理互联网医院机构的硬件设备信息，包括设备名称、型号、类型、产地、购买日期、价值金额、使用情况等。支持按设备名称和型号搜索，支持新增、编辑和删除操作。

## 触发条件
- 管理员登录管理后台，进入"机构管理 → 设备信息"页面

## 步骤详解

### 第一步：查看设备列表
进入页面后，系统自动加载设备列表，表格展示以下列：
- **设备编号**：系统自动生成的唯一编号（格式：SBBH+年月日时分秒+3位随机数）
- **设备名称**
- **设备型号**
- **产地**：从字典数据映射为中文显示
- **设备类型**：从字典数据映射为中文显示
- **使用情况**：从字典数据映射为中文显示
- **创建时间**
- **修改时间**
- **操作**：编辑、删除按钮

### 第二步：搜索设备
页面顶部提供搜索条件：
- **设备名称**：输入框，精确/模糊匹配（最多20字）
- **设备型号**：输入框
- **产地**：下拉选择（字典：sbcd）
- **使用情况**：下拉选择（字典：sbsyqk）

点击搜索后，系统调用分页接口查询匹配结果。

### 第三步：新增设备
点击"新增设备"按钮（需要权限：hosp:devices:write），弹出新增表单：

**必填字段**（带红色星号）：
- 设备名称（最多20字）
- 设备类型（下拉选择：字典sblx）
- 产地（下拉选择：字典sbcd）
- 使用情况（下拉选择：字典sbsyqk）
- 购买日期（日期选择器）

**选填字段**：
- 生产厂家（最多120字）
- 设备型号（最多200字）
- 设备价值金额
- 理论设计寿命（年）
- 购进时新旧况（下拉：字典sbxjqk）
- 同批购进相同型号设备台数
- 设备参数（最多500字）
- 用途（最多500字）
- 备注（最多100字）

点击"确定"提交，新增成功后列表自动刷新。新增时系统会自动生成设备编号并关联当前机构信息。

### 第四步：编辑设备
点击表格中的"编辑"按钮，弹出编辑表单（与新增表单相同），预填已有数据，修改后保存。

### 第五步：删除设备
点击"删除"按钮，系统弹出二次确认："此操作将永久删除该项，是否继续？"
- 确认：执行删除，成功后提示"操作成功"
- 取消：提示"已取消删除"

## 字典依赖

该功能依赖以下字典分类，需提前在字典管理中配置：

| 字典编码 | 字典名称 | 用途 |
|---------|---------|------|
| sbcd | 设备产地 | 产地下拉选项 |
| sblx | 设备类型 | 设备类型下拉和列表翻译 |
| sbsyqk | 设备使用情况 | 使用情况下拉和列表翻译 |
| sbxjqk | 购进时新旧况 | 采购状况下拉 |

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 机构信息不存在 | 新增时报错"机构信息不存在" |
| 必填字段未填写 | 表单校验不通过，红色提示文字 |
| 无写入权限 | 新增/编辑/删除按钮不可见 |
| 删除确认取消 | 提示"已取消删除"，不执行操作 |
| 设备名称超过20字 | 输入框自动截断限制 |
| 生产厂家超过120字 | 输入框自动截断限制 |

## 权限控制
- 查看设备列表：hosp:devices 权限前缀
- 新增/编辑设备：hosp:devices:write 权限
- 删除设备：继承hosp:devices权限

## 关联功能
- 字典管理——设备类型、产地、使用情况、新旧况均依赖字典配置
- 互联网机构信息管理——新增设备时关联当前机构代码
