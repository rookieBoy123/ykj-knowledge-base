# 门诊问诊订单管理流程

> 文档版本：v2.17.2 | 面向读者：运营 / 客服

## 概述

管理员在后台查看所有线上问诊订单，支持多维度筛选、查看问诊对话/视频记录、手动结束问诊、查看知情同意书确认信息。

## 触发条件

- 管理员登录后台后，进入「门诊服务订单」菜单
- 需要排查患者投诉、核对问诊记录、监控问诊进度

## 步骤详解

### 第一步：筛选问诊订单

管理员可以通过以下条件组合筛选问诊订单：

| 筛选条件 | 说明 |
|---------|------|
| 问诊编号 | 精确搜索，最多输入30个字符 |
| 服务类型 | 多选：图文问诊、视频问诊、咨询服务、快速问诊、快速续方、医生复诊、商城问诊。如果系统开启了AI医生，还会多出「AI图文问诊」选项 |
| 问诊流程 | 多选：在线问诊、线上复诊、慢病医保、预约挂号、快速开方。如果系统开启了AI医生，还会多出「AI问诊」选项 |
| 问诊状态 | 单选：全部/已预约/问诊中/问诊结束/取消问诊 |
| 支付时间 | 日期范围选择 |
| 问诊时间 | 日期范围选择 |
| 就诊时间 | 日期范围选择 |
| 有无病历 | 有/无 |
| 医生姓名 | 最多输入5个字 |
| 患者姓名 | 最多输入5个字 |
| 所属团队 | 仅在系统开启了专家团队功能后显示 |
| 所属机构 | 仅在系统开启了多机构功能后显示 |

点击「查询」按钮，每页显示10条记录。

### 第二步：浏览问诊订单列表

列表以表格形式展示每条问诊的核心信息：

| 列名 | 说明 |
|------|------|
| 问诊编号 | 唯一标识，鼠标悬停显示完整编号 |
| 服务类型 | 图文问诊/视频问诊等 |
| 问诊流程 | 在线问诊/线上复诊/慢病医保等 |
| 医生姓名 | 接诊医生的姓名 |
| 支付时间 | 患者完成支付的精确时间 |
| 诊费 | 显示两位小数的金额 |
| 患者姓名 | 默认脱敏显示（如"张**"），点击可切换显示完整姓名 |
| 患者联系方式 | 默认脱敏，点击后请求后端解密并显示完整手机号（同时其他行的手机号恢复脱敏） |
| 问诊时间 | 问诊创建时间 |
| 就诊时间 | 预约的就诊时间 |
| 状态 | 已预约/问诊中/问诊结束/取消问诊 |
| 有无病历 | 有/无 |
| 操作 | 查看详情、结束问诊、视频记录、知情同意书确认信息 |

### 第三步：查看问诊详情（对话记录）

点击某条订单的「查看详情」，系统弹出对话记录窗口：

- **普通问诊**：加载IM（即时通讯）群组的历史消息，按时间倒序展示。窗口内包含文字、图片、语音、视频等多种消息类型。滚动到顶部会自动加载更早的消息（每次20条）。窗口带有水印（当前登录用户名@医院名称），防止信息泄露。
- **AI图文问诊**：打开专门的AI对话记录窗口，分为AI发言（左侧白色气泡+AI标志）和用户发言（右侧蓝色气泡），支持文字消息和业务卡片消息。点击底部按钮可查看AI预问诊报告。

### 第四步：查看视频记录

点击「视频记录」按钮（仅视频问诊类型可见），弹出窗口列出该问诊的所有视频通话记录。每条记录显示创建时间，点击「播放」按钮在内嵌播放器中观看回放。

### 第五步：查看知情同意书确认信息

点击「知情同意书确认信息」按钮（仅当问诊已签署知情同意书时可见），弹出窗口展示：
- 确认人姓名
- 网络IP地址
- 设备信息
- 微信用户openId
- 同意书签署时间

### 第六步：手动结束问诊

对于状态为「问诊中」（flowStatus=20）的订单，管理员可以点击「结束问诊」。系统弹出二次确认弹窗"确认是否结束所选问诊?"，确认后调用结束接口，刷新列表。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 问诊编号为空时点击结束问诊 | 提示「问诊编号为空！」，操作取消 |
| 搜索条件中选择了「全部」流程（serviceClassifies=0） | 后端忽略该筛选条件，返回所有流程的记录 |
| 问诊状态选择「已预约」或「问诊中」 | 前端将复合值（如"20,0"）解析为flowStatus和isCheckIn两个独立参数传给后端 |
| 问诊对话记录为空 | 弹窗显示空列表 |
| 视频记录列表为空 | 显示空状态占位图 |

## 关联功能

- 跳转至「问诊详情」页面查看问诊完整信息
- 跳转至「患者详情」查看患者个人资料
- 与「问诊室」功能关联（管理员可在问诊室模拟医生操作）
- 与「处方管理」关联（查看该问诊关联的处方）
