# 疾病知识管理流程

> 文档版本：2026-07-03 | 面向读者：运营人员、内容管理员

## 概述

疾病知识是医学百科下的一个内容管理模块，用于维护疾病的科普知识信息。每条疾病记录包含疾病名称、疾病分类和疾病详情（富文本内容）。管理员可以新增、编辑、删除疾病知识条目。新增和编辑操作通过弹窗完成，无需跳转页面。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 医学百科 → 疾病知识
- **前置条件**：拥有对应菜单权限的管理员

## 步骤详解

### 第一步：浏览疾病知识列表

进入页面后，系统自动加载疾病知识列表，每页显示10条记录。列表展示以下信息：
- **疾病名称**：疾病的标准名称
- **疾病分类**：该疾病所属的分类（如内科、外科等，由系统字典 `jbzs-jbfl` 维护）
- **最后修改时间**：该条目的最近编辑时间

管理员可以通过筛选条件过滤数据：
- **疾病名称**：按名称搜索（最多20字）
- **疾病分类**：下拉选择，选项来源于字典 `jbzs-jbfl`

### 第二步：新增疾病知识

点击列表上方的「新增」按钮，弹出新增弹窗（宽度800px）。弹窗包含：
- **疾病名称**：必填，最多20字
- **疾病分类**：必填，下拉选择
- **疾病详情**：必填，通过富文本编辑器编辑，可插入图片和排版

填写完成后点击「确定」提交。新增成功后，弹窗关闭，列表自动回到第一页刷新。

### 第三步：编辑和删除

点击「编辑」按钮，弹出编辑弹窗并自动回显已有数据。编辑保存后列表保持当前页刷新。删除操作包含二次确认弹窗。支持单条删除和批量删除。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 疾病名称为空 | 表单校验提示"请填写疾病名称" |
| 疾病分类未选择 | 表单校验提示"请选择疾病分类" |
| 疾病详情为空 | 表单校验提示"请填写疾病详情" |
| 批量删除未勾选 | 静默无响应 |

## 关联功能

- 通过服务端 API 提供给小程序端的疾病知识展示
- 疾病分类来源于系统字典管理
