# 问诊室管理流程

> 文档版本：2026-07-02 | 面向读者：运营人员、客服人员

## 概述

问诊室是管理后台中医生进行在线问诊操作的核心工作台，采用三栏布局：左侧问诊单列表、中间聊天对话区、右侧问诊资料区（问诊详情/病历记录/处方记录）。只允许医生角色登录使用，非医生角色会看到"请登录医生角色查看!"的提示。

## 触发条件

- 前置条件：必须以"医生"角色登录管理后台
- 入口路径：后台 → 诊疗管理 → 问诊室

## 步骤详解

### 第一步：查看问诊单列表（左侧栏）

页面左侧显示问诊单列表，按状态分组显示（待接诊、进行中、已完成等标签页）。医生可以：
- 切换不同状态的标签页，查看对应问诊单
- 点击某条问诊单，中间和右侧区域会加载对应的问诊详情

### 第二步：接诊（中间栏）

选中问诊单后，在中间聊天区可以：
- 查看患者与医生的IM对话记录
- 点击"接诊"按钮，系统提示"接诊成功"，问诊单自动从"待接诊"移至"进行中"
- 接诊后可以开始与患者进行图文交流

### 第三步：发起视频通话

在聊天区可以发起视频问诊：
- 点击视频按钮，系统生成房间号并发送卡片邀请给患者
- 卡片包含医生姓名、职称和"加入"按钮
- 患者点击加入后，双方进入视频通话房间
- 视频结束后返回聊天界面

### 第四步：写病历（右侧栏）

接诊后可以切换右侧栏为"写病历"模式：
- 点击聊天区"写病历"按钮，右侧切换为病历编辑面板
- 填写病历信息后点击"病历保存"，系统提示"病历保存成功"
- 也可以点击"取消"退出病历编辑模式，回到问诊资料面板
- 已保存的病历可以后续更新（"病历更新成功"）

### 第五步：开医嘱/处方

病历保存后可以开具处方：
- 点击聊天区"开医嘱"按钮
- 如果还未写病历，系统弹出提示"请先写病历再开医嘱!"，需先完成第四步
- 病例已填写后，弹出开方弹窗，可以开具中西药处方
- 处方提交后完成本次诊疗

### 第六步：结束问诊

- 点击"结束问诊"，系统二次确认"确定结束问诊吗？"
- 确认后问诊结束，状态变为已完成，操作不可逆
- 左侧问诊单列表自动切换到"已完成"标签页

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 非医生角色登录 | 显示"请登录医生角色查看!"空白页，无法进入问诊室 |
| 未选择任何问诊单 | 中间栏显示"选择患者查看详细信息"提示 |
| 开医嘱但未写病历 | 弹出"请先写病历再开医嘱!"确认弹窗 |
| 结束问诊操作取消 | 提示"操作取消"，不做任何状态变更 |
| 视频通话获取房间号失败 | 提示"响应错误，请联系管理人员" |

## 关联功能

- 门诊服务订单管理：问诊订单的源头管理
- 处方管理：开具的处方可在处方管理模块查看
- 病历查询：填写的病历可在病历查询中查看
