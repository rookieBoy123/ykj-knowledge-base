# 患者线上就诊病历综合查询流程

> 文档版本 2026-07-03 | 面向读者：运营管理员

## 概述
管理员可通过患者身份证号查询其在平台上的所有线上就诊记录，以时间线形式展示每次复诊的病历摘要（门诊号、接诊医生、主诉、诊断），点击可查看完整病历详情以及关联的处方信息。

## 触发条件
- 管理员登录管理后台，通过带身份证号和授权令牌的链接访问病历查询页面
- 通常在患者管理模块中，点击某个患者的"查看病历"入口跳转而来

## 步骤详解

### 第一步：身份验证
- 页面从URL参数中读取身份证号（idCard）和授权令牌（token）
- 如果缺少任一参数，页面不加载数据
- 系统通过身份证号查询该患者在所有平台账号下的患者ID

### 第二步：加载病历时间线
- 系统获取患者所有关联的就诊记录
- 左侧以时间线列表形式展示，每条记录包含：
  - **门诊号** + "复诊"标签
  - **接诊医生**：姓名 + 科室 + 职称
  - **主诉**：患者的病情描述
  - **诊断**：西医诊断、中医辨病、中医辨证（区分展示）
  - **电子病历按钮**：如果有PDF文件，显示蓝色按钮可查看

### 第三步：查看病历详情
点击左侧时间线中的任意一条记录，右侧展示该次就诊的完整病历详情：

**病历信息区域**：
- 就诊人：姓名、性别、年龄（岁/月）
- 门诊号
- 初诊标志：固定显示"复诊"
- 监护人信息（如有）：姓名、身份证号
- 就诊类型：图文问诊/视频问诊/咨询服务等
- 就诊机构：系统配置的机构名称
- 就诊时间
- 就诊科室
- 接诊医生

**病史信息**：
- 主诉、现病史、既往史、过敏史
- 个人史、家族史、月经史、婚育史
- 中医四诊（脉象、舌象）——仅中医病历显示
- 常见体征
- 治法——仅中医病历显示
- 治疗意见——仅西医病历显示
- 辅助检查

**诊断信息**：
- 西医诊断
- 中医辨病（中医模式）
- 中医辨证（中医模式）

**病历资料**：
- 患者上传的图片资料，点击可放大预览
- 无资料显示"记录遗失/不在身边"

### 第四步：查看处方信息
病历下方展示该次就诊关联的处方：

**成药处方（type=1）**：
- 表格展示：序号、药品名称、规格、剂型、每次用量、用药途径、用药频次、数量
- 处方医生、审核药师
- 电子处方PDF链接

**中药处方（type=2）**：
- 药材名称和用量列表
- 剂数、用药方法（内服/外用）、服用频率
- 用药说明（煎煮方法）
- 处方医生、审核药师
- 电子处方PDF链接

### 第五步：查看电子病历/处方PDF
点击"电子病历"或"电子处方"按钮，通过内置PDF阅读器打开对应的文件。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| URL缺少身份证号或令牌 | 页面不加载，不发起任何请求 |
| 根据身份证号查不到患者 | 提示"没有该患者相关信息" |
| 患者无线上就诊记录 | 显示"暂无线上就诊记录!" |
| 病历图片加载失败 | 不显示该图片区域 |

## 关联功能
- 患者管理——通过患者列表进入此页面
- 电子病历PDF阅读——系统内置PDF查看器
- 门诊问诊订单管理——相关订单可跳转查看
