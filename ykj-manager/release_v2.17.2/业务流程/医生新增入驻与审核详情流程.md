# 医生新增入驻与审核详情流程

> 文档版本 v1.0 | 面向运营/审核人员

## 概述

当新医生申请入驻平台时，运营人员在管理后台为其创建档案并填写完整的从业信息。对于"待审核"状态的医生，需要逐项审核基本资料、从业资质、资格证书和执业证书，确认无误后审核通过；如发现信息不完整或不符要求，可驳回并填写原因。审核通过后的医生档案如需修改，走编辑资料流程。

## 触发条件

- **前置条件**：运营人员有医生管理的操作权限
- **入口路径**：医院管理 → 医生管理 → 医生审核列表 → 点击某条待审核记录进入详情，或点击"新增医生"按钮

## 步骤详解

### 第一步：填写基础信息

页面分为两大场景：新增医生和审核/编辑已有医生。基础信息包含以下字段：

- **头像图片**：上传200×200像素或1:1比例的图片作为头像（必填）
- **医生姓名**：最多5个汉字。在审核编辑模式下，只有待审核状态（flowStatus=10）的医生才可以修改姓名
- **性别**：下拉选择（男/女）
- **手机号码**：11位数字
- **出生日期**：日期选择器
- **所在科室**：从医院科室树中选择，支持搜索过滤
- **医生职务**：从字典数据中选择（如主任医师、副主任医师等）
- **民族**：山东地区（isShanDongArea）专属字段，下拉选择
- **擅长疾病**：文本域，最多500字
- **医生简介**：文本域，最多500字
- **对症询医**：当系统开启"医生疾病匹配"功能时显示。远程搜索疾病分类，最多选10个

### 第二步：填写从业资料

- **主要执业医院**：远程搜索医院名称，选中后自动回填
- **职务类别**：从字典数据中选择
- **全科医生**：单选（是/否）
- **开始执业日期**：日期选择器
- **上海地区专有字段**：参加工作日期、执业机构

### 第三步：上传资格证书和执业证书

从业资质证明需要上传三类证书图片：

1. **从业资格证**：上传证书正面图片 + 填写资格证编号、资格获取时间
2. **执业证书**：上传证书正面和背面各一张 + 填写执业证书编号、发证日期、执业范围（从树形字典中选择）
3. **其他相关证明**：最多上传5张图片，可删除

### 第四步：查看CA认证状态

如果系统开启了CA认证功能（caflagPage），页面显示"CA认证状态"区域。显示当前医生的电子签章认证状态，如"未认证""已认证"等。

### 第五步：查看审核结果（被驳回时）

当医生状态为"已驳回"（flowStatus=30）时，显示审核结果区域，展示驳回原因。

如果医生已通过审核且有线上问诊记录，表单大部分字段变为只读（isForbidden=true），防止随意修改已上线医生的信息。

### 第六步：设置所属机构

如果系统开启了机构管理功能（organizationEnabled），可以选择该医生归属的运营机构（支持远程搜索、多选）。

### 第七步：设置工作类型和服务项目

工作类型选择：
- **医生**：勾选后将展示问诊服务配置
- **医助**：辅助医生接诊患者做诊前诊后服务

当选择"医生"工作类型后，额外的配置项：
- **测试账号**：勾选后该医生仅用于测试，不对信息进行备案，诊疗数据不上传
- **不展示在平台**（visible=0）：勾选后医生不在全平台展示，患者只能通过扫邀请码找到该医生

### 第八步：开启服务项目

已入驻医生（编辑模式）可以配置8种服务项目：

| 服务类型 | 可配置项 | 特殊说明 |
|---------|---------|---------|
| 图文问诊 | 开关、价格（元）、时长（小时/次） | 基础服务 |
| 视频问诊 | 开关、价格（元）、时长 | 时长单位由系统配置 |
| 咨询服务 | 开关、价格（元）、时长 | — |
| 快速续方 | 开关、续方诊金（元） | 开启快速续方必须先开启图文问诊，否则保存时提示"开启快速续方的同时请开启图文问诊" |
| 会诊服务 | 开关、会诊诊金（元） | 诊金有价格区间限制（feeAreaMin~feeAreaMax），由医生职务决定 |
| 医生复诊 | 开关、复诊诊金（元） | 复诊诊金与处方订单一起收取 |
| 快速问诊 | 仅勾选 | 平台服务，免费，24小时服务时间，默认可开处方（系统可配置禁止） |
| 商城问诊 | 仅勾选 | 平台服务，需系统开启商城功能后才显示 |

附加辅助功能：
- **医助辅助接诊**：图文/视频问诊开启后可选，自动匹配医助做诊前诊后服务
- **AI预问诊**：需系统开启IPC功能，启用后AI助手辅助预问诊收集病史

隐藏诊费设置：勾选后，医生在小程序端无法自行修改服务开启状态和诊费。

### 第九步：设置接诊模式（预约功能开启时）

当系统开启了预约挂号功能时显示：
- **实时接诊模式**：医生在线即接诊，离线时提醒患者等待
- **预约接诊模式**：医生需设置排班，患者预约挂号后就诊

选择预约接诊模式后，可额外设置"过号是否接诊"：
- 是：超过预约时间患者仍可签到，医生可继续接诊
- 否：超过预约时间患者不可签到，问诊自动取消

### 第十步：提交审核

页面底部操作按钮根据状态不同而变化：

| 状态 | 按钮 |
|------|------|
| 新增 | 确定（提交创建） |
| 待审核（flowStatus=10） | 驳回 + 确定（审核通过） |
| 已审核（flowStatus=20） | 确定（修改资料） |
| 只读模式 | 返回 |

- **审核通过**：修改 flowStatus 从10改为20，先保存医生服务项目配置，再调用审核通过接口
- **驳回**：弹出驳回弹窗，必须填写驳回原因（不能为空或纯空格，最多255字），然后提交驳回（flowStatus=30）

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 表单必填项未填 | 提交时提示"请完善资料" |
| 快速续方开启但图文问诊未开启 | 保存时弹出警告："开启快速续方的同时请开启图文问诊" |
| 驳回原因为空或全空格 | 提示"驳回原因不能为空且不能为空格" |
| 医生已通过审核且有线上记录 | 基础资料大部分字段置灰不可编辑 |
| 头像上传 | 限制200×200像素或1:1比例 |
| 手机号格式错误 | 前端限制11位数字 |
| 医生姓名超过5字 | 输入框限制最大5字 |
| 擅长疾病/简介超过500字 | 输入框限制最大500字 |
| 对症询医选择超过10个 | 限制最多10个 |
| 其他证明图片超过5张 | 上传组件限制最多5张 |
| 会诊诊金超出职务范围 | 输入框限制feeAreaMin~feeAreaMax |

## 关联功能

- **医生列表管理**：审核通过后进入已入驻医生列表
- **医生服务项目管理**：可配置的各服务类型
- **CA认证管理**：电子签章认证状态查看
- **排班管理**：预约接诊模式下需要设置排班
