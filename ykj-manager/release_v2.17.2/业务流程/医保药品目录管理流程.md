# 医保药品目录管理流程

> 文档版本：2026-07-02 | 面向读者：医保管理人员

## 概述

管理医保基础药品目录数据，涵盖西药（成药）和中药饮片两大类。支持查询、下载模板、批量导入、导出和删除操作。目录数据包含药品的收费项目等级（甲/乙/丙类）和不同参保类型（城职/城乡）的先支付比例，是医保结算的核心基础数据。

## 触发条件

- 前置条件：拥有医保管理权限，导入导出需要特定权限
- 入口路径：后台 → 系统管理 → 医保管理 → 医保药品目录

## 步骤详解

### 第一步：切换药品类型

页面顶部有两个标签页：
- **成药类型**：西药（成药）目录，显示注册名称、商品名称、剂型、规格、包装、批准文号等
- **中药饮片**：中药目录，显示饮片名称、药材名称、炮制方法、功效分类、性味归经、功能主治等

切换标签时，查询条件会自动更新为对应类型的搜索项。

### 第二步：筛选查询

成药品类的筛选条件：
- **收费等级**：从字典中选择（甲类/乙类/丙类），动态加载自系统字典"收费项目等级"
- **注册名称**：输入药品注册名称搜索
- **通用名称**：输入药品通用名搜索
- **药品代码**：输入药品代码搜索
- **药品本位码**：输入本位码搜索

中药饮片的筛选条件：
- **收费等级**：同上
- **中药名称**：输入中药饮片名称
- **药材名称**：输入药材名称
- **中药代码**：输入中药代码

### 第三步：导入药品数据

运营步骤：
1. 点击"**下载模板**"按钮，根据当前选中的药品类型（成药/中药）下载对应的 Excel 导入模板
2. 按模板格式填充药品数据
3. 点击"**导入**"按钮，选择填好的 Excel 文件上传

导入过程中：
- 页面显示"正在导入数据"的加载动画（带加载图标）
- 系统每隔 5 秒轮询一次导入任务状态
- 导入完成后自动刷新列表
- 如果导入失败，弹出错误提示显示失败原因

### 第四步：导出药品数据

点击"**导出**"按钮（需要导出权限），系统显示"加载中"提示，生成并下载当前筛选条件下的药品数据 Excel 文件。

### 第五步：删除药品

支持单条删除，点击删除按钮，二次确认"此操作将永久删除该项, 是否继续?"，确认后删除并刷新列表。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 导入数据中 | 页面显示加载动画，每隔5秒轮询任务状态 |
| 导入失败 | 弹出错误提示，显示具体的失败原因信息 |
| 导入权限不足 | 导入和下载模板按钮不显示（需要权限 `med:medicare-medicine:import`） |
| 导出权限不足 | 导出按钮不显示（需要权限 `med:medicare-medicine:export`） |
| 删除确认 | 二次确认："此操作将永久删除该项, 是否继续?" |
| 数据为空翻页 | 查询结果为空时自动回退到上一页 |

## 关联功能

- 收费项目管理：收费等级字典数据来自系统字典
- 医保结算：患者支付时根据药品的收费等级和先支付比例计算自付金额
- 处方开具：医生开具处方时调用的药品列表部分来源于此目录
