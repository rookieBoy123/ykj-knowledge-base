# 处方列表查询与管理流程

> 文档版本：2026-07-01 | 面向读者：运营管理员、药事管理员

## 概述

管理员通过后台「处方列表」页面查看平台上所有医生开具的处方，支持多维度筛选搜索和查看电子处方详情。

## 触发条件

- **前置条件**：登录管理后台，有处方管理权限
- **入口路径**：诊疗管理 → 处方管理 → 处方列表

## 步骤详解

### 第一步：进入处方列表页

进入页面后，系统自动加载所有处方记录，每页显示10条。列表展示以下信息：

| 列名 | 说明 |
|------|------|
| 处方编号 | 唯一标识，可通过此编号搜索 |
| 处方类型 | 成药处方 或 中药处方 |
| 处方医生 | 开具处方的医生姓名 |
| 所属团队 | 医生所属的专家团队（仅团队功能开启时显示） |
| 科室 | 医生所属科室 |
| 开具时间 | 处方创建时间 |
| 诊断 | 医生的诊断结果 |
| 患者姓名 | 脱敏显示（只显示首字+星号），点击可查看全名 |
| 状态 | 处方当前状态（见下方状态说明） |

### 第二步：筛选查找处方

管理员可以使用以下条件精准查找处方：

- **医生姓名**：输入医生姓名搜索，最多5个字
- **科室**：通过树形下拉选择科室，支持模糊搜索
- **处方类型**：可选择「成药处方」或「中药处方」
- **处方编号**：输入处方编号精确查找
- **开具时间**：选择日期范围（开始日期～结束日期）
- **状态**：选择处方当前所处阶段
- **所属团队**：按专家团队筛选（仅团队功能开启时显示）

### 第三步：查看电子处方

每条处方记录都有「查看电子处方」操作按钮，但仅在处方已生成PDF文件时才显示。

点击后：
- 如果是PDF文件：在新窗口打开PDF查看器查看电子处方
- 如果是图片格式：弹出大图预览，可放大查看

### 处方状态说明

| 状态 | 含义 |
|------|------|
| 开方中 | 医生正在编辑处方，尚未提交 |
| 待审核 | 处方已提交，等待药师审核 |
| 审核中 | 药师正在进行审核 |
| 审核通过 | 药师审核通过，处方已生效 |
| 审核不通过 | 药师审核拒绝，需医生修改后重新提交 |
| 已失效 | 处方因各种原因（过期、撤回等）已失效 |

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 患者姓名脱敏 | 列表默认显示「张***」格式，点击可切换显示全名。出于隐私保护考虑，全名只在当前会话可见 |
| 团队功能未开启 | 如果系统配置中关闭了专家团队功能，列表中将不显示「所属团队」列和「所属团队」筛选条件 |
| 无电子处方文件 | 如果处方尚未生成PDF（treatmentOrderUrl为空），「查看电子处方」按钮不显示 |
| 搜索结果为空 | 列表显示空状态，可调整筛选条件后重新搜索 |

## 关联功能

- **药师审方**：从处方列表可了解有哪些处方等待审核
- **医生管理**：通过处方的「处方医生」可定位到具体医生的开方记录
- **电子处方验签**：在「处方验签」页面，可上传PDF文件验证处方的CA签章真实性（仅支持本平台的电子处方验签，文件限PDF格式且不超过2MB）
