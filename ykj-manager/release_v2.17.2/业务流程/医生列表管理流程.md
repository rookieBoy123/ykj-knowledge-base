# 医生列表管理流程

> 文档版本：2026-07-02 | 面向读者：产品运营、管理员

## 概述

后台运营人员通过医生列表管理所有注册医生的信息，包括查询筛选、启用/禁用账号、查看医生详情、管理医生邀请码、变更手机号、管理会诊管理员权限等。

## 触发条件

- **前置条件**：具有医生管理权限的后台账号
- **入口路径**：左侧菜单 → 医院管理 → 医生管理 → 医生列表

## 步骤详解

### 第一步：筛选查询

支持多维度筛选医生：

| 筛选维度 | 说明 |
|---------|------|
| 医生姓名 | 文本搜索，最多5字 |
| 工作类型 | 医生/医助 |
| 科室 | 树形选择，支持搜索 |
| 所属医院 | 远程搜索下拉 |
| 职称 | 字典下拉 |
| 账号状态 | 启用/禁用，默认全部 |
| 所属机构 | 机构下拉（机构模式时显示） |
| 开启服务 | 多选：图文问诊/视频问诊/咨询服务/快速问诊/快速续方/医生复诊/商城问诊 |
| 关联商务 | 文本搜索（非严格模式时显示） |
| 是否会诊管理员 | 是/否 |

### 第二步：查看医生列表

表格展示以下信息：

| 列名 | 说明 |
|------|------|
| 编号 | 医生ID |
| 医生姓名 | 医生姓名 |
| 所属医院 | 医院名称 |
| 科室 | 科室名称 |
| 职称 | 职称名称 |
| 联系电话 | 默认为脱敏状态，点击可查看完整号码（同一时间只能查看一个） |
| 工作类型 | 医生/医助 |
| 开启服务 | 显示该医生开通的服务类型 |
| 所属机构 | 机构名称（机构模式时显示） |
| 关联商务 | 邀请该医生注册的商务人员 |
| 关注量 | 患者关注数 |
| 在线状态 | 在线/离线 |
| 状态开关 | 启用/禁用，带二次确认 |
| CA认证状态 | CA证书认证情况（开启CA时显示） |
| CA用户同步状态 | 未审核/身份审核通过/证书签发/设置签章/用户注销/申请拒绝/用户停用 |
| 注册时间 | 注册日期时间 |
| 操作 | 编辑、邀请码、变更手机号、同步资料、去认证、设为/取消会诊管理员 |

### 第三步：编辑医生

点击「编辑」按钮，跳转到医生详情页，可修改医生的基本信息和专业信息。

### 第四步：管理邀请码

点击「邀请码」按钮，弹出二维码弹窗：
- 显示医生的专属邀请二维码
- 可点击「刷新二维码」生成新的邀请码

### 第五步：变更手机号

点击「变更手机号」按钮，弹出手机号变更弹窗：
- 输入11位新手机号
- 实时校验手机号格式（以1开头的11位数字）
- 校验手机号是否已被其他医生使用，若已存在提示「手机号码已存在，请重新输入」

### 第六步：管理账号状态

每行医生记录都有一个启用/禁用的开关按钮：
- 点击启用：二次确认「是否确认启用该医生账号」
- 点击禁用：二次确认「是否确认禁用该医生账号，禁用后该医生账号不能正常使用」

### 第七步：CA相关操作（开启CA时显示）

- **同步资料**：将医生信息同步到CA平台。仅对「未认证」或「认证失败」状态的医生可用
- **去认证**：引导医生前往CA平台完成实名认证。仅对「认证中」状态的医生可用

### 第八步：管理会诊管理员权限

点击「设为会诊管理员」或「取消会诊管理员」可管理医生的会诊管理权限。这两个按钮互斥显示。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 手机号格式错误 | 红色提示「手机号格式错误」 |
| 手机号已被占用 | 红色提示「手机号码已存在，请重新输入」 |
| 手机号为空 | 红色提示「手机号不能为空」 |
| 医生已被禁用 | 状态开关显示为禁用，该医生无法正常使用小程序 |
| 严格模式 | 隐藏「关联商务」相关字段和信息 |
| 分页数据为空且不在第一页 | 自动回退到上一页 |
