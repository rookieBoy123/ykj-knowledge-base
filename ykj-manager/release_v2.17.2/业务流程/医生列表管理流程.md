# 医生列表管理流程

> 文档版本：2026-07-02 | 面向读者：管理员角色

## 概述

管理员在后台的医生管理模块中查看、搜索、筛选所有注册医生，管理医生的账号状态（启用/禁用）、编辑医生资料、查看邀请码、变更手机号、管理会诊管理员权限、处理CA认证等。

## 触发条件

- **前置条件**：管理员已登录管理后台，拥有医生管理权限
- **入口路径**：左侧菜单 → 医院管理 → 医生管理 → 医生列表

## 步骤详解

### 第一步：筛选医生

页面顶部提供多维度筛选条件：

- **医生姓名**：输入框模糊搜索，最多5个字
- **工作类型**：下拉选择"全部 / 医生 / 医助"
- **科室**：树形下拉选择，科室数据从医院科室树动态加载
- **所属医院**：远程搜索下拉框，输入医院名称模糊匹配
- **职称**：下拉选择，职称数据从字典表（`yszyjszwdm`）加载
- **账号状态**：启用（10）/ 禁用（20）/ 全部
- **所属机构**：显示条件取决于系统配置 `organizationEnabled` 开关，仅在多机构模式下显示
- **开启服务**：多选下拉，可同时筛选开通了图文问诊/视频问诊/咨询服务/快速问诊/快速续方/医生复诊/商城问诊的医生
- **是否会诊管理员**：是/否
- **关联商务**：非严格模式下显示，输入商务姓名模糊搜索

点击搜索按钮后列表刷新。从医生详情页返回时列表保持缓存状态不刷新。

### 第二步：查看医生列表

列表列展示以下信息：

| 列名 | 说明 |
|------|------|
| 编号 | 医生ID |
| 医生姓名 | - |
| 所属医院 | - |
| 科室 | - |
| 职称 | - |
| 联系电话 | 默认脱敏显示，点击后可查看完整号码（需验证权限） |
| 工作类型 | 医生/医助 |
| 开启服务 | 显示该医生开通的服务名称列表 |
| 所属机构 | 多机构模式下显示 |
| 关联商务 | 非严格模式下显示，展示关联的商务姓名 |
| 关注量 | 患者关注数 |
| 在线状态 | 在线（绿）/ 离线（灰） |
| 状态 | 开关按钮，可切换启用/禁用 |
| CA认证状态 | CA功能启用时显示：未认证/认证中/认证成功/认证失败 |
| CA用户同步状态 | 显示CA同步详细状态：未审核/身份审核通过/证书签发/设置签章/用户注销/申请拒绝/用户停用 |
| 注册时间 | - |

### 第三步：管理医生账号状态

点击列表中的状态开关：
- **禁用**：弹出确认提示"是否确认禁用该医生账号，禁用后该医生账号不能正常使用"，确认后状态变为禁用（20），医生无法登录和使用小程序
- **启用**：弹出确认提示"是否确认启用该医生账号"，确认后状态变为启用（10）

### 第四步：编辑医生资料

点击"编辑"按钮跳转到医生详情编辑页面（`/mgmt/hosp/doctormanage/doctorlist/info`），可修改医生基本信息、执业信息、服务配置等。

### 第五步：查看和管理邀请码

点击"邀请码"按钮弹出二维码弹窗：
- 显示医生专属的患者邀请二维码
- 点击"刷新二维码"生成新的邀请码
- 患者扫码后可关注该医生

### 第六步：变更手机号

点击"变更手机号"弹出表单弹窗：
- 输入新的手机号码（11位，实时校验格式）
- 后端校验手机号是否已被其他医生/药师使用，重复则提示"手机号码已存在，请重新输入"
- 修改成功后弹出确认提示

### 第七步：管理CA认证

当系统启用CA功能（`ca.enabled`）时：

- **同步资料**：仅在医生认证状态为"未认证"（1）或"认证失败"（4）时可用，点击后将医生信息同步到CA平台
- **去认证**：仅在认证状态为"认证中"（2）时可用，点击后弹出二维码引导医生完成认证流程
- CA认证状态列实时展示认证进度

### 第八步：管理会诊管理员权限

- **设为会诊管理员**：对非管理员医生显示，点击直接生效，医生获得审核会诊申请的权限
- **取消会诊管理员**：对已是管理员的医生显示，点击弹出确认提示，确认后取消权限

### 第九步：分页与缓存

- 列表支持分页，每页默认10条
- 从医生编辑页返回时列表保持缓存（不重新加载）
- 从其他页面进入时自动重置搜索条件并重新加载

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 手机号格式错误 | 校验不通过，提示"手机号格式错误" |
| 手机号已被其他医生使用 | 提示"手机号码已存在，请重新输入" |
| 禁用医生账号 | 二次确认弹窗 + 提示禁用后该医生不能正常使用 |
| 非会诊管理员审核会诊 | 后端拦截，提示"您不是会诊管理员, 无法审核会诊" |
| CA功能未启用 | CA相关列和按钮不显示 |
| 严格模式 | "关联商务"筛选条件和列均隐藏 |
| 非多机构模式 | "所属机构"筛选和列不显示 |

## 关联功能

- **医生详情编辑**：编辑医生的基本信息和执业信息
- **医生排班管理**：为医生设置出诊班次
- **医生服务配置**：设置医生各服务的价格和时长
- **会诊审核管理**：会诊管理员审批会诊申请
- **商务管理**：商务与医生的邀请关系
