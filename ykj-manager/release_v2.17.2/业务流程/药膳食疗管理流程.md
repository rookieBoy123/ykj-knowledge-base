# 药膳食疗管理流程

> 文档版本：2026-07-03 | 面向读者：运营人员、内容管理员

## 概述

药膳食疗是医学百科下的一个内容管理模块，用于维护药膳配方信息。每个药膳条目包含名称、封面图、功效、适应症、功能作用、主料、辅料、做法和食物拓展等信息。管理员可以新增、编辑、删除药膳食疗条目。新增和编辑操作通过弹窗完成。

## 触发条件

- **入口路径**：管理后台 → 运营管理 → 医学百科 → 药膳食疗
- **前置条件**：拥有对应菜单权限的管理员

## 步骤详解

### 第一步：浏览药膳食疗列表

进入页面后，系统自动加载药膳食疗列表，每页显示10条记录。列表展示以下信息：
- **药膳名称**：药膳的标准名称
- **功效**：多个功效以顿号分隔显示
- **适应症**：多个适应症以顿号分隔显示
- **功能作用**：药膳的功能描述
- **主料**：主要食材
- **辅料**：辅助食材
- **做法**：制作方法
- **最后修改时间**：最近编辑时间

管理员可以通过筛选条件过滤数据：
- **药膳名称**：按名称搜索（最多20字）
- **食材**：下拉选择，从系统字典 `yssl-sc` 获取（注意：食材筛选时使用字典编码对应的中文名称进行查询）
- **功效**：下拉选择，从系统字典 `yssl-gx` 获取
- **适应症**：下拉选择，从系统字典 `yssl-syz` 获取

### 第二步：新增药膳食疗

点击「新增」按钮，弹出新增弹窗（宽度650px）。弹窗包含：
- **药膳名称**：必填，最多20字
- **展示封面图**：上传封面图，要求200×200像素或1:1比例，不超过2MB
- **功效**：多选下拉，从字典中选择（可选）
- **适应症**：多选下拉，从字典中选择（可选）
- **功能作用**：必填，文本域，最多300字
- **主料**：必填，文本输入，最多100字
- **辅料**：文本输入，最多50字（可选）
- **做法**：必填，文本域，最多300字
- **食物拓展**：文本域，最多300字（可选）

### 第三步：编辑和删除

编辑通过弹窗完成，自动回显已有数据。删除支持单条和批量，均有二次确认。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药膳名称为空 | 表单校验提示"请填写药膳名称" |
| 功能作用为空 | 表单校验提示"请填写功能作用" |
| 主料为空 | 表单校验提示"请填写主料" |
| 做法为空 | 表单校验提示"请填写做法" |
| 新增成功后 | 自动回到列表第一页 |

## 关联功能

- 通过服务端 API 提供给小程序端的药膳食疗展示
- 食材、功效、适应症的下拉选项来源于系统字典
