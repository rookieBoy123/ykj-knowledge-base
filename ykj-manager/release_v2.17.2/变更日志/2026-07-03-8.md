# 变更日志 2026-07-03 (第8轮)

## 新增业务流程（5篇）

1. **互联网机构信息管理流程** — 管理互联网医院在监管平台的备案信息，包括机构基本信息、实体医院信息、证照资料等。涉及30+个配置字段和多重校验。
2. **推荐医生排序与管理流程** — 管理患者端推荐医生名单，支持添加/删除/排序（排至最前/上移/下移/排至最后），排序决定小程序展示顺序。
3. **疾病类别（症状）管理流程** — 管理患者端"对症询医"的症状分类，患者端最多展示8个症状，支持排序和显示/隐藏控制。
4. **健康咨询资讯管理流程** — 管理患者端健康科普文章，支持新增/编辑/显示隐藏/批量删除，文章分类来自字典"患者内容分类"。
5. **药房证照授权管理流程** — 管理医生与药房的授权绑定关系，控制医生可为哪些药房开方，需要 `med:pharmacy-doctors:write` 权限。
