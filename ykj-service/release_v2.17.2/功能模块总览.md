# 源康健互联网医院 — 服务端功能模块总览

> 文档版本：v1.0 | 最后更新：2026-07-01 | 面向读者：产品/运营/非技术人员

## 概述

源康健互联网医院服务端是一个基于 Java Spring Boot 的多模块微服务架构系统，为小程序医生端、患者端和管理后台提供统一的业务支撑。系统共划分为 30 个功能模块，覆盖从用户注册、医生审核、问诊接诊、处方开具、药品管理、订单支付、医保结算、随访管理到远程会诊等全链路业务。

---

## 核心业务模块

### 会员模块（member）
管理平台所有用户（患者、医生、管理员）的注册、登录、个人信息和身份绑定。

- **用户注册**：支持手机号+微信授权注册，默认头像昵称自动生成
- **个人信息编辑**：昵称、头像更新，手机号绑定（需短信验证）
- **微信绑定**：openId 与 memberId 绑定、解绑、换绑
- **金豆积分**：金豆增减、会员等级管理
- **银行卡管理**：医生银行卡信息存储（用于收益结算）
- **收货地址管理**：增删改查、默认地址设置、三级地区联动
- **搜索历史**：本地搜索记录保存（最多12条）

### 问诊模块（inquiry）
管理在线问诊的完整生命周期，是平台最核心的业务模块。

- **问诊发起**：患者选择医生后发起图文/视频/电话问诊
- **医生接诊**：医生接诊、转诊、协助预约
- **问诊结束**：医生主动结束问诊或自动超时结束
- **问诊取消**：支持患者取消、医生拒诊、系统超时取消
- **预约问诊**：排班制预约问诊，支持医生排班表
- **病历管理**：问诊病历创建、编辑、查看（支持图片附件）
- **处方管理**：处方草稿/保存/提交、医生签章、药师审核
- **用药需求**：用药需求提交与处理
- **续方/快速续方**：历史处方复用和快速开方
- **检查报告**：检验/检查/体检报告的上传、查看和管理
- **AI 医生分身**：AI 预问诊、AI 医生 PC 端协同
- **收益结算**：医生问诊收益计算和结算

### 订单模块（order）
管理平台所有交易订单，是资金流转的核心。

- **订单创建**：支持多来源类型（问诊/处方/预约/会诊/服务包等）
- **订单支付**：微信支付（JSAPI/扫码）、支付宝、医保支付三种渠道
- **支付回调**：微信/支付宝支付回调处理、医保支付回调
- **支付状态查询**：主动查单、历史渠道补单、防重复支付
- **退款管理**：退款申请→审核→原路退回/手动退款
- **订单发货**：处方药品订单的物流发货
- **订单签收**：自动签收（发货15天后）、取货码签收
- **发票管理**：订单发票申请和导出
- **费用明细**：订单收费项拆分（诊费、药费、运费等）
- **代支付**：分享订单链接给他人代付

### 问诊模块（consultation）
管理群会诊和机构间远程会诊。

- **群会诊**：医生发起→邀请其他医生→患者确认→视频会诊→填写报告→CA签章
- **机构间会诊**：机构A发起→邀请机构B→B审核→视频会诊→生成报告
- **会诊医生管理**：主治医生、参会医生、邀请医生
- **会诊文件管理**：上传/查看会诊所需影像文件（支持DICOM）
- **会诊报告**：填写、审核、签章（支持医网信/源康健PaaS/放心签平台）
- **视频通话**：基于腾讯TRTC的视频/音频通话服务

### 药品管理模块（medicine）
管理药房、药品库、处方模板和公共药方。

- **药房管理**：自营药房和第三方药房信息维护
- **药品库**：西药/中成药/中药饮片库的统一管理
- **药品导入**：Excel批量导入、模板校验
- **药品黑名单**：特殊管制药品限制
- **煎药服务**：代煎药配制和加工服务管理
- **药品库存**：药房库存、药品上下架
- **处方模板**：医生个人处方模板（中药/成药/公用三类）
- **加工服务**：药品炮制加工服务管理

### 微信医保模块（yb）
管理微信医保支付的对接。

- **医保支付**：微信医保授权、医保支付、医保结算
- **医保退款**：医保订单退款处理
- **医保结算上传**：医保结算数据上报
- **慢病管理**：慢性病种管理和医保定点管理

### 随访模块（followup）
管理医生对患者的定期随访。

- **随访计划**：创建、编辑、删除随访计划
- **随访任务**：生成随访任务（按时间节点触发）
- **随访模板**：模板创建与管理（含计划节点+内容配置）
- **随访内容**：三种内容类型——患教资料、问卷、提醒
- **患者随访**：患者端查看和执行随访任务

### 量表模块（scale）
管理健康测评量表。

- **量表库**：量表创建、编辑、分类管理
- **量表分类**：量表分类标签管理
- **题目管理**：量表题目（选择题、问答题、图片题等）
- **评分标准**：量表评分规则配置
- **测评记录**：用户测评记录管理（未完成/已完成）
- **测评发送**：医生向患者发送量表测评

### 医院管理模块（hospital）
管理医院基础信息和医生资源。

- **医生管理**：医生注册、审核、信息维护
- **医生排班**：医生出诊排班表（按日期/时间段）
- **医生服务类型**：图文/视频/电话/线下门诊等服务配置
- **诊费设置**：医生各服务类型的收费标准
- **常用语管理**：医生快捷回复常用语
- **慢病科室**：慢病科室与病种关联
- **专家团队**：专家团队创建和成员管理
- **医生视频**：医生科普视频管理
- **收支管理**：收费项目管理
- **AI 医生**：AI 医生分身管理

### HIS对接模块（his）
管理医院信息系统对接。

- **医生同步**：HIS医生数据同步
- **科室同步**：HIS科室数据同步
- **药品同步**：HIS药品数据同步
- **病历同步**：HIS病历/处方数据同步（线下门诊记录）

### 系统管理模块（system）
管理平台级配置和运营工具。

- **字典管理**：系统字典维护（各类下拉选项）
- **用户管理**：后台管理员账号管理
- **角色权限**：角色、菜单权限配置
- **系统设置**：平台级配置参数管理
- **页面装修**：小程序首页/功能页的自定义布局
- **素材管理**：图片/视频等素材库
- **消息通知**：站内消息推送管理
- **定时任务**：系统定时任务调度管理
- **登录日志**：管理后台登录日志审计

### 预约模块（reservation）
管理线下服务预约。

- **服务项目管理**：体检/核酸/健康证/检查项目配置
- **预约管理**：预约创建、取消、核销
- **电子发票**：预约服务电子发票开具

### 其他辅助模块
- **即时通讯（im）**：腾讯IM集成，管理聊天账号和群组
- **实时音视频（rtc）**：腾讯TRTC视频通话管理
- **内容安全（security）**：用户Token管理、接口鉴权
- **归档（archiving）**：历史病历数据归档
- **CA签章（ca）**：电子签名认证（医网信/源康健/放心签）
- **人脸识别（faceid）**：医生实名认证人脸核验
- **订阅消息（subscribe）**：微信小程序订阅消息推送
- **开放平台（open）**：第三方医院系统对接API
- **标准化（std）**：标准科室诊断、标准药品编码映射

## 模块间关系

```
患者端小程序 → 问诊模块(inquiry) → 订单模块(order) → 支付/退款
              → 药品模块(medicine) → 处方/药品库
              → 随访模块(followup) → 医生发起随访
              → 量表模块(scale) → 健康测评
              → 会员模块(member) → 用户信息管理

医生端小程序 → 问诊模块(inquiry) → 接诊/处方/病历
              → 会诊模块(consultation) → 群会诊/远程会诊
              → HIS模块(his) → 线下病历同步
              → 医院模块(hospital) → 排班/诊费/常用语

管理后台     → 所有mgmt接口 → 审核/配置/查询/统计
```
