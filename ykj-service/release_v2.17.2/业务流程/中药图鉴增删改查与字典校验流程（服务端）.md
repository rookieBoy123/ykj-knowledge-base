# 中药图鉴增删改查与字典校验流程（服务端）

> 文档版本 2026-07-04 | 面向读者：产品/技术 | 服务模块：ykj-service-hospital

## 概述

中药图鉴是中医百科中展示中药知识的模块。服务端提供增删改查接口，校验药味和适应症的字典有效性，并自动生成拼音供 App 端检索。

## 接口清单

| 操作 | 接口路径 | 方法 | 权限 |
|------|---------|------|------|
| 分页查询 | `/mgmt/wiki/herb-illustrations` | GET | `hosp:wiki:herb-illustrations:read` |
| 详情查询 | `/mgmt/wiki/herb-illustrations/{id}` | GET | `hosp:wiki:herb-illustrations:read` |
| 新增 | `/mgmt/wiki/herb-illustrations` | POST | `hosp:wiki:herb-illustrations:write` |
| 编辑 | `/mgmt/wiki/herb-illustrations/{id}` | PUT | `hosp:wiki:herb-illustrations:write` |
| 删除 | `/mgmt/wiki/herb-illustrations/{id}` | DELETE | `hosp:wiki:herb-illustrations:write` |
| 批量删除 | `/mgmt/wiki/herb-illustrations/_batch-delete` | POST | `hosp:wiki:herb-illustrations:write` |

## 核心规则

### 1. 名称唯一性校验

- 新增/编辑时检查中药名称是否重复
- 编辑时排除自身ID
- 重名时抛出："中药名称已存在"

### 2. 字典有效性校验

新增/编辑时校验以下两个字典：

| 校验内容 | 字典编码 | 说明 |
|---------|---------|------|
| 药味 | `zytj-yw` | 如酸、苦、甘、辛、咸等，多选 |
| 适应症 | `zytj-syz` | 可多选，可为空 |

药味每次必校验（多选列表整体校验），适应症仅在有值时校验。

### 3. 拼音自动生成

调用 `ChinesePinyinUtil.getPinYin()` 生成拼音，存入 `pinyin` 字段。

### 4. 字典转译

查询时自动将药味和适应症的编码转为中文名称展示。药味编码如 `ku` → "苦"，适应症编码如 `gm` → "感冒"。

### 5. 与经典名方的差异

| 特性 | 经典名方 | 中药图鉴 |
|------|---------|---------|
| 字典校验 | 3个字典（类型+出处+适应症） | 2个字典（药味+适应症） |
| 药味字段 | 无 | 有（多选） |
| 书籍出处 | 有（必选） | 无 |
| 封面图 | 可选 | 可选 |

## 异常处理

| 异常情况 | 返回信息 |
|---------|---------|
| 名称重复 | "中药名称已存在" |
| 药味编码不存在 | "字典:zytj-yw, 编码:xxx不存在" |
| 适应症编码不存在 | "字典:zytj-syz, 编码:xxx不存在" |
| 删除ID为空 | "请正确选择要删除的中药图解" |
