# 标准药品信息管理流程（服务端）

> 文档版本：2026-07-03 | 面向读者：产品经理、技术运营

## 概述

标准药品信息管理用于维护系统中的标准药品基础数据。标准药品库是整个系统药品管理的"基准数据"，包括药品通用名、规格、剂型、生产厂家等信息。管理后台对标准药品的操作通过服务端接口完成。

## 触发条件

- **管理入口**：管理后台 → 医院管理 → 标准管理 → 标准药品信息
- **服务模块**：`ykj-service-std`
- **权限要求**：需要标准药品相关的读写权限

## 接口说明

### 1. 分页查询

**接口**：`GET /mgmt/std/stdMed`

支持分页查询和条件筛选（药品通用名、标准编码等），返回标准药品列表。

### 2. 新增标准药品

**接口**：`POST /mgmt/std/stdMed`

管理员录入新的标准药品信息，包括药品通用名、标准编码、剂型、规格等基础信息。

### 3. 修改标准药品

**接口**：`PUT /mgmt/std/stdMed`

修改已有标准药品的信息。

### 4. 查询详情

**接口**：`GET /mgmt/std/stdMed/detail`

按 ID 或药品通用名查询标准药品的详细信息。

### 5. 导入

**接口**：`POST /mgmt/std/stdMed/import`

支持通过 Excel 文件批量导入标准药品数据。

## 关联功能

- 标准药品诊断对照管理
- 药房药品管理（药房药品需关联到标准药品）
- 处方开具时的药品选择
- 服务模块：`ykj-service-std`
