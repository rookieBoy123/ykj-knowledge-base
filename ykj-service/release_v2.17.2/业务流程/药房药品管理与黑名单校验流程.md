# 药房药品管理与黑名单校验流程

> 文档版本 v1.0 | 面向读者：药房管理员、运营人员

## 概述

药房药品管理的核心服务，负责药品的新增、编辑、删除、导入导出、库存管理和黑名单检测。药品分为中药（草药）和成药（西药/中成药）两种类型，各自有不同的属性字段和管理逻辑。

## 核心功能

### 一、药品搜索与黑名单拦截

**医生搜索药品时**，系统进行以下处理：
- 自动获取医生已授权的药房列表（未授权任何药房时提示「医生暂未授权任何药房, 请联系管理员」）
- 成药搜索时检测搜索关键词是否命中药品黑名单（如精麻毒管制类药品），命中时提示「该药品为精麻毒等管制类药品，按规定不可线上开方。」
- 支持药品名称拼音首字母搜索（如输入英文字母按字符串相似度排序）

### 二、药品新增

**中药新增流程**：
1. 根据药品通用名在药品库中查找对应记录
2. 药品库中不存在时提示「药品库不存在该药品，通用名=xxx」
3. 从药品库同步基本信息（别名、名称、剂型、单位）并关联药品库ID
4. 检测药品名是否命中黑名单
5. 校验是否已存在相同药品（如果已存在提示「药品已存在」）
6. 校验福建地区批准文号是否必填
7. 校验第三方药房药品必填项

**成药新增流程**：
1. 根据药监ID在药品库中查找对应记录
2. 药品库中不存在时提示「药品库不存在该药品，药监ID=xxx」
3. 从药品库同步基本信息并自动生成拼音首字母
4. 插入药品主表和成药详情表

### 三、药品编辑

编辑时对不可修改字段进行保护：
- **中药不可修改**：通用名、药品名称、剂型、单位
- **成药不可修改**：通用名、药品名称、剂型、分类、生产厂家、规格、剂量单位、单位

每次编辑均需重新校验第三方药房必填项和福建地区批准文号。

### 四、药品导入导出

**导出**：支持按药品类型分别导出中药和成药的Excel文件，自动添加序号列。

**导入**：支持批量Excel导入，每行数据处理时：
- 自动生成拼音首字母（如果未填）
- 已存在的药品覆盖更新，新增的药品新建
- 导入失败的记录在错误列表中显示错误原因

### 五、库存扣减

下单时系统扣减库存：
- 检查药品是否存在
- 检查药房是否开启库存管理
- 库存设置为-1表示不限制库存
- 根据药品库中的换算比例转换库存单位
- 库存不足时返回失败，阻止下单

### 六、药品禁用与删除

- 支持按药品ID列表批量禁用（设置状态为禁用）
- 支持按药品库ID批量删除药品
- 删除时同时清除中药/成药子表数据

## 状态流转

```
药品状态：正常（启用）→ 禁用
导入模式：新药品 → 新增; 已存在药品 → 覆盖更新
库存模式：-1=不限制库存; ≥0=限制库存（下单时扣减）
```

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 医生未授权任何药房 | 后端拦截，提示「医生暂未授权任何药房, 请联系管理员」 |
| 搜索命中黑名单药品 | 后端拦截，提示「该药品为精麻毒等管制类药品，按规定不可线上开方。」 |
| 药品库中不存在 | 后端拦截，提示「药品库不存在该药品」 |
| 药房已存在相同药品 | 后端拦截，提示「药品已存在」 |
| 药品ID参数错误 | 后端拦截，提示「参数错误」 |
| 库存不足 | 返回失败，阻止库存扣减 |
| 同一药房存在多个相同名称药品 | 后端拦截，提示「药房里存在多个相同名称的药材, 请联系管理员」 |
| 药品类型未知 | 后端拦截，提示「未知的药品类型」 |
| 导出失败 | 后端拦截，提示「药房药品导出失败」 |

## 关联模块

- **药品库服务** (ykj-service-medicine)：提供标准药品库数据
- **药房服务**：管理药房授权和库存开关
- **黑名单服务**：管理精麻毒管制类药品清单
- **医保药品服务**：医保药品目录
- **处方结算服务**：下单时调用扣库存
