# 药品加工工艺与类别管理流程（服务端）

> 文档版本 | 面向读者：开发人员 / 产品经理

## 概述
中药在配药时需要经过特定的加工处理（如切片、炒制、醋制等），系统提供加工工艺和加工类别的管理功能，用于维护这些工艺选项。药房在配置中药加工服务时，可以从已维护的工艺和类别中选择。

## 触发条件
- **数据来源**：管理后台药品加工工艺/类别管理页面
- **服务模块**：`ykj-service-medicine`

## 核心流程

### 1. 加工工艺管理

**接口前缀**：`/mgmt/medicine/processing-technology`
**权限前缀**：`med:processing-technology`

| 操作 | 接口 | 说明 |
|------|------|------|
| 新增 | `POST /` | 添加新的加工工艺 |
| 编辑 | `PUT /{id}` | 修改已有工艺 |
| 删除 | `DELETE /{id}` | 删除单个工艺 |
| 批量删除 | `POST /batch_delete` | 传入ID列表批量删除 |
| 查询详情 | `GET /{id}` | 查看单个工艺 |
| 分页查询 | `GET /` | 分页浏览全部工艺 |

### 2. 加工类别管理

**接口前缀**：`/mgmt/medicine/processing-category`
**权限前缀**：`med:processing-category`

| 操作 | 接口 | 说明 |
|------|------|------|
| 新增 | `POST /` | 添加新的加工类别 |
| 编辑 | `PUT /{id}` | 修改已有类别 |
| 删除 | `DELETE /{id}` | 删除单个类别 |
| 查询详情 | `GET /{id}` | 查看单个类别 |
| 列表查询 | `GET /list` | 获取全部类别列表 |

### 3. 业务关系

```
加工类别（Category）
  └── 加工工艺（Technology）
       └── 药房中药加工服务配置
```

加工类别是加工工艺的上层分类，例如"炒制"可能是一个类别，其下包含"清炒""麸炒""砂炒"等具体工艺。

### 4. 状态流转

```
新增工艺/类别 ──→ 持久化到数据库 ──→ 药房可以在配置中引用
编辑工艺/类别 ──→ 更新数据库 ──→ 已引用的配置同步更新
删除工艺/类别 ──→ 从数据库移除 ──→ 已引用的配置需单独处理
```

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 新增时数据校验失败 | 返回验证错误信息 |
| 删除已被引用的工艺/类别 | 取决于业务逻辑层的引用检查 |

## 关联功能
- **药房中药加工服务配置**：药房选择加工工艺时从此处获取选项
- **处方中药代加工**：患者选择加工服务时基于此配置
