# 处方预结算与医保费用上传流程

> 文档版本 v1.0 | 面向读者：产品、运营、客服

## 概述

描述患者将处方转为订单时，系统进行费用预结算（展示药品费、加工费、运费等明细）以及医保订单的费用明细上传和支付处理的完整后端链路。

## 触发条件

- **前置条件**：医生已开具处方且状态可结算
- **入口路径**：处方详情页点击"去购药"或"去结算"

## 步骤详解

### 第一步：预结算（费用预估）

患者在结算前可先查看费用预估明细：

- **处方来源**：根据门诊编号查询处方信息
- **处方状态校验**：检查处方是否处于可结算状态
- **药房信息**：获取药房名称、配送方式（快递/自取）、加工服务信息
- **收货地址**：获取患者默认收货地址（快递配送时）
- **加工费用**：确认加工服务技术方案和费用
- **费用项计算**：
  - 药品金额（单价 × 数量）
  - 加工服务费（中药代煎等）
  - 运费（根据收货地址和药房计算）
- **定金判断**：
  - 药房支持定金支付
  - 货到付款功能已开启
  - 处方金额高于最低定金门槛
  - 满足以上三个条件则显示定金选项
- **预结算结果**：返回费用明细列表和总金额（总金额如为负数则置为0）

### 第二步：正式结算（创建订单）

用户确认费用后点击"提交订单"：

- **重复订单检查**：同一处方编号已存在有效订单则拒绝创建（提示"已存在订单信息"）
- **费用计算**：正式结算时药品金额不合并服务费（费用项独立显示）
- **创建订单**：调用订单服务创建订单（详见"订单生命周期管理流程"）
- **订单号回填**：将生成的订单号回填到处方记录中

### 第三步：医保费用明细上传

医保订单需要先上传费用明细才能进行医保支付：

- **费用明细组装**：
  - 处方药品信息（国家药品编码、药品名称、单价、数量、报销类别等）
  - 诊断信息（疾病诊断编码和名称）
  - 患者医保信息（医保编号、参保地区、医保类型）
  - 医生信息
  - 诊查费（问诊服务费作为费用明细的一项，超过10元按10元计算）
- **重复上传处理**：如果该订单已有费用明细记录，先取消历史上传，再重新上传
- **授权码处理**：
  - 已有授权码：直接使用，获取支付授权号和经纬度
  - 无授权码但有历史记录：复用历史记录的支付授权号（切换个账开关场景）
- **个账支付判断**：
  - 如果系统支持个人账户支付且用户选择使用个账，标记为个账支付
  - 不支持或不使用个账，标记为0（不使用个账）

### 第四步：医保预支付

费用明细上传完成后执行医保预支付：

- **获取支付订单**：调用医保支付平台生成支付订单号
- **支付订单号保存**：将医保支付订单号、支付授权号等信息保存到订单医保扩展表

### 第五步：订单详情查询（已创建订单）

管理员或患者查询订单时可获取完整详情：

- **订单状态**：支付状态、配送状态
- **支付渠道展示**：
  - 普通支付：显示"微信""支付宝"等
  - 医保混合支付：显示"统筹支付+个人账户+微信支付"的组合方式
- **费用明细**：区分药品费、加工费、运费、诊查费等
- **收货地址**：快递配送订单显示收件人、电话、地址
- **支付方式**：快递配送或到店自取

### 第六步：医保结算明细查询

医保支付的订单支持查询结算明细：

- **查询条件**：必须是医保支付订单
- **结算明细**：调用医保平台获取分项明细（统筹支付金额、个人账户支付金额、现金支付金额等）
- **生成报销凭证**：将结算明细渲染为图片供保存

## 医保支付渠道

医保订单的付款方式组合：

| 支付渠道 | 说明 |
|---------|------|
| 统筹支付（PlanAsAWhole） | 医保统筹基金支付部分 |
| 个人账户（IndividualAccount） | 医保个人账户支付部分 |
| 微信自费（InsuranceWechat） | 个人自费部分通过微信支付 |

三种渠道的金额之和等于订单总金额。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 已存在订单信息 | 提示"已存在订单信息" |
| 处方不存在 | 提示"处方不存在，无法创建订单" |
| 药房信息不存在 | 提示"找不到对应药房信息" |
| 订单信息不存在 | 提示"订单信息不存在！" |
| 订单费用项信息不存在 | 提示"订单费用项信息不存在！" |
| 订单收货地址不存在（快递配送） | 提示"该订单收货地址信息不存在" |
| 非医保订单查询明细 | 提示"该订单不支持查询明细" |
| 未找到订单结算信息 | 提示"未找到订单结算信息" |
| 处方/问诊/患者信息异常 | 提示"处方信息异常"/"问诊信息异常"/"患者信息异常" |
