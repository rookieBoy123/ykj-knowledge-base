# 小程序登录日志脱敏与安全审计流程（服务端）

> 文档版本：2026-07-04 | 面向读者：后端开发、安全审计

## 概述

`MgmtSystemLoginLogController` 统一管理后台登录日志和小程序用户登录日志的查询。通过 `sysClient` 参数区分客户端类型，对 App 端用户手机号执行自动解密和脱敏处理。

## 接口清单

| 方法 | 路径 | 功能 | 权限 |
|------|------|------|------|
| GET | `/mgmt/system/login-logs` | 分页查询登录日志 | `sys:login-log:read` |

该接口通过 `DefaultPageEndpoint` 分页端点模式实现，自动提供分页查询能力。

## 核心逻辑

### 客户端区分

通过请求参数 `sysClient` 区分：

- **`sysClient = "app"`**：小程序端用户登录日志 → 执行手机号脱敏
- **`sysClient = "mgmt"`**（或其他）：管理后台管理员登录日志 → 不脱敏

### 手机号脱敏流程（仅 App 端）

```
查询数据库（加密手机号）
    ↓
crypto.decrypt() 解密得到明文手机号
    ↓
DesensitizedUtil.mobilePhone() 脱敏 → 138****1234
    ↓
返回给前端展示
```

代码位置：`MgmtSystemLoginLogController.assembleAppUserAccount()` 方法（第 57-63 行）

### 操作系统和浏览器解析

登录日志中不直接存储 OS 和浏览器名称，而是存储完整的 User-Agent 字符串。在返回数据时，通过 Hutool 的 `UserAgentUtil.parse()` 动态解析：

```java
public String getOs() {
    UserAgent userAgentObj = UserAgentUtil.parse(this.getUserAgent());
    return userAgentObj.getOs().getName();
}

public String getBrowser() {
    UserAgent userAgentObj = UserAgentUtil.parse(this.getUserAgent());
    return userAgentObj.getBrowser().getName();
}
```

这意味着前端无需解析 User-Agent，服务端已自动提取为可读的 OS 和浏览器名称。

### 查询参数

支持以下筛选条件：

| 参数 | 类型 | 说明 |
|------|------|------|
| userAccount | String | 用户名/账号 |
| ip | String | IP 地址 |
| startTime | Date | 开始时间 |
| endTime | Date | 结束时间 |
| result | Boolean | 登录状态（true=成功, false=失败） |
| sysClient | SysClient | 客户端类型（app/mgmt） |

## 数据存储

登录日志实体 `LoginLog` 存储以下字段：

| 字段 | 说明 |
|------|------|
| id | 主键自增 |
| userAccount | 加密存储的手机号/账号 |
| ip | 登录 IP 地址 |
| userAgent | 完整的 User-Agent 字符串 |
| loginWay | 登录方式（如微信授权、手机号） |
| loginTime | 登录时间 |
| result | 登录是否成功 |
| reason | 登录失败原因（成功时为空） |
| sysClient | 客户端类型标识 |

## 安全设计

1. **手机号加密存储**：数据库中的 `userAccount` 使用 `Crypto` 加密
2. **传输层脱敏**：API 返回前对 App 端手机号执行 `DesensitizedUtil.mobilePhone()` 脱敏
3. **权限控制**：接口受 `sys:login-log:read` 权限保护
4. **客户端隔离**：管理后台和小程序使用同一接口但执行不同的安全策略

## 对比：后台登录日志 vs 小程序登录日志

| 维度 | 后台登录日志 | 小程序登录日志 |
|------|------------|--------------|
| 查询参数 sysClient | mgmt 或不传 | app |
| 账号脱敏 | 不脱敏（显示管理员账号） | 脱敏（手机号中间四位隐藏） |
| 登录方式 | 账号密码/短信验证码 | 微信授权/手机号 |
| 前端页面 | systemmanage/system/loginLog | systemmanage/system/appLoginLog |
