# 问诊预约与患者分诊流程（服务端）

> 文档版本 1.0 | 面向非技术读者 | 2026-07-02

## 概述

问诊预约模块管理患者在线预约医生问诊的完整后端逻辑。患者选择医生和服务类型后，系统创建问诊预约单，关联就诊人信息，支持预约排班和非预约即时问诊两种模式。问诊中支持量表下发、健康商品推荐等辅助功能。

## 触发条件

- 患者在医生详情页选择服务并发起问诊
- 医生端查看自己的问诊列表
- 患者端查看自己的问诊记录

## 核心流程

### 一、问诊预约单创建

患者选择医生和服务后，系统创建问诊预约单，记录：
- 患者信息（姓名、性别、年龄、证件号）
- 医生信息（姓名、科室、职称、所属医院）
- 服务类型（图文问诊/视频问诊/咨询服务/快速问诊/AI问诊等）
- 病情描述/咨询内容
- 预约时间（如有排班）

预约模式区分：
- **预约问诊**（serviceClassify=Registration）：需要预约时段，到时间后签到
- **即时问诊**（非预约）：患者付款后直接进入问诊

### 二、问诊列表查询

**患者端列表**：按四种类型分类查询

| 类型 | 筛选条件 | 排序 |
|------|---------|------|
| 待支付（draft） | flowStatus=10 | 创建时间倒序 |
| 已预约（wait） | flowStatus=20 + 预约模式 + 未签到 | 就诊时间升序 |
| 问诊中（underway） | flowStatus=20 - 预约但未签到 | 创建时间升序 |
| 已完成（finish） | flowStatus=30 | 创建时间倒序 |

**医生端列表**：按三种类型分类
- 待接诊（draft）
- 问诊中（underway）
- 已完成（finish）

### 三、问诊详情查询

通过问诊编号查询完整问诊信息：
- 自动计算患者年龄（根据出生日期）
- 支持按病历ID关联病历
- 支持按多个问诊编号批量查询

### 四、IM消息推送

问诊过程中，系统通过IM（即时通讯）群组向患者发送各种消息：

**1. 接诊消息**：医生接诊后自动发送接诊系统消息

**2. 转诊消息**：医生流转问诊（如从兼职转全职）时发送转诊通知，包含：
- 转诊原因说明
- 新接诊医生信息

**3. 量表测评消息**：医生下发量表后，向群组发送测评卡片：
- 测评结果链接（可直接跳转）
- 适用患者ID

**4. 健康商品推荐**：医生可向患者推荐商城商品，发送商品卡片：
- 商品名称和规格
- 商品价格
- 商品图片
- "去购买"按钮（含商品ID和规格ID参数）

### 五、就诊标记

医生可以在问诊记录上标记"已就诊"（doctorVisitFlag），用于区分问诊中和已线下就诊的病例。

### 六、预约量统计

支持按日期范围和医生ID统计某段时间内的问诊预约数量，用于医生排班和数据分析。

## 问诊状态流转

```
草稿(10) ──支付成功──→ 问诊中(20)
                           │
                    ┌──────┼──────┐
                    │      │      │
                 医生接诊  预约签到  转诊流转
                    ↓      ↓      ↓
                 问诊中  问诊中   问诊中(新医生)
                    │      │      │
                    └──────┼──────┘
                           │
                    问诊结束(30)
                           │
                    ┌──────┴──────┐
                    │             │
                 评价医生      查看病历
                    │             │
                 评价完成      病历签署
```

## 医生接诊状态（doctorFlagStatus）

| 状态码 | 说明 |
|-------|------|
| 10 | 助理医生待接诊 |
| 20 | 助理医生问诊中 |
| 30 | 主治医生待问诊 |
| 40 | 主治医生问诊中 |
| 50 | 主治医生结束，助理医生再次问诊中 |
| 60 | 问诊结束 |

## 异常处理

| 情况 | 系统处理 | 提示信息 |
|------|---------|---------|
| 问诊ID为空 | 拒绝操作 | "参数错误" |
| 问诊编号不存在 | 返回空数据 | 无明确提示 |
| 非医生角色查询医生端 | 拒绝操作 | "当前用户非医生角色" |
| 医生未配置排班 | 不显示预约时段 | 患者端仅显示即时问诊选项 |
| 查询类型参数错误 | 拒绝操作 | "参数错误" |
| 医生ID为空 | 拒绝操作 | "当前用户非医生角色" |

## 关联模块

- **医生服务模块**（ykj-service-hospital）：医生信息查询、排班查询
- **患者服务模块**：就诊人信息查询和绑定
- **IM服务模块**（ykj-service-im）：即时通讯群组消息推送
- **导诊模块**（ykj-service-guidance）：导诊数据关联
- **量表模块**（ykj-service-scale）：量表测评下发
- **商城模块**（ykj-service-mall）：健康商品推荐
- **病历模块**（ykj-service-inquiry）：病历关联
- **会员模块**（ykj-service-member）：会员关系绑定
