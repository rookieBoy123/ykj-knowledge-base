# 药品生命周期管理与第三方同步流程

> 文档版本 v1.0 | 面向读者：技术运维、后台管理员

## 概述
描述药房药品从添加到维护的完整后端处理逻辑，包括中药和成药的分开管理、库存扣减、第三方药房药品同步、药品导入导出、黑名单拦截等。药品数据同时关联药房和药品库两层关系。

## 触发条件
- 后台管理员在药房管理中添加/编辑药品
- 定时任务触发第三方药房药品信息同步
- 医生开方时查询和校验药品
- 订单支付后扣减药品库存

## 核心流程

### 流程一：药品添加（中药）
1. **药房管理员提交中药信息** → 系统校验该药房下是否已存在同名药材
2. **校验药品库** → 查找国家标准药品库中是否存在该药材（按通用名匹配）
   - 不存在 → 提示"药品库不存在该药品，通用名=XX"
3. **校验黑名单** → 检查药品名称是否在精麻毒管制类药品黑名单中
   - 命中 → 拦截添加
4. **校验第三方药房** → 如果是第三方药房，校验必填字段
5. **校验福建地区** → 福建地区药房必须填写批准文号
6. **设置默认库存** → 如未填库存，默认设为-1（不限库存）
7. **写入数据** → 主表（MedicalDrugs）+ 中药子表（MedicalDrugsHerbal）
   - 不允许修改部分：通用名、药品名、剂型、单位（从药品库继承）
8. **拼音自动生成** → 导入时如未填拼音首字母，根据通用名自动生成

### 流程二：药品添加（成药）
1. **药房管理员提交成药信息** → 按药监ID查找药品库匹配
   - 不存在 → 提示"药品库不存在该药品，药监ID=XX"
2. **校验重复** → 同一药监ID在同一药房不可重复添加
3. **设置药品库信息** → 通用名、药品名、剂型、分类、生产厂家、规格、剂型单位、药品单位
4. **不允许修改部分** → 编辑时重置为空：通用名、药品名、剂型、药品分类、生产厂家、规格、剂型单位、药品单位
5. **写入数据** → 主表 + 成药子表（MedicalDrugsPatent）

### 流程三：药品编辑
- **中药编辑** → 通用名/药品名/剂型/单位不可修改（传null），校验第三方和福建地区
- **成药编辑** → 通用名/药品名/剂型/分类/厂家/规格/剂型单位/药品单位不可修改
- 编辑时校验第三方药房必填项和福建地区批准文号

### 流程四：库存扣减
1. **参数校验** → 药品ID不能为空
2. **库存开关检查** → 药房未开启库存管理，直接返回成功
3. **无限库存跳过** → 库存为-1表示不限库存，直接返回成功
4. **库存单位转换** → 如果药品库的换算比例不为1（如袋→克），先进行单位换算
5. **库存不足检查** → 当前库存 < 扣减数量时返回失败
6. **执行扣减** → 库存 = 当前库存 - 扣减数量

### 流程五：第三方药品信息同步（定时任务）
1. **拉取图片** → 遍历所有启用的第三方药房，查找缺少图片的药品
   - 调用第三方接口获取药品图片
   - 图片URL通过OSS转存后更新到数据库
2. **拉取商品信息** → 遍历启用的第三方药房，按更新时间增量拉取
   - 分页拉取（每页100条）
   - 更新：商品名、通用名、库存数量、规格、生产地
   - 请求间隔：每次拉取间隔7秒（防止第三方接口限流）
   - 注意：进货价不在此处更新（由导入时定义）

### 流程六：药品导入导出
- **导出（中药/成药）** → 按类型导出为Excel，带序号、样式和模板处理器
- **导入** → 读取Excel文件，逐行解析并保存
  - 导入时拼音字段为非必填，自动根据通用名生成
  - 已存在的药品（按药房+通用名/药监ID匹配）执行覆盖更新
  - 每行解析失败记录到错误列表返回

### 流程七：药品查询（医生端）
- **中药查询** → 支持拼音搜索，按字符串相似度排序
- **成药查询** → 搜索时先检查黑名单关键词
  - 命中黑名单 → 直接拦截提示"该药品为精麻毒等管制类药品，按规定不可线上开方"
- **药房授权校验** → 医生未指定药房时，自动查询该医生授权的所有药房
  - 无授权药房 → 提示"医生暂未授权任何药房，请联系管理员"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药品库不存在该药品 | 前端提示"药品库不存在该药品" |
| 药品已存在 | 前端提示"药品已存在"（导入模式自动覆盖更新）|
| 黑名单药品搜索 | 前端提示"该药品为精麻毒等管制类药品，按规定不可线上开方" |
| 医生未授权任何药房 | 前端提示"医生暂未授权任何药房，请联系管理员" |
| 库存不足 | 返回失败，日志记录 |
| 第三方药房药品信息错误 | 日志记录错误，跳过该药房继续处理 |
| 药品导入失败 | 返回每行的具体错误信息 |
| 福建地区批准文号必填 | 校验拦截 |
| 药房ID为空 | 前端提示"药房ID不能为空" |
