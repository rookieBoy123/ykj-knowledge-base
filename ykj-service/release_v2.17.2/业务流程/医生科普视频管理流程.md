# 医生科普视频管理流程

> 文档版本 1.0 | 面向运营人员

## 概述
运营人员在后台管理医生科普视频，将医生录制的科普视频信息同步到视频库中，支持关联具体医生，在小程序端以医生科普栏目的形式展示给患者。

## 前置条件
- 已登录管理后台
- 拥有视频管理权限
- 视频资源已上传到系统

## 核心流程

### 1. 新增医生科普视频
管理员上传视频时，系统进行四重校验：
- **视频存在性校验**：检查视频库中是否存在该视频（通过videoId查询）
- **分类存在性校验**：确保选择的视频分类有效
- **医生存在性校验**：如关联了医生，确认医生已注册
- **名称重复校验**：检查视频名称是否与已有内容重复

校验通过后：
- 在医生视频表中创建记录（保存分类、名称、封面、视频ID、关联医生）
- 同步更新视频库中的视频名称

### 2. 编辑视频信息
修改已存在的医生视频，系统再次执行四重校验：
- 视频记录存在性
- 视频资源本身存在性
- 分类存在性
- 名称排己校验（排除自身，防止与其他视频重名）

更新视频记录和同步视频库名称。

### 3. 删除视频
- 校验视频记录存在性
- 不存在的视频返回"视频不存在"错误
- 物理删除记录（不可恢复）

### 4. 视频详情查询
查询单条视频的完整信息，自动关联：
- 分类名称
- 关联医生的完整资料（姓名、头像、科室、医院、职称、执业状态）
- 视频播放地址（原视频URL和转码后URL）

### 5. 视频分页查询
支持三维筛选：分类ID + 视频名称 + 启用状态，按创建时间倒序排列。每条结果自动附加分类名称和关联医生姓名。

## 异常处理
| 情况 | 系统处理方式 |
|------|------------|
| 视频资源不存在 | 新增/编辑时抛"视频不存在" |
| 分类不存在 | 抛分类不存在错误 |
| 医生不存在 | 抛医生不存在错误 |
| 视频名称重复 | 抛名称已存在错误，需改名 |
| 删除不存在的视频 | 抛"视频不存在" |
| 关联医生已被禁用 | 分页查询时仍显示，但标记状态为禁用 |

## 关联功能
- 管理后台视频管理：视频分类管理和视频列表
- 小程序患者端：医生科普栏目查看视频
- 医生列表：查看医生关联的视频数量
