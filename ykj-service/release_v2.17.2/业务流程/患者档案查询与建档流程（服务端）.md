# 患者档案查询与建档流程（服务端）

> 文档版本 v1.0 | 面向读者：开发人员、系统管理员

## 概述

患者档案是互联网医院的核心数据，关联会员账号、HIS系统患者信息、慢病登记、标签等。服务端提供患者档案的查询、快速建档（从HIS同步）、解绑档案等功能。

## 接口总览

| 接口 | 方法 | 权限 | 说明 |
|------|------|------|------|
| `/mgmt/patient/patients/_findIdsByIdCard` | GET | `hosp:patients:read` | 根据身份证号查询患者ID |
| `/mgmt/patient/patients/_findByIdCard` | GET | `hosp:patients:read` | 根据身份证号查询患者详情 |
| `/mgmt/patient/patients/_quick-filing` | POST | `hosp:patients:write` | 快速建档（从HIS同步） |
| `/mgmt/patient/patients/_unbind` | POST | `hosp:patients:write` | 解绑档案 |
| `/mgmt/patient/patients`（分页） | GET | `hosp:patients:page` | 患者分页查询 |
| `/mgmt/patient/patients/{id}` | GET | `hosp:patients:get` | 患者详情查询 |

## 业务流程详解

### 快速建档（从HIS同步）

这是患者档案创建的核心流程，用于管理员手动触发从HIS系统同步患者信息。

**流程步骤**：

1. **本地查询**：先根据身份证号在本系统查询患者是否存在
2. **已存在 → 直接返回**：如果患者已建档，直接返回现有患者信息
3. **不存在 → 从HIS查询**：
   - 调用 `ApiHisPatientService.getPatInfo(idCard)` 查询HIS系统中的患者信息
   - 如果HIS中也查不到（`hisPatientDto == null`），抛出异常："从HIS查询档案失败，请联系管理员!"
4. **查询/创建会员**：
   - 用HIS患者手机号调用 `ApiMemberService.findByPhone()` 查询会员
   - 如果会员不存在，调用 `ApiMemberService.register()` 注册新会员
5. **创建患者档案**：
   - 将HIS患者信息转换为 `PatientDto`
   - 设置 `memberId`（关联会员）
   - 设置 `relationshipCode = "0"`（默认关系为"本人"）
   - 调用 `PatientService.add()` 创建患者记录
6. **绑定HIS关系**：
   - 调用 `ApiHisPatientService.addRelation()` 绑定HIS患者ID与本系统患者ID的关联关系
   - 传入 `hisPatId`、`patientId`、`birthday`

**返回**：创建或查询到的患者信息（`PatientVo`）

### 根据身份证号查询

- **`_findIdsByIdCard`**：仅返回患者ID列表，用于快速校验
- **`_findByIdCard`**：返回患者完整信息，包括姓名、性别、出生日期、年龄（通过 `AgeUtils.getAgeStr()` 计算）

### 患者分页查询

继承 `DefaultPageEndpoint` 接口，由 `PatientAssembler.findMgmtPage()` 实现：
- 支持多条件搜索（`PatientSearchDomain`）
- 调用 `PatientServiceImpl` 的 `findPage` 方法
- 数据来源于 `Patient` 实体对应的数据库表

### 患者详情查询

继承 `DefaultGetEndpoint` 接口，由 `PatientAssembler.findDetails()` 实现：
- 返回患者详细信息，包括关联的会员账号、患者标签、慢病登记、病历记录等
- 管理员在后台点击患者时调用此接口

### 解绑档案

- 接收 `UnbindVo`（含 `patientId`）
- 调用 `PatientService.unbind()` 解除患者档案的绑定关系
- 用于处理数据异常或患者要求注销的场景

## 权限控制

| 权限码 | 可执行操作 |
|--------|-----------|
| `hosp:patients:read` | 根据身份证查询患者 |
| `hosp:patients:write` | 快速建档、解绑档案 |
| `hosp:patients:page` | 患者分页查询 |
| `hosp:patients:get` | 患者详情查询 |

## 异常分支

| 异常 | 触发条件 | HTTP状态 |
|------|---------|----------|
| 从HIS查询档案失败 | 调用HIS接口返回null | 400，提示"从HIS查询档案失败，请联系管理员!" |
| 身份证号为空 | 快速建档时 `idCard` 为空 | 400（@NotEmpty校验） |

## 数据模型关联

- **Patient**（患者档案）← 关联 → **Member**（会员账号）通过 `memberId`
- **Patient** ← 关联 → **HIS患者**通过 `hisPatId`（`ApiHisPatientService.addRelation`）
- **PatientVo** 包含年龄计算（`AgeUtils.getAgeStr(birthday)`），输出格式如"30岁3月"

## 关联服务

- **ApiHisPatientService**（HIS模块）：查询HIS患者信息、绑定HIS患者关系
- **ApiMemberService**（会员模块）：查询/注册会员账号
- **PatientService**（医院模块）：患者档案CRUD
- **PatientAssembler**：数据组装和转换
