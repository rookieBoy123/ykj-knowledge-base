# 支付宝支付功能开关配置流程（服务端）

> 文档版本：v1.0 | 最后更新：2026-07-05 | 面向读者：产品经理、运维人员

## 概述

支付宝支付是一项全局功能开关，管理员可以在后台全局配置中启用或关闭。关闭后，小程序的收银台将不展示支付宝支付选项；开启后，支付宝支付出现在支付方式列表中，且管理员可以进一步配置支付宝商户参数（应用ID、公私钥等）。

> ⚠️ 注意：开启支付宝支付功能开关仅控制前端的"是否展示支付宝支付方式"，实际的支付二维码生成依赖于该功能下的支付宝商户配置是否填写完整。

## 触发条件

- **前置条件**：系统管理员登录管理后台，具有全局配置权限
- **入口路径**：管理后台 → 系统管理 → 全局配置 → 功能开关 → 支付宝支付

## 步骤详解

### 第一步：开启功能开关

管理员在全局配置页面的"功能开关"分组中找到"支付宝支付"开关：

- **开关位置**：功能开关 > 支付宝支付
- **配置属性**：`ykj.func.alipayEnabled`
- **类型**：布尔值（是/否）
- **默认值**：否（关闭）
- **配置说明**："开启后，支付收银台将展示支付宝支付方式"

管理员将开关设为"是"即可启用。

### 第二步：功能开关生效

开启后，系统在构建订单支付渠道列表时会进行判断：

- **开关关闭**：收银台仅展示微信支付和微信扫码支付（根据医保支付开关和校验结果可能也展示医保支付）
- **开关开启**：收银台额外展示支付宝支付选项，默认不选中

### 第三步：配置支付宝商户参数

支付宝功能开关打开后，全局配置中出现"支付宝支付"配置分组（该分组依赖功能开关，关闭时不展示）：

| 配置项 | 说明 | 是否必填 |
|--------|------|---------|
| 应用ID | 支付宝商户AppId | 是 |
| 支付宝公钥 | 用于验签支付宝返回数据 | 是 |
| 商户私钥 | 用于签名商户请求数据 | 是 |
| 回调地址 | 支付结果通知地址 | 是 |

如果仅开启了功能开关但未填写商户参数，调用支付宝扫码预支付接口时会因缺少配置而失败。

### 第四步：收银台展示效果

功能开关开启后，用户在小程序收银台看到的效果：

1. 支付方式列表中出现"支付宝支付"选项
2. 用户选择支付宝支付后，系统调用预支付接口生成支付宝收款二维码
3. 用户跳转到支付宝教程页，按指引保存二维码并用支付宝App扫码支付

## 开关的依赖关系

支付宝支付功能开关是`settings.yaml`中支付宝商户配置的**前置依赖**：

```
支付宝支付功能开关（ykj.func.alipayEnabled）
  └── 支付宝支付（商户配置分组）
        ├── 应用ID（ykj.payment.aliPay.appId）
        ├── 支付宝公钥（ykj.payment.aliPay.alipayPublicKey）
        ├── 商户私钥（ykj.payment.aliPay.merchantPrivateKey）
        └── 回调地址（ykj.payment.aliPay.notifyUrl）
```

即：功能开关必须为"是"，才会展示下方的商户参数配置组。关闭开关时，商户配置被隐藏。

## 异常处理

| 情况 | 处理方式 |
|------|---------|
| 功能开关关闭 | 收银台不展示支付宝支付方式（后端判断） |
| 功能开关开启但商户参数未配置 | 用户选择支付宝支付后，预支付接口可能因缺少参数而失败，提示"支付失败，请联系客服或选择其他支付方式" |
| 二维码生成失败 | 前端提示"支付二维码生成失败，请稍后重试！" |
| 订单已支付（错误码40001） | 弹窗"该订单已支付成功，无需重复支付" |
| 订单已失效（错误码40002） | 弹窗"订单已取消关闭，未扣款，请重新下单/开方" |

## 关联功能

- **收银台统一支付**：支付宝支付选项的展示受此开关控制
- **全局配置管理**：功能开关与商户配置在统一的后台配置页面管理
- **微信支付配置**：与支付宝支付同属支付渠道配置体系
- **订单支付流程**：支付方式列表的构建依赖此开关

## 对比：微信支付与支付宝支付的配置差异

| 维度 | 微信支付 | 支付宝支付 |
|------|---------|-----------|
| 功能开关 | 无独立开关（始终启用） | 有独立功能开关 |
| 默认选中 | 是（第一个） | 否 |
| 商户配置 | 微信appId/商户号/API密钥/证书 | 应用ID/支付宝公钥/商户私钥 |
| 前端实现 | 拉起微信小程序支付收银台 | 生成二维码，用户用支付宝App扫码 |
