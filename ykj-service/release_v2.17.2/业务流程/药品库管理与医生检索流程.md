# 药品库管理与医生检索流程

> 文档版本 2026-07-02 | 面向读者：产品经理、运营、客服

## 概述

描述管理后台药品库（成药和中药两套体系）的信息管理、导入导出、库存扣减、以及医生端药品搜索的全链路后端逻辑。涵盖药品从新增到医生在问诊过程中搜索和使用的完整流程。

## 触发条件

- **后台管理员**：新增/编辑/删除/禁用药品、导入导出药品数据
- **定时任务**：第三方药房药品信息和图片的自动同步
- **医生端**：开处方时搜索中药或成药

## 步骤详解

### 第一步：药品新增（后台管理员）

管理员在后台选择药房后新增中药或成药：

**新增中药**：
1. 系统先到药品库查找该中药（按通用名匹配）。若药品库不存在该中药，拒绝新增提示"药品库不存在该药品"
2. 校验同一药房内不可重复添加相同通用名的中药（"药品已存在"）
3. 校验药品是否在系统黑名单中（"药品已被加入黑名单"）
4. 校验第三方药房的进货价和来源编码不能为空
5. 校验福建地区批准文号是否已填写（地区适配规则）
6. 默认库存设为"-1"（无限库存），自动生成拼音首字母
7. 新增成功后自动更新药房药品数量统计

**新增成药**：
1. 系统按药监ID到药品库匹配。若药品库不存在，拒绝新增
2. 校验同一药房内不可重复添加同一药监ID的成药
3. 其余与中药相同（黑名单、进货价、批准文号校验）

### 第二步：药品编辑

编辑药品时，以下字段不可修改（系统自动从药品库同步覆盖为空值）：
- **中药**：通用名、药品名称、剂型、药品单位
- **成药**：通用名、药品名称、剂型、药品分类、生产厂家、规格数量、用量单位、药品单位

编辑后重新校验第三方药房进货价、来源编码和批准文号。

### 第三步：Excel导入

管理员可选择导入中药或成药的Excel文件：

1. 系统逐行读取，每行采用"先新增、失败则转编辑"的策略：先尝试新增，若新增失败则尝试编辑更新
2. 拼音首字母自动生成（若导入时为空）
3. 最终返回成功列表和失败列表（含行号和失败原因）
4. 导入中会检查黑名单关键词（精麻毒等管制药品）

### 第四步：库存扣减

当处方下单扣减药品库存时：

1. **库存管理开关**：先检测该药房是否开启了库存管理，未开启则直接返回成功
2. **无限库存标记**：库存为"-1"表示无限库存，直接返回成功不扣减
3. **单位转换**：若药品库的转换比例不为1，需将扣减数量按比例换算
4. **库存不足**：当前库存 < 需扣减数量时，记录错误日志并返回失败（不抛异常，避免中断订单流程）

### 第五步：第三方药品同步

系统中存在三个定时任务负责与第三方药房同步：

1. **药品信息同步**：每7秒间隔拉取第三方药房的药品信息（名称、库存、规格等），按sourceCode匹配更新本地药品
2. **图片拉取**：扫描所有无图片但有sourceCode的药品，逐个从第三方拉取图片URL并上传到OSS
3. **库存同步**：每3秒拉取一次库存信息

### 第六步：医生端搜索药品

医生在开方时搜索中药：

1. **药房授权校验**：医生只能搜索已授权药房的药品。若医生未授权任何药房，提示"医生暂未授权任何药房, 请联系管理员"
2. **拼音搜索**：输入英文时，按Jaro-Winkler拼音相似度算法排序（相似度越高越靠前）
3. **中文搜索**：输入中文时，按名称文本相似度排序
4. **搜索结果**：若全量匹配到拼音完全一致的药品，再按名称二次排序，然后将这些药品排在最前面
5. **黑名单拦截**：医生搜索成药时，若搜索词命中黑名单关键词，直接提示"该药品为精麻毒等管制类药品，按规定不可线上开方"

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药品已存在（同一药房重复添加） | 提示"药品已存在"，拒绝新增 |
| 药品库无对应药品 | 提示"药品库不存在该药品"，拒绝新增 |
| 药品在黑名单中 | 提示"药品已被加入黑名单"，拒绝新增 |
| 第三方药房未填进货价 | 提示"第三方药房药品进货价不能为空" |
| 第三方药房未填来源编码 | 提示"第三方药房药品来源编码不能为空" |
| 福建地区批准文号未填 | 提示"批准文号不能为空" |
| 导入数据部分失败 | 返回成功/失败列表（含行号和原因），成功的已导入，失败的需修正后重导 |
| 医生未授权任何药房 | 提示"医生暂未授权任何药房, 请联系管理员" |
| 医生未授权指定药房 | 提示"您暂未授权该药房" |
| 搜索管制药品关键词 | 提示"该药品为精麻毒等管制类药品，按规定不可线上开方" |
| 药品库存不足 | 静默失败（记录日志），不影响订单创建流程 |
| 库存管理未开启 | 直接跳过库存扣减，不报错 |
| 同一药房存在多个同名中药 | 提示"药房里存在多个相同名称的药材, 请联系管理员" |

## 关联功能

- 药品信息管理流程（后台管理页面 → 此项的后台操作页面）
- 处方开具流程（医生搜索药品 → 添加到处方）
- 第三方药房对接（定时同步任务）
- 药品黑名单管理
