# 问卷模板创建与问题库保存流程

> 文档版本：2026-07-02 | 面向读者：医生、运营管理员

## 概述

本流程描述平台中医生创建问卷模板、编辑问卷、管理问卷状态的完整服务端逻辑。问卷可用于问诊时收集患者信息，医生可创建私有问卷或公用问卷，问卷题目可选择同步保存到问题库供复用。

## 触发条件

- **医生**在小程序「问卷管理」中创建或编辑问卷
- **管理员**在后台管理公用问卷
- **医生**启用/禁用某个问卷

## 步骤详解

### 一、问卷创建

1. **问卷编号自动生成**：编号格式为"WJ"+时间戳（精确到毫秒），如"WJ20260702153045123"，确保全局唯一。
2. **标题唯一性校验**：
   - 同一医生不能创建相同标题的私有问卷
   - 公用问卷之间也不能重名
   - 重名时提示"您当前已有相同标题的问卷"
3. **标题长度限制**：不超过300字，超过提示"问卷标题最大字数不能超过：300字"。
4. **副标题长度限制**：不超过300字（可选）。
5. **私有问卷自动清除**：如果问卷被设为非私有（isPrivate=false），系统自动将doctorId设为null，转为公用问卷。
6. **问卷题目处理**：
   - 保存前先删除该问卷下的所有旧题目（如果存在），再重新保存新题目
   - 题目至少1题，没有题目提示"问卷问题不能为空"
7. **问题库同步**：如果题目勾选了"保存到问题库"，系统尝试将该题目同步保存到公共问题库。同步失败不影响问卷本身的保存，仅在返回消息中提示"以下问题保存至问题库失败：xxx"。

### 二、题目校验

系统对每道题目进行严格校验：

1. **题目文字**：不能为空，否则提示"问卷题目名称不能为空"
2. **题目类型**：必须选择题目类型（单选/多选/问答题等），否则提示"问卷题目类型不能为空"
3. **题目序号**：必须有序号（seq），否则提示"问卷题目序号不能为空"
4. **选项校验**：
   - 问答题（EQ）和量表题（RBQ）可以不设选项
   - 单选和多选题必须至少有2个选项，不足2个提示"单选题选项不能低于2个"
   - 没有选项时提示"单选题答案选项不能为空"

### 三、问卷编辑

1. **存在性检查**：编辑时必须指定问卷ID，否则提示"参数错误，未指定要查询的问卷"。
2. **标题唯一性**：编辑时同样需要校验标题唯一性，但排除当前问卷自身（避免"自己和自己重名"）。
3. **所有权校验（小程序端）**：医生只可编辑自己创建的问卷，修改他人问卷时提示"您无法修改当前问卷"。
4. **全量替换策略**：编辑保存时，旧的题目和选项会被全部删除，替换为新提交的数据。

### 四、问卷列表查询

1. **私有问卷查询规则**：指定isPrivate=true时，查询该医生的私有问卷+所有公用问卷（医生既能看到自己的也能看到公用的）。
2. **公用问卷查询**：isPrivate=false时，只查询公用问卷。
3. **排序策略**：私有问卷优先显示（isPrivate升序），再按创建时间倒序。
4. **筛选支持**：编号模糊搜索、标题模糊搜索、启用状态筛选、创建时间范围筛选。
5. **题目数量统计**：每条问卷自动附带题目数量（questionCount），无需额外查询。

### 五、问卷启用/禁用

1. **即时切换**：changeUsableState方法即时修改enabled字段，不用审核。
2. **状态变更**：支持单独启用或禁用，通过Boolean参数控制。

### 六、问卷删除

1. **单条删除**：删除问卷→删除该问卷下所有题目→删除所有题目对应的答案选项。
2. **批量删除**：遍历每一条，逐一执行删除操作。任意一条失败不影响其他条（但没有try-catch保护，所以实际上是会中断的）。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 问卷标题为空 | 提示"问卷标题不能为空" |
| 问卷标题超过300字 | 提示"问卷标题最大字数不能超过：300字" |
| 同一医生创建同名问卷 | 提示"您当前已有相同标题的问卷" |
| 问卷题目为空 | 提示"问卷问题不能为空" |
| 题目类型未选择 | 提示"问卷题目类型不能为空" |
| 单选题选项不足2个 | 提示"单选题选项不能低于2个" |
| 医生编辑他人问卷 | 提示"您无法修改当前问卷" |
| 保存到问题库失败 | 不影响问卷保存，返回提示"以下问题保存至问题库失败" |

## 关联功能

- **问题库（QuestionLibrary）**：题目可同步保存到公共问题库，供其他问卷复用
- **问诊设置**：医生在问诊设置中关联问卷，问诊时自动发送给患者
- **问诊流程**：患者进入问诊前填写关联的问卷
