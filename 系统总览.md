# 源康健互联网医院 - 系统总览

> 最后更新：2026-06-29

---

## 系统简介

源康健互联网医院是一个在线医疗服务平台，连接患者、医生、药房三方，提供从预约挂号、在线问诊、电子处方到药品配送的完整医疗服务闭环。

## 系统组成

```
┌──────────────────────────────────────────────┐
│              源康健互联网医院                    │
├──────────────┬──────────────┬─────────────────┤
│  微信小程序    │   管理后台    │   服务端（待接入）  │
│  (患者+医生端) │  (运营+药房端) │                 │
└──────────────┴──────────────┴─────────────────┘
```

### 微信小程序
- **患者端**：预约挂号、在线问诊、电子处方、药品购买、健康管理
- **医生端**：在线接诊、处方开具、患者管理、收入查看
- **诊所端**：诊所管理、销售数据

技术栈：uni-app + Vue 3

### 管理后台
- **管理端 (mgmt)**：医院管理、病历管理、财务管理、运营管理、系统管理
- **药房端 (phar)**：处方审核、订单处理、库存管理

技术栈：Vue 3 + Element Plus + TypeScript

### 服务端（待接入）
提供 API 接口和业务逻辑处理，待获取代码权限后补充。

## 核心业务流程

1. **在线问诊流程**：患者预约 → 医生接诊 → 在线沟通 → 开具处方 → 患者支付 → 药房配药 → 物流配送
2. **药品购买流程**：患者搜索药品 → 下单 → 支付 → 药房处理 → 发货 → 收货
3. **医生入驻流程**：医生提交申请 → 管理员审核 → 通过后上线接诊
