# 云康健微信小程序 — 角色说明

> 文档版本：release/v2.17.2 | 面向读者：客服、运营、产品经理等非技术人员

---

## 角色概览

本小程序支持三种用户角色，同一微信账号可在不同角色之间切换：

| 角色 | 简称 | 典型用户 | 核心场景 |
|------|------|----------|----------|
| **患者** | 患者端 | 普通就医用户及其家庭成员 | 找医生、在线问诊、买药、管理健康 |
| **医生** | 医生端 | 平台注册的执业医师 | 接诊患者、开处方、写病历、管随访 |
| **商务/诊所** | 商务端 | 诊所或平台的商务管理人员 | 管理医生、查看销售数据 |

用户首次进入小程序默认为**游客模式**，可浏览医生列表和部分内容。完成微信授权登录后即可使用完整功能。医生和商务角色需要通过资质审核才能正常使用相关功能。

---

## 一、患者角色

### 患者能做什么

#### 找医生与问诊
- 浏览平台所有医生，按科室、地区、擅长领域搜索
- 查看医生个人主页（执业信息、擅长领域、患者评价）
- 发起**图文问诊**：通过文字、图片与医生在线聊天
- 发起**视频问诊**：预约时间后与医生视频通话
- 使用**快速问诊**：不指定医生，系统自动匹配
- 使用**AI预问诊**：由AI先收集病情，再转接真人医生
- 参与**群会诊**：多位医生同时为自己会诊

#### 健康管理
- 为自己和家人（就诊人）建立健康档案
- 填写和管理病史信息
- 进行**健康量表测评**，查看测评结果和趋势
- 浏览**医学百科**（药膳食疗、经络穴位、疾病知识等）
- 阅读健康科普资讯

#### 药品与支付
- 查看医生开具的**电子处方**和用药建议
- 在线购买处方药品，填写收货地址，查看物流
- 通过微信支付完成问诊费和药费支付
- 部分地区支持**医保支付**
- 申请电子发票

#### 预约与记录
- 预约医生的问诊服务，选择合适时间
- 查看和管理所有预约记录
- 查看历次问诊记录、会诊记录
- 对已完成的问诊进行**评价**

#### 其他
- 管理常用收货地址
- 查看积分
- 提交意见反馈
- 使用**黔康码**（贵州电子健康卡）

### 患者的典型使用路径

```
打开小程序 → 浏览首页/搜索医生 → 选择医生 → 查看医生详情 
→ 选择服务类型 → 填写病情 → 支付 → 等待接诊 → 在线问诊 
→ 查看处方/建议 → 购药 → 评价
```

## 二、医生角色

### 医生能做什么

#### 接诊与问诊
- 查看**待接诊列表**，选择接诊或拒绝患者
- 与患者进行图文聊天交流，使用常用语提高效率
- 与患者进行视频问诊
- 参与多医生联合视频会诊
- 查看问诊详情和历史记录

#### 开具处方
- 为患者**开具电子处方**（支持西药、中成药、中药饮片）
- 使用**处方模板**快速开方
- 通过**快速开方**功能简化开方流程
- 管理个人常用药房（添加常用药材和药品）
- 查看历史处方记录
- 对处方进行电子签名认证

#### 写病历
- 为患者编写电子病历
- 查看患者线下门诊病历
- 查看和修改已编写的病历

#### 患者管理
- 查看与自己建立医患关系的患者列表
- 查看患者详细信息和病史
- 通过二维码邀请新患者

#### 随访管理
- 创建和管理**随访计划**
- 创建可复用的**随访模板**
- 向患者推送患教资料、设置用药提醒、发布调查问卷
- 查看患者随访任务的完成进度
- 对随访进行总结

#### 收入与数据
- 查看个人**收益数据**（问诊收入、开方收入）
- 绑定银行卡用于提现
- 查看当天接诊数和开方数统计
- 查看平台通知消息

#### 个人设置
- 编辑执业信息和擅长领域
- 设置各类问诊服务的收费标准
- 设置出诊时间安排
- 配置电子签章

#### AI功能
- 创建和配置**AI医生分身**，让AI替代自己进行初步问诊
- 查看AI分身的问诊记录
- 使用AI辅助功能（提问建议、病历书写辅助、图片识别）

### 医生的典型使用路径

```
打开小程序 → 切换到医生角色 → 查看待接诊列表 → 接诊患者 
→ 图文/视频问诊 → 填写病历 → 开具处方 → 发起随访 → 跟踪患者康复
```

## 三、商务/诊所角色

### 商务能做什么

#### 医生管理
- 查看所邀请的医生列表及状态
- 通过二维码或链接邀请新医生加入平台

#### 销售数据
- 查看药品销售的**月度统计数据**
- 查看药品销售的**年度统计数据**
- 了解医生开方量、药品销量等经营数据

#### 个人管理
- 编辑个人基本资料
- 查看商务资质审核状态

### 商务的典型使用路径

```
打开小程序 → 切换到商务角色 → 查看销售数据 → 管理医生 → 邀请新医生
```

## 四、角色切换说明

同一微信账号如果同时拥有多个角色（如某用户既是患者也是医生），可在小程序内进行角色切换：

- **切换入口**：个人中心或首页提供角色切换功能
- **切换方式**：点击切换按钮，选择目标角色后自动跳转到对应角色的首页
- **游客模式**：未登录或未认证状态下可使用部分浏览功能
