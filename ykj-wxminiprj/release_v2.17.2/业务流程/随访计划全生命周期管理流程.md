# 随访计划全生命周期管理流程

> 文档版本：2026-07-01 | 面向读者：医生

## 概述

医生为自己的患者创建个性化的随访计划（含多个时间节点的患教/提醒/问卷），系统按时自动发送，医生可随时查看进度、取消计划或填写随访总结。

## 触发条件

- 医生在问诊结束后，从患者聊天室或工具栏进入"发起随访"
- 医生从"我的患者"列表中选择就诊过的患者发起随访
- 从随访模板中选择预设方案快速创建

## 步骤详解

### 第一步：选择随访患者

进入"发起随访"页面后看到：
- 顶部搜索框，可按姓名搜索患者
- 患者标签区：显示所有已有的患者标签，点击标签可筛选患者（支持多选+重置），最多显示两行标签，超出部分点击"更多"弹窗查看
- 患者列表：展示头像、姓名、性别年龄、已有的随访计划标签
- 点击患者前会先验证该患者是否有过就诊记录——没有就诊过的患者不可发起随访，弹出提示："随访暂只支持对您就诊过的患者发起"
- 每页10条，滚动到底自动加载更多

### 第二步：选择随访方案

验证通过后进入方案选择页：
- **新建空白方案**：从零开始配置
- **从模板创建**：选择一个已有的随访模板，自动填充计划名称和内容
- 如果患者已有同名进行中的随访计划，弹窗提示"患者已有该计划的随访"，可选择取消或继续创建

### 第三步：配置随访计划

进入计划创建/编辑页：

**① 计划名称**（必填）
- 输入随访计划名称，如"高血压术后随访"

**② 就诊时间**（必填）
- 选择患者的就诊日期，同时展示患者最近的就诊时间列表供参考

**③ 添加计划节点**
- 每个节点包含：
  - **发送时间**：相对于就诊时间的间隔（如"1周后"）
  - **发送内容**：患教资料、提醒文字、问卷（三种类型可混合）
- 第一个计划节点自动添加，后续节点可点击"添加计划"新增
- 除第一个节点外，每个节点右上角有"删除计划"按钮
- 点击"添加发送内容"进入内容选择页

### 第四步：选择随访内容

内容选择页有三个选项卡：

**患教资料**：
- 列表展示已发布的患教文章标题，支持搜索
- 点击标题可预览文章详情
- 点击右侧勾选框选中/取消

**提醒**：
- 列表展示提醒内容，支持搜索
- 点击勾选时弹出文本框（只读模式），可查看内容
- 也可直接在输入框中编辑提醒文字（最长100字），保存后自动选中

**问卷**：
- 列表展示已发布的问卷标题，支持搜索
- 点击标题可预览问卷详情
- 点击右侧勾选框选中/取消

每页10条，三种类型各独立分页。底部显示已选内容数量和"已选内容"入口（可查看/删除已选项）。点击"确认选择"返回计划页。

### 第五步：提交计划

点击底部"执行计划"按钮前，系统校验：
- 计划名称不能为空
- 就诊时间不能为空
- 每个计划节点的随访日期不能过期
- 每个计划节点至少有一种内容

通过后弹出"执行计划成功"提示，3秒后自动跳转到计划详情页。

### 第六步：查看与操作计划详情

计划详情页展示：
- **患者信息**：头像、姓名、年龄、性别
- **计划名称**和**状态标签**（进行中/已完成/已取消）
- **就诊时间**（只读）
- **计划节点列表**：每个节点显示间隔时间、发送时间、状态（待发送/已发送）、包含的内容项（点击可查看详情）

操作按钮根据状态变化：
- **进行中**：显示"取消随访"按钮 → 点击弹出取消原因输入框（必填）→ 确认后状态变为"已取消"，按钮变为"查看取消原因"
- **已完成**：
  - 如果已填写总结 → 显示"查看随访总结"
  - 如果未填写总结 → 显示"填写随访总结"（输入框，必填）
- **已取消**且医生角色：显示"查看取消原因"

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 患者未就诊过 | 弹窗提示"随访暂只支持对您就诊过的患者发起，该患者未就诊过，不可发起随访" |
| 计划名称为空 | 提示"请输入计划名称" |
| 未选就诊时间 | 提示"请选择就诊时间" |
| 随访日期已过 | 提示"计划{N}的随访日期已过，请重新选择" |
| 节点无内容 | 提示"请添加发送内容" |
| 取消原因未填 | 提示"请填写取消原因" |
| 随访总结未填 | 提示"请填写随访总结" |
| 模板中内容被禁用 | 已禁用的内容不展示在计划节点中（但模板数据仍保留） |

## 关联功能

- 从"我的患者"进入发起随访（通过患者标签筛选）
- 从模板快速创建 → 跳转到模板选择页
- 计划节点中的内容跳转到对应的内容详情页（患教/提醒弹窗/问卷详情）
- 计划详情中的已存在计划可跳转查看
