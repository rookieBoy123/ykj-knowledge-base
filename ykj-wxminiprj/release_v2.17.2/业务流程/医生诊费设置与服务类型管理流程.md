# 医生诊费设置与服务类型管理流程

> 文档版本 v1.0 | 面向读者：医生

## 概述
医生可以在小程序端自主配置自己提供的问诊服务类型和价格。支持的服务类型包括图文问诊、视频问诊、咨询服务、快速续方、会诊服务、医生复诊等。医生可以随时开启或关闭某种服务，并调整对应的诊费和时长。

## 触发条件
- **前置条件**：已认证的医生账号
- **入口路径**：医生端 → 我的 → 诊费设置

## 步骤详解

### 第一步：查看服务列表
进入页面后展示医生所有可提供的服务类型，每类服务显示为一个卡片：
| 服务类型 | 说明 |
|---------|------|
| 图文问诊 | 可接诊和开处方，可设置问诊价格和时长 |
| 视频问诊 | 可接诊和开处方，可设置问诊价格和时长 |
| 咨询服务 | 仅做咨询不开处方，可设置咨询价格和时长 |
| 快速续方 | 患者可通过历史处方快速续方，可设置价格 |
| 会诊服务 | 可设置会诊价格 |
| 医生复诊 | 可设置复诊价格 |

每种服务类型卡片右上角有一个**开关按钮**，可随时开启或关闭该服务。

### 第二步：开启/关闭服务
- 拨动服务开关，绿色表示开启，灰色表示关闭
- 关闭后，患者端不再显示该医生的此项服务
- 开启后可进一步设置价格和时长

### 第三步：设置问诊价格
- 点击服务卡片中的价格区域（如"50元"），弹出价格选择器
- 从系统预设的价格档位中选择合适的金额
- 图文问诊和视频问诊的服务类型标签显示"可接诊和开处方"
- 咨询服务的标签显示"仅做咨询不开处方"

### 第四步：设置问诊/咨询时长
- 点击服务卡片中的时长区域（如"24小时"或"请选择"）
- 弹出时长选择器，从预设的时长档位中选择
- 时长单位可能是"小时"或"分钟"
- 未设置时长时显示"请选择"文字

### 第五步：特殊服务说明
- **咨询服务**底部有温馨提示："仅可以做咨询服务，不可以开处方"及最小咨询时长提示
- **快速续方**底部有说明："开通快速续方后，患者可通过历史处方快速续方；若关闭，则匹配新的医生完成续方申请"
- **会诊服务和复诊服务**价格单位为"元/次"

## 异常处理
| 情况 | 系统处理方式 |
|------|-------------|
| 关闭所有服务 | 患者端该医生页面不显示任何服务入口 |
| 未设置时长就开启服务 | 时长区域显示"请选择"，患者端可能无法正常下单 |
| 价格选择器无数据 | 显示空白选择器，无法选择价格 |
| 开关切换网络失败 | 开关可能回弹到之前状态 |
| 服务列表加载失败 | 页面显示空白 |

## 关联功能
- **患者端医生详情页**：患者看到的服务类型和价格取决于此处的配置
- **问诊下单**：患者下单时价格和时长从医生设置中读取
- **收益管理**：医生收益与定价直接相关
