# 量表测评答题与结果查看流程

> 文档版本 2026-07-02 | 面向读者：患者、医生

## 概述

患者或医生可以通过量表测评对自己的健康状况进行量化评估。系统提供多种类型的量表（如焦虑自评、抑郁筛查等），用户逐题作答后自动计算得分并展示测评结果，包括总分、雷达图和评测解读。

## 触发条件

- **前置条件**：已有就诊人信息（患者ID），已选择某个量表并进入答题
- **入口路径**：
  - 患者端：量表列表 → 量表详情 → 开始测评
  - 医生端：患者病历 → 量表记录 → 选择量表开始测评

## 步骤详解

### 第一步：进入答题页面

用户进入答题页面后看到：
- 顶部显示"测评答题"标题
- 进度条显示当前答题进度（如"第3题/共10题"）
- 题目内容区：题目类型标签（单选题/多选题）+ 题目文字
- 选项列表：每个选项前有序号（如"1、完全没有"），点击选中后选项高亮并显示勾选图标
- 底部按钮区：上一题（非第一题时显示）、下一题（非最后一题时显示）、提交（最后一题时显示）

### 第二步：逐题作答

**单选题**（题目类型 0）：
- 点击选项后自动提交答案，500毫秒延迟后自动跳转下一题
- 如果已答过该题（修改答案），点击新选项后自动覆盖旧答案并跳转下一题
- 最后一题时点击选项只更新选中状态，不会自动跳转，需点击"提交"按钮

**多选题**（题目类型 1）：
- 点击选项进行切换（选中/取消），支持同时选择多个选项
- 至少选择一项后"下一题"按钮才可点击（否则灰色禁用）
- 点击"下一题"后才提交答案并跳转
- 最后一题时需点击"提交"按钮完成整个测评

**上一题功能**：
- 回退到已答题目，已选答案会回显，用户可以修改答案

### 第三步：提交测评

当所有题目答完后，点击"提交"按钮：
1. 系统先检查是否至少选择了一个答案，如果未选择提示"请答完题目再提交"
2. 提交最后一题答案到服务器
3. 后端校验：单选题不能有多个答案、答案必须在题目选项中、测评已完成不能再修改
4. 如果是最后一题，后端自动生成测评结果（计算总分、维度分）
5. 提交成功后自动跳转到测评结果页

### 第四步：查看测评结果

测评结果页展示：
- **圆弧进度条**：展示总分相对于量表总分的百分比，中心显示"XX分"
- **结果等级**：如"轻度焦虑"，位于进度条下方居中展示
- **雷达图**：当量表有3个及以上维度时展示，直观显示各维度得分对比（如"躯体化"、"焦虑"、"抑郁"等维度，维度名超10字自动截断）
- **评测解读**：详细解释该分数所代表的意义和临床建议
- **免责提示**："该测试结果仅作为评估工具，不能以此作为诊断。若对评测结果有疑问，可咨询医生。"
- **测评人信息**：姓名、性别、年龄、测评时间
- **问诊信息**（如有关联问诊）：问诊医生、科室、问诊时间
- **问诊对话记录**（如有关联问诊）：点击可跳转到IM聊天记录

### 第五步：后续操作

- **查看题目明细**：点击"查看题目明细"可查看每道题的具体作答情况
- **咨询医生**：如果当前是患者角色且未关联问诊，底部显示"咨询医生"按钮，点击跳转到找医生页面，携带患者ID和测评记录ID

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 进入页面缺少必要参数（recordId/currentQuestionId/scaleId/patientId） | 提示"缺少必要参数"，页面无数据 |
| 未选择任何答案点击下一题 | 多选题：下一题按钮灰色不可点；单选题：自动选中并跳转 |
| 提交时未选择答案 | 弹窗提示"请答完题目再提交" |
| 单选题选择了多个答案（后端校验） | 后端返回错误"单选题不可以有多个答案！" |
| 提交的答案不在题目选项中（后端校验） | 后端返回错误"该答案不存在于问题选项中！" |
| 测评已完成再次尝试修改答案（后端校验） | 后端返回错误"已测评完成，不能再次修改答案！" |
| 网络异常导致答案提交失败 | 答案未保存，用户需重新答题 |

## 关联功能

- 跳转到**测评结果页**：提交完成后自动跳转（redirectTo）
- 跳转到**题目明细页**：testResultDetail/index（查看每道题详细作答情况）
- 跳转到**找医生页**：specialsearchdoctor/index（咨询医生，携带患者ID和测评记录ID）
- 跳转到**IM聊天页**：timIM/pages/TUI-Chat/chat（查看问诊对话记录）
