# 处方详情双角色查看与操作流程

> 文档版本 2026-07-03 | 面向读者：产品经理、运营、客服

## 概述
这是一份「用药建议/处方」在医生端和患者端的统一详情页，根据当前登录角色（医生/患者）展示不同的操作按钮和信息。支持成药处方和中药饮片两种类型，覆盖从处方生成、审核签字、支付领取到订单查看的完整环节。

## 触发条件

**医生端入口**：
- 从问诊室中点击"开方"后提交的处方
- 从「处方记录」列表点击某条处方记录
- 从快速续方成功后跳转

**患者端入口**：
- 收到医生分享的处方消息（微信订阅消息/分享卡片）
- 从「我的处方」列表点击某条处方
- 从快速开方领取页面跳转

## 步骤详解

### 第一步：页面加载
用户进入处方详情页，系统自动：
- 根据处方单号查询处方完整信息（药品清单、诊断、患者信息、审核状态、支付状态）
- 如果医生刚提交处方且状态为"审核中"，系统会获取CA电子签名的唯一标识
- 识别当前是医生还是患者身份，决定后续展示的按钮和操作
- 如果是快速开方类型，查询关联的多个处方（顶部多标签切换）

### 第二步：查看处方基本信息
页面展示以下信息区域：

**基础信息区**：
- 处方单号、开具日期
- 患者姓名、性别、年龄
- 如果已支付，显示"已支付"标记图

**诊断信息区**：
- 西医诊断（如：高血压）
- 中医辨病（如：眩晕）
- 中医辨证（如：肝阳上亢）

**药品信息区**：
- 成药处方：药品名称、规格、每次用量、用药途径、用药频次、数量
- 中药饮片：药材名称、用量、加工工艺（如代煎/自煎）、剂数、用药方法
- 患者未支付时，中药处方的药品明细可能被隐藏（由系统配置控制）

**医生/药师信息**：
- 开具医生姓名
- 审核药师姓名（审核通过后显示）

**底部说明**：
- 展示处方注意事项（如"处方当日有效，请及时支付"）

### 第三步：查看电子处方
如果处方已审核通过且未过期：
- 点击"查看电子处方"可打开电子处方PDF文件
- 患者端：中药处方需先支付后才能查看电子处方

### 第四步：患者端操作

**情况A：待支付处方**
- 底部显示"付款"按钮
- 点击后进入确认处方页面，可选择配送地址、支付方式
- 如果是对外合作机构的处方，可能不显示付款按钮（由服务端判断）

**情况B：医生代付待领取**
- 底部显示"确认领取处方并支付"按钮
- 点击后绑定就诊人信息，完成处方归属确认
- 如果该就诊人已被其他手机号绑定，系统提示解绑

**情况C：已支付**
- 底部显示"订单详情"按钮，可跳转查看物流配送进度

**情况D：处方已失效**
- 弹窗提示"处方单已失效，请联系医生重新为您开具处方"
- 按钮变为不可操作状态

### 第五步：医生端操作

**情况A：待审核处方（状态=20）且问诊未结束**
- 底部显示"编辑"按钮
- 点击可跳转到处方编辑页面，修改药品、诊断等信息
- 问诊结束后（超过24小时），编辑按钮消失

**情况B：审核中（状态=25）且有CA签名需求**
- 底部显示"签字"按钮
- 点击调起电子签名（支持医网信或放心签两种第三方平台）
- 签名成功后自动跳转到成功页面

**情况C：审核通过/待支付**
- 底部显示"发到患者微信"按钮——通过微信分享卡片发送给患者
- 底部显示"引用处方"按钮——将当前处方内容复用到新处方（商城问诊和机构处方不可复用）
- 底部显示"代支付"按钮——医生帮患者垫付药费
- 底部显示"指定处方接收人"按钮——将处方指定给特定患者

**情况D：处方关联了问诊**
- 底部显示"返回问诊室"按钮，可快速回到医生工作台

### 第六步：快速续方入口
如果当前处方是从快速续方进入的：
- 页面底部显示"快速续方"按钮
- 点击前会检查医生是否有该药房的授权
- 无授权提示"当前医生没有该药房授权，请联系管理员"

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 获取CA签字ID失败 | 弹窗提示"获取处方签字ID失败，请联系管理员"，自动返回上一页 |
| 处方已失效（审核不通过/已过期） | 按钮隐藏，药品信息不可见，弹窗提示"处方单已失效" |
| 处方已被撤回 | 患者端药品信息隐藏（显示"撤回审核中"标记） |
| 就诊人被其他手机号绑定 | 弹窗提示"同一就诊人只能绑定在一个手机号"，提供"去解绑"入口 |
| 患者未实名 | 提示"该处方患者未实名，不可指定" |
| 问诊单号缺失 | 提示"问诊单号缺失" |
| 处方过期未支付 | 药品信息不展示，底部按钮消失 |
| 中药处方患者未支付 | 根据系统配置可能隐藏药品明细（"未支付不可查看"或"始终不可查看"） |
| 药房授权不足 | 提示"当前医生没有该药房授权，请联系管理员" |
| 签字回调超时（10秒） | 显示等待状态，如果超时则提示"签字失败，请重新签字" |

## 状态流转

### 处方审核状态
```
待处方医生审核(20) → 待药师审核(20) → 审核中(25) → 审核通过(30) 
                                                 → 审核不通过(40)
```

### 处方整体状态
```
待生成(10) → 待审核(20) → 审核中(25) → 审核通过(30) → 完成
                                    → 审核不通过(40) → 已失效
                                    → 已失效(50)
```

### 订单状态（关联）
```
待付款(10) → 待发货(20) → 已发货待签收(30) → 已签收待反馈(40) → 正常完成(60)
         → 已取消(70) → 已失效(80) → 已退款(90) → 待退款(50)
```

## 关联功能
- 处方编辑（addOrEditRp页面）——编辑按钮跳转
- 药品下单（drugOrder页面）——付款按钮跳转
- 订单详情（orderDetail页面）——已支付时跳转
- 快速续方（quickPrescribe页面）——续方按钮跳转
- 问诊室（医生工作台）——返回问诊室按钮跳转
- 电子处方PDF查看——系统内置PDF阅读器
- 会员绑定/解绑——处方领取流程
