# 商务销售数据查看流程

> 文档版本 1.0 | 面向角色：商务运营人员

## 概述

商务人员通过小程序查看所邀请医生的药品销售情况，支持按月、按年维度汇总，并可下钻至单个医生的详细销售数据。

## 触发条件

- **前置条件**：使用商务角色账号登录
- **入口路径**：商务工作台 → 月销售数据 / 年销售数据 / 近期邀请

## 页面结构

商务销售数据查看由三个页面组成，层层递进：

```
月销售数据（sales/index）
  ├── 年销售数据（sales/yearindex）
  └── 医生个人销售详情（salesData/index）
```

近期邀请（invitation/index）也可跳转至医生个人销售详情。

---

## 步骤详解

### 第一步：查看月销售数据

**入口**：商务工作台 → 月销售数据

用户看到什么：

- 顶部显示当前选中月份（如"2026年 7月"），点击可切换月份
- 下方显示该月药品总销量（如"销售128"），表示该月共卖出128个品种的药品
- 已邀请医生的列表，每位医生显示：头像、姓名、职称、科室、医院，以及本月销量数字
- 点击任一医生可查看该医生的详细销售数据
- 如果当月没有邀请记录，显示"暂无邀请记录"空状态

用户能做什么：

- 切换月份查看不同时间段的数据
- 点击医生进入个人详情页
- 向下滚动加载更多医生

数据范围：选中的整月（月初第一天 00:00:00 到月末最后一天 23:59:59）

---

### 第二步：查看年销售数据

**入口**：月销售数据页面右上角"年"Tab

用户看到什么：

- 顶部显示当前选中年份（如"2026年"），点击可切换年份
- 下方显示该年度药品总销量
- 与月销售数据相同的医生列表布局，但数据范围变为整年（1月1日到12月31日）

与月销售数据的区别：
| 对比维度 | 月销售数据 | 年销售数据 |
|---------|-----------|-----------|
| 时间粒度 | 按月 | 按年 |
| 数据范围 | 月初到月末 | 年初到年末 |
| 切换控件 | 月份选择器 | 年份选择器 |
| 页面标题 | "月销售数据" | "年销售数据" |

---

### 第三步：查看医生个人销售详情

**入口**：在月/年销售数据或近期邀请中点击某一医生

用户看到什么：

- 页面顶部带渐变背景，展示医生基本信息（头像、姓名、职称、科室、医院、是否三甲）
- 三个统计数字：总订单量、总销售额（元）、销售药品数（品种）
- 三个时间Tab：今日、近7日、近30日
- 药品销售明细表格：药品名称 + 数量
- 表格数据分页加载，加载完显示"- 已显示全部 -"

用户能做什么：

- 切换"今日/近7日/近30日"查看不同时间段的药品销量
- 表格顶部显示"订单量"和当前日期
- 向下滚动加载更多药品记录

---

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 当月/当年无邀请记录 | 显示空状态插图 + "暂无邀请记录" |
| 医生无销售额数据 | 显示"0"或"暂无" |
| 医生无头像 | 显示默认头像图片 |
| 医生无职称/科室/医院 | 显示"暂无" |
| 列表加载到底 | 显示"- 已显示全部 -" |
| 个人详情中药品列表为空 | 显示"- 已显示全部 -"，无loading状态 |

---

## 关联功能

- **近期邀请**：查看已邀请的医生列表，点击医生同样跳转个人销售详情
- **商务工作台**：商务角色的首页入口
- **医生管理**：管理后台可查看更详细的医生销售统计

## 数据来源说明

商务端能看到哪些医生的数据，取决于该商务人员通过邀请码/分享链接成功邀请注册的医生范围。未通过该商务邀请的医生不在列表中。
