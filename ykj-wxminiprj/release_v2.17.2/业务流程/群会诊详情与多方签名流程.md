# 群会诊详情与多方签名流程

> 文档版本 2026-07-02 | 面向产品运营、客服

## 概述

群会诊详情页展示会诊的完整信息，包括会诊状态进度、时间安排、患者信息、参会医生列表、费用信息和各方签章。医生和患者在此页面完成会诊相关的签名操作，最终推进会诊流程。

## 触发条件

- 医生或患者从会诊列表/通知进入
- 前置条件：拥有会诊编号（consultationNo）
- 入口路径：会诊列表 → 会诊详情页

## 步骤详解

### 第一步：查看会诊状态进度

页面顶部展示会诊状态进度条（仅会诊开始前且角色为医生时显示），直观展示当前会诊所处阶段。

如果是已终止的会诊（flowStatus > 5），显示审核状态说明。

### 第二步：查看会诊时间

会诊未结束时（flowStatus < 6），显示会诊时间信息组件。

### 第三步：查看患者信息和会诊信息

依次展示：
- **患者信息**：来自患者信息组件
- **会诊信息**：会诊基本资料
- **会诊医生列表**（flowStatus > 2 时显示）：所有参会医生的信息

### 第四步：支付会诊费用（患者端）

当角色为患者且会诊状态为「待收费」（flowStatus=3）时，显示会诊费用卡片，展示应支付的会诊金额。

### 第五步：患者签名（患者端）

会诊状态为「待收费」且患者尚未签名时，显示签名入口：
- 左侧文字"患者签名"
- 右侧箭头 + "点击签名"文字

点击跳转手写签名板页面，签名完成后返回，页面自动更新签名状态。

### 第六步：医生签名（医生端）

医生通过底部操作按钮触发签名流程。

**签名前置检查 - CA电子签章流程**：

系统先检查是否开启了CA签章功能：
- **开启CA**：走CA电子签章流程
- **关闭CA**：走手写签名流程

**CA签章流程的分支**：

1. **查询CA用户状态**：先获取医生的CA账号信息
   - 无CA账号 → 弹窗提示"未完成CA电子签章"，可点击「确定」进行账号同步
   - 账号同步后自动重新触发签名

2. **CA认证状态判断**：
   - **未认证（authStatus=1）**：弹出CA账号同步弹窗，点击确认同步资料
   - **认证中（authStatus=2）**：提示认证中，引导去完成认证
   - **认证失败（authStatus=4）**：提示认证失败，可重新认证
   - **认证成功（authStatus=3）**：根据平台类型分发签名

3. **签名平台分发**：
   - **医网信（platform=1）或源康健PaaS（platform=3）**：检查本地环境，有则调起医网信签名组件，否则弹出提示引导
   - **放心签（platform=4）**：检查签名图片缓存，有则直接使用，否则弹出签名弹窗
   - **其他平台**：提示"暂不支持"

4. **手写签名**（CA未开启时）：直接跳转手写签名板

### 第七步：签名结果处理

**CA签名成功后**：
- 页面显示loading"查询结果中"，开始轮询查询签名结果
- 每1.5秒查询一次，最多查询7次（约10.5秒）
- 查到签名结果 → 页面自动刷新
- 7次未查到 → 强制标记已签名，防止重复点击

**手写签名成功后**：
- 如果是审核医生扫码签名：弹窗"签名成功"→ 点击「返回首页」跳转医生首页
- 如果是主诊医生签名：提示"签名成功"→ 页面自动刷新

**签名失败**：
- 放心签取消：关闭弹窗，不做处理
- 放心签错误：提示"签字失败，请重试"
- 手写签名失败：调用签名失败处理函数

### 第八步：会诊被移出处理

当医生被移出视频会诊室时，弹出提示："您已被请出视频会诊室请等待会诊结果"，提供「关闭」按钮确认。

## 异常处理

| 情况 | 系统处理方式 |
|-----|------------|
| CA未同步 | 弹窗提示，可点击确认同步账号资料 |
| CA认证中 | 弹窗提示认证状态，引导完成认证 |
| CA认证失败 | 弹窗提示，可重新认证 |
| CA签名轮询超时 | 强制标记签名完成，防止重复签名 |
| 被移出会诊室 | 弹窗提示"您已被请出视频会诊室" |
| 签名提交失败 | 弹窗提示错误信息 |
| 签名环境未安装 | 弹出引导弹窗，提示需先完成CA签章 |
| 不支持的CA平台 | 提示"暂不支持" |

## 关联功能

- 会诊列表 → 会诊详情页
- 会诊详情页 → 手写签名板
- 会诊详情页 → CA签名组件
- 会诊详情页 → 医生首页（审核医生签名后跳转）
