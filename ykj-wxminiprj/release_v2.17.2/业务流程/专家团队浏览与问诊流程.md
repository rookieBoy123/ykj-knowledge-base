# 专家团队浏览与问诊流程

> 文档版本 2026-07-03 | 面向角色：患者

## 概述

专家团队是由一位首席专家和多位数名医生组成的专业诊疗团队。患者可以浏览团队详情，了解服务流程和团队成员，最后选择一位团队医生发起问诊。患者不能直接选择首席专家问诊，需由团队医生初诊后判断是否需要预约首席专家。

## 触发条件

- **前置条件**：系统开启了专家团队功能
- **入口路径**：患者端首页 → 专家团队入口 → 专家团队列表

## 步骤详解

### 第一步：浏览专家团队列表

列表页展示所有已启用（isShow=1）的专家团队，每个团队显示：
- 团队头像（若无则显示默认头像）
- 团队名称
- 团队简介（最多三行，超出省略）

支持通过顶部搜索框输入团队名称关键字进行搜索。

### 第二步：进入团队详情页

点击团队卡片进入详情页，包含以下信息区域：

**团队介绍**：显示团队名称、头像和简介。简介超过三行时可点击"展开详细"查看完整内容。

**首席专家**：显示首席专家的头像、姓名、职称，以及一句话简介。点击"去【医生名】医生主页查看更多信息"可进入该首席专家的医生主页。

**服务流程**：
1. 第一步：选择团队医生 → 可选择团队医生发起问诊
2. 第二步：团队医生首诊 → 经团队医生判断病情，如需进一步看诊，团队医生将为您预约首席专家线上/线下的号源
3. 第三步：首席专家看诊

**团队成员**：列出所有团队医生，点击任一医生可跳转至该医生的服务选择页面。

**服务内容**：以富文本形式展示团队可提供的服务内容。

### 第三步：选择医生发起问诊

点击底部"去问诊"按钮，弹出团队医生选择面板（底部弹出），选择一位医生后跳转到医生的服务页面，后续走标准问诊流程。

**温馨提示**：
1. 患者选择专家团队问诊时，目前仅支持选择非首席专家的团队医生发起问诊
2. 医生接诊后，会根据患者病情需要再确认是否为患者预约线下首席专家门诊号
3. 若患者病情与本团队简介不符，医生团队有权拒诊

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 团队暂无成员 | "去问诊"按钮显示为"去问诊（暂无团队成员）"且不可点击，按钮颜色变灰 |
| 无专家团队 | 列表显示空状态"暂无专家团队" |
| 搜索无结果 | 列表区域显示空 |
| 医生无简介 | 显示"医生暂无填写" |
