# 医生问卷创建与管理全流程

> 文档版本 | 面向读者：医生

## 概述

医生可以创建、编辑、管理自己的问诊问卷。问卷包含四种题型（单选、多选、问答、图片），可以从平台题库中复用题目，也可以关联到问诊设置中，让患者在新创建问诊时自动收到问卷。

## 触发条件

- **前置条件**：医生已登录小程序医生端
- **入口路径**：医生端 → 问诊设置页 → "全部问卷"区域 → 点击问卷或底部操作按钮

## 步骤详解

### 第一部分：问卷管理主页（问诊设置页）

页面分为上下两个区域：

**1. 问诊问卷开关与关联设置**
- 顶部开关："患者创建问诊时填写问诊问卷"，开启/关闭自动发送问卷功能
- 开关下方展示多种问诊类型（如初诊、复诊、咨询等），每种类型需分别选择绑定一份问卷
- 点击任意类型的行选择区域（蓝色字体"请选择"或已选问卷名称），弹出问卷选择器
- 选择器中可搜索和浏览"我的问卷"，选中后该类型即关联该问卷

**2. 全部问卷列表**
- 展示当前医生的所有"我的问卷"（私人问卷）和"公用问卷"
- 每条问卷显示：图标、名称（两行截断）、"我的问卷"(橙色标签)或"公用问卷"(蓝色标签)、题目总数
- **我的问卷**支持左滑删除（红色"删除"按钮），公用问卷不可删除
- 点击任意问卷进入问卷详情页
- 列表支持滚动分页加载（每页10条），滑到底部自动加载更多

**3. 底部操作按钮**
- **从已有问卷创建**（青绿色）：弹出问卷选择器，选中一份已有问卷后跳转到创建页，自动带入其所有题目，标题自动追加"--副本"
- **创建新问卷**（蓝色）：直接跳转到空白创建页

### 第二部分：创建/编辑问卷

**问卷标题与说明**
- 问卷名称（必填）：最多20字，大号加粗字体输入框
- 问卷说明（选填）：最多300字，用于描述问卷用途（如"流感专用问卷"）

**题目列表**
- 每个题目显示：题型标签（单选蓝/多选黄/问答绿/图片红，四种颜色）、题号、题目文字、是否必填标记
- 每道题右上角有"⋮"操作菜单，支持：
  - **编辑**：跳转到编辑题目页，修改题目内容、选项等
  - **删除**：从问卷中移除该题，后续题目序号自动重排
  - **置顶**：将该题移动到问卷第1题，其余题目序号顺延
  - **上移/下移**：与相邻题目交换位置
- 题目预览效果：单选/多选题展示答案选项框，问答题展示备注文字，图片题展示上传图标+示例图片区域

**添加题目的两种方式**
- **"新建问题"**（蓝色）：跳转到创建题目页，可选择单选/多选/问答/图片四种类型新建一道题，保存后自动加入问卷末尾
- **"从题库选择"**（青绿色）：跳转到平台共享问题库，可多选题目批量添加到问卷

**底部按钮**
- **"保存问卷"**：提交校验后保存。按钮根据表单完整性自动启用/禁用：
  - 问卷名称非空 + 至少有一道题 → 蓝色可点击
  - 任一条件不满足 → 灰色禁用

### 第三部分：问卷提交与校验

点击"保存问卷"时，系统执行以下检查：

1. **重名检查**：如果问卷中存在相同文字的题目，弹出Toast提示"问卷中存在相同的题目"，阻止保存
2. **提交数据**：
   - **创建模式**：调用创建接口，数据包含标题、说明、题目列表，且标记为`isPrivate: true`（我的问卷）、`enabled: true`（启用）
   - **编辑模式**：调用编辑接口，更新问卷信息
3. **成功反馈**：弹出"创建成功"或"编辑成功"的绿色勾提示，1.5秒后自动返回上一页

### 第四部分：问卷详情查看

从列表点击进入查看问卷详情：

- 展示完整的问卷内容：标题、说明（虚线分隔）、所有题目（带题型标签和序号）
- 图片题的示例图片可点击大图预览
- **底部按钮差异化**：
  - 来自"我的问卷"：底部显示蓝色"编辑"按钮，点击进入编辑模式
  - 来自"公用问卷"：底部显示青绿色"创建为我的问卷"按钮，点击可基于此公用问卷创建一份副本作为自己的问卷

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 问卷名称未填写或题目列表为空 | "保存问卷"按钮灰色禁用 |
| 问卷中存在相同文字内容的题目 | 弹出Toast"问卷中存在相同的题目"，阻止保存 |
| 问诊类型未选择关联问卷 | 显示灰色"请选择"文字，该类型不会在患者创建问诊时发送问卷 |
| 删除我的问卷 | 弹出确认弹窗"删除后数据将不可恢复"，确认后删除并从列表中移除 |
| 删除公用问卷 | 左滑操作禁用（`disabled`），不可删除公用问卷 |
| 基于已有问卷创建副本 | 标题自动添加"--副本"后缀，以创建模式提交而非编辑模式 |
| 问卷开关关闭 | 患者创建问诊时不再自动发送问卷，但已发送的问卷不受影响 |
