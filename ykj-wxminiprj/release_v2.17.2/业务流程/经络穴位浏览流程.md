# 经络穴位浏览流程

> 文档版本：v2.17.2 | 面向读者：患者

## 概述

患者在中医百科模块浏览经络穴位知识库，支持「按经络查询」和「按部位查询」两种方式切换，通过左侧分类导航+右侧内容列表的双栏布局查找穴位，查看详情（含拼音、功能作用、富文本图文详解）。

## 触发条件

- **前置条件**：无特殊要求，任何已登录的患者均可使用
- **入口路径**：中医百科 → 经络穴位

## 步骤详解

### 第一步：进入经络穴位页

用户看到有国风背景的页面，顶部标题「经络穴位」，下方是搜索框和两个查询方式选项卡：

- **按经络查询**：如手太阴肺经、手阳明大肠经、足阳明胃经等（来自字典 `jlxw-jl`），列出十二正经和奇经八脉
- **按部位查询**：如头部、面部、颈部、胸部、腹部、背部、上肢、下肢等（来自字典 `jlxw-bw`）

默认选中「按经络查询」，选项卡切换时左侧分类列表和右侧穴位列表同时更新。被选中的选项卡下方有金色下划线指示。

### 第二步：左侧选分类，右侧看穴位

页面采用**双栏布局**：

- **左侧**（灰色背景）：竖向滚动列表，展示当前查询方式下的所有分类。选中的分类项高亮（深色背景+白色文字），滚动时自动定位到选中项。
- **右侧**（白色背景）：展示当前分类下的穴位列表，每行包含小正方形缩略图+穴位名称，点击可进入详情。

切换分类时，右侧列表自动刷新并回到顶部。每页加载**15条**，滚动到底部自动加载更多。

顶部搜索框可输入关键词按穴位名称模糊搜索，输入后实时刷新右侧列表。

### 第三步：查看穴位详情

点击穴位条目进入详情页，从上到下展示：

1. **封面大图**（750×750）：穴位示意图
2. **标题卡片**（白底圆角）：拼音（灰色小字）+ 穴位名称（大号粗体）+ 功能作用（一行灰色简介，最多两行截断）
3. **详细内容区**：富文本 HTML 渲染（`details` 字段），通常包含穴位定位、主治病症、操作手法、配伍等信息，支持图文混排

### 第四步：分享穴位

在列表页分享时标题固定「经络穴位」，携带当前查询类型和分类参数；在详情页分享时标题为穴位名称，携带ID。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 某分类下无穴位 | 右侧列表显示空状态+「暂无数据」 |
| 搜索无匹配 | 显示空状态，可修改搜索词或切换分类 |
| 详情页功能作用超长 | 限两行显示，超出部分截断省略 |
| 网络请求失败 | 页面保持上次数据显示，不自动重试 |
| 分享后进入时携带了筛选参数 | 自动选中对应分类并滚到可视区（0.5秒延迟定位） |

## 关联功能

- **中医百科首页**：返回百科首页，可切换到其他模块
- **微信分享**：好友点击分享链接可直接进入经络穴位列表或详情
