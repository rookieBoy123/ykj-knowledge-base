# 快速开方流程

> 文档版本：v1.0 | 最后更新：2026-07-01 | 面向读者：产品/运营

## 概述

快速开方是医生端的一个独立开方功能，不需要先有问诊记录即可直接为患者开具处方。医生选择患者后，通过中药/成药两个选项卡切换，填写诊断信息和药品清单，支持十八反十九畏自动校验、保存为模板和电子签章。

---

## 触发条件

- 医生需要为线下就诊的患者快速开具处方
- 医生不需要通过完整问诊流程就能开方
- 适用于复诊/老患者场景

---

## 步骤详解

### 第一步：选择患者

进入快速开方页面后：

- 选择就诊患者（通过历史患者列表或搜索）
- 系统加载患者的病历和过敏史信息
- 患者信息区域显示姓名、性别、年龄等基本信息

### 第二步：选择处方类型并填写药品

通过顶部选项卡切换中药（zy）和成药（cy）两种处方类型：

**成药店方**：
- 选择药房（从医生关联的药房列表中选择）
- 搜索和添加药品（支持拼音首字母搜索）
- 设置每个药品的用法用量（用药方式、使用频次、每次用量、使用天数）
- 填写医嘱
- 支持从处方模板导入和历史处方导入

**中药处方**：
- 选择药房
- 搜索和添加中药饮片（按名称或拼音）
- 每种药材设置克数、煎法（先煎/后下/包煎等）
- 设置副数和煎药方法

### 第三步：填写诊断信息

诊断信息区域填写：

- **辨病**：中医辨病结果
- **诊断**：中医辨证分型
- **辨证**：辨证要点

根据医生擅长领域自动匹配默认诊断选项。

### 第四步：中药校验

提交中药处方前，系统自动进行十八反十九畏校验：

- 检查处方中是否存在配伍禁忌（如甘草反甘遂等）
- 检查有毒中药剂量是否超标
- 如有问题弹出校验弹窗，医生可选择：
  - **修改药品**：返回编辑调整
  - **忽略继续**：确认无误后继续签章（弹窗中需要二次确认）

对于因药房库存导致自动删除的药品，也会弹出提示。

### 第五步：保存为模板（可选）

医生可以选择将本次处方保存为模板：

- 打开"保存为模版处方"开关
- 输入模板名称
- 提交开方时同步保存模板

### 第六步：确认开方与签章

点击底部「确认开方」按钮：

- 首先进行表单校验（患者、诊断、药品等完整性检查）
- 对中成药检查用法用量是否完整
- 对中药执行十八反十九畏校验
- 校验通过后调用电子签章：
  - 医网信签章：调用医网信SDK进行人脸识别或手写签名
  - 放心签签章：弹出签章弹窗完成签名
- 签章完成后处方生效
- 可选择查看处方记录或返回首页

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未选择患者 | 底部确认按钮禁用，无法提交 |
| 未填写诊断 | 底部确认按钮禁用 |
| 未添加药品 | 底部确认按钮禁用 |
| 中成药用法用量不完整 | 按钮禁用，需补全所有药品的用法用量 |
| 十八反十九畏冲突 | 弹出校验弹窗，列出冲突项 |
| 有毒中药超量 | 弹出校验弹窗，警示超量药品 |
| 药房库存删除药品 | 自动删除弹窗，需确认 |
| 签章失败 | 显示失败原因，可重试 |
| 未选择药房 | 校验不通过，提示选择 |

---

## 关联功能

- **处方模板**：可从模板导入药品，也可保存为模板
- **历史处方**：可从历史处方导入药品
- **电子签章**：医网信/放心签签章支持
- **中药校验**：十八反十九畏自动检测
