# 医生端随访计划创建与管理流程

> 文档版本 2026-07-02 | 面向使用者：医生

## 概述

随访计划是医生在患者就诊后，为患者安排的一系列定期跟进任务。医生可以创建包含患教资料、提醒和问卷的随访计划，系统按设定时间自动发送给患者。医生可随时查看计划详情、填写随访总结或取消计划。

## 触发条件

- **前置条件**：医生已与患者有过就诊关系
- **入口路径**：医生工作台 → 发起随访 → 选择患者 → 创建随访计划
- **备用入口**：从随访模板列表选择模板后进入

## 步骤详解

### 第一步：选择患者

进入发起随访页面：
- 顶部搜索栏：输入患者姓名实时搜索
- 患者标签筛选：显示医生为患者设置的标签，可多选筛选
  - 标签展示限制两行，点击「更多」展开全部标签
  - 选中的标签高亮显示，更多按钮显示已选数量
  - 支持重置所有标签
- 患者列表：每页 10 条，触底加载更多
  - 展示患者头像、姓名、性别、年龄
  - 头像根据性别和年龄自动判断（男/女/6 岁以下儿童）
  - 已有随访计划的患者卡片下方显示随访计划标签，可点击跳转查看
- 点击患者前先校验是否有就诊关系：
  - 有就诊关系：跳转创建计划页
  - 无就诊关系：弹窗提示「随访暂只支持对您就诊过的患者发起」

### 第二步：填写计划基本信息

进入创建随访计划页面后：

- **计划名称**（必填）：自由输入
- **就诊时间**（必填）：从患者最近的就诊时间列表中选择
- 如果从模板进入，自动回填模板的计划名称和任务内容

### 第三步：添加随访任务（计划项）

每个计划项包含：
- **间隔时间**：距就诊时间后多少天/周/月
- **间隔单位**：天（1）/ 周（2）/ 月（3）
- **任务内容**：可添加患教资料（type=0）、提醒文字（type=1）、问卷（type=2）
  - 可以添加多条内容（点击任意一条跳转对应详情页）
  - 提醒内容可点击编辑，弹出文本框最多 100 字
- **删除按钮**：第二个及之后的计划项可删除
- **添加新计划**：点击「添加计划」按钮增加一个空白计划项

首条计划项默认间隔为 1 周，其余新建计划项默认间隔为 0（未设置）

### 第四步：保存并执行

点击底部「执行计划」按钮：

1. **校验**：
   - 计划名称不能为空 → 提示「请输入计划名称」
   - 就诊时间不能为空 → 提示「请选择就诊时间」
   - 计划项的随访日期不能已过期 → 提示「计划N的随访日期已过，请重新选择」
   - 每个计划项必须至少有一条内容 → 提示「请添加发送内容」

2. **提交**：调用添加随访接口，成功后提示「执行计划成功」，3 秒后自动跳转计划详情页

### 第五步：查看和管理随访计划

进入计划详情页后：

**页面信息展示**：
- 患者信息（姓名、性别、年龄、头像）
- 计划名称和状态标签
- 就诊时间
- 各计划项的间隔时间、状态、发送时间、包含的任务内容

**状态说明**：
- **进行中**（status=0）：显示「取消」按钮
- **已完成**（status=1）：显示「填写随访总结」或「查看随访总结」按钮
- **已取消**（status=2）：显示「查看取消原因」按钮

**操作**：
- 取消随访：点击取消按钮 → 弹出输入框填写取消原因（必填）→ 确认弹窗「确认取消该患者的随访计划吗？确认后，该随访计划将不再发送给患者」→ 提交后提示「已取消」并返回上一页
- 填写随访总结：已完成状态下 → 输入随访总结内容 → 保存
- 查看随访总结：已有总结内容时可直接查看

### 第六步：查看随访任务详情

从计划详情页点击具体任务内容可查看：

- **患教资料**（课程）：跳转课程详情页展示患教内容
- **提醒**：弹出弹窗显示提醒文字内容
- **问卷**：跳转问卷详情页展示问卷题目和填写结果

任务详情页面区分医生端和患者端：
- **医生端**：显示患者信息，各任务有「完成/待办」状态
- **患者端**：显示医生信息，随访时间
- 提醒类任务自动标记为已完成（`changeRemindStatus`）

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 计划名称为空 | 提示「请输入计划名称」 |
| 就诊时间未选 | 提示「请选择就诊时间」 |
| 随访日期已过 | 提示具体哪个计划项日期已过 |
| 发送内容为空 | 提示「请添加发送内容」 |
| 取消原因未填 | 提示「请填写取消原因」 |
| 随访总结未填 | 提示「请填写随访总结」 |
| 患者无就诊关系 | 弹窗提示不可发起随访 |
| 无患者标签 | 显示「暂无患者标签」 |
| 无患者记录 | 显示空状态「暂无患者」 |
| 已显示全部数据 | 底部显示「- 已显示全部 -」 |

## 关联功能

- **随访模板管理**：创建随访时可从模板导入（模板创建/列表/详情）
- **患者标签管理**：通过标签筛选患者
- **患教资料管理**：课程内容的创建和查看
- **问卷管理**：随访问卷的创建和填写
- **随访任务详情**：患者端查看和完成随访任务
