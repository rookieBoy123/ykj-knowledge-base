# 医生处方记录管理流程

> 文档版本：release/v2.17.2 | 面向读者：客服、运营

## 概述

医生在「处方记录」页面中查看和管理自己开具的所有处方。支持按8种审核状态筛选、按患者姓名搜索、分页加载。每张处方卡片上可执行查看详情、删除、撤回、签名、驳回、分享、发送等操作。

## 触发条件

### 前置条件
- 用户已登录为医生角色
- 医生至少开过一次处方

### 入口路径
- **医生端首页** → 「处方记录」（普通模式）
- **问诊工作台** → 某次问诊 → 查看已开处方（已开处方模式，标题变为「已开处方」，无筛选栏）

## 步骤详解

### 第一步：查看处方列表

每页15条处方，支持触底加载更多。

**8种状态筛选标签**（普通模式）：

| 标签 | 说明 |
|------|------|
| 全部处方 | 不筛选 |
| 审核通过 | 处方已通过审核 |
| 审核中 | 药师正在审核 |
| 待审核 | 已提交等待审核 |
| 审核不通过 | 药师驳回 |
| 开方中 | 正在开具 |
| 已撤回 | 医生主动撤回 |
| 处方草稿 | 未提交的草稿 |

切换标签后自动回到顶部并重新加载第1页。

**搜索**：支持输入患者姓名模糊搜索。

### 第二步：处方卡片上的操作

**通用操作**：
- **查看详情**：跳转到患者端处方详情页
- **微信分享**：标题格式「{医生名}医生为「{患者名}」开具的处方」

**处方签名（CA电子签章）**：
- 仅待签名处方显示
- 严格模式需先人脸识别
- 支持医网信、源康健PaaS、放心签三种平台

**删除处方**：
- 仅「审核不通过」的处方可删除
- 二次确认弹窗「确认删除该处方吗？删除之后无法恢复」

**撤回处方**：
- 仅「待审核」和「审核中」的处方可撤回
- 二次确认弹窗「确认撤回吗？」

**驳回处方**（医生级别）：
- 必填驳回原因，最多500字

**处方消息发送**：
- 如果患者绑定多个微信，弹出选择窗口
- 选择后处方消息发送到对应微信聊天中

### 第三步：分页加载

滚动到底部自动加载下一页，全部加载完显示已加载全部。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 接口异常 | 弹窗提示错误信息，仅「知道了」 |
| 撤回失败 | 弹窗提示后端错误信息 |
| 驳回未填写原因 | 提示「请输入驳回原因」 |
| 严格模式未人脸识别 | 跳转人脸识别页 |

## 关联功能

- 处方详情页、人脸识别、微信分享
