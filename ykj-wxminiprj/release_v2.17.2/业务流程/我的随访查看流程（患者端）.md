# 我的随访查看流程（患者端）

> 文档版本 2026-07-01 | 面向读者：患者

## 概述

患者可以在"我的随访"中查看医生分配给自己的随访任务，按家庭成员筛选、按状态分类浏览，查看每条随访的详细内容（患教文章、提醒、问卷），并完成待办项。

## 触发条件

- 从"我的"页面点击"我的随访"入口进入
- 医生发送新的随访任务后，患者从小程序消息通知进入

## 步骤详解

### 第一步：选择家庭成员

页面顶部显示家庭成员横滑列表，自动默认选中第一个成员。切换成员后，下方的随访任务列表和数据统计自动刷新。

### 第二步：浏览随访任务

每位成员的随访任务按日期分组展示。每条随访卡片包含：
- 发送日期（格式：XX年XX月XX日）
- 状态标签（待办=红色 / 已完成=灰色）
- 随访内容列表，每项带有类型图标：
  - 📖 患教文章
  - 🔔 提醒
  - 📋 问卷
- 每项内容的标题和完成状态（未完成标记"待办"红色文字）
- 医生信息（头像+姓名+科室+职称）
- "查看详情"按钮

### 第三步：筛选切换

顶部状态栏支持三个选项卡：
- **全部**：所有随访任务
- **待办**：仅显示 status=1（有未完成内容项）的任务
- **已完成**：仅显示 status=2（全部内容已完成）的任务

切换选项卡时立即重新查询列表。

### 第四步：分页与吸顶

- 每页展示5条随访任务，滚动到底部自动加载更多
- 当页面向下滚动超过100像素时，状态选项卡自动吸顶固定在页面顶部
- 已加载全部数据时底部显示"- 已显示全部 -"

### 第五步：查看详情与完成待办

点击"查看详情"进入随访任务详情页，可查看每项随访内容的具体详情：

- **患教文章**：直接阅读，患者端会自动标记为"已完成"
- **提醒**：以底部弹窗面板展示提醒文字内容
- **问卷**：进入答题页面完成问卷填写

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 该成员没有任何随访任务 | 显示空状态插图+"暂无随访记录" |
| 切换成员时网络请求失败 | 保持上一个成员的显示 |
| 问卷未完成全部题目 | "提交"按钮保持灰色不可点击 |
| 页面加载中 | 显示全局加载动画 |

## 关联功能

- 跳转到：随访任务详情（患病/问卷/提醒内容）
- 跳转到：量表答题页面（如果随访含问卷）
- 关联：医生端"发起随访"功能
