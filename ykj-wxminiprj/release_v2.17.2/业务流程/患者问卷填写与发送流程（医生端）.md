# 患者问卷填写与发送流程（医生端）

> 文档版本：release/v2.17.2 | 面向读者：医生

## 概述

医生在问诊过程中可以向患者发送问卷，患者填写后医生可以在「查看患者填写问卷」页面追踪填写进度。支持查看已填写和未填写的问卷，并可随时发送新的问卷给患者。

## 触发条件

- **前置条件**：医生已进入某个问诊的聊天室，该问诊已绑定了问卷
- **入口路径**：问诊聊天室 → 发送问卷入口 → 「查看患者填写问卷」页面

## 步骤详解

### 第一步：进入问卷追踪页

页面展示该问诊下绑定的所有问卷列表。每条问卷显示：

- **填写状态角标**：右上角显示状态标签
  - 「未填写」：红色背景，患者尚未填写
  - 「已填写」：蓝色背景，患者已经完成填写
- **问卷图标**：标准问卷图标
- **问卷标题**：最多两行显示，超出省略
- **题目数量**：显示「共XX题」

### 第二步：查看问卷填写详情

点击任意问卷卡片，跳转到该患者的问卷填写详情页。详情页展示：

- **问卷标题和说明**
- **每道题的完整内容**：包括题目文字、题目类型（单选/多选/问答/图片，不同类型有不同颜色标签）、是否为必填
- **患者的回答**：每道题显示患者的选择或输入内容
- **单选题显示**：选项列表，患者选中的选项高亮
- **多选题显示**：选项列表，患者选中的多个选项高亮
- **问答题显示**：患者输入的文字内容
- **图片题显示**：患者上传的图片列表，点击可预览

### 第三步：发送新问卷

医生点击底部「发送问卷」按钮，弹出一个问卷选择弹窗：

1. 弹窗列出医生可用的所有问卷（包括「我的问卷」和「公用问卷」）
2. 医生选中一份问卷（单选）
3. 点击确认发送
4. 如果未选择问卷就点确认，提示「请先选择问卷」
5. 发送成功后提示「问卷发送成功」，列表自动刷新

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 该问诊尚未绑定任何问卷 | 显示空状态「暂无问卷」，可点击发送问卷 |
| 发送问卷时未选择 | 提示「请先选择问卷」 |
| 患者尚未填写任何问题 | 详情页下患者回答部分为空 |
| 医生同时查看多个问诊 | 通过inquiryNo区分不同问诊的问卷数据 |

## 关联功能

- **问诊聊天室（IM）**：从聊天室进入问卷管理，发送的问卷会以消息形式推送给患者
- **问卷详情（inquiryDetail）**：点击查看患者对每道题的具体回答
- **问卷创建（create）**：医生可创建自己的问卷用于发送
- **问卷设置（set）**：配置哪些场景自动发送问卷

## 数据刷新机制

- 进入页面（onShow）自动从后端获取最新的问卷填写状态
- 发送新问卷后自动刷新列表
- 返回聊天室后再进入会重新获取最新状态

## 补充说明

- 同一问诊可以向患者发送多份问卷
- 患者填写后，问卷状态从「未填写」变为「已填写」
- 问卷发送功能需要医生已配置问卷（在「问诊设置」中），否则弹窗中无可选问卷
- 患者端填写问卷后，医生端需要手动刷新或重新进入页面才能看到最新状态
