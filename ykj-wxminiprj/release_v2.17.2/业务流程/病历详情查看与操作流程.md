# 病历详情查看与操作流程

> 文档版本：release/v2.17.2 | 面向读者：客服、运营

## 概述

医生在完成问诊后可以查看患者的完整病历内容，包括患者信息、病史、问诊问卷、诊断结果、治疗意见、病历资料、问诊对话记录、处方记录和医生签名。医生还可以进行撤销病历、编辑病历、CA电子签名等关键操作。患者在特定条件下也可以查看病历详情。

## 触发条件

### 前置条件
- 医生已成功接诊并对患者进行了问诊
- 医生已为患者编写了病历（病历ID已生成）
- 病历需由当前医生本人查看（跨医生查看会提示身份不一致）

### 入口路径
- **医生端**：从问诊记录 → 查看问诊详情 → 点击「查看病历」进入
- **患者端**：从健康档案 → 诊疗记录 → 点击病历进入（只读模式）
- **快速续方流程中**：从续方页面 → 查看已开处方 → 点击病历进入（参数 isContinue=true）

## 步骤详解

### 第一步：进入病历详情页

页面加载时通过病历ID查询病历数据，同时查询关联的问诊信息。

**页面展示区域从上到下依次为**：

1. **患者信息卡片**：显示患者头像（按性别自动切换）、姓名、性别、年龄（≥1岁显示「X岁」，<1岁显示「X月龄」）、监护人信息（如有）、就诊科室、就诊时间。医生角色下右上角有「拨打电话」按钮。

2. **门慢医保标签**：如果本次问诊属于门慢医保复诊，蓝色标签展示慢病病种名称。

3. **病史信息区域**：以表格形式展示多项病史（过敏史、既往史、个人史、家族史、婚育史、月经史、舌象、脉象、体征、治疗史等）。每项有则显示有，无则显示否。

4. **问诊问卷记录**：展示患者在发起问诊时填写的问卷答案。

5. **临床诊断区域**：显示三项诊断——辨病、诊断、辨证。三种诊断类型分别归类展示。

6. **治疗意见**：展示医生填写的治疗意见文本。

7. **病历资料**：展示医生上传的病历相关图片文件。

8. **问诊对话记录**：一张卡片入口，点击「>」跳转到聊天对话记录。

9. **处方记录**：展示关联的处方列表，如果是快速续方流程进入，标记为续方模式。

10. **医生签名**：展示医生的签名，支持两种格式——图片格式（base64编码）和文字格式。

### 第二步：底部操作按钮（仅医生角色）

底部按钮在同时满足以下条件时显示：
- 当前角色为医生
- 系统处于严格模式
- 问诊状态 flowStatus=30（问诊已结束）
- 距离问诊结束时间不超过24小时

按钮组合根据签名状态动态变化：
- 待签名+满足条件：显示「撤销病历」（红色）+「签字」
- 待签名+不满足条件：单独显示「签字」
- 已签名/无需签名：显示「撤销病历」（红色）+「编辑病历」

三种操作详情：

1. **签字**（CA电子签章）：
   - 先检查医生身份是否一致，不一致则提示「当前用户与待签名病历归属的用户不一致，不可签名」
   - 一致后初始化签名获取caUniqueId
   - 支持3种签名平台：医网信、源康健PaaS、放心签
   - 签名成功后每1.5秒轮询病历状态，最多轮询7次（约10秒）

2. **撤销病历**：
   - 弹出底部输入框，必须填写「撤销原因」（必填，最多500字）
   - 确认后调用撤销接口，成功后提示「已撤销」，1.5秒后自动返回

3. **编辑病历**：
   - 跳转到病历编辑页

### 第三步：医生拨打电话

医生可点击患者信息卡右上角的「拨打电话」按钮。如果患者没有登记电话号码，提示「暂无可拨打电话」。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 当前医生与病历所属医生不一致 | 弹窗提示不一致，强制返回首页 |
| CA签名初始化失败 | 不显示签字入口 |
| 签名轮询超时（7次×1.5秒） | 停止轮询，不提示错误 |
| 撤销病历未填写原因 | 提示「请输入撤销原因」 |
| 患者无电话号码 | 提示「暂无可拨打电话」 |
| 放心签取消操作 | 静默关闭弹窗 |
| 放心签失败 | 提示「签字失败，请重试」 |

## 关联功能

- 病历编辑页、「问诊对话记录」→聊天室、处方详情页
- 撤销成功后自动返回上一页
