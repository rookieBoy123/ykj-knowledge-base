# AI医生问诊记录查询流程

> 文档版本 2026-07-03 | 面向医生用户

## 概述

医生可以查看其AI医生分身处理的所有历史问诊记录。支持按患者姓名搜索、按问诊日期筛选、按患者年龄区间筛选，点击任一记录可直接进入该AI问诊室查看完整对话。

## 触发条件

- **前置条件**：医生已开启AI医生分身功能，且有患者通过AI医生进行过问诊
- **入口路径**：医生端 → AI医生分身 → 问诊记录

## 步骤详解

### 第一步：进入问诊记录页

进入页面后，顶部标题栏显示"医生分身问诊记录"。页面分为两个区域：
- **搜索与筛选区**：顶部的搜索栏和两个下拉筛选条件
- **记录列表区**：下方按时间倒序排列的问诊记录卡片

### 第二步：按患者姓名搜索

在顶部的搜索框中输入患者姓名，系统会实时搜索（输入即触发，无需点击搜索按钮）。搜索结果按分页展示，每次显示10条记录。

搜索采用防抖处理（200毫秒延迟），避免频繁请求。

### 第三步：按问诊日期筛选

点击"问诊日期"下拉选项，弹出日期选择器：
1. 选择某一天作为筛选条件
2. 系统用该日期作为查询范围（startDate = endDate = 选择的日期）
3. 自动搜索并刷新列表
4. 点击日期选择器的清除按钮，可取消日期筛选，恢复显示全部记录

### 第四步：按年龄区间筛选

点击"年龄区间"下拉选项，弹出双列滚动选择器：
1. 左侧选择最低年龄（1-120岁），右侧选择最高年龄（1-120岁）
2. 系统会校验：最低年龄不能高于最高年龄，否则提示"请选择正确的年龄区间"
3. 如果未选择任何年龄就点"确认"，系统提示"请先选择年龄区间"
4. 确认后，选中区间显示在筛选栏（如"25岁-45岁"），自动搜索

年龄区间选择器底部有两个按钮：
- **取消**：放弃选择，清除年龄筛选条件，刷新为全部记录
- **确认**：应用选择的年龄区间进行筛选

### 第五步：查看问诊记录卡片

每条问诊记录以卡片形式展示，信息包括：
- 患者姓名
- AI医生分身名称
- 问诊时间
- 问诊状态

### 第六步：进入AI问诊室

点击任意记录卡片，系统会跳转到对应的AI问诊对话室，医生可以查看完整的AI与患者的对话记录。跳转使用问诊编号（bizNo/inquiryNo）作为参数。

### 第七步：加载更多

向下滚动列表到底部时，自动加载下一页数据。如果当前已是最后一页，则不再触发加载。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 无问诊记录 | 显示空状态页面，提示"暂无问诊记录"，附带插画图标 |
| 年龄区间选择错误（最低>最高） | 提示"请选择正确的年龄区间"，不执行搜索 |
| 未选年龄就点确认 | 提示"请先选择年龄区间" |
| 网络加载失败 | 已加载的记录保持显示，不会因加载失败清空列表 |
| 搜索结果为空 | 正常显示空列表，无额外提示 |

## 关联功能

- **AI医生分身对话**：点击记录可查看完整的AI问诊对话
- **AI医生分身设置**：医生开启/配置AI医生分身的入口
- **患者管理**：患者姓名来自患者注册信息
