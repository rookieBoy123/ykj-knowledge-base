# 健康档案总览流程

> 文档版本：2026-07-01 | 面向读者：患者

## 概述

患者查看自己及家人的健康档案，包括已登记的慢病病种、线上就诊病历和线下（HIS同步）就诊病历，支持多个就诊人切换。

## 触发条件

- 从"我的"页面进入"健康档案"
- 携带指定就诊人ID参数进入（如从问诊详情跳转）

## 步骤详解

### 第一步：切换就诊人

进入页面后：
- 顶部显示当前就诊人的基本信息（姓名、性别、年龄）
- 如果有多个家庭成员，顶部有就诊人横滑列表，点击可切换
- 切换就诊人时，下方病历数据自动重新加载
- 页面滚动超过30像素时，顶部导航栏变为白色背景，就诊人列表固定在顶部

### 第二步：查看慢病病种

如果系统开启了医保支付功能且当前就诊人有已登记的慢病病种，显示蓝色标签列表：
- 展示每个病种名称（如"高血压""糖尿病"）
- 以蓝色背景标签横向排列
- 无慢病记录时不显示此区域

### 第三步：查看线上/线下病历

病历区域有两个选项卡：

**线上病历**（互联网问诊记录）：
- 按时间倒序展示该就诊人的历次线上问诊病历
- 每条记录显示：就诊时间、接诊医生、主诉摘要
- 点击可查看完整病历详情（与问诊详情页相同）
- 每页5条，滚动加载更多

**线下病历**（HIS系统同步）：
- 展示从医院HIS系统同步的线下门诊病历
- 每条记录显示：门诊号、就诊医生、就诊时间
- 点击查看完整病历详情（与问诊病历格式不同，包含更多体检信息）
- 每页5条，滚动加载更多

两种病历独立分页，各自滚动到底部时加载更多。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 无就诊人 | 显示错误状态（componentStatus='error'），页面空白 |
| 无线上病历 | 病历列表为空 |
| 无线下病历 | 病历列表为空 |
| HIS同步中断 | 线下病历为空，不影响线上病历查看 |
| 切换就诊人时 | 自动重置病历分页到第一页 |

## 关联功能

- 点击线上病历 → 查看完整病历详情（含诊断、处方）
- 点击线下病历 → 查看HIS同步的完整门诊记录
- 从"我的家庭成员"可编辑就诊人信息
- 慢病病种信息由医保系统同步
