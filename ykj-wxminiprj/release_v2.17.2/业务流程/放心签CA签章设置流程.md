# 放心签CA签章设置流程

> 文档版本：release/v2.17.2 | 面向读者：医生、运营

## 概述

放心签是平台的CA电子签章方案之一，医生需要先设置签名密码，再完成手写签章，才能在开方、病历等环节使用电子签名。设置完成后，系统会自动显示医生已有的签章图片（如有）。

## 触发条件

- **前置条件**：医生已完成身份认证，平台已为医生开通放心签CA账号
- **入口路径**：医生端 → 「签章设置」页面（两个入口：「设置签名密码」和「设置签章」）

## 步骤详解

### 第一步：进入签章设置首页

医生看到两个设置入口卡片：「设置签名密码」和「设置签章」。如果医生之前从未设置过，必须先完成密码设置，再进行签章设置。两者有先后依赖关系——没有密码无法确认签章。

### 第二步：设置签名密码

1. 点击「设置签名密码」，系统自动获取当前医生的手机号并向该号码发送短信验证码
2. 页面顶部显示蓝色提示条：「已向手机号 138****1234 发送验证码」
3. 医生需要在「验证码」输入框中填入收到的6位数字验证码
4. 在「签名密码」输入框中输入6位数字密码（输入隐藏）
5. 在「确认密码」输入框中再次输入相同密码
6. 点击「确定」提交

**校验规则**：
- 两次输入密码必须一致，不一致时提示「两次密码不一致，请重新输入」
- 如果验证码错误或密码格式不正确，提示具体错误信息
- 手机号丢失时提示「手机号丢失，请重新登录」

**支持重新获取验证码**：验证码发送后有60秒倒计时，按钮变为「60s」→「59s」...倒计时结束后恢复为「获取验证码」，可重新获取。

### 第三步：签名密码设置成功后选择下一步操作

成功后弹出成功弹窗，有两个按钮：
- **「确定」**：返回上一页，签章密码已生效
- **「设置签章」**：直接跳转到设置签章页面

### 第四步：设置签章（手写签章）

1. 进入签章设置页面后，系统自动查询医生已有的签章图片并显示
2. 如果还没有签章，页面中央显示灰色区域，底部有蓝色「手写签章」按钮
3. 点击「手写签章」→ 跳转到手写签名板页面，医生用手指签名（至少3笔）
4. 签名完成后返回签章设置页，签章图片替换为刚写好的签名
5. 点击「确认」→ 弹出底部密码输入弹窗

### 第五步：输入签名密码确认

1. 底部弹出6位密码输入框（每个数字一格，输入后显示圆点）
2. 输入完整6位密码后，系统自动提交签章图片和密码到放心签平台
3. 如果签章设置成功，提示「签章设置成功！」
4. 系统自动清除临时的签名板数据

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未先手写签章就点确认 | 提示「请先完成手写签章」，不会弹出密码输入框 |
| 密码输入错误 | 后端返回错误信息后弹窗提示，签章未保存 |
| 短信验证码发送失败 | 接口返回失败信息，倒计时不会启动 |
| 手机号为空 | 提示「手机号丢失，请重新登录」，无法继续 |
| 密码输入不满6位 | 需要重新输入，不满6位不会触发提交 |

## 关联功能

- **处方开具**：签章设置完成后，开具处方时可选择放心签平台进行CA电子签名
- **病历编辑**：严格模式下病历保存后需要签名，可使用已设置的签章
- **群会诊签名**：会诊申请签名环节可选择放心签平台
- **处方记录**：处方记录中可对处方进行签名操作

## 补充说明

- 签章设置为一次性配置，配置后长期有效，无需每次开方都重新设置
- 签名密码是6位纯数字，忘记密码需通过「忘记密码」流程重置（需短信验证）
- 签章图片可以在签名板重新绘制后覆盖
- 放心签是平台三种CA签章平台之一，另外两种是医网信和源康健PaaS，医生实际使用哪种由平台后台配置决定
