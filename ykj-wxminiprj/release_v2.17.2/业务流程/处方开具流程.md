# 处方开具流程（医生端）

> 文档版本：release/v2.17.2 | 面向读者：运营、客服、产品经理

---

## 概述

医生在问诊过程中或问诊结束后，为患者开具电子处方。处方分为**成药处方（西药/中成药）**和**中药处方（中药饮片）**两种类型，均需经过电子签名（签章）后方可生效。支持从处方模板、历史处方快速导入用药方案。

## 触发条件

- 医生正在进行图文/视频问诊，需要为患者开具处方
- 医生通过"快速开方"功能直接为已有医患关系的患者开方（无需问诊）
- 医生通过"快速续方"为患者续开历史处方

## 前置条件

- 医生已通过资质审核，账号处于启用状态
- 如果是正常开方：需要有对应的问诊编号（inquiryNo）
- 如果是快速开方：需要患者的姓名、身份证号、手机号等基本信息
- 医生已完成CA电子签章证书的下载和认证
- 如果开启中医科模式，需当前医生科室为中医科室

## 步骤详解

### 步骤一：进入开方页面

**入口路径：**
- 问诊工作台 → 点击"开具处方"按钮 → 跳转到处方编辑页（传入问诊编号）
- 患者管理 → 选择患者 → "快速开方"
- 处方记录 → 选择历史处方 → "快速续方"
- 处方模板 → 选择模板 → 使用模板开方

**页面内容：**
- 顶部：中药处方/成药处方切换标签（中医科医生默认中药，西医科医生默认成药）
- 患者信息区：展示患者姓名、年龄、性别、身份证号等基本信息
- 慢病医保提示：若患者为门慢医保复诊患者，显示蓝色提示条"该患者为门慢医保复诊请尽量使用医保目录内药品"
- 医保定点提示：若患者已定点且医生支持医保，显示定点信息

### 步骤二：填写诊断信息

医生填写诊断相关内容：

1. **辨病**（中医）：选择或输入疾病对应的中医辨病结果
2. **诊断**（西医）：填写西医诊断，如系统配置了"诊断必填"开关，此项必须填写
3. **辨证**（中医）：选择或输入中医辨证分型

以上内容支持从AI辅助中获取建议，也可手动输入。

### 步骤三：添加药品

#### 成药处方（西药/中成药）

医生从药房中选择药品加入处方：

1. 选择发药药房（系统自动匹配该医生的默认药房）
2. 点击"添加药品"按钮，搜索药品名称
3. 选择药品后，填写每次用量、用法（口服/外用等）、频次、天数
4. 可添加多个药品，支持从处方模板或历史处方导入
5. 填写医嘱（用药建议的补充说明）
6. 可选择是否免运费（需要后台开启该配置）

#### 中药处方（中药饮片）

医生从药房中选择中药饮片：

1. 选择发药药房
2. 添加中药饮片，设置每味药的剂量（克数）
3. 选择加工工艺（如代煎、打粉等，取决于药房提供的服务）
4. 填写用法用量（如"每日一剂，分两次煎服"）
5. 支持从处方模板或历史处方导入

**校验规则：**
- 中药处方会进行"十八反十九畏"配伍禁忌检查
- 有毒中药剂量超量检查（超量时弹窗提示）
- 黑名单药品自动删除（弹窗提示"处方中存在黑名单药品系统已自动删除"）

### 步骤四：选择保存方式

在页面底部，医生可选择是否将本次处方**保存为处方模板**：
- 开启"保存为模版处方"开关 → 输入模板名称 → 提交后自动创建模板
- 快速开方/对外上传场景下不显示此选项

### 步骤五：提交处方

点击底部"确认"按钮触发提交流程：

1. **表单校验**：校验诊断信息、药品列表、用法用量完整性
2. **中药特殊校验**（中药处方）：
   - 检查"十八反十九畏"配伍禁忌
   - 检查有毒中药剂量是否超量
   - 如有问题，弹窗展示具体冲突项，医生可逐项修改
3. **药房库存校验**：检查所选药房是否有库存
4. **药房授权校验**：检查药房是否有该处方类型（中药/成药）的授权
5. **CA电子签名**：校验通过后触发电子签章

### 步骤六：电子签名（签章）

提交前必须完成CA电子签名，系统支持三种签章平台：

- **医网信（ywxPlugin）**：需要下载签章证书，设置签章密码
- **源康健PaaS**：平台内置签名服务
- **放心签**：第三方电子签名

**签名流程：**
1. 系统根据配置选择签章方式
2. 弹出签名组件，医生完成签名操作
3. 签名成功后自动提交处方
4. 签名失败时提示重试，并记录失败原因

### 步骤七：完成

提交成功后：
- 处方进入"待审核"状态（如果平台开启了药师审方流程）
- 系统发送处方消息到患者问诊聊天中
- 患者可在"用药建议"中查看处方详情

**快速开方特殊处理：**
- 底部有两个按钮："存为草稿"和"确认发送给患者"
- 存为草稿后弹窗提示"处方已存为草稿，可在[我的-处方记录-处方草稿]查看"，可选择"查看处方记录"或"返回首页"
- 草稿处方不会发送给患者，医生可稍后编辑并发送

## 异常处理

| 异常场景 | 提示文案 | 处理方式 |
|---------|---------|---------|
| 药品中包含黑名单药品 | "处方中存在黑名单药品系统已自动删除" | 系统自动移除黑名单药品，弹窗告知；医生可继续添加其他药品 |
| 中药配伍禁忌（十八反十九畏） | 具体冲突项展示 | 弹窗列出冲突，医生可修改后提交或确认忽略 |
| 有毒中药剂量超量 | 超量提示 | 弹窗提示，医生可调整剂量 |
| 药房库存不足 | 库存不足提示 | 切换其他药房或调整药品 |
| 药房无该类型授权 | 授权校验失败 | 切换其他有授权的药房 |
| CA签名失败 | 签名失败提示 | 重新签名，多次失败检查证书状态 |
| 处方中存在必填项未填 | 表单校验提示 | 补充必填信息后重试 |
| 页面返回时修改未保存 | 无显式提示（表单内容保留在页面中） | 返回页面时数据保留，除非主动退出 |

## 关联功能

- **处方记录**：查看已开具的所有处方及状态（待审核/审核通过/审核不通过/已失效）
- **处方模板**：管理常用处方模板，快速复用
- **历史处方**：查看患者的过往处方，支持快速续方
- **用药需求**：处理患者提交的用药需求申请
- **药品详情**：查看单个药品的详细信息
- **药师审方**（管理后台）：处方提交后由药师审核
