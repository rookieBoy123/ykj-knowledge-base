# 经典名方浏览流程

> 文档版本：release/v2.17.2 | 面向读者：患者、普通用户

## 概述

患者通过"经典名方"（名方今用）功能浏览中医经典方剂知识库，支持按方剂类型、适应疾病、来源书籍三种维度浏览，也可通过搜索直接查找名方。

## 触发条件

### 前置条件
- 已登录小程序（患者角色）
- 无其他特殊限制

### 入口路径
- 中医百科模块 → 经典名方入口

## 步骤详解

### 第一步：进入经典名方列表

页面顶部显示大标题"名方今用"，下方是搜索框（提示文字"请输入关键词搜索"），再往下是三个筛选选项卡——"按类型""按疾病""按书籍"。默认选中"按类型"。

页面分为左右两栏布局：
- **左侧分类列表**：纵向滚动的分类标签，当前选中的选项卡高亮显示（金色底线+粗体黑色文字），其余显示灰色文字
- **右侧名方列表**：对应分类下的名方列表，每条显示方剂封面图（正方形圆角缩略图）和方剂名称，每页15条，滚动到底自动加载更多

### 第二步：切换筛选维度

点击"按疾病"或"按书籍"选项卡后：
- 左侧分类列表自动切换为对应的分类数据（疾病名或书籍名）
- 右侧名方列表随之刷新

三个维度共享同一个右侧列表区域，切换维度时结果自动刷新，当前选中的分类值自动默认为该维度下的第一个分类。

### 第三步：选择具体分类

点击左侧分类列表中的任意一个分类：
- 该分类高亮选中（金色渐变底+白色文字+阴影）
- 右侧名方列表立即刷新，只显示该分类下的名方

分类数据（方剂类型、适应症、相关书籍）均来自后台字典配置（字典编码分别为jtmf-lx、jtmf-syz、jtmf-sj），分类的展示顺序也由后台决定。

### 第四步：搜索名方

在搜索框中输入关键词（最多20个字）：
- 输入内容后自动触发搜索
- 搜索结果按当前选中的筛选维度+分类+关键词三者叠加过滤

### 第五步：查看名方详情

点击列表中的任意名方进入详情页，内容包括：
- **顶部大图**：名方的封面插图（750×750像素），未配置时显示默认占位图
- **标题区**：方剂拼音 + 方剂名称（大号粗体）+ 功效主治摘要（灰色小字，最多两行）
- **内容区**：四个选项卡——"基本用法""药物作用""用药方法""注意事项"，每个选项卡下是富文本格式的详细说明

### 第六步：分享

详情页支持微信分享，分享标题为当前方剂名称。好友点击后直接进入该名方的详情页，筛选参数（维度、分类）也会带入。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 后台未配置名方封面图 | 显示默认占位图（与中药图解不同样式的默认图） |
| 某分类下没有名方 | 显示空状态图+文字"暂无数据" |
| 搜索关键词无匹配结果 | 同样显示空状态 |
| 后台字典未配置分类 | 分类列表为空，名方列表也可能为空 |
| 从分享链接进入（带参数） | 自动识别参数中的维度+分类，定位到对应筛选项并滚动到高亮位置 |
