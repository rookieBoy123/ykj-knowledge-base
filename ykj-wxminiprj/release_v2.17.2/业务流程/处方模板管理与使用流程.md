# 处方模板管理与使用流程

> 文档版本：release/v2.17.2 | 面向读者：医生、运营

## 概述

医生可以将常用的处方保存为模板，方便在开方时一键复用，避免重复填写。模板分为中药处方模板、成药处方模板和公用药方三大类。

## 触发条件

- **前置条件**：医生已登录，且至少有一个门诊号（发起问诊或接诊后）
- **入口路径**：在开方页面或快速开方页面点击"使用模板"→跳转到处方模板列表页

## 步骤详解

### 第一步：选择模板类型

进入页面后，顶部有三个选项卡：**中药处方**、**成药处方**、**公用药方**。

- 中药和成药模板是医生个人创建的，每位医生只能看到自己的
- 公用药方由平台管理员创建，所有医生都可以使用
- 页面顶部有黄色提示条："温馨提示：处方模板处方中如存在黑名单药品系统自动删除"

### 第二步：浏览和搜索模板

每个选项卡下是一个可滚动的列表，每页显示5条模板。每条模板卡片包含：

- **模板名称**：医生创建时填写的标题
- **诊断信息**：中药/成药模板显示诊断、辨病、辨证（三项都可能出现），公用模板显示诊断
- **药品清单**：以逗号分隔的药品名称列表，最多显示两行
- **操作按钮**：
  - **预览**：点击进入模板详情页，查看完整的处方内容
  - **使用**（仅在从开方页进入时有）：自动返回开方页并填充模板数据
  - **编辑**（中药/成药模板独有）：跳转到模板编辑页面修改
  - **删除**（中药/成药模板独有）：弹出二次确认弹窗后删除

页面支持**搜索**功能，可以在顶部搜索框中输入模板名称进行模糊搜索。输入后自动重置到第1页。

### 第三步：新建模板

中药和成药选项卡底部有固定的"添加处方模板"按钮。点击后跳转到处方编辑页面：

- 中药模板：跳转到模板版开方页，预填"中药"类型
- 成药模板：跳转到模板版开方页，预填"成药"类型
- 进入编辑页面前会清空之前的缓存药品数据

公用药方没有新建入口——公用药方只能由平台管理员在后台创建。

### 第四步：预览模板详情

点击"预览"进入模板详情页，可以看到：

- **模板名称**（只读显示）
- **临床诊断区**（中药/成药模板）：诊断、辨病、辨证（有数据才显示）
- **开方区（中药）**：RP标识+药材列表（双列布局）、每味药的煎煮方法、用量、缺货状态标记（红色"药材缺货，请替换"）
- **用法用量区（中药）**：用法（内服/外用）、总剂数、每日剂数、每剂分数次服用、每次用量
- **用药医嘱区（中药）**：用药说明、用药时间（带红色星号）、用药禁忌
- **开方区（成药）**：药品图片+名称+规格+用法用量+数量×单位、已失效标记
- **底部"使用"按钮**：点击自动返回开方页并填充数据

### 第五步：使用模板开方

点击"使用"按钮后，系统会查找页面栈中是否存在开方页（处方开具或快速开方页面）：

1. 如果找到：自动返回到开方页，通过事件通道将模板数据传回
2. 对于中药/成药模板：传递`templateType=template`和处方号`businessId`
3. 对于公用药方：传递`templateType=public`和模板ID`businessId`
4. 如果找不到相关页面：无法使用（可能从非开方入口进入）

### 特殊场景：从开方页快速进入

当从开方页的"使用模板"进入时，页面参数会限制展示：

- **仅限成药模板模式**（`label=1&isUse=1`）：只显示成药模板，隐藏其他选项卡，隐藏"添加"按钮和"编辑"按钮
- **仅限中药模板模式**（`label=2&isUse=1`）：只显示中药模板
- **仅限非公用模式**（`notGol=1`）：隐藏公用药方选项卡，根据医生的医疗类型（中医/西医）自动选择默认选项卡

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 某选项卡下没有任何模板 | 显示空状态提示"暂无记录"，底部保留"添加"按钮 |
| 删除模板 | 弹出确认弹窗"确认删除处方模板吗？"，点击"确认"后调用删除接口，成功后刷新列表 |
| 模板中有药品已缺货 | 预览页对应的药品显示"药材缺货，请替换"红色提示 |
| 模板中有药品已失效 | 预览页对应的药品打上"已失效"灰色标记（该药品仍在清单中展示） |
| 公用药方中部分药品在当前药房不存在 | 使用模板时会自动过滤（待确认） |
| 搜索无匹配结果 | 显示"暂无记录"空状态 |
