# CA签章与签名密码设置流程

> 文档版本 v1.0 | 面向读者：医生

## 概述

医生设置电子签章（手写签名图片）和签名密码，用于处方的CA电子签章。流程分两步：先设置6位签名密码（通过短信验证码验证身份），再手写绘制签章图片并确认保存。

## 触发条件

- **前置条件**：医生已通过CA认证，拥有CA账号
- **场景**：医生开具处方需要进行CA电子签章时
- **入口路径**：医生端 → 个人中心 → 签名密码 / 设置签章

## 步骤详解

### 第一步：设置签名密码（signPassword页面）

页面顶部显示蓝色提示条："已向手机号XXX（已脱敏）发送验证码"。

**表单包含三个字段**：
1. **验证码**：6位数字验证码，右侧有"获取验证码"按钮（60秒倒计时，防重复发送）
2. **签名密码**：6位数字密码，密码输入框（黑点显示）
3. **确认密码**：6位数字，需与签名密码完全一致

**操作流程**：
1. 页面加载时自动调用短信发送接口，向医生绑定的手机号发送验证码
2. 如果手机号丢失（如未绑定），弹窗提示"手机号丢失，请重新登录"，阻止继续操作
3. 用户输入验证码和两次密码后点击"确定"
4. 系统校验两次密码是否一致 → 不一致则提示"两次密码不一致，请重新输入"
5. 系统调用密码设置接口（pwdSetting），传入CA账号、密码、短信验证码
6. 设置成功后弹窗显示"签名密码设置成功"，提供两个按钮：
   - **确定**：返回上一页
   - **设置签章**：跳转到签章设置页面

**验证码重发**：点击"获取验证码"触发60秒倒计时，倒计时期间按钮禁用显示"XXs"，倒计时结束恢复为"获取验证码"。

### 第二步：手写签章设置（signSetting页面）

页面中间是签章预览区域（灰色背景），底部有"手写签章"按钮和操作区。

**页面加载**：自动查询当前医生是否已有签章图片，如有则预填充到预览区。

**操作流程**：
1. 点击"手写签章"按钮 → 跳转到签名画板页面（`signatureSketchpad`）
2. 在画板上手写签名后返回本页面，签章预览区显示手写结果（通过Vuex全局状态传递base64图片数据）
3. 点击"确认"按钮 → 如果未手写签章，提示"请先完成手写签章"
4. 如果已有签章图片 → 弹出6位密码验证框（底部弹出式面板）
5. 密码输入框隐藏（通过透明input覆盖密码方块实现），每输入一位密码，对应方块显示黑点
6. 输入满6位后自动提交：调用签章设置接口（stampImageSetting），传入CA账号、签章图片、密码
7. 设置成功后弹窗提示"签章设置成功！"，同时清空Vuex中的画板缓存

**取消操作**：点击"取消"按钮直接返回上一页。

### 第三步：签名密码的后续使用

医生在开具处方进行CA签章时，需要输入此6位密码来验证签名身份。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 手机号丢失 | 弹窗提示"手机号丢失，请重新登录"，阻止操作 |
| 两次密码不一致 | 弹窗提示"两次密码不一致，请重新输入" |
| 未手写签章就点确认 | 弹窗提示"请先完成手写签章" |
| 密码/验证码错误 | 接口返回100400，显示具体错误信息（如"密码输入错误或者格式不正确"） |
| 短信发送失败 | 接口返回错误，倒计时不启动 |
| CA账号不存在 | 无法获取当前医生信息，页面加载失败 |

## 关联功能

- 处方CA电子签章：开具处方时的签名验证依赖此处设置的签章和密码
- CA认证流程：CA账号由平台管理员开通
- 签名画板：手写签名的独立页面，跨页面通过Vuex传递数据
