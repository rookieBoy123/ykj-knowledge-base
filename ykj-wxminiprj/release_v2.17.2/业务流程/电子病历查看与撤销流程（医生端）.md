# 电子病历查看与撤销流程（医生端）

> 文档版本 1.0 | 面向非技术读者 | 2026-07-01

## 概述

医生在问诊结束后可以查看已生成的电子病历详情，并在满足条件时编辑或撤销病历。病历包含患者基本信息、病情信息（主诉/现病史/既往史/过敏史等8项）、诊疗信息（诊断/辨病/辩证/治疗意见）以及医生签名。页面同时支持线上问诊病历和线下HIS同步的线下病历两种查看模式。

## 触发条件

- 从问诊聊天室/患者详情/病历列表等入口点击进入
- 传入参数：`medicalRecordId`（病历记录ID）

## 线上病历查看流程

### 第一步：页面加载

进入页面后显示医院名称标题和"电子病历"副标题，然后调用接口获取病历详情数据。

### 第二步：患者基本信息

左上角展示患者头像（根据性别和年龄自动选择：6岁以下显示儿童头像，男性显示男士头像，女性显示女士头像）+ 姓名 + 性别 + 年龄。

信息区展示：

- 问诊单号
- 就诊科室
- 就诊时间
- 身份证号
- 联系方式
- 现居住地

### 第三步：病情信息区（可折叠）

默认展开显示，点击"收起信息"可折叠为固定高度（避免页面过长）。包含以下8项病史信息：

主诉、中医四诊（舌像+脉象）、现病史、既往史、过敏史、个人史、家族史、婚育史、常见体征、治法、辅助检查、病历资料（图片可点击放大预览）、治法。

未填写的字段统一显示"未填写"。

### 第四步：诊疗信息区

展示诊断（西医诊断和中医诊断分类展示：diagnosisType=1为诊断、=2为辨病、=3为辩证）、治疗意见。

### 第五步：医生签名区

- **手写签名（图片型）**：如果签名是base64图片格式，直接显示签名图片
- **文字签名**：如果不是图片格式，显示为带灰色背景的加粗文字

### 第六步：查看原始电子病历

如果有电子病历PDF文件，底部显示"查看原始电子病历"按钮，点击可在线预览PDF文件。

## 病历撤销流程

### 撤销条件（5个条件必须全部满足）

1. 当前用户角色为"医生"
2. 平台开启了严格模式（`strict=true`）
3. 该病历关联的问诊单存在
4. 问诊状态为"已完成"（flowStatus=30）
5. 当前时间在问诊结束后**24小时内**（超时不可操作）

### 撤销操作步骤

1. 点击底部"撤销病历"按钮（红色危险按钮）
2. 底部弹出撤销原因输入面板
3. 输入撤销原因（必填，最多500字）
4. 点击"确认"提交
5. 成功后提示"已撤销"，1.5秒后自动返回上一页

### 编辑操作

如果满足撤销条件，底部同时显示"编辑病历"按钮，点击跳转到病历编辑页面。

## 线下病历查看流程

线下病历（从HIS同步的病历）查看流程独立：

### 差异点

- 无问诊单号（线下就诊）
- 显示医生姓名和就诊科室
- 额外的"体格检查"区块：体温、脉率、呼吸频率、BMI、体重、身高、收缩压/舒张压
- "中医四诊"独立区块
- "辅助检查"独立区块
- "初步诊断"独立区块
- "病状分类"独立区块
- "治疗意见"独立区块
- 无撤销/编辑功能（线下病历不可修改）
- 底部仅"查看原始电子病历"一个按钮

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 缺少病历记录ID | 提示"缺少病历记录ID" |
| 撤销原因未填写 | 提示"请输入撤销原因" |
| 超过问诊结束24小时 | 底部撤销/编辑按钮不显示 |
| 非医生角色 | 底部撤销/编辑按钮不显示 |
| 非严格模式 | 底部撤销/编辑按钮不显示 |
| 原始病历PDF不存在 | 提示"暂无原始电子病历" |

## 关联功能

- **病历编辑**：跳转到病历编辑页（带入问诊单号、病历ID等参数）
- **问诊详情**：从问诊详情页的病历入口进入
- **图片预览**：点击病历资料图片可全屏预览
