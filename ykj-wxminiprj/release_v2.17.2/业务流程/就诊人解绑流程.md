# 就诊人解绑流程

> 文档版本：v2.17.2 | 面向读者：患者

## 概述

患者需要对已绑定的就诊人进行解绑操作（例如更换绑定的身份信息），通过短信验证码验证身份后完成解绑。解绑成功后该就诊人与当前微信账号解除关联，可重新绑定新的身份信息。

## 触发条件

- **前置条件**：用户已登录，就诊人列表中已有关联的就诊人
- **入口路径**：我的 → 就诊人管理 → 选择某就诊人 → 解绑操作（具体入口位置取决于产品配置）

## 步骤详解

### 第一步：进入解绑验证页

用户看到解绑页面，顶部标题「就诊人解绑」。

页面顶部有红色提示区域：带感叹号图标和提示文字「请获取当前绑定手机号的验证码进行解绑，解绑成功后将绑定新的手机号」。

下方是验证表单：

| 区域 | 说明 |
|------|------|
| 手机号 | 自动填充当前就诊人绑定的手机号，**不可修改** |
| 验证码 | 需手动输入，右侧有「获取验证码」按钮 |
| 解绑按钮 | 手机号和验证码均填写后可用，否则置灰（半透明蓝色） |

底部有联系客服入口：「若该手机号已停用/无法接收短信，请联系客服处理」。

### 第二步：获取验证码

点击「获取验证码」按钮，系统发送短信到当前绑定手机号。按钮进入**60秒倒计时**状态，显示「重新发送Xs」，倒计时结束前不可再次点击。倒计时结束后恢复为「获取验证码」可重新发送。

### 第三步：提交解绑

输入收到的验证码后，「解绑」按钮由灰变蓝（可点击）。点击后：
1. 显示加载中
2. 验证码校验通过后，就诊人与当前微信账号解绑
3. 弹出「解绑成功」提示
4. 1.5秒后自动返回上一页

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 未输入验证码 | 「解绑」按钮置灰不可点击 |
| 验证码过期或错误 | 接口返回错误，提示用户重新获取 |
| 手机号已停用无法接收短信 | 用户需点击底部「联系客服」通过微信客服处理 |
| 缺少必要参数（patientId） | 提示「缺少必要参数」，无法获取验证码 |
| 解绑接口失败 | 不弹出成功提示，用户可重试 |

## 关联功能

- **就诊人管理**：解绑后的就诊人在列表中可能消失或变为可重新绑定状态
- **微信客服**：遇到手机号相关问题可联系客服
