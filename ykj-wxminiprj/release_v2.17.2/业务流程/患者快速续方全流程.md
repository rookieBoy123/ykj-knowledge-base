# 患者快速续方全流程

> 文档版本 2026-07-03 | 面向角色：患者

## 概述

快速续方是针对有线上或线下有效处方、正按方吃药且病情稳定的患者提供的便捷续方通道。患者可以从历史处方列表中直接选择处方进行续方，系统自动匹配原处方医生或推荐新医生。

## 触发条件

- **前置条件**：患者已登录，有 6 个月内的有效处方
- **入口路径**：患者端首页 → 快速续方入口

## 步骤详解

### 第一步：进入快速续方页面

页面顶部显示提示说明："快速续方适用有线上或线下有效处方，正按方吃药并且病情稳定的患者"。

**上传处方续方**：如果患者有线下处方，可以点击"+ 上传处方续方"，进入快速问诊流程（serviceType=4）。

### 第二步：查看历史处方列表

页面展示最近 6 个月的有效处方，每个处方卡片包含：
- 处方类型（线上/线下）
- 处方编号
- 患者姓名
- 诊断/辨病辨证信息
- 处方医生

卡片底部有两个按钮："查看详情"和"快速续方"。

### 第三步：点击"快速续方"

系统按以下顺序检查：

1. **检查医生是否开通了快速续方服务**：如果医生未开通，弹出提示"医生未开通快速续方服务，您可以向医生发起问诊咨询购药"。
   - 普通患者：可选择"匹配新医生"或"取消"
   - 特殊患者：可选择"去问诊"（跳转到医生详情页）

2. **检查是否有进行中的问诊**：如果该患者和医生之间已有未完成的快速续方问诊：
   - 有未支付的订单 → 提示"有未支付问诊单，是否前往支付？"，可"去支付"或"取消"
   - 有进行中但已支付的 → 提示"有正在进行的问诊，是否前往？"，可"前往问诊"或"取消"

3. **检查医生是否在线**：
   - 原医生在线 → 直接进入下单流程
   - 原医生不在线 → 弹出"您的医生暂不在线，是否为您匹配新的医生？"
     - 选择"匹配新医生" → 系统按药房匹配新医生
     - 选择"取消" → 留在当前页面

4. **匹配新医生**：
   - 匹配成功 → 自动创建问诊参数并跳转到下单页面
   - 匹配失败 → 弹出"暂无医生在线，请稍后再试"

### 第四步：下单创建问诊

进入下单页面后走标准问诊下单流程，serviceType=5（快速续方）。下单时携带的续方处方编号会关联到问诊中。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 无处方记录 | 显示空状态"暂无处方记录"，普通用户提示"可选择上传线下处方进行续方" |
| 医生未开通快速续方 | 弹出提示，可选择匹配新医生或转去问诊 |
| 原医生不在线 | 弹出提示，可选择匹配新医生或取消 |
| 无可匹配的医生 | 弹出"暂无医生在线，请稍后再试" |
| 有未完成问诊 | 提示前往支付或继续问诊 |
| 处方来源为商城（serviceType=8） | "快速续方"按钮不显示 |
