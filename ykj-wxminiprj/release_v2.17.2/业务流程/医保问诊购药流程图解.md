# 医保问诊购药流程图解

> 文档版本：release/v2.17.2 | 面向读者：客服、运营、产品经理等非技术人员

## 概述

平台通过图文并茂的方式向患者展示医保问诊和购药的完整流程，帮助患者理解如何在平台使用医保。页面分为「普通医保」和「慢病医保」两条路径，各有独立的步骤说明。

---

## 触发条件

- **入口**：从医生详情页或问诊流程中的医保说明链接进入
- **前置条件**：无特殊限制，任何患者都可以查看此流程说明页（纯静态展示，不涉及数据加载）

---

## 页面结构

### 顶部信息区

- **标题栏**：「医保问诊购药流程」，透明背景
- **医院名称展示**：从系统全局配置中读取并展示「【XX医院】医院依托实体YY机构」
- **说明文字**：
  - 普通医保：符合条件的广州参保人需先在YY机构定点
  - 慢病医保：需先完成线下慢病备案
- **激活指引**：绿底区域提示「暂仅支持微信认证本人使用医保业务服务」，指导用户进入「我的医保凭证」小程序激活电子医保卡

### 路径一：普通医保（5步）

| 步骤 | 操作描述 | 配图说明 |
|------|---------|---------|
| 第一步 | 在「名医预约」医生列表或「推荐医生」中选择有「可医保」标记的医生 | 展示筛选医生的界面截图 |
| 第二步 | 费别选择「医保」，确定定点为指定机构（若不定点可选择自费） | 展示费别选择界面 |
| 第三步 | 发起问诊，医生开方后点击付款 | 展示付款按钮界面 |
| 第四步 | 支付处方，确认报销金额（展示医保结算明细） | 展示费用明细页面 |
| 第五步 | 查看报销明细 | 展示报销明细页 |

### 路径二：慢病医保（6步）

多了一个前置步骤「选择慢病医保复诊入口」，后面的步骤与普通医保类似：

| 步骤 | 操作描述 |
|------|---------|
| 第一步 | 选择「慢病医保」复诊入口 |
| 第二步 | 添加慢病备案患者，选择问诊病种 |
| 第三步 | 选择支持该慢病病种的医生，发起问诊 |
| 第四步 | 发起问诊，医生开方后点击付款 |
| 第五步 | 支付处方，确认报销金额 |
| 第六步 | 查看报销明细 |

---

## 关联功能

- **门慢医保复诊**：患者通过hisForm页面选择病种后搜索可医保医生
- **医保支付**：hisPrePay页面实际操作医保结算
- **报销明细**：claimDetail页面查看报销详情
