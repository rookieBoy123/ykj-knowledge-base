# AI医生分身设置与管理流程

> 文档版本 v1.0 | 面向读者：医生用户

## 概述

医生为AI分身设置问诊价格，查看团队医生列表，以及查看AI分身的历史问诊记录。

## 触发条件

- 前置条件：医生已登录且已配置AI医生分身
- 入口路径：医生端"我的" → AI医生分身列表 → 点击某个AI分身 → 进入设置页

## 步骤详解

### 第一步：查看AI分身信息

页面加载时根据URL参数中的AI医生ID获取详情：
- 当前问诊价格（红色显示，格式：XX元/次）
- 价格设置范围提示（可设置范围：0~XX元）
- 团队医生列表（显示该AI分身关联的真人医生）

### 第二步：设置问诊价格

点击价格区域 → 底部弹出价格设置面板：
- 输入框：支持数字输入（包括整数或最多两位小数）
- 快捷价格选项：10元、20元、40元、60元（点击即填入）
- 确认按钮

**价格校验规则**：
- 不能为空："请输入价格"
- 必须是数字或最多两位小数："请输入有效的价格，可输入整数或最多两位小数"
- 不能超过最大价格："超出最大价格XX元"

设置成功后刷新AI分身详情。

### 第三步：查看问诊记录

点击"查看问诊记录" → 跳转到AI问诊记录列表页，可查看该AI分身处理过的所有问诊记录。

### 第四步：查看团队医生

页面直接展示团队医生列表，每位医生显示基本信息卡片（姓名、科室、职称等）。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 缺少AI医生ID | 提示"缺少医生ID" |
| 价格为空 | 提示"请输入价格" |
| 价格格式不正确 | 提示"请输入有效的价格，可输入整数或最多两位小数" |
| 价格超出最大限制 | 提示"超出最大价格XX元" |
| 团队医生为空 | 不显示医生列表 |

## 关键规则

- **价格上限**：最大价格由后台配置（cmConfig.aiDoctor.maximumPrice），不同医院可配置不同上限
- **价格精度**：支持整数和最多两位小数（如 10.50 元）
- **快捷价格**：固定为10/20/40/60元四个选项，方便医生快速设置
- **团队医生**：显示的是该AI分身关联的真人医生列表，AI问诊时可能会转接给这些医生

## 关联功能

- AI分身问诊对话（患者与AI聊天）
- AI问诊记录列表
- 医生信息管理
