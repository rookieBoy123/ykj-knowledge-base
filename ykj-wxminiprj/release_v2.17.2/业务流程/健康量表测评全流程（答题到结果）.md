# 健康量表测评全流程（答题到结果）

> 文档版本：release/v2.17.2 | 面向读者：客服、运营、产品经理

## 概述

患者选择一个健康量表（如心理测评、症状自评等），逐题作答完成后提交，系统自动计算得分并展示测评结果（含总分进度图、维度雷达图、评测解读），还可以查看答题明细、得分趋势图、咨询医生。

## 触发条件

- 在"量表测评"列表中选择一个量表，点击进入详情页
- 在"我的测评"记录中点击某条历史记录继续答题
- 医生端从患者健康档案的"量表记录"进入

## 前置条件

- 患者需选择一名就诊人（patientId），如果从问诊流程中进入则自动取当前就诊人
- 如果是医生端，需要携带参数 `patientId` 指定测评患者
- 量表详情需要有效的 `id`（量表ID）

## 步骤详解

### 第一步：量表详情页（了解量表信息）

进入量表详情页后，展示以下内容：

- **量表基本信息**：量表图标、名称、简介、题目总数
- **患者选择区**（患者端）：如果未指定就诊人，页面底部显示就诊人选择组件，要求先选择测评就诊人；如果已指定，显示就诊人姓名和信息
- **医生端**：额外展示"问卷列表"，可以预览量表中所有题目
- **底部按钮**：
  - 患者端：显示"开始测评"绿色按钮
  - 如果患者未选择就诊人，按钮不显示，需先添加测评就诊人

点击"开始测评"前，系统会自动创建一个测评记录（`addTestRecord`），获取本次测评的 `recordId` 和第一题ID。

### 第二步：答题页（逐题作答）

跳转到答题页后，页面顶部显示**答题进度条**（当前题号/总题数，百分比进度）。

**题目类型与答题规则**：

| 题型 | 规则 | 操作方式 |
|------|------|---------|
| **单选题**（type=0） | 点击选项自动提交答案，自动跳到下一题（有500毫秒延迟让患者看清已选） | 点击选项即可 |
| **多选题**（type=1） | 点击选项记录选择（可多选/取消），需手动点击"下一题"才提交 | 点击选项选中→点击"下一题"提交 |

**翻题操作**：

- **上一题**：如果当前题不是第一题（有 `previousId`），显示"上一题"按钮，点击查看已答过的题目，可以修改答案
- **下一题**：多选时需选中至少一个选项，"下一题"按钮才变为可点击（蓝色）。
- **最后一题**：进度条显示第N/N题，此时"下一题"按钮消失，底部显示"提交"按钮。单选和多选都需要点击"提交"才最终提交。

**答案提交类型**：

- **首次答题（add）**：之前该题没有答案，走新增提交接口
- **修改答案（edit）**：之前该题已有答案（患者从"上一题"返回修改），走编辑接口，提交时需要携带 `answerId`

### 第三步：测评结果页

答完所有题目并点击"提交"后，跳转到测评结果页。页面分为以下区域：

**1. 得分展示区**：
- 圆弧进度图（半圆环状），直观展示得分占比
- 中间大字显示总分："XX分"
- 下方文字显示结果等级（如"轻度焦虑"、"正常"等）

**2. 维度雷达图**（当维度≥3个时显示）：
- 各维度评分雷达图，展示在不同维度上的得分情况
- 维度名称超过10字自动截断

**3. 评测解读区**：
- 展示量表的详细解读文本。如果该量表未配置解读内容，显示"暂无测评信息"
- 底部免责声明："该测试结果仅作为评估工具，不能以此作为诊断。若对评测结果有疑问，可咨询医生。"

**4. 测评信息区**：
- 测评人姓名、性别、年龄
- 测评时间
- 如果本次测评关联了某次问诊（`inquiryNo` 存在），额外显示问诊医生和问诊时间

**5. 操作按钮**：

- **查看题目明细**：跳转到答题详情页，查看每道题的题目、选项和患者答案
- **咨询医生**（仅当测评不关联问诊时显示）：跳转到专家搜索页，可按疾病查找医生进行咨询
- **问诊对话记录**（仅当关联问诊时显示）：点击可进入对应的问诊聊天室，查看与医生的对话

### 第四步：趋势图（多次测评对比）

从量表测评记录页的"趋势图"入口进入，通过事件通道（eventChannel）接收参数：

- 顶部展示量表图标和名称
- 四个预设时间段切换：近一周、近一月、近三月、近半年
- 折线图展示各次测评的总分变化趋势
- X轴为测评次数序号，Y轴为总分
- 点击不同时间段重新请求数据并刷新图表

### 第五步：答题详情（回顾答案）

从测评结果页点击"查看题目明细"进入，展示：
- 量表基本信息（与详情页相同）
- 问题列表：每道题显示题目内容和患者的选择（选中项高亮）
- 患者端额外区分动态样式（主题色）

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未选就诊人就点击"开始测评" | 提示"请先添加测评就诊人" |
| 缺少必要参数（recordId/currentQuestionId/scaleId/patientId） | 提示"缺少必要参数" |
| 最后一题未选择答案点"提交" | 提示"请答完题目再提交" |
| 测评结果中维度数<3个 | 不展示雷达图 |
| 测评结果无解读内容 | 显示"暂无测评信息" |
| 评测量表未配置解读 | 底部仍显示免责声明 |

- **专家搜索/咨询医生**（跳转目标）：`/packagePatient1/pages/specialsearchdoctor/index`
- **问诊聊天室**（关联问诊时）：`/timIM/pages/TUI-Chat/chat`
- **测评趋势图**（历史对比）：`/packagePatient2/pages/scale/trendChart/index`
- **测评记录列表**（医生端查看）：`/packagePatient2/pages/scale/testRecord/index`
