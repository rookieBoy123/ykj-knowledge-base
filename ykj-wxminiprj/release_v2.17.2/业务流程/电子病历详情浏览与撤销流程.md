# 电子病历详情浏览与撤销流程

> 文档版本 v1.0 | 面向读者：产品运营、医生用户

## 概述

医生在问诊结束后查看患者的电子病历详情，包括患者基本信息、病情信息（主诉、现病史、既往史、过敏史等）和诊疗信息（诊断、辨病、辨证、治疗意见）。在问诊结束后24小时内，医生可以撤销或编辑病历。

## 触发条件

- 前置条件：医生已完成问诊，产生了病历记录
- 入口路径：从问诊对话或病历列表中点击病历记录进入

## 步骤详解

### 第一步：查看病历基本信息

页面顶部展示互联网医院标题和"电子病历"副标题。核心信息区包含：
- 问诊单号（右上角显示）
- 患者头像（根据性别和年龄自动选择男/女/儿童头像）
- 患者姓名、性别、年龄标签
- 就诊科室、就诊时间
- 身份证号、联系方式、现居住地

### 第二步：浏览病情信息

"病情信息"区域展示完整的病历记录：
- 主诉
- 中医四诊（舌像、脉象，仅中医科显示）
- 现病史、既往史、过敏史
- 个人史、家族史、婚育史
- 常见体征
- 治法（仅中医科显示）
- 辅助检查
- 病历资料（患者上传的图片，可点击放大预览）

病情信息默认**折叠展示**（高度限制300rpx），点击"展开全部"可查看完整内容。

### 第三步：浏览诊疗信息

"诊疗信息"区域展示医生的诊断结果：
- 诊断（西医诊断名称）
- 辨病（中医辨病，仅中医科显示）
- 辩证（中医辩证，仅中医科显示）
- 治疗意见（非中医科显示）

### 第四步：查看医生签名

页面底部展示医生的签名，可能是签名图片或文字形式。

### 第五步：查看原始电子病历

如果病历生成了PDF文件（medicalRecordUrl不为空），页面底部会显示"查看原始电子病历"按钮，点击可预览原始电子病历文件。

### 第六步：撤销或编辑病历（限时操作）

在**严格模式**下，当医生身份且问诊状态为"已结束"（flowStatus=30），且距离问诊结束时间不超过24小时时，页面底部显示两个操作按钮：
- **撤销病历**：点击弹出撤销原因输入框，必填撤销原因（最多500字），确认后病历状态变为"已撤销"
- **编辑病历**：点击跳转到病历编辑页面，可修改病历内容

超过24小时后，撤销和编辑按钮不再显示。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 缺少病历记录ID | 弹出提示"缺少病历记录ID"并阻止加载 |
| 问诊结束超过24小时 | 撤销和编辑按钮隐藏，医生无法操作 |
| 非严格模式 | 撤销和编辑按钮始终隐藏 |
| 撤销原因未填写 | 弹出提示"请输入撤销原因"，阻止提交 |
| 病历数据中某字段为空 | 显示"未填写"或"暂无"等默认文案 |
| 医生签名是图片格式 | 以缩略图形式展示签名 |
| 医生签名是文字格式 | 以灰色背景标签展示医生姓名 |

## 关联功能

- 病历编辑 → 点击"编辑病历"跳转至病历编辑页
- 问诊对话 → 返回上一页回到问诊对话
- 原始电子病历 → 调用文件预览功能查看PDF
