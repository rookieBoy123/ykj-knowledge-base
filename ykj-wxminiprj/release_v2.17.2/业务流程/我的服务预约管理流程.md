# 我的服务预约管理流程

> 文档版本 2026-07-01 | 面向读者：患者

## 概述

患者可以在"我的预约"中查看自己所有线下医疗服务（体检、核酸、健康证、检查）的预约记录，按状态筛选，并查看每个预约的详细信息（含核销码、取消预约等操作）。

## 触发条件

- 从"我的"页面点击"我的预约"入口进入
- 预约状态发生变化后自动刷新

## 步骤详解

### 第一步：进入预约列表

页面默认显示"全部"预约记录，顶部有5个状态选项卡可切换：

| 选项卡 | 说明 |
|--------|------|
| 全部 | 显示所有状态的预约 |
| 待支付 | 已生成预约但未付款 |
| 待核销 | 已付款，到院时出示核销码 |
| 已核销 | 已到院完成服务 |
| 已取消 | 已取消的预约（含退款） |

### 第二步：浏览预约卡片

每条预约记录显示为一张卡片，包含服务名称、预约信息摘要和当前状态。列表支持下拉滚动到底部自动加载更多（每页10条）。

### 第三步：查看预约详情

点击任意预约卡片进入详情页，可查看：
- 完整的服务信息
- 预约时间和地点
- 支付金额和支付状态
- 待核销状态下的条形码（512×134像素，供工作人员扫描核销）
- 取消预约按钮（待支付/待核销状态下可见）

### 第四步：取消预约

在详情页点击取消按钮，系统弹出二次确认提示。已付款的预约会提示"退款将在48小时内到账"。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 没有任何预约记录 | 显示空状态插图+"暂无记录" |
| 某个状态选项卡下无数据 | 显示空列表 |
| 取消预约时网络异常 | 提示取消失败，保留当前状态 |
| 多次快速点击取消按钮 | 防重复点击保护 |

## 关联功能

- 跳转到：预约详情页（含核销码和取消操作）
- 跳转到：服务详情页（查看服务介绍）
- 关联：服务预约下单流程
