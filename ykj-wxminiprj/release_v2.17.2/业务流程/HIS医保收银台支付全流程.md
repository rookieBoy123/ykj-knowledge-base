# HIS医保收银台支付全流程

> 文档版本 v1.0 | 面向读者：产品、运营、客服

## 概述

用户在药品订单确认后进入收银台页面，可选择微信支付或医保支付完成付款。选择医保支付时需要先经过医保电子凭证授权，然后在医保预结算页面确认费用明细（医保基金支付、个人账户支付、现金支付三部分），最后调起医保小程序完成支付。

## 触发条件

- **前置条件**：用户已在药品订单页点击"去支付"，订单已成功创建
- **入口路径**：药品订单详情 → 点击"去支付" → 自动跳转到收银台

## 步骤详解

### 第一步：进入收银台

用户进入收银台页面后看到：
- 顶部显示待支付金额（大号红色字体）
- 支付剩余时间倒计时（时:分:秒），超时后订单自动失效
- 支持的支付方式列表（单选）：
  - **微信支付**（默认选中）
  - **医保支付**（需满足条件才可用）

每种支付方式旁边有圆形选择框，选中的显示蓝色实心圆。不可用的支付方式显示灰色禁用图标，下方提示不可用原因。

### 第二步：选择支付方式

用户点击某个支付方式切换选择：

- **选微信支付**：直接跳到第四步"确认支付"
- **选医保支付**：系统自动执行以下检查：
  1. 检查是否有患者ID（无则提示"缺少患者id，请联系客服"并自动切回微信支付）
  2. 判断是否为慢病订单（慢病订单跳过定点检查）
  3. 非慢病：查询该就诊人是否已医保定点
     - 已定点 → 允许使用医保支付
     - 未定点 → 弹出提示"该就诊人暂未定点[医保名称]，定点后才可使用医保支付"
       - 点击"确认定点" → 调起定点接口
       - 点击"暂不定点" → 自动切回微信支付

### 第三步（医保支付）：医保电子凭证授权

用户选中医保并确认定点后，点击"确认支付"：

1. 系统先请求订阅消息授权（用于后续推送支付结果通知）
2. 跳转到医保官方小程序进行电子凭证身份授权
3. 授权结果：
   - **授权成功**（retCode=0）→ 携带授权码（authCode）进入医保预结算页面
   - **取消授权**（retCode=503608）→ 弹出提示"您已经取消医保电子凭证身份授权，医保移动支付已中止"

### 第四步（医保支付）：医保预结算确认

进入医保预结算页面，用户看到：

**费用明细区**：
| 费用项目 | 说明 |
|---------|------|
| 费用总额 | 处方全部药品原价 |
| 医保基金支付 | 医保统筹基金报销金额 |
| 个人帐户支付 | 医保个人账户支付（可开关） |
| 其他抵扣金额 | 其他优惠/抵扣 |
| 现金支付 | 用户最终需自付金额 |

**个人账户支付开关**（如医保支持个人账户）：
- 显示"使用"/"不使用"两个按钮，当前选中高亮
- 切换开关后需要重新获取医保授权码（跳转医保小程序重新授权）
- 切换后费用明细实时更新

**处方明细**：
- 点击"查看明细"展开底部面板
- 显示每张处方的药品名称、数量、单价、金额
- 金额为自费部分时会高亮显示

底部显示"医保移动支付"标识。

### 第五步：确认支付

页面底部显示"您还需支付：¥XX.XX"和"去支付"按钮。

点击"去支付"后：
1. 页面显示加载动画
2. 调起医保支付小程序完成支付
3. 支付小程序返回后，系统查询订单状态确认支付结果

### 第六步：支付结果处理

**支付成功**：
- 自动跳转到药品订单详情页

**支付状态未知**（未查询到支付状态）：
- 弹出提示框"未查询到支付状态，请根据您的情况点击下面的按钮"
- 提供两个选项：
  - "支付未成功"→ 关闭弹窗，留在当前页可重试
  - "支付已成功"→ 再次确认订单状态，已支付则跳转订单详情

**支付超时**（倒计时归零）：
- 弹出提示"订单支付超时，请重新发起"
- 点击"知道了"后返回上一页

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 缺少患者ID | 提示"缺少患者id，请联系客服"，自动切回微信支付 |
| 医保参数配置缺失 | 弹出提示"医保参数有误，是否重新获取？"，用户确认后重新拉取配置 |
| 未获取到授权码 | 提示"未获取到授权码，请重新医保电子凭证身份授权" |
| 取消医保授权 | 提示"您已经取消医保电子凭证身份授权，医保移动支付已中止"，返回收银台重新选择 |
| 医生非医保医师 | 提示"当前医生非医保医师，不支持医保支付，只能自费" |
| 就诊人未定点 | 提示"该就诊人暂未定点[医保名称]，定点后才可使用医保支付" |
| 定点失败 | 自动切回微信支付 |
| 医保信息查询失败 | 提示具体错误信息，返回收银台 |
| 微信扫码支付 | 小程序收银台不开放微信扫码支付（channel=4的支付方式被过滤掉） |

## 关联功能

- 药品订单详情（支付完成后跳转）
- 处方详情（支付失败/超时返回）
- 就诊人管理（医保定点）
- 医保移动支付小程序（外部跳转）
