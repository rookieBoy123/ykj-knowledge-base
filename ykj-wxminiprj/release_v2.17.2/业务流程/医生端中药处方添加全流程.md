# 医生端中药处方添加全流程

> 文档版本 v1.0 | 面向读者：产品经理、运营人员

## 概述
医生在开处方时，为处方添加中药品种。支持拼音首字母搜索药品、自定义克数键盘输入、煎煮方法选择、历史用药快捷导入、快速录方（文字解析）和图片识别录方，系统自动校验配伍禁忌和超量问题。

## 触发条件
- **前置条件**：医生已选择药房（需已授权）、进入处方开具页面
- **入口路径**：处方开具页 → 点击添加中药 → 进入本页面
- **页面接收参数**：药房编号、问诊单号（用于图片识别场景）

## 步骤详解

### 第一步：搜索添加药品（拼音输入）
医生在页面顶部看到已选药品区（显示已选X味 + 总克数g + 总价¥），下方是药品输入框，点击后弹出专用键盘。

- **两种输入方式**：页面提供"快速录方"和"专用键盘"两种模式，默认使用专用键盘
- **拼音搜索**：通过键盘输入药材名称拼音首字母（如输入"DG"搜索"当归"），系统实时展示匹配的药材列表，每条显示药材名和单价
- **选择药材**：点击搜索结果的药材卡片，自动添加到已选列表中
- **滚动加载**：搜索结果支持横向滚动查看更多匹配药材

### 第二步：调整用量（数字键盘）
每个已添加的药材下方显示用量输入框，点击可弹出数字键盘：

- **颗粒转换提示**：如果该药材的库存单位与用量单位不一致（如库存按袋、每袋含3克），页面顶部会显示黄色提示条，说明换算关系，建议输入库存单位的整数倍
- **常用剂量按钮**：键盘上方显示预设剂量按钮（根据字典配置的常用剂量，如5g、10g、15g、20g、25g），医生可一键填入
- **增减按钮**：每种药材用量旁有"－""＋"按钮，可快速增减
- **克数精度**：支持小数点后两位，系统自动计算总克数和总价
- **单位显示**：每种药材显示其单位（克/袋等）

### 第三步：选择煎煮方法
每种药材可单独设置煎煮方法：

- 点击药材的煎煮方法选择器，弹出底部弹窗
- 可选煎煮法包括：先煎、后下、包煎、另煎、烊化、冲服等（从字典动态加载）
- 选中后点击"保存"按钮确认
- 未选择煎煮方法的药材显示"请选择"

### 第四步：历史用药快捷导入
当输入框为空时，键盘区域上方展示"历史用药"列表：

- 默认折叠模式：展示前5个历史药材（可点击展开查看全部）
- 展开模式：显示全部历史药材 + "全部删除"按钮 + 收起按钮
- 点击历史药材名称直接添加到已选列表
- 历史数据按药房存储，不同药房的历史独立

### 第五步：快速录方（文字解析）
切换到"快速录方"模式后：

- **格式要求**：按"药材名 + 克数 + 煎法"格式输入，多药材用逗号或空格隔开
- **示例**："石莲子15先煎，黄荟10，地骨皮10"
- 点击"确认录入"，系统自动解析文本、匹配药房药材并添加到已选列表
- **重复检测**：如果录入的药材与已有药材重复，提示"XX与原有药材重复"
- **药房无此药**：解析出的药材在该药房不存在时，标记为"药房无此药"（灰色显示）

### 第六步：AI图片识别录方
在快速录方模式下，如果启用了AI功能，显示"图片识别"按钮：

- **识别方式选择**：
  - 拍照：调用相机拍摄纸质处方
  - 相册：从相册选择处方图片
  - 病历资料：从当前问诊的病历图片中选择
- 识别成功后，解析出的药材文本自动追加到快速录方输入框末尾

### 第七步：药品管理操作
在已选药品列表中：

- **删除药材**：点击药材左侧红色删除按钮
- **单排/双排切换**：点击"双排"/"单排"按钮切换视图密度
- **清空全部**：点击"清空"按钮，二次确认后移除所有已选药材
- **超量提示**：如果药材用量超过规定上限，显示红色"超量"标签，点击可查看上限说明

### 第八步：提交（配伍校验）
点击"添加完成"按钮时（按钮在未选药材时禁用），系统进行配伍校验：

- **十八反十九畏检查**：检测处方中是否存在配伍禁忌组合
- **有毒中药剂量检查**：检测有毒药材是否超出安全剂量
- **配伍弹窗**：如有问题，弹出校验弹窗，列出具体的冲突项，医生可选择：
  - 确认继续（忽略警告）
  - 返回修改

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药房无该药材 | 药材卡片显示"药房无此药"标记，无法正常使用 |
| 快速录方药材重复 | 弹窗提示"XX与原有药材重复"，跳过该药材 |
| 快速录方药材不存在 | 药材标记为"inValid"，红色显示 |
| 配伍禁忌（十八反十九畏） | 弹窗展示冲突详情，医生可确认继续或返回修改 |
| 有毒药材超量 | 弹窗展示超量详情 |
| 颗粒库存单位不匹配 | 页面顶部黄色提示条说明换算关系 |
| 清空全部 | 二次确认弹窗："确定清空药品吗？确定后，所有选择的药品将被移除" |

## 关联功能
- 处方开具（选择完成后返回处方页面继续填写）
- 药品详情（点击药材可跳转查看详情）
- 快速开方（共享同一药材添加组件）
- AI图片识别（跳转到专门的识别页面）
