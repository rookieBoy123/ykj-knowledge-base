# 就诊人信息管理流程

> 文档版本：release/v2.17.2 | 面向读者：运营、客服 | 最后更新：2026-06-29

---

## 概述

患者可添加/编辑就诊人信息，包括基本信息、过敏史、现病史、既往史。支持绑定监护人（适用于未成年就诊人）。

---

## 一、触发条件

- 首次问诊前需要创建就诊人
- 为家人添加就诊人信息
- 更新已有就诊人的病史

## 二、前置条件

- 用户已登录

---

## 三、步骤详解

### 步骤1：进入就诊人信息页面

**页面**：`healthRecords/add.vue`（套餐患者2分包），页面标题：「就诊人信息」。

### 步骤2：填写基本信息

| 字段 | 说明 | 必填 |
|-----|------|------|
| 姓名 | 就诊人真实姓名 | 是 |
| 性别 | 男(1)/女(2)，默认男 | 可选 |
| 身份证 | 用于年龄推算 | 是（通过birthday校验） |
| 出生日期 | 从身份证解析 | 是 |
| 有无监护人 | 0=无/1=有 | 可选 |
| 监护人姓名 | 当hadGuardian=1时必填 | 条件必填 |
| 监护人身份证 | 当hadGuardian=1时必填 | 条件必填 |

### 步骤3：填写病史信息

每项病史三步：选择「有/无」→ 有则填写详情 → 确认。

| 病史类型 | 字段 | 说明 |
|---------|------|------|
| 过敏史 | hadAllergy/allergy | 过敏源信息 |
| 现病史 | hadMedicalHistory/medicalHistory | 当前疾病 |
| 既往史 | hadPreviousHistory/previousHistory | 过往病史 |

### 步骤4：保存

点击「保存」按钮，校验规则：

| 校验规则 | 提示 |
|---------|------|
| 姓名为空 | 请输入就诊人姓名 |
| 身份证为空 | 请输入身份证 |
| 有监护人但姓名空 | 请输入监护人姓名 |
| 有监护人但身份证空 | 请输入监护人身份证 |
| 有过敏史但空 | 请输入过敏史 |
| 有现病史但空 | 请输入现病史 |
| 有既往史但空 | 请输入既往史 |

校验通过后显示"实名信息认证中，请稍等"，调用接口提交。保存成功自动返回。

**注意**：如果选择「无监护人」，提交时会自动清空监护人信息不传给后端。

---

## 四、关联功能

- **健康档案**：查看就诊人的全部病历
- **在线问诊**：发起问诊需选择就诊人
- **量表测评**：测评前需选择就诊人
- **药品下单**：购药需确认就诊人身份
