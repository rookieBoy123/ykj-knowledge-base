# 医生视频浏览流程（患者端）

> 文档版本 2026-07-01 | 面向读者：患者

## 概述

患者可以在小程序的"医生视频"专区中，像刷短视频一样上下滑动观看医生发布的科普视频。每个视频下方展示医生信息卡片，可直接点击"去问诊"进入该医生的服务选购页。

## 触发条件

- 从首页"医生视频"模块点击进入
- 通过分享链接进入指定视频
- 从医生详情页点击视频进入

## 步骤详解

### 第一步：选择视频分类

页面顶部显示横向可滚动的视频分类标签（如"全部""中医养生""慢病管理"等），点击切换分类。首个标签为"全部"，分类数据由后台动态配置。当前选中的分类下方有高亮下划线标识。

### 第二步：上下滑动观看

页面展示竖屏全屏视频列表，支持手指上下滑动切换视频（类似抖音/快手模式）：
- 上滑 = 下一个视频
- 下滑 = 上一个视频
- 当前视频自动播放，上一个视频自动暂停
- 点击视频画面可暂停/继续播放
- 视频支持循环播放
- 每次播放自动记录观看行为（用于数据统计）

### 第三步：查看医生信息

每个视频下方悬浮一张半透明背景的医生信息卡片（仅当视频关联了医生时显示），包含：
- 医生头像（圆形）
- 医生姓名 + 职称
- 所属科室 + 医院名称
- "去问诊"按钮（仅医生状态为"已启用"时显示，绿色主题色）

### 第四步：去问诊

点击"去问诊"按钮直接跳转到该医生的服务选购页面，可选择图文问诊、视频问诊等服务类型并下单。

### 第五步：分享

点击右上角可分享当前正在观看的视频。分享标题为视频名称，好友点击后直接定位到该视频。

## 特殊场景

- **分享进入指定视频**：如果通过分享链接进入且携带了视频ID（id参数），系统会将该视频排到列表第一位，其他视频按原顺序排列。如果该视频已不存在（被删除），1.5秒后弹出提示"当前视频已找不到！"并自动切换到"全部"分类。
- **分类为空**：当某个分类下没有任何视频时，显示空状态插图+"暂无视频"+"请切换其他分类观看"
- **视频未关联医生**：不显示医生信息卡片，仅显示视频标题

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 分享的视频已被删除 | 弹出"当前视频已找不到！"提示，1.5秒后切换到"全部"分类 |
| 分类下无视频 | 显示空状态"暂无视频，请切换其他分类观看" |
| 医生已停用 | 不显示"去问诊"按钮 |
| 视频加载失败 | 显示视频播放器自带的错误状态 |

## 关联功能

- 关联：首页医生视频模块
