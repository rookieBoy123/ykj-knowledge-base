# 医生详情页选择服务与下单流程

> 文档版本：release/v2.17.2 | 面向读者：客服、运营、产品经理

## 概述

患者从医生列表点击进入医生详情页，浏览医生的基本信息、服务项目、患者评价后，选择适合自己的问诊服务类型发起问诊或预约。这是患者从"找医生"到"发起问诊"的关键中间环节。

## 触发条件

- 患者在首页/搜索/推荐医生列表点击某位医生
- 患者在"我的关注"/"问诊过的医生"点击某位医生
- 从AI医生详情页点击"查看专家主页"跳转真人医生详情
- 从门慢医保搜索医生列表点击某位医生
- 从快速续方页面匹配到的在线医生列表点击进入

## 前置条件

- 患者必须已登录（未登录时点击服务会引导登录）
- 需要携带医生的唯一标识（doctorId或uniqueId）
- 如果是线上复诊模式，需传入 `isOnlineFollowUp=true` 标识

## 步骤详解

### 第一步：查看医生详情页

进入医生详情页后，页面从上到下分为以下几个区域：

**医生基础信息区**（最顶部）：
- 医生头像、姓名、职称、科室
- 所在医院和擅长领域
- 医生简介（支持展开/收起，默认展示3行）

**专家团队与AI分身区**：
- 如果医生有专家团队，展示"专家团队问诊"入口，点击跳转专家团队详情
- 如果医生有AI数字分身，展示"AI分身问诊"入口，点击发起AI问诊

**问诊服务区**（或预约服务区）：
- 如果医生开通了预约服务（`consultationType=1` 且系统开关 `registrationEnabled` 开启），展示为**预约服务区**：列出医生的门诊排班时间段，患者可选时间段预约
- 如果医生是普通问诊模式，展示为**问诊服务区**：列出医生已开通的服务类型（图文问诊、视频问诊、快速续方等），每项显示服务名称、价格、服务时长、服务说明入口

**患者评价区**（底部）：
- 评价标签汇总（如"耐心(12)"、"专业(8)"），数字表示该标签被提及次数
- 具体评价列表，每条展示星级、评价内容、评价人、时间
- 评价列表支持滚动加载（每页10条），滚动到底自动加载更多
- 如果该医生尚无评价，显示空状态："还没有评价呢"

### 第二步：选择服务并下单

**普通问诊模式**（点击问诊服务区的某个服务）：

1. 系统先检查患者是否已登录，未登录弹出登录引导
2. 对于预约服务，检查所选号段是否可用（调用号段校验接口），不可用时提示"当前号段不可用"
3. 系统显示加载中，获取医生的在线状态：
   - **医生在线**（`onlineStatus=1`）或**有医助协助**（`isDoctorAssisted=true`）：直接跳转到"填写病情信息"页面，开始问诊流程
   - **医生离线**：弹窗提示"医生当前时间不在线，是否继续问诊，等待医生上线后接诊？"，并显示超时说明（如"注：如医生在X小时后无接诊将自动取消并退款"）
     - 点击"取消"：关闭弹窗，留在当前页面
     - 点击"继续问诊"：仍然跳转到填写病情信息页

**预约模式**（点击某排班时间段）：

1. 同样先检查登录状态
2. 检查号段可用性
3. 获取医生在线状态检查（与普通模式逻辑相同）
4. 跳转到填写病情信息页时，会携带 `scheduleItemNo`（排班号段编号）参数

### 第三步：查看服务说明

每个问诊服务旁边有"服务说明"入口（问号图标），点击后：
1. 系统获取该服务的协议说明内容（通过协议编码查询）
2. 弹出居中弹窗，滚动展示服务说明的富文本内容
3. 点击"我知道了"关闭弹窗

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未登录就点击服务 | 弹出登录引导弹窗，点击"确定"跳转登录页 |
| 预约号段不可用 | 提示"当前号段不可用"，不继续下单 |
| 获取医生状态失败 | 提示"获取医生状态失败，请稍后重试" |
| 医生离线 | 弹窗告知离线，患者可选择继续等待或取消 |
| 医生在线但有超时风险 | 弹窗显示超时小时数，提醒患者注意 |
| 评价列表为空 | 显示空状态占位图"还没有评价呢" |

## 关联功能

- **填写病情信息页**（跳转目标）：`/packagePatient1/pages/medicalRecord/addRecord/index`
- **专家团队详情**（如有）：`/packagePatient2/pages/expertTeamDetail/index`
- **AI分身问诊**（如有）：跳转AI对话页
- **医生搜索/列表**（返回来源）
