# 医生收益结算流程

## 触发条件
医生登录后从个人中心或底部导航进入「收益指数」页面，查看自己的问诊/开方收入和结算记录。

## 前置条件
- 当前角色为「医生」
- 医生信息（doctorInfo）已存储在本地

## 步骤详解

### 1. 进入收益指数页
页面顶部展示「我的银行卡」卡片区域：
- **「我的银行卡」入口**（右上角）：跳转到银行卡绑定页面
- **「我的指数」**：显示当前余额（`remainingSum`，单位为元），默认显示「0.00」
- 卡片有渐变蓝色背景和阴影效果

### 2. 切换收益/结算记录
页面有两个选项卡：
- **收益记录**：记录每一笔问诊和开方的收入（金额显示为蓝色正值）
- **结算记录**：记录平台已结算的金额（金额显示为红色，表示支出/结算方向）

### 3. 查看收益记录

**时间筛选**：
- 收益记录模式下，顶部出现月份选择器（picker mode="date" fields="month"）
- 默认显示当前月份（如「2024-06」）
- 切换月份后自动重新查询

**收入统计**：
- 月份选择器右侧显示该月的收入小计：`收入 ¥XXX.XX`

**收益明细列表**：
- 每条记录展示：收入描述（如「图文问诊」「处方开方」）、创建时间、金额
- 金额以蓝色字体显示
- 列表支持上拉加载更多（每页10条）
- 无数据时显示空状态插画和「暂无收益记录」

### 4. 查看结算记录

- 切换到「结算记录」选项卡
- 展示平台对医生的结算记录
- 金额以红色字体显示
- 同样支持分页加载
- 无数据时显示「暂无结算记录」

### 5. 绑定银行卡
点击「我的银行卡」入口跳转到银行卡绑定页面：
- 填写银行卡号、开户行信息
- 用于收益提现

## 收益记录的时间范围
收益记录按月查询：
- 选择某个月份后，查询范围为该月1日00:00:00至下月1日00:00:00
- 跨年自动处理（12月查询范围为12月1日至次年1月1日）
- 同时调用统计接口查询该月总收入

## 异常处理

| 异常场景 | 系统行为 |
|---------|---------|
| 查询收益记录接口失败 | 列表为空，显示「暂无收益记录」 |
| 查询结算记录接口失败 | 列表为空，显示「暂无结算记录」 |
| 统计接口失败 | 收入小计不更新，保持上一次的值 |

## 关联功能
- 跳转到「银行卡绑定页」→ 管理提现银行卡
- 来源于「问诊详情页」→ 看完问诊后查看收益
- 来源于「处方详情页」→ 开具处方后查看收益

## 重要说明
- 收益记录展示的是每一笔具体收入（未扣除平台抽成前的原始收入）
- 结算记录展示的是平台实际结算给医生的金额（可能已扣除平台费用）
- 页面顶部的「我的指数」（余额）是医生当前可提现的总余额
- 月份选择器切换时，滚动位置自动重置到列表顶部
- 数据未加载完时底部显示「加载中」的加载动画
