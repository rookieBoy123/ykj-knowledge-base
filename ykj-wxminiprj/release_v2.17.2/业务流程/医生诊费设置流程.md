# 医生诊费设置流程

> 文档版本 v1.0 | 面向读者：医生

## 概述

医生在个人设置中配置自己能提供哪些问诊服务、每种服务的价格和时长。共支持七种服务类型，医生可以独立开关每种服务。

## 触发条件

- **前置条件**：医生已通过平台审核，获得接诊资格
- **入口路径**：医生端 → 个人中心 → 诊费设置

## 服务类型一览

系统支持七种服务类型，每种可独立启用/关闭：

| 序号 | 服务类型 | 是否含处方 | 配置项 |
|:---|:---|:---|:---|
| 1 | 图文问诊 | 可接诊和开处方 | 开关 + 价格 + 时长 |
| 2 | 视频问诊 | 可接诊和开处方 | 开关 + 价格 + 时长 |
| 3 | 咨询服务 | 仅做咨询、不开处方 | 开关 + 价格 + 时长 |
| 5 | 快速续方 | 患者通过历史处方快速续方 | 开关 + 价格 |
| 6 | 会诊服务 | 多医生协作 | 开关 + 价格 |
| 7 | 医生复诊 | 医生通过病历详情发起 | 开关 + 价格 |

## 步骤详解

### 第一步：进入诊费设置页面

页面显示医生已开通的所有服务，按服务类型分组展示。每项服务包含：
- 服务名称和标签（"可接诊和开处方" 或 "仅做咨询不开处方"）
- 开启/关闭开关（蓝色滑块）
- 价格设置（点击可修改）
- 时长设置（图文/视频/咨询类型专属）

如果医生暂未开通任何接诊服务，页面显示空状态提示："暂未开启接诊服务，请联系管理人员开启"。

### 第二步：启用或关闭服务

点击某项服务的开关（switch）即可启用或关闭。启用后，下方出现价格和时长配置项；关闭后，配置项折叠隐藏。

**快速续方的特殊依赖**：开启"快速续方"服务前，系统会检查"图文问诊"是否已开启。如果图文问诊未开启，弹窗提示"请先开启图文问诊"，并提供两个选项：
- **取消**：放弃开启快速续方
- **确认**：同时开启图文问诊和快速续方

### 第三步：设置价格

点击价格显示区域，弹出价格选择面板：

- **快速选择**：面板提供预设价格按钮（0元、20元、40元、60元、80元、100元、120元、140元），点击即可快速填入
- **自定义输入**：也可以直接在输入框中输入任意金额
- **价格区间提示**：输入框的占位提示文字显示允许的价格区间（如"建议价格区间为0-200元"）
- **确认保存**：点击"确认"按钮保存价格

图文问诊、视频问诊、咨询服务的价格单位为"元"；会诊服务显示"元/次"；复诊服务显示"元/次"。

### 第四步：设置时长

点击时长显示区域，弹出弹窗：

- 弹窗标题区分："问诊时长"（图文/视频） 或 "咨询时长"（咨询服务）
- 输入框输入数字，单位（分钟/小时）固定显示在旁边
- 确认时系统按服务类型校验最短时长：
  - **图文问诊**：输入的是小时数，最小时长由平台配置决定（默认0.5小时，即30分钟）
  - **视频问诊**：输入的是分钟数，最短由平台配置决定（默认30分钟）
  - **咨询服务**：输入的是小时数，最短由平台配置决定（默认12小时）
- 不满足最小时长的，提示具体的最低要求（如"时长最小为0.5个小时"）
- 后台存储时，图文问诊的时长会自动转换为分钟单位

**咨询服务的特殊说明**：页面底部显示提示"1、仅可以做咨询服务，不可以开处方；2、咨询服务时长至少为XX小时以上"。

### 第五步：保存生效

每次修改价格或时长后，系统自动提交保存所有服务配置。保存成功后页面刷新，显示最新的价格和时长。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 诊费价格超过配置上限 | 弹窗提示"诊费价格不能大于XXX"，阻止保存 |
| 诊费价格低于配置下限 | 弹窗提示"诊费价格不能小于XXX"，阻止保存 |
| 问诊时长低于最小值 | 弹窗提示"时长最小为X个小时/分钟"，阻止保存 |
| 快速续方依赖图文问诊 | 弹窗提示"请先开启图文问诊"，引导医生先开启 |
| 无服务权限 | 页面显示空状态："暂未开启接诊服务，请联系管理人员开启" |
| 价格区间配置未加载 | 默认价格区间 0 ~ 99999 元 |

## 关联功能

- 患者端问诊下单：患者看到的服务价格由此页面配置生效
- 处方开方流程：图文问诊/视频问诊类型的服务才可开处方
- 快速续方流程：依赖图文问诊开启后患者才可发起快速续方
- 会诊流程：会诊价格由此处配置
