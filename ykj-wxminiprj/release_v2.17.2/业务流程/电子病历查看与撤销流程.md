# 电子病历查看与撤销流程

> 文档版本 2026-07-02 | 面向产品运营、客服

## 概述

医生可以在小程序端查看已签名的电子病历完整信息，包括患者基本信息、病情信息（主诉、现病史、既往史等）、诊疗信息（诊断、辨病、辩证、治疗意见）以及医生签名。在特定条件下，医生还可以撤销或编辑病历。

## 触发条件

- 医生在病历记录列表中点击某条病历
- 前置条件：医生已登录，病历记录ID有效
- 入口路径：医生端病历相关列表 → 电子病历详情页

## 步骤详解

### 第一步：查看病历头部信息

页面顶部展示：
- **医院名称**（系统配置标题，如"XX互联网医院"）
- **电子病历**标题
- **问诊单号**：如有关联问诊则显示
- **患者头像**：根据性别（男/女）和年龄（6岁以下显示儿童头像）自动选择

### 第二步：查看患者基本信息

患者信息卡片展示：
- 姓名 + 性别标签 + 年龄
- 就诊科室
- 就诊时间（来源于问诊预约记录）
- 身份证号
- 联系方式
- 现居住地（无则显示"暂无"）

### 第三步：查看病情信息（可折叠）

病情信息区域包含以下字段，每个字段未填写时显示"未填写"：
- **主诉**：患者主要病情描述
- **中医四诊**：舌像 + 脉象（仅中医问诊时显示）
- **现病史**
- **既往史**
- **过敏史**
- **个人史**
- **家族史**
- **婚育史**
- **常见体征**
- **治法**（仅中医诊断时显示）
- **辅助检查**
- **病历资料**：图片预览（点击可放大查看）

病情信息区域超过一定高度时可折叠，点击「展开全部」/「收起信息」切换。

### 第四步：查看诊疗信息

诊疗信息区域展示：
- **诊断**：西医诊断结果，多个用逗号分隔
- **辨病**：中医辨病结果（仅中医诊断时显示）
- **辩证**：中医辩证结果（仅中医诊断时显示）
- **治疗意见**：西医治疗意见（仅西医诊断时显示）

诊断类型由后端数据的 diagnosisType 字段区分：1=西医诊断、2=中医辨病、3=中医辩证。

### 第五步：查看医生签名

页面底部展示医生签名：
- 签名是图片格式（base64）：直接展示签名图片
- 签名是文字格式：展示医生姓名

如果存在原始电子病历文件，显示「查看原始电子病历」按钮，点击可预览PDF。

### 第六步：撤销病历（条件显示）

底部操作栏仅在以下条件**全部满足**时显示：
1. 当前登录角色是「医生」
2. 系统开启了严格模式（strict配置）
3. 病历关联了问诊记录
4. 问诊状态为「问诊结束」（flowStatus=30）
5. 距问诊结束不超过24小时

满足条件时显示两个按钮：
- **撤销病历**（红色/危险按钮）：点击弹出撤销原因输入框，必填（最多500字），确认后提交撤销
- **编辑病历**（普通按钮）：点击跳转病历编辑页

撤销成功后显示"已撤销"提示，1.5秒后自动返回上一页。

### 第七步：编辑病历

点击「编辑病历」按钮跳转到病历编辑页面，携带以下参数：
- 问诊编号（inquiryNo）
- 问诊预约ID（inquiryAppointId）
- 病历记录ID（medicalRecordId）

## 异常处理

| 情况 | 系统处理方式 |
|-----|------------|
| 缺少病历记录ID | 提示"缺少病历记录ID" |
| 撤销原因未填写 | 提示"请输入撤销原因" |
| 不在24小时内 | 底部撤销/编辑按钮不显示 |
| 非医生角色 | 底部撤销/编辑按钮不显示 |
| 无关联问诊记录 | 底部撤销/编辑按钮不显示（问诊信息为空时禁用） |

## 关联功能

- 病历列表 → 电子病历详情页
- 电子病历详情页 → 病历编辑页
- 电子病历详情页 → PDF预览
