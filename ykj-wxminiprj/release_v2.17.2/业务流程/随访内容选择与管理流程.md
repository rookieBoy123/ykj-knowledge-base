# 随访内容选择与管理流程

> 文档版本：release/v2.17.2 | 面向读者：医生

## 概述

医生在创建或编辑随访计划时，需要为每个随访节点选择推送内容。系统提供三种内容类型（患教资料、提醒、问卷），医生可以跨类型多选，也可以随时查看已选内容并编辑。

## 触发条件

- **前置条件**：医生已进入随访计划创建/编辑页面，点击了某个随访节点的内容添加
- **入口路径**：创建随访计划页 → 选择随访内容 → 跳转到随访内容选择页

## 步骤详解

### 第一步：进入内容选择页

页面顶部有三个选项卡，默认选中第一个：

| 选项卡 | 内容类型 | 说明 |
|-------|---------|------|
| 患教资料 | 文章、科普资料 | 患者阅读的健康教育文章 |
| 提醒 | 自定义提醒文字 | 发送给患者的提醒，最多100字 |
| 问卷 | 调查问卷 | 患者填写的健康问卷 |

每个选项卡支持关键字搜索（通过顶部搜索框），搜索后自动重置到第1页。

### 第二步：浏览和选择患教资料（选项卡一）

1. 列表展示所有患教资料，每条显示标题（最多两行截断）
2. 点击任意资料可跳转到详情页查看完整内容
3. 点击右侧复选框选中/取消选中该资料
4. 支持分页加载（每页10条），滚动到底自动加载更多

### 第三步：浏览和选择提醒内容（选项卡二）

1. 列表展示所有已保存的提醒内容，每条显示文字（最多两行截断）
2. 点击任意提醒 → 弹出文字编辑框：
   - 「提醒内容」显示已有内容，可编辑
   - 最多100字
   - 如果是从快捷选中进入（勾选复选框），文字默认为只读
3. 点击右侧复选框快捷选中，或点击文字区域编辑后选中
4. 编辑完成后点击「保存」：如果提醒文字为空则提示「提醒内容不能为空」

### 第四步：浏览和选择问卷（选项卡三）

1. 列表展示所有可用问卷，每条显示问卷标题（最多两行截断）
2. 点击任意问卷跳转到问卷详情页查看题目
3. 点击右侧复选框选中/取消选中

### 第五步：查看已选内容

页面底部固定栏显示：
- **已选内容入口**：展示已选总数（红色数字角标），点击后弹出底部抽屉
- **确认选择按钮**：未选择任何内容时灰色不可点击，选择后变蓝

抽屉中列出所有已选内容：
- 每条显示类型图标（患教资料/提醒/问卷）和标题/内容
- 每条右侧有删除按钮，点击可移除该项
- 如果没有任何已选内容，显示空状态提示

### 第六步：确认选择并返回

点击「确认选择」或抽屉中的「确认选择」，已选内容通过事件通道回传给创建随访计划页面。返回后，创建页面自动更新随访节点的内容列表。

## 三种内容类型说明

| 类型 | 数据来源 | 可编辑 | 可多选 |
|------|---------|--------|--------|
| 患教资料 | 平台知识库统一管理 | 不可编辑（只读） | 是 |
| 提醒 | 医生自定义文本 | 可编辑（100字限制） | 是 |
| 问卷 | 医生创建或公用问卷 | 不可编辑（可通过创建副本修改） | 是 |

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 某类型下没有内容 | 显示空状态「暂无内容」 |
| 提醒保存时内容为空 | 提示「提醒内容不能为空」 |
| 未选择任何内容点确认 | 确认按钮灰色不可点击 |
| 选项卡下已全部加载 | 底部显示「- 已显示全部 -」 |

## 关联功能

- **创建随访计划**：选择内容的目标页面，接收选中内容列表
- **患教资料详情（course）**：查看患教资料全文
- **问卷详情（questionnaire）**：查看问卷题目和选项
- **随访计划详情**：已创建的随访计划中可看到已选内容

## 补充说明

- 已选内容在切换选项卡后保留，各部分之间有记忆关联（通过_id唯一标识）
- 搜索仅在当前选项卡生效，切换选项卡会重置搜索词
- 患教资料和问卷的内容由平台统一管理，医生只能选择不能修改
- 提醒内容是医生自由输入的，可以按需创建不同用途的提醒
