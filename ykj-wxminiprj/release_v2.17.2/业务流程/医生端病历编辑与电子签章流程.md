# 医生端病历编辑与电子签章流程

> 文档版本 2026-07-02 | 面向读者：产品、运营、客服

## 概述

医生在问诊过程中为患者编写病历，记录病史信息、临床诊断、治疗意见和病历资料，完成后进行电子签名（CA签章或手写签名），确保病历的合法性和完整性。

## 触发条件

- **前置条件**：医生已接诊、当前处于「问诊中」状态
- **入口路径**：问诊聊天页 → 点击「编辑病历」按钮
- **页面参数**：inquiryNo（问诊单号）、serviceType（服务类型）、medicalRecordId（已有病历ID则编辑模式）、comeFrom（来源页面）

## 步骤详解

### 第一步：页面初始化（三种数据来源）

页面加载时按以下优先级填充数据：

1. **本地缓存优先**：如果离开页面时系统自动缓存了填写到一半的病历数据，再次进入时会优先恢复缓存内容
2. **已有病历查询**：如果已存在病历ID，从服务器查询并回显历史病历数据（诊断、病史、主诉、治疗意见等）
3. **问诊卡数据**：如果是首次填写，从问诊卡中自动提取患者基本信息（姓名、性别、年龄）、主诉和病情描述

### 第二步：查看患者基本信息

页面顶部展示患者卡片，包含：
- **基本信息**：姓名、性别、年龄（6岁以下显示月龄）、监护人信息（如有）
- **就诊信息**：所属科室、就诊时间
- **慢病标签**：如患者登记了慢病病种，显示蓝色慢病标签
- **诊疗记录入口**：点击「诊疗记录」按钮可跳转患者健康档案页

### 第三步：填写病史信息

医生可逐项填写以下病史内容，每项都可以独立勾选「有」或「无」：

| 病史类型 | 填写内容 |
|---------|---------|
| 过敏史 | 药物/食物过敏情况 |
| 现病史 | 本次发病情况描述 |
| 既往史 | 既往疾病和治疗史 |
| 个人史 | 生活习惯、烟酒史等 |
| 家族史 | 家族遗传病史 |
| 婚育史 | 婚姻生育情况 |
| 月经史 | 女性患者月经情况 |
| 舌像 | 中医舌诊描述 |
| 脉象 | 中医脉诊描述 |
| 常见体征 | 体温、血压等 |
| 辅助检查 | 检查检验结果 |

> **中/西医差异**：中医科医生显示「舌像」「脉象」等中医特有的病史字段；西医科医生不显示。

### 第四步：填写临床诊断

医生可添加三类诊断（至少填写一项）：

- **辨病**（中医）：中医疾病名称
- **辨证**（中医）：中医证型名称  
- **诊断**（西医）：基于ICD-10标准的西医疾病诊断

每种诊断均可从标准诊断库中搜索选择，支持关键词模糊匹配，已选诊断按编码自动去重，最多可添加5个诊断标签。某些由系统下发的诊断标签不可删除（只读状态）。

> **区域适配**：上海地区的医院要求「治法/治疗意见」字段必填。

### 第五步：填写治疗意见与上传资料

- **治疗意见**（西医模式）：输入治疗建议、药物使用、手术方案、康复措施等
- **治法**（中医模式）：描述中医治疗方案和方法
- **病历资料**：可上传检查报告、影像资料等图片附件

### 第六步：AI辅助（可选）

当医院开启了AI辅助功能时，页面右下角显示AI图标。点击后弹出AI辅助面板，系统自动从问诊对话中提取：
- **主诉**：自动填充病情描述
- **诊断**：AI推荐的诊断建议（如与已有诊断不重复则自动追加）
- **病史信息**：7项病史内容（过敏史、现病史、既往史、个人史、家族史、婚育史、月经史）自动填充

所有AI生成内容仅供参考，医生可修改或删除。

### 第七步：保存病历

点击「保存」按钮后：

1. **完整性校验**：检查病史是否填写完整、诊断是否至少填写一项（辨病或诊断至少有一项）
2. **区域校验**：上海地区额外校验「治疗意见/治法」是否填写
3. **创建或更新**：根据是否有病历ID决定创建新病历还是更新已有病历

### 第八步：电子签名

保存病历后进入签名环节，根据医院配置走不同路径：

| 模式 | 行为 |
|------|------|
| **严格模式** | 弹出确认弹窗「确认已完整编写病历？」→「是」打开手写签字板 →「否」暂存草稿（签名=未签） |
| **运营模式** | 后台自动签医生姓名，无需手动操作 |
| **开启CA** | 根据后端返回的caUniqueId调用CA电子签章组件 |

> **CA签章**：如果医院启用了CA电子签章，系统会自动创建签章记录。支持四种CA平台（医网信/腾讯电子签/源康健PaaS/放心签），签名流程根据平台不同有所差异。

### 第九步：保存成功

保存成功后弹出成功提示：
- **问诊中状态**：显示「开具处方」和「返回问诊」两个按钮（仅医生具备开方权限时显示「开具处方」）
- **非问诊中**：仅显示「返回」按钮
- **快速问诊且禁止开方**：不显示「开具处方」按钮

点击「开具处方」会检查CA认证状态，未完成CA认证则引导完成后再跳转。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 诊断和辨病都未填写 | 提示「请添加临床诊断信息（诊断、辨病至少填写一项）」 |
| 上海地区未填写治疗意见/治法 | 提示「请输入治疗意见」或「请输入治法」 |
| 病史信息未填写完整 | 提示检查并滚动到未填写项 |
| CA未认证 | 弹出「未完成CA电子签章不可接诊，请先完成签章」→引导下载证书 |
| CA认证中 | 提示「CA认证正在处理中，请稍后再试」 |
| CA认证失败 | 提示失败原因，引导重新认证 |
| 医网信证书缺失 | 跳转下载证书页面 |
| 放心签未设置签名图片 | 跳转设置签章页面 |
| 页面切换时 | 自动缓存当前填写内容到本地，下次进入自动恢复 |

## 关联功能

- **患者健康档案**：点击「诊疗记录」按钮跳转查看患者全部健康记录
- **开具处方**：保存病历后可直接跳转处方编辑页面
- **问诊对话**：返回问诊聊天页继续沟通
- **AI辅助**：AI自动提取问诊内容填充病历
