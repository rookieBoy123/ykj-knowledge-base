# 收货地址管理流程

> 文档版本：v2.17.2 | 面向读者：患者

## 概述

患者在药品下单时可管理收货地址，支持地址的新增、编辑、删除和设为默认。下单场景下可从地址列表中选择已有地址并带回下单页。

## 触发条件

- **前置条件**：用户已登录，从药品下单页或地址管理入口进入
- **入口路径**：
  - 下单场景：药品下单页 → 选择收货地址 → 进入地址列表
  - 管理场景：我的 → 地址管理

## 步骤详解

### 第一步：进入地址列表页

用户看到地址列表页，顶部标题「地址管理」，底部固定按钮「新增地址」。

- 如果已有地址：以卡片列表展示，每张卡片显示收货人姓名、电话、省/市/区、详细地址。若该地址是默认地址，卡片带绿色「默认」标记。
- 如果无地址：显示空状态插图+「暂无收货地址」文字。

列表支持滚动分页加载，每页**10条**。

### 第二步：新增地址

点击「新增地址」按钮，进入新增/编辑表单页，包含以下字段：

| 字段 | 说明 | 校验规则 |
|------|------|---------|
| 姓名 | 收货人姓名 | 必填，不能为空字符串 |
| 电话 | 收货人手机号 | 必填，手机号格式校验（11位数字） |
| 地区 | 省/市/区 | 必填，点击弹出区域选择器 |
| 详细地址 | 街道门牌等 | 必填，不能为空字符串 |
| 设为默认 | 开关 | 可选 |

底部操作区根据模式显示：
- **新增模式**：左侧「取消」（返回上一页），右侧「保存」
- **编辑模式**：左侧「删除」（红色边框按钮），右侧「保存」

### 第三步：填写表单

1. **选择地区**：点击「地区」行弹出三级联动选择器（省→市→区），选择后地区行自动填充「xx省xx市xx区」
2. **填写详细地址**：输入街道、门牌号、楼层房间号等
3. **设置默认**：打开开关，该地址将标记为默认收货地址

### 第四步：表单校验与保存

点击「保存」按钮后，依次校验：

1. 姓名不能为空 → 提示「请输入收货人姓名」
2. 手机号格式校验 → 提示（由手机号校验规则决定）
3. 地区不能为空 → 提示「请选择收货地区」
4. 详细地址不能为空 → 提示「请填写详细地址」

全部通过后调用保存接口，成功后弹出「保存成功」提示，1.5秒后自动返回地址列表。

### 第五步：删除地址

编辑模式下点击「删除」按钮，弹出确认弹窗「确定要删除吗？」，用户可选择「取消」或「确定」。
- 点击「确定」后调用删除接口，成功后弹出「删除成功」，1.5秒后返回列表。
- 注意：删除操作不可恢复。

### 第六步：下单场景选择地址

当从药品下单页进入地址列表时（URL携带参数 `from=order`），点击任意地址卡片会：
1. 选中该地址，卡片高亮
2. 将地址ID存入本地缓存（`addressCache`）
3. 自动返回上一页（下单页），下单页从缓存中读取地址信息

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 表单校验不通过 | 对应字段下出现提示文字，保存按钮可被点击（仅校验拦截） |
| 保存请求失败 | 不显示成功提示，停留在当前页面，用户可重新点击保存 |
| 删除请求失败 | 弹窗关闭，地址保留，不显示成功提示 |
| 地址列表为空 | 显示空状态插图，引导用户新增 |
| 进入编辑页但地址ID无效 | 加载不到地址详情，可能只显示空表单 |

## 关联功能

- **药品下单**：下单时选择收货地址
- **我的页面**：从我的页面入口管理所有地址
- **微信联系客服**：页底部提示「若该手机号已停用/无法接收短信，请联系客服处理」（仅解绑场景，地址管理无此功能）
