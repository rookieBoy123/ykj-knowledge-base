# 收货地址管理流程

> 文档版本：2026-07-02 | 面向读者：患者用户

## 概述

患者在小程序中管理药品配送的收货地址，支持新增、编辑和删除地址。每个地址包含收货人姓名、手机号、省市区地区和详细地址，可设置默认地址。

## 触发条件

- 前置条件：已登录小程序的用户
- 入口路径：患者端 → 我的 → 收货地址

## 步骤详解

### 第一步：新增收货地址

进入新增地址页面（路径参数 type=add）：
1. 填写**收货人姓名**（必填，输入框提示"请输入收货人姓名"）
2. 填写**手机号**（必填，数字键盘输入，提示"请输入收货人手机号"，自动校验格式）
3. 选择**地区**（必填，点击弹出省市区三级联动选择器，标题"选择地区"，选后自动拼接为"省+市+区"显示）
4. 填写**详细地址**（必填，多行文本输入，提示"街道门牌、楼层房间号等信息"）
5. 可开启"**设为默认地址**"开关

保存时进行4项校验：
- 姓名为空→提示"请输入收货人姓名"
- 手机号格式错误→提示手机号格式错误
- 未选地区→提示"请选择收货地区"
- 详细地址为空→提示"请填写详细地址"

保存成功后提示"保存成功"，自动返回上一页。

### 第二步：编辑收货地址

进入编辑地址页面（路径参数 type=edit + addressId）：
- 自动加载已有地址信息填入各字段
- 修改后点击保存，同样经过4项校验
- 底部显示"删除"和"保存"两个按钮

### 第三步：删除收货地址

在编辑地址页面底部点击"删除"按钮：
- 弹出确认弹窗"确定要删除吗？"
- 确认后删除并提示"删除成功"
- 自动返回上一页

### 第四步：设置默认地址

在地址编辑页面，打开"设为默认地址"开关即可：
- 系统自动将其他地址取消默认（通常每个用户只有一个默认地址）
- 默认地址在下单时优先选中

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 姓名为空 | 提示"请输入收货人姓名" |
| 手机号格式错误 | 通用的手机号格式校验提示 |
| 未选择地区 | 提示"请选择收货地区" |
| 详细地址为空 | 提示"请填写详细地址" |
| 删除确认取消 | 关闭弹窗，不做任何操作 |

## 关联功能

- 药品下单配送：下单时选择收货地址，默认地址优先
- 处方药品配送：开具处方后购药配送使用收货地址
