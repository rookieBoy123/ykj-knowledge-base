# 医生快速开方流程

> 文档版本：2026-07-02 | 面向读者：产品运营、客服、医生

## 概述

快速开方是医生不需要先建立问诊就可以直接为患者开具处方的快捷功能。医生可以在患者信息、用药需求和处方模板之间自由切换，支持中药处方和成药处方两种模式。

## 触发条件

- **前置条件**：医生已登录小程序，具备开方工作类型权限
- **入口路径**：从医生首页或其它入口进入快速开方页面
- **传入参数**：可携带患者ID（锁定患者）或留空（手动选择患者）

## 步骤详解

### 第一步：选择处方类型

进入页面后，系统根据医生的科室类型自动决定默认处方模式：
- **中医科医生**：默认显示「中药处方」标签页，可切换到「成药处方」
- **西医科室医生**：只显示成药处方模式

医生可以在中药和成药之间切换（中医科医生）。

### 第二步：选择或确认患者

- 如果从其他页面携带患者ID进入，患者信息会自动加载且不可修改（灰色显示，可点击删除重选）
- 如果没有携带患者，医生可以手动选择患者
- 选择患者后，系统自动加载该患者最新的病史信息（过敏史、现病史、既往史等）

### 第三步：填写临床诊断

在诊断区域录入诊断信息：
- **中医科医生**：需填写辨病、辨证。如开启了「西医诊断必填」配置，还需同时填写西医诊断
- **西医科室医生**：填写诊断（西医诊断）

诊断支持搜索选择标准诊断编码。

### 第四步：录入处方药品

**中药处方模式**：
- 搜索并添加中药材，填写每味药材的用量
- 选择药房（决定药品来源和价格）
- 设置用法用量：内服/外用、剂数、每日N剂、分N次服用
- 选择加工工艺（如代煎、打粉等），工艺决定收费方式（按重量/按剂数）
- 填写用药医嘱：煎煮说明、用药时间、用药禁忌（多选）
- 可设置免运费（仅开通过此权限的医生）

**成药处方模式**：
- 搜索并添加成药药品，选择规格和数量
- 选择药房
- 为每味药品填写用法用量（每日次数、每次用量、用药天数、用药方式）
- 填写用药医嘱文字
- 可设置免运费

药品搜索结果会标注缺货/库存不足状态。

### 第五步：选择是否保存为处方模板

页面底部有一个开关「保存为模版处方」，开启后需要输入模板名称。提交时会同时保存处方和模板。

### 第六步：提交处方

页面底部有两个按钮：
- **存为草稿**：保存为草稿处方，可在「我的-处方记录-处方草稿」中查看和继续编辑
- **确认发送给患者**：正式开方并发送给患者

点击「确认发送给患者」后：
1. 系统首先校验患者信息完整性
2. 校验中药/成药的诊断信息
3. 检查药房授权（医生是否被授权使用该药房）
4. 校验药品库存（缺货会提示更换药品）
5. 中药处方额外校验：十八反十九畏、有毒中药超量（弹窗确认）
6. 如果配置了CA电子签章，会触发签名流程（医网信/放心签/源康健PaaS）
7. 严格模式下先人脸识别再签名
8. 签名成功后处方生效

### 第七步：处方结果

- 保存草稿成功：弹出提示「处方已存为草稿」，可点击「查看处方记录」或「返回首页」
- 正式开方成功：跳转到开方成功页面

## 页面缓存机制

快速开方页面在离开时会自动缓存填写的内容。下次进入时：
- 如果医生科室类型没变，会恢复上次填写的内容（包括药品、诊断等）
- 如果医生科室类型变了（如从中医科切换为西医科），缓存会被清除
- 一旦成功保存或开方，缓存会自动清除

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 患者信息未填写完整 | 提示对应字段必填，阻止提交 |
| 诊断信息为空 | 提示「请添加临床诊断信息」 |
| 药品列表为空 | 校验不通过，阻止提交 |
| 药品库存不足/缺货 | 提示「药品缺货或库存不足，请更换药品」 |
| 药房授权问题 | 提交时校验，无授权阻止提交 |
| 药材存在配伍禁忌 | 弹出十八反十九畏警告弹窗，需医生确认 |
| 有毒中药超量 | 弹出警告弹窗，列出超量药材 |
| 中药用法用量不完整 | 校验不通过，需完善 |
| 成药用法用量不完整（对外上传时） | 提示「请将已添加的药品用法用量填写完整」 |
| 存为模板但未填名称 | 提示「请输入模板名称」 |
| CA证书未完成 | 弹出提示「未完成CA电子签章不可接诊」 |
| 提交中重复点击 | 按钮置灰并显示加载状态，防止重复提交 |
| 黑名单药品 | 系统自动删除，弹出提示「处方中存在黑名单药品系统已自动删除」 |

## 数据来源

- **处方模板**：可从模板列表选择已有模板，一键填充到表单
- **历史处方**：可从历史处方列表选择，一键填充到表单
- **公共处方**：可从公共处方库引入
- **快速续方**：从快速续方入口带入

这些来源通过事件通道（eventChannel）将模板数据传递给快速开方页面。

## 关联功能

- 处方模板列表（选择模板后回传数据）
- 历史处方列表（选择历史处方后回传数据）
- 处方记录（存为草稿后可跳转查看）
- 医生首页（返回首页入口）
- CA电子签章（签名组件）
- 人脸识别（严格模式下签名前需人脸识别）
