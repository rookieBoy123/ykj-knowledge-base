# 医生问卷题目创建流程

> 文档版本：release/v2.17.2 | 面向读者：医生

## 概述

医生在创建问卷时，可以逐题添加问题。每个问题支持四种类型（单选、多选、问答、图片），每种类型的填写内容和校验规则不同。创建完成后，题目通过事件通道回传给问卷创建页面。

## 触发条件

- **前置条件**：医生正在创建或编辑一份问卷
- **入口路径**：问卷创建页面 → 点击「添加题目」 → 创建题目页面

## 步骤详解

### 第一步：选择题目类型

页面顶部展示四种题目类型，以图标+文字形式排列：

| 类型 | 说明 | 患者端交互 |
|------|------|-----------|
| 单选（圆形图标） | 从多个选项中选择一个 | 点击单选 |
| 多选（方框图标） | 从多个选项中选择多个 | 点击多选 |
| 问答（笔图标） | 开放式文字回答 | 文字输入框 |
| 图片（上传图标） | 患者上传图片 | 图片上传区域 |

点击某个类型后，该类型高亮（蓝色背景、白色文字），其他类型恢复默认样式。切换类型时，之前填写的选项数据会被清空。

### 第二步：填写题目内容

每种类型都有一个文本输入框填写题目（最多100字）。输入框下方显示可根据类型变化的配置区域。

**单选和多选**：
1. 输入选项内容（每个选项最多500字）
2. 点击「新建选项」添加更多选项
3. 每个选项右侧有删除按钮（×），点击可删除该选项
4. **校验规则**：单选/多选题的选项不能少于2项，如果少于2项或任意选项为空，底部「保存本题」按钮为灰色不可点击
5. 底部提示文字：单选题显示「单选题选项不能少于2项」，多选题显示「多选题选项不能少于2项」

**问答题**：
1. 题目下方显示提示信息输入框（最多100字）
2. 提示信息用于引导患者作答（如「请描述疼痛的具体部位和程度」）
3. 页面有说明文字：「为帮助患者更好地理解问题，您可设置问题回答示例和回答指导」

**图片题**：
1. 可输入提示信息（最多100字）
2. 可上传示例图片（最多8张），说明文字：「为帮助患者更准确上传的图片，请具体描述所需图片的信息（如身体部位名称）和上传示例图片」
3. 已经上传的示例图片可点击预览、可删除（删除时二次确认：「是否删除该图片？」）
4. 上传已达8张上限时提示「最多可选择8张上传」

### 第三步：设置是否必填

所有题目类型底部都有一个「本题必填」复选框。默认勾选状态。点击可切换必填/非必填。患者填问卷时，必填题目未作答会阻止提交。

### 第四步：提交保存

1. 点击底部「保存本题」
2. **重复题目检查**：系统检查已存在的所有题目中是否有相同类型+相同题目的——如果有，提示「该题型已存在相同的题目，请修改题目后保存」
3. 如果检查通过，通过事件通道将题目数据传回问卷创建页面
4. 自动返回上一页，问卷创建页面的题目列表新增此题

## 校验规则汇总

| 校验项 | 条件 | 提示文案 |
|-------|------|---------|
| 题目为空 | 题目输入框无内容 | 保存按钮灰色不可点击 |
| 选项不足 | 单选/多选题选项少于2个 | 保存按钮灰色不可点击 |
| 选项为空 | 任意选项内容为空 | 保存按钮灰色不可点击 |
| 题目重复 | 相同类型已有相同题目 | 「该题型已存在相同的题目，请修改题目后保存」 |

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 保存成功 | 自动返回上一页，题目列表更新 |
| 切换题目类型 | 清空所有已填选项数据 |
| 图片上传失败 | 平台通用上传错误提示 |

## 关联功能

- **问卷创建页面（create）**：接收新建的题目，管理所有题目的顺序
- **问卷详情（detail）**：创建完成后可在详情页查看所有题目
- **问诊问卷发送**：创建完成的问卷可发送给患者填写
- **题目库（questionLib）**：公共题目库中的题目可被复用

## 补充说明

- 题目创建是独立的页面，与问卷创建页面通过事件通道（eventChannel）通信
- 题目保存后不可直接编辑，需要在问卷编辑页面对已有题目进行编辑
- 从已有问卷创建副本时，原问卷的所有题目会一并复制
