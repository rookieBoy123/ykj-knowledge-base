# 线下电子病历查看流程

> 文档版本 v1.0 | 面向读者：医生

## 概述

医生查看患者在线下医院就诊产生的电子病历（由HIS系统同步），了解患者的就诊信息、体格检查、中医四诊、辅助检查、诊断及治疗意见。这是一个纯查看页面，不支持编辑。

## 触发条件

- **前置条件**：患者在线下医院有就诊记录，且病历已同步到系统
- **入口路径**：医生端 → 患者病历详情 → 点击某条线下病历记录

## 页面结构

病历页面包含以下信息区块：

1. **患者基本信息**：头像（男女不同图标）、姓名、性别、年龄
2. **就诊信息**：医生姓名、就诊科室、就诊时间
3. **病情信息**：主诉、现病史、既往史、过敏史、个人史、家族史、婚育史、月经史（默认折叠显示300rpx高度）
4. **体格检查**：体温(℃)、脉率(次/min)、呼吸频率(次/min)、BMI、体重(kg)、身高(cm)、收缩压/舒张压(mmHg)、体格检查文字描述
5. **中医四诊**：望闻问切信息
6. **辅助检查**：检查结果文字描述
7. **初步诊断**：诊断结论
8. **病状分类**：疾病分类
9. **治疗意见**：治疗建议

## 步骤详解

### 第一步：进入病历页面

医生从患者病历列表中选择一条线下病历记录进入。页面顶部有半透明背景装饰（渐变背景图），中部滚动展示病历详情，底部固定显示"查看原始电子病历"按钮。

### 第二步：浏览病情信息

病情信息区域展示八个字段（主诉、现病史、既往史、过敏史、个人史、家族史、婚育史、月经史）。

**折叠机制**：默认情况下，病情信息区域高度限制为300rpx（约150px），超出部分被隐藏。底部显示"展开全部"按钮（带向下箭头图标），点击后展开全部内容，按钮文字变为"收起信息"（箭头反转向上）。

### 第三步：查看体格检查

体格检查区列出生命体征数据：体温、脉率、呼吸频率、BMI、体重、身高、血压。血压以"收缩压/舒张压"格式显示，如只填了收缩压则单独显示。如果填写了"体格检查"文字描述字段，额外显示在底部。

### 第四步：查看其他信息区

依次浏览中医四诊、辅助检查、初步诊断、病状分类、治疗意见等区域。所有未填写的字段统一显示"未填写"。

### 第五步：查看原始电子病历

点击底部"查看原始电子病历"按钮，系统调用文件预览功能打开病历原文件（如PDF或图片）。如果没有原始病历文件，弹窗提示"暂无原始电子病历"。

## 异常处理

| 情况 | 系统处理方式 |
|:---|:---|
| 病历记录不存在 | 接口返回空数据，页面不显示内容 |
| 某字段为空 | 统一显示"未填写" |
| 原始病历文件不存在 | 弹窗提示"暂无原始电子病历" |
| 病情信息过长 | 默认折叠300rpx，可点击"展开全部" |
| 页面ID参数丢失 | 页面静默不加载数据 |

## 关联功能

- 在线电子病历查看（`elMedical/index.vue`）：支持撤销/编辑的在线病历
- 病历编辑：医生的病历编辑页面
- HIS数据同步：线下病历由HIS系统同步至平台
