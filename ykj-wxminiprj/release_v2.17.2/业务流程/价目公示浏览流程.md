# 价目公示浏览流程

> 文档版本 | 面向读者：患者用户

## 概述
平台提供公开透明的价格公示页面，患者可以在此查看药品价格、医生诊费和会诊费用三项价目信息，方便在就诊前了解费用情况。

## 触发条件
- **前置条件**：进入小程序，通常从首页导航或"我的"页面入口进入
- **入口路径**：小程序 → 价目公示

## 步骤详解

### 第一步：选择查看类型
页面顶部有三个标签页可供切换：
- **药品价格**：查看各药房药品的售价
- **诊费价格**：查看各医生不同问诊方式的费用
- **会诊价格**：查看远程会诊的费用标准

进入页面默认显示「药品价格」标签。

### 第二步：查看药品价格（药品价格标签）

**筛选功能**：
- **药房选择**：顶部下拉菜单可选择查看哪个药房的药品。如果有配置默认药房且开启了默认药房功能，则默认选中默认药房；否则显示所有药房供选择。
- **药品类型**：可切换"成药"和"中药"两个类型

**列表展示**：
| 列名 | 说明 |
|------|------|
| 药品名称 | 药品的通用名称 |
| 规格 | 药品规格信息，无数据时显示"--" |
| 生产厂家 | 药品生产厂家，无数据时显示"--" |
| 价格（元） | 药品售价 |

支持滚动加载更多，每次加载30条。

**空状态处理**：
- 如果系统未配置任何药房，页面显示空状态提示："暂无可用药房"、"请联系管理员配置药房信息"，表格区域隐藏。

### 第三步：查看诊费价格（诊费价格标签）

**列表展示**：
| 列名 | 说明 |
|------|------|
| 医生姓名 | 医生名称 |
| 科室 | 所属科室 |
| 职称 | 医生职称 |
| 诊费 | 分图文、视频、续方三种方式分别显示 |

诊费明细：
- **图文**：图文问诊费用，大于0显示金额，等于0显示"免费"
- **视频**：视频问诊费用，大于0显示金额，等于0显示"免费"
- **续方**：续方费用，大于0显示金额，等于0显示"免费"

支持滚动加载更多，每次加载30条。

### 第四步：查看会诊价格（会诊价格标签）

**列表展示**：
| 列名 | 说明 |
|------|------|
| 项目名 | 按医生职称分类（如"主任医师会诊"） |
| 计价单位 | 会诊的计费单位 |
| 价格（元） | 显示为区间（如 200~500） |
| 备注 | 固定展示："每增加一位医生加收对应医生职称费用" |

会诊价格一次性加载全部数据，不支持分页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 无可用药房 | 显示"暂无可用药房"空状态提示 |
| 药品无规格信息 | 显示"--" |
| 药品无生产厂家 | 显示"--" |
| 诊费为0 | 显示"免费" |
| 诊费字段不存在（null） | 不显示该行问诊方式 |

## 关联功能
- **药房配送**：药品价格与药房关联，药房信息来自药房管理模块
- **医生问诊**：医生诊费与医生问诊服务关联
- **远程会诊**：会诊价格与远程会诊申请关联
