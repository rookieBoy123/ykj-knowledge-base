# 医生身份认证与资料提交流程

> 文档版本 v1.0 | 面向读者：新注册医生

## 概述

新注册的医生需要通过小程序提交个人资料和四类证件照片进行身份认证。提交后由后台运营人员审核，审核通过后才能正式接诊。已审核通过的医生也可以编辑可修改字段更新资料。

## 触发条件

- **前置条件**：已登录小程序，且当前处于患者角色
- **入口路径**：
  - 首次注册：通过商务分享的邀请二维码扫码进入，自动跳转到资料填写页
  - 已有草稿：再次扫码或手动进入时自动加载草稿数据继续填写
  - 编辑资料：审核通过后从医生首页"编辑资料"入口进入（已审核字段锁定不可改）

## 步骤详解

### 第一步：填写基本信息

- **头像**：点击头像区域上传个人正面免冠照（建议穿医生工作服），上传后实时预览
- **姓名**（必填）：输入真实姓名，审核通过后不可修改
- **性别**（必填）：选择"男"或"女"，审核通过后不可修改
- **电话**（必填）：自动填充会员手机号，不可修改
- **邮箱**（选填）：输入个人邮箱地址
- **城市**（选填）：通过省市区三级联动选择执业所在城市

### 第二步：填写专业信息

- **医院**（必填）：点击弹出搜索弹窗，输入医院名称远程搜索，支持滚动加载更多。选择医院后自动回填，审核通过后不可修改
- **科室**（必填）：树形结构选择科室（如"内科→心内科"），审核通过后不可修改
- **职称**（必填）：下拉选择框，从字典加载（如主任医师、副主任医师等），审核通过后不可修改

### 第三步：填写个人介绍（选填）

- **个人介绍**：点击"请输入"弹出底部文字输入面板，输入完成后点击"确认"。编辑模式下显示"已输入N字"
- **擅长领域**（选填）：同上，弹出面板输入

### 第四步：上传证件照片

**身份认证**（必填）：
- 支持四种证件类型：身份证（默认）、护照、港澳居民来往内地通行证、台湾居民来往大陆通行证
- 点击"证件类型"可切换，切换后证件照示例图自动变化
- 身份证/通行证需上传正反两面各一张；护照仅需主页一面
- 每种证件类型有对应的提示文案（如"请上传身份证正面"）

**医师资格证书**（必填）：
- 上传人像页和国徽页各一张

**医师执业证书**（必填）：
- 上传人像页和国徽页各一张

**其他相关证明**（选填，最多5张）：
- 每张图片可点击上传，已上传的图片右上角显示删除按钮
- 上传满5张后不再显示添加按钮，编辑模式下也不显示删除按钮

### 第五步：确认提交

页面底部有两个按钮：
- **"返回"按钮**：
  - 如果已提交过审核（状态为"审核中"），直接返回上一页
  - 如果是草稿状态，弹出确认弹窗"是否返回患者首页？"，确认后保存草稿并切换回患者角色
- **"确认"按钮**：触发提交校验

**提交前的校验规则**：
1. 头像不能为空 → "请上传头像"
2. 姓名不能为空 → "请输入姓名"
3. 性别必须选择 → "请选择性别"
4. 手机号必须为有效格式 → "手机号码格式不正确"
5. 医院必须选择 → "请选择医院"
6. 科室必须选择 → "请选择科室"
7. 职称必须选择 → "请选择职称"
8. 身份证正面必须上传 → 根据证件类型提示对应文案
9. 身份证反面必须上传（护照除外）→ 对应提示文案
10. 医师资格证书正面必须上传 → "请上传医师资格证书正面"
11. 医师执业证书正面必须上传 → "请上传医师执业证书正面"

全部通过后提交审核，页面跳转到"资料审核中"状态页。

### 编辑模式（已审核通过的医生）

- 编辑模式下，审核通过的字段（姓名、性别、医院、科室、职称）显示为灰色禁用状态，前面有锁图标
- 可编辑的字段：头像、邮箱、城市、个人介绍、擅长领域
- 重新提交审核时需微信订阅消息授权
- 如果是重新提交审核（cxsh 参数），会将状态重置为"审核中"
- 如果是普通编辑，直接更新医生信息

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 退出填写页 | 如果已有已提交的审核，直接返回；如果是草稿，弹窗询问是否保存草稿后返回 |
| 编辑模式修改已审核字段 | 已审核字段被禁用（灰色+锁图标），无法修改 |
| 证件类型切换 | 已上传的证件照会保留；护照类型不显示国徽页上传区域 |
| 医院搜索无结果 | 显示空状态，提示"暂无数据" |
| 提交时校验不通过 | 逐项弹出错误提示，停留在当前页 |
| 保存草稿失败 | 不阻塞返回首页流程 |

## 关联功能

- **审核状态页**（`audittatus/index.vue`）：提交后跳转查看审核进度
- **医生首页**（`doctor/index/index.vue`）：审核通过后进入的医生端首页
- **扫码入口**：通过商务分享的二维码进入注册流程，携带业务员ID（accessBusinessId）或邀请医生ID（friendDoctorId）
- **医网信同步**：内部测试功能，可同步医生信息到监管平台
