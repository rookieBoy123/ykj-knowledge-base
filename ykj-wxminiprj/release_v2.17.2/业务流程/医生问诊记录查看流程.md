# 医生问诊记录查看流程

> 文档版本：2026-07-05 | 面向读者：产品、运营、客服

## 概述

医生在「问诊记录」页面可以查看自己接诊过的所有患者问诊列表，包括问诊中、已结束和已取消三种状态。点击任意一条记录可进入对应问诊聊天室，查看完整的问诊对话和处方记录。

## 触发条件

- **前置条件**：医生已登录并进入医生工作台
- **入口路径**：医生工作台 → 问诊记录（导航菜单）

## 步骤详解

### 第一步：进入问诊记录列表

医生进入页面后，系统自动按时间倒序加载问诊记录列表，每页显示 20 条。每条记录展示：

- **患者信息**：头像、姓名、性别（男女图标）、年龄
- **服务类型**：图文问诊、视频问诊等，以蓝色标签显示
- **问诊状态**：右上角灰色圆角标签，分为"问诊中""问诊结束""已取消"
- **消息摘要**：
  - 问诊中的记录显示病情描述（咨询问题）
  - 问诊结束的记录显示病情描述
  - 已取消的记录显示取消原因
- **就诊时间**（如有）：显示预约时间范围，格式为"YYYY-MM-DD HH:mm 至 MM-dd HH:mm"

### 第二步：滚动加载更多

页面支持上拉底部自动加载更多记录（分页加载），每次追加 20 条。当全部记录加载完后不再触发加载。

### 第三步：查看问诊详情

医生点击任意一条记录后，系统调用 `publicGoEndChatRoom` 方法跳转到问诊聊天室。该方法会根据问诊状态（问诊中 / 已结束）进入不同的聊天界面。进入聊天室后可以查看完整的问诊对话、处方和病历资料。

### 第四步：空状态

如果医生当前没有任何问诊记录，页面显示"暂无记录"的空状态提示，不显示任何卡片。

## 消息摘要规则

系统会根据问诊的最后一条消息自动生成摘要，优先级如下：

1. **预约时间消息**（flowStatus=20）：自动解析为「预约时间成功，请提前5分钟进入诊室」
2. **门诊编号消息**（含outpatientNo）：显示「你有一条订单消息」
3. **JSON格式消息**（普通聊天）：显示「你有一条未读消息」
4. **非JSON格式**：
   - 空消息 → 「暂无消息」
   - .mp3 文件 → 「[语音]」
   - .jpg/.png/.jpeg 文件 → 「[图片]」
   - .mp4 文件 → 「[视频]」
   - 含 roomID → 「[问诊视频]」

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 没有问诊记录 | 显示"暂无记录"空状态 |
| 已取消的记录 | 显示取消原因（remark字段），如为空则显示"暂无" |
| 无就诊时间 | 不显示时间行（仅当mainStartTime有值时显示） |
| 患者信息缺失 | 姓名显示为空字符串，性别/年龄可能有空值 |

## 关联功能

- 点击记录 → 跳转到问诊聊天室，可查看问诊对话和处方
- 问诊状态变化由系统自动更新，医生无需手动操作
