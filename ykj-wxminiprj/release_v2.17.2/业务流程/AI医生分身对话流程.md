# AI医生分身对话流程

> 文档版本 | 面向读者：患者、医生

## 概述

患者通过AI医生分身进行初步问诊咨询。AI会根据患者描述自动进行追问、智能分析，并在对话结束后生成健康报告。若患者需要进一步诊疗，可转接专家团队的真人医生继续问诊。

## 触发条件

- **前置条件**：医生开通了AI医生分身功能，患者已提交问诊申请并获得问诊编号
- **入口路径**：从问诊订单详情或消息通知中点击进入AI对话界面

## 步骤详解

### 第一步：进入AI对话

页面加载时，系统自动判断当前状态：

- **新问诊（尚无对话记录）**：自动创建AI问诊会话，患者可开始输入症状描述
- **进行中的问诊**：加载历史对话记录，并继续当前会话
- **已结束但未生成报告**：自动触发生成健康报告
- **已结束且已生成报告**：展示完整的报告卡片，并推荐专家团队医生

### 第二步：对话交互

患者和AI之间以聊天界面进行问答：

- **患者输入**：在底部输入框输入症状、不适等问题（支持图片上传），点击发送
- **AI逐字回复**：AI的回答会以流式传输的方式逐字显示，模拟真实打字效果，让等待不再枯燥
- **常见问题提示**：AI在每次回复后会给出3个与当前对话相关的常见问题供患者选择，点击即可自动填入输入框，减少打字
- **多轮对话**：AI会根据患者的回答自动追问细节（如症状持续时间、严重程度、有无过敏等），引导完成诊断所需的信息采集

### 第三步：问诊结束与报告生成

当AI判定已收集到足够信息时，对话结束：

- **状态变为"已结束"**：输入框隐藏，无法再发送消息
- **自动生成报告**：页面显示"正在生成报告，请稍候"，几秒后AI报告卡片出现在聊天区域中
- **报告内容包括**：主要症状分析、可能涉及的健康问题、建议措施等

### 第四步：转接专家团队

问诊结束后，如患者需要开药或进一步诊疗：

- 页面底部展示"问诊已结束"分隔线和温馨提示："如需开药或进一步问诊可选择专家团队的医生问诊"
- 列出该AI医生关联的专家团队真人医生列表（含姓名、科室、职称）
- 点击任意医生卡片，跳转到医生详情+选择服务页面，可发起点对点真人问诊

### 第五步：退出对话

- 顶部返回按钮：回到上一页面（如消息通知、问诊订单列表）
- 若从首页入口进入（`goHome=1`参数），返回按钮将跳转到首页

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 进入页面时问诊单不存在（患者角色） | 自动创建新的AI问诊会话，然后重新加载 |
| 问诊ID缺失 | 弹出提示"缺少问诊ID"，无法发送消息 |
| 最后一条消息由AI发出且会话进行中 | 自动请求AI生成常见问题推荐，供患者选择 |
| 对话记录仅有一条且为患者发送 | 自动触发BEGIN请求启动AI对话流程 |
| 报告生成失败 | 静默处理，不显示加载状态，不弹出错误提示 |
| 无关联专家团队医生 | 不显示"问诊已结束"区域，页面底部留白 |
