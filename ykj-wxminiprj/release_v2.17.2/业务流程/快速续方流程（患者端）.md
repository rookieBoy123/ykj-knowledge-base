# 快速续方流程（患者端）

> 文档版本 2026-07-01 | 面向读者：患者

## 概述

快速续方是针对"有有效处方、病情稳定、只需继续按原方吃药"的患者提供的便捷续方通道。患者无需重新问诊，选择历史处方即可让医生快速开具续方，也可上传线下处方进行续方。

## 触发条件

- 从"我的"→"快速续方"入口进入
- 仅显示最近6个月内的有效处方

## 步骤详解

### 第一步：查看处方列表

页面顶部显示蓝色提示条："快速续方适用有线上或线下有效处方，正按方吃药并且病情稳定的患者"。下方展示"仅展现最近6个月的有效处方"的处方列表。

每张处方卡片包含：
- 处方类型（中药/成药）、处方编号
- 患者姓名
- 诊断信息（成药显示诊断、中药显示辨病+辨证）
- 处方医生姓名
- "查看详情"和"快速续方"两个操作按钮
- 每页5条，滚动自动加载更多

### 第二步：选择续方方式

**方式一：选择历史处方续方**

点击处方卡片上的"快速续方"按钮，系统依次进行以下检查：

1. **医生是否开通快速续方服务**：如果医生未开通，弹出提示（分两种情况）：
   - 普通患者：「原处方医生暂不方便续方，是否为您匹配新的医生？」
   - 特殊患者：「医生未开通快速续方服务，您可以向医生发起问诊咨询购药」，并引导去问诊

2. **是否已有进行中的问诊**：如果该医生已有未完成的快速续方问诊，弹出提示：
   - 有未支付订单：「您有未支付问诊单，是否前往支付？」
   - 进行中问诊：「您有正在进行的问诊，是否前往？」

3. **是否需要匹配新医生**：如果原处方医生当前不接诊，弹出「您的医生暂不在线，是否为您匹配新的医生？」

4. **直接进入支付**：所有检查通过后，跳转到收银台生成问诊订单并支付。

**方式二：上传处方续方**

对于线下纸质处方，点击"上传处方续方"按钮。需先检查登录状态（未登录引导登录），然后通过微信订阅消息授权后跳转到病历资料上传页面，选择服务类型为"快速续方"。

### 第三步：支付与完成

支付成功后进入正常的问诊流程。医生接诊后可查看患者上传的处方图片或历史处方信息，开具新的电子处方。

### 特殊场景

- **匹配新医生**：系统根据原处方所属药房（pharmacyId）为新医生匹配，匹配成功后自动创建快速续方问诊订单
- **无医生在线**：当匹配不到新医生时，弹出"暂无医生在线，请稍后再试"
- **续方处方关联**：续方时会自动携带原处方编号（continuedPrescriptionNo）便于医生参考

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 6个月内无有效处方 | 显示空状态："暂无处方记录"，并提示可上传线下处方 |
| 原处方医生未开通快速续方 | 弹出提示，引导匹配新医生或去问诊 |
| 已有进行中问诊 | 弹出提示，引导前往支付或前往问诊 |
| 匹配不到新医生 | 弹出提示"暂无医生在线，请稍后再试" |
| 未登录时点上传处方 | 强制引导到登录页面 |
| 患者为特殊身份 | 医生未开通时引导去问诊，而非匹配新医生 |
| 消息订阅未授权 | 跳过订阅继续跳转 |
