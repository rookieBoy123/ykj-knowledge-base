# 快速续方流程（患者端）

> 文档版本：2026-07-06 | 面向读者：患者用户

## 概述

快速续方是面向有线上或线下有效处方、正在按方吃药且病情稳定的患者，提供的一键续方购药服务。患者可选择历史处方快速续方，或上传线下处方发起续方问诊。

## 触发条件

- **前置条件**：患者已登录，拥有至少一张有效处方（6个月内）
- **入口路径**：小程序首页 → 快速续方

## 步骤详解

### 第一步：进入快速续方页

患者进入页面后看到：
- 顶部提示语："快速续方适用有线上或线下有效处方，正按方吃药并且病情稳定的患者"
- 「上传处方续方」按钮（点击后走快速问诊流程，服务类型标记为"图文急诊"）
- 处方列表区域，显示最近6个月的有效处方

### 第二步：选择处方

患者在处方列表中点击某张处方：
- 可点击处方卡片查看处方详情（跳转到用药建议详情页）
- 可点击"快速续方"按钮发起续方

### 第三步：医生资格检查

点击"快速续方"后，系统依次检查：

1. **医生是否开通快速续方服务**：如果医生未开通，弹出提示"医生未开通快速续方服务，您可以向医生发起问诊咨询购药"
   - 特殊患者通道：点击「去问诊」跳转到医生详情页购买问诊服务
   - 普通患者：点击「匹配新医生」系统自动分配其他医生

2. **是否存在未完成的问诊**：如果该患者对该医生有未完成的快速续方问诊
   - 有未支付订单：弹出提示"您有未支付问诊单，是否前往支付？"→ 点击后跳转订单详情页
   - 有进行中问诊：弹出提示"您有正在进行的问诊，是否前往？"→ 点击后跳转到聊天页面

3. **医生是否在线**：系统判断是否需要匹配新医生
   - 如果原医生在线：直接保存问诊参数到缓存，跳转到创建订单页
   - 如果原医生不在线：弹出"您的医生暂不在线，是否为您匹配新的医生？"

### 第四步：匹配新医生

当原医生不在线时：
- 系统根据原处方所属药房，自动匹配该药房下可用的新医生
- 匹配成功后，跳转到创建订单页面（服务类型标记为"快速续方"）
- 如果该药房没有医生在线：弹出"暂无医生在线，请稍后再试"

### 第五步：创建订单

跳转到订单创建页后，患者正常完成支付即可。支付完成后进入问诊流程，医生可查看续方处方并为患者开具新处方。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 没有有效处方（6个月内） | 显示空状态"暂无处方记录"，可选择上传线下处方续方 |
| 医生未开通快速续方 | 弹窗提示"医生未开通快速续方服务" |
| 特殊患者且医生未开通 | 引导"去问诊"购买问诊服务 |
| 存在未支付问诊单 | 弹窗提示，可选择去支付或取消 |
| 存在进行中问诊 | 弹窗提示，可前往聊天页面继续问诊 |
| 医生不在线 | 弹出"匹配新医生"确认框 |
| 药房无可用医生 | 弹出"暂无医生在线，请稍后再试" |
| 未登录 | 引导跳转到登录页面 |

## 关联功能

- **处方详情**：点击处方卡片可查看处方明细
- **创建订单**：续方后进入订单创建与支付流程
- **即时通讯**：已支付的问诊通过IM聊天页面与医生沟通
- **上传处方续方**：通过图文急诊服务类型进入病历提交页面
