# 线下门诊病历详情查看流程

> 文档版本：release/v2.17.2 | 面向读者：患者、医生

## 概述

患者和医生可以查看从HIS系统同步过来的线下门诊病历的完整详情，包括患者信息、诊断、体格检查、中医四诊、处方信息等。

## 触发条件

- **前置条件**：有从HIS同步的线下病历记录
- **入口路径**：健康档案→诊疗记录→线下病历列表→点击某条病历

## 步骤详解

### 第一步：查看患者基本信息

进入详情页后，顶部展示患者的基本信息卡片：
- **头像**：根据性别自动切换男性/女性默认头像
- **姓名**：患者姓名
- **性别和年龄**：例如"男 35岁"
- **医生信息**：接诊医生姓名
- **科室**：就诊科室
- **就诊时间**：线下就诊的日期时间

### 第二步：查看主诉和病史

病历分为多个区块，每个区块有一个带蓝色竖线的标题：

- **主诉**：患者就诊时口述的主要不适
- **现病史**：当前疾病的发生、发展过程
- **既往史**：以前的疾病和治疗情况
- **其他病史**：分为四项子信息
  - 个人史
  - 家族史
  - 婚育史
  - 月经史
  - 过敏史

所有字段如果源数据为空，则显示为空白（不会显示"无"等占位文字）。

### 第三步：查看体格检查

展示八项基本生命体征测量值：
- 体温（℃）
- 脉率（次/min）
- 呼吸频率（次/min）
- BMI（体重指数）
- 体重（kg）
- 身高（cm）
- 收缩压/舒张压（mmHg）——用"/"分隔，如果舒张压为空则只显示收缩压
- 体格检查描述文字（条件显示）

空值字段不显示数值。

### 第四步：查看诊断和治疗

以下区域依次展示：

- **中医四诊**：望闻问切的记录
- **辅助检查**：如血常规、B超等检查内容
- **初步诊断**：医生的诊断结论
- **病状分类**：疾病分类
- **治疗意见**：医生的处理建议

每个区域的空字段显示为空白。

### 第五步：查看处方信息

页面有一个专门的分区标题"处方信息"（带处方图标）。

如果有处方，每条处方一张卡片展示：
- **处方类型和种类**：如"中药处方(普通)"、"西药处方(急诊)"
- **处方编号**：线下处方的门诊号

**西药处方**（type=5）展示：
- 药品名称 + 数量 × 单位
- 剂型、规格
- 用法用量：每天几次，每次多少，使用多少天

**中药处方**（type=4）展示：
- RP标识 + 药品列表（含克数和煎煮备注）
- 剂数、用药方法、用药说明
- 下方显示处方医生和审核药师

处方种类含义：
| 编码 | 含义 |
|------|------|
| 1 | 治疗项目 |
| 2 | 理疗项目 |
| 3 | 体检项目 |
| 4 | 中药处方 |
| 5 | 西药处方 |
| 6 | 附加项目 |
| 7 | 材料 |

处方类型含义：
| 编码 | 含义 |
|------|------|
| 0 | 普通处方 |
| 1 | 急诊处方 |
| 2 | 儿科处方 |
| 3 | 麻醉、精一处方 |
| 4 | 精二处方 |
| 5 | 其他 |

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 缺少病历ID参数 | 页面无法初始化 |
| 接口返回非200或无数据 | 不渲染任何内容 |
| 某字段为空 | 该字段位置显示空白（不显示"无"或占位文字） |
| 无处方信息 | 处方区域为空（但标题仍然显示） |
| 性别非1也非2 | 年龄旁显示"未知" |
