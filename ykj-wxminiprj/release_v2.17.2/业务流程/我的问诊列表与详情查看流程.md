# 我的问诊列表与详情查看流程

> 文档版本 1.0 | 面向非技术读者 | 2026-07-02

## 概述

患者在小程序中查看自己发起的所有问诊记录，包括待支付、问诊中、已预约、已完成四种状态的问诊。每个问诊卡片展示医生信息、问诊人、病情描述等关键信息，点击可进入问诊室继续对话或查看详情。问诊详情页还展示病历、会诊记录、对话记录等完整信息。

## 触发条件

- 患者发起问诊后需要查看进度
- 患者需要继续与医生的对话
- 患者需要查看已完成问诊的病历和评价
- 入口：小程序 → 我的 → 我的问诊

## 步骤详解

### 第一步：问诊列表页

进入「我的问诊」页面后，顶部显示四个Tab标签页：

| Tab | 对应状态 | 说明 |
|-----|---------|------|
| 待支付 | flowStatus=10（草稿） | 已填写病情但未付款的问诊 |
| 问诊中 | flowStatus=20（问诊中） | 正在进行的问诊，可进入问诊室 |
| 已预约 | flowStatus=20 + 预约模式 + 未签到 | 已预约但尚未到就诊时间的问诊 |
| 已完成 | flowStatus=30（问诊结束） | 已结束的问诊，可查看详情和评价 |

### 第二步：查看问诊卡片

每个问诊卡片展示：
- **医生信息**：头像、姓名、科室、职称
- **状态标签**：右上角显示当前状态（待支付/问诊中/已预约/已完成/已取消等）
- **问诊人**：姓名、性别、年龄
- **服务类型**：图文问诊/视频问诊/咨询服务/快速问诊/快速续方等
- **病情描述/咨询内容**：患者填写的症状或问题
- **问诊时间**：创建时间
- **预约就诊时间**（已预约Tab）：显示预约的具体时段

区分服务类型的标签：咨询服务显示「咨询人」「咨询内容」「咨询时间」，其他服务显示「问诊人」「病情描述」「问诊时间」。咨询服务不区分问诊中/已预约，状态标签显示「咨询中」。

### 第三步：操作按钮

根据不同状态显示不同操作：

| 状态 | 按钮 | 点击行为 |
|------|------|---------|
| 待支付（flowStatus=10） | 「立即支付」 | 跳转订单详情页完成支付 |
| 问诊中（flowStatus=20） | 「进入问诊室」 | AI问诊跳AI对话页，普通问诊跳IM聊天室 |
| 已取消（flowStatus=40） | 「查看详情」 | 查看详情时弹出取消原因提示 |
| 已完成（flowStatus>20） | 「查看详情」 | 跳转问诊详情页 |

已取消的问诊卡片底部显示红色提示框，展示取消原因。

### 第四步：问诊详情页

点击「查看详情」后进入问诊详情页，展示：

**医生信息区域**：医生头像、姓名、科室、职称、医院。点击可跳转医生详情或AI医生详情页。

**问诊详情区域**：
- 问诊人/咨询人：姓名、性别、年龄
- 服务类型
- 病情描述/咨询内容
- 问诊/咨询时间
- 就诊时间（非AI问诊且非咨询时显示）
- 结束时间

**问诊问卷**：展示问诊时填写的问卷内容（如症状问卷、健康评估等）。

**病历详情**（非咨询和非AI问诊）：点击跳转病历详情页。如病历尚未生成，提示"未生成病历"。

**会诊记录**（如有）：显示本次问诊关联的会诊记录列表，包括会诊目的和会诊时间，点击可跳转会诊详情。

**问诊对话记录**：点击进入聊天记录查看页（已结束的问诊可查看历史消息，已取消的问诊弹出提示）。

**评价医生**（问诊结束后）：问诊结束（flowStatus=30）且非AI问诊时，显示「评价医生」入口，可对本次问诊的医生进行评价。

### 第五步：就诊时间显示规则

就诊时间的显示根据医生接诊状态动态调整：
- 待接诊：不显示就诊时间
- 接诊中（无需预约）：显示接诊开始时间
- 接诊中（需预约、未预约）：显示接诊开始时间
- 接诊中（需预约、已预约）：显示接诊开始时间
- 转诊场景（兼职医生转全职医生）：显示开始时间+结束时间

### 第六步：分页加载

列表每页15条，支持上拉加载更多。滚动到底部自动加载下一页，全部加载完毕后显示"没有更多数据了"。若某Tab下无数据，显示空状态插图"暂无记录"。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 某Tab无数据 | 显示空状态插图和"暂无记录" |
| 已取消的问诊 | 点击查看详情时弹出提示框，展示取消原因 |
| 网络加载失败 | 保留已加载的数据，不影响浏览 |
| AI问诊无病历 | 不显示病历详情入口 |
| 咨询服务 | 不显示病历详情和评价医生入口 |
| 当前用户非医生角色查看医生端 | 提示"当前用户非医生角色" |

## 关联功能

- **问诊室聊天**：点击"进入问诊室"跳转IM聊天
- **AI医生对话**：AI问诊跳转专用AI对话页
- **订单支付**：待支付订单跳转订单详情
- **病历查看**：跳转病历详情页
- **会诊详情**：跳转会诊详情页
- **医生评价**：跳转评价页
- **医生详情**：跳转医生服务选择页
