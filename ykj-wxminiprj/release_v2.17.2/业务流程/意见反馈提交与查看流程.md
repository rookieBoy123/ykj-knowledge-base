# 意见反馈提交与查看流程

> 文档版本：v2.17.2 | 面向读者：患者

## 概述

患者向平台提交意见或建议，包括选择反馈类型、填写文字描述、上传截图（最多3张）、留下联系电话。提交后可查看历史反馈记录及平台的处理回复。

## 触发条件

- **前置条件**：用户已登录
- **入口路径**：我的 → 意见反馈

## 步骤详解

### 第一步：进入反馈列表页

用户看到反馈列表页，顶部标题「意见反馈」。

- 如果有反馈记录：以卡片列表展示，每条包含反馈类型（蓝色粗体）、处理状态（已处理=蓝色/未处理=红色）、反馈内容、上传截图（如有）、联系电话、提交时间。如果平台已回复，在卡片底部显示灰色背景区域，内容为「处理反馈：xxx」，回复文字超过约44个字时默认折叠，可点击「展开」查看全部。
- 如果无反馈记录：显示空状态。

列表每页加载**10条**，滚动到底部自动加载更多。每次进入页面（`onShow`）会自动刷新列表。

底部固定按钮「反馈意见」，点击进入提交页面。

### 第二步：填写反馈

进入提交页，标题「提交意见反馈」，包含三个必填项：

#### 1. 反馈类型（必选）

点击「请选择」弹出底部面板，罗列所有反馈类型选项（来自字典 `fklx`，如「功能建议」「Bug反馈」「服务体验」等）。每个选项右侧有选中/未选中图标，**只能单选**。选中后点击「确定」关闭面板，页面显示选中的类型名称。

#### 2. 问题和建议（必填，至少5个字）

在文本输入框中输入意见或建议，最多**500字**。输入框下方可上传截图辅助说明：
- 最多上传**3张**图片
- 点击虚线框区域调起图片选择器
- 上传的图片显示缩略图，右上角有删除按钮
- 点击缩略图可预览大图
- 点击删除按钮弹出确认框「是否删除该图片？」
- 数量达到3张后，点击上传区域提示「最多可选择3张上传」

#### 3. 联系电话（必填）

输入11位手机号，格式校验不通过时提示。

### 第三步：提交反馈

点击底部「提交」按钮，系统校验：

1. 反馈类型为空 → 提示「请选择反馈类型」
2. 意见内容为空 → 提示「请输入意见或建议」
3. 意见内容少于5个字 → 提示「意见或建议至少五个字」
4. 手机号格式不正确 → 提示错误信息

全部通过后显示加载中，提交成功弹出「提交成功」并自动返回列表页（1.5秒延迟）。提交失败则弹出「提交失败」。

### 第四步：查看处理结果

在反馈列表页可以看到每条反馈的处理状态：

- **未处理**（红色）：平台人员尚未查看或处理
- **已处理**（蓝色）：平台人员已处理并可能附有回复内容

回复内容显示在灰色背景区域，长文本折叠。反馈提交后用户**不能修改或删除**已有反馈。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 图片超过3张 | 提示「最多可选择3张上传」 |
| 意见少于5字 | 提示「意见或建议至少五个字」 |
| 未选择反馈类型 | 提示「请选择反馈类型」 |
| 手机号格式错误 | 提示格式错误信息 |
| 提交接口失败 | 提示「提交失败」，用户可重试 |
| 反馈列表为空 | 显示空状态，引导用户提交第一个反馈 |

## 关联功能

- **我的页面**：入口位置
- **管理后台**：运营人员在管理后台查看和处理反馈
