# 就诊人添加与编辑管理流程

> 文档版本 1.0 | 面向非技术读者 | 2026-07-02

## 概述

患者在小程序中管理就诊人信息，包括添加新的就诊人、编辑已有就诊人资料、删除就诊人。就诊人是患者使用平台各项服务（问诊、开方、购药）的身份基础，系统会校验身份信息的真实性和年龄合规性。

## 触发条件

- **添加就诊人**：首次使用服务时，或为家人添加就诊人时
- **编辑就诊人**：已有就诊人信息需要修改时
- **删除就诊人**：不再需要某个就诊人时
- 入口路径：小程序 → 我的 → 就诊人管理 → 添加/编辑

## 步骤详解

### 第一步：进入就诊人编辑页

页面根据是否传入就诊人ID自动判断模式：
- **添加模式**：页面标题显示「添加就诊人」，所有字段可自由填写
- **编辑模式**：页面标题显示「编辑就诊人」，已认证的信息（姓名、证件号）显示为锁定状态，不可修改

页面顶部显示就诊人姓名输入框，输入身份证号后自动识别性别和年龄，显示在姓名右侧（如"男 | 28岁"）。

### 第二步：选择证件类型并填写证件号

用户可从四种证件类型中选择：
- 身份证（默认）：输入18位身份证号，系统自动校验格式并从号码中提取性别和出生日期
- 护照：9位字母和数字组合
- 港澳居民来往内地通行证：H/M前缀+8位数字
- 台湾居民来往大陆通行证：8位数字

选择非身份证类型时，页面会额外显示性别（男女单选）和出生日期选择器，出生日期不能晚于6年前（不满6岁儿童不能在线问诊）。

### 第三步：选择成员关系

用户从下拉列表中选择与就诊人的关系：本人、父母、配偶、子女、亲属、其他。

### 第四步：填写健康信息（可选）

如果系统开启了海迪设备功能，需填写就诊人身高（厘米），限正整数。

### 第五步：填写地址信息（按需）

根据医院配置，地址填写有三种模式：
- 不收集地址：该区域不显示
- 收集且必填：需选择省市区（县乡镇）并填写详细地址
- 收集非必填：可选择填写

### 第六步：添加监护人（不满6岁儿童）

添加6岁以下儿童时，系统强制要求填写监护人信息：
- 监护人姓名
- 监护人与儿童关系
- 监护人身份证号

在严格运营模式下，直接禁止添加6岁以下儿童，弹出提示「不满6岁儿童不能在互联网医院进行问诊，请到线下实体医院问诊」。

### 第七步：保存就诊人

点击「保存」按钮，系统执行以下校验：
1. 姓名不能为空
2. 证件类型必须选择
3. 证件号码不能为空且格式正确
4. 非身份证类型需选择性别和出生日期
5. 成员关系必须选择
6. 如开启了海迪功能，身高必填且为正整数
7. 如地址配置为必填，省市区和详细地址必填

校验通过后，系统调用实名认证接口进行身份验证。验证通过后：
- 如使用海迪设备扫码登录场景，自动绑定设备
- 如从问诊流程进入，将就诊人信息同步到问诊缓存
- 成功保存后自动返回上一页

### 第八步：编辑已认证就诊人

已认证的就诊人，姓名和证件号锁定不可修改（显示锁图标）。可修改的信息包括：
- 成员关系
- 身高
- 所在地区和详细地址
- 监护人信息

### 第九步：删除就诊人

在编辑模式下，底部显示红色「删除就诊人」按钮。点击后弹出二次确认弹窗，明确提示「删除就诊人后相关病历也一并删除」。确认删除后，如果当前被删除的是正在问诊中选中的就诊人，会同时清除问诊缓存。

### 第十步：解绑已绑定就诊人

如果添加或编辑时，系统检测到该就诊人已被其他手机号绑定，会弹出提示：同一就诊人只能绑定在一个手机号，当前就诊人已绑定[手机号]，是否进行解绑？用户可选择「去解绑」跳转到解绑验证页面。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 姓名为空 | 提示"请输入就诊人姓名" |
| 证件类型未选 | 提示"请输入就诊人证件类型" |
| 证件号为空 | 提示"请输入就诊证件号码" |
| 证件号格式错误 | 输入框下方红字提示"证件号格式有误，请检查后更正提交" |
| 身份证校验不通过 | 红色错误提示，无法提交 |
| 成员关系未选 | 提示"请选择就与就诊人的成员关系" |
| 身高未填（海迪模式） | 提示"请输入就诊人身高" |
| 身高非正整数 | 提示"身高只能输入正整数" |
| 所在地区未选（必填模式） | 提示"请选择所在地区" |
| 详细地址为空（必填模式） | 提示"请填写详细地址" |
| 严格模式下不满6岁 | 弹窗提示"不满6岁儿童不能在互联网医院进行问诊" |
| 非严格模式下不满6岁无监护人 | 自动弹出监护人填写弹窗 |
| 就诊人已被其他手机号绑定 | 弹窗提示已绑定手机号，可去解绑 |
| 实名认证失败 | 弹窗显示具体失败原因 |

## 关联功能

- **问诊流程**：添加就诊人后可同步到问诊信息填写页
- **海迪设备**：扫码绑定就诊人与体检设备
- **就诊人列表**：保存后返回列表页，新就诊人自动出现在列表中
- **监护人管理**：6岁以下儿童绑定监护人信息
