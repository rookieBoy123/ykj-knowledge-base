# 创建随访计划流程

> 文档版本：release/v2.17.2 | 面向读者：客服、运营

## 概述

医生从患者列表中选择已就诊患者后，可以创建随访计划。随访计划由「计划名称 + 就诊时间 + 多个随访任务」组成。每个任务设置间隔时间和发送内容（患教资料/提醒内容/问卷）。医生也可以从随访模板库中选择已有模板快速创建。

## 触发条件

### 前置条件
- 医生已与患者建立就诊关系
- 患者已有至少一次就诊记录（用于提取就诊时间）

### 入口路径
- **医生端** → 患者管理 → 选择患者 → 发起随访 → 选择模板 → 进入创建页
- **从随访模板进入**：选择模板后直接进入，标题变为「随访计划详情」，自动填充模板数据

## 步骤详解

### 第一步：进入页面

页面加载时：
1. 接收 patientId、memberId、可选 templateId
2. 自动创建默认计划（间隔1周，无内容）
3. 查询患者最近就诊时间
4. 如有模板ID则拉取模板详情并自动填充

### 第二步：填写计划名称

必填项，如「高血压术后随访」。

### 第三步：选择就诊时间

从患者就诊记录日期列表中选择。

### 第四步：添加随访任务

每个任务包含：
- **间隔时间**：距离就诊日的间隔（数字 + 单位：天/周/月）
- **发送内容**：三种类型
  - 患教资料：健康教育文章/视频
  - 提醒内容：纯文本，最多100字
  - 问卷：健康量表

计划管理：
- 点击「+ 添加计划」新增空白计划
- 第2个及之后的计划可删除（第1个为默认计划不可删）
- 点击提醒内容可编辑文本

### 第五步：提交前校验（按顺序）

| 序号 | 校验项 | 不通过提示 |
|------|--------|-----------|
| 1 | 计划名称不为空 | 「请输入计划名称」 |
| 2 | 就诊时间不为空 | 「请选择就诊时间」 |
| 3 | 随访日期未过期 | 「计划N的随访日期已过，请重新选择」 |
| 4 | 每计划至少1条内容 | 「请添加发送内容」 |

### 第六步：提交

成功后提示「执行计划成功」（3秒），自动跳转到随访计划详情页。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 任意校验不通过 | 吐司提示对应信息，不提交 |
| 就诊时间列表为空 | 选择器显示空 |
| 模板中已禁用内容 | 自动过滤不展示 |

## 关联功能

- 跳转到随访计划详情页（提交成功）
- 跳转到内容详情（患教/问卷）
- 从随访模板库进入
