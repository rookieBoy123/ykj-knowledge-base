# 医生收益与结算查看流程

> 文档版本：v1.0 | 面向读者：医生

## 概述

医生在小程序端查看自己的收入明细和结算记录，包括累计收益指数、按月查询的收入流水、结算打款记录，以及管理绑定银行卡。

## 触发条件

- **前置条件**：以医生角色登录
- **入口路径**：医生端「我的」→「收益指数」

## 步骤详解

### 第一步：查看收益总览

进入页面后，顶部展示蓝色收益卡片：

1. **「我的银行卡」入口**（右上角胶囊按钮）：点击跳转到银行卡绑定页
2. **「我的指数」**：显示医生当前的累计剩余金额（remainingSum），格式为 ¥XX.XX
3. **卡片设计**：蓝色渐变背景，金额采用超大字重（96rpx）突出显示

如果医生尚未绑定银行卡，仍可查看收益记录，但无法提现结算。

### 第二步：切换收益记录/结算记录

卡片下方有两个选项卡：

- **收益记录**（recordType=1）：默认显示。展示每一笔收入明细，按时间倒序排列
- **结算记录**（recordType=2）：展示已经打款的结算记录，金额为红色显示

切换选项卡时：
1. 列表和分页状态全部重置
2. 重新加载对应数据

### 第三步：筛选收益月份

仅在"收益记录"模式下：

1. 顶部显示月份选择器（picker mode="date" fields="month"），默认为当前年月
2. 右侧显示当月收入统计金额（¥XX.XX）
3. 选择不同月份后：重新查询该月的收入明细，更新统计金额
4. 时间范围：选中月份的1号 00:00:00 到下月1号 00:00:00

### 第四步：浏览记录列表

每条记录卡片展示：

1. **描述**：收入来源说明（description），如"问诊收入""处方收入"
2. **时间**：收入产生的时间（createTime）
3. **金额**：蓝色文字（收益记录）或红色文字（结算记录），格式化为浮点显示

列表支持滚动分页（每页10条），滑到底部自动加载更多。没有记录时显示空状态图"暂无收益记录"或"暂无结算记录"。

### 第五步：管理银行卡（关联跳转）

点击「我的银行卡」跳转到银行卡绑定页，可：
- 新增银行卡：填写银行卡号、银行名称、开户行、卡主姓名、身份证号
- 编辑已有银行卡（每个医生仅一张）
- 卡主姓名需与注册医生姓名一致

绑定成功后1.5秒自动返回收益页。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 无收益记录 | 显示空状态"暂无收益记录" |
| 无结算记录 | 显示空状态"暂无结算记录" |
| 选择月份无数据 | 统计金额显示0，列表为空 |
| 网络异常 | 显示加载失败 |
| 银行卡未绑定 | 跳转到绑定页新增 |

## 数据说明

1. **收益记录**：实时展示每一笔收入，金额实时更新
2. **结算记录**：展示已完成的打款记录，结算记录可能滞后于收益记录（取决于打款周期）
3. **我的指数**：等于所有收益累加扣除已结算金额后的剩余金额
