# 医生端快速开方流程

> 文档版本 2026-07-02 | 面向使用者：医生

## 概述

快速开方是医生无需经过问诊流程，直接为患者开具处方的功能。适用于线下已看诊完成、仅需补开处方的场景。支持成药处方和中药处方两种类型，医生可以单独开具或附带模板保存。

## 触发条件

- **前置条件**：医生已在小程序注册并开通快速开方权限
- **入口路径**：医生工作台 → 快速开方
- **两种模式**：
  - **新开处方**：选择患者后进入空白开方页
  - **编辑草稿**：从「我的-处方记录-处方草稿」继续未完成的处方
  - **使用模板**：从处方模板列表或历史处方中选择导入

## 步骤详解

### 第一步：选择患者

进入快速开方页面后，首先选择就诊患者。

- 如果从处方草稿列表进入（`operationType=edit`），患者信息已自动填充
- 如果从患者详情进入（`patientId`传参），自动加载患者信息和最新病史
- 如果是新开处方，患者信息可编辑，需填写姓名、性别、年龄等基本信息
- 中医科医生与西医医生的处方默认类型不同（中医科默认中药、西医默认成药）

页面顶部的 Tab 切换取决于医生科室类型：中医科显示「中药处方 | 成药处方」，西医显示「成药处方 | 中药处方」

### 第二步：填写诊断信息

诊断信息区域包含：
- **西医诊断**：搜索 ICD-10 诊断编码，支持去重、最多 5 个
- **中医辨病**（中医科）：中医疾病诊断
- **中医辨证**（中医科）：中医证型诊断

诊断数据来源优先级：病历已有诊断 > 问诊卡诊断 > 旧处方携带诊断（仅在当前诊断区域为空时回填）

### 第三步：选择药房和添加药品

根据当前 Tab 位置，进入成药或中药开方区域：

**成药开方**：
- 选择发药药房（需校验医生是否有该药房的授权、药房是否启用、库存是否可用）
- 搜索添加药品，填写用法用量每条必须完整
- 如果是对外上传处方，需额外检查用法用量完整性

**中药开方**：
- 选择药房（同样需要授权/启用/库存三重校验）
- 通过拼音简码搜索或从模板/历史导入中药饮片
- 填写克数（小数两位限制）、煎煮方法（特殊煎法标记）
- 配伍校验：十八反十九畏、有毒中药超量、药材库存
- 设置加工工艺（如颗粒剂），按重量计算则需填每日次数和每次用量

### 第四步：设置费用和医嘱

- **医事服务费**（中药）：服务费 + 咨询费 + 总费用，可编辑
- **成药诊金**：可设置挂号费（需医生开通服务类型 7）
- **用法用量**（中药）：服用方式、服用方法、用药天数
- **用法备注**（中药）：额外的用法说明文字
- **医嘱**（中药）：用药时间、煎药方法、禁忌事项
- **医嘱**（成药）：自由文本输入框
- **运费设置**：如果医生开通了免运费权限，可勾选免除运费

### 第五步：提交或存为草稿

底部固定两个按钮：

**「存为草稿」按钮**：
- 如果勾选了「保存为模版处方」，则走完整校验流程后保存
- 如果未勾选模板且无药品，直接保存基本信息
- 保存成功后弹出成功提示框，可「查看处方记录」或「返回首页」
- 页面切换或退出时自动缓存全部数据，再次进入可恢复

**「确认发送给患者」按钮**：
- 第一步：校验患者信息必填
- 第二步：根据处方类型校验诊断、药品等必填项；中医科下配置西医诊断必填则还需西医诊断
- 第三步：中药处方检查药材数量为空、校验配伍（十八反十九畏/有毒超量）、校验库存
- 第四步：如果勾选模板且未填名称，提示「请输入模板名称」
- 第五步：如果是草稿编辑状态走编辑接口，否则走新增接口
- 第六步：如果返回需要签章（`authSign=true`），触发电子签名流程
- 第七步：签名完成后跳转到处方成功页

### 草稿恢复机制

- 页面隐藏（`onHide`）和卸载（`onUnload`）时自动保存缓存
- 保存成功后清除缓存（`hasSave=true`）
- 再次进入时从缓存恢复全部数据，包括诊断、药品、药房、费用等
- 切换患者（传入 `patientId`）时清除患者缓存但保留其他缓存

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 药房未授权 | 不展示该药房，无法选择 |
| 药品库存不足或缺货 | 弹出提示「药品缺货或库存不足，请更换药品」 |
| 中药药材未填克数 | 提示「药材：【名称】请输入数量！」 |
| 中药配伍冲突（十八反十九畏/超量） | 弹出验证弹窗，列出违规药材，需医生确认或修改 |
| 黑名单药品 | 自动删除并弹出提示「处方中存在黑名单药品系统已自动删除」 |
| 处方已提交但签名失败 | 记录日志，提示「CA签名异常，请稍后再试」，保留页面缓存 |
| 开方接口返回 400422 | 弹出配伍校验弹窗，医生可确认或返回修改 |
| 草稿保存失败 | 弹窗显示错误原因 |
| 对外上传处方用法用量不完整 | 提示「请将已添加的药品用法用量填写完整」 |
| 签名取消（用户主动关闭） | 不缓存数据，跳转到处方详情页 |

## 关联功能

- **处方模板管理**：开方时可选择导入模板，完成后可保存为模板
- **历史处方列表**：开方时可选择导入历史处方
- **处方记录**：草稿箱和已发送处方列表
- **药品搜索与添加**：中药饮片添加页（拼音搜索+专用键盘）
- **电子签章**：CA 签章（医网信/源康健PaaS/放心签）或手写签名
- **订阅消息**：发送处方前请求患者订阅通知
