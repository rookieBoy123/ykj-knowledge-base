# 中药饮片添加与处方配伍流程

> 文档版本 1.0 | 面向读者：医生、产品、运营

## 概述

医生在开具中药处方时，通过拼音简码搜索或历史用药快速选择药材，逐味添加并设定克数和煎煮方法，最终通过配伍校验（十八反十九畏、有毒药品、超量警告）后完成中药配伍。

## 触发条件

- 医生进入"添加药品"页面（从开方页面的"添加中药"入口）
- 必须已选择药房（pharmacyId）

## 步骤详解

### 第一步：进入页面，选择输入方式

进入页面后，底部自动弹出专用键盘。医生有三种方式添加药材：

1. **专用键盘输入拼音简码**：点击药品输入框，使用底部英文键盘输入药材拼音首字母（如输入"GC"搜索"甘草"），系统实时搜索匹配的药材
2. **快速录方（智能识别）**：切换到"快速录方"模式，在文本框中按"药材名+克数+煎法"格式自由输入。例如："石莲子 15 先煎，黄芩10，地骨皮10"。系统自动解析药材名和克数
3. **历史用药快捷选择**：无搜索输入时，显示该药房的历史常用药材列表（最多10味），点击直接添加

### 第二步：逐味添加药材

每添加一味药材后：
- 页面顶部实时显示"已选N味"和总克数、预估总价
- 药材卡片显示：名称、克数（可+/-调整）、煎煮方法选择
- 如果药材在当前药房不存在，标记为"药房无此药"（红色标识）

专用键盘输入克数时：
- 支持小数（最多两位）
- 常用剂量快捷按钮（5g/10g/15g/20g/25g）一键设置
- 中药颗粒按库存单位转换：如果1袋=2克，提示"建议输入数量是2的倍数"

### 第三步：选择煎煮方法

每味药材可单独设置煎煮方法，点击药材卡片的煎煮方法区域弹出选择面板：
- 选项从字典"中药特殊煎煮方法"（zytsjzff）中获取
- 常见选项：先煎、后下、包煎、另煎、烊化、冲服等
- 选择后卡片显示对应煎法名称

### 第四步：智能识别录入（可选）

点击"图片识别"按钮，支持两种方式：
- **拍照/相册**：拍摄处方图片，AI自动识别药材名称和克数，结果自动追加到快速录入文本框
- **病历资料**：跳转病历资料页，识别病历中的处方信息

### 第五步：添加完成与配伍校验

点击"添加完成"按钮，系统执行以下校验：

1. **数量校验**：检查是否所有药材都填写了数量（克数>0），否则提示"药材：【名称】请输入数量！"
2. **配伍禁忌校验**（verifyZyDrugs）：后端检查是否存在：
   - 十八反/十九畏（配伍禁忌冲突）
   - 有毒中药超量使用
   - 某些药材的双重超量警告
3. 校验通过→药材列表返回开方页面，自动回填
4. 校验不通过→弹出"校验中药"弹窗，逐条展示问题，医生需确认或修改

### 第六步：页面间数据同步

- 每次药材变更自动通过事件通道（eventChannel）同步至开方页面
- 清空药品、删除单味药材都会实时更新开方页
- 配伍校验状态（已确认/未确认）也同步，开方页据此控制是否允许继续

## 页面功能说明

| 功能 | 说明 |
|------|------|
| 清空 | 二次确认弹窗："确定清空药品吗？确定后，所有选择的药品将被移除。" |
| 单排/双排 | 切换药材卡片布局，单排更紧凑，双排更宽阔 |
| 历史用药 | 最近使用的前10味药材，按药房缓存（pharmacyYlist），可展开全部并支持删除历史 |
| 常用剂量 | 默认5/10/15/20/25g；中药颗粒按库存单位转换后动态计算 |

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未填写克数就提交 | 提示"药材：【名称】请输入数量！"，拒绝提交 |
| 重复添加同一药材 | 提示"请勿重复添加药品！" |
| 药材在药房不存在 | 标记为"药房无此药"（红色），该药材不会被提交 |
| 输入超过两位小数 | 提示"仅支持输入小数点后两位" |
| 配伍禁忌（十八反十九畏） | 弹出警告弹窗，列出冲突详情 |
| 有毒中药超量 | 药材卡片显示"超量"标记（含上限克数），弹出超量说明弹窗 |
| AI图片识别失败 | 不会阻断流程，可手动输入 |
| 药房无药材可选 | 搜索无结果时提示"无药品" |

## 关联功能

- 开方页面（处方开具）：药材列表通过事件通道回传
- 图片识别（AI处方识别）：拍照/相册/病历资料识别药材
- 药房管理：药材库存、价格、单位来自药房系统
