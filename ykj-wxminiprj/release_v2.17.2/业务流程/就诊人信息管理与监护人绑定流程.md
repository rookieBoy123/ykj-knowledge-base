# 就诊人信息管理与监护人绑定流程

> 文档版本 v1.0 | 面向读者：患者用户

## 概述

患者可以在小程序中添加、编辑、删除就诊人（家庭成员）信息。每个就诊人需要填写真实姓名和证件号码，部分场景需要填写所在地区和详细地址。6岁以下儿童需要绑定监护人信息。已实名认证的就诊人部分字段会被锁定。

## 触发条件

- **前置条件**：已登录小程序，处于患者角色
- **入口路径**：
  - 添加就诊人：从健康档案页点击"添加就诊人"按钮进入
  - 编辑就诊人：在就诊人列表中点击某个就诊人进入编辑页
  - 问诊流程中：发起问诊时如果未选择就诊人，也会引导添加

## 步骤详解

### 第一步：选择证件类型

支持四种证件类型：
- 身份证（默认）、护照、港澳居民来往内地通行证、台湾居民来往大陆通行证
- 选择非身份证类型时，会额外显示"性别"和"出生日期"字段（因为非身份证证件号无法自动解析性别和年龄）

### 第二步：填写就诊人信息

**基础信息**：
- **就诊人姓名**（必填）：输入真实姓名
- **证件号码**（必填）：输入证件号码后，系统自动校验格式：
  - 身份证：18位，自动解析性别和年龄，显示"男/女 | X岁"
  - 护照：9位字母+数字
  - 港澳通行证：H/M + 8位数字
  - 台湾通行证：8位数字
- **成员关系**（必填）：选择与您的关系（本人/父母/配偶/子女/亲属/其他）
- **身高**（海迪设备场景下必填）：仅输入正整数（cm），用于海迪设备身高测量

**地址信息**（根据配置决定）：
- 不收集（配置=1）：不显示地址相关字段
- 收集必填（配置=2）：必须选择省市区+填写详细地址
- 收集非必填（配置=3）：可选填写

### 第三步：年龄校验与监护人

- **身份证用户**：自动从证件号解析年龄
- **非身份证用户**：手动选择出生日期，日历限制不能选择6年内日期
- **严格模式**：不允许6岁以下儿童问诊，弹出提示"不满6岁儿童不能在互联网医院进行问诊，请到线下实体医院问诊"
- **非严格模式+6岁以下儿童**：弹出监护人信息弹窗，需填写：
  - 监护人姓名
  - 监护人身份证号
  - 监护人与儿童的关系

### 第四步：保存提交

点击"保存"按钮触发提交前校验：
1. 就诊人姓名不能为空
2. 证件类型不能为空
3. 证件号码不能为空
4. 非身份证类型必须选性别和出生日期
5. 添加模式需校验证件号格式
6. 成员关系必须选择
7. 海迪场景下身高不能为空且必须为正整数
8. 地址必填模式下省市区和详细地址不能为空

提交后显示"实名信息认证中，请稍等"，认证通过后自动返回上一页。

### 编辑就诊人

- 已认证的就诊人：姓名和证件号码字段被锁定（灰色+锁图标），不可修改
- 编辑模式下同样触发年龄校验和监护人逻辑
- 编辑成功后显示"信息编辑成功"提示，2秒后自动返回

### 删除就诊人

- 点击"删除就诊人"按钮，弹窗提示"删除就诊人后相关病历也一并删除，确认要删除吗？"
- 确认后执行删除，同时清除该就诊人的问诊缓存（inquirySync），并通知其他页面重置就诊人信息

### 海迪设备登录

如果当前环境中存在海迪设备登录参数：
- 添加/编辑就诊人成功后自动调用海迪设备登录接口
- 登录成功后提示"确认成功"，1秒后返回
- 如果二维码过期（错误码400406），弹出提示"二维码已过期，请点击屏幕二维码后重新扫码登录"
- 重新扫码后自动重试绑定

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 证件号格式错误 | 输入框下方显示红色提示"证件号格式有误，请检查后更正提交" |
| 同一就诊人被重复绑定 | 弹窗提示"同一就诊人只能绑定在一个手机号，当前就诊人已绑定XXX，是否进行解绑？"，可点击"去解绑"跳转解绑页 |
| 6岁以下儿童（严格模式） | 弹窗提示"不满6岁儿童不能在互联网医院问诊"，只能点"我知道了"关闭 |
| 6岁以下儿童（非严格模式） | 弹出监护人信息弹窗，必须填写监护人后才能保存 |
| 海迪设备二维码过期 | 弹窗提示重新扫码登录 |
| 删除就诊人取消 | 弹窗点击"取消"，关闭弹窗不删除 |
| 提交时校验不通过 | 逐项弹出红色提示，停留在当前页 |

## 关联功能

- **健康档案列表**（`healthRecords/list.vue`）：就诊人列表页，可查看和管理所有就诊人
- **就诊人解绑**（`getAuthCode/index.vue`）：重复绑定时的解绑流程，需短信验证码
- **问诊流程**：添加就诊人后自动回填到问诊单中
- **海迪设备**：就诊人绑定后可与海迪身高体重设备联动
