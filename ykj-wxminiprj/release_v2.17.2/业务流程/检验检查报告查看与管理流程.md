# 检验检查报告查看与管理流程

> 文档版本：v1.0 | 面向读者：患者、医生

## 概述

患者和医生可以在小程序中查看来自线下医院HIS系统同步的检验报告和检查报告。检验报告展示各项指标数据（项目名称、检验结果、参考值、单位），检查报告展示影像图片和诊断结论。患者端还可以查看体测报告（需配置海迪体测仪）。

## 触发条件

### 前置条件
- 就诊人已在系统中绑定
- 医院HIS系统已将报告数据同步到平台
- 体测报告需要系统开启了「海迪体测仪」功能开关（`haidiEnabled`）

### 入口路径
- 患者端：从「我的」→「报告查询」进入
- 医生端：从患者详情页进入，需携带患者ID参数（`patientId`）

## 步骤详解

### 第一步：选择就诊人和报告类型

用户进入报告查询页后看到：
- **就诊人横滑切换**（仅患者端）：顶部展示所有已绑定的就诊人，默认选中第一个。切换就诊人后自动刷新报告列表。
- **报告类型切换**：页面右上角提供三个选项卡：

| 选项卡 | 对应类型 | 说明 |
|--------|---------|------|
| 检查报告 | type=2（默认） | 影像检查报告，如X光、CT、B超等 |
| 检验报告 | type=1 | 化验报告，如血常规、尿常规等 |
| 体测报告 | type=3 | 仅在开启海迪体测仪时显示，含体成分和体态两类 |

- 切换类型时自动重置页码并刷新列表，每页展示10条记录。

### 第二步：浏览报告列表

报告列表以卡片形式展示，每张卡片包含：
- **报告名称**（如「胸部CT平扫」「血常规」）
- **检查号**：优先显示医院原始编号（`hisReportNo`），无则显示系统编号
- **检查时间**
- **报告医生 | 检查科室**（以分隔符「I」连接）
- **「查看报告」按钮**（蓝色主题色）

滚动到底部自动加载下一页，已全部显示时提示「已显示全部」。无数据时展示「暂无检查报告」或「暂无检验报告」空状态。

### 第三步：查看报告详情

点击「查看报告」进入详情页，根据报告类型展示不同内容：

**A. 检验报告详情**

顶部医院名称和「检验报告单」标题，包含：
- **患者信息区**：姓名、性别、年龄、申请科室、申请医生、检查科室、检查医生、检查项目
- **指标数据表**：四列表格 —— 项目名称 | 检验结果 | 参考值 | 单位（每行一项指标，空字段显示「-」）
- **诊断结果意见**区域（空则显示「无」）
- **备注**区域（空则显示「无」）
- **报告签署区**：报告医生姓名+手写签名图片、报告时间、审核医生、审核时间
- **电子报告入口**：如有电子报告图片，显示「查看电子报告」按钮，点击可全屏预览
- **免责声明**：底部注明「本报告仅供临床医师参考不作证明之用」

**B. 检查报告详情**

与检验报告结构相似，但指标数据表替换为：
- **影像图片区**：一张图片占满整行，多张图片一行两张排列，点击可放大预览
- **检查所见**：描述影像所见内容（空则显示「无」）
- **诊断结果意见**（空则显示「无」）
- **查看影像按钮**（仅当存在`dicom_url`即DICOM医学影像链接时显示）：跳转到外部医学影像浏览器

**C. 报告文件类型处理**

如果报告的数据类型是文件（PDF或图片）而非结构化数据：
- PDF：使用系统PDF查看器打开
- 图片（png/jpg/jpeg）：全屏展示图片

### 第四步：患者端的后续操作

患者查看报告详情后，底部显示「去问诊」按钮，点击跳转到指定医生搜索页面，方便患者携带报告结果去咨询医生。

## 体测报告特殊说明

体测报告来源于第三方海迪体测仪设备，提供两类评估：
- **体成分**（reportType=1）：蓝绿渐变背景，展示身体成分分析
- **体态**（reportType=2）：橙色渐变背景，展示体态评估

体测报告列表支持额外筛选：
- **类型筛选**：全部 / 体成分 / 体态
- **日期筛选**：日历组件仅高亮有检测记录的日期，避免用户选择空日期

点击体测报告跳转到海迪小程序内的报告详情页（跨小程序跳转）。

底部备注：「本报告数据来源于海迪体测仪，结果仅供参考。如有健康疑问，请咨询专业医生。」

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未选择就诊人时进入 | 患者端自动选第一个就诊人并加载数据；医生端以传入的`patientId`为准 |
| 报告id缺失 | 弹出提示「报告id缺失」，不展示任何内容 |
| 报告为PDF文件 | 自动调用PDF查看器打开 |
| 报告文件后缀不是图片 | 跳转到结构化详情页而非文件预览 |
| 体测报告无可用日期 | 日期选择器不打开，提示「暂无可选日期」 |
| 体测仪功能未开启 | 体测报告选项卡不显示 |
| 无报告数据 | 显示空状态插图和对应提示文字 |

## 关联功能

- 报告详情页底部「去问诊」→ 跳转到医生搜索页
- 检查报告「查看影像」→ 跳转外部DICOM浏览器
- 电子报告图片 → 全屏图片预览
- 体测报告 → 跳转到海迪小程序报告页
