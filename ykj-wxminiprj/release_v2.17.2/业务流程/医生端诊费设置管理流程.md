# 医生端诊费设置管理流程

> 文档版本 2026-07-02 | 面向读者：运营、产品、客服

## 概述

医生可在小程序端独立配置自己的各项服务收费标准。包括图文问诊、视频问诊、咨询服务、快速续方、会诊服务、复诊服务等六种服务类型的价格和时长设置。每种服务可独立开启或关闭。

## 触发条件

- **前置条件**：医生已入驻并登录
- **入口路径**：医生端"我的"→"诊费设置"
- **岗位参数**：页面携带 positionCategory 参数（医生岗位类型）

## 步骤详解

### 第一步：系统加载当前服务配置

进入页面后，系统同时查询：
1. **当前医生的开通服务**（getCurrentDoctorServicePro）：获取每种服务的启用状态、当前价格、当前时长
2. **医院费用设置范围**（getConsultationFeesettings）：获取该岗位类型的费用区间（feeAreaMin～feeAreaMax）

如果没有开通任何服务，页面显示空状态："暂未开启接诊服务，请联系管理人员开启"。

### 第二步：查看各项服务

页面按服务类型排序展示，每种服务一张卡片：

| 服务类型 | 图示提示 | 可设置项 |
|---------|---------|---------|
| **图文问诊**（serviceType=1） | "可接诊和开处方" | 开关、价格（0-200元建议区间）、时长（最低xx小时） |
| **视频问诊**（serviceType=2） | "可接诊和开处方" | 开关、价格、时长（最低xx分钟） |
| **咨询服务**（serviceType=3） | "仅做咨询不开处方" | 开关、价格、时长（最低xx小时） |
| **快速续方**（serviceType=5） | "开通后患者可通过历史处方快速续方" | 开关、价格 |
| **会诊服务**（serviceType=6） | - | 开关、价格（元/次） |
| **复诊服务**（serviceType=7） | "启用后医生可通过患者病历详情发起复诊" | 开关、价格（元/次） |

每种服务最右侧有开关按钮，开启后下方显示价格和时长设置入口。

### 第三步：开启/关闭服务

点击开关按钮切换服务状态：

- **快速续方特殊校验**：开启快速续方时，必须先已开启图文问诊。如果图文未开启，弹窗"请先开启图文问诊"并提供确认按钮（同时开启图文和快速续方）或取消
- **关闭服务**：直接关闭，无需额外确认

### 第四步：设置问诊/咨询价格（图文、视频、咨询、快速续方）

点击价格区域，底部弹出价格选择面板：

1. **手动输入**：可自由输入数字金额
2. **快捷选择**：底部有预设金额按钮（0.00 / 20.00 / 40.00 / 60.00 / 80.00 / 100.00 / 120.00 / 140.00），点击快速填入
3. 价格范围受医院费用设置约束，输入框显示提示"建议价格区间为0-200元"
4. 点击"确认"提交

### 第五步：设置问诊/咨询时长

点击时长区域，弹出时长设置对话框：

- **图文问诊**：输入小时数，最小值为配置的 imageTextMinDuration（默认0.5小时），提交时自动转换为分钟
- **视频问诊**：输入分钟数，最小值为配置的 videoMinDuration（默认30分钟）
- **咨询服务**：输入小时数，最小值为配置的 consultMinDuration（默认12小时），页面显示提示"咨询服务时长至少为12小时以上"

点击确认后保存。

### 第六步：设置会诊价格

点击会诊价格区域，弹出专用对话框：

1. 输入金额（元/次）
2. 校验：不能大于医院费用上限，不能小于下限
3. 超出上限提示"诊费价格不能大于xxx"
4. 低于下限提示"诊费价格不能小于xxx"
5. 点击"确认"保存

### 第七步：设置复诊价格

点击复诊价格区域，弹出对话框输入金额（元/次），确认后保存。

## 费用设置范围

每种服务的价格范围由医院管理员在后台配置（feeAreaMin～feeAreaMax），默认 0～99999 元。医生设置的价格必须在此范围内。

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 无任何服务 | 显示空状态插画和文字"暂未开启接诊服务，请联系管理人员开启" |
| 开启快速续方时图文未开启 | 弹窗"请先开启图文问诊"，提供确认（同时开启）和取消按钮 |
| 图文问诊时长低于最小值 | 提示"时长最小为xx个小时" |
| 视频问诊时长低于最小值 | 提示"时长最小为xx分钟" |
| 咨询时长低于最小值 | 提示"时长最小为xx小时" |
| 会诊价格超过上限 | 提示"诊费价格不能大于xxx" |
| 会诊价格低于下限 | 提示"诊费价格不能小于xxx" |

## 关联功能

- **医生入驻**：医生入驻后才能设置诊费
- **患者端问诊**：患者看到的价格即医生设置的诊费
- **后台费用管理**：管理员可配置费用区间
