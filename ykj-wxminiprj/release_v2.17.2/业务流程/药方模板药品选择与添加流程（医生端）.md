# 药方模板药品选择与添加流程（医生端）

> 文档版本：2026-07-06 | 面向读者：医生用户

## 概述

医生在创建或编辑药方模板时，通过药品搜索和选择功能从药房药品库中添加药品到药方中。支持药品搜索、用法用量配置、库存状态展示和防重复添加。

## 触发条件

- **前置条件**：医生已登录，正在编辑药方模板
- **入口路径**：药方模板编辑页 → 点击"添加药品"按钮

## 步骤详解

### 第一步：进入药品选择页

医生通过事件通道接收药方模板编辑页已有的药品列表（用于标记已选药品），页面加载时：
- 自动获取药品使用频率字典（如每日1次、每日2次等）
- 获取用药途径字典（如口服、外用等）
- 获取每次用量单位字典（如片、粒、袋等）
- 默认展示药房下的全部药品列表

### 第二步：搜索药品

医生可在搜索框输入药品名称或拼音首字母搜索药品。搜索结果自动分页加载，滚动到底部自动加载更多。

### 第三步：查看药品状态

每个药品根据库存和状态显示不同按钮：

| 药品状态 | 显示按钮 |
|---------|---------|
| 有库存或无库存管理（库存量>0 或 无库存限制 或 库存=-1）且未禁用 | 显示「选择」按钮 |
| 开启库存管理且库存为0 | 显示「缺货」按钮（灰色不可点击） |
| 药品被禁用（status=0） | 显示「禁用」按钮（灰色不可点击） |

- 点击药品名称可跳转到药品详情页查看完整信息
- 如果是从药品详情页修改用法后返回，会自动将修改后的药品添加到已选列表

### 第四步：选择药品并配置用法

点击「选择」按钮弹出用法配置面板，医生可以设置：

- **每次用量**：默认取药品建议用量，可加减调整（最低为1）
- **用药途径**：从字典中选择（如口服、外用、含服等）
- **用药频次**：从字典中选择（如每日1次、每日2次等），自动关联周期单位（日/周）
- **使用天数**：默认取药品建议天数，可加减调整（最低为1）
- **药品数量**：默认1，可加减调整

配置完成后点击确定，药品加入已选列表。

### 第五步：限制与防重复

- **最多添加5个药品**：如果已选药品已达5个，选择新药品时提示"最多只能添加五个药品"
- **防重复**：如果选择的药品已在列表中，自动更新该药品的用法用量配置，不会重复添加
- **数量为0的药品自动过滤**：如果医生将药品数量改为0，该药品会被自动移出已选列表

### 第六步：完成选择

所有药品添加完毕后，点击底部「选好了」按钮返回上一页，通过事件通道将已选药品列表同步给药方模板编辑页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 缺少药房ID参数 | 提示"缺少关键id"，无法加载药品列表 |
| 药品库存为0 | 按钮显示"缺货"且不可点击 |
| 药品被禁用 | 按钮显示"禁用"且不可点击 |
| 已选药品已达5个 | 提示"最多只能添加五个药品" |
| 药品列表为空 | 显示空状态"暂无药品" |
| 已显示全部药品 | 列表底部显示"- 已显示全部 -" |

## 关联功能

- **药品详情**：点击药品可查看完整信息并修改用法用量
- **药方模板编辑**：选择完成后数据回传药方模板编辑页
- **药品搜索**：通过药品名称或拼音首字母搜索
