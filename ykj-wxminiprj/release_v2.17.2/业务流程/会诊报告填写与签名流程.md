# 会诊报告填写与签名流程

> 文档版本：release/v2.17.2 | 面向读者：医生

## 概述

群会诊中，主治医生和参会医生在会诊结束后分别填写会诊报告和会诊意见，并通过CA电子签名或手写签名完成提交。主治医生写的是"会诊报告"，参会医生写的是"会诊意见"，两者在同一填写页面中根据医生身份自动区分。

## 触发条件

### 前置条件
- 医生已受邀加入某个群会诊
- 会诊申请已通过审核进入适宜填写阶段
- 当前角色必须是"医生"

### 入口路径
- 患者端：会诊报告列表页 → 点击"立即填写"按钮
- 医生端：会诊详情 → 进入报告列表 → 点击"立即填写"

## 步骤详解

### 第一步：进入会诊报告页

进入后会看到：
- **患者信息区**：如果当前医生是主治医生（自己的问诊单），顶部显示"xxx（男/女 | N岁）的会诊报告"。如果是参会医生看别人的问诊单，显示"xx医生的会诊意见"并附上患者基本信息卡片（姓名、性别、年龄、监护人关系等）。
- **大文本输入区**：默认高度占屏幕一半的文本输入框。主治医生看到的提示文字为"请输入会诊报告"，参会医生看到的为"请输入会诊意见"。

### 第二步：填写报告内容

在文本框中输入会诊报告或会诊意见内容。输入内容后，底部的"确认提交"按钮由灰色变为可点击状态。（按钮在输入框为空时禁用）

### 第三步：确认提交

点击"确认提交"按钮后的流程分支：

**如果是主治医生（报告提交人）**：弹出二次确认弹窗，提示"治疗方案提交后，会诊医生不能再填写会诊报告"，确认后进入签名流程。

**如果是参会医生（写会诊意见）**：不弹二次确认，直接进入签名流程。

### 第四步：CA电子签名流程

提交报告后，系统根据后台配置决定签名方式：

1. **如果医院开通了CA电子签章**：
   - 系统自动获取医生的CA认证状态
   - **未认证（authStatus=1）**：弹出提示"未完成CA电子签章，请先完成签章"，引导医生同步CA账号资料
   - **认证中（authStatus=2）**：弹出"认证中"提示，可选择前往认证页面
   - **认证成功（authStatus=3）**：根据签章平台分支处理——医网信/源康健PaaS会弹出电子签章组件让医生进行电子签名；放心签会打开放心签签名弹窗；腾讯电子签暂不支持
   - **认证失败（authStatus=4）**：弹出"认证失败"提示

2. **如果医院未开通CA电子签章**：跳转手写签名板页面，医生在白板上手写签名

### 第五步：签名完成

签名成功后自动返回上一页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 没有填写任何内容就点"确认提交" | 按钮为灰色，无法点击 |
| CA认证状态为"未认证" | 弹出提示框，引导同步认证资料 |
| CA认证状态为"认证失败" | 弹出提示框，告知认证失败 |
| 签名组件加载失败 | 弹出提示"签字失败，请重试" |
| 医生退出签名页面 | 放心签平台区分"取消"和"出错"两种情况，取消时不做处理，出错时提示重试 |
| 医生的CA账号尚未同步 | 弹出提示"未完成CA电子签章，请先完成签章" |

## 关联功能

- **会诊报告列表页**：填写完成后返回报告列表，可看到已填写的内容
- **会诊报告详情页**：点击已填写的报告/意见，可查看完整的报告内容、提交医生签名、提交时间
- **会诊详情页**：可查看整个会诊的状态和参与医生信息
- **手写签名板**（packagePatient1/pages/signatureSketchpad）：未开通CA时的备选签名方式
- **放心签设置**（packageDoctor1/pages/fxqSetting）：放心签签章密码和签章设置
- **医网信签章**（packageDoctor1/pages/ywx）：医网信相关签章操作
