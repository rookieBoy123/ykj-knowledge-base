# 医生端会诊申请提交与签章流程

> 文档版本 2026-07-02 | 面向读者：运营、产品、客服

## 概述

医生在问诊过程中，发现患者病情复杂需要多科室协作时，可以发起会诊申请。填写会诊信息后，需完成电子签章（CA签章或手写签名）才能正式提交。会诊申请需经过审核和患者确认两个后续环节。

## 触发条件

- **前置条件**：医生正在接诊中（有inquiryNo），且为中医医生（isTCM）
- **入口路径**：从问诊对话页发起会诊申请
- **草稿编辑**：如果已有草稿状态的会诊申请（consultationNo + flowStatus=0），进入编辑模式

## 步骤详解

### 第一步：系统自动加载基础信息

进入会诊申请页后，系统自动加载：
- **患者基本信息**：姓名、性别、年龄、监护人关系
- **病情描述**：从问诊记录中带出
- **会诊类型**：从字典（sqhzyy）加载可选的会诊原因
- **会诊目的**：从字典（hzmd）加载可选的目的选项

如果是**编辑草稿**（已有consultationNo），回显之前填写的内容。

### 第二步：填写当前诊断

医生需要填写当前诊断，根据医生类型不同：
- **中医**：需填写辨病（diseasesResult）和辨证（dialecticalList）
- **西医**：填写诊断（diagnosisList）

支持通过诊断搜索组件选择。

### 第三步：选择会诊原因和目的

两项皆为多选：
- **会诊原因**：患方要求 / 患者病情需要（从字典配置加载）
- **会诊目的**：明确诊断 / 指导治疗（从字典配置加载）

### 第四步：选择会诊科室

通过会诊科室组件选择需要参与的科室，可多选。每个科室包含科室名称和科室编号。

### 第五步：设置会诊时间和时长

- **预计开始时间**：选择会诊开始的具体时间
- **会议时长**：设置会诊持续时长

### 第六步：提交审核

点击底部"提交审核"按钮前，系统校验所有必填项：
- 病情描述不能为空
- 当前诊断不能为空（中医需辨病+辨证，西医需诊断）
- 会诊原因至少选一项
- 会诊目的至少选一项
- 会诊科室至少选一项
- 会诊时长不能为空
- 预计开始时间不能为空

校验通过后先订阅消息通知，然后执行提交：
- **新增**：调用consultationsApply创建会诊申请
- **编辑草稿**：调用consultationsEdit编辑已有草稿

### 第七步：电子签章

提交后进入签章环节，根据医院配置不同：

**方式一：CA电子签章**（ca.enabled=true）
1. 检查医生是否有CA用户编号（caUserNo）
2. 无CA编号：弹出CA认证引导，引导医生同步资料
3. 有CA编号且认证状态：
   - **未认证（authStatus=1）**：引导完善CA认证
   - **认证中（authStatus=2）**：提示认证进度
   - **认证成功（authStatus=3）**：
     - 医网信平台（platform=1/3）：检测本地证书→获取签章数据→打开签名组件
     - 放心签平台（platform=4）：检测签名图片→打开放心签签名弹窗
   - **认证失败（authStatus=4）**：显示失败原因，引导重新认证

**方式二：手写签名**（ca.enabled=false）
跳转到手写签名板页面，医生手写签名后自动提交

### 第八步：签章完成后

签章成功后，跳转到会诊状态详情页（statusDetail），可查看会诊的完整流程状态：
- 会诊申请 → 会诊审核 → 患者确认

## 会诊状态流转

页面顶部显示三个步骤的状态条：
1. **会诊申请**（当前步骤，isActive=true）
2. **会诊审核**（待处理）
3. **患者确认**（待处理）

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 缺少问诊数据 | 提示"缺少问诊数据，请联系管理员！" |
| 缺少会诊数据（编辑草稿） | 提示"缺少会诊数据，请联系管理员！" |
| 未完成CA签章 | 弹窗"未完成CA电子签章，请先完成签章"，提供取消/确定按钮 |
| 医网信本地证书缺失 | 弹窗引导前往医网信页面安装证书 |
| 放心签签名图片缺失 | 弹窗引导设置签名密码 |
| 签字失败 | 提示"签字失败，请重试" |
| 必填项未填 | "提交审核"按钮保持灰色禁用状态 |
| 订阅消息通知失败 | 不阻断提交流程 |

## 关联功能

- **会诊状态详情**（statusDetail）：查看会诊进度
- **医网信设置**（ywx/xz、ywx/rest、ywx/qz）：医网信证书管理
- **放心签设置**（fxqSetting/signPassword、fxqSetting/signSetting）：放心签签名管理
- **手写签名板**（signatureSketchpad）：手写签名
