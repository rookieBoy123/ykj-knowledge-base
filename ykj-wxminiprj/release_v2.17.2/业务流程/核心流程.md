# 云康健微信小程序 — 核心业务流程

> 文档版本：release/v2.17.2 | 面向读者：客服、运营、产品经理等非技术人员

---

## 流程一：在线图文问诊（患者发起 → 医生接诊 → 结束问诊）

这是平台最核心的业务流程，患者通过图文聊天的方式向医生咨询健康问题。

### 步骤说明

1. **患者选择医生**  
   患者在首页浏览医生列表，或通过搜索功能按科室、地区等条件找到合适的医生。

2. **选择问诊类型**  
   进入医生详情页，查看医生提供的服务项目（如图文问诊、视频问诊），选择"图文问诊"。

3. **填写病情信息**  
   系统引导患者填写问诊卡，包括：主要症状、持续时间、过往病史、用药情况等。此步骤也称为"填写问诊卡"。

4. **确认费用并支付**  
   系统展示问诊费用明细，患者确认后通过微信支付完成付款。

5. **等待医生接诊**  
   支付成功后，问诊进入"待接诊"状态。医生的待接诊列表会收到新的问诊请求。

6. **医生接诊**  
   医生查看患者填写的病情信息后，点击"接诊"按钮，问诊正式开始。

7. **图文交流**  
   患者和医生通过IM聊天界面进行图文沟通。患者可发送文字、图片描述病情；医生可询问细节、给出初步判断，并使用常用语提高回复效率。

8. **医生给出建议/开处方**  
   问诊结束后，医生可给出用药建议或开具电子处方。处方需经医生电子签名（签章）确认。

9. **患者查看建议/处方**  
   患者可在问诊详情中查看医生的用药建议和处方内容。

10. **评价医生**  
   问诊结束后，患者可对本次问诊进行评价打分。

## 流程二：视频问诊（预约 → 视频通话 → 处方/病历）

适用于需要"面对面"沟通的远程就医场景，患者可与医生进行实时视频通话。

### 步骤说明

1. **预约视频服务**  
   患者在选择医生后，选择"视频问诊"服务类型，查看医生的出诊时间并选择合适的时段进行预约。

2. **支付预约费用**  
   确认预约信息后，通过微信支付完成付款。

3. **等待预约时间**  
   预约成功后，患者可在"我的预约"中查看预约详情。到预约时间时，双方进入视频房间。

4. **进入视频房间**  
   医生和患者各自点击进入视频房间，系统基于腾讯TRTC技术建立实时音视频连接。

5. **视频问诊交流**  
   双方进行面对面的视频交流。医生可查看患者病情、询问病史。支持视频转接（将患者转给其他医生进行视频会诊）。

6. **医生编写病历**  
   视频问诊结束后，医生为患者编写电子病历，记录诊断结果和治疗建议。

7. **开具处方（可选）**  
   如有需要，医生可为患者开具电子处方。

8. **患者查看病案/处方**  
   患者可在问诊记录中查看病历详情和处方内容。

## 流程三：处方开药与购药（医生开方 → 患者支付 → 药品配送）

连接线上问诊与线下药品配送的完整闭环。

### 步骤说明

1. **医生开具处方**  
   在问诊过程中或结束后，医生通过"开具处方"功能为患者开药。医生可从个人药房中快速选择药品，也可使用处方模板快速生成。

2. **处方签名确认**  
   医生对处方进行电子签章，确保处方的合法性和真实性。

3. **患者查看处方**  
   患者在"用药建议"或"处方"页面查看医生开具的处方明细，包括药品名称、用法用量、注意事项等。

4. **下单购药**  
   患者确认处方后，点击"去购药"进入药品订单页面，选择配送方式（快递配送或到店自取）。

5. **填写收货地址**  
   如选择快递配送，需填写或选择收货地址。患者可在"地址管理"中提前维护常用地址。

6. **支付药费**  
   确认药品订单信息后，通过微信支付完成付款。部分地区支持医保支付。

7. **查看订单与物流**  
   支付完成后，患者可在"药品订单"中查看订单状态和物流轨迹。

8. **药品配送**  
   药品由合作药房发货配送，患者签收药品。

## 流程四：预约挂号（选择服务 → 预约时间 → 就诊）

为患者提供提前预约医生服务时间的功能。

### 步骤说明

1. **查看医生服务**  
   患者进入医生详情页，查看医生提供的服务项目列表（图文问诊、视频问诊等）。

2. **选择预约日期**  
   选择需要的服务类型后，查看该医生可预约的日期和时间段。

3. **填写预约信息**  
   选择就诊人（本人或家庭成员）、填写病情简要描述。

4. **确认预约并支付**  
   确认预约信息，完成费用支付后预约生效。

5. **查看预约**  
   在"我的预约"中查看预约详情。如需取消，可在规定时间内取消预约。

6. **按时就诊**  
   在预约时间，医生和患者通过图文或视频方式完成问诊。

## 流程五：医生随访管理（创建计划 → 执行任务 → 患者完成）

医生对患者进行长期健康跟踪和管理的功能。

### 步骤说明

1. **创建随访计划**  
   医生在"随访管理"中新建随访计划，可基于已有模板快速创建，也可自定义计划内容。

2. **配置随访内容**  
   随访内容可包含三种类型：
   - **患教资料**：推送给患者的健康教育文章或课程；
   - **提醒**：用药提醒、复诊提醒等；
   - **问卷**：需要患者填写的健康问卷。

3. **发起随访**  
   选择目标患者，设置随访周期和起止时间后发起随访。

4. **患者接收随访**  
   患者端在"我的随访"中可查看医生发起的随访计划，按任务要求完成学习、填写问卷等。

5. **医生查看进度**  
   医生可查看每个随访任务的完成情况，包括患者是否阅读了患教资料、问卷填写结果等。

6. **随访总结**  
   随访周期结束后，医生可对本次随访进行总结记录。

---

## 补充说明

- 以上流程中涉及支付环节均通过微信支付完成，医保相关功能通过HIS系统对接实现。
- 所有问诊过程产生的消息记录、处方、病历等信息均会长期保存，患者和医生可随时回看。
- AI辅助功能（AI预问诊、AI医生分身）可嵌入在图文问诊流程中，在正式医生接诊前由AI先行收集患者信息。
