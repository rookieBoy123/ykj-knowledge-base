# 会诊申请提交与签章流程（医生端）

> 文档版本：v1.0 | 面向读者：医生

## 概述

医生在问诊过程中，认为患者需要多学科专家会诊时，发起会诊申请。包括填写会诊信息、选择会诊科室和时间、提交申请并完成CA签章的全流程。

## 触发条件

- **前置条件**：医生已接诊患者，有问诊单号（inquiryNo）
- **入口路径**：问诊聊天室→发起会诊→进入会诊申请页
- **传入参数**：inquiryNo（问诊单号），如果是撤销后修改则额外传入consultationNo（会诊单号）

## 步骤详解

### 第一步：页面加载

进入会诊申请页后，页面顶部显示三步进度条：会诊申请（当前激活）→ 会诊审核 → 患者确认。

系统自动并行加载：
1. **字典数据**：会诊原因选项（字典`sqhzyy`）、会诊目的选项（字典`hzmd`）
2. **如果是编辑已有申请**（有consultationNo）：加载已有会诊详情并回填所有字段
3. **如果是新申请**：加载问诊卡信息，自动填充患者数据和病情描述

### 第二步：填写会诊信息

页面从上到下依次填写：

**1. 患者信息区**（只读）：
- 显示患者姓名、性别、出生日期、年龄（ageStr）
- 如有监护人，显示监护人关系

**2. 病情描述**：
- 从问诊卡自动带入，医生可修改
- 支持文本输入，不能为空

**3. 当前诊断**：
- 中医医生：显示辨病+辨证两个入口（不需要考虑diagnoseRequireEnabled）
- 西医医生：显示诊断入口
- 会诊申请必须填写诊断

**4. 会诊类型**：
- 固定显示"普通会诊"
- 不可修改

**5. 会诊原因**（多选）：
- 备选项从字典`sqhzyy`动态加载
- 如"患方要求""患者病情需要"
- 至少勾选一项

**6. 会诊目的**（多选）：
- 备选项从字典`hzmd`动态加载
- 如"明确诊断""指导治疗"
- 至少勾选一项

**7. 会诊科室**：
- 可多选科室（科室树选择器）
- 提交时取科室名称（label）和科室编号（departmentNo）

**8. 会诊时间和时长**：
- 选择会诊开始时间（meetingStartTime）
- 选择预计时长（meetingPlanDuration）

### 第三步：提交申请

「提交审核」按钮初始为灰色，需满足以下所有条件后才变为可点击：

1. 病情描述非空
2. 诊断已填写（中医需辨病+辨证，西医需诊断）
3. 会诊原因至少选一项
4. 会诊目的至少选一项
5. 会诊科室至少选一个
6. 开始时间和时长均已设置

点击「提交审核」：
1. 先触发微信订阅消息授权
2. 判断当前是会诊状态：
   - **草稿状态（flowStatus=0）**：调用`consultationsEdit`编辑已有会诊
   - **新申请**：调用`consultationsApply`创建新会诊
3. 提交成功后进入签章流程

### 第四步：CA签章

签章流程根据系统配置分支：

**如果CA已开启**：
1. 检查医生的caUserNo：
   - 为空 → 弹出"未完成CA电子签章，请先完成签章"，引导同步资料
   - 有值 → 查询CA信息（getCaInfo）

2. 根据认证状态处理：
   - 未认证（authStatus=1）→ 弹出引导弹窗，点击「确定」→同步资料到签章平台
   - 认证中（authStatus=2）→ 弹出引导去认证；医网信/源康健PaaS→调用`initSign`；腾讯电子签→跳转小程序认证
   - 认证成功（authStatus=3）→ 根据平台类型调用签章组件：
     - 医网信/源康健PaaS：检查本地证书→有则调用签章，无则弹窗提示下载证书
     - 放心签：检查本地签章图片→有则调用签章，无则弹窗提示
   - 认证失败（authStatus=4）→ 弹出引导重新认证

**如果CA未开启**：
直接跳转到手写签名板（signatureSketchpad）。

### 第五步：签章完成

- **医网信/源康健PaaS签章成功**→ 跳转到会诊状态详情页（statusDetail）
- **放心签签章成功**→ 跳转到会诊状态详情页
- **签章失败/取消**→ 关闭签章组件，显示错误提示或不做处理

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 表单不完整 | 「提交审核」按钮保持灰色，不可点击 |
| CA未认证 | 弹出引导弹窗，提示先完成签章认证 |
| 医网信/源康健PaaS本地无证书 | 弹出弹窗提示下载证书，可跳转ywx/xz页面 |
| 放心签本地无签章图片 | 弹出弹窗提示设置签章，可跳转signPassword |
| 签章平台=腾讯电子签 | 弹出提示"暂不支持" |
| 签章失败 | 显示"签字失败，请重试" |
| 微信订阅消息授权取消 | 仍然继续提交流程 |

## 关联功能

- **会诊审核**：申请提交后进入会诊审核流程
- **会诊详情**：通过statusDetail查看会诊进度和状态
- **CA签章管理**：可跳转到医网信下载证书页（ywx/xz）或放心签设置页（fxqSetting/signPassword）
