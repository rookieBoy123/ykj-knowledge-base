# 特色疗法服务包浏览与购买流程

> 文档版本：release/v2.17.2 | 面向读者：客服、运营、产品经理

## 概述

患者通过"特色疗法"入口浏览平台推出的各类健康服务套餐（如产后康复包、慢病管理包等），查看套餐包含的服务项目和服务介绍，确认购买后完成微信支付，购买后可在"专属权益"中查看和使用已购服务。

## 触发条件

- 从首页导航菜单点击"特色疗法"入口
- 从个人中心相关入口进入
- 通过微信分享链接进入

## 前置条件

- 患者已登录
- 平台已配置至少一个健康服务包

## 步骤详解

### 第一步：浏览服务包列表

进入"特色疗法"页面后：

- 顶部搜索框：输入关键词搜索服务包（如"康复"、"慢病"），输入后自动重新搜索
- 服务包卡片列表：每个卡片展示封面图（无封面图时显示默认占位图）、服务包名称、包含的服务项目摘要（`itemsStr`）、价格、去购买按钮
- 支持滚动到底自动加载更多（每页10条）
- 如果搜索无结果或列表为空，显示空状态"暂无数据"

### 第二步：查看服务包详情

点击任意服务包卡片进入详情页，页面顶部展示：
- 封面图（160×160像素）、服务包名称、价格

**两个选项卡切换**：

1. **服务详情**（默认选中）：
   - 包含的服务项目：逐项列出服务包内的具体项目（如"图文问诊3次"、"体检套餐1次"等）
   - 服务介绍：富文本内容，展示详细的服务说明

2. **购买须知**：
   - 展示购买相关的注意事项（富文本内容）
   - 如果后端未配置，显示"无"

### 第三步：确认购买

底部固定区域显示：
- **协议勾选**：复选框 + "我已阅读并同意《服务协议》与《隐私策略》"
  - 点击《服务协议》跳转查看服务协议全文
  - 点击《隐私策略》跳转查看隐私策略全文
- **立即购买按钮**：显示"立即购买 ¥XX"

点击"立即购买"前的校验：
- 未勾选协议：提示"请先阅读并同意服务协议与隐私策略"
- 已勾选协议：先调用预下单接口（`prePurchase`）校验购买资格，成功后调用下单接口（`purchase`）生成订单，然后跳转到微信支付页面完成付款

### 第四步：查看已购权益

购买成功后，进入"专属权益"（会员权益）页面：

- **两个选项卡**：生效中 / 已失效
- 生效中的权益：列表展示已购买且在有效期内的服务包，每个卡片显示服务包名称和可用服务项目
- 已失效的权益：展示已过期或已用完的服务包
- 点击某个权益的服务项目：弹出底部面板，展示该服务项目的详细说明（富文本内容）
- 支持滚动加载更多（每页10条）

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 未勾选协议点击购买 | 提示"请先阅读并同意服务协议与隐私策略" |
| 预下单校验不通过 | 接口返回错误信息（如库存不足、不在售卖期等），不生成订单 |
| 微信支付取消/失败 | 返回详情页，可重新点击购买 |
| 搜索无结果 | 显示空状态"暂无数据" |
| 服务包列表为空 | 同上 |
| 已失效权益无数据 | 显示"暂无记录" |
