# 医生端病历信息查看流程

> 文档版本：2026-07-02 | 面向读者：医生用户

## 概述

医生在小程序中查看指定患者的完整病历信息，包含患者基本信息、已登记慢病病种、诊疗记录入口、已登记病历列表。支持医生查看线下病历、量表记录、随访计划和检查报告，并可对病历记录进行微信分享。

## 触发条件

- 前置条件：以"医生"角色登录小程序
- 入口路径：医生端 → 患者管理 → 点击某位患者 → 查看病历信息

## 步骤详解

### 第一步：查看患者基本信息

页面顶部显示患者信息卡片：
- 患者头像（根据年龄和性别自动匹配：6岁以下显示儿童头像，男性显示男头像，女性显示女头像）
- 患者姓名、性别、年龄
- 如有监护人，显示"监护人：XXX（关系）"

### 第二步：查看患者标签

患者基本信息下方显示患者标签区，医生可以：
- 查看已有的患者标签
- 添加或修改患者标签，修改后自动保存
- 标签更新失败时弹窗提示"就诊人信息更新失败，请稍后重试"

### 第三步：查看诊疗记录入口

显示四类诊疗记录快捷入口，根据系统配置和角色权限动态显示：
1. **线下病历**：查看患者在合作医院的线下就诊记录（医生角色显示）
2. **量表记录**：查看患者完成的量表测评记录（需开启量表功能+医生角色）
3. **随访计划**：查看该患者的随访计划（需开启随访功能+医生角色）
4. **查报告**：查看患者的检查检验报告（需开启报告功能+医生角色）

### 第四步：查看已登记慢病病种

如果系统开启了医保支付功能且患者已登记慢病病种，显示"已登记慢病病种"区域：
- 以标签形式展示所有已登记的慢病病种名称
- 如未登记或无医保功能，该区域不显示

### 第五步：浏览病历记录列表

页面下方显示患者的线上病历记录列表：
- 每条病历卡片包含门诊号、医生、主诉、诊断（区分西医诊断和中医诊断）等信息
- 每页展示5条，滚动到底部自动加载更多
- 如果医生开通了复诊服务且系统开启复诊开关，病历卡片显示"复诊"按钮
- 支持将病历通过微信分享给患者（点击分享生成带门诊号的处方详情链接）

## 异常处理

| 情况 | 系统处理方式 |
|------|------------|
| 修改患者标签失败 | 弹窗提示"就诊人信息更新失败，请稍后重试" |
| 病历记录查询失败 | 停止加载状态 |
| 病历为空 | 不再显示加载动画 |
| 已到最后一页 | 停止滚动加载（不再触发） |
| 功能开关关闭 | 对应功能入口不显示（量表/随访/报告） |

## 关联功能

- 患者端健康档案：患者端查看自己的病历记录
- 医生端接诊：从问诊中跳转查看患者病历
- 随访计划：快捷入口直接跳转到该患者的随访计划列表
- 量表记录：快捷入口跳转到患者的量表测评记录
