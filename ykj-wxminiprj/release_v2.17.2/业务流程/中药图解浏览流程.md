# 中药图解浏览流程

> 文档版本：release/v2.17.2 | 面向读者：患者、普通用户

## 概述

患者通过"中药图解"功能浏览中药材知识库，支持按中药的"五味"或按"适应疾病"两种方式浏览，也可通过搜索框直接查找。

## 触发条件

### 前置条件
- 已登录小程序（患者角色）
- 无其他特殊限制

### 入口路径
- 中医百科模块 → 中药图解入口

## 步骤详解

### 第一步：进入中药图解列表

页面顶部显示大标题"中药图解"，下方是搜索框（提示文字"请输入关键词搜索"），再往下是两个筛选选项卡——"按五味"和"按疾病"。默认选中"按五味"。

页面分为左右两栏：
- **左侧分类列表**：纵向滚动的中药分类标签（如"甘味""苦味""辛味"等或"感冒""咳嗽"等疾病名），每个分类显示在独立的圆角卡片上。当前选中的分类高亮显示为金色渐变底色+白色文字。
- **右侧药材列表**：对应分类下的中药列表，每条显示药材封面图（正方形圆角缩略图）和药材名称，每页15条，滚动到底自动加载更多。

### 第二步：切换筛选维度

点击"按疾病"选项卡后：
- 左侧分类列表自动切换为疾病分类（如"感冒""咳嗽""头痛"等）
- 右侧药材列表随之刷新，展示该疾病维度下的药材

两种维度互斥，切换时对侧的筛选值自动清空。

### 第三步：选择具体分类

点击左侧分类列表中的任意一个分类（如"甘味"）：
- 该分类高亮选中
- 右侧药材列表立即刷新，只显示该分类下的药材
- 可以再次点击同一个分类取消选中

### 第四步：搜索药材

在搜索框中输入关键词（最多20个字）：
- 输入框右侧出现清除按钮
- 输入内容后自动触发搜索（无需按回车）
- 搜索结果按当前选中的筛选维度+分类+关键词三者叠加过滤

### 第五步：查看药材详情

点击列表中的任意药材，进入药材详情页。详情页包含：
- **顶部大图**：药材的封面插图（750×750像素），如果后台未配置图片则显示默认占位图
- **标题区**：药材拼音 + 药材名称（大号粗体）+ 功效摘要（灰色小字，最多两行）
- **内容区**：四个选项卡——"基本用法""药物作用""用药方法""注意事项"，每个选项卡下是富文本格式的详细内容，可滚动浏览
- **吸顶导航**：向上滚动超出标题区后，顶部导航栏会显示药材名称，分类选项卡吸顶固定，方便切换

### 第六步：分享

详情页支持微信分享，分享标题为当前药材名称。好友点击后直接进入该药材的详情页。

## 异常处理

| 情况 | 系统处理方式 |
|------|-------------|
| 后台未配置药材封面图 | 显示默认占位图 |
| 某分类下没有药材 | 显示空状态图+文字"暂无数据" |
| 搜索关键词无匹配结果 | 同样显示空状态 |
| 后台字典未配置分类 | 分类列表为空，药材列表也可能为空 |
| 详情页富文本内容为空 | 对应选项卡下内容区为空白 |

## 关联功能

- **中医百科首页**：中药图解是中医百科的子模块
- **疾病知识**（packagePatient2/pages/medWiki/diseaseKnowledge）：同属中医百科的另一个模块
- **经典名方**（packagePatient2/pages/medWiki/classicFamousPrescriptions）：按经典名方浏览
- **药膳食疗**（packagePatient2/pages/medWiki/medicatedDietTherapy）：药膳食疗知识
- **经络穴位**（packagePatient2/pages/medWiki/meridiansAcupoints）：经络穴位图解
