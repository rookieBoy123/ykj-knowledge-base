# 变更日志 2026-07-03 (第10轮)

## 新增业务流程（4篇）

1. **医生人脸识别校验流程** — 医生登录和开方时通过摄像头录制视频进行人脸身份验证，支持摄像头/麦克风权限检测、授权引导和识别失败重试。根据 type 参数区分登录校验（login）和开方校验（rp）。
2. **患者快速续方全流程** — 患者从6个月内历史处方中选择处方进行快速续方，系统检查医生是否开通快速续方服务→是否有进行中的问诊→医生是否在线，必要时匹配新医生或引导去问诊。
3. **专家团队浏览与问诊流程** — 患者浏览专家团队列表→查看团队详情（首席专家、服务流程、团队成员、服务内容）→选择非首席专家的团队医生发起问诊。支持关键字搜索。
4. **AI问诊报告查看流程** — 展示AI预问诊后生成的报告，包含主诉、七类病史、体检信息、AI诊断结果。页面明确提示AI诊断仅供参考。
